## 知识点讲解

同学们，刚才我们聊到了“代码注释”的重要性。那么，到底什么是注释呢？

1.  **什么是注释？**
    注释是写在代码中，用来解释代码作用、功能、思路的文字。它们是写给我们程序员看的，不是写给计算机看的！计算机在执行程序的时候，会直接“跳过”注释，就像你读一本故事书，会跳过书上的插画说明一样，直接看故事内容。

2.  **为什么需要注释？**
    *   **帮助自己**：当你写完一段代码，过了一段时间再回头看，可能会忘记当时为什么要这样写。有了注释，你就能很快回忆起来。
    *   **帮助别人**：如果你和小伙伴一起写程序，或者你的程序被别人使用，注释能帮助他们快速理解你的代码，提高合作效率。
    *   **调试程序**：有时候我们想暂时禁用一行代码，又不想删除它，就可以把它变成注释。这样，这行代码就不会被执行，方便我们检查程序哪里出了问题。

3.  **C++ 中的单行注释 `//`**
    在 C++ 语言中，最常用的一种注释是单行注释，它的标志是两个斜杠 `//`。
    *   **语法**：当你在一行代码中写下 `//` 之后，从 `//` 开始到这行代码的末尾，所有内容都会被视为注释。
    *   **特点**：它只能注释一行内容。如果你想注释多行，需要每一行都加上 `//`。

    **举个例子**：
    想象一下你写了一句魔法咒语 `cout << "Hello, world!";`，它会让你在屏幕上看到“Hello, world!”。如果你想告诉别人这句咒语的作用，你就可以这样写：

    ```cpp
    cout << "Hello, world!"; // 这行代码会在屏幕上输出“Hello, world!”
    ```

    看，`//` 后面的文字就是注释，计算机在运行的时候会忽略它，只会执行 `cout << "Hello, world!";` 这部分。

    **还有一种多行注释 `/* ... */`**：
    C++ 还有另一种注释方式，叫做多行注释，用 `/*` 开始，用 `*/` 结束。它可以在 `/*` 和 `*/` 之间注释多行内容。比如：
    ```cpp
    /*
     这是多行注释的例子。
     它可以跨越多行。
     计算机也会忽略它。
    */
    ```
    不过，对于我们初学者来说，先掌握好 `//` 单行注释就足够啦！因为它简单、直观，用起来也很方便。我们以后会学到更多关于多行注释的用法。

4.  **什么时候加注释？加什么内容？**
    *   **关键代码**：当代码比较复杂，或者功能比较特别时，可以加注释。
    *   **变量说明**：解释变量是用来做什么的，比如 `int n; // n 用来存储输入的数字个数`。
    *   **功能模块**：一段代码完成一个特定任务时，可以在这段代码前面加一个注释来概括它的功能。
    *   **不是所有代码都需要注释**：太简单的代码，比如 `a = 1;` 就不需要注释“把 1 赋值给 a”，因为这很明显。注释是为了让不明显的地方变得明显。

记住，好的注释就像指路牌，能让大家更容易地找到方向！

## 代码示例

下面我们来看一些 C++ 代码，并学习如何在其中添加注释。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    // 这是一个简单的 C++ 程序，演示如何使用注释

    // 声明一个整数变量 a
    int a;

    // 声明另一个整数变量 b
    int b;

    // 提示用户输入第一个数字
    cout << "请输入第一个数字: ";

    // 从键盘读取用户输入的第一个数字，并存储到变量 a 中
    cin >> a;

    // 提示用户输入第二个数字
    cout << "请输入第二个数字: ";

    // 从键盘读取用户输入的第二个数字，并存储到变量 b 中
    cin >> b;

    // 计算两个数字的和
    int s = a + b; // s 代表 sum (和)

    // 在屏幕上输出计算结果
    cout << "这两个数字的和是: " << s << endl;

    // 程序运行结束，返回 0 表示成功
    return 0;
}
```

**代码解释**：
*   `// 这是一个简单的 C++ 程序，演示如何使用注释`：这行注释说明了整个程序的目的。
*   `int a; // 声明一个整数变量 a`：这行注释解释了 `int a;` 的作用，即声明一个用来存储整数的变量 `a`。
*   `cin >> a; // 从键盘读取用户输入的第一个数字，并存储到变量 a 中`：这行注释详细说明了 `cin >> a;` 的功能。
*   `int s = a + b; // s 代表 sum (和)`：这行注释不仅解释了 `int s = a + b;` 的计算过程，还说明了变量 `s` 的含义。
*   `// 程序运行结束，返回 0 表示成功`：这行注释说明了 `return 0;` 的含义。

通过这些注释，即使是第一次看到这段代码的同学，也能很快明白每一部分代码是用来做什么的，是不是很棒呢？

## 练习题目

同学们，我们来做几道小练习，检验一下今天学习的成果吧！

**题目部分**

**选择题：**

1.  在 C++ 语言中，以下哪个符号用于表示单行注释？
    A.  `/*`
    B.  `//`
    C.  `--`
    D.  `##`

2.  关于代码注释，以下说法正确的是：
    A.  注释是给计算机看的，它会执行注释中的指令。
    B.  注释会增加程序的运行时间。
    C.  注释的主要作用是提高代码的可读性，方便程序员理解。
    D.  注释只能放在代码的开头。

3.  下面这段代码，运行后会在屏幕上输出什么？
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;
    int main()
    {
        int a = 5; // 定义变量 a 并赋值为 5
        // int b = 10; // 定义变量 b 并赋值为 10
        cout << a << endl;
        return 0;
    }
    ```
    A.  5
    B.  10
    C.  5 10
    D.  什么都不输出

**填空题：**

4.  请在下面代码的横线处填写合适的注释，说明 `n` 变量的用途。
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;
    int main()
    {
        int n; // __________________________
        cout << "请输入一个整数: ";
        cin >> n;
        cout << "你输入的整数是: " << n << endl;
        return 0;
    }
    ```

5.  请在下面代码的横线处填写合适的注释，说明 `s = a + b;` 这行代码的作用。
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;
    int main()
    {
        int a = 10;
        int b = 20;
        int s;
        s = a + b; // __________________________
        cout << "和是: " << s << endl;
        return 0;
    }
    ```

---

**答案与解析**

**选择题答案：**

1.  **B. `//`**
    *   **解析**：在 C++ 中，`//` 用于表示单行注释。`/*` 是多行注释的开始，`--` 和 `##` 不是 C++ 中的注释符号。

2.  **C. 注释的主要作用是提高代码的可读性，方便程序员理解。**
    *   **解析**：
        *   A 错误，注释是给程序员看的，计算机在编译时会忽略注释，不会执行。
        *   B 错误，注释不会被编译成机器码，所以不会增加程序的运行时间。
        *   D 错误，注释可以放在代码的任何位置，只要符合语法规则即可。

3.  **A. 5**
    *   **解析**：代码中 `int b = 10;` 这行被 `//` 注释掉了，所以它不会被执行。程序只会定义变量 `a` 并输出 `a` 的值，即 5。

**填空题答案：**

4.  `n 用来存储用户输入的整数` (或其他类似描述，如：`定义一个整数变量 n`)
    *   **解析**：注释应该解释 `int n;` 的作用，即声明一个整数变量，并说明它的用途。

5.  `计算变量 a 和 b 的和，并将结果存储到变量 s 中` (或其他类似描述，如：`将 a 和 b 相加，结果赋给 s`)
    *   **解析**：注释应该说明 `s = a + b;` 这行代码的具体功能，即执行加法运算并将结果赋值给 `s`。
