## 趣味引入

同学们，大家有没有玩过这样的游戏：小明是机器人，你给他下指令。

如果我们想让小明做两件事：先“向前走5步”，再“向左转90度”。
之前我们是这样做的：
你：“小明，向前走5步。”
小明：`（走完）`
你：“小明，向左转90度。”
小明：`（转完）`

这样是不是有点慢呀？如果我想一次性告诉小明所有指令，让他连续完成呢？
比如，你一次性说：“小明，向前走5步，然后向左转90度！”
小明：`（走完，再转完）`

在C++编程中，我们之前学习了怎么让程序一次只输入一个数据。比如：
`cin >> a;` // 只能输入一个数给变量a

那如果我需要输入好几个数字或者字母，难道要写好几行 `cin` 吗？
比如想输入你的学号、身高和体重：
```cpp
// 这样写虽然可以，但有点麻烦
cin >> x; // 学号
cin >> h; // 身高
cin >> w; // 体重
```
有没有更“高效”的方法，像一次性给小明下达多个指令那样，让程序一次性接收多个数据呢？
答案是肯定的！今天我们就来学习这个超级方便的技巧！

## 知识点讲解

同学们，C++ 语言非常聪明，它允许我们使用一个 `cin` 语句就读取多个数据。这个秘诀就是利用 `>>` 运算符的“链式”功能！

1.  **单个数据输入回顾**
    我们知道，要输入一个整数给变量 `a`，我们这样写：
    ```cpp
    int a;
    cin >> a; // 用户输入一个整数，比如 10
    ```

2.  **多个数据一次性输入**
    如果我们要输入两个整数给变量 `a` 和 `b`，可以这样写：
    ```cpp
    int a;
    int b;
    cin >> a >> b; // 用户输入两个整数，比如 10 和 20
    ```
    是不是很简单？只需要在 `cin` 后面连续使用 `>>` 运算符，并跟上你想要接收数据的变量名就可以了！

3.  **输入时的分隔符**
    当程序执行 `cin >> a >> b;` 这条语句时，它会等待你输入。你需要输入两个数据，并且用**空格键**、**Tab 键**或者按一下**回车键**来把它们分开。
    *   例如，你可以输入 `10 20` 然后按回车。
    *   你也可以输入 `10` 按回车，再输入 `20` 按回车。
    *   你还可以输入 `10` 按 Tab 键，再输入 `20` 按回车。
    程序会聪明地识别这些分隔符，并把第一个数据给 `a`，第二个数据给 `b`。

4.  **不同数据类型混合输入**
    `cin` 也很棒，它甚至可以一次性读取不同类型的数据！
    比如，我们要输入一个字符 `c`、一个整数 `i` 和一个小数 `d`：
    ```cpp
    char c;
    int i;
    double d;
    cin >> c >> i >> d; // 用户输入：A 10 3.14
    ```
    只要你输入的顺序和数据类型与 `cin` 后面的变量顺序和类型匹配，程序就能正确地把数据存入对应的变量中。

5.  **小提示**
    *   **数据类型要匹配**：如果你要求输入整数，但用户输入了字母，程序可能会出错或者得到意想不到的结果。所以，在输入时要细心哦！
    *   **多余的输入**：如果你输入的数据比程序要求的要多，比如程序要求输入两个数，你却输入了三个，那么多出来的那个数据会被“忽略”或者留在输入缓冲区，等待下一个 `cin` 语句来读取。

现在，我们来看一些具体的代码例子，让大家亲手体验一下！

## 代码示例

我们来用代码实际操作一下，看看一次性输入多个数据是多么方便！

### 示例1：输入两个整数并计算它们的和

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    // 声明两个整数变量，用来存放用户输入的数据
    int a;
    int b;

    // 提示用户输入
    cout << "请输入两个整数（用空格隔开）：" << endl;

    // 使用一个 cin 语句，一次性读取两个整数
    cin >> a >> b;

    // 计算它们的和
    int s = a + b;

    // 输出结果
    cout << "你输入的第一个数是：" << a << endl;
    cout << "你输入的第二个数是：" << b << endl;
    cout << "它们的和是：" << s << endl;

    return 0;
}
```
**运行示例**：
如果你运行这个程序，当它提示你输入时，你可以输入 `15 25` 然后按回车。
程序会输出：
```
你输入的第一个数是：15
你输入的第二个数是：25
它们的和是：40
```
是不是很神奇？

### 示例2：输入一个字符和两个小数

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    // 声明一个字符变量和两个小数变量
    char c;
    double d1;
    double d2;

    // 提示用户输入
    cout << "请输入一个字符和两个小数（用空格隔开）：" << endl;

    // 使用一个 cin 语句，一次性读取不同类型的数据
    cin >> c >> d1 >> d2;

    // 输出你输入的数据
    cout << "你输入的字符是：" << c << endl;
    cout << "你输入的第一个小数是：" << d1 << endl;
    cout << "你输入的第二个小数是：" << d2 << endl;

    return 0;
}
```
**运行示例**：
如果你运行这个程序，当它提示你输入时，你可以输入 `K 1.23 4.56` 然后按回车。
程序会输出：
```
你输入的字符是：K
你输入的第一个小数是：1.23
你输入的第二个小数是：4.56
```
看，即使是不同类型的数据，`cin` 也能完美地一次性处理！太方便了！

4.  **填空题**：
    如果程序要求输入两个整数 (`cin >> x >> y;`)，但用户只输入了一个整数 `50`，按下回车后程序会 `__`。

    char c;

### 编程题

**基础练习**：
编写一个 C++ 程序，要求用户一次性输入三个整数，然后计算这三个整数的平均值并输出。
例如：输入 `10 20 30`，输出 `平均值是：20`。

**进阶练习**：
编写一个 C++ 程序，模拟一个简单的学生信息录入。
程序需要用户一次性输入：
1.  学生的学号（整数）
2.  学生的语文成绩（整数）
3.  学生的数学成绩（整数）
4.  学生的英语成绩（整数）
然后，程序需要输出该学生的学号，以及他的总分。
例如：输入 `1001 90 85 92`，输出 `学号1001的总分是：267`。

---

### 练习题目答案与解析

1.  **填空题答案**：`>>`
    **解析**：`>>` 是 C++ 中用于输入（提取）数据的运算符，它可以连续使用，实现一次性读取多个数据。

2.  **选择题答案**：A
    **解析**：`cin` 会按照变量的声明顺序依次将输入的数据赋值给变量。所以第一个输入 `10` 给了 `a`，第二个输入 `20` 给了 `b`。

3.  **选择题答案**：D
    **解析**：在 C++ 的 `cin` 默认行为中，空格、回车和 Tab 键都被视为数据的分隔符。逗号 `,` 则会被当作普通字符处理，而不是分隔符。如果输入 `10,20`，程序可能会尝试将 `10` 赋值给第一个变量，然后将 `,` 视为下一个输入的一部分，导致类型不匹配等问题。

4.  **填空题答案**：等待用户输入第二个整数
    **解析**：`cin` 语句会一直等待，直到它接收到足够的数据来填充所有指定的变量。如果只输入了一个整数，程序会暂停，等待用户输入第二个整数。

5.  **选择题答案**：B
    **解析**：`cin` 会按照变量的类型和顺序进行匹配。`'A'` 是一个字符，匹配 `char c`；`10.5` 是一个小数，匹配 `double d`。
