## 知识点讲解

### 1. 为什么需要 `double` 类型？

`int` 类型只能存储整数（例如：1, 100, -5），不能存储小数（例如：3.14, 0.5, -9.9）。
而现实世界中，很多数据都包含小数，比如身高、体重、价格、温度、圆周率等。
`double` 类型就是C++中用来表示**浮点数**（也就是带小数点的数字）的一种数据类型。

### 2. `double` 类型的声明与赋值

声明一个 `double` 变量的语法和 `int` 类似，只需要把 `int` 换成 `double`：

```cpp
double a; // 声明一个名为 a 的 double 类型变量
double b = 3.14159; // 声明并初始化一个名为 b 的 double 类型变量
```

你可以像给 `int` 变量赋值一样给 `double` 变量赋值：

```cpp
double price;
price = 19.99; // 给 price 变量赋值 19.99
double average = (85.5 + 90.0 + 92.5) / 3.0; // 也可以通过计算结果赋值
```
**注意：** 在进行小数计算时，为了保证结果是小数，至少有一个操作数要是小数类型（例如 `3.0` 而不是 `3`）。如果所有操作数都是整数，即使结果是小数，C++也会先进行整数除法，再赋值给 `double` 变量，这可能会导致精度损失。

### 3. `double` 类型的输入和输出

`double` 类型的变量同样可以使用 `cin` 进行输入，使用 `cout` 进行输出：

```cpp
double radius;
cout << "请输入圆的半径: ";
cin >> radius; // 从键盘输入一个小数，存储到 radius 中

double area = 3.14159 * radius * radius;
cout << "圆的面积是: " << area << endl; // 输出计算出的面积
```

**关于输出精度：**
默认情况下，`cout` 输出 `double` 类型时，可能会根据数值的大小自动调整输出的位数，或者使用科学计数法。在竞赛中，我们经常需要输出固定位数的小数。这时，我们需要用到一些特殊的设置。

虽然我们是Level 1，不需要深入理解，但知道怎么用可以帮助我们更好地控制输出：

```cpp
#include <bits/stdc++.h> // 包含所有常用头文件，也包括了iomanip
using namespace std;

int main()
{
    double a = 123.456789;
    double b = 1.0 / 3.0; // 结果是 0.3333...

    cout << "默认输出 a: " << a << endl;
    cout << "默认输出 b: " << b << endl;

    // 控制输出精度，例如保留 2 位小数
    cout << fixed << setprecision(2) << "保留两位小数输出 a: " << a << endl;
    cout << fixed << setprecision(2) << "保留两位小数输出 b: " << b << endl;

    // 如果想恢复默认输出，可以再次调用 setprecision(6) 或其他默认值
    cout << setprecision(6) << "恢复默认输出 a: " << a << endl;

    return 0;
}
```
*   `fixed`：表示以普通小数形式输出，而不是科学计数法。
*   `setprecision(n)`：表示输出 `n` 位小数（与 `fixed` 配合使用时）。

### 4. `double` 类型的运算特点（精度）

`double` 类型虽然可以存储小数，但它在计算机内部的存储方式决定了它不能**精确**表示所有的小数。有些小数，比如 0.1，在计算机内部是无法用有限的二进制位精确表示的，就像 1/3 无法用有限的十进制小数精确表示一样（0.333...）。

这意味着，当我们进行一些小数运算时，结果可能不是我们数学上期望的那么精确。例如：

```cpp
double x = 0.1;
double y = 0.2;
double z = x + y; // 理论上应该是 0.3

cout << fixed << setprecision(20) << z << endl; // 可能会输出 0.30000000000000004441
```
看到没？`0.1 + 0.2` 的结果可能不是严格的 `0.3`。对于 Level 1 的我们，只需要记住：**小数运算可能存在微小的误差，不要完全相信两个 `double` 变量是否“严格相等”**。未来我们会学习如何比较两个 `double` 变量是否“足够接近”。

但在大多数日常计算中，`double` 提供的精度已经足够高，可以满足我们的需求了。

## 代码示例

让我们来编写一个简单的程序，计算一个商品的总价。

```cpp
#include <bits/stdc++.h> // 包含所有常用的头文件
using namespace std;

int main()
{
    // 声明变量
    double unitPrice; // 商品单价
    int quantity;     // 购买数量
    double totalPrice; // 总价

    // 1. 获取商品单价
    cout << "请输入商品的单价 (例如 12.50): ";
    cin >> unitPrice;

    // 2. 获取购买数量
    cout << "请输入购买的数量 (整数): ";
    cin >> quantity;

    // 3. 计算总价
    // 注意：unitPrice 是 double，quantity 是 int。
    // 在乘法运算中，C++ 会自动将 int 类型的 quantity 转换为 double 类型，
    // 然后再进行 double 类型的乘法，确保结果是 double。
    totalPrice = unitPrice * quantity;

    // 4. 输出结果
    cout << "--------------------" << endl;
    cout << "您购买的商品单价是: " << fixed << setprecision(2) << unitPrice << " 元" << endl;
    cout << "购买数量是: " << quantity << " 件" << endl;
    cout << "总价是: " << fixed << setprecision(2) << totalPrice << " 元" << endl; // 通常价格会保留两位小数

    return 0;
}
```

**运行示例：**
如果输入单价 `12.99`，数量 `3`：
输出会是：
```
请输入商品的单价 (例如 12.50): 12.99
请输入购买的数量 (整数): 3
--------------------
您购买的商品单价是: 12.99 元
购买数量是: 3 件
总价是: 38.97 元
```

## 练习题目

### 选择题/填空题

1.  以下哪个数据类型最适合存储商品价格 `19.99`？
    A. `int`
    B. `char`
    C. `double`
    D. `bool`

2.  要声明一个名为 `height` 的 `double` 类型变量，并初始化为 `1.80`，正确的代码是：
    A. `int height = 1.80;`
    B. `double height = 1.80;`
    C. `height double = 1.80;`
    D. `double height; height = "1.80";`

3.  C++中，用于从键盘输入 `double` 类型数据的语句是 `cin >> ______;`。

4.  以下C++代码的输出结果是：
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;
    int main() {
        double x = 5.0;
        double y = 2.0;
        double z = x / y;
        cout << z << endl;
        return 0;
    }
    ```
    A. `2`
    B. `2.5`
    C. `2.0`
    D. `5/2`

5.  如果想让 `double` 变量 `pi` （值为 `3.1415926`）输出时只显示两位小数，应该使用 `cout << fixed << setprecision(______) << pi << endl;`。

---

### 答案与解析

1.  **答案：C**
    **解析**：`int` 只能存储整数，`char` 存储字符，`bool` 存储真假。`double` 是专门用来存储带小数的数字的。

2.  **答案：B**
    **解析**：`double` 是数据类型关键字，`height` 是变量名，`=` 是赋值运算符，`1.80` 是 `double` 类型的值。选项A类型错误，选项C语法错误，选项D将字符串赋给 `double` 变量是错误的。

3.  **答案：变量名**
    **解析**：`cin >> 变量名;` 是C++中通用的输入语法，适用于各种数据类型。

4.  **答案：B**
    **解析**：`x` 和 `y` 都是 `double` 类型，`5.0 / 2.0` 的结果是 `2.5`，也会是 `double` 类型。默认输出会显示 `2.5`。

5.  **答案：2**
    **解析**：`setprecision(n)` 用于设置输出的小数位数，当与 `fixed` 结合使用时，表示保留 `n` 位小数。
