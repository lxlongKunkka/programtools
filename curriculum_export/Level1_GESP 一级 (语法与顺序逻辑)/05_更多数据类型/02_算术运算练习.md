## 趣味引入

场景：小明去超市买零食，他买了 3 包薯片，每包 5 元，又买了 2 瓶果汁，每瓶 7 元。  
已知单价，引出：  
- 总花费是多少？  
- 平均每件商品花费多少？  
- 如果带 50 元，找回多少钱？  

通过这个真实场景，让学生带着问题进入算术运算的世界。

===NEXT===

## 知识点讲解

1. 算术运算符  
   - 加法：`+`  
   - 减法：`-`  
   - 乘法：`*`  
   - 除法：`/`（整型除法会向零取整）  
   - 取余：`%`（得到除法后的余数）

2. 运算优先级  
   - 先乘除后加减  
   - 同级运算从左到右  

3. 整型运算特性  
   - `a / b` 中如果 `a` 和 `b` 都是 `int`，则结果是整数，自动舍弃小数部分  
   - `a % b` 的结果范围是 `[0, b-1]`（当 a、b 为正数时）

4. 拓展：类型转换  
   - 如果要得到小数结果，需要将参与运算的变量转换为浮点数

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a = 3;            // 薯片袋数
    int b = 5;            // 薯片单价
    int c = 2;            // 果汁瓶数
    int d = 7;            // 果汁单价

    // 1. 计算总花费
    int totalCost = a * b + c * d;
    cout << "总花费: " << totalCost << " 元" << endl;

    // 2. 计算平均每件商品的价格（取整）
    int itemCount = a + c;
    int averageCost = totalCost / itemCount;
    cout << "平均每件商品花费: " << averageCost << " 元（整型运算）" << endl;

    // 3. 如果带 50 元，找回多少钱
    int change = 50 - totalCost;
    cout << "找回: " << change << " 元" << endl;

    // 4. 演示取余运算
    int modExample = totalCost % itemCount;
    cout << "余数示例 (totalCost % itemCount): " << modExample << endl;

    return 0;
}
```

关键行注释已放在代码中，帮助同学理解每步运算。
