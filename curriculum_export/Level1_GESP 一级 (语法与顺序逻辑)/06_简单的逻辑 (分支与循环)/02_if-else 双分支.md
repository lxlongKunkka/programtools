## 知识点讲解

我们已经知道，计算机程序会按照我们写的代码一行一行地执行。但是，有了条件判断语句，程序就可以变得更“聪明”，它会根据我们给定的条件来选择不同的执行路径！

### 1. 什么是双分支结构？

双分支结构，顾名思义，就是程序根据一个条件的判断结果，选择执行两个不同的“分支”中的一个。就像一条路分成了左右两条，你只能选择其中一条走。

*   **分支一**：当条件成立（真）时执行的代码。
*   **分支二**：当条件不成立（假）时执行的代码。

### 2. `if-else` 语句的语法

在 C++ 中，我们使用 `if-else` 关键字来实现双分支结构。它的基本语法是这样的：

```cpp
if (条件表达式)
{
    // 如果“条件表达式”为真（成立），就执行这里的代码
    // 这被称为 if 分支
}
else
{
    // 如果“条件表达式”为假（不成立），就执行这里的代码
    // 这被称为 else 分支
}
```

*   `if` 后面跟着一对小括号 `()`，里面写上我们要判断的**条件表达式**。
*   紧接着 `if (条件表达式)` 的是一对大括号 `{}`，里面是当条件为真时要执行的代码。
*   `else` 关键字表示“否则”的意思。
*   紧接着 `else` 的是一对大括号 `{}`，里面是当条件为假时要执行的代码。

**重要提示**：`if` 和 `else` 后面的两个大括号 `{}` 是用来包裹多行代码的。如果每个分支只有一行代码，那么大括号可以省略，但为了代码的清晰和避免出错，我们**强烈建议你总是使用大括号**！

### 3. 条件表达式

条件表达式的结果只能是“真”或“假”。我们通常使用**比较运算符**来构造条件表达式：

| 运算符 | 含义         | 示例       | 结果 (假设 a=5, b=3) |
| :----- | :----------- | :--------- | :------------------- |
| `>`    | 大于         | `a > b`    | 真                   |
| `<`    | 小于         | `a < b`    | 假                   |
| `>=`   | 大于等于     | `a >= b`   | 真                   |
| `<=`   | 小于等于     | `a <= b`   | 假                   |
| `==`   | 等于         | `a == b`   | 假                   |
| `!=`   | 不等于       | `a != b`   | 真                   |

### 4. `if-else` 的执行流程

当程序遇到 `if-else` 语句时，它会：

1.  首先计算 `if` 后面的**条件表达式**的值。
2.  如果条件表达式的结果是**真** (true)，那么程序会执行 `if` 后面大括号里的代码块。
3.  如果条件表达式的结果是**假** (false)，那么程序会跳过 `if` 后面大括号里的代码块，直接执行 `else` 后面大括号里的代码块。
4.  **注意**：`if` 和 `else` 两个分支的代码**只会有一个被执行**！程序会根据条件选择一条路径走，绝不会两条都走，也不会两条都不走。

让我们用一个简单的例子来理解：判断一个数是偶数还是奇数。
*   **条件**：这个数能被2整除吗？（即 `n % 2 == 0`）
*   **如果真**：输出“偶数”。
*   **如果假**：输出“奇数”。

是不是很简单呢？现在，让我们通过代码来实际操作一下！

## 代码示例

### 示例 1：判断一个数是正数还是非正数

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n; // 定义一个整数变量n
    cout << "请输入一个整数：";
    cin >> n; // 从键盘读取一个整数到变量n

    if (n > 0) // 判断条件：如果n大于0
    {
        cout << n << " 是一个正数。" << endl; // 如果条件为真，执行这里
    }
    else // 否则（如果n不大于0，即n小于等于0）
    {
        cout << n << " 是一个非正数。" << endl; // 如果条件为假，执行这里
    }

    return 0;
}
```

**运行结果示例：**
输入：`5`
输出：`5 是一个正数。`

输入：`-3`
输出：`-3 是一个非正数。`

输入：`0`
输出：`0 是一个非正数。`

### 示例 2：判断一个学生的成绩是否及格

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int s; // 定义一个整数变量s表示分数
    cout << "请输入学生的分数（0-100）：";
    cin >> s; // 从键盘读取分数

    if (s >= 60) // 判断条件：如果分数大于等于60
    {
        cout << "恭喜你，成绩及格了！" << endl; // 如果条件为真，执行这里
    }
    else // 否则（如果分数小于60）
    {
        cout << "很遗憾，成绩不及格，请继续努力！" << endl; // 如果条件为假，执行这里
    }

    return 0;
}
```

**运行结果示例：**
输入：`85`
输出：`恭喜你，成绩及格了！`

输入：`59`
输出：`很遗憾，成绩不及格，请继续努力！`

### 示例 3：判断一个数是奇数还是偶数

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int x; // 定义一个整数变量x
    cout << "请输入一个整数：";
    cin >> x; // 从键盘读取一个整数

    // 判断条件：x除以2的余数是否等于0。
    // % 是取模运算符，它计算除法的余数。
    if (x % 2 == 0) // 如果余数为0，说明是偶数
    {
        cout << x << " 是一个偶数。" << endl; // 如果条件为真，执行这里
    }
    else // 否则（余数不为0，即为1），说明是奇数
    {
        cout << x << " 是一个奇数。" << endl; // 如果条件为假，执行这里
    }

    return 0;
}
```

**运行结果示例：**
输入：`10`
输出：`10 是一个偶数。`

输入：`7`
输出：`7 是一个奇数。`
