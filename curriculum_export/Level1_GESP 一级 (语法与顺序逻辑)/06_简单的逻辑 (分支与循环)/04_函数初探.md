## 知识点讲解

1. 引入头文件  
   ```cpp
   #include <bits/stdc++.h>
   using namespace std;
   ```
   `bits/stdc++.h` 集成了所有常用头文件，包括 `<cmath>` 等。

2. 常用数学函数  
   - floor(x)：向下取整，返回不大于 x 的最大整数。  
   - ceil(x)：向上取整，返回不小于 x 的最小整数。  
   - round(x)：四舍五入，返回最接近 x 的整数（.5 向远离 0 的方向）。  
   - sqrt(x)：开平方，返回 x 的非负平方根。  
   - pow(x, y)：幂运算，返回 x 的 y 次幂。  
   - abs(x)：绝对值，返回 x 的非负值（整型或浮点均可）。  
   - swap(a, b)：交换两个变量的值。

3. 函数调用形式  
   - 参数类型：大多数函数接受 `double` 类型，`abs` 对整型和浮点型均重载。  
   - 返回值：根据计算结果决定类型（整型或浮点型）。  
   - 注意：调用后记得接收返回值或将其嵌入表达式。

4. 简单→复杂示例  
   - 先计算 sqrt，再用 floor 取整：`int r = floor(sqrt(10.0));`  
   - 先算 pow，再 round 取整：`int p = round(pow(2.5, 3));`  
   - 交换两个成绩：`swap(score1, score2);`

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    double x = 3.7;
    double y = -2.3;
    int a = 5;
    int b = -12;

    // floor、ceil、round
    cout << "floor(3.7) = " << floor(x) << endl;     // 3
    cout << "ceil(3.7)  = " << ceil(x)  << endl;     // 4
    cout << "round(3.7) = " << round(x) << endl;     // 4
    cout << "round(-2.3)= " << round(y) << endl;     // -2

    // sqrt、pow
    cout << "sqrt(9.0)  = " << sqrt(9.0)    << endl; // 3
    cout << "pow(2.0,3) = " << pow(2.0, 3.0) << endl; // 8

    // abs
    cout << "abs(-12)   = " << abs(b)       << endl; // 12

    // swap
    cout << "Before swap: a=" << a << ", b=" << b << endl;
    swap(a, b);
    cout << "After  swap: a=" << a << ", b=" << b << endl;

    return 0;
}
```

## 练习题目

### 题目部分

1. floor(−1.2) 的值是（ ）。  
   A. −1  B. −2  C. 1  D. 2  

2. 以下表达式中，返回 27 的是（ ）。  
   A. pow(3, 3)  
   B. sqrt(27)  
   C. round(pow(3, 2.7))  
   D. ceil(pow(2, 4))  

3. 把变量 x 与 y 互换，可使用（ ）。  
   A. floor(x, y)  
   B. swap(x, y)  
   C. ceil(x, y)  
   D. abs(x, y)  

4. 填空：`int r = ___(sqrt(16.0) + round(2.3));` 使 r 的值为 6。

5. 填空：`int t = abs(___);` 如果希望 t 等于 5，则括号内应填入 ______。

### 答案与解析

1. 答案：B  
   解析：floor（向下取整）对负数更负，-1.2 → -2。

2. 答案：A  
   解析：pow(3,3)=27；其他选项结果均不为 27。

3. 答案：B  
   解析：swap 即交换两个变量的值。

4. 答案：round  
   解析：sqrt(16.0)=4，round(2.3)=2，4+2=6。

5. 答案：-5  
   解析：abs(-5)=5。
