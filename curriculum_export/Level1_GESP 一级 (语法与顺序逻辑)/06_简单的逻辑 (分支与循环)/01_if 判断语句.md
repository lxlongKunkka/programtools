## 知识点讲解

### 1. 什么是条件判断？
条件判断，简单来说，就是程序根据某个条件是否成立，来决定接下来要做什么。如果条件成立（是真的），就执行某些代码；如果条件不成立（是假的），就不执行这些代码，或者执行另外一些代码。

### 2. `if` 判断语句的基本结构
`if` 语句是最基本的条件判断语句。它的语法结构就像这样：

```cpp
if (条件表达式)
{
    // 如果“条件表达式”为真，就执行这里的代码
}
// 如果“条件表达式”为假，则跳过上面的代码块，从这里继续执行
```

*   **`if` 关键字：** 告诉编译器，接下来要进行一个条件判断了。
*   **`()` 括号：** 里面放着我们的“条件表达式”。
*   **`条件表达式`：** 这是一个会得到“真”或“假”结果的式子。
*   **`{}` 大括号：** 里面放着当条件为真时，程序要执行的代码。我们称之为“代码块”。

### 3. 神奇的关系运算符
条件表达式里面，我们通常会用到一些特殊的符号，它们用来比较两个值的大小或是否相等，我们叫它们**关系运算符**。

| 运算符 | 含义         | 例子         | 结果（假设 a=5, b=3） |
| :----- | :----------- | :----------- | :-------------------- |
| `>`    | 大于         | `a > b`      | 真 (5 > 3 是真的)     |
| `<`    | 小于         | `a < b`      | 假 (5 < 3 是假的)     |
| `>=`   | 大于等于     | `a >= b`     | 真 (5 >= 3 是真的)    |
| `<=`   | 小于等于     | `a <= b`     | 假 (5 <= 3 是假的)    |
| `==`   | 等于         | `a == b`     | 假 (5 == 3 是假的)    |
| `!=`   | 不等于       | `a != b`     | 真 (5 != 3 是真的)    |

**特别注意：**
*   判断“等于”要用两个等号 `==`，而不是一个等号 `=`。一个等号 `=` 是赋值的意思，表示把右边的值放进左边的变量里。
*   条件表达式的结果只有两种：真（通常用数字 1 表示）或假（通常用数字 0 表示）。

### 4. `if` 语句的执行流程
想象一下，程序就像一个侦探：
1.  它走到 `if` 语句这里。
2.  它会先去检查 `()` 里面的“条件表达式”。
3.  如果条件表达式是真的（比如 `5 > 3`），侦探就会进入 `{}` 里面，执行里面的代码。
4.  如果条件表达式是假的（比如 `5 < 3`），侦探就会跳过 `{}` 里面的代码，直接走到 `if` 语句后面的代码继续执行。

1.  `a > b`
2.  `a == b`
3.  `a <= 10`
4.  `b != 7`

（答案：1.真 2.假 3.真 4.假）

## 代码示例

让我们用代码来实际操作一下吧！

### 示例 1：判断一个数是否大于 10

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    int a; // 定义一个整数变量 a
    cout << "请输一个整数: "; // 提示用户输入
    cin >> a; // 读取用户输入的整数到变量 a

    // 判断 a 是否大于 10
    if (a > 10)
    {
        cout << "你输入的数 " << a << " 大于 10。" << endl; // 如果条件为真，就输出这句话
    }

    cout << "程序结束。" << endl; // 无论条件是否为真，都会执行这句话

    return 0;
}
```

**运行演示：**
*   如果输入 `12`：
    ```
    请输一个整数: 12
    你输入的数 12 大于 10。
    程序结束。
    ```
*   如果输入 `5`：
    ```
    请输一个整数: 5
    程序结束。
    ```

### 示例 2：判断一个数是否为偶数

一个数如果是偶数，那么它除以 2 的余数就是 0。我们可以用 `%` (取模运算符) 来计算余数。

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    int a; // 定义一个整数变量 a
    cout << "请输一个整数: "; // 提示用户输入
    cin >> a; // 读取用户输入的整数到变量 a

    // 判断 a 除以 2 的余数是否为 0
    if (a % 2 == 0)
    {
        cout << "你输入的数 " << a << " 是一个偶数。" << endl; // 如果条件为真，就输出这句话
    }

    cout << "判断完成。" << endl;

    return 0;
}
```

**运行演示：**
*   如果输入 `8`：
    ```
    请输一个整数: 8
    你输入的数 8 是一个偶数。
    判断完成。
    ```
*   如果输入 `3`：
    ```
    请输一个整数: 3
    判断完成。
    ```

### 示例 3：判断考试是否及格

假设及格分数是 60 分。

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    int s; // 定义一个整数变量 s 代表分数
    cout << "请输你的考试分数: "; // 提示用户输入分数
    cin >> s; // 读取用户输入的分数到变量 s

    // 判断分数是否大于或等于 60
    if (s >= 60)
    {
        cout << "恭喜你，及格了！" << endl; // 如果分数及格，就输出这句话
    }

    cout << "考试结果已公布。" << endl;

    return 0;
}
```

**运行演示：**

    ```

    考试结果已公布。
    ```

### 练习题目答案与解析

**1. 选择题：**
**答案：** B
**解析：** `=` 是赋值运算符，用于将右侧的值赋给左侧的变量。`==` 才是关系运算符，用于判断两个值是否相等。

**2. 填空题：**
**答案：** `条件表达式`
**解析：** `if` 语句括号中需要填写一个能够判断真假的条件表达式。

**3. 选择题：**
**答案：** B
**解析：** `x` 的值是 15，`15 < 10` 这个条件是假的，所以结果是“假”。在 C++ 中，“假”通常用数字 0 表示。

**4. 填空题：**
**答案：**
```
Hello
World
```
**解析：** 输入 `20`，`n` 的值是 20。条件 `n > 10` (即 `20 > 10`) 为真，所以会执行 `if` 语句块内的 `cout << "Hello" << endl;`。接着，`if` 语句后面的 `cout << "World" << endl;` 无论条件真假都会执行。

**5. 选择题：**
**答案：** A
**解析：** 输入 `5`，`m` 的值是 5。条件 `m % 2 != 0` (即 `5 % 2 != 0`，`5 % 2` 的结果是 1，`1 != 0` 为真)。所以会执行 `if` 语句块内的 `cout << "Odd" << endl;`。
