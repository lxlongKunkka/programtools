## 知识点讲解

我们刚才说了，`for` 循环就像一个勤劳的小助手，可以帮我们重复做事情。那它到底是怎么工作的呢？

### `for` 循环的基本结构

`for` 循环的语法长这样：

```cpp
for (初始化语句; 条件表达式; 更新语句)
{
    // 循环体：当条件表达式为真时，会重复执行这里的代码
}
```

是不是看起来有点像一个“三明治”？我们来一层一层地揭开它的秘密：

1.  **`初始化语句` (int i = 1; )**：
    *   这部分代码只会在 `for` 循环**最开始的时候执行一次**。
    *   它的作用通常是创建一个计数器变量，并给它一个初始值。比如，我们要从 1 开始数，就可以写 `int i = 1;`，表示我们的计数器 `i` 从 1 开始。

2.  **`条件表达式` (i <= 100; )**：
    *   这部分代码在**每次循环体执行之前都会被检查**。
    *   如果条件表达式的结果是 `true` (真)，那么循环体里的代码就会被执行。
    *   如果条件表达式的结果是 `false` (假)，那么循环就会立即停止，程序会跳过循环体，继续执行 `for` 循环后面的代码。
    *   比如，我们要数到 100，就可以写 `i <= 100;`，意思是只要 `i` 小于或等于 100，就继续数。

3.  **`更新语句` (i++; )**：
    *   这部分代码在**每次循环体执行完毕后都会被执行**。
    *   它的作用通常是改变计数器的值，让它一步步接近或远离终止条件。
    *   比如，我们要一个一个地数，就可以写 `i++;`，意思是每次数完一个数后，让 `i` 增加 1。`i++` 是 `i = i + 1` 的简写形式哦！

### `for` 循环的执行流程

让我们用一个例子来模拟一下 `for` 循环是怎样工作的：我们要从 1 数到 3。

```cpp
for (int i = 1; i <= 3; i++)
{
    cout << i << endl; // 打印当前的数字 i
}
```

1.  **第一步：初始化**
    *   `int i = 1;`：电脑创建一个变量 `i`，并把它的值设为 1。
    *   （这步只执行一次！）

2.  **第二步：检查条件**
    *   `i <= 3;`：现在 `i` 是 1，`1 <= 3` 是 `true` 吗？是的！
    *   条件为真，进入循环体。

3.  **第三步：执行循环体**
    *   `cout << i << endl;`：打印 `i` 的值，也就是 1。

4.  **第四步：更新**
    *   `i++;`：`i` 的值从 1 变成了 2。

5.  **第五步：再次检查条件**
    *   `i <= 3;`：现在 `i` 是 2，`2 <= 3` 是 `true` 吗？是的！
    *   条件为真，进入循环体。

6.  **第六步：再次执行循环体**
    *   `cout << i << endl;`：打印 `i` 的值，也就是 2。

7.  **第七步：再次更新**
    *   `i++;`：`i` 的值从 2 变成了 3。

8.  **第八步：再次检查条件**
    *   `i <= 3;`：现在 `i` 是 3，`3 <= 3` 是 `true` 吗？是的！
    *   条件为真，进入循环体。

9.  **第九步：再次执行循环体**
    *   `cout << i << endl;`：打印 `i` 的值，也就是 3。

10. **第十步：再次更新**
    *   `i++;`：`i` 的值从 3 变成了 4。

11. **第十一步：最后检查条件**
    *   `i <= 3;`：现在 `i` 是 4，`4 <= 3` 是 `true` 吗？不是！是 `false`！
    *   条件为假，循环停止！程序会跳出 `for` 循环，执行后面的代码。

看，是不是像一个自动化的计数器？只要我们告诉它从哪里开始、到哪里结束、每次加多少，它就能准确无误地完成任务！

## 代码示例

现在，就让我们用 `for` 循环来实现我们最初的目标：打印从 1 到 100 的所有数字！

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    // 使用 for 循环打印从 1 到 100 的数字
    // 初始化：int i = 1;  计数器 i 从 1 开始
    // 条件：  i <= 100; 只要 i 小于或等于 100，就继续循环
    // 更新：  i++;      每次循环结束后，i 的值增加 1
    for (int i = 1; i <= 100; i++)
    {
        cout << i << endl; // 打印当前的数字 i，并换行
    }

    return 0;
}
```

**运行结果会是：**
```
1
2
3
...
99
100
```
（中间省略了很多行，但实际运行会打印出所有数字哦！）

同学们，看到了吗？仅仅几行代码，就完成了手动数 100 个数字的繁琐工作！这就是 `for` 循环的魔力！

### 小试牛刀：打印 10 到 1

如果我想从 10 倒着数到 1 呢？我们只需要稍微修改一下 `for` 循环的三个部分：

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    // 使用 for 循环打印从 10 到 1 的数字
    // 初始化：int i = 10; 计数器 i 从 10 开始
    // 条件：  i >= 1;   只要 i 大于或等于 1，就继续循环
    // 更新：  i--;      每次循环结束后，i 的值减少 1 (i-- 是 i = i - 1 的简写)
    for (int i = 10; i >= 1; i--)
    {
        cout << i << endl; // 打印当前的数字 i，并换行
    }

    return 0;
}
```
**运行结果会是：**
```
10
9
8
...
2
1
```

是不是很简单？只要理解了 `初始化`、`条件` 和 `更新` 这三部分，你就可以让 `for` 循环做各种各样的重复任务啦！

## 练习题目

### 选择题/填空题

1.  以下 `for` 循环代码会打印出多少个数字？
    ```cpp
    for (int i = 0; i < 5; i++)
    {
        cout << i << endl;
    }
    ```
    A. 4 个
    B. 5 个
    C. 6 个
    D. 无限个

2.  要打印从 10 到 1 的所有数字，以下哪个 `for` 循环的 `条件表达式` 和 `更新语句` 是正确的？
    ```cpp
    for (int i = 10; _______; _______)
    {
        cout << i << endl;
    }
    ```
    A. `i < 1; i++`
    B. `i <= 1; i--`
    C. `i >= 1; i--`
    D. `i > 0; i++`

3.  以下 `for` 循环会输出什么？
    ```cpp
    for (int i = 3; i <= 7; i = i + 2)
    {
        cout << i << " ";
    }
    ```
    A. `3 4 5 6 7 `
    B. `3 5 7 `
    C. `3 6 9 `
    D. `3 4 5 `

4.  在 `for (int i = 1; i <= 10; i++)` 中，`i++` 的作用是：
    A. 初始化循环变量 `i`
    B. 判断循环是否继续执行
    C. 在每次循环结束后，将 `i` 的值增加 1
    D. 打印 `i` 的值

5.  如果希望一个 `for` 循环从 `1` 开始，到 `n` 结束（包含 `n`），并且每次加 `1`，那么 `for` 循环的结构应该是：`for (int i = 1; _______; i++)`。请填空。

### 答案与解析

1.  **答案：B**
    *   **解析**：`i` 从 0 开始，当 `i` 等于 0, 1, 2, 3, 4 时，`i < 5` 都为真，会执行循环体。当 `i` 变为 5 时，`5 < 5` 为假，循环停止。所以会打印 0, 1, 2, 3, 4，共 5 个数字。

2.  **答案：C**
    *   **解析**：
        *   要从 10 倒数到 1，所以条件应该是 `i >= 1` (当 `i` 大于等于 1 时继续)。
        *   每次递减，所以更新语句应该是 `i--` (每次 `i` 减 1)。

3.  **答案：B**
    *   **解析**：
        *   `i` 初始化为 3。
        *   第一次循环：`i=3`，`3 <= 7` 为真，打印 3。`i` 变为 `3+2=5`。
        *   第二次循环：`i=5`，`5 <= 7` 为真，打印 5。`i` 变为 `5+2=7`。
        *   第三次循环：`i=7`，`7 <= 7` 为真，打印 7。`i` 变为 `7+2=9`。
        *   第四次循环：`i=9`，`9 <= 7` 为假，循环结束。
        *   所以输出是 `3 5 7 `。

4.  **答案：C**
    *   **解析**：
        *   A 选项是 `int i = 1;` 的作用。
        *   B 选项是 `i <= 10;` 的作用。
        *   C 选项 `i++` 正是每次循环后递增计数器的作用。
        *   D 选项是 `cout << i << endl;` 的作用。

5.  **答案：`i <= n`**
    *   **解析**：要包含 `n`，所以条件应该是小于或等于 `n`。
