## 趣味引入

同学们，大家好！今天我们要一起探索一个非常有趣且强大的数学工具，它能帮我们解决那些看似简单却又让人头疼的“数数”问题。

想象一下，你有一串项链，上面有 4 颗珠子。你手里有黑、白两种颜色的染料，想给这 4 颗珠子染色。请问，你能染出多少种“本质不同”的项链呢？

我们来试试看：
*   如果只看珠子的颜色排列，那每颗珠子都有 2 种选择，一共是 2 * 2 * 2 * 2 = 16 种。
*   但项链是可以旋转的！比如，“黑白黑白”和“白黑白黑”可能看起来不同，但只要把项链旋转一下，它们就变成了同一种。
*   甚至，有些项链还可以翻转！如果允许翻转，那“黑黑白白”和““黑白白黑”（从左到右）可能也是同一种。

是不是有点乱？到底什么叫做“本质不同”？我们平时数数，都是数“排列组合”，但当对象具有“对称性”时，比如可以旋转、翻转，很多看似不同的排列其实是同一个东西的不同表现形式。

传统的计数方法在这里就失效了。今天，我们要学习的 **Burnside 引理** 和它的“升级版” **Polya 定理**，正是为了解决这类问题而生的！它们就像一对火眼金睛，能帮助我们透过表象，直接数出那些真正“本质不同”的方案。准备好了吗？让我们一起进入奇妙的组合数学世界吧！

## 深度知识点讲解

### 1. 什么是“本质不同”？——群论的初探

要理解 Burnside 引理和 Polya 定理，我们首先要理解它们背后的数学基础——**群论**。别担心，我们只学习最核心、最直观的部分。

**核心思想：** 当我们说两个方案“本质相同”时，通常意味着它们可以通过某种“操作”互相转换。这些操作，比如旋转、翻转，它们本身就构成了一个特殊的数学结构，叫做“群”。

#### 1.1 群 (Group)

一个**群**是由一个集合 $G$ 和一个二元运算 $*$ 组成的，满足以下四个条件：
1.  **封闭性：** 对于任意 $a, b \in G$， $a * b$ 仍然在 $G$ 中。
2.  **结合律：** 对于任意 $a, b, c \in G$， $(a * b) * c = a * (b * c)$。
3.  **单位元：** 存在一个元素 $e \in G$，使得对于任意 $a \in G$， $e * a = a * e = a$。
4.  **逆元：** 对于任意 $a \in G$，都存在一个元素 $a^{-1} \in G$，使得 $a * a^{-1} = a^{-1} * a = e$。

**生活类比：**
*   **旋转操作：** 想象一个正方形，我们可以把它旋转 0 度、90 度、180 度、270 度。这些旋转操作构成一个群。
    *   “操作”就是群的元素。
    *   “连续做两个旋转”就是群的运算。
    *   旋转 0 度就是单位元。
    *   旋转 90 度的逆元是旋转 270 度。
*   **翻转操作：** 如果再加上翻转操作（沿着对称轴翻转），这些旋转和翻转操作也构成一个群，叫做“二面体群”。

#### 1.2 群作用 (Group Action)

现在我们有了“操作”的集合——群 $G$，我们还需要一个被操作的集合 $X$。比如，我们要染色的方案集合。
**群作用**就是群 $G$ 中的每个元素 $g$ 都能作用在集合 $X$ 中的每个元素 $x$ 上，得到一个新的元素 $g(x) \in X$，并且满足：
1.  单位元 $e$ 作用在任何 $x$ 上，都得到 $x$ 本身：$e(x) = x$。
2.  连续作用：$g(h(x)) = (g * h)(x)$。

**生活类比：**
*   **群 $G$：** 正方形的旋转操作。
*   **集合 $X$：** 所有可能的正方形顶点染色方案。
*   **群作用：** 某个旋转操作作用在一个染色方案上，得到一个新的染色方案。

#### 1.3 轨道 (Orbit) 与 不动点 (Fixed Point)

*   **轨道 (Orbit)：** 对于集合 $X$ 中的一个元素 $x$，它的轨道 $O(x)$ 是所有可以通过群 $G$ 中的操作从 $x$ 得到的元素的集合。
    $O(x) = \{g(x) \mid g \in G\}$
    **核心思想：** 在同一个轨道里的元素，就是我们说的“本质相同”的方案。我们要数的就是轨道的数量！
*   **不动点 (Fixed Point)：** 对于群 $G$ 中的一个操作 $g$，如果 $g(x) = x$，那么 $x$ 就是操作 $g$ 的一个**不动点**。也就是说，操作 $g$ 作用在 $x$ 上， $x$ 保持不变。我们用 $X^g$ 表示操作 $g$ 的所有不动点组成的集合， $|X^g|$ 是不动点的数量。

**生活类比：**
*   **轨道：** 某一种染色方案，经过所有旋转操作后，能变成的所有方案的集合。这个集合里所有的方案都算作一种“本质不同”的方案。
*   **不动点：** 比如，一个全黑的项链，无论怎么旋转，它都还是全黑的。所以全黑的项链是所有旋转操作的不动点。一个“黑白黑白”的项链，旋转 90 度会变成“白黑白黑”，但旋转 180 度后它还是“黑白黑白”，所以它是旋转 180 度这个操作的不动点。

### 2. Burnside 引理：数轨道的魔法公式

Burnside 引理提供了一个直接计算轨道数量的公式：
**轨道的数量 = (1/|G|) * Σ_{g ∈ G} |X^g|**

其中：
*   $|X/G|$ 表示轨道的数量（也就是我们想求的“本质不同”的方案数）。
*   $|G|$ 是群 $G$ 中元素的数量（操作的总数）。
*   Σ_{g ∈ G} 表示对群 $G$ 中的所有操作 $g$ 求和。
*   $|X^g|$ 是操作 $g$ 的不动点数量。

**公式的直观理解：**
Burnside 引理告诉我们，要数有多少种本质不同的方案，我们只需要：
1.  列出所有可能的“操作”（群 $G$ 的所有元素）。
2.  对于每一种操作 $g$，计算有多少种方案 $x$ 在这个操作下保持不变（$|X^g|$）。
3.  把所有操作的不动点数量加起来。
4.  最后除以操作的总数 $|G|$。

**为什么是这样？** （高层次的证明思想）
这个公式的巧妙之处在于，它将对“轨道”的计数转化为了对“不动点”的计数。
我们可以构造一个二元组 $(x, g)$ 的集合 $S = \{(x, g) \mid x \in X, g \in G, g(x) = x\}$。
我们从两个角度来计算 $|S|$：
1.  **按 $g$ 计数：** $|S| = Σ_{g \in G} |X^g|$ (这就是公式右侧的求和部分)。
2.  **按 $x$ 计数：** $|S| = Σ_{x \in X} |Stab(x)|$，其中 $Stab(x) = \{g \in G \mid g(x) = x\}$ 叫做 $x$ 的稳定子群。
    群论中有一个重要的“轨道-稳定子定理”：$|O(x)| \cdot |Stab(x)| = |G|$。
    所以 $|Stab(x)| = |G| / |O(x)|$。
    那么 $|S| = Σ_{x \in X} (|G| / |O(x)|)$。
    由于同一个轨道里的所有 $x$ 的 $|O(x)|$ 都相同，我们可以把求和改成对轨道求和：
    $|S| = Σ_{O \in X/G} Σ_{x \in O} (|G| / |O(x)|) = Σ_{O \in X/G} |O| \cdot (|G| / |O|) = Σ_{O \in X/G} |G| = |X/G| \cdot |G|$。
    因此，从两个角度计数得到：$Σ_{g \in G} |X^g| = |X/G| \cdot |G|$。
    移项就得到了 Burnside 引理：$|X/G| = (1/|G|) \cdot Σ_{g \in G} |X^g|$。

### 3. Polya 定理：Burnside 的染色计数利器

Burnside 引理已经很强大了，但对于染色问题，计算 $|X^g|$（某个操作的不动点数量）仍然可能很复杂。Polya 定理正是为此而生，它提供了一个更直接的方法来计算 $|X^g|$。

**核心思想：置换的循环分解与不动点**

任何一个置换（群 $G$ 中的操作 $g$ 都可以看作一个置换）都可以唯一地分解成若干个**不相交的循环**的乘积。

**生活类比：**
*   一个旋转 90 度的置换作用在正方形的 4 个顶点上：顶点 1 -> 2 -> 3 -> 4 -> 1。这构成一个长度为 4 的循环：(1 2 3 4)。
*   一个旋转 180 度的置换：顶点 1 -> 3 -> 1，顶点 2 -> 4 -> 2。这构成两个长度为 2 的循环：(1 3)(2 4)。

**Polya 定理的关键洞察：**
如果一个染色方案 $x$ 是操作 $g$ 的不动点，那么它必须满足：在操作 $g$ 的作用下，所有颜色保持不变。这意味着，置换 $g$ 的每一个循环内部的所有元素都必须染成相同的颜色。

举例：如果 $g = (1 2 3)(4 5)$，要使染色方案不变，那么 1, 2, 3 必须同色，4, 5 必须同色。
如果一个置换 $g$ 被分解成 $c(g)$ 个不相交的循环，并且我们有 $k$ 种颜色。那么要使一个染色方案成为 $g$ 的不动点，每个循环都可以独立选择一种颜色。所以，不动点数量 $|X^g| = k^{c(g)}$。

**Polya 定理的表述：**
对于一个作用在 $n$ 个对象上的群 $G$，我们用 $k$ 种颜色对这 $n$ 个对象进行染色。本质不同的染色方案数是：
**N = (1/|G|) * Σ_{g ∈ G} k^{c(g)}**

其中 $c(g)$ 是置换 $g$ 被分解成的不相交循环的个数。

**对比 Burnside 和 Polya：**
Polya 定理实际上是 Burnside 引理的一个特例或推广。它将 Burnside 引理中的 $|X^g|$ 替换成了 $k^{c(g)}$，极大地简化了染色问题的计算。

#### 3.1 循环指标多项式 (Cycle Index Polynomial)

Polya 定理还可以用**循环指标多项式**来表示，这是一种更通用的形式。
对于一个群 $G$ 和其作用在 $n$ 个对象上的所有置换 $g$，定义 $c_i(g)$ 为置换 $g$ 中长度为 $i$ 的循环的个数。
群 $G$ 的循环指标多项式 $Z_G(x_1, x_2, \ldots, x_n)$ 为：
$Z_G(x_1, x_2, \ldots, x_n) = (1/|G|) * Σ_{g ∈ G} x_1^{c_1(g)} x_2^{c_2(g)} \ldots x_n^{c_n(g)}$

**Polya 定理（通用形式）：** 如果我们有 $k$ 种颜色，并且每种颜色有不同的“权值”，那么本质不同的染色方案的“总权值和”可以通过将循环指标多项式中的 $x_i$ 替换为 $(颜色1的权值)^i + (颜色2的权值)^i + \ldots + (颜色k的权值)^i$ 来计算。
对于最常见的简单计数问题（所有颜色权值都为 1），我们只需将所有的 $x_i$ 替换为 $k$。
此时，$x_1^{c_1(g)} x_2^{c_2(g)} \ldots x_n^{c_n(g)} = k^{c_1(g)} k^{c_2(g)} \ldots k^{c_n(g)} = k^{c_1(g) + c_2(g) + \ldots + c_n(g)} = k^{c(g)}$，其中 $c(g)$ 是总循环个数。这又回到了上面的简化形式。

### 常见误区：
1.  **混淆对象：** 到底是染顶点、边还是面？不同的对象集合，群作用的对象不同，置换的分解也不同。
2.  **遗漏置换：** 一定要列举出群中的所有置换，不能漏掉。特别是像正方形、立方体这类物体，除了旋转还有翻转。
3.  **循环分解错误：** 准确地分解置换为不相交循环是关键。
4.  **计算 $|G|$ 错误：** 群的大小 $|G|$ 必须准确。
5.  **Polya 定理的适用范围：** Polya 定理主要用于染色问题。对于其他类型的计数问题，可能需要回到 Burnside 引理。

## 典型例题精讲

### 例题 1：项链染色问题 (只考虑旋转)

**问题：** 用 $k$ 种颜色染 $n$ 颗珠子的项链，如果通过旋转能互相得到的项链视为同一种。问有多少种本质不同的项链？

**分析：**
1.  **对象集合 $X$：** 所有 $k^n$ 种可能的染色方案。
2.  **群 $G$：** 旋转群 $C_n$。它由 $n$ 个元素组成，分别是旋转 $0, 1, \ldots, n-1$ 个单位（每个单位是 $360/n$ 度）。所以 $|G|=n$。
    *   设 $g_j$ 表示旋转 $j$ 个单位（$j=0, \ldots, n-1$）。
3.  **置换 $g_j$ 的循环分解：**
    *   旋转 $j$ 个单位，会将珠子 $i$ 移到 $(i+j) \pmod n$ 的位置。
    *   这个置换的循环个数 $c(g_j)$ 是 $gcd(j, n)$。
    *   **解释：** 考虑珠子 0，它会经过 0 -> j -> 2j -> ... -> mj 循环回到 0。这个循环的长度是 $n/gcd(j, n)$。一共有 $gcd(j, n)$ 个这样的循环。
4.  **应用 Polya 定理：**
    本质不同的项链数 = $(1/|G|) \cdot Σ_{j=0}^{n-1} k^{c(g_j)}$
    $= (1/n) \cdot Σ_{j=0}^{n-1} k^{gcd(j, n)}$

**举例：** $n=3$ 颗珠子，$k=2$ 种颜色。
*   $|G|=3$。
*   $j=0$（旋转 0 度）：$gcd(0, 3) = 3$。不动点数 $k^3 = 2^3 = 8$。
*   $j=1$（旋转 120 度）：$gcd(1, 3) = 1$。不动点数 $k^1 = 2^1 = 2$。
*   $j=2$（旋转 240 度）：$gcd(2, 3) = 1$。不动点数 $k^1 = 2^1 = 2$。
*   总数 = $(1/3) \cdot (2^3 + 2^1 + 2^1) = (1/3) \cdot (8 + 2 + 2) = 12/3 = 4$。
    这 4 种分别是：全黑、全白、黑白白（旋转得到白黑白、白白黑）、黑黑白（旋转得到黑白黑、白黑黑）。

**优化：** 观察到 $gcd(j, n)$ 的值只可能是 $n$ 的因子。我们可以遍历 $n$ 的所有因子 $d$，计算有多少个 $j \in \{0, \ldots, n-1\}$ 使得 $gcd(j, n) = d$。
这个数量正是欧拉函数 $\phi(n/d)$。
所以公式可以优化为：
本质不同的项链数 = $(1/n) \cdot Σ_{d|n} \phi(n/d) \cdot k^d$
或者，更常见的形式是：
本质不同的项链数 = $(1/n) \cdot Σ_{d|n} \phi(d) \cdot k^{n/d}$
（其中 $d|n$ 表示 $d$ 是 $n$ 的因子，$d$ 遍历 $n$ 的所有因子）

**复杂度分析：**
*   直接求和：O(N log N) (N次GCD运算)。
*   优化后：O(sqrt(N) + N_prime_factors) (求因子 O(sqrt(N))，欧拉函数 O(sqrt(N)) 或预处理 O(N log log N))。对于大 N，优化后的性能更好。

### 例题 2：正方形顶点染色问题 (考虑旋转和翻转)

**问题：** 用 $k$ 种颜色染正方形的 4 个顶点，如果通过旋转或翻转能互相得到的方案视为同一种。问有多少种本质不同的染色方案？

**分析：**
1.  **对象集合 $X$：** 所有 $k^4$ 种可能的染色方案。
2.  **群 $G$：** 正方形的对称群，即二面体群 $D_4$。它包含 8 个元素（4 个旋转，4 个翻转）。所以 $|G|=8$。
    我们用顶点 1, 2, 3, 4 顺时针编号。
    *   **旋转 (4 个)：**
        *   **$g_1$: 旋转 0 度 (恒等置换)**
            置换：(1)(2)(3)(4)
            循环分解：4 个长度为 1 的循环。$c(g_1)=4$。
            不动点数：$k^4$。
        *   **$g_2$: 旋转 90 度**
            置换：(1 2 3 4)
            循环分解：1 个长度为 4 的循环。$c(g_2)=1$。
            不动点数：$k^1$。
        *   **$g_3$: 旋转 180 度**
            置换：(1 3)(2 4)
            循环分解：2 个长度为 2 的循环。$c(g_3)=2$。
            不动点数：$k^2$。
        *   **$g_4$: 旋转 270 度**
            置换：(1 4 3 2)
            循环分解：1 个长度为 4 的循环。$c(g_4)=1$。
            不动点数：$k^1$。
    *   **翻转 (4 个)：**
        *   **$g_5, g_6$: 沿着对边中点连线翻转 (垂直轴和水平轴)**
            *   $g_5$ (垂直轴，穿过 1-2 中点和 3-4 中点)：(1 2)(3 4)
                循环分解：2 个长度为 2 的循环。$c(g_5)=2$。
                不动点数：$k^2$。
            *   $g_6$ (水平轴，穿过 1-4 中点和 2-3 中点)：(1 4)(2 3)
                循环分解：2 个长度为 2 的循环。$c(g_6)=2$。
                不动点数：$k^2$。
        *   **$g_7, g_8$: 沿着对角线翻转 (主对角线和副对角线)**
            *   $g_7$ (主对角线，穿过 1 和 3)：(1)(3)(2 4)
                循环分解：2 个长度为 1 的循环，1 个长度为 2 的循环。$c(g_7)=3$。
                不动点数：$k^3$。
            *   $g_8$ (副对角线，穿过 2 和 4)：(2)(4)(1 3)
                循环分解：2 个长度为 1 的循环，1 个长度为 2 的循环。$c(g_8)=3$。
                不动点数：$k^3$。

3.  **应用 Polya 定理：**
    本质不同的方案数 = $(1/8) \cdot (k^4 + k^1 + k^2 + k^1 + k^2 + k^2 + k^3 + k^3)$
    $= (1/8) \cdot (k^4 + 2k^3 + 3k^2 + 2k^1)$

**举例：** $k=2$ 种颜色（黑白）。
*   总数 = $(1/8) \cdot (2^4 + 2 \cdot 2^3 + 3 \cdot 2^2 + 2 \cdot 2^1)$
    $= (1/8) \cdot (16 + 2 \cdot 8 + 3 \cdot 4 + 2 \cdot 2)$
    $= (1/8) \cdot (16 + 16 + 12 + 4)$
    $= (1/8) \cdot (48) = 6$。

**这 6 种方案是：**
1.  全黑
2.  全白
3.  1 黑 3 白 (或 3 黑 1 白)
4.  2 黑 2 白 (相邻)
5.  2 黑 2 白 (对角)
6.  (没有其他情况了)

**复杂度分析：** 对于固定大小的群 (如 $D_4$ 的 $|G|=8$)，计算复杂度是常数。

ll gcd(ll a, ll b) {
    while (b) {
        a %= b;
        std::swap(a, b);
    }
    return a;
}

// 快速幂函数：计算 base^exp
ll power(ll base, ll exp) {
    ll res = 1;
    while (exp > 0) {
        if (exp % 2 == 1) { // 如果指数是奇数
            res *= base;
        }
        base *= base; // 基数平方
        exp /= 2;     // 指数减半
    }
    return res;
}
