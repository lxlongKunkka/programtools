# GESP 七级 (图·哈希·进阶DP)

## 教学目标
1. 掌握树形结构的基本概念与遍历。
2. 理解贪心算法的设计思想。
3. 掌握简单的动态规划。

## 核心知识点
- 树：二叉树的概念、性质、存储。
- 树的遍历：前序、中序、后序遍历。
- 贪心：区间调度、背包问题的贪心策略（部分）。
- 动态规划：递推公式，爬楼梯问题，最大子段和。

## 专题列表

1. **DFS 进阶**
   > 回溯与剪枝优化、全排列/子集生成、搜索顺序优化
2. **BFS 进阶**
   > 多源 BFS、双向 BFS、A* 启发式搜索思想
3. **快速幂**
   > 快速幂算法（整数/矩阵）、取模运算与幂运算应用
4. **倍增与 LCA**
   > 倍增法、最近公共祖先（LCA）与树上路径查询
5. **离散化与 ST 表**
   > 坐标压缩（离散化）与 ST 表静态 RMQ
6. **哈希**
   > 哈希表原理、字符串哈希（多项式哈希）与应用
7. **表达式求值**
   > 栈模拟中缀/后缀表达式、运算符优先级处理
8. **单调栈与单调队列**
   > 在栈与队列基础上维护单调性：单调栈 O(n) 解决下一个更大/更小元素与柱状图最大矩形；单调队列 O(n) 求滑动窗口最值，并作为区间DP和斜率优化DP的核心工具。
9. **区间 DP**
   > 区间动态规划经典模型：矩阵链乘、石子归并、括号合并
10. **树上 DP**
   > 树形动态规划、以某节点为根的状态转移
11. **二维 DP**
   > 二维状态的动态规划、方格路径类、编辑距离等
