## 知识点讲解

1. 已知→未知  
   - 复习：已学会的一维数组声明与基本循环  
   - 引出：如何在数组中查找、统计、排序等更复杂的操作  

2. 简单→复杂  
   - 求和与求平均  
   - 查找最大值/最小值及其下标  
   - 计数（如统计满足特定条件的元素个数）  
   - 冒泡排序（实践手写排序过程）  

3. 理论→实践  
   - 每个操作都先讲解思路：  
     a) 遍历一次得到结果（求和、找最大最小）  
     b) 嵌套循环实现排序  
   - 再通过代码示例演示，并配合调试工具观察运行过程

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n = 5;
    int a[100] = {23, 45, 12, 37, 29};

    // 1. 求和与平均
    int sum = 0;
    for (int i = 0; i < n; i++)
    {
        sum += a[i];
    }
    cout << "Sum = " << sum << endl;
    cout << "Average = " << sum * 1.0 / n << endl;

    // 2. 查找最大值及其下标
    int maxVal = a[0];
    int maxPos = 0;
    for (int i = 1; i < n; i++)
    {
        if (a[i] > maxVal)
        {
            maxVal = a[i];
            maxPos = i;
        }
    }
    cout << "Max value = " << maxVal << ", at index " << maxPos << endl;

    // 3. 冒泡排序（从小到大）
    for (int i = 0; i < n - 1; i++)
    {
        for (int j = 0; j < n - 1 - i; j++)
        {
            if (a[j] > a[j + 1])
            {
                int t = a[j];
                a[j] = a[j + 1];
                a[j + 1] = t;
            }
        }
    }
    cout << "Sorted array: ";
    for (int i = 0; i < n; i++)
    {
        cout << a[i] << " ";
    }
    cout << endl;

    return 0;
}
```

## 练习题目

基础题（选择或填空，共 4 道）

1. 定义 `int a[10];` 后，访问 `a[10]` 会怎样？  
2. 已有 `int a[5] = {1, 3, 5, 7, 9};`，执行 `cout << a[2];` 输出是（ ）。  
3. 冒泡排序中，如果外层循环 i 从 `0` 到 `n-2`，内层循环应是（ ）。  
   A. `for (int j = 0; j < n; j++)`  
   B. `for (int j = 0; j < n - i; j++)`  
   C. `for (int j = 0; j < n - 1 - i; j++)`  
4. 填空：要统计数组 `a[ ]` 中所有偶数个数，可用变量 `cnt = 0;`，遍历时判断 `if (_________) cnt++;`  

进阶题（1 道）

5. 编写一段代码，输入 `n` 个整数存入数组 `a`，输出第二大值（假设各元素互不相同）。请用手写算法，不允许调用 `sort`。  

—— 先做题，后对答案 ——  
  

**答案与解析**

1. 越界访问，结果未定义；可能输出垃圾值或导致运行错误。  
2. a[2] = 5。  
3. 答案 C。外层 i 越多，内层比较长度要减 `i`。  
4. 填空：`a[i] % 2 == 0`  
5. 示例思路：  
   - 先用一次遍历找出最大值 `max1` 和次大值 `max2`  
   - 代码示例：  
```cpp
int max1 = a[0], max2 = a[1];
if (max2 > max1)
{
    swap(max1, max2);
}
for (int i = 2; i < n; i++)
{
    if (a[i] > max1)
    {
        max2 = max1;
        max1 = a[i];
    }
    else if (a[i] > max2)
    {
        max2 = a[i];
    }
}
cout << max2 << endl;
```
