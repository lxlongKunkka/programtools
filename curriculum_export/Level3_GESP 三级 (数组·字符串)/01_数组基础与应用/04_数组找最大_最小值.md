## 知识点讲解

### 1. 数组小回顾

我们之前学习过，数组就像是一个“大柜子”，里面有很多“小格子”，每个小格子都可以存放一个相同类型的数据。每个小格子都有一个编号，我们称之为“下标”或“索引”，它总是从 `0` 开始。

*   **声明数组**：`int a[10];` // 声明一个可以存放10个整数的数组，下标从0到9。
*   **访问元素**：`a[0]` 是第一个元素，`a[9]` 是最后一个元素。
*   **遍历数组**：通常使用 `for` 循环来访问数组中的每一个元素。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a[5] = {10, 20, 30, 40, 50}; // 声明并初始化一个有5个元素的数组

    // 遍历并打印数组中的所有元素
    for (int i = 0; i < 5; i++) {
        cout << "a[" << i << "] = " << a[i] << endl;
    }

    return 0;
}
```

### 2. 查找最大值（Max Value）的思路

我们要找数组中的最大值，就像在很多数字中挑出最大的一个。我们可以这样想：

1.  **先假设**：我们先假设数组中的第一个元素就是最大值。把它存到一个叫做 `max_val` 的变量里。
2.  **逐个比较**：然后，我们从数组的第二个元素开始，一个一个地和我们“假设的最大值”进行比较。
3.  **更新最大值**：如果当前这个元素比 `max_val` 还要大，那么说明我们之前的假设不对，当前的这个元素才是更大的，所以我们要更新 `max_val`，让它变成这个更大的值。
4.  **遍历完成**：当所有元素都比较完之后，`max_val` 里存的就是整个数组中真正的最大值了！

### 3. 查找最小值（Min Value）的思路

查找最小值和查找最大值的思路非常相似，只是比较的方向变了：

1.  **先假设**：同样，我们先假设数组中的第一个元素就是最小值。把它存到一个叫做 `min_val` 的变量里。
2.  **逐个比较**：然后，我们从数组的第二个元素开始，一个一个地和我们“假设的最小值”进行比较。
3.  **更新最小值**：如果当前这个元素比 `min_val` 还要小，那么说明我们之前的假设不对，当前的这个元素才是更小的，所以我们要更新 `min_val`，让它变成这个更小的值。
4.  **遍历完成**：当所有元素都比较完之后，`min_val` 里存的就是整个数组中真正的最小值了！

**小技巧：** 为什么我们通常用数组的第一个元素来初始化 `max_val` 和 `min_val` 呢？因为这样可以确保 `max_val` 和 `min_val` 一定是数组中的一个真实值，而且可以处理数组中包含正数、负数或0的各种情况。

## 代码示例

### 示例一：查找数组中的最大值

```cpp
#include <bits/stdc++.h> // 包含所有常用头文件
using namespace std;      // 使用标准命名空间

int main() {
    int a[6] = {15, 8, 23, 7, 30, 12}; // 声明并初始化一个包含6个整数的数组
    int n = 6;                         // 数组的元素个数

    // 假设数组的第一个元素是最大值
    int max_val = a[0];

    // 从数组的第二个元素开始遍历，与max_val进行比较
    for (int i = 1; i < n; i++) {
        // 如果当前元素比max_val大，就更新max_val
        if (a[i] > max_val) {
            max_val = a[i];
        }
    }

    cout << "数组中的最大值是: " << max_val << endl; // 输出结果

    return 0;
}
```

### 示例二：查找数组中的最小值

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a[7] = {100, 50, 200, 10, 30, 90, 5}; // 声明并初始化一个包含7个整数的数组
    int n = 7;                                // 数组的元素个数

    // 假设数组的第一个元素是最小值
    int min_val = a[0];

    // 从数组的第二个元素开始遍历，与min_val进行比较
    for (int i = 1; i < n; i++) {
        // 如果当前元素比min_val小，就更新min_val
        if (a[i] < min_val) {
            min_val = a[i];
        }
    }

    cout << "数组中的最小值是: " << min_val << endl; // 输出结果

    return 0;
}
```

### 示例三：用户输入数据，同时查找最大值和最小值

这个例子更贴近实际应用，让程序处理用户输入的数据。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; // 用于存储数组的元素个数
    cout << "请输入数组的元素个数 (1-100之间): ";
    cin >> n;

    // 为了简化，我们假设数组最大可以有100个元素
    int a[100];

    cout << "请依次输入 " << n << " 个整数:" << endl;
    // 循环读取用户输入的n个整数到数组中
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    // 假设数组的第一个元素同时是最大值和最小值
    int max_val = a[0];
    int min_val = a[0];

    // 从数组的第二个元素开始遍历
    for (int i = 1; i < n; i++) {
        // 查找最大值
        if (a[i] > max_val) {
            max_val = a[i];
        }
        // 查找最小值
        if (a[i] < min_val) {
            min_val = a[i];
        }
    }

    cout << "你输入的数组中的最大值是: " << max_val << endl;
    cout << "你输入的数组中的最小值是: " << min_val << endl;

    return 0;
}
```

## 练习题目

同学们，是时候检验一下你们的学习成果了！请认真阅读以下题目，并选出或填入你认为正确的答案。

### 选择题/填空题

**1. 选择题**
给定一个整数数组 `int a[5] = {12, 5, 8, 20, 15};`，以下代码片段用于查找数组中的最大值，请问 `max_val` 的最终值是多少？
```cpp
int max_val = a[0];
for (int i = 1; i < 5; i++) {
    if (a[i] > max_val) {
        max_val = a[i];
    }
}
// cout << max_val << endl;
```
A. 12
B. 5
C. 20
D. 15

**2. 填空题**
给定一个整数数组 `int a[4] = {7, 3, 9, 1};`，以下代码片段用于查找数组中的最小值，请在横线处填入正确的代码。
```cpp
int min_val = a[0];
for (int i = 1; i < 4; i++) {
    if (a[i] ______ min_val) { // 填空1
        min_val = a[i];
    }
}
// cout << min_val << endl; // 最终输出结果是 ______ // 填空2
```

**3. 选择题**
以下关于数组查找最大值和最小值的说法，哪一个是**不正确**的？
A. 查找最大值时，通常将 `max_val` 初始化为数组的第一个元素。
B. 查找最小值时，`for` 循环的起始索引通常是 `0`。
C. 查找最大值时，如果当前元素 `a[i]` 大于 `max_val`，就需要更新 `max_val`。
D. 查找最小值和最大值的算法思路基本相同，只是比较符号相反。

**4. 填空题**
如果有一个数组 `int scores[3] = {90, 90, 90};`，使用我们课堂上学习的方法查找最大值，那么 `max_val` 的最终值是 `______`。

**5. 选择题**
在 C++ 中，数组的索引（下标）总是从哪个数字开始？
A. 1
B. 0
C. 2
D. 任意整数

---

### 答案与解析

**1. 选择题**
**答案：C**
**解析：**
*   `max_val` 初始化为 `a[0]`，即 `12`。
*   `i = 1` 时，`a[1]` 是 `5`，`5` 不大于 `12`，`max_val` 仍为 `12`。
*   `i = 2` 时, `a[2]` 是 `8`，`8` 不大于 `12`，`max_val` 仍为 `12`。
*   `i = 3` 时, `a[3]` 是 `20`，`20` 大于 `12`，`max_val` 更新为 `20`。
*   `i = 4` 时, `a[4]` 是 `15`，`15` 不大于 `20`，`max_val` 仍为 `20`。
*   循环结束，`max_val` 为 `20`。

**2. 填空题**
**答案：**
填空1: `<`
填空2: `1`
**解析：**
*   查找最小值时，需要比较当前元素是否**小于**当前的 `min_val`。
*   `min_val` 初始化为 `a[0]`，即 `7`。
*   `i = 1` 时，`a[1]` 是 `3`，`3 < 7`，`min_val` 更新为 `3`。
*   `i = 2` 时，`a[2]` 是 `9`，`9` 不小于 `3`，`min_val` 仍为 `3`。
*   `i = 3` 时，`a[3]` 是 `1`，`1 < 3`，`min_val` 更新为 `1`。
*   循环结束，`min_val` 为 `1`。

**3. 选择题**
**答案：B**
**解析：**
*   A 正确，这是最常用的初始化方法。
*   B 不正确。在用数组第一个元素初始化 `min_val` 或 `max_val` 后，循环通常从数组的**第二个元素**开始遍历（即 `i` 从 `1` 开始），因为第一个元素已经参与了初始化。
*   C 正确，这是更新最大值的逻辑。
*   D 正确，两种算法的结构非常相似，只是比较符 `>` 和 `<` 的区别。

**4. 填空题**
**答案：90**
**解析：**
*   `max_val` 初始化为 `scores[0]`，即 `90`。
*   `i = 1` 时，`scores[1]` 是 `90`，`90` 不大于 `max_val`（即 `90`），`max_val` 仍为 `90`。
*   `i = 2` 时，`scores[2]` 是 `90`，`90` 不大于 `max_val`（即 `90`），`max_val` 仍为 `90`。
*   循环结束，`max_val` 为 `90`。

**5. 选择题**
**答案：B**
**解析：**
在 C++ 和许多其他编程语言中，数组的索引（下标）都是从 `0` 开始的。这意味着一个有 N 个元素的数组，其索引范围是 `0` 到 `N-1`。
