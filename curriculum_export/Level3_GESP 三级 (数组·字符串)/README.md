# GESP 三级 (数组·字符串)

## 教学目标
1. 掌握一维数组的进阶应用与多维数组。
2. 理解字符串的处理方法。
3. 掌握函数的定义与调用，理解参数传递。

## 核心知识点
- 数组算法：查找（线性查找）、简单排序（冒泡/选择）。
- 字符串：`string` 类型，字符数组，常用字符串函数。
- 函数：函数声明、定义、返回值、值传递。
- 进制转换：二进制、十进制、十六进制的转换。

## 专题列表

1. **数组基础与应用**
   > 从一维数组的定义与访问出发，掌握遍历、求和、查找最大最小值、计数（桶思想）等核心操作，为字符串和高级算法打好数据容器基础。
2. **字符串处理**
3. **进制与编码**
4. **位运算**
5. **枚举进阶**
   > 以数组为容器深化多维枚举，并探索子集枚举（位运算辅助）与剪枝优化，提升穷举搜索的效率与代码规范性。
6. **模拟进阶**
   > 在掌握基础模拟思路后，深入数组模拟栈与队列、矩阵方格模拟、字符串模拟，强化将抽象过程转化为精确代码的综合实现能力。
