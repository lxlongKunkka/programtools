## 知识点讲解

字符串查找与替换，顾名思义，就是在一个大字符串里找到一个小字符串，然后把它换成另一个字符串。

### 1. 字符串查找 (String Search)

**目标：** 在一个较长的字符串（我们称之为“源字符串”或“文本”）中，找到一个较短的字符串（我们称之为“模式串”或“子串”）第一次出现的位置。

**核心思想：**
我们可以想象成两根尺子：一根是源字符串尺子，另一根是模式串尺子。我们把模式串尺子从源字符串尺子的最左边开始对齐，然后逐个字符比较。如果完全匹配，我们就找到了！如果匹配失败，就把模式串尺子向右移动一位，再重新开始比较，直到源字符串的末尾。

**举个例子：**
源字符串 `s = "ababa"`
模式串 `p = "aba"`

1.  `s`: `a b a b a`
    `p`: `a b a`
    比较：`s[0]` vs `p[0]` (a==a), `s[1]` vs `p[1]` (b==b), `s[2]` vs `p[2]` (a==a)。匹配成功！在索引 0 处找到。

**C++ 实现思路 (使用 `char[]` 数组):**

*   我们需要一个外层循环，控制模式串在源字符串中的起始位置。
*   内层循环则负责从当前起始位置开始，逐个字符比较模式串和源字符串的对应部分。
*   如果内层循环完整匹配了模式串的每一个字符，就说明找到了。

```cpp
// 查找函数：在源字符串 s 中查找模式串 p 第一次出现的位置
// 返回第一次出现的起始索引，如果没找到则返回 -1
int findString(char s[], char p[]) {
    int n = 0; // 源字符串 s 的长度
    while (s[n] != '\0') {
        n++;
    }

    int m = 0; // 模式串 p 的长度
    while (p[m] != '\0') {
        m++;
    }

    // 如果模式串比源字符串长，肯定找不到
    if (m > n) {
        return -1;
    }

    // 外层循环：控制模式串在源字符串中的起始位置 i
    for (int i = 0; i <= n - m; i++) {
        bool match = true; // 假设当前位置匹配
        // 内层循环：比较模式串的每个字符
        for (int j = 0; j < m; j++) {
            if (s[i + j] != p[j]) {
                match = false; // 发现不匹配
                break;         // 立即跳出内层循环
            }
        }
        if (match) {
            return i; // 找到匹配，返回起始索引
        }
    }
    return -1; // 遍历完所有可能位置，没有找到
}
```

### 2. 字符串替换 (String Replace)

**目标：** 在一个源字符串中，将找到的模式串替换成另一个新的字符串。

**核心思想：**
替换比查找要复杂一些，因为替换后的字符串长度可能和原来的模式串长度不一样。
例如：
*   把 "apple" 替换成 "banana" (长度不变)
*   把 "cat" 替换成 "doggy" (变长了)
*   把 "elephant" 替换成 "mouse" (变短了)

如果直接在原字符串数组上修改，当长度变化时，就需要移动大量的字符，这非常容易出错。
**更安全、更简单的做法是：创建一个新的字符串数组，将替换后的结果逐步构建在新数组中。**

**C++ 实现思路 (使用 `char[]` 数组):**

1.  我们需要一个足够大的新 `char[]` 数组来存储替换后的结果。
2.  遍历源字符串。
3.  每当我们找到一个模式串的匹配，就不是拷贝模式串，而是拷贝替换串到新数组中。
4.  如果没有找到模式串的匹配，就拷贝源字符串的当前字符到新数组中。

**举个例子：**
源字符串 `s = "I like apples and apples."`
模式串 `p = "apples"`
替换串 `r = "bananas"`

**替换过程：**
*   从 `s` 的开头开始，`"I like "` 部分没有匹配 `p`，直接复制到新字符串。
*   在 `s` 的某个位置找到 `p` (`"apples"`)，这时不复制 `"apples"`, 而是复制 `r` (`"bananas"`) 到新字符串。
*   继续从 `s` 中 `p` 的后面开始，`" and "` 部分没有匹配，直接复制。
*   再次找到 `p` (`"apples"`)，复制 `r` (`"bananas"`)。
*   最后 `"."` 没有匹配，直接复制。
*   最终得到 `"I like bananas and bananas."`

```cpp
// 替换函数：在源字符串 s 中查找模式串 p，并替换为替换串 r
// 注意：这个函数会创建一个新的字符串来存储结果，并返回结果字符串的指针
// 这是一个简化版本，只替换第一个找到的模式串。
// 如果要替换所有，需要循环调用查找并更新起始搜索位置。
// 为了简化，我们假设结果字符串的长度不会超过 MAX_LEN
const int MAX_LEN = 1000; // 定义一个足够大的缓冲区大小

void replaceFirst(char s[], char p[], char r[], char result[]) {
    int n = 0; // 源字符串 s 的长度
    while (s[n] != '\0') {
        n++;
    }

    int m = 0; // 模式串 p 的长度
    while (p[m] != '\0') {
        m++;
    }

    int k = 0; // 替换串 r 的长度
    while (r[k] != '\0') {
        k++;
    }

    int idx = 0; // result 数组的当前写入位置
    bool replaced = false; // 标记是否已经替换过一次

    for (int i = 0; i < n; ) { // 遍历源字符串 s
        // 尝试在当前位置 i 匹配模式串 p
        bool match = true;
        if (!replaced && i <= n - m) { // 只有还没替换过，并且剩余长度足够才尝试匹配
            for (int j = 0; j < m; j++) {
                if (s[i + j] != p[j]) {
                    match = false;
                    break;
                }
            }
        } else {
            match = false; // 已经替换过，或者剩余长度不足，不再尝试匹配
        }

        if (match) {
            // 如果匹配成功，将替换串 r 拷贝到 result
            for (int j = 0; j < k; j++) {
                result[idx++] = r[j];
            }
            i += m; // 源字符串的索引跳过被替换的模式串长度
            replaced = true; // 标记已经替换过
        } else {
            // 如果不匹配，或者已经替换过了，将源字符串当前字符拷贝到 result
            result[idx++] = s[i];
            i++; // 源字符串索引前进一位
        }
    }
    result[idx] = '\0'; // 添加字符串结束符
}

// 替换所有函数：在源字符串 s 中查找模式串 p，并替换为替换串 r
// 同样，会创建一个新的字符串来存储结果
void replaceAll(char s[], char p[], char r[], char result[]) {
    int n = 0; // 源字符串 s 的长度
    while (s[n] != '\0') {
        n++;
    }

    int m = 0; // 模式串 p 的长度
    while (p[m] != '\0') {
        m++;
    }

    int k = 0; // 替换串 r 的长度
    while (r[k] != '\0') {
        k++;
    }

    int idx = 0; // result 数组的当前写入位置

    for (int i = 0; i < n; ) { // 遍历源字符串 s
        // 尝试在当前位置 i 匹配模式串 p
        bool match = true;
        if (i <= n - m) { // 剩余长度足够才尝试匹配
            for (int j = 0; j < m; j++) {
                if (s[i + j] != p[j]) {
                    match = false;
                    break;
                }
            }
        } else {
            match = false; // 剩余长度不足，不尝试匹配
        }

        if (match) {
            // 如果匹配成功，将替换串 r 拷贝到 result
            for (int j = 0; j < k; j++) {
                result[idx++] = r[j];
            }
            i += m; // 源字符串的索引跳过被替换的模式串长度
        } else {
            // 如果不匹配，将源字符串当前字符拷贝到 result
            result[idx++] = s[i];
            i++; // 源字符串索引前进一位
        }
    }
    result[idx] = '\0'; // 添加字符串结束符
}
```

**小提示：** `char[]` 数组在 C++ 中需要手动管理内存和字符串结束符 `'\0'`。所以在使用时一定要小心，确保数组足够大，并且在字符串末尾加上 `'\0'`。

## 代码示例

让我们把刚才学习的查找和替换功能，放到一个完整的 C++ 程序里运行一下吧！

```cpp
#include <bits/stdc++.h> // 包含所有常用头文件
using namespace std;     // 使用标准命名空间

const int MAX_STRING_LEN = 1005; // 定义一个足够大的字符串缓冲区长度

// 字符串查找函数：在源字符串 s 中查找模式串 p 第一次出现的位置
// 返回第一次出现的起始索引，如果没找到则返回 -1
int findString(char s[], char p[]) {
    int n = 0; // 源字符串 s 的长度
    while (s[n] != '\0') {
        n++;
    }

    int m = 0; // 模式串 p 的长度
    while (p[m] != '\0') {
        m++;
    }

    if (m > n) { // 如果模式串比源字符串长，肯定找不到
        return -1;
    }

    for (int i = 0; i <= n - m; i++) {
        bool match = true;
        for (int j = 0; j < m; j++) {
            if (s[i + j] != p[j]) {
                match = false;
                break;
            }
        }
        if (match) {
            return i;
        }
    }
    return -1;
}

// 字符串替换函数：在源字符串 s 中查找模式串 p，并替换为替换串 r
// 结果存储在 result 数组中，替换所有匹配项
void replaceAll(char s[], char p[], char r[], char result[]) {
    int n = 0; // 源字符串 s 的长度
    while (s[n] != '\0') {
        n++;
    }

    int m = 0; // 模式串 p 的长度
    while (p[m] != '\0') {
        m++;
    }

    int k = 0; // 替换串 r 的长度
    while (r[k] != '\0') {
        k++;
    }

    int idx = 0; // result 数组的当前写入位置

    for (int i = 0; i < n; ) { // 遍历源字符串 s
        bool match = true;
        if (i <= n - m) { // 剩余长度足够才尝试匹配
            for (int j = 0; j < m; j++) {
                if (s[i + j] != p[j]) {
                    match = false;
                    break;
                }
            }
        } else {
            match = false; // 剩余长度不足，不尝试匹配
        }

        if (match) {
            // 如果匹配成功，将替换串 r 拷贝到 result
            for (int j = 0; j < k; j++) {
                result[idx++] = r[j];
            }
            i += m; // 源字符串的索引跳过被替换的模式串长度
        } else {
            // 如果不匹配，将源字符串当前字符拷贝到 result
            result[idx++] = s[i];
            i++; // 源字符串索引前进一位
        }
        // 确保不会超出 result 缓冲区
        if (idx >= MAX_STRING_LEN * 2 - 1) { // 留出空间给结束符，乘以2是因为替换后可能变长
            // 如果结果字符串太长，进行截断并发出警告
            // 实际应用中需要更严谨的错误处理或动态内存分配
            cerr << "Warning: Result string truncated due to buffer limit!" << endl;
            break;
        }
    }
    result[idx] = '\0'; // 添加字符串结束符
}


int main() {
    char source[MAX_STRING_LEN];
    char pattern[MAX_STRING_LEN];
    char replacement[MAX_STRING_LEN];
    char result[MAX_STRING_LEN * 2]; // 替换后字符串可能变长，预留更大空间

    cout << "===== 字符串查找示例 =====" << endl;
    cout << "请输入源字符串 (含空格请用getline): ";
    cin.getline(source, MAX_STRING_LEN); // 读取一整行字符串

    cout << "请输入要查找的模式串: ";
    cin.getline(pattern, MAX_STRING_LEN);

    int pos = findString(source, pattern);
    if (pos != -1) {
        cout << "模式串 \"" << pattern << "\" 在源字符串 \"" << source << "\" 中第一次出现的位置是: " << pos << endl;
    } else {
        cout << "模式串 \"" << pattern << "\" 在源字符串中未找到。" << endl;
    }

    cout << "\n===== 字符串替换示例 =====" << endl;
    cout << "请输入源字符串 (含空格请用getline): ";
    // 清空 cin 缓冲区，以防上次 cin.getline 之后留有换行符
    cin.clear();
    // cin.ignore(numeric_limits<streamsize>::max(), '\n'); // 在某些情况下可能需要，这里每次都是getline，所以不是必须
    cin.getline(source, MAX_STRING_LEN);

    cout << "请输入要查找并替换的模式串: ";
    cin.getline(pattern, MAX_STRING_LEN);

    cout << "请输入替换为的字符串: ";
    cin.getline(replacement, MAX_STRING_LEN);

    // 调用替换函数
    replaceAll(source, pattern, replacement, result);

    cout << "原始字符串: " << source << endl;
    cout << "替换后字符串: " << result << endl;

    return 0;
}
```
**注意:** 在 `main` 函数中，如果连续使用 `cin.getline()`，或者在 `cin >> var;` 之后使用 `cin.getline()`，可能会遇到输入缓冲区遗留换行符的问题。在竞赛中，通常会使用 `scanf("%s", s)` 读取单词，或 `fgets(s, MAX_STRING_LEN, stdin)` 读取带空格的行。本示例为了简化交互，统一使用 `cin.getline`。为了避免潜在的缓冲区问题，在第二个 `cin.getline` 前添加了 `cin.clear();`。

1.  在 C++ 中，`char` 数组表示字符串时，其末尾必须以字符 `____` (ASCII 值为 0) 作为结束标志。
2.  我们实现的 `findString` 函数中，外层循环控制的是模式串在源字符串中的 `____` 位置。
3.  如果源字符串 `s = "apple"`, 模式串 `p = "pp"`, 替换串 `r = "xx"`。使用 `replaceAll` 函数后，`result` 数组的内容将是 `____`。

---

### 答案与解析

**选择题答案：**

1.  **C**
    *   **解析：** C 选项 `char s[5] = "hello";` 声明了一个只能存储 5 个字符的数组。而 "hello" 字符串实际需要 5 个字符加上一个 `\0` 结束符，共 6 个字符。因此，`s[5]` 无法完整存储 "hello" 并包含结束符，会导致缓冲区溢出或未定义行为。A、B、D 选项都提供了足够的空间。

2.  **A**
    *   **解析：** `findString` 函数查找的是模式串第一次出现的位置。在 "ababa" 中，"aba" 第一次出现在索引 0 的位置 (`s[0]s[1]s[2]`)。

3.  **B**
    *   **解析：** 当替换后的字符串长度与原模式串长度不同时，直接在原数组中进行修改（A 和 D）会导致字符移动复杂，容易出错，甚至可能覆盖掉后面的有用数据。创建新数组（B）是更安全、更易于理解和实现的方法。C 选项的 `memset` 不适用于替换操作。

**填空题答案：**

1.  `\0`
    *   **解析：** 这是 C 风格字符串的约定，`\0` (空字符) 标记字符串的结束。

2.  起始
    *   **解析：** 外层循环变量 `i` 控制模式串 `p` 在源字符串 `s` 中开始比较的索引位置。

3.  `apxxle`
    *   **解析：** 源字符串 `apple`，模式串 `pp` 位于索引 1 (`s[1]s[2]`)。替换为 `xx` 后，`s[0]`='a'不变，`s[1]s[2]`变为 `xx`，`s[3]s[4]`='le'不变。所以结果是 `apxxle`。

### 课后巩固作业 (结果性检验)

1.  **必做题：**
    *   请编写一个 C++ 程序，要求用户输入一个句子，然后输入一个单词和一个字符。程序需要找出句子中所有该单词出现的位置，并将该单词的**第一个字符**替换为用户输入的字符，然后输出修改后的句子。
        *   **示例：**
            *   输入句子: `I like apple and orange.`
            *   输入单词: `apple`
            *   输入字符: `X`
            *   输出: `I like Xpple and orange.`
    *   **要求：** 必须使用 `char[]` 数组实现，不能使用 C++ STL 的 `string` 对象或 `string` 类的方法。

2.  **选做题 (挑战题)：**
    *   请编写一个 C++ 程序，实现一个简单的“敏感词过滤”功能。程序需要维护一个敏感词列表（例如，`char sensitiveWords[3][20] = {"bad", "ugly", "stupid"};`）。用户输入一个句子，程序需要将句子中所有敏感词替换为 `***` (三个星号)。
        *   **示例：**
            *   输入句子: `You are a bad and ugly boy.`
            *   输出: `You are a *** and *** boy.`
    *   **要求：** 同样使用 `char[]` 数组，并考虑如何处理多个敏感词的查找和替换。
