### 深度知识点讲解

#### 1. 什么是字符串哈希？—— 给字符串一个“身份证号”

字符串哈希的核心思想，就是把一个字符串，通过某种计算方式，映射成一个数字（通常是一个 `long long` 类型的整数）。这个数字，就是这个字符串的“哈希值”或者说“指纹”。

**怎么映射呢？** 我们可以把字符串想象成一个 `P` 进制的数。

假设我们有一个字符串 `S = "abc"`。
我们可以给每个字符一个数值：`'a' -> 1`, `'b' -> 2`, `'c' -> 3` (或者直接用 ASCII 码，例如 `'a' -> 97`)。
然后，我们可以选择一个基数 `P` (比如 `P=131` 或 `P=233`)，以及一个模数 `M` (比如 `M=10^9 + 7` 或 `M=10^9 + 9`)。

那么字符串 `S` 的哈希值就可以这样计算：
`Hash("abc") = ( ('a'的数值) * P^2 + ('b'的数值) * P^1 + ('c'的数值) * P^0 ) % M`

如果 `'a' -> 1, 'b' -> 2, 'c' -> 3`，并且 `P=10`，那么 `Hash("abc") = (1 * 10^2 + 2 * 10^1 + 3 * 10^0) % M = (100 + 20 + 3) % M = 123 % M`。
这就像我们把字符串 `abc` 看作是 10 进制数 123。

这种方法叫做**多项式哈希（Polynomial Hashing）**。

#### 2. 为什么需要字符串哈希？—— 飞一般的比较速度

传统上，比较两个字符串是否相等，需要逐个字符地比较，时间复杂度是 `O(L)`，其中 `L` 是字符串的长度。如果我们要在一个长字符串中查找一个子串，可能需要 `O(N * L)` 的时间，`N` 是长字符串的长度。

有了哈希值，比较两个字符串就变成了比较两个数字！两个数字的比较是 `O(1)` 的！这就像是：
*   你要判断两个人是不是同一个人，不需要看他们的长相、口音、习惯，只需要看他们的身份证号是不是一样，`O(1)`搞定！
*   你要判断一个文件是不是另一个文件的某个段落，不需要读完整个段落，只需要计算这个段落的“指纹”，再和目标指纹比较，`O(1)`搞定！

这样一来，很多字符串操作的效率就能大大提升。

#### 3. 怎么想到的？—— 从数字系统到字符串的类比

这个想法来源于我们熟悉的数字系统。
一个十进制数 `123`，我们知道它是 `1 * 10^2 + 2 * 10^1 + 3 * 10^0`。
一个二进制数 `101`，我们知道它是 `1 * 2^2 + 0 * 2^1 + 1 * 2^0`。
本质上，就是每个位上的数字乘以一个基数的幂次，然后求和。

字符串也可以这样看！把每个字符看作是“位上的数字”，选择一个合适的“基数 `P`”，然后加起来。为了防止数值过大溢出，我们每次计算都对一个大质数 `M` 取模。

#### 4. 哈希碰撞怎么办？—— 双保险：双哈希！

虽然哈希值是字符串的“指纹”，但指纹也可能“撞车”！
不同的字符串计算出相同的哈希值，这种情况就叫做**哈希碰撞（Hash Collision）**。
这就像两个人恰好身份证号一样，或者两份不同的文件算出了相同的“指纹”！
哈希碰撞是非常危险的，它会导致我们的判断出错：把不同的字符串误认为是相同的。

为了降低哈希碰撞的概率，我们通常采用以下策略：
*   **选择合适的基数 `P`**: `P` 应该是一个质数，且大于字符集的大小（例如，如果只包含小写字母，`P` 至少大于 26）。常用的 `P` 有 131, 13331, 233, 23333 等。
*   **选择合适的模数 `M`**: `M` 也应该是一个大质数，例如 `10^9 + 7`, `10^9 + 9`, `998244353` 等。
*   **双哈希（Double Hashing）**: 这是最常用的方法！我们不只计算一个哈希值，而是计算两个哈希值！用两组不同的 `(P, M)` 来计算。
    *   `Hash1 = ( ('a'的数值) * P1^2 + ('b'的数值) * P1^1 + ('c'的数值) * P1^0 ) % M1`
    *   `Hash2 = ( ('a'的数值) * P2^2 + ('b'的数值) * P2^1 + ('c'的数值) * P2^0 ) % M2`
    如果两个字符串的 `(Hash1, Hash2)` 都相同，那么它们是相同字符串的概率就非常高了，几乎可以认为是 100% 相同。这就像用指纹和面部识别双重验证，安全性大大提高！

#### 5. 如何计算子串哈希？—— 前缀哈希与 O(1) 区间查询

我们不可能每次都重新计算一个子串的哈希值，那样效率又回去了。
幸运的是，我们可以利用**前缀哈希**来在 `O(1)` 时间内计算任意子串的哈希值。

假设我们已经预处理出了字符串 `S` 的所有前缀的哈希值：
`h[i]` 表示 `S[0...i-1]` 这个前缀的哈希值。
`h[0] = 0` (空字符串的哈希值)
`h[1] = (S[0]的数值) % M`
`h[2] = ( (S[0]的数值) * P + (S[1]的数值) ) % M`
`h[i] = ( h[i-1] * P + (S[i-1]的数值) ) % M`

现在，我们想计算子串 `S[L...R]` (长度为 `len = R - L + 1`) 的哈希值。
`Hash(S[L...R]) = ( (S[L]的数值) * P^(len-1) + ... + (S[R]的数值) * P^0 ) % M`

我们可以发现：
`h[R+1] = ( (S[0]的数值) * P^R + ... + (S[L-1]的数值) * P^(R-L+1) + (S[L]的数值) * P^(len-1) + ... + (S[R]的数值) * P^0 ) % M`
`h[L] = ( (S[0]的数值) * P^(L-1) + ... + (S[L-1]的数值) * P^0 ) % M`

我们把 `h[L]` 乘以 `P^len`，再对 `M` 取模，就得到了 `S[0...L-1]` 的哈希值“左移”了 `len` 位：
`h[L] * P^len % M = ( (S[0]的数值) * P^(L-1+len) + ... + (S[L-1]的数值) * P^len ) % M`

所以，子串 `S[L...R]` 的哈希值可以这样计算：
`Hash(S[L...R]) = ( h[R+1] - (h[L] * P^len) % M + M ) % M`
注意，`+ M` 是为了处理减法可能出现的负数。
这里的 `P^len` 需要预先计算好，存放在一个数组 `p_pow[len]` 中。

**预处理过程：**
1.  计算 `p_pow[i] = P^i % M`。
2.  计算 `h[i]`，即 `S[0...i-1]` 的哈希值。

**整个过程的时间复杂度：**
*   预处理：`O(N)`，`N` 是字符串长度。
*   查询任意子串哈希值：`O(1)`。
这使得哈希成为解决很多字符串问题的强大工具。

#### 6. 哈希的局限性与应用场景

*   **局限性**: 哈希是一种概率性算法。理论上，哈希碰撞是无法完全避免的，只能通过双哈希等方法将概率降到极低，低到在实际竞赛中可以忽略不计。
*   **应用场景**:
    *   **字符串匹配**: 判断一个模式串是否是文本串的子串。
    *   **最长公共子串/子序列**: 结合二分答案。
    *   **回文串判断**: 比较正向子串哈希和反向子串哈希。
    *   **统计不同子串**: 将所有子串哈希值存入 `std::set` 或排序去重。
    *   **循环同构**: 判断两个字符串是否通过循环位移可以相互得到。

#### 例题 1: 查找所有出现次数大于等于 `k` 的子串

**题目描述**: 给定一个字符串 `s` 和一个整数 `k`。找出所有在 `s` 中出现次数大于等于 `k` 的子串，并输出这些子串（可以不要求去重，只要求找出）。为了简化问题，我们只输出其中一个满足条件的子串即可，或者统计满足条件的子子串的数量。这里我们统计出现次数大于等于 `k` 的**不同**子串的数量。

**输入**:
`s = "ababa"`
`k = 2`

**输出**:
`3` (子串 "a", "b", "ab", "ba", "aba", "bab", "ababa" 中，"a" 出现3次，"b" 出现2次，"ab" 出现2次，"aba" 出现2次。不同子串有 "a", "b", "ab", "aba" 四个，其中出现次数 >= 2 的有 "a", "b", "ab", "aba"。题目简化为统计数量，那么答案是4。如果要求输出子串，则需进一步处理。)
我们这里实现统计不同子串的数量。

**思路分析**:
1.  **生成所有子串**: 一个长度为 `N` 的字符串有 `N * (N+1) / 2` 个子串。
2.  **计算哈希值**: 对每个子串计算其双哈希值 `(h1, h2)`。
3.  **统计频率**: 将每个子串的哈希值对 `(h1, h2)` 存入 `std::map<std::pair<long long, long long>, int>` 中，统计每个哈希值对出现的次数。
4.  **筛选**: 遍历 `map`，统计出现次数大于等于 `k` 的键值对的数量。

**具体步骤**:
1.  预处理 `P1` 和 `P2` 的幂次，以及字符串 `s` 的双前缀哈希值。
2.  遍历所有可能的子串长度 `len` 从 `1` 到 `n`。
3.  对于每个 `len`，遍历所有可能的起始位置 `i` (从 `0` 到 `n - len`)。
4.  利用区间哈希函数 `get_hash(i, i + len - 1)` 获取当前子串的哈希值对。
5.  将哈希值对存入 `map` 并更新计数。
6.  最后遍历 `map`，统计 `count >= k` 的哈希对数量。

**代码实现**:

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <map>
#include <utility> // For std::pair

// 使用 Allman 风格大括号

const long long P1 = 131; // 第一个基数
const long long M1 = 1e9 + 7; // 第一个模数

const long long P2 = 233; // 第二个基数
const long long M2 = 1e9 + 9; // 第二个模数

std::vector<long long> p_pow1; // P1 的幂次
std::vector<long long> h1;     // 前缀哈希值1

std::vector<long long> p_pow2; // P2 的幂次
std::vector<long long> h2;     // 前缀哈希值2

// 初始化哈希所需数据
void init_hash(const std::string& s)
{
    int n = s.length();

    p_pow1.resize(n + 1);
    h1.resize(n + 1, 0);
    p_pow2.resize(n + 1);
    h2.resize(n + 1, 0);

    p_pow1[0] = 1;
    p_pow2[0] = 1;

    for (int i = 0; i < n; ++i)
    {
        // 计算 P1 的幂次
        p_pow1[i + 1] = (p_pow1[i] * P1) % M1;
        // 计算前缀哈希1
        h1[i + 1] = (h1[i] * P1 + (s[i] - 'a' + 1)) % M1;

        // 计算 P2 的幂次
        p_pow2[i + 1] = (p_pow2[i] * P2) % M2;
        // 计算前缀哈希2
        h2[i + 1] = (h2[i] * P2 + (s[i] - 'a' + 1)) % M2;
    }
}

// 获取子串 s[l...r] 的哈希值对 (0-indexed, inclusive)
std::pair<long long, long long> get_hash(int l, int r)
{
    long long current_h1 = (h1[r + 1] - (h1[l] * p_pow1[r - l + 1]) % M1 + M1) % M1;
    long long current_h2 = (h2[r + 1] - (h2[l] * p_pow2[r - l + 1]) % M2 + M2) % M2;
    return {current_h1, current_h2};
}

int main()
{
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    std::string s;
    int k;
    std::cin >> s >> k;

    init_hash(s);
    int n = s.length();

    std::map<std::pair<long long, long long>, int> m;

    for (int i = 0; i < n; ++i)
    {
        for (int j = i; j < n; ++j)
        {
            // 子串 s[i...j]
            std::pair<long long, long long> current_sub_hash = get_hash(i, j);
            m[current_sub_hash]++;
        }
    }

    int ans = 0;
    for (auto const& [key, val] : m)
    {
        if (val >= k)
        {
            ans++;
        }
    }

    std::cout << ans << std::endl;

    return 0;
}

```

**复杂度分析**:
*   **时间复杂度**:
    *   `init_hash`: `O(N)`。
    *   生成所有子串并计算哈希值：`N` 个起始位置，每个起始位置最多 `N` 个长度。总共有 `O(N^2)` 个子串。每个子串的哈希值计算是 `O(1)`。
    *   `map` 插入和查找的平均时间复杂度是 `O(log(N^2))` (因为最多有 `N^2` 个不同的哈希值对)。
    *   总时间复杂度：`O(N^2 * log(N^2))`，即 `O(N^2 log N)`。
*   **空间复杂度**: `O(N)` 用于存储幂次和前缀哈希数组，`O(N^2)` 用于存储 `map` 中的哈希值对。总空间复杂度 `O(N^2)`。

#### 例题 2: 最长回文子串 (长度)

**题目描述**: 给定一个字符串 `s`，找出其最长回文子串的长度。

**输入**:
`s = "babad"`

**输出**:
`3` (最长回文子串可以是 "bab" 或 "aba")

**思路分析**:
回文串的特点是正读和反读都一样。
我们可以利用哈希来快速判断一个子串是否是回文串：比较它的正向哈希值和反向哈希值。

1.  **预处理**:
    *   计算字符串 `s` 的正向双前缀哈希值 `(h1, h2)` 和 `(p_pow1, p_pow2)`。
    *   计算字符串 `s` 的反向字符串 `s_rev` 的双前缀哈希值 `(h_rev1, h_rev2)` 和 `(p_pow1, p_pow2)`。
    *   `get_hash(l, r)` 返回 `s[l...r]` 的正向哈希。
    *   `get_hash_rev(l, r)` 返回 `s_rev` 中对应 `s[l...r]` 的子串的正向哈希。
        *   如果 `s[l...r]` 是 `s` 中的子串，那么它在 `s_rev` 中对应的子串是 `s_rev[n-1-r ... n-1-l]`。
        *   所以 `get_hash_rev(l, r)` 实际上是调用 `get_hash_rev_internal(n-1-r, n-1-l)`。

2.  **二分答案**: 最长回文子串的长度可以在 `1` 到 `N` 之间。我们可以二分这个长度 `len`。
    *   对于一个给定的长度 `len`，我们检查是否存在一个长度为 `len` 的回文子串。
    *   如何检查？遍历所有长度为 `len` 的子串 `s[i...i+len-1]`。
        *   计算 `s[i...i+len-1]` 的正向哈希值 `H_fwd = get_hash(i, i+len-1)`。
        *   计算 `s[i...i+len-1]` 的反向哈希值 `H_rev = get_hash_rev(i, i+len-1)`。
        *   如果 `H_fwd == H_rev`，则找到了一个长度为 `len` 的回文子串，返回 `true`。
    *   如果 `check(len)` 为 `true`，说明可能存在更长的回文串，`ans = len`, `low = len + 1`。
    *   如果 `check(len)` 为 `false`，说明太长了，`high = len - 1`。

3.  **注意奇偶长度**: 回文串可以有奇数长度（中心是一个字符）或偶数长度（中心是两个字符）。
    *   可以分别二分奇数长度和偶数长度，然后取最大值。
    *   或者，在 `check(len)` 函数中，遍历所有起始位置 `i`，检查 `s[i...i+len-1]` 是否是回文。

**代码实现**:

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm> // For std::max, std::reverse

// 使用 Allman 风格大括号

const long long P1 = 131; // 第一个基数
const long long M1 = 1e9 + 7; // 第一个模数

const long long P2 = 233; // 第二个基数
const long long M2 = 1e9 + 9; // 第二个模数

std::vector<long long> p_pow1; // P1 的幂次
std::vector<long long> h_fwd1; // 正向字符串前缀哈希值1
std::vector<long long> h_bwd1; // 反向字符串前缀哈希值1 (用于计算原字符串子串的反向哈希)

std::vector<long long> p_pow2; // P2 的幂次
std::vector<long long> h_fwd2; // 正向字符串前缀哈希值2
std::vector<long long> h_bwd2; // 反向字符串前缀哈希值2

// 初始化哈希所需数据
void init_hash(const std::string& s)
{
    int n = s.length();

    p_pow1.resize(n + 1);
    h_fwd1.resize(n + 1, 0);
    h_bwd1.resize(n + 1, 0);

    p_pow2.resize(n + 1);
    h_fwd2.resize(n + 1, 0);
    h_bwd2.resize(n + 1, 0);

    p_pow1[0] = 1;
    p_pow2[0] = 1;

    for (int i = 0; i < n; ++i)
    {
        // 计算 P1 的幂次
        p_pow1[i + 1] = (p_pow1[i] * P1) % M1;
        // 计算正向字符串前缀哈希1
        h_fwd1[i + 1] = (h_fwd1[i] * P1 + (s[i] - 'a' + 1)) % M1;
        // 计算反向字符串前缀哈希1 (从 s[n-1] 到 s[0] 构造)
        h_bwd1[i + 1] = (h_bwd1[i] * P1 + (s[n - 1 - i] - 'a' + 1)) % M1;

        // 计算 P2 的幂次
        p_pow2[i + 1] = (p_pow2[i] * P2) % M2;
        // 计算正向字符串前缀哈希2
        h_fwd2[i + 1] = (h_fwd2[i] * P2 + (s[i] - 'a' + 1)) % M2;
        // 计算反向字符串前缀哈希2
        h_bwd2[i + 1] = (h_bwd2[i] * P2 + (s[n - 1 - i] - 'a' + 1)) % M2;
    }
}

// 获取正向子串 s[l...r] 的哈希值对 (0-indexed, inclusive)
std::pair<long long, long long> get_forward_hash(int l, int r)
{
    long long current_h1 = (h_fwd1[r + 1] - (h_fwd1[l] * p_pow1[r - l + 1]) % M1 + M1) % M1;
    long long current_h2 = (h_fwd2[r + 1] - (h_fwd2[l] * p_pow2[r - l + 1]) % M2 + M2) % M2;
    return {current_h1, current_h2};
}

// 获取子串 s[l...r] 的反向哈希值对 (0-indexed, inclusive)
// 实际上是获取 s_rev[N-1-r ... N-1-l] 的正向哈希
std::pair<long long, long long> get_backward_hash(int l, int r, int n)
{
    int rev_l = n - 1 - r; // 在反向字符串中的起始位置
    int rev_r = n - 1 - l; // 在反向字符串中的结束位置
    
    long long current_h1 = (h_bwd1[rev_r + 1] - (h_bwd1[rev_l] * p_pow1[rev_r - rev_l + 1]) % M1 + M1) % M1;
    long long current_h2 = (h_bwd2[rev_r + 1] - (h_bwd2[rev_l] * p_pow2[rev_r - rev_l + 1]) % M2 + M2) % M2;
    return {current_h1, current_h2};
}

// 检查是否存在长度为 len 的回文子串
bool check(int len, int n)
{
    if (len == 0) return true; // 空串也是回文
    if (len > n) return false;

    for (int i = 0; i <= n - len; ++i)
    {
        // 子串 s[i ... i + len - 1]
        std::pair<long long, long long> fwd_hash = get_forward_hash(i, i + len - 1);
        std::pair<long long, long long> bwd_hash = get_backward_hash(i, i + len - 1, n);
        
        if (fwd_hash == bwd_hash)
        {
            return true;
        }
    }
    return false;
}

int main()
{
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    std::string s;
    std::cin >> s;
    int n = s.length();

    init_hash(s);

    int ans = 0;
    // 二分偶数长度回文串
    int low = 2, high = n / 2 * 2; // 偶数长度
    int current_max_even = 0;
    while (low <= high)
    {
        int mid = low + (high - low) / 2;
        if (mid % 2 != 0) // 确保是偶数
        {
            mid--;
        }
        if (mid <= 0) // 避免负数或0
        {
            low = 2; // 继续向右找
            continue;
        }

        if (check(mid, n))
        {
            current_max_even = mid;
            low = mid + 2;
        }
        else
        {
            high = mid - 2;
        }
    }
    ans = std::max(ans, current_max_even);

    // 二分奇数长度回文串
    low = 1, high = (n - 1) / 2 * 2 + 1; // 奇数长度
    int current_max_odd = 0;
    while (low <= high)
    {
        int mid = low + (high - low) / 2;
        if (mid % 2 == 0) // 确保是奇数
        {
            mid++;
        }
        if (mid > n) // 避免超长
        {
            high = n;
            continue;
        }
        if (mid <= 0) // 避免负数或0
        {
            low = 1;
            continue;
        }

        if (check(mid, n))
        {
            current_max_odd = mid;
            low = mid + 2;
        }
        else
        {
            high = mid - 2;
        }
    }
    ans = std::max(ans, current_max_odd);

    std::cout << ans << std::endl;

    return 0;
}

```

**复杂度分析**:
*   **时间复杂度**:
    *   `init_hash`: `O(N)`。
    *   `check(len)` 函数：遍历 `O(N)` 个子串，每个子串 `O(1)` 哈希查询。总计 `O(N)`。
    *   二分查找长度：长度的范围是 `O(N)`，所以二分次数是 `O(log N)`。
    *   总时间复杂度：`O(N log N)`。
*   **空间复杂度**: `O(N)` 用于存储幂次和前缀哈希数组。

#### 例题 3: 统计不同子串的数量

**题目描述**: 给定一个字符串 `s`，统计它有多少个不同的子串。

**输入**:
`s = "aba"`

**输出**:
`5` (不同的子串有 "a", "b", "ab", "ba", "aba")

**思路分析**:
1.  **生成所有子串**: 和例题 1 类似，遍历所有可能的起始位置 `i` 和结束位置 `j`，得到 `s[i...j]`。
2.  **计算哈希值**: 对每个子串计算其双哈希值 `(h1, h2)`。
3.  **去重**: 将所有子串的哈希值对存入 `std::set<std::pair<long long, long long>>` 中。`set` 会自动处理重复元素。
4.  **统计**: `set` 的大小就是不同子串的数量。

**代码实现**:

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <set>      // For std::set
#include <utility>  // For std::pair

// 使用 Allman 风格大括号

const long long P1 = 131; // 第一个基数
const long long M1 = 1e9 + 7; // 第一个模数

const long long P2 = 233; // 第二个基数
const long long M2 = 1e9 + 9; // 第二个模数

std::vector<long long> p_pow1; // P1 的幂次
std::vector<long long> h1;     // 前缀哈希值1

std::vector<long long> p_pow2; // P2 的幂次
std::vector<long long> h2;     // 前缀哈希值2

// 初始化哈希所需数据
void init_hash(const std::string& s)
{
    int n = s.length();

    p_pow1.resize(n + 1);
    h1.resize(n + 1, 0);
    p_pow2.resize(n + 1);
    h2.resize(n + 1, 0);

    p_pow1[0] = 1;
    p_pow2[0] = 1;

    for (int i = 0; i < n; ++i)
    {
        // 计算 P1 的幂次
        p_pow1[i + 1] = (p_pow1[i] * P1) % M1;
        // 计算前缀哈希1
        h1[i + 1] = (h1[i] * P1 + (s[i] - 'a' + 1)) % M1;

        // 计算 P2 的幂次
        p_pow2[i + 1] = (p_pow2[i] * P2) % M2;
        // 计算前缀哈希2
        h2[i + 1] = (h2[i] * P2 + (s[i] - 'a' + 1)) % M2;
    }
}

// 获取子串 s[l...r] 的哈希值对 (0-indexed, inclusive)
std::pair<long long, long long> get_hash(int l, int r)
{
    long long current_h1 = (h1[r + 1] - (h1[l] * p_pow1[r - l + 1]) % M1 + M1) % M1;
    long long current_h2 = (h2[r + 1] - (h2[l] * p_pow2[r - l + 1]) % M2 + M2) % M2;
    return {current_h1, current_h2};
}

int main()
{
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    std::string s;
    std::cin >> s;

    init_hash(s);
    int n = s.length();

    std::set<std::pair<long long, long long>> distinct_substring_hashes;

    for (int i = 0; i < n; ++i)
    {
        for (int j = i; j < n; ++j)
        {
            // 子串 s[i...j]
            std::pair<long long, long long> current_sub_hash = get_hash(i, j);
            distinct_substring_hashes.insert(current_sub_hash);
        }
    }

    std::cout << distinct_substring_hashes.size() << std::endl;

    return 0;
}

```

**复杂度分析**:
*   **时间复杂度**:
    *   `init_hash`: `O(N)`。
    *   生成所有子串并计算哈希值：`O(N^2)`。
    *   `set` 插入的平均时间复杂度是 `O(log S)`，其中 `S` 是 `set` 的大小，最多 `N^2` 个元素。
    *   总时间复杂度：`O(N^2 * log(N^2))`，即 `O(N^2 log N)`。
*   **空间复杂度**: `O(N)` 用于存储幂次和前缀哈希数组，`O(N^2)` 用于存储 `set` 中的哈希值对。总空间复杂度 `O(N^2)`。

### 代码实现模板

这是一个通用的双哈希模板，适用于大多数字符串哈希问题。

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <utility> // For std::pair

// 使用 Allman 风格大括号

// 定义两个不同的基数和模数，用于双哈希
const long long P1 = 131; // 常用的基数，也可以是 233, 2333, 13331 等
const long long M1 = 1e9 + 7; // 常用的大质数模数

const long long P2 = 2333; // 第二个基数
const long long M2 = 1e9 + 9; // 第二个大质数模数

// 存储 P1 的幂次，p_pow1[i] = P1^i % M1
std::vector<long long> p_pow1;
// 存储字符串的前缀哈希值1，h1[i] 表示 s[0...i-1] 的哈希值
std::vector<long long> h1;

// 存储 P2 的幂次，p_pow2[i] = P2^i % M2
std::vector<long long> p_pow2;
// 存储字符串的前缀哈希值2，h2[i] 表示 s[0...i-1] 的哈希值
std::vector<long long> h2;

/**
 * @brief 初始化哈希所需的数据结构：幂次数组和前缀哈希数组。
 * @param s 待处理的字符串。
 * @note 假设字符串只包含小写字母，字符值映射为 s[i] - 'a' + 1。
 *       如果包含其他字符，可以根据需求调整映射方式 (例如直接使用 s[i])。
 */
void init_hash(const std::string& s)
{
    int n = s.length();

    // 调整 vector 大小，n+1 是为了 h[0] 和 p_pow[0]
    p_pow1.resize(n + 1);
    h1.resize(n + 1, 0); // h1[0] = 0 (空前缀)
    p_pow2.resize(n + 1);
    h2.resize(n + 1, 0); // h2[0] = 0 (空前缀)

    // P^0 = 1
    p_pow1[0] = 1;
    p_pow2[0] = 1;

    // 预处理幂次和前缀哈希
    for (int i = 0; i < n; ++i)
    {
        // 计算 P1 的幂次
        p_pow1[i + 1] = (p_pow1[i] * P1) % M1;
        // 计算前缀哈希1
        // (s[i] - 'a' + 1) 将字符映射为 1 到 26 的整数，避免字符 'a' 映射为 0 导致乘法失效
        h1[i + 1] = (h1[i] * P1 + (s[i] - 'a' + 1)) % M1;

        // 计算 P2 的幂次
        p_pow2[i + 1] = (p_pow2[i] * P2) % M2;
        // 计算前缀哈希2
        h2[i + 1] = (h2[i] * P2 + (s[i] - 'a' + 1)) % M2;
    }
}

/**
 * @brief 获取字符串 s 中子串 s[l...r] 的双哈希值。
 * @param l 子串的起始索引 (0-indexed, inclusive)。
 * @param r 子串的结束索引 (0-indexed, inclusive)。
 * @return 包含两个 long long 哈希值的 std::pair。
 * @note 计算公式: Hash(s[l...r]) = (h[r+1] - h[l] * P^(r-l+1) % M + M) % M
 */
std::pair<long long, long long> get_substring_hash(int l, int r)
{
    // 子串长度
    int len = r - l + 1;

    // 计算第一个哈希值
    long long current_h1 = (h1[r + 1]
