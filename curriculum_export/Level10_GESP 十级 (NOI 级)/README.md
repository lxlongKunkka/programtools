# GESP 十级 (NOI 级)

对应 NOI/IOI 水平，包含高级字符串、可持久化数据结构、LCT、点分治、多项式算法、高级数论与组合数学等。

## 专题列表

1. **字符串高级**
   > 后缀数组（SA+LCP）、后缀自动机（SAM）、广义后缀自动机
2. **可持久化数据结构进阶**
   > 在掌握可持久化线段树（主席树）基础后，深入可持久化并查集的版本回退技巧，并综合应用各类可持久化结构解决带版本查询的复杂问题。
3. **高级数据结构**
   > 可合并堆（左偏树/斜堆）、线段树合并与分裂、树套树
4. **LCT 动态树**
   > Link-Cut Tree：支持链路切割、动态树上路径维护
5. **树上高级算法**
   > 点分治、树上启发式合并（DSU on Tree）、虚树、KD树
6. **DP 极限优化**
   > 四边形不等式优化 DP（决策单调性）、WQS 二分（凸优化）
7. **多项式算法**
   > FFT 快速傅里叶变换、NTT 快速数论变换及多项式基本操作
8. **高级数论**
   > 莫比乌斯反演、狄利克雷卷积、积性函数、原根
9. **高级组合数学**
   > 斯特林数、母函数（EGF/OGF）、Burnside 引理与 Polya 定理
10. **计算几何高级**
   > 半平面交、旋转卡壳、闵可夫斯基和
