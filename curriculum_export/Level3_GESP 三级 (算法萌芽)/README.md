# Level 3：GESP 三级 (算法萌芽)

## 教学目标
1. 掌握一维数组的进阶应用与多维数组。
2. 理解字符串的处理方法。
3. 掌握函数的定义与调用，理解参数传递。

## 核心知识点
- 数组算法：查找（线性查找）、简单排序（冒泡/选择）。
- 字符串：`string` 类型，字符数组，常用字符串函数。
- 函数：函数声明、定义、返回值、值传递。
- 进制转换：二进制、十进制、十六进制的转换。
