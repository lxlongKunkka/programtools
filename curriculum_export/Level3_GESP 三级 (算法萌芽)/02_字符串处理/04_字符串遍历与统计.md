## 知识点讲解

### 1. 什么是字符串？

在 C++ 中，字符串其实就是由一串字符组成的“字符数组”。每个字符都有一个位置（索引），从 0 开始。字符串的最后，系统会自动添加一个特殊的字符 `\0` (空字符)，它标志着字符串的结束。

*   **声明和初始化:**
    我们可以像声明普通数组一样声明字符数组，然后给它赋值。
    ```cpp
    char s[100]; // 声明一个最大能存储 99 个可见字符的字符串（因为要留一个位置给 '\0'）
    char s1[] = "Hello"; // 声明并初始化，编译器会自动计算长度并添加 '\0'
    char s2[10] = {'H', 'i', '!', '\0'}; // 也可以这样初始化
    ```

### 2. 字符串的输入与输出

*   **输出：`cout << s;`**
    直接使用 `cout` 可以输出整个字符串，它会一直输出直到遇到 `\0`。

*   **输入：**
    1.  **`cin >> s;`**
        这种方式会从标准输入读取字符，直到遇到**空格、回车或制表符**为止。它不能读取包含空格的字符串。
        例如：输入 `Hello World`，`s` 中只会存储 `Hello`。

    2.  **`cin.getline(s, size);`**
        这种方式可以读取一整行，包括空格，直到遇到回车符或者读取了 `size-1` 个字符。`size` 参数表示字符数组的最大容量（包括 `\0`）。
        例如：`cin.getline(s, 100);` 可以读取一行最多 99 个字符（加上 `\0` 共 100 个）。

### 3. 获取字符串的长度

我们可以使用 `strlen()` 函数来获取字符串的有效长度（不包括 `\0`）。
这个函数在 `<cstring>` 头文件中，不过我们通常使用 `#include <bits/stdc++.h>` 就能直接使用。

```cpp
char s[] = "Programming";
int len = strlen(s); // len 的值将是 11
```

### 4. 字符串的遍历

遍历字符串就是从头到尾访问字符串中的每一个字符。我们通常使用循环来完成。

*   **方法一：利用 `strlen()` 获取长度**
    先获取字符串的长度 `len`，然后从索引 0 遍历到 `len-1`。
    ```cpp
    char s[] = "Hello";
    int len = strlen(s);
    for (int i = 0; i < len; i++) {
        // 访问 s[i]
        // cout << s[i] << endl;
    }
    ```

*   **方法二：利用字符串结束标志 `\0`**
    循环条件是当前字符不等于 `\0`。
    ```cpp
    char s[] = "Hello";
    for (int i = 0; s[i] != '\0'; i++) {
        // 访问 s[i]
        // cout << s[i] << endl;
    }
    ```
    这两种方法都可以实现字符串的遍历，在实际应用中，你可以选择你更喜欢或更方便的一种。

### 5. 常用字符判断函数

在统计字符串时，我们经常需要判断一个字符是字母、数字还是空格。C++ 提供了一些非常方便的函数：
（这些函数在 `<cctype>` 头文件中，同样 `#include <bits/stdc++.h>` 即可使用）

*   `isalpha(c)`：判断 `c` 是否为字母（a-z 或 A-Z）。
*   `isdigit(c)`：判断 `c` 是否为数字（0-9）。
*   `isspace(c)`：判断 `c` 是否为空格字符（包括空格、制表符、换行符等）。
*   `isupper(c)`：判断 `c` 是否为大写字母。
*   `islower(c)`：判断 `c` 是否为小写字母。

这些函数都返回一个非零值（表示真）或零（表示假）。

### 6. 字符串统计与查找

结合遍历和字符判断函数，我们就可以实现各种统计和查找功能了！

**例：统计一个字符串中大写字母、小写字母、数字和空格的个数。**
思路：定义四个计数器，遍历字符串，对每个字符进行判断，然后相应的计数器加一。

**例：查找某个字符是否在字符串中出现，并记录第一次出现的位置。**
思路：遍历字符串，如果找到目标字符，就记录当前索引并结束循环。

## 代码示例

### 示例 1: 字符串输入、输出与长度

```cpp
#include <bits/stdc++.h> // 包含所有常用头文件

using namespace std;

int main()
{
    char s[100]; // 声明一个字符数组来存储字符串

    cout << "请输入一个不含空格的字符串: ";
    cin >> s; // 使用 cin >> 读取字符串，遇到空格、回车停止
    cout << "你输入的是: " << s << endl;
    cout << "字符串长度是: " << strlen(s) << endl; // 使用 strlen 获取长度

    // 清空输入缓冲区，为 getline 做准备
    cin.ignore(numeric_limits<streamsize>::max(), '\n'); // 忽略当前行剩余的所有字符

    cout << "请输入一个可能含空格的字符串: ";
    cin.getline(s, 100); // 使用 cin.getline 读取一整行，包括空格
    cout << "你输入的是: " << s << endl;
    cout << "字符串长度是: " << strlen(s) << endl;

    return 0;
}
```

### 示例 2: 遍历字符串并统计元音字母个数

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    char s[100];
    int count = 0; // 计数器，统计元音字母个数

    cout << "请输入一个字符串: ";
    cin.getline(s, 100); // 读取一行字符串

    cout << "你输入的字符串是: " << s << endl;

    // 遍历字符串的两种方式之一：利用 '\0' 结束标志
    for (int i = 0; s[i] != '\0'; i++)
    {
        char c = tolower(s[i]); // 将字符转换为小写，方便判断
        if (c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u')
        {
            count++; // 如果是元音字母，计数器加一
        }
    }

    cout << "字符串中元音字母的个数是: " << count << endl;

    return 0;
}
```

### 示例 3: 统计不同类型字符的个数

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    char s[100];
    int upper_count = 0; // 大写字母计数
    int lower_count = 0; // 小写字母计数
    int digit_count = 0; // 数字计数
    int space_count = 0; // 空格计数
    int other_count = 0; // 其他字符计数

    cout << "请输入一个字符串: ";
    cin.getline(s, 100);

    // 遍历字符串的另一种方式：利用 strlen()
    int len = strlen(s);
    for (int i = 0; i < len; i++)
    {
        if (isupper(s[i])) // 判断是否为大写字母
        {
            upper_count++;
        }
        else if (islower(s[i])) // 判断是否为小写字母
        {
            lower_count++;
        }
        else if (isdigit(s[i])) // 判断是否为数字
        {
            digit_count++;
        }
        else if (isspace(s[i])) // 判断是否为空格字符
        {
            space_count++;
        }
        else
        {
            other_count++; // 其他字符
        }
    }

    cout << "统计结果如下:" << endl;
    cout << "大写字母个数: " << upper_count << endl;
    cout << "小写字母个数: " << lower_count << endl;
    cout << "数字个数: " << digit_count << endl;
    cout << "空格个数: " << space_count << endl;
    cout << "其他字符个数: " << other_count << endl;

    return 0;
}
```

### 示例 4: 查找字符首次出现的位置

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    char s[100];
    char target_char;
    int pos = -1; // 用 -1 表示未找到

    cout << "请输入一个字符串: ";
    cin.getline(s, 100);

    cout << "请输入要查找的字符: ";
    cin >> target_char; // 读取一个字符

    int len = strlen(s);
    for (int i = 0; i < len; i++)
    {
        if (s[i] == target_char) // 如果当前字符与目标字符相等
        {
            pos = i; // 记录位置
            break; // 找到后立即退出循环
        }
    }

    if (pos != -1)
    {
        cout << "字符 '" << target_char << "' 首次出现在位置: " << pos << endl;
    }
    else
    {
        cout << "字符 '" << target_char << "' 在字符串中未找到。" << endl;
    }

    return 0;
}
```

## 练习题目

### 选择题/填空题

1.  声明一个字符数组 `char s[20];`，它最多可以存储多少个**可见字符**？
    A. 19
    B. 20
    C. 21
    D. 0

2.  以下 C++ 代码片段：
    ```cpp
    char s[100];
    cin >> s;
    // 假设用户输入 "Hello World"
    cout << s;
    ```
    程序的输出结果是：
    A. Hello World
    B. Hello
    C. World
    D. (空)

3.  `strlen("C++\0Programming")` 的返回值是：
    A. 3
    B. 6
    C. 13
    D. 0

4.  下列哪个函数可以判断一个字符 `c` 是否为大写字母？
    A. `isdigit(c)`
    B. `isalpha(c)`
    C. `islower(c)`
    D. `isupper(c)`

5.  以下哪种循环结构可以正确遍历字符串 `char str[100];` 中的所有有效字符（不包括结束符 `\0`）？（多选）
    A. `for (int i = 0; i < strlen(str); i++) { /* ... */ }`
    B. `for (int i = 0; str[i] != '\0'; i++) { /* ... */ }`
    C. `for (int i = 1; i <= strlen(str); i++) { /* ... */ }`
    D. `for (int i = 0; i <= strlen(str); i++) { /* ... */ }`

---

### 答案与解析

1.  **A. 19**
    *   **解析：** 字符数组的容量需要包含字符串的结束标志 `\0`。因此，如果数组大小为 20，那么最多只能存储 19 个实际可见的字符。

2.  **B. Hello**
    *   **解析：** `cin >> s;` 遇到空格、回车或制表符就会停止读取。当用户输入 "Hello World" 时，`cin` 读取到 "Hello" 后遇到空格就停止了，所以 `s` 中只存储了 "Hello"。

3.  **A. 3**
    *   **解析：** `strlen()` 函数计算的是从字符串起始位置到第一个 `\0` 之间的字符个数（不包括 `\0` 本身）。在 `"C++\0Programming"` 中，第一个 `\0` 出现在 'C', '+', '+' 之后，所以长度是 3。

4.  **D. `isupper(c)`**
    *   **解析：**
        *   `isdigit(c)` 判断是否为数字。
        *   `isalpha(c)` 判断是否为字母（不区分大小写）。
        *   `islower(c)` 判断是否为小写字母。
        *   `isupper(c)` 专门用于判断是否为大写字母。

5.  **A, B**
    *   **解析：**
        *   A 选项：`for (int i = 0; i < strlen(str); i++)` 从索引 0 开始到 `strlen(str) - 1` 结束，正确遍历了所有有效字符。
        *   B 选项：`for (int i = 0; str[i] != '\0'; i++)` 从索引 0 开始，直到遇到字符串结束标志 `\0`，也正确遍历了所有有效字符。
        *   C 选项：`for (int i = 1; i <= strlen(str); i++)` 从索引 1 开始，漏掉了第一个字符 `str[0]`。
        *   D 选项：`for (int i = 0; i <= strlen(str); i++)` 会访问到 `str[strlen(str)]`，这个位置是 `\0`，通常我们不需要处理它，而且如果循环体内有对 `str[i]` 的操作，可能会导致不必要的行为。所以此选项不推荐。
