## 知识点讲解

### 1. 快速回顾：什么是二进制？

我们平时使用的数字是十进制，有 0 到 9 十个数字。而计算机只认识两种状态：开（1）和关（0），所以它使用的是二进制。
每个 0 或 1，我们称之为“位”（bit）。
例如：
*   十进制的 5，二进制是 `101` (1 * 2^2 + 0 * 2^1 + 1 * 2^0 = 4 + 0 + 1 = 5)
*   十进制的 10，二进制是 `1010` (1 * 2^3 + 0 * 2^2 + 1 * 2^1 + 0 * 2^0 = 8 + 0 + 2 + 0 = 10)

### 2. 左移运算符 `<<`：乘法小能手

想象一下，十进制数 5 乘以 10，我们把 5 变成了 50，相当于在 5 后面加了一个 0。
在二进制中，左移操作 `<<` 就是把一个数的所有二进制位向左移动指定的位数，右边空出来的位置用 0 填充。

**规则**：`n << k` 表示将数字 `n` 的二进制表示向左移动 `k` 位。
**效果**：`n << k` 等价于 `n * (2^k)`。

**举例**：
*   `5 << 1`：
    *   5 的二进制是 `101`
    *   向左移动 1 位，右边补 0：`101` -> `1010`
    *   `1010` 转换成十进制是 10。
    *   所以 `5 << 1` 的结果是 10，这和 `5 * (2^1)` 也就是 `5 * 2 = 10` 的结果是一样的！
*   `5 << 2`：
    *   5 的二进制是 `101`
    *   向左移动 2 位，右边补 0：`101` -> `10100`
    *   `10100` 转换成十进制是 20。
    *   所以 `5 << 2` 的结果是 20，这和 `5 * (2^2)` 也就是 `5 * 4 = 20` 的结果是一样的！

**小提示**：左移操作可以非常高效地实现乘以 2 的幂次方的运算，比普通的乘法运算速度更快哦！

### 3. 右移运算符 `>>`：除法小能手

右移操作 `>>` 和左移操作相反，它将一个数的所有二进制位向右移动指定的位数。

**规则**：`n >> k` 表示将数字 `n` 的二进制表示向右移动 `k` 位。
**效果**：对于正整数 `n`，`n >> k` 等价于 `n / (2^k)` 的整数部分。

**举例**：
*   `10 >> 1`：
    *   10 的二进制是 `1010`
    *   向右移动 1 位，左边补 0 (对于正数)：`1010` -> `0101` (最右边的 0 被移出去了)
    *   `0101` 转换成十进制是 5。
    *   所以 `10 >> 1` 的结果是 5，这和 `10 / (2^1)` 也就是 `10 / 2 = 5` 的结果是一样的！
*   `20 >> 2`：
    *   20 的二进制是 `10100`
    *   向右移动 2 位，左边补 0 (对于正数)：`10100` -> `00101` (最右边的 00 被移出去了)
    *   `00101` 转换成十进制是 5。
    *   所以 `20 >> 2` 的结果是 5，这和 `20 / (2^2)` 也就是 `20 / 4 = 5` 的整数部分是一样的！

**重要提示**：
1.  右移操作对于正整数来说，就相当于整数除法。
2.  对于负数，右移操作的行为在不同的计算机系统上可能会略有不同，所以我们在信奥比赛中，通常只对正整数使用右移操作。今天的课程，我们也只关注正整数的位移。

## 代码示例

让我们用 C++ 代码来实际感受一下位移操作的威力吧！

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    int n = 5;
    int m = 10;
    int x = 20;

    // 左移操作示例
    cout << "--- 左移操作 (<<) 示例 ---" << endl;
    cout << n << " << 1 = " << (n << 1) << endl; // 5 * 2^1 = 10
    cout << n << " << 2 = " << (n << 2) << endl; // 5 * 2^2 = 20
    cout << n << " << 3 = " << (n << 3) << endl; // 5 * 2^3 = 40

    cout << endl;

    // 右移操作示例
    cout << "--- 右移操作 (>>) 示例 ---" << endl;
    cout << m << " >> 1 = " << (m >> 1) << endl; // 10 / 2^1 = 5
    cout << x << " >> 1 = " << (x >> 1) << endl; // 20 / 2^1 = 10
    cout << x << " >> 2 = " << (x >> 2) << endl; // 20 / 2^2 = 5
    cout << x << " >> 3 = " << (x >> 3) << endl; // 20 / 2^3 = 20 / 8 = 2 (整数部分)

    // 位移操作的实际应用：快速计算
    int a = 7;
    int b = 4; // 相当于 2^2
    int c = 8; // 相当于 2^3

    cout << endl;
    cout << "--- 实际应用示例 ---" << endl;
    cout << a << " 乘以 4 (使用 <<): " << (a << 2) << endl; // 7 * 4 = 28
    cout << a << " 除以 2 (使用 >>): " << (a >> 1) << endl; // 7 / 2 = 3 (整数部分)

    return 0;
}
```

**代码运行结果**：
```
--- 左移操作 (<<) 示例 ---
5 << 1 = 10
5 << 2 = 20
5 << 3 = 40

--- 右移操作 (>>) 示例 ---
10 >> 1 = 5
20 >> 1 = 10
20 >> 2 = 5
20 >> 3 = 2

--- 实际应用示例 ---
7 乘以 4 (使用 <<): 28
7 除以 2 (使用 >>): 3
```

从上面的代码和运行结果中，我们可以清晰地看到位移操作是如何实现快速乘除的。是不是很神奇呢？

    A. `p << 4`
    B. `p << 6`
    C. `p >> 6`
    D. `p * 64` (虽然正确，但要求用位移操作)

    A. 6
    B. 12
    C. 25
    D. 50

### 编程题

1.  **基础练习：位移计算器**
    编写一个程序，读取一个整数 `n` 和一个整数 `k`。然后分别输出 `n` 左移 `k` 位的结果和 `n` 右移 `k` 位的结果。
    **输入样例**：
    `10 2`
    **输出样例**：
    `40`
    `2`

2.  **进阶练习：偶数减半，奇数加倍**
    编写一个程序，读取一个整数 `a`。如果 `a` 是偶数，则将其值右移 1 位（相当于除以 2）；如果 `a` 是奇数，则将其值左移 1 位（相当于乘以 2）。最后输出处理后的 `a` 值。
    **输入样例 1**：
    `7`
    **输出样例 1**：
    `14`
    **输入样例 2**：
    `10`
    **输出样例 2**：
    `5`

---

### 答案与解析

**选择题/填空题答案**

1.  **C. 12**
    *   解析：`3` 的二进制是 `11`。`3 << 2` 表示 `11` 向左移动 2 位，变成 `1100`。`1100` 转换为十进制是 `8 + 4 = 12`。或者 `3 * (2^2) = 3 * 4 = 12`。

2.  **B. 7**
    *   解析：`15` 的二进制是 `1111`。`15 >> 1` 表示 `1111` 向右移动 1 位，变成 `0111`。`0111` 转换为十进制是 `4 + 2 + 1 = 7`。或者 `15 / (2^1) = 15 / 2 = 7` (整数部分)。

3.  **D. 88**
    *   解析：`11 << 3` 等于 `11 * (2^3) = 11 * 8 = 88`。

4.  **B. `p << 6`**
    *   解析：将整数 `p` 乘以 `64`，因为 `64 = 2^6`，所以使用左移 6 位的操作 `p << 6`。

5.  **B. 12**
    *   解析：先计算 `25 >> 2`。`25` 的二进制是 `11001`。向右移动 2 位变成 `00110`，即十进制的 `6`。
    *   然后计算 `6 << 1`。`6` 的二进制是 `110`。向左移动 1 位变成 `1100`，即十进制的 `12`。
    *   所以最终结果是 `12`。
