## 趣味引入

同学们，你们有没有玩过“猜数字”游戏？比如，我心里想了一个1到100之间的数字，让你猜。你可能会从1开始，一个一个地猜：1、2、3、4……直到猜中为止。或者你可能会从中间开始猜，比如50，如果我告诉你“大了”，你再猜25，如果我告诉你“小了”，你再猜37……

我们今天学习的“枚举法”，就像是第一种“一个一个地猜”的方法，只不过它更“聪明”一些，它会把所有可能的情况都“猜”一遍，然后找出那些符合我们要求的“正确答案”！

比如，小明有一个神奇的盒子，里面装着一个三位数。他告诉你这个三位数有一个特点：它的个位数字、十位数字和百位数字加起来正好等于10。他想让你帮他找到所有符合这个条件的三位数。

你会怎么找呢？
是不是可以从100开始，一个一个地检查到999？
对于每一个数，比如123，我们计算它的各位数字之和：1+2+3=6。不等于10，所以123不是答案。
再比如145，1+4+5=10。Bingo！145是一个答案！
像这样，把所有可能的三位数都检查一遍，这就是我们今天要学习的“枚举法”！是不是很简单呢？

## 知识点讲解

### 1. 什么是枚举法？
**枚举法**（也叫做**暴力破解法**或**穷举法**）是一种最直接、最基础的算法思想。它的核心思想是：将问题的所有可能情况都一一列举出来，然后对每种情况进行判断，找出那些满足特定条件的解。

想象一下，你有一个装满各种形状积木的箱子，要找出所有红色的圆形积木。枚举法就是把箱子里所有的积木都拿出来看一遍，如果是红色的圆形，就把它挑出来。

### 2. 枚举法的“三要素”
要成功使用枚举法解决问题，我们需要明确三个关键要素：

1.  **枚举对象**：我们要“一个一个地尝试”的是什么？
    *   在猜数字游戏中，枚举对象是1到100之间的每一个整数。
    *   在找三位数的问题中，枚举对象是100到999之间的每一个三位数。
2.  **枚举范围**：我们要从哪里开始尝试？到哪里结束？
    *   这个范围必须包含所有可能的解，并且不能遗漏。
    *   例如，找三位数，范围就是从100到999。
3.  **判断条件**：每尝试一个对象后，我们如何判断它是不是我们想要的答案？
    *   在猜数字游戏中，判断条件可能是“是否等于我心中的数字”。
    *   在找三位数的问题中，判断条件是“各位数字之和是否等于10”。

### 3. 枚举法的优点和缺点
*   **优点**：
    *   **简单直观**：思想非常容易理解，实现起来也相对简单。
    *   **通用性强**：在没有其他更优算法时，枚举法总能找到问题的解（如果存在的话）。
    *   **保证正确**：只要枚举范围正确且判断条件无误，枚举法找到的解一定是正确的。
*   **缺点**：
    *   **效率低下**：当枚举范围非常大时，需要尝试的情况会非常多，程序运行时间会很长，甚至无法在规定时间内给出结果。这就像大海捞针，虽然能捞到，但可能要花很久很久。

### 4. 什么时候用枚举法？
*   **问题规模小**：枚举范围不大，尝试所有情况不会花费太多时间。
*   **没有明显规律**：问题没有特别明显的数学公式、贪心策略或其他高级算法可以快速解决。
*   **需要找出所有解**：题目要求列出所有符合条件的答案，而不是只求一个。

### 5. C++ 实现思路
在 C++ 中，我们通常使用 `for` 循环来实现枚举。

```cpp
// 伪代码示例
for (枚举对象 从 起始值 到 结束值) {
    if (判断条件 成立) {
        // 找到一个解，进行相应的处理（比如打印出来）
    }
}
```

## 代码示例

### 示例1：找出所有两位数，其个位数字和十位数字之和等于5。

**分析：**
*   **枚举对象**：两位数 `x`。
*   **枚举范围**：从10到99。
*   **判断条件**：`x` 的十位数字加上个位数字等于5。
    *   如何获取一个两位数 `x` 的十位数字？ `x / 10`
    *   如何获取一个两位数 `x` 的个位数字？ `x % 10`

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    cout << "所有个位数字和十位数字之和为5的两位数有：" << endl;
    // 枚举范围：从10到99的所有两位数
    for (int i = 10; i <= 99; i++) {
        // 获取十位数字
        int a = i / 10;
        // 获取个位数字
        int b = i % 10;
        
        // 判断条件：十位数字和个位数字之和是否为5
        if (a + b == 5) {
            cout << i << " "; // 如果满足条件，就打印出来
        }
    }
    cout << endl; // 打印完所有数字后换行
    return 0;
}
```

### 示例2：水仙花数
**问题描述**：水仙花数是指一个三位数，它的每个位上的数字的立方和等于它本身。例如：153 = 1^3 + 5^3 + 3^3。请找出所有水仙花数。

**分析：**
*   **枚举对象**：三位数 `x`。
*   **枚举范围**：从100到999。
*   **判断条件**：`x` 的百位数字的立方 + 十位数字的立方 + 个位数字的立方 等于 `x` 本身。
    *   如何获取一个三位数 `x` 的百位数字？ `x / 100`
    *   如何获取一个三位数 `x` 的十位数字？ `(x / 10) % 10`
    *   如何获取一个三位数 `x` 的个位数字？ `x % 10`

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    cout << "所有水仙花数有：" << endl;
    // 枚举范围：从100到999的所有三位数
    for (int i = 100; i <= 999; i++) {
        // 获取百位数字
        int a = i / 100;
        // 获取十位数字
        int b = (i / 10) % 10;
        // 获取个位数字
        int c = i % 10;
        
        // 判断条件：各位数字的立方和是否等于原数
        if (a * a * a + b * b * b + c * c * c == i) {
            cout << i << " "; // 如果满足条件，就打印出来
        }
    }
    cout << endl;
    return 0;
}
```
**小提示**：这里的 `a * a * a` 就是 `a` 的立方。

### 编程题参考答案

1.  **基础题：找出所有两位数，其十位数字大于个位数字。**

    ```cpp
    #include <bits/stdc++.h>
    using namespace std;

    int main() {
        cout << "所有十位数字大于个位数字的两位数有：" << endl;
        for (int i = 10; i <= 99; i++) {
            int a = i / 10; // 十位数字
            int b = i % 10; // 个位数字
            if (a > b) {
                cout << i << endl; // 满足条件则打印
            }
        }
        return 0;
    }
    ```

2.  **进阶题：鸡兔同笼问题。**

    ```cpp
    #include <bits/stdc++.h>
    using namespace std;

    int main() {
        int h, f; // h为头数，f为脚数
        cout << "请输入头数h和脚数f：";
        cin >> h >> f;

        bool found = false; // 标记是否找到解

        // 枚举鸡的数量，从0只到h只
        for (int x = 0; x <= h; x++) {
            // 根据头数和鸡的数量，计算兔的数量
            int y = h - x; 
            
            // 兔子的数量不能是负数
            if (y < 0) { 
                continue; // 跳过不合理的y值
            }

            // 判断当前鸡兔数量是否满足脚数条件
            if (2 * x + 4 * y == f) {
                cout << "鸡: " << x << ", 兔: " << y << endl;
                found = true;
                // 如果只要求输出一组解，找到后就可以退出循环
                break; 
            }
        }

        if (!found) {
            cout << "无解" << endl;
        }

        return 0;
    }
    ```
