# GESP 八级 (图论算法·数学)

## 教学目标
1. 掌握图论基础与常用算法。
2. 深入理解动态规划与搜索优化。
3. 具备综合运用算法解决复杂问题的能力。

## 核心知识点
- 图：图的存储（邻接矩阵/邻接表），图的遍历。
- 最短路径：Dijkstra 算法, Floyd 算法。
- 进阶DP：背包问题（0/1背包、完全背包），最长公共子序列 (LCS)。
- 综合能力：模拟大作业，复杂逻辑实现。

## 专题列表

1. **组合数学**
2. **数论进阶**
   > 欧拉函数、逆元（费马小定理/线性逆元）、容斥原理
3. **并查集**
   > 路径压缩与按秩合并的并查集，及带权、扩展并查集
4. **优先队列与堆**
   > 堆的原理、STL priority_queue 的使用与贪心中的应用
5. **Floyd 全源最短路**
   > Floyd-Warshall 全源最短路算法与传递闭包
6. **最短路径**
   > 掌握单源最短路的三种算法：Dijkstra（贪心+堆优化）处理非负权图，Bellman-Ford处理负权边，SPFA作为队列优化版本并用于判负环，理解各算法适用场景。
7. **最小生成树**
8. **拓扑排序**
   > DAG 有向无环图的拓扑排序与依赖关系建模
9. **树的直径与重心**
   > 两次 DFS 求树的直径、树的重心及其应用
10. **树上差分**
   > 树上路径差分（点差分与边差分）
11. **树状数组**
   > 利用 lowbit 位运算实现高效的单点修改与前缀查询（O(log n)），掌握差分树状数组支持区间修改，作为线段树的轻量替代方案广泛应用于计数、逆序对、动态前缀和等场景。
12. **线段树**
   > ## 📚 模块简介
线段树是一种常用的区间数据结构，支持快速查询和更新区间信息，能在竞赛里高效处理区间求和、最值等操作。

## 🎯 学习目标
- 掌握线段树的基本结构与特点  
- 能够使用 C++ 实现线段树的建树、查询和修改操作  
- 运用线段树解决区间求和、区间最值等竞赛题  

## 🚀 学习建议
- 先从简单的区间求和场景入手，手动模拟线段树过程  
- 在 C++ 中练习写出递归建树与区间操作，加深理解  

## 🌟 亮点
- **动态区间维护**：实时更新区间数据，适应多种查询需求  
- **高效复杂度**：单次查询修改只需 O(log n)，助你在竞赛中抢分！  

加油！你一定能掌握线段树，变成数据结构小达人！
