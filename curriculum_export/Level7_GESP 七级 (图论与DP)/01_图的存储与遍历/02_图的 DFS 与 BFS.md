## 知识点讲解

我们已经知道，图是由节点（顶点）和边组成的。在计算机中，我们通常用**邻接表**来表示图，因为它在处理稀疏图（边相对较少）时效率更高，也更方便进行遍历。

### 1. 图的邻接表表示回顾

邻接表本质上是一个数组，数组的每个元素都是一个链表（或C++中的`vector`）。`adj[i]` 存储了与节点 `i` 相连的所有节点。

```cpp
// 假设有 n 个节点，m 条边
// vector<vector<int>> adj; // 声明一个邻接表
// adj.resize(n + 1);       // 初始化大小，如果节点从1开始编号

// 对于每条边 u -> v
// adj[u].push_back(v);
// 如果是无向图，还需要：
// adj[v].push_back(u);
```

### 2. 深度优先搜索（DFS）

**核心思想**：DFS就像一个不撞南墙不回头的探险家。它从某个节点出发，尽可能深地访问它的邻居节点。当一条路径走到尽头（没有未访问的邻居节点）时，它会回溯到上一个节点，尝试访问该节点的其他未访问邻居。

**搜索策略**：
1.  选择一个起始节点。
2.  访问该节点。
3.  标记该节点为已访问。
4.  递归地对其**未访问的邻居节点**重复上述步骤。

**实现方式**：通常使用**递归**来实现，也可以使用**栈**来模拟递归过程。

**特点**：
*   **回溯**：当一条路径走到底时，会返回到上一个节点继续探索。
*   **栈行为**：访问路径类似栈的“后进先出”特性。
*   **应用场景**：查找连通分量、判断图是否有环、拓扑排序、迷宫寻路（找到一条路径即可）。

**DFS的伪代码**：

```
DFS(u):
  标记 u 为已访问
  打印 u
  对于 u 的每一个邻居 v:
    如果 v 未被访问:
      DFS(v)
```

### 3. 广度优先搜索（BFS）

**核心思想**：BFS就像水波纹一样，从一个中心点开始，一层一层地向外扩散。它会先访问所有距离起始节点为1的节点，然后是所有距离为2的节点，依此类推。

**搜索策略**：
1.  选择一个起始节点。
2.  访问该节点。
3.  标记该节点为已访问，并将其加入一个**队列**。
4.  当队列不为空时：
    *   从队列中取出一个节点 `u`。
    *   访问 `u` 的所有**未访问的邻居节点 `v`**。
    *   将 `v` 标记为已访问，并加入队列。

**实现方式**：**队列**。

**特点**：
*   **层序遍历**：按照距离起始节点的远近，逐层访问节点。
*   **最短路径**：在无权图中，BFS可以找到从起始节点到其他所有节点的最短路径。
*   **队列行为**：访问顺序类似队列的“先进先出”特性。
*   **应用场景**：查找无权图的最短路径、社交网络中查找最近的朋友、网页爬虫、连通块计数。

**BFS的伪代码**：

```
BFS(s):
  创建一个队列 Q
  将 s 加入 Q
  标记 s 为已访问
  当 Q 不为空时:
    从 Q 中取出节点 u
    打印 u
    对于 u 的每一个邻居 v:
      如果 v 未被访问:
        标记 v 为已访问
        将 v 加入 Q
```

### 4. DFS与BFS的比较

| 特性       | 深度优先搜索 (DFS)                               | 广度优先搜索 (BFS)                               |
| :--------- | :----------------------------------------------- | :----------------------------------------------- |
| **思想**   | 一条路走到黑，碰壁回头                           | 层层推进，先近后远                               |
| **实现**   | 递归（或栈）                                     | 队列                                             |
| **遍历顺序** | 深入探索，回溯                                   | 逐层探索，由近及远                               |
| **应用**   | 连通性、环检测、拓扑排序、迷宫寻路（任意路径） | 无权图最短路径、连通块计数、社交网络关系、迷宫寻路（最短路径） |
| **时间复杂度** | O(V + E) (V是节点数，E是边数)                  | O(V + E)                                         |
| **空间复杂度** | O(V) (递归栈深度最坏情况)                      | O(V) (队列存储最坏情况)                          |

**重要提示**：无论是DFS还是BFS，我们都需要一个 `visited` 数组（或 `bool` 数组）来记录哪些节点已经被访问过，以避免死循环（在有环图中）和重复访问。

## 代码示例

我们用一个简单的无向图来演示DFS和BFS的实现。
图结构：
1 -- 2
|    |
3 -- 4
|
5

节点：1, 2, 3, 4, 5
边：(1,2), (1,3), (2,4), (3,4), (3,5)

```cpp
#include <bits/stdc++.h> // 包含大部分标准库
using namespace std;

// 邻接表表示图
vector<vector<int>> adj;
// 访问标记数组，用来记录节点是否被访问过
vector<bool> vis;
int n_nodes; // 节点数量

// 深度优先搜索 (DFS) 函数
void dfs(int u) {
    vis[u] = true; // 标记当前节点u为已访问
    cout << u << " "; // 访问节点u

    // 遍历所有与u相邻的节点v
    for (int v : adj[u]) {
        if (!vis[v]) { // 如果邻居v未被访问
            dfs(v);    // 递归地对v进行DFS
        }
    }
}

// 广度优先搜索 (BFS) 函数
void bfs(int start_node) {
    queue<int> q; // 创建一个队列
    
    q.push(start_node); // 将起始节点加入队列
    vis[start_node] = true; // 标记起始节点为已访问

    // 当队列不为空时，循环执行
    while (!q.empty()) {
        int u = q.front(); // 取出队列头部节点u
        q.pop();           // 将u从队列中移除
        cout << u << " ";  // 访问节点u

        // 遍历所有与u相邻的节点v
        for (int v : adj[u]) {
            if (!vis[v]) { // 如果邻居v未被访问
                vis[v] = true; // 标记v为已访问
                q.push(v);     // 将v加入队列
            }
        }
    }
}

int main() {
    n_nodes = 5; // 假设有5个节点，编号从1到5
    adj.resize(n_nodes + 1); // 邻接表大小设置为 n_nodes + 1，方便使用1-based索引
    vis.resize(n_nodes + 1, false); // 访问标记数组初始化为false

    // 构建图的边
    adj[1].push_back(2);
    adj[2].push_back(1);

    adj[1].push_back(3);
    adj[3].push_back(1);

    adj[2].push_back(4);
    adj[4].push_back(2);

    adj[3].push_back(4);
    adj[4].push_back(3);
    
    adj[3].push_back(5);
    adj[5].push_back(3);

    cout << "DFS Traversal (starting from node 1): ";
    dfs(1); // 从节点1开始进行DFS
    cout << endl;

    // 重置访问标记数组，以便进行BFS
    fill(vis.begin(), vis.end(), false); 

    cout << "BFS Traversal (starting from node 1): ";
    bfs(1); // 从节点1开始进行BFS
    cout << endl;

    return 0;
}
```

**代码运行结果示例**：
```
DFS Traversal (starting from node 1): 1 2 4 3 5 
BFS Traversal (starting from node 1): 1 2 3 4 5 
```
**注意**：DFS的遍历结果可能因邻接表中存储顺序不同而有所差异，但只要是合法的深度优先遍历，都是正确的。BFS的遍历结果在无权图且起始节点固定时通常是唯一的。

### 练习题目答案与解析

**选择题/填空题答案**：

1.  **B. 广度优先搜索 (BFS)**
    *   **解析**：BFS 按照“层”的顺序遍历，因此在无权图中，它总是先找到距离最短的路径。Dijkstra 和 A* 是针对带权图的最短路径算法。

2.  **B. 栈**
    *   **解析**：DFS 的递归调用本质上就是利用了系统栈。如果手动实现非递归DFS，则需要显式使用一个栈。

3.  **A. 队列**
    *   **解析**：BFS 的核心思想是层序遍历，这与队列的“先进先出”特性完美契合。

4.  **O(V + E)**
    *   **解析**：V 是节点数，E 是边数。无论 DFS 还是 BFS，每个节点和每条边都只会被访问一次（因为有 `visited` 数组），所以时间复杂度与图的大小成线性关系。

5.  **避免重复访问和死循环**
    *   **解析**：在有环图中，如果没有 `visited` 数组，遍历可能会无限循环。同时，`visited` 数组也能确保每个节点只被处理一次，保证算法效率。

---

**编程题解答提示**：

**题目一：连通块计数**
*   **思路**：遍历所有节点。如果遇到一个未被访问的节点，说明我们发现了一个新的连通分量。从这个节点开始执行一次DFS（或BFS），标记所有与它连通的节点为已访问。每执行一次这样的DFS/BFS，连通分量计数器就加一。
*   **关键**：一个全局的 `visited` 数组，一个循环遍历所有节点的 `for` 循环，以及在循环内调用DFS/BFS。

**题目二：查找路径**
*   **思路**：从起始节点 `s` 开始进行一次DFS（或BFS）。在遍历过程中，检查是否访问到了目标节点 `t`。
*   **关键**：在DFS或BFS函数中，一旦当前访问的节点 `u` 等于目标节点 `t`，就可以立即返回 `true`（表示找到路径）。如果整个遍历结束都没有找到 `t`，则返回 `false`。
