## 趣味引入

想象一下，你是一名邮票收藏家，每张邮票有一个独一无二的编号，编号可能非常大（如上亿）。  
*问题来了*：当你想快速统计某个区间内（例如编号 1000000 到 2000000）的邮票数量时，直接用数组显然不现实。  
已知：  
- 邮票编号不连续，且范围极大  
- 我们需要频繁插入新邮票并查询某一区间的数量  

今天，我们就来学习如何用“**离散化技术 + 树状数组**”来高效地维护和查询这类大范围编号的问题！

## 知识点讲解

1. 已知→未知  
   - 已知：树状数组能在 O(log n) 内完成更新与前缀和查询  
   - 未知：如何把“稀疏且巨大的编号”变为“紧凑且连续的下标”  

2. 离散化原理  
   - 收集所有要出现的编号，去重后排序  
   - 利用 map 或 lower_bound 将原编号映射到 1…m 的范围  
   - 示例：原序列 {50, 10, 20, 50} → 唯一值 {10,20,50} → 映射：10→1,20→2,50→3  

3. 树状数组回顾  
   - lowbit(x) = x & (-x)  
   - update(i, v)：对下标 i 增加 v  
   - query(i)：求前 i 项和  

4. 离散化+树状数组结合流程  
   a. 读入所有操作，收集要离散化的值到 vector `b`  
   b. 对 `b` 排序并去重  
   c. 对原操作中的每个值，用 `lower_bound` 映射到新下标  
   d. 用树状数组在映射后下标上执行 update/query

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int n;
vector<int> b;            // 用于存放离散化基准
vector<int> bit;          // 树状数组

// 计算低位
int lowbit(int x)
{
    return x & -x;
}

// 在树状数组上做点更新：给下标 i 加上 v
void update(int i, int v)
{
    int m = bit.size() - 1;
    while (i <= m)
    {
        bit[i] += v;
        i += lowbit(i);
    }
}

// 查询前缀和：求 1..i 的和
int query(int i)
{
    int s = 0;
    while (i > 0)
    {
        s += bit[i];
        i -= lowbit(i);
    }
    return s;
}

int main()
{
    // 假设有 n 次操作：1 x （插入 x）、2 l r（查询 [l,r] 区间个数）
    cin >> n;
    vector<pair<int, pair<int, int>>> ops;
    for (int i = 0; i < n; i++)
    {
        int t, x, y;
        cin >> t;
        if (t == 1)
        {
            cin >> x;
            ops.push_back({1, {x, 0}});
            b.push_back(x);
        }
        else
        {
            cin >> x >> y;
            ops.push_back({2, {x, y}});
            b.push_back(x);
            b.push_back(y);
        }
    }

    // 离散化基准
    sort(b.begin(), b.end());
    b.erase(unique(b.begin(), b.end()), b.end());
    int m = b.size();

    bit.assign(m + 1, 0);

    // 处理操作
    for (auto &op : ops)
    {
        int t = op.first;
        int x = op.second.first;
        int y = op.second.second;

        // 离散化映射
        int idx = lower_bound(b.begin(), b.end(), x) - b.begin() + 1;

        if (t == 1)
        {
            // 插入一次 x
            update(idx, 1);
        }
        else
        {
            int idx2 = lower_bound(b.begin(), b.end(), y) - b.begin() + 1;
            // query(idx2) - query(idx-1)
            int ans = query(idx2) - query(idx - 1);
            cout << ans << "\n";
        }
    }

    return 0;
}
```

## 练习题目

### 选择题 / 填空题

1. 离散化的第 1 步是：  
   A. 对原始数据排序并去重  
   B. 建立长度为 1e9+7 的数组  
   C. 直接在树状数组上使用原值  

2. lowbit(12) 的值是 ______。

3. 给定原序列 {100, 50, 100, 200}，离散化后 200 映射到的下标是 ______。

4. 在树状数组中，执行 `update(i, v)` 的关键语句是 ______。

5. 已知离散化后下标为 3 的元素前缀和为 5，想查询第 3 个数到第 6 个数的区间和，应计算 ______。

### 答案与解析

1. A  
   解析：离散化先要对原始数据排序并去重，才能映射到连续下标。

2. 4  
   解析：12 的二进制为 1100，lowbit = 0100 = 4。

3. 3  
   解析：唯一值集合 {50,100,200}，200 在第三位。

4. `i += lowbit(i);`  
   解析：更新时需不断跳到下一影响位。

5. `query(6) - query(2)`  
   解析：区间 [3,6] 的和等于前 6 项和减去前 2 项和。
