## 趣味引入
真实场景：  
想象你在运营一个在线游戏，用数组记录每个玩家的得分。每天玩家的得分会随时更新，你需要高效地查询任意前 k 名玩家的总得分。  
- 已知方案：直接维护前缀和数组，更新一次需要 O(n) 时间，查询 O(1)  
- 未知目标：能否将“更新”和“查询”都优化到 O(log n)？

引出问题：如果能让每次更新只影响少量位置，就能把更新从 O(n) 降到 O(log n，而这正是树状数组要解决的核心问题。

===NEXT===

## 知识点讲解
1. 复习前缀和数组  
   - `s[i] = a[1] + … + a[i]`，查询前 i 项 O(1)，更新某个 a[k] 需要重建后续所有 s[j]，O(n)

2. lowbit 概念  
   - 定义：lowbit(x) = x & (-x)  
   - 作用：提取二进制中最低位的 1，决定每个节点管理的区间长度  
   - 示例：x = 12 (1100₂)，lowbit(12) = 4 (0100₂)

3. 树状数组结构  
   - 用一维数组 f[1..n] 存储“跳跃区间和”  
   - f[i] 存储区间 (i - lowbit(i) + 1) 到 i 的和  
   - 更新时：从 i 出发，每次加上 lowbit(i) 的长度，最坏 O(log n)  
   - 查询时：从 i 出发，每次减去 lowbit(i) 的长度，累计到 0，最坏 O(log n)

4. 单点更新与前缀和查询  
   - update(x, v)：将 a[x] 增加 v  
     for (i = x; i ≤ n; i += lowbit(i)) f[i] += v  
   - query(x)：计算 a[1] + … + a[x]  
     sum = 0  
     for (i = x; i > 0; i -= lowbit(i)) sum += f[i]  
     return sum

5. 复杂度分析  
   - 单次更新 O(log n)  
   - 单次前缀和查询 O(log n)  
   - 空间 O(n)

## 代码示例
```cpp
#include <bits/stdc++.h>
using namespace std;

class Fenwick
{
public:
    int n;
    vector<int> f;

    Fenwick(int n_)
    {
        n = n_;
        f.assign(n + 1, 0);
    }

    // 计算 lowbit
    int lowbit(int x)
    {
        return x & -x;
    }

    // 单点更新：将位置 x 增加 v
    void update(int x, int v)
    {
        for (int i = x; i <= n; i += lowbit(i))
        {
            f[i] += v;
        }
    }

    // 前缀和查询：求 1..x 的和
    int query(int x)
    {
        int s = 0;
        for (int i = x; i > 0; i -= lowbit(i))
        {
            s += f[i];
        }
        return s;
    }
};

int main()
{
    int n, q;
    cin >> n >> q;                 // 输入数组大小和操作数
    Fenwick t(n);

    while (q--)
    {
        int op, x, v;
        cin >> op >> x >> v;
        if (op == 1)
        {
            // 单点更新
            t.update(x, v);
        }
        else if (op == 2)
        {
            // 前缀和查询
            cout << t.query(x) << "\n";
        }
    }
    return 0;
}
```

## 练习题目

### 题目
1. lowbit(14) = (   )。  
2. 树状数组单次查询复杂度为 O(   )。  
3. 在 `update(x, v)` 中，循环体内的 `i += lowbit(i)` 作用是：  
   A. 找到下一个父节点  
   B. 跳过已经更新的区间  
   C. 结束循环  
4. 给定初始全 0，执行 `update(2,5)`、`update(5,3)` 后，`query(5)` 的结果是 (   )。  
5. （填空）树状数组中，f[i] 存储的是区间 (   ) 到 i 的和。

### 答案与解析
1. 答案：2  
   解析：14₁₀ = 1110₂，lowbit(14) = 0010₂ = 2  
2. 答案：log n  
   解析：查询过程每次 i 减少一个 lowbit，循环次数 ≤ 二进制位数  
3. 答案：A  
   解析：`i += lowbit(i)` 是沿着树状数组向上跳到父节点  
4. 答案：8  
   解析：前缀和 = 5（位置2）+ 3（位置5）= 8  
5. 答案：[i - lowbit(i) + 1]  
   解析：f[i] 存储从 i−lowbit(i)+1 到 i 的元素和
