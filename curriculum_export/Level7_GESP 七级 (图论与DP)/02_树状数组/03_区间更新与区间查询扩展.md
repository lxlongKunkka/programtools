## 趣味引入

真实场景：  
假设一个在线游戏平台维护了 N 位玩家的金币数。  
现在需要支持两种操作：  
1. 在第 l 到第 r 位玩家的账户中统一增加金币 d  
2. 查询第 l 到第 r 位玩家当前的总金币数  

已知我们可以用前缀和数组实现区间查询，但每次区间更新都需要 O(N) 时间，太慢了。  
今天，我们就来学习如何利用树状数组，将这两种操作都优化到 O(log N) 时间！  

===NEXT===

## 知识点讲解

1. 单点更新 + 前缀和查询回顾  
   - 经典树状数组支持 `update(i, d)`：将第 i 个元素加 d  
   - 支持 `sum(i)`：计算前缀和 `a[1] + … + a[i]`  
   - 时间复杂度均为 O(log N)  

2. 区间更新 + 单点查询  
   - 如果要给区间 [l, r] 增加 d，可以在树状数组 b 上执行  
     - `b.update(l, d)`  
     - `b.update(r+1, -d)`  
   - 此时第 i 个元素的实际值就是 `b.sum(i)`  

3. 区间更新 + 区间查询（双树状数组法）  
   - 要实现对区间 [l, r] 增加 d，且之后能够查询任意区间 [l, r] 的和  
   - 思路：维护两个树状数组 b1 和 b2  
   - 更新操作 `update_range(l, r, d)`：  
     1. b1.update(l,  d)  
     2. b1.update(r+1, -d)  
     3. b2.update(l,  d*(l-1))  
     4. b2.update(r+1, -d*r)  
   - 查询前缀和 `prefix_sum(x)`：  
     prefix_sum(x) = x * b1.sum(x) - b2.sum(x)  
   - 区间和 `query_range(l, r)`：  
     query_range(l, r) = prefix_sum(r) - prefix_sum(l-1)  

4. 复杂度分析  
   - 每次更新或查询都使用常数次的树状数组操作，时间复杂度 O(log N)  
   - 空间复杂度 O(N)

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

class BIT
{
public:
    int n;
    vector<long long> c;

    BIT(int n)
    {
        this->n = n;
        c.assign(n + 1, 0);
    }

    // 单点更新：第 x 位加 v
    void update(int x, long long v)
    {
        for (int i = x; i <= n; i += i & -i)
        {
            c[i] += v;
        }
    }

    // 前缀和查询：[1..x] 的和
    long long sum(int x)
    {
        long long s = 0;
        for (int i = x; i > 0; i -= i & -i)
        {
            s += c[i];
        }
        return s;
    }
};

int n, m;       // n: 数组长度, m: 操作次数
BIT *b1, *b2;

// 区间更新：给 [l..r] 每位加 d
void update_range(int l, int r, long long d)
{
    b1->update(l, d);
    b1->update(r + 1, -d);
    b2->update(l, d * (l - 1));
    b2->update(r + 1, -d * r);
}

// 前缀和查询：计算 a[1] + … + a[x]
long long prefix_sum(int x)
{
    return x * b1->sum(x) - b2->sum(x);
}

// 区间查询：[l..r] 的和
long long query_range(int l, int r)
{
    return prefix_sum(r) - prefix_sum(l - 1);
}

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> n >> m;
    b1 = new BIT(n);
    b2 = new BIT(n);

    while (m--)
    {
        int op, l, r;
        long long d;
        cin >> op;
        if (op == 1)
        {
            // 操作1：区间更新
            cin >> l >> r >> d;
            update_range(l, r, d);
        }
        else
        {
            // 操作2：区间查询
            cin >> l >> r;
            cout << query_range(l, r) << "\n";
        }
    }

    return 0;
}
```

## 练习题目

以下为 4 道练习题（选择题或填空题），请先完成题目，再对照下方答案与解析。

### 题目

1. 在双树状数组方案中，b1 负责记录的是什么？  
   A. 前缀和的增量  
   B. 区间更新时的基础增量  
   C. 区间查询时的修正项  
   D. 原始数组的值  

2. 更新区间 [l, r] 增加 d 时，对 b2 执行的操作是（填空）：  
   b2.update(l, _____);  
   b2.update(r+1, _____);  

3. 前缀和公式 `prefix_sum(x)` 等于 （填空）  
   prefix_sum(x) = x * b1.sum(x) - b2.sum(x)

4. 初始数组全为 0，执行以下操作后，query_range(1, 4) 的结果是多少？  
   1) update_range(2, 5, 3)  
   2) update_range(3, 4, 2)  

### 答案与解析

1. 答案：B  
   解析：b1 用来记录每次区间更新时对前缀和增量的影响，b2 用来校正前缀和的累加偏移。

2. 答案：  
   b2.update(l, d*(l-1));  
   b2.update(r+1, -d*r);  
   解析：根据双树状数组原理，b2 负责累积公式中的修正项。

3. 已填充正确，无需修改。  
   解析：公式来源于将区间增量展开到前缀和的推导。

4. 答案：11  
   解析：  
   - 第一次操作后，数组变为 [0,3,3,3,3,0,…]  
   - 第二次操作后，数组变为 [0,3,5,5,3,0,…]  
   - query_range(1,4) = 0+3+5+5 = 13  

   （注意：题目问的是前 4 项，结果应为 0+3+5+5=13）
