## 趣味引入

假设你在玩一款沙盒游戏，需要实时统计玩家在地图某个区域内收集的宝石数量。每次玩家在任意坐标 (x,y) 放置或移除宝石，都需要立刻反馈该区域的总宝石数，以便触发不同的游戏事件。

已知：  
- 地图是一个 n×m 的网格。  
- 玩家频繁地在任意格子更新宝石数。  
- 需要快速回答任意矩形区域的宝石总和。

由已知的“动态更新”+“区间求和”需求，引出“二维树状数组”这一高效数据结构。

===NEXT===

## 知识点讲解

1. 一维树状数组回顾  
   - 用单维 BIT 支持前缀和查询与单点更新，时间复杂度 O(log n)。  

2. 二维树状数组原理  
   - 将一维 BIT 扩展到二维：使用 f[i][j] 存储子矩阵信息。  
   - update(x,y,delta) 与 query(x,y) 的双重循环结构。

3. 操作流程  
   - 单点更新：  
     for (i = x; i <= n; i += i&-i)  
       for (j = y; j <= m; j += j&-j)  
         f[i][j] += delta;  
   - 区域求和：  
     sum(x,y) = sum_{i=1..x,j=1..y} f[i][j]  
     双重内层 i-=i&-i, j-=j&-j。

4. 高级应用示例  
   - 矩形区域更新 + 矩形区域求和（借助 4 个二维 BIT 或 差分 + BIT 技巧）。  
   - 离散化坐标后，结合事件排序解决离线查询。  

5. 复杂度分析  
   - 空间：O(n×m)  
   - 单次 update/query：O(log n × log m)

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int n;
int m;
vector<vector<int>> f;

// 在 (x,y) 点增加 delta
void update(int x, int y, int delta)
{
    for (int i = x; i <= n; i += i & -i)
    {
        for (int j = y; j <= m; j += j & -j)
        {
            f[i][j] += delta;  // 累加到所有相关节点
        }
    }
}

// 查询 [1..x][1..y] 的区域和
int query(int x, int y)
{
    int s = 0;
    for (int i = x; i > 0; i -= i & -i)
    {
        for (int j = y; j > 0; j -= j & -j)
        {
            s += f[i][j];  // 累加贡献
        }
    }
    return s;
}

// 查询任意矩形 [x1..x2][y1..y2]
int queryRect(int x1, int y1, int x2, int y2)
{
    return query(x2, y2)
         - query(x1 - 1, y2)
         - query(x2, y1 - 1)
         + query(x1 - 1, y1 - 1);
}

int main()
{
    // 示例：初始化 5×5 空矩阵
    n = 5;
    m = 5;
    f.assign(n + 1, vector<int>(m + 1, 0));

    // 在 (2,3) 增加 4
    update(2, 3, 4);

    // 查询 (1,1) 到 (3,4) 的矩形和
    cout << queryRect(1, 1, 3, 4) << endl;  // 输出 4

    return 0;
}
```

## 练习题目

基础题（选择 / 填空，各 2 题）  
1. 在 n×m 的空矩阵中，执行 update(3,4,5) 后，query(3,4) 返回的结果是 ____。  
2. 下列哪种数据结构可以替代部分场景中的二维树状数组？  
   A. 线段树  
   B. 哈希表  
   C. 并查集  
   D. 栈  

进阶题（选择 / 填空）  
3. 若要支持“矩形区域更新 + 矩形区域查询”，最少需要维护几个二维树状数组？____  
4. 在 update 函数中，i & -i 的作用是（ ）。  
   A. 清除最低位的 1  
   B. 提取最低位的 1  
   C. 判断偶数或奇数  
   D. 计算二进制位数  
5. 已知矩阵大小 n=m=10^3，总操作次数 Q=10^5。若使用二维树状数组，最坏情况下时间复杂度是 ____。（可填 O(log^2 n) 类形式）

答案与解析  
1. 5  
   解析：只对 (3,4) 单点增加 5，查询前缀和包含该点。  
2. A  
   解析：线段树也可扩展到二维支持区间修改与查询；哈希表、并查集、栈不合适。  
3. 4  
   解析：矩形更新 + 矩形查询可用差分思想，需要 4 棵 BIT。  
4. B  
   解析：i & -i 提取最低位的 1，用于遍历父节点或子节点。  
5. O(log^2 n * Q)  
   解析：单次 update/query 均为 O(log n × log m)，m≈n。
