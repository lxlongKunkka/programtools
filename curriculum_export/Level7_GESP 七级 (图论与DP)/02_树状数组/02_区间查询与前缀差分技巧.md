## 趣味引入

现实场景：  
想象一下，你在学校图书馆做值班记录。每天需要给一段书架上的书籍做打标签操作：  
- 老师可能对第 L 到 R 号书架加上“新到书籍”标签；  
- 你需要查询任意书架上当前标签的数量。  

如果使用暴力方法，每次修改要遍历区间，查询也要累计区间，效率都极低。  
“有没有一种数据结构，既能快速给区间打标签，又能快速查询某个位置或某个区间的标签数？”  
今天，我们就用「树状数组 + 前缀差分」来解决这个问题！

## 知识点讲解

1. 树状数组回顾  
   - 存储结构：`int f[N]`，下标通过 `lowbit(i)=i&-i` 确定区间  
   - 操作：  
     • `add(i, v)`：在位置 i 加上 v，影响到所有包含 i 的区间  
     • `sum(i)`：查询前缀和 `a[1..i]`  

2. 区间查询（Range Sum Query）  
   - 原理：`sum(l, r) = sum(r) - sum(l-1)`  
   - 只需两次前缀和查询，时间 O(log n)  

3. 前缀差分技巧（Difference Array）  
   - 定义差分数组 `d[i] = a[i] - a[i-1]`，则原数组前缀和 `a[i] = sum_{j=1..i} d[j]`  
   - 区间更新 `add_range(l, r, v)`  
     • 做 `add(d, l, v)`  
     • 做 `add(d, r+1, -v)`  
   - 点查询 `query_point(i) = sum(d, i)`  

4. 综合运用  
   - 点更新 + 区间查询：直接使用树状数组 `add` + `sum(r)-sum(l-1)`  
   - 区间更新 + 点查询：基于差分数组，对 `d` 做两次点更新，再 `sum(i)`  

5. 拓展（可选）  
   - 如何在 O(log n) 内 支持“区间更新 + 区间查询”？  
   - 需要维护两个树状数组，方法略

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

// 树状数组（Fenwick Tree）+ 前缀差分 实现区间修改 + 点查询
int n, m;
vector<long long> f;

// lowbit 计算
int lowbit(int x)
{
    return x & -x;
}

// 在 f 上做点修改：f[i] += v
void add(int i, long long v)
{
    while (i <= n)
    {
        f[i] += v;
        i += lowbit(i);
    }
}

// 查询前缀和 sum_{1..i}
long long sum(int i)
{
    long long s = 0;
    while (i > 0)
    {
        s += f[i];
        i -= lowbit(i);
    }
    return s;
}

// 区间修改 [l, r] += v
void add_range(int l, int r, long long v)
{
    add(l, v);
    add(r + 1, -v);
}

// 点查询 a[pos]
long long query_point(int pos)
{
    return sum(pos);
}

int main()
{
    // 读入 n （位置数） 和 m （操作数）
    cin >> n >> m;
    f.assign(n + 2, 0);

    // 初始数组 a[i]，转换为差分后建树
    for (int i = 1; i <= n; i++)
    {
        long long x;
        cin >> x;
        // 差分：d[i] = a[i] - a[i-1]
        add(i, x);
        add(i + 1, -x);
    }

    // 处理 m 次操作
    // op = 1: 区间更新 l, r, v
    // op = 2: 点查询 pos
    while (m--)
    {
        int op;
        cin >> op;
        if (op == 1)
        {
            int l, r;
            long long v;
            cin >> l >> r >> v;
            add_range(l, r, v);
        }
        else
        {
            int pos;
            cin >> pos;
            cout << query_point(pos) << "\n";
        }
    }

    return 0;
}
```

关键行注释已标注，帮助同学理清每一步操作。

## 练习题目

请先完成下面 4 道题目，再对照答案自测。

1. （选择题）树状数组中，`add(i, v)` 的时间复杂度是  
   A. O(1)  
   B. O(log n)  
   C. O(n)  
   D. O(n log n)

2. （填空题）使用差分技巧实现区间更新时，需要对差分数组 d 做两次点更新：  
   add(d, ___, v)  
   add(d, ___, -v)

3. （选择题）要查询原数组 a 在 [l, r] 的区间和，我们可以通过下列哪种方式？  
   A. sum(r) + sum(l)  
   B. sum(r) - sum(l - 1)  
   C. sum(r) * sum(l - 1)  
   D. 无法查询

4. （判断题）差分数组 d[i] = a[i] - a[i-1]，则原数组 a[i] = sum_{j=1..i} d[j]。  
   A. 正确  
   B. 错误

===

### 答案与解析

1. 答案：B  
   解析：每次 `add` 会沿着 `i += lowbit(i)` 最多执行 O(log n) 步。  

2. 答案：l, r+1  
   解析：在差分数组上做 `add(d, l, v)` 和 `add(d, r+1, -v)`，才能让区间 `[l, r]` 的每个位置都加上 v。  

3. 答案：B  
   解析：区间和 `a[l] + a[l+1] + ... + a[r] = sum(r) - sum(l-1)`。  

4. 答案：A  
   解析：差分定义和前缀和还原公式正好对应。
