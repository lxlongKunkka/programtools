## 深度知识点讲解

### 1. 树形DP基础：向下路径的计算

**概念本质：**
树形DP，顾名思义，就是在树这种特殊图结构上进行的动态规划。由于树的层级结构（从根到叶子），我们通常可以利用DFS的递归特性，从叶子节点向根节点方向（或者说，从子树向父节点）逐步计算出DP状态。

**生活类比：**
想象你是一棵大树的树干。你的孩子（子节点）们各自向下延伸出最长的树枝。如果你想知道从你这里向下最长的树枝有多长，你只需要看看你的孩子们谁向下延伸得最长，然后把他们的最长长度加上你到他们那段的长度（通常是1，表示一条边），就是从你这里向下最长的树枝了。

**关键点剖析：**
我们定义 `d1[u]` 表示从节点 `u` 出发，只沿着子节点方向（即不经过 `u` 的父节点），能够到达的最远叶子节点的路径长度（边数）。

- **叶子节点：** 如果 `u` 是一个叶子节点（没有子节点），那么从 `u` 出发向下最长的路径长度就是 `0`。
- **非叶子节点：** 对于非叶子节点 `u`，它向下最长的路径，必然经过它的某个子节点 `v`。因此，`d1[u]` 就等于 `1`（`u` 到 `v` 的边）加上从 `v` 出发向下最长的路径长度 `d1[v]`。我们要在所有子节点中，选择使得这个和最大的那个。
    所以，`d1[u] = max(1 + d1[v])`，其中 `v` 是 `u` 的所有子节点。

**计算过程（DFS实现）：**
1. 从任意节点开始DFS遍历（通常选择1号节点作为根）。
2. 在DFS访问一个节点 `u` 时，先递归访问它的所有子节点 `v`。
3. 当所有子节点 `v` 的 `d1[v]` 都计算完成后，再计算 `d1[u]`。
   - 初始化 `d1[u] = 0`。
   - 遍历 `u` 的所有子节点 `v`，更新 `d1[u] = max(d1[u], 1 + d1[v])`。

**常见误区：**
- 忘记初始化 `d1[u]` 为0，导致叶子节点或只有一个子节点的节点计算错误。
- 混淆路径长度是计算边数还是节点数。在树的直径问题中，通常指边数。

### 2. 树的直径的两种计算方法

**定义：** 树的直径是指树中任意两个节点之间最长路径的长度。这条路径可能经过树中的任何节点。

#### 方法一：两次DFS（或BFS）

**核心思想：**
这个方法基于一个重要的性质：从树中任意一点 `s` 出发，通过DFS/BFS能到达的最远点 `a`，一定是该树某条直径的一个端点。然后，从点 `a` 出发，再进行一次DFS/BFS，能到达的最远点 `b`，就是这条直径的另一个端点。`a` 到 `b` 的距离就是树的直径。

**生活类比：**
你在一片大树林里迷路了，想找到最远的两棵树。
1. 你随便走到一棵树 `S` 旁边。
2. 你开始向四周探索，一直走到离 `S` 最远的那棵树 `A`。
3. 接着，你从 `A` 树出发，再次向四周探索，一直走到离 `A` 最远的那棵树 `B`。
那么，恭喜你，`A` 和 `B` 之间的距离，就是这片树林里任意两棵树之间最远的距离！

**关键点剖析：**
1. **第一次DFS：** 从任意节点（比如节点1）开始DFS，计算所有节点到节点1的距离。同时记录距离节点1最远的节点 `a`。
   - DFS函数需要传递当前节点 `u`、父节点 `p` 和当前距离 `d`。
   - 更新全局变量 `max_d` 和 `furthest_node`。
2. **第二次DFS：** 从节点 `a` 开始DFS，计算所有节点到节点 `a` 的距离。同时记录距离节点 `a` 最远的节点 `b`。
   - `a` 到 `b` 的距离就是树的直径。

**计算过程：**
```cpp
// 第一次DFS
int furthest_node = 1; // 记录最远节点
int max_dist = 0;      // 记录最远距离
void dfs1(int u, int p, int d) {
    if (d > max_dist) {
        max_dist = d;
        furthest_node = u;
    }
    // 遍历u的邻居v
    for (int i = head[u]; i != 0; i = nxt[i]) {
        int v = to[i];
        if (v == p) continue; // 避免回到父节点
        dfs1(v, u, d + 1); // 假设边权为1
    }
}

// 第二次DFS，与dfs1类似，只是从furthest_node开始
// 最终max_dist就是直径
```

**常见误区：**
- 忘记在DFS中传递父节点参数 `p`，导致DFS无限循环或走回头路。
- 距离计算错误，例如忘记加1。

#### 方法二：单次DFS的树形DP

**核心思想：**
树的直径，要么是某个节点 `u` 向下延伸的两条最长路径之和，要么就是某个节点 `u` 向下延伸的最长路径本身（如果直径只有一个端点在 `u` 的子树内）。我们可以通过一次DFS遍历，在计算 `d1[u]` 的同时，也计算出经过 `u` 的最长路径。

**生活类比：**
你是一个“节点老板”，想知道从你这里经过的最长路径有多长。
1. 你让你的每个“子节点员工”汇报：从他们那里向下最长的“业绩路径”有多长？
2. 你收到所有报告后，从中选出“最长业绩路径”和“次长业绩路径”。
3. 那么，从你这里经过的最长路径，就是把你的“最长业绩路径”和“次长业绩路径”连起来。
4. 你把这个长度和你之前计算过的“历史最长路径”比较，更新一下“历史最长路径”。

**关键点剖析：**
1. **`d1[u]`：** 从 `u` 出发，向下延伸的最长路径长度（同前）。
2. **`d2[u]`：** 从 `u` 出发，向下延伸的**次长**路径长度。
   - 为什么要次长？因为直径如果经过 `u`，它会由两条从 `u` 向下的路径组成，这两条路径必须走向不同的子树。为了最大化长度，我们自然选择最长和次长的。
3. **直径更新：** 在计算 `d1[u]` 和 `d2[u]` 的过程中，`d1[u] + d2[u]` 就是经过 `u` 的最长路径。我们用一个全局变量 `ans` 来记录所有 `d1[u] + d2[u]` 中的最大值，这个最大值就是树的直径。

**计算过程（DFS实现）：**
```cpp
int ans = 0; // 全局变量，记录树的直径
// d1[u]：从u向下最长路径
// d2[u]：从u向下次长路径
// 可以在一个DFS函数中同时计算并更新直径
void dfs_dp(int u, int p) {
    d1[u] = 0; // 初始化最长路径
    // d2[u] 可以不用显式存储，在循环中动态更新即可
    int max1 = 0, max2 = 0; // max1是当前子树最长，max2是次长

    for (int i = head[u]; i != 0; i = nxt[i]) {
        int v = to[i];
        if (v == p) continue;
        dfs_dp(v, u); // 递归计算子节点v的d1[v]

        // 从u到v的路径长度为 1 + d1[v]
        int current_path_len = 1 + d1[v]; 
        if (current_path_len > max1) {
            max2 = max1; // 原来的最长变为次长
            max1 = current_path_len; // 更新最长
        } else if (current_path_len > max2) {
            max2 = current_path_len; // 更新次长
        }
    }
    d1[u] = max1; // u向下最长路径就是max1
    ans = max(ans, max1 + max2); // 更新全局直径
}
```

**常见误区：**
- `d1[u]` 和 `d2[u]` 的更新逻辑容易出错，特别是当 `current_path_len` 插入时，需要正确处理原有的 `max1` 和 `max2` 的值。
- 忘记初始化 `ans` 为0。
- 混淆 `d1[u]` 和 `max1` 的作用。`max1` 是在 `u` 的子节点中找到的最长路径，赋值给 `d1[u]`。

### 复杂度分析

两种方法的时间复杂度都是 `O(N+M)`，其中 `N` 是节点数，`M` 是边数。对于树来说，`M = N-1`，所以都是 `O(N)`。因为每个节点和每条边都会被访问常数次。

### 例题1：计算树上所有节点向下最长路径

**题目描述：**
给定一棵N个节点的树，节点编号从1到N。请你为每个节点 `u`，计算从 `u` 出发，只沿着子节点方向（不经过父节点）能够到达的最远叶子节点的路径长度（边数）。

**思路分析：**
这正是我们知识点讲解中提到的 `d1[u]` 的直接应用。我们可以通过一次DFS，从叶子节点向根节点回溯计算。
1. **数据结构：** 使用邻接表存储树，为了区分父子关系，DFS时需要传入父节点 `p`。
2. **DFS函数：** `dfs(u, p)`
   - `d1[u]` 初始化为0。
   - 遍历 `u` 的所有邻居 `v`。
   - 如果 `v` 是 `p`，跳过。
   - 递归调用 `dfs(v, u)`。
   - `d1[u] = max(d1[u], 1 + d1[v])`。
3. **调用：** 从任意节点（如1号节点）开始DFS：`dfs(1, 0)`（0表示根节点没有父节点）。

**C++代码实现：**

```cpp
#include <bits/stdc++.h>

const int N = 100005; // 最大节点数

int head[N]; // 邻接表头
int nxt[2 * N]; // 存储下一条边的索引
int to[2 * N];  // 存储边的目标节点
int idx = 1;    // 边的计数器，从1开始

int d1[N]; // 存储从节点u向下最长路径的长度

void add(int u, int v) {
    to[idx] = v;
    nxt[idx] = head[u];
    head[u] = idx++;
}

// DFS函数，计算从u向下最长路径
// u: 当前节点
// p: u的父节点
void dfs(int u, int p) {
    d1[u] = 0; // 初始化：如果u是叶子，则向下最长路径为0

    for (int i = head[u]; i != 0; i = nxt[i]) {
        int v = to[i];
        if (v == p) { // 如果v是父节点，跳过
            continue;
        }
        dfs(v, u); // 递归计算子节点v的d1值
        // 从u到v有一条边，长度为1
        // 所以从u经过v向下最长路径为 1 + d1[v]
        d1[u] = std::max(d1[u], 1 + d1[v]);
    }
}

int main() {
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    int n; // 节点数
    std::cin >> n;

    for (int i = 0; i < n - 1; ++i) {
        int u, v;
        std::cin >> u >> v;
        add(u, v);
        add(v, u); // 无向图，添加两条边
    }

    // 从任意节点开始DFS，这里选择1号节点作为根，0作为虚拟父节点
    dfs(1, 0);

    // 输出每个节点向下最长路径
    for (int i = 1; i <= n; ++i) {
        std::cout << "Node " << i << ": Downward longest path = " << d1[i] << "\n";
    }

    return 0;
}
```

**复杂度分析：**
- 时间复杂度：`O(N)`。每个节点和每条边只被访问常数次。
- 空间复杂度：`O(N)`。用于存储邻接表和 `d1` 数组。

### 例题2：计算树的直径（两次DFS方法）

**题目描述：**
给定一棵N个节点的树，节点编号从1到N。请计算这棵树的直径（即任意两个节点之间最长路径的长度）。

**思路分析：**
采用两次DFS的方法。
1. **第一次DFS：** 从任意节点（例如1号节点）出发，找到距离它最远的节点 `a`。
2. **第二次DFS：** 从节点 `a` 出发，找到距离它最远的节点 `b`。
3. `a` 到 `b` 的距离就是树的直径。

**C++代码实现：**

```cpp
#include <bits/stdc++.h>

const int N = 100005; // 最大节点数

int head[N]; // 邻接表头
int nxt[2 * N]; // 存储下一条边的索引
int to[2 * N];  // 存储边的目标节点
int idx = 1;    // 边的计数器

int max_d;         // 记录当前DFS过程中发现的最远距离
int furthest_node; // 记录当前DFS过程中发现的距离最远的节点

void add(int u, int v) {
    to[idx] = v;
    nxt[idx] = head[u];
    head[u] = idx++;
}

// DFS函数，用于寻找距离起始节点最远的节点
// u: 当前节点
// p: u的父节点
// d: 当前节点u到起始节点的距离
void dfs_find_furthest(int u, int p, int d) {
    if (d > max_d) {
        max_d = d;
        furthest_node = u;
    }

    for (int i = head[u]; i != 0; i = nxt[i]) {
        int v = to[i];
        if (v == p) {
            continue;
        }
        dfs_find_furthest(v, u, d + 1); // 边权为1
    }
}

int main() {
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    int n; // 节点数
    std::cin >> n;

    for (int i = 0; i < n - 1; ++i) {
        int u, v;
        std::cin >> u >> v;
        add(u, v);
        add(v, u); // 无向图
    }

    // 第一次DFS：从任意节点（例如1）出发，找到距离它最远的节点A
    max_d = -1; // 初始化为-1，确保能被0覆盖
    furthest_node = -1;
    dfs_find_furthest(1, 0, 0); // 从节点1开始，距离为0，0作为虚拟父节点
    int a = furthest_node; // 得到直径的一个端点A

    // 第二次DFS：从节点A出发，找到距离它最远的节点B
    max_d = -1; // 重置最大距离
    furthest_node = -1;
    dfs_find_furthest(a, 0, 0); // 从节点A开始，距离为0
    // int b = furthest_node; // 得到直径的另一个端点B

    std::cout << "The diameter of the tree is: " << max_d << "\n";

    return 0;
}
```

**复杂度分析：**
- 时间复杂度：`O(N)`。两次DFS，每次都是 `O(N)`。
- 空间复杂度：`O(N)`。邻接表存储。

### 例题3：计算树的直径（单次DFS的树形DP方法）

**题目描述：**
给定一棵N个节点的树，节点编号从1到N。请计算这棵树的直径。

**思路分析：**
采用单次DFS的树形DP方法。
1. 对于每个节点 `u`，计算从 `u` 向下最长路径 `d1[u]` 和次长路径 `d2[u]`（可以在DFS过程中用局部变量 `max1`, `max2` 维护）。
2. 经过 `u` 的最长路径就是 `d1[u] + d2[u]`。
3. 用一个全局变量 `ans` 记录所有 `d1[u] + d2[u]` 中的最大值，这个最大值就是树的直径。

**C++代码实现：**

```cpp
#include <bits/stdc++.h>

const int N = 100005; // 最大节点数

int head[N]; // 邻接表头
int nxt[2 * N]; // 存储下一条边的索引
int to[2 * N];  // 存储边的目标节点
int idx = 1;    // 边的计数器

int d1[N]; // 存储从节点u向下最长路径的长度 (虽然这里没显式用d2数组，但概念上是需要的)
int ans;   // 全局变量，记录树的直径

void add(int u, int v) {
    to[idx] = v;
    nxt[idx] = head[u];
    head[u] = idx++;
}

// DFS函数，同时计算从u向下最长路径和更新树的直径
// u: 当前节点
// p: u的父节点
void dfs_dp_diameter(int u, int p) {
    d1[u] = 0; // 初始化：u向下最长路径长度

    int max1 = 0; // 记录从u向下最长路径的长度
    int max2 = 0; // 记录从u向下次长路径的长度

    for (int i = head[u]; i != 0; i = nxt[i]) {
        int v = to[i];
        if (v == p) {
            continue;
        }
        dfs_dp_diameter(v, u); // 递归计算子节点v的d1值

        // 从u到v有一条边，长度为1
        // 从u经过v向下延伸的路径长度为 1 + d1[v]
        int current_path_len = 1 + d1[v];

        if (current_path_len > max1) {
            max2 = max1; // 原来的最长变为次长
            max1 = current_path_len; // 更新最长
        } else if (current_path_len > max2) {
            max2 = current_path_len; // 更新次长
        }
    }
    d1[u] = max1; // 节点u向下最长路径就是max1

    // 经过u的直径可能由max1和max2组成
    // 更新全局的树的直径
    ans = std::max(ans, max1 + max2);
}

int main() {
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    int n; // 节点数
    std::cin >> n;

    for (int i = 0; i < n - 1; ++i) {
        int u, v;
        std::cin >> u >> v;
        add(u, v);
        add(v, u); // 无向图
    }

    ans = 0; // 初始化直径为0
    dfs_dp_diameter(1, 0); // 从节点1开始DFS，0作为虚拟父节点

    std::cout << "The diameter of the tree is: " << ans << "\n";

    return 0;
}
```

**复杂度分析：**
- 时间复杂度：`O(N)`。一次DFS遍历所有节点和边。
- 空间复杂度：`O(N)`。邻接表和 `d1` 数组。

## 代码实现模板

这里提供一个通用的树形DP和直径计算的C++模板，采用原始数组实现邻接表和DFS。

```cpp
#include <bits/stdc++.h> // 包含大部分常用头文件

const int MAXN = 100005; // 树的最大节点数

// 邻接表存储树
int head[MAXN]; // head[u] 存储节点u发出的第一条边的索引
int nxt[2 * MAXN]; // nxt[i] 存储第i条边的下一条边的索引
int to[2 * MAXN];  // to[i] 存储第i条边的目标节点
int idx = 1;       // 边的计数器，从1开始

// 向邻接表中添加一条边 (u -> v)
void add(int u, int v) {
    to[idx] = v;
    nxt[idx] = head[u];
    head[u] = idx++;
}

// ---------------------------------------------------
// 方法一：计算向下最长路径 (d1[u])
// ---------------------------------------------------
int d1[MAXN]; // d1[u]: 从节点u出发，向下（子树方向）最长路径的长度

// DFS函数，计算每个节点的d1值
// u: 当前节点
// p: u的父节点，用于避免回溯
void dfs_downward_path(int u, int p) {
    d1[u] = 0; // 初始化：如果u是叶子节点，则向下最长路径为0

    for (int i = head[u]; i != 0; i = nxt[i]) {
        int v = to[i];
        if (v == p) { // 避免访问父节点
            continue;
        }
        dfs_downward_path(v, u); // 递归访问子节点
        // 从u到v有一条边（长度为1），加上v向下最长路径d1[v]
        d1[u] = std::max(d1[u], 1 + d1[v]);
    }
}

// ---------------------------------------------------
// 方法二：树的直径 - 两次DFS法
// ---------------------------------------------------
int max_dist_global; // 全局变量，记录当前DFS过程中发现的最远距离
int furthest_node_global; // 全局变量，记录当前DFS过程中发现的距离最远的节点

// DFS函数，用于寻找距离起始节点最远的节点
// u: 当前节点
// p: u的父节点
// d: 当前节点u到起始节点的距离
void dfs_find_furthest(int u, int p, int d) {
    if (d > max_dist_global) {
        max_dist_global = d;
        furthest_node_global = u;
    }

    for (int i = head[u]; i != 0; i = nxt[i]) {
        int v = to[i];
        if (v == p) {
            continue;
        }
        dfs_find_furthest(v, u, d + 1); // 边权为1
    }
}

// ---------------------------------------------------
// 方法三：树的直径 - 单次DFS的树形DP法
// ---------------------------------------------------
int tree_diameter_ans; // 全局变量，记录树的直径

// DFS函数，同时计算d1[u]并更新树的直径
// u: 当前节点
// p: u的父节点
void dfs_dp_diameter(int u, int p) {
    // d1[u] 在这里直接用局部变量 max1 替代，避免重复定义
    // max1: 从u向下最长路径长度
    // max2: 从u向下次长路径长度
    int max1_path_from_u = 0;
    int max2_path_from_u = 0;

    for (int i = head[u]; i != 0; i = nxt[i]) {
        int v = to[i];
        if (v == p) {
            continue;
        }
        dfs_dp_diameter(v, u); // 递归计算子节点v的DP值

        // 从u经过v向下延伸的路径长度为 1 + (从v向下最长路径)
        // 注意：这里的 (从v向下最长路径) 实际上就是 dfs_dp_diameter 返回后 v 的 d1[v]
        // 为了避免额外数组，我们假设 dfs_dp_diameter(v,u) 结束后，d1[v] 已经计算好了
        // 实际上，更常见的做法是把 d1[u] 作为一个返回值。
        // 为了和前面保持一致，我们假定 d1[v] 已经通过某种方式得到了（例如，一个全局数组）
        // 在这个模板中，我们直接用 max1_path_from_u, max2_path_from_u 来维护 d1[u] 的逻辑
        // 并在函数结束时将 max1_path_from_u 赋值给 d1[u] (如果需要的话)

        // 为了简化，我们直接使用 d1[v] 作为子节点的最长路径
        // 那么 dfs_dp_diameter 应该在返回前把计算好的 d1[u] 存入 d1[u] 数组
        // 这里的 d1[u] 数组被重用，所以需要注意上下文
        
        // 修正：为了避免混淆，让 dfs_dp_diameter 计算并设置 d1[u]
        // 并且在函数内部用局部变量 max1, max2 来更新全局 diameter
        
        // 递归调用会计算子节点v的d1值
        // 这里需要一个 d1 数组来存储每个节点的向下最长路径
        // 假设 d1 数组已经声明在全局
        dfs_downward_path(v, u); // 确保d1[v]已计算

        int current_path_len = 1 + d1[v];
        if (current_path_len > max1_path_from_u) {
            max2_path_from_u = max1_path_from_u;
            max1_path_from_u = current_path_len;
        } else if (current_path_len > max2_path_from_u) {
            max2_path_from_u = current_path_len;
        }
    }
    // 更新当前节点u的d1值
    d1[u] = max1_path_from_u; // u向下最长路径就是max1_path_from_u

    // 经过u的直径可能由两条从u向下最长的路径组成
    tree_diameter_ans = std::max(tree_diameter_ans, max1_path_from_u + max2_path_from_u);
}


// 主函数示例
int main() {
    std::ios_base::sync_with_stdio(false);
    std::cin.tie(NULL);

    int n; // 节点数
    std::cin >> n;

    // 构建树
    for (int i = 0; i < n - 1; ++i) {
        int u, v;
        std::cin >> u >> v;
        add(u, v);
        add(v, u); // 无向图
    }

    // ---------------------------------------------------
    // 示例用法：
    // 1. 计算所有节点向下最长路径
    // dfs_downward_path(1, 0);
    // for (int i = 1; i <= n; ++i) {
    //     std::cout << "Node " << i << " downward path: " << d1[i] << "\n";
    // }

    // 2. 树的直径 - 两次DFS法
    // max_dist_global = -1;
    // furthest_node_global = -1;
    // dfs_find_furthest(1, 0, 0); // 第一次DFS，从1开始
    // int a = furthest_node_global;

    // max_dist_global = -1; // 重置
    // furthest_node_global = -1;
    // dfs_find_furthest(a, 0, 0); // 第二次DFS，从A开始
    // std::cout << "Tree diameter (Two DFS): " << max_dist_global << "\n";

    // 3. 树的直径 - 单次DFS的树形DP法
    tree_diameter_ans = 0; // 初始化直径
    // 注意：这里的dfs_dp_diameter内部会调用dfs_downward_path来填充d1数组
    // 实际应用中，通常会把这两种逻辑合并到一个DFS函数中，或者让dfs_dp_diameter返回d1[u]
    // 为了模板清晰，这里保持了分离，但需确保d1[v]在调用时已准备好
    // 实际实现时，dfs_dp_diameter 通常会自己计算 d1[u]
    // 简化 dfs_dp_diameter 的实现，让它直接计算 d1[u]
    // 并且在循环中直接使用递归调用返回的 d1[v] 值，而不是依赖全局 d1 数组
    // 这里为了符合GSP7禁止使用STL容器的原则，d1[u]必须是全局数组
    
    // 重新实现 dfs_dp_diameter，使其独立且正确填充d1数组
    // 并在计算过程中更新全局直径
    // 为了避免重复代码，这里假设 d1 数组是用来存储向下最长路径的
    // d1[u] = 从u向下最长路径
    // tree_diameter_ans = 树的直径
    
    // 新的dfs_dp_diameter_combined
    // 这个函数会计算每个节点的d1值，并更新全局直径
    std::function<void(int, int)> dfs_dp_diameter_combined = 
        [&](int u, int p) {
        d1[u] = 0; // 初始化：u向下最长路径长度

        int max1_child_path = 0; // 从u的某个子节点向下最长路径
        int max2_child_path = 0; // 从u的某个子节点向下次长路径

        for (int i = head[u]; i != 0; i = nxt[i]) {
            int v = to[i];
            if (v == p) {
                continue;
            }
            dfs_dp_diameter_combined(v, u); // 递归计算子节点v的d1值

            int current_path_through_child = 1 + d1[v]; // 从u经过v向下的路径长度

            if (current_path_through_child > max1_child_path) {
                max2_child_path = max1_child_path;
                max1_child_path = current_path_through_child;
            } else if (current_path_through_child > max2_child_path) {
                max2_child_path = current_path_through_child;
            }
        }
        d1[u] = max1_child_path; // 更新u向下最长路径

        // 经过u的直径可能由两条从u向下最长的路径组成
        tree_diameter_ans = std::max(tree_diameter_ans, max1_child_path + max2_child_path);
    };

    dfs_dp_diameter_combined(1, 0); // 从节点1开始DFS
    std::cout << "Tree diameter (Single DFS DP): " << tree_diameter_ans << "\n";


    return 0;
}
```

**详细注释：**
- 邻接表使用 `head`, `nxt`, `to`, `idx` 数组实现，高效且符合GSP要求。
- `add(u, v)` 函数用于添加边，由于是无向图，通常需要 `add(u, v)` 和 `add(v, u)` 两次调用。
- DFS函数都包含 `u` (当前节点) 和 `p` (父节点) 参数，防止DFS走回头路，陷入死循环。
- 路径长度通常计算边数，所以每经过一条边就 `+1`。
- `std::max` 用于取最大值。
- `std::ios_base::sync_with_stdio(false); std::cin.tie(NULL);` 用于加速C++的I/O。
- 代码模板中提供了三种方法的实现框架，在 `main` 函数中可以根据需要选择性启用。为了简化，`dfs_dp_diameter` 被合并为 `dfs_dp_diameter_combined` 以展示一个更整合的单次DFS DP实现。
