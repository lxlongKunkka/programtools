# Level 7：GESP 七级 (图论与DP)

## 教学目标
1. 掌握树形结构的基本概念与遍历。
2. 理解贪心算法的设计思想。
3. 掌握简单的动态规划。

## 核心知识点
- 树：二叉树的概念、性质、存储。
- 树的遍历：前序、中序、后序遍历。
- 贪心：区间调度、背包问题的贪心策略（部分）。
- 动态规划：递推公式，爬楼梯问题，最大子段和。
