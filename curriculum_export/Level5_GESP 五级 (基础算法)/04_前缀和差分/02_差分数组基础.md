# 差分数组基础

## 教学目标

知识  
- 理解差分数组的定义：diff[i]=a[i]−a[i−1]  
- 掌握如何通过差分数组进行区间加值操作：diff[l]+=x，diff[r+1]−=x  
- 学会从差分数组还原原数组：对diff做一次前缀和  

能力  
- 能够手动构建、修改和还原差分数组  
- 能够通过差分思想将区间更新的时间复杂度由O(nm)优化到O(n+m)  
- 培养对算法优化思路的分析与验证能力  

素养  
- 培养严谨的代码风格与注释习惯  
- 培养算法思维与“抽象—还原”的解决问题能力  
- 通过真实场景激发学习兴趣与探究精神

## 趣味引入

假设学校的花坛分成 n 个相连的花槽，编号从1到n。一位老师每天给某一段连着的花槽浇水：把区间[L,R]每个槽增加 x 升水。  
问：在经过 m 次相似操作后，每个花槽最终共增加了多少升水？  

  
已知  
- 花槽编号 1…n  
- 每次操作的 L、R、x  

未知  
- 每个花槽的最终增加量 a[1…n]  

如果每次操作都对 a[L…R]依次累加，最坏要 O(n×m)。差分数组可以将 m 次操作合并到 O(m) 步，再一次前缀和还原，整体 O(n+m)，效果立竿见影！

===NEXT===

## 知识点讲解

1. 差分数组定义  
   - 对原数组 a[1…n]，定义差分数组 diff[1…n]：  
     diff[1]=a[1]；  
     对于 i>1，diff[i]=a[i]−a[i−1]。  

2. 区间加值操作  
   - 要将 a[L…R] 全部加上 x，只需：  
     diff[L]+=x；  
     如果 R+1≤n，则 diff[R+1]−=x。  
   - 原理：调整差分，两边增减，使得还原时区间内累加了 x。  

3. 从差分数组还原原数组  
   - 令 a[1]=diff[1]；  
   - 对 i=2…n，a[i]=a[i−1]+diff[i]。  

4. 复杂度分析  
   - 构造初始 diff：O(n)  
   - m 次区间操作：O(m)  
   - 还原 a：O(n)  
   - 总计 O(n+m)，相比 O(nm) 大幅优化。

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n, m;
    // 输入花槽数量 n 和操作次数 m
    cin >> n >> m;

    // 原数组 a 未显式存储，仅用差分数组 diff
    // 下标从1到n，额外空间 diff[n+2] 用于处理 R+1 位
    static long long diff[100005];
    for (int i = 1; i <= n; i++)
    {
        diff[i] = 0;
    }
    diff[n + 1] = 0;

    // 读取 m 次区间加值操作
    for (int k = 0; k < m; k++)
    {
        int L, R;
        long long x;
        cin >> L >> R >> x;
        diff[L] += x;         // 区间左端加 x
        if (R + 1 <= n)
        {
            diff[R + 1] -= x; // 区间右端下一个位置减 x
        }
    }

    // 还原并输出结果
    // 当前累加和 sum 即为 a[i]
    long long sum = 0;
    for (int i = 1; i <= n; i++)
    {
        sum += diff[i];  // 前缀和，将差分还原为原值
        cout << sum << " ";
    }
    cout << "\n";

    return 0;
}
```

## 练习题目

### 题目部分

1. （填空）差分数组 diff 的定义：diff[i]=________。  
2. （选择）要将 a[L…R] 全部加上 x，下列操作中正确的是  
   A. diff[L]−=x；diff[R+1]+=x  
   B. diff[L]+=x；diff[R+1]−=x  
   C. diff[L]+=x；diff[R]−=x  
   D. diff[L]−=x；diff[R]−=x  
3. （填空）从 diff 数组还原原数组 a 时，a[i]=________。  
4. （进阶选择）已知 n=5，初始 a=[1,2,3,4,5]，对区间[2,4]加 10，最终 a[3] 等于多少？  
   A. 13  B. 12  C. 14  D. 15  

### 答案与解析

1. diff[i]=a[i]−a[i−1]。（i>1；并且 diff[1]=a[1]）  
2. B。对 diff[L]+=x, diff[R+1]−=x 可以让 [L,R] 范围在还原时每项多加 x。  
3. a[i]=a[i−1]+diff[i]。（对 i=1…n，令 a[1]=diff[1]）  
4. A。原 a=[1,2,3,4,5]，只对 2≤i≤4 增加 10，故 a[3]=3+10=13。

   要求先输入 n、m；接着 m 行操作；最后输出所有查询结果。
