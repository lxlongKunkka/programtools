## 趣味引入

场景：学校组织运动会，每个班级在不同时间段使用操场。当多次轮流使用后，如何快速统计每个时间点操场被使用的次数？  
已知：每次使用都是一个时间区间 [l, r]。如果逐秒累加，数据量大时效率低。  
未知：能否在 O(1) 时间内完成一次区间加1？如何在所有操作结束后快速得出最终使用次数？

通过“差分+前缀和”方法，将“秒级累加”变为常数更新，最后统一还原，极大提升效率。

===NEXT===

## 知识点讲解

1. 已知→差分  
   - 定义原数组 a[i] 表示时间点 i 的使用次数  
   - 构造差分数组 d[i]：  
     d[1] = a[1]，d[i] = a[i] - a[i-1]（i > 1）

2. 区间更新  
   - 要对 [l, r] 每个元素加 v，只需：  
     d[l] += v  
     d[r+1] -= v  （若 r+1 ≤ n）

3. 前缀和还原  
   - 更新完成后，通过前缀和恢复 a[i]：  
     a[1] = d[1]  
     a[i] = a[i-1] + d[i]（i > 1）

4. 时间复杂度对比  
   - 暴力每次区间更新 O(n) → 总 O(n·m)  
   - 差分更新 O(1)，还原 O(n) → 总 O(n + m)

5. 综合应用  
   - 可以在一次读写中完成所有更新，再一次性还原并查询，节省大量时间

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n, m;
    // n: 数组长度，m: 操作次数
    cin >> n >> m;

    // 原数组a与差分数组d，大小n+2防止越界
    static int a[100005];
    static int d[100005];

    // 读取初始数组
    for (int i = 1; i <= n; i++)
    {
        cin >> a[i];
    }

    // 构造初始差分数组
    d[1] = a[1];
    for (int i = 2; i <= n; i++)
    {
        d[i] = a[i] - a[i - 1];
    }

    // m次区间更新：区间[l, r]加v
    for (int i = 0; i < m; i++)
    {
        int l, r, v;
        cin >> l >> r >> v;
        d[l] += v;            // 差分头加v
        if (r + 1 <= n)
        {
            d[r + 1] -= v;    // 差分尾+1减v
        }
    }

    // 前缀和还原最终数组
    for (int i = 1; i <= n; i++)
    {
        a[i] = (i == 1 ? d[i] : a[i - 1] + d[i]);
    }

    // 输出结果
    for (int i = 1; i <= n; i++)
    {
        cout << a[i] << " ";
    }
    cout << "\n";

    return 0;
}
```

## 练习题目

### 题目部分

1. （选择题）以下差分更新操作哪个能实现对区间 [4, 8] 加 5？  
   A. d[4] -= 5     d[8] += 5  
   B. d[4] += 5     d[8] -= 5  
   C. d[3] += 5     d[8] -= 5  
   D. d[4] += 5     d[9] -= 5

2. （填空题）差分数组 d 定义为 d[i] = a[i] - a[i-1]，则原数组 a[i] 可通过 _________ 恢复。

3. （选择题）已知初始 a = [2, 2, 2, 2]，执行一次区间更新对 [1,4] 加 3，差分 d 的第 5 个位置应为（  ）  
   A. 0    B. -3    C. +3    D. 无意义

4. （填空题）在差分数组中，如果要对整个数组每个元素减 x，应对 d[1] 执行 ______，对 d[n+1] 执行 ______。（若 n+1 越界可忽略）

5. （选择题）差分 + 前缀和方法的时间复杂度是 O(    )。  
   A. O(n·m)    B. O(n + m)    C. O(n log n)    D. O(m)

### 答案与解析

1. 答案 D  
   解析：区间 [4,8] 加 5 应 d[4] += 5，d[8+1] -= 5。

2. 答案 a[i] = a[i-1] + d[i]（或前缀和）

3. 答案 B  
   解析：更新 [1,4] 时做 d[1]+=3，d[5]-=3。

4. 答案 d[1] -= x ， d[n+1] += x  
   解析：整个区间都减 x，相当于头部减，尾部+抵消。

5. 答案 B  
   解析：差分更新 O(1)·m 次 + 还原 O(n)，合计 O(n + m)。
