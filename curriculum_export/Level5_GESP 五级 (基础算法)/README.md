# GESP 五级 (基础算法)

## 教学目标
1. 掌握常用的数学算法（质数、GCD等）。
2. 理解递归算法的思想与实现。
3. 掌握二维数组的高级应用。

## 核心知识点
- 数论基础：素数判定、最大公约数 (GCD)、最小公倍数 (LCM)。
- 递归：递归的定义、终止条件、递归栈（如斐波那契数列）。
- 位运算：按位与、或、异或、左移、右移。

5. **二分法**
   > ## 📚 模块简介
这个模块将带你探索两种高效的算法思想：二分查找和贪心策略，它们能帮你更快、更聪明地解决问题！

## 🎯 学习目标
- 掌握二分查找的基本原理，能在有序数据中快速定位目标。
- 理解二分答案的运用场景，学会将复杂问题转化为可二分判断的简单问题。
- 学习贪心策略的核心思想，能设计并实现简单的贪心算法解决优化问题。

## 🚀 学习建议
- **多动手实践**：二分和贪心都非常注重逻辑推导，多写代码才能真正掌握。
- **分析问题特征**：学会识别哪些问题可以用二分或贪心解决，这是关键。
- **画图辅助理解**：对于二分查找的区间变化，画图能帮助你更清晰地理解。

## 🌟 亮点
- **效率神器**：二分查找能将搜索时间大大缩短，是处理海量数据的利器！
- **智慧解题**：贪心策略教会你每一步都做出“局部最优”选择，最终导向“全局最优”的巧妙思路。
- **竞赛必备**：这两种算法在信息学奥赛中频繁出现，是冲击高分不可或缺的技能！

加油，小程序员们，用智慧和算法点亮你的编程之路吧！

9. **递归算法**
   > ## 📚 模块简介
递归是函数直接或间接调用自身的编程技巧，将大问题拆解成小问题，简化复杂逻辑，在数学、排序、遍历等场景中大显身手。

## 🎯 学习目标
- 理解递归的基本原理与结构  
- 能用 C++ 编写简单递归函数解决问题  
- 掌握递归归纳法，分析递归时间复杂度  

## 🚀 学习建议
- 先通过画递归调用树理解流程  
- 对比循环与递归写同一算法，体会差异  

## 🌟 亮点
- **自顶向下思维**：将问题拆成子问题，思路直观  
- **代码简洁优雅**：用更少的代码解决复杂问题  

递归虽有挑战，但练习多了就会发现它无比强大，加油!
