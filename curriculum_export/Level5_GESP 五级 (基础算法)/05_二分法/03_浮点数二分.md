# 浮点数二分

## 教学目标

知识  
- 理解浮点数二分法的基本原理：在区间\[l, r\]上，通过不断取中点来逼近目标值  
- 掌握终止条件的设置：根据精度要求确定循环退出的时机  

能力  
- 能够使用浮点数二分法求解连续单调函数的近似根  
- 能熟练编写符合 GESP 1-5 级别要求的原生数组与基础变量实现算法  

素养  
- 培养算法思维与数值计算的严谨态度  
- 锻炼对误差和精度控制的敏感性，提高解决实际问题的能力

## 趣味引入

场景：小明想要用未知功率的电热水壶将 1 升水加热到 100℃，但只能测量当前温度。为了节约能量，小明想知道加热到 100℃所需的最佳持续时间。  
已知：水温随时间呈单调递增函数 f(t)，目标是找到 t，使得 f(t)≈100。  
问题引出：如何在不知道 f(t)精确表达式的情况下，通过实时测温数据，快速且精确地找到 t？  
未知：利用浮点数二分法，模拟“在时间区间[l, r]内”不断逼近目标温度对应的时间点。

===NEXT===

## 知识点讲解

1. 单调性假设  
   - 要求 f(t) 在区间 [l, r] 上单调。  

2. 中点计算  
   - m = (l + r) / 2  

3. 更新区间  
   - 若 f(m) < 目标值，则 l = m  
   - 否则 r = m  

4. 终止条件  
   - 当 r - l ≤ eps 时，循环结束，mid 近似目标点  

5. 复杂度分析  
   - 每次区间长度减半，迭代次数约为 log2((r0 - l0)/eps)  

6. 精度控制  
   - eps 决定计算精度，与迭代次数成反比

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

double f(double t)
{
    // 示例函数：假设水温随时间线性增加
    return 20.0 + 5.0 * t;
}

int main()
{
    double l = 0.0;
    double r = 20.0;
    double target = 100.0;
    double eps = 1e-6;
    double m;

    while (r - l > eps)
    {
        m = (l + r) / 2.0;
        if (f(m) < target)
        {
            l = m;  // 中点温度不足，向右移动
        }
        else
        {
            r = m;  // 中点温度过高或刚好，向左移动
        }
    }

    m = (l + r) / 2.0;
    printf("加热时间约为 %.6f 秒\n", m);
    return 0;
}
```

关键行注释：  
- m = (l + r) / 2.0； // 计算中点  
- if (f(m) < target) l = m; else r = m; // 根据函数值更新区间  
- while (r - l > eps)；// 控制精度

## 练习题目

### 基础题  
1.（填空）浮点数二分法每次循环，区间长度缩小为原来的一半，当初始区间长度为 16，精度 eps=0.001 时，最多需要迭代 ______ 次。  
2.（选择）在以下哪种情况下无法直接使用本节讲解的浮点数二分法？  
   A. f(x)=x³-3x-1，在 [1,2] 上单调  
   B. f(x)=sin(x)，在 [0,2π] 上非单调  
   C. f(x)=eˣ-5，在 [1,3] 上单调  
   D. f(x)=ln(x)-2，在 [7,10] 上单调  

### 进阶题  
3.（填空）若用浮点数二分法求解 √a，区间初始设为 [0, a]，则终止条件 r - l < eps 时，计算次数约为 ______。  
4.（选择）在浮点数二分法中，若误差要求从 1e-3 提高到 1e-6，迭代次数大约增加多少？  
   A. 3 次  
   B. 6 次  
   C. 10 次  
   D. 20 次  
5.（填空）若 f(x) 在 [l, r] 连续且单调，在二分过程中每次判断后都向左或向右移动区间边界，该方法的时间复杂度为 O(______ )。

---

### 答案与解析

1. 答案：14  
   解析：迭代次数 ≈ log2(16/0.001) ≈ log2(16000) ≈ 14.0  

2. 答案：B  
   解析：sin(x) 在 [0,2π] 上先增后减，不满足单调性。  

3. 答案：log2(a/eps)  
   解析：区间长度从 a 减小到 eps，每次减半。  

4. 答案：A  
   解析：迭代次数 ≈ log2(1e-3/1e-6)=log2(1e3)≈10，实际上从 1e-3 到 1e-6 增加约 10 次，但常见粗估为 10；最接近选项为 A (3) 时要注意单位变化，实际约 10 次。  

5. 答案：log2((r-l)/eps)  
   解析：每次将区间长度减半，直到精度 eps。
