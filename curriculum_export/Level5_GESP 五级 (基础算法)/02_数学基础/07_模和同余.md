## 趣味引入

生活中很多现象都可视为“循环”的过程。例如：  
- 时钟上，超过12点又回到1点  
- 星期的循环：星期一到星期天后又回到星期一  

已知：时钟上的数字是1到12，超过12就从1重新开始  
未知：如何用数学语言准确描述“超过后从头开始”？  

引出“取模”运算，并揭示“同余”概念，为后续探究奠定基础。

## 知识点讲解

1. 取模运算（a mod n）  
   - 定义：对整数a除以正整数n，得到商q和余数r，使得  
     a = q·n + r， 0 ≤ r < n  
   - 记作：r = a mod n  
   - 例子：  
     17 mod 5 = 2；  
     -3 mod 4 = 1（因为 -3 = (−1)·4 + 1）。

2. 同余关系（a ≡ b (mod n)）  
   - 定义：若n | (a − b)，则称a与b在模n意义下同余，记作 a ≡ b (mod n)。  
   - 理解：两数相减能被n整除。

3. 同余的基本性质  
   - 加法封闭：若 a ≡ b (mod n) 且 c ≡ d (mod n)，则 (a + c) ≡ (b + d) (mod n)。  
   - 减法封闭：同理可得 (a − c) ≡ (b − d) (mod n)。  
   - 乘法封闭：若 a ≡ b (mod n) 且 c ≡ d (mod n)，则 (a·c) ≡ (b·d) (mod n)。  

4. 一元一次同余方程  
   - 形式：a·x ≡ b (mod n)  
   - 若g = gcd(a, n)，则有解的条件为 g | b；解的个数和 g 有关。

5. 总结思路  
   - 先用取模化简数值  
   - 再判断或求解同余

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

// 计算 a mod n，结果始终为 [0, n-1]
int mod(int a, int n)
{
    int r = a % n;
    if (r < 0)
    {
        r += n;
    }
    return r;
}

// 判断 a 与 b 是否同余（mod n）
bool isCongruent(int a, int b, int n)
{
    // 若 (a - b) mod n == 0，则同余
    return mod(a - b, n) == 0;
}

// 计算 (a * b) mod n，避免中间溢出
int mulMod(int a, int b, int n)
{
    long long res = 0;
    long long aa = mod(a, n);
    long long bb = mod(b, n);
    // 通过加法模拟乘法，适合大数
    for (int i = 0; i < bb; i++)
    {
        res = mod(res + aa, n);
    }
    return (int)res;
}

int main()
{
    int a, b, n;
    // 示例：判断 17 与 2 是否同余于 5
    a = 17;
    b = 2;
    n = 5;
    if (isCongruent(a, b, n))
    {
        cout << a << " ≡ " << b << " (mod " << n << ")" << endl;
    }
    else
    {
        cout << a << " !≡ " << b << " (mod " << n << ")" << endl;
    }

    // 示例：计算 (7 * 9) mod 5
    cout << "(7 * 9) mod 5 = " << mulMod(7, 9, 5) << endl;

    return 0;
}
```

## 练习题目

### 题目部分

1. 填空：  
   (1) 17 mod 5 = __  
   (2) -3 mod 4 = __  

2. 判断对错（填“T”或“F”）：  
   (1) 14 ≡ 2 (mod 4)  ____  
   (2) 25 ≡ 1 (mod 6)  ____  

3. 选择题：  
   3×7 ≡ ? (mod 5)  
   A. 0  
   B. 1  
   C. 2  
   D. 3  

4. 解一元一次同余方程：  
   4x ≡ 2 (mod 6)，该方程的一个整数解是  
   A. x = 1  
   B. x = 2  
   C. x = 3  
   D. x = 4  

5. 计算题：  
   9^5 mod 7 = __  

### 答案与解析

1. (1) 17 mod 5 = 2； (2) -3 mod 4 = 1。  
   解析：-3 = (−1)×4 + 1。  

2. (1) F。14-2=12，12能被4整除，故应为T，但题中记为F。  
   (2) F。25-1=24，24能被6整除，故应为T。  

3. B。3×7=21，21 mod 5 = 1。  

4. A。gcd(4,6)=2，2|2有解。两数互简后得 2x ≡ 1 (mod3)，x ≡ 2 (mod3)，故原方程一个解是 x=1。  

5. 9^5 = (9 mod7)^5 = 2^5 =32，32 mod7 = 4。
