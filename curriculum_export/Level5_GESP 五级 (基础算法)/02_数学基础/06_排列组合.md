## 知识点讲解

1. 排列P(n,k)：  
   - 定义：从n个不同元素中，任取k个元素并按顺序排列的总数。  
   - 公式：P(n,k)=n×(n−1)×…×(n−k+1)  
2. 组合C(n,k)：  
   - 定义：从n个不同元素中，不考虑顺序地选出k个元素的总数。  
   - 公式：C(n,k)=P(n,k)÷k!=n!/(k!×(n−k)!)  
3. 二者关系：C(n,k)=P(n,k)/k!  
4. 常见边界：当k>n时，P(n,k)=0，C(n,k)=0；当k=0时，P(n,0)=1，C(n,0)=1。  

简单→复杂：  
- 先用公式计算小规模数据，再用程序处理大数据。  
- 通过“循环”实现公式，逐步优化到高效算法。

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

// 计算 n 的阶乘 n!
long long factorial(int n)
{
    long long f = 1;
    for (int i = 1; i <= n; i++)
    {
        f = f * i;               // 累乘得到阶乘
    }
    return f;
}

// 计算排列数 P(n, k)
long long P(int n, int k)
{
    if (k > n)
    {
        return 0;               // k>n 时无排列
    }
    long long res = 1;
    for (int i = 0; i < k; i++)
    {
        res = res * (n - i);    // 从 n 开始连乘 k 项
    }
    return res;
}

// 计算组合数 C(n, k)
long long C(int n, int k)
{
    if (k > n)
    {
        return 0;               // k>n 时无组合
    }
    return P(n, k) / factorial(k);  // 组合 = 排列 / k!
}

int main()
{
    int n, k;
    cout << "请输入 n k：" << endl;
    cin >> n >> k;
    // 输出排列数与组合数
    cout << "P(" << n << "," << k << ") = " << P(n, k) << endl;
    cout << "C(" << n << "," << k << ") = " << C(n, k) << endl;
    return 0;
}
```

## 课堂互动

1. 提问环节  
   - “如果 n=5，k=3，P(5,3)和C(5,3)分别是多少？为什么？”  
   - “当 k=0 时，为什么 P(n,0)=1，C(n,0)=1？”  
2. 小组任务  
   - 每组发一个扑克牌中的4张牌：  
     a. 计算并列举所有P(4,2)的排列；  
     b. 列举所有C(4,2)的组合。  
   - 比赛：最先列举完整且正确小组获胜。

## 练习题目

基础题（选择题/填空题，共4题）

1. 【填空】P(5,3)=____。  
2. 【单选】下列哪个是 C(7,2) 的正确值？  
   A. 14  
   B. 21  
   C. 35  
   D. 28  
3. 【填空】从8个学生中选出3人组成小组，不考虑顺序，共有____种选法。  
4. 【单选】在6个不同数字中任选4个，并按顺序排列，其排列数是( )。  
   A. 360  
   B. 504  
   C. 180  
   D. 15  

答案与解析  
1. 答案：60  
   解析：P(5,3)=5×4×3=60。  
2. 答案：B  
   解析：C(7,2)=7×6/2=21。  
3. 答案：56  
   解析：C(8,3)=8×7×6/(3×2×1)=56。  
4. 答案：A  
   解析：P(6,4)=6×5×4×3=360。
