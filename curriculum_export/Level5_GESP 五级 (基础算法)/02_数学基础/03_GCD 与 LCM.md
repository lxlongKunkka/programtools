## 知识点讲解

### 1. 最大公约数 (GCD - Greatest Common Divisor)

**定义**：几个整数中，能同时整除它们的最大正整数，就叫做它们的**最大公约数**。
例如：
*   12 的约数有：1, 2, 3, 4, 6, 12
*   18 的约数有：1, 2, 3, 6, 9, 18
*   12 和 18 的公约数有：1, 2, 3, 6
*   其中最大的公约数是 6。所以，GCD(12, 18) = 6。

#### 如何求 GCD？

**方法一：列举法（暴力枚举）**
*   **思路**：从 1 开始，遍历到两个数中较小的那个数，如果某个数能同时整除这两个数，就记下它。遍历结束后，最大的那个数就是 GCD。
*   **优点**：简单直观，容易理解。
*   **缺点**：当数字很大时，效率很低。

**方法二：更相减损术**
*   **历史**：这是我国古代数学著作《九章算术》中记载的一种求最大公约数的方法。
*   **核心思想**：如果两个数 a 和 b 不相等，就用大数减去小数，得到一个差。然后用这个差和较小的数继续重复这个过程，直到两个数相等为止。这个相等的数就是它们的 GCD。
*   **原理**：`gcd(a, b) = gcd(a - b, b)` (当 `a > b`)。
*   **举例**：求 GCD(18, 12)
    1.  18 和 12。18 > 12，用 18 - 12 = 6。现在是 (12, 6)。
    2.  12 和 6。12 > 6，用 12 - 6 = 6。现在是 (6, 6)。
    3.  两个数相等，都是 6。所以 GCD(18, 12) = 6。

**方法三：辗转相除法（欧几里得算法）**
*   **历史**：这是古希腊数学家欧几里得提出的算法，是目前最常用、效率最高的求 GCD 的方法。
*   **核心思想**：`gcd(a, b) = gcd(b, a % b)` (其中 `a % b` 表示 `a` 除以 `b` 的余数)。当余数为 0 时，当前的 `b` 就是 GCD。
*   **原理**：如果 `a` 能被 `b` 整除，那么 `b` 就是 `a` 和 `b` 的 GCD。否则，`a` 和 `b` 的 GCD 等于 `b` 和 `a % b` 的 GCD。
*   **举例**：求 GCD(18, 12)
    1.  `gcd(18, 12)`
    2.  `18 % 12 = 6`。所以 `gcd(18, 12)` 变为 `gcd(12, 6)`。
    3.  `12 % 6 = 0`。余数为 0，所以当前的 `b` (即 6) 就是 GCD。
    4.  所以 GCD(18, 12) = 6。

*   **举例**：求 GCD(24, 36)
    1.  `gcd(24, 36)` (为了方便，我们通常将大数放在前面，但 `gcd(a, b)` 和 `gcd(b, a)` 结果是一样的)
    2.  `gcd(36, 24)`
    3.  `36 % 24 = 12`。所以 `gcd(36, 24)` 变为 `gcd(24, 12)`。
    4.  `24 % 12 = 0`。余数为 0，所以当前的 `b` (即 12) 就是 GCD。
    5.  所以 GCD(24, 36) = 12。

### 2. 最小公倍数 (LCM - Least Common Multiple)

**定义**：几个整数中，能同时被它们整除的最小正整数，就叫做它们的**最小公倍数**。
例如：
*   12 的倍数有：12, 24, 36, 48, 60, 72, ...
*   18 的倍数有：18, 36, 54, 72, ...
*   12 和 18 的公倍数有：36, 72, ...
*   其中最小的公倍数是 36。所以，LCM(12, 18) = 36。

#### 如何求 LCM？

我们不需要单独发明一种求 LCM 的新算法，因为 LCM 和 GCD 之间存在一个非常美妙的关系！

**GCD 与 LCM 的关系**：
对于任意两个正整数 `a` 和 `b`，它们的乘积等于它们的 GCD 与 LCM 的乘积。
即：`a * b = gcd(a, b) * lcm(a, b)`

所以，我们可以通过这个关系来求 LCM：
`lcm(a, b) = (a * b) / gcd(a, b)`

**重要提示**：在编程计算时，为了防止 `a * b` 结果过大导致溢出，我们通常会先除后乘：
`lcm(a, b) = a / gcd(a, b) * b`

*   **举例**：求 LCM(12, 18)
    1.  我们已经知道 GCD(12, 18) = 6。
    2.  根据公式：LCM(12, 18) = (12 * 18) / GCD(12, 18) = (12 * 18) / 6 = 216 / 6 = 36。
    3.  或者：LCM(12, 18) = 12 / GCD(12, 18) * 18 = 12 / 6 * 18 = 2 * 18 = 36。

## 代码示例

同学们，接下来我们用 C++ 语言来实现刚才学习的算法。请记住，在信奥比赛中，我们通常会使用单字母变量名，并且大括号要独占一行哦！

```cpp
#include <bits/stdc++.h> // 包含所有常用头文件
using namespace std;       // 使用标准命名空间

// 方法一：更相减损术求 GCD
// 函数名：gcd_sub
// 参数：两个整数 a 和 b
// 返回值：a 和 b 的最大公约数
int gcd_sub(int a, int b) {
    // 循环直到 a 和 b 相等
    while (a != b) {
        if (a > b) {
            a = a - b; // 如果 a 大于 b，则 a 减去 b
        } else {
            b = b - a; // 如果 b 大于 a，则 b 减去 a
        }
    }
    return a; // 当 a == b 时，a（或 b）就是 GCD
}

// 方法二：辗转相除法（欧几里得算法）求 GCD - 递归实现
// 函数名：gcd_euclid_recursive
// 参数：两个整数 a 和 b
// 返回值：a 和 b 的最大公约数
int gcd_euclid_recursive(int a, int b) {
    // 递归终止条件：如果 b 为 0，则 a 就是 GCD
    if (b == 0) {
        return a;
    }
    // 递归调用：gcd(a, b) 等于 gcd(b, a % b)
    return gcd_euclid_recursive(b, a % b);
}

// 方法二：辗转相除法（欧几里得算法）求 GCD - 非递归（迭代）实现
// 函数名：gcd_euclid_iterative
// 参数：两个整数 a 和 b
// 返回值：a 和 b 的最大公约数
int gcd_euclid_iterative(int a, int b) {
    int r; // 用于存储余数
    // 循环直到余数为 0
    while (b != 0) {
        r = a % b; // 计算 a 除以 b 的余数
        a = b;     // 将 b 的值赋给 a
        b = r;     // 将余数 r 的值赋给 b
    }
    return a; // 当 b 为 0 时，a 就是 GCD
}

// 利用 GCD 求 LCM
// 函数名：lcm_calc
// 参数：两个整数 a 和 b
// 返回值：a 和 b 的最小公倍数
long long lcm_calc(int a, int b) {
    // 为了防止 a * b 溢出，先除以 GCD 再乘以另一个数
    // 注意：这里使用 long long 类型来存储 LCM 结果，因为 LCM 可能会比较大
    return (long long)a / gcd_euclid_iterative(a, b) * b;
}

int main() {
    int x, y; // 定义两个整数变量

    cout << "请输入两个正整数：" << endl; // 提示用户输入
    cin >> x >> y; // 读取用户输入的两个整数

    cout << "使用更相减损术求 GCD(" << x << ", " << y << ") = " << gcd_sub(x, y) << endl;
    cout << "使用辗转相除法（递归）求 GCD(" << x << ", " << y << ") = " << gcd_euclid_recursive(x, y) << endl;
    cout << "使用辗转相除法（迭代）求 GCD(" << x << ", " << y << ") = " << gcd_euclid_iterative(x, y) << endl;
    cout << "LCM(" << x << ", " << y << ") = " << lcm_calc(x, y) << endl;

    return 0; // 程序正常结束
}

```

**代码解释：**
*   `gcd_sub(int a, int b)`：实现了更相减损术。通过 `while (a != b)` 循环，不断用大数减小数，直到两数相等。
*   `gcd_euclid_recursive(int a, int b)`：实现了辗转相除法的递归版本。当 `b` 为 0 时，`a` 就是 GCD；否则，递归调用 `gcd_euclid_recursive(b, a % b)`。
*   `gcd_euclid_iterative(int a, int b)`：实现了辗转相除法的非递归（迭代）版本。使用 `while (b != 0)` 循环，不断更新 `a` 和 `b` 的值，直到 `b` 为 0。
*   `lcm_calc(int a, int b)`：利用 `a * b = gcd(a, b) * lcm(a, b)` 的关系来计算 LCM。为了避免中间结果 `a * b` 溢出，我们先将 `a` 除以 `gcd(a, b)`，再乘以 `b`。注意，LCM 的结果可能很大，所以我们使用了 `long long` 类型来存储。
*   `main()` 函数：负责获取用户输入，调用上述函数并打印结果。

### 练习题目答案与解析

1.  **答案**：B. 12
    **解析**：
    *   24 的约数：1, 2, 3, 4, 6, 8, 12, 24
    *   36 的约数：1, 2, 3, 4, 6, 9, 12, 18, 36
    *   公约数：1, 2, 3, 4, 6, 12
    *   最大公约数是 12。

2.  **答案**：B. 75
    **解析**：
    *   首先求 GCD(15, 25)：
        *   GCD(25, 15) -> GCD(15, 25 % 15) -> GCD(15, 10)
        *   GCD(15, 10) -> GCD(10, 15 % 10) -> GCD(10, 5)
        *   GCD(10, 5) -> GCD(5, 10 % 5) -> GCD(5, 0)
        *   所以 GCD(15, 25) = 5。
    *   然后求 LCM(15, 25) = (15 / GCD(15, 25)) * 25 = (15 / 5) * 25 = 3 * 25 = 75。

3.  **答案**：A. 12, 6, 0
    **解析**：
    *   GCD(48, 18)
    *   第一次：48 % 18 = 12。现在是 GCD(18, 12)。余数是 12。
    *   第二次：18 % 12 = 6。现在是 GCD(12, 6)。余数是 6。
    *   第三次：12 % 6 = 0。现在是 GCD(6, 0)。余数是 0。
    *   最终 GCD 是 6。

4.  **答案**：`a * b`
    **解析**：根据公式 `a * b = gcd(a, b) * lcm(a, b)`。如果 `gcd(a, b) = 1`，那么 `lcm(a, b) = (a * b) / 1 = a * b`。

5.  **答案**：B
    **解析**：
    *   A 是更相减损术的迭代实现。
    *   B 是辗转相除法的迭代实现。`t = x % y; x = y; y = t;` 是典型的迭代更新步骤。
    *   C 是辗转相除法的递归实现。
    *   D 是暴力枚举法。
