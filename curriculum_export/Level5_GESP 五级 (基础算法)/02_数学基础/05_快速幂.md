# 快速幂

## 教学目标

知识  
- 理解快速幂（binary exponentiation）的基本思想：将指数的二进制拆分，利用平方取幂减少乘法次数。  
- 掌握快速幂的时间复杂度 Θ(log n) 原因分析。  

能力  
- 会手动推导小例子（如 3¹³）的快速幂计算过程。  
- 能熟练实现迭代版快速幂算法（C++、原生类型、Allman 风格）。  

素养  
- 培养二分思想与分治思维，激发算法优化意识。  
- 体会数学思想在编程中的落地，增强学习兴趣。

## 趣味引入

现实场景：  
想象你正在设计一款奇幻游戏，角色获得了一个宝物，可以让攻击力瞬间提升 a^n 倍。但如果直接执行 n 次普通乘法，会让程序卡顿。  
提问：  
- 有没有更快的方法？  
- 二进制与“平方”有什么关系？  

由此引出“快速幂”——用 Θ(log n) 次乘法，瞬间算出 a^n，让游戏既炫酷又流畅！

## 知识点讲解

1. 从暴力到优化  
   已知：aⁿ = a×a×…×a，共 n 次乘法（Θ(n)）。  
   未知：如何将乘法次数降到 Θ(log n)？  

2. 二进制拆分  
   - 将 n 用二进制表示：n = (nₜ nₜ₋₁ … n₁ n₀)₂  
   - aⁿ = a^(nₜ2ᵗ + … + n₁2¹ + n₀2⁰)  
   - 仅对二进制位为 1 的那几项做乘法：aⁿ = ∏[i|nᵢ=1] a^(2ⁱ)  

3. 平方取幂  
   - 快速生成序列：a^(2⁰), a^(2¹), …, a^(2ᵗ)  
   - 关系：a^(2ⁱ) = (a^(2ⁱ⁻¹))²  

4. 迭代版本流程  
   初始化 r = 1，  
   当 n > 0 时：  
     - 若 n%2 == 1，则 r = r * a  
     - a = a * a  
     - n = n / 2  
   返回 r  

5. 时间复杂度  
   每次循环 n 减半，共执行 Θ(log n) 次乘法。

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

long long fastPow(long long a, long long n)
{
    // r 存储最终结果，初始为 1
    long long r = 1;
    // 持续二分指数
    while (n > 0)
    {
        // 如果当前最低位为 1，就乘到结果上
        if (n % 2 == 1)
        {
            r = r * a;
        }
        // 平方 a，准备计算下一位 2^k
        a = a * a;
        // 右移一位，相当于除以 2
        n = n / 2;
    }
    return r;
}

int main()
{
    long long a, n;
    cin >> a >> n;          // 输入底数 a 和指数 n
    cout << fastPow(a, n)   // 输出 a^n
         << endl;
    return 0;
}
```

## 练习题目

基础题（选择/填空，共 4 题）

1. 快速幂算法的时间复杂度是 ______ 。  
2. 在快速幂迭代版本中，变量 r 初始值为（  ）。  
   A. 0  
   B. 1  
   C. a  
   D. n  
3. 若 n 的二进制表示为 10110₂，则需要乘到结果中的 a^(2ⁱ) 项有（  ）。  
   A. i = 1, 2, 4  
   B. i = 1, 2, 4, 5  
   C. i = 1, 2, 4, 0  
   D. i = 1, 2, 4, 3  
4. 填空：在迭代版本中，更新 a = a * a 的目的是计算 ______ 。  

答案与解析：

1. Θ(log n)  
   解析：每次将 n 减半，共进行 ~log₂n 步循环，每步常数次乘法。  
2. B  
   解析：r 用来累乘有效幂次，初始乘法中性元是 1。  
3. D  
   解析：10110₂ 从低位到高位分别是 0,1,1,0,1，对应 i=1,2,4。  
4. 2ᵏ 次幂（即上一轮 a^(2ᵏ⁻¹) 的平方）。
