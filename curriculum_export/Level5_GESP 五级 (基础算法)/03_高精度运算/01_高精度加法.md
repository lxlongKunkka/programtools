## 知识点讲解

既然计算机自带的数据类型不够用，我们就得自己想办法来表示和计算这些“超大”的数字。

### 1. 什么是高精度运算？

高精度运算，顾名思义，就是处理那些精度要求非常高，或者数字位数非常多的运算。当数字的位数超过了标准数据类型（如 `int`、`long long`）的表示范围时，我们就需要采用高精度算法。

### 2. 如何存储大整数？

我们不能直接把大整数存成一个数字变量。但是，我们可以把它当作一串字符来处理！
比如数字 `12345`，我们可以把它看作字符串 `"12345"`。
然后，我们可以把字符串中的每一个字符（数字）单独取出来，存储到一个数组中。

**关键点：逆序存储**
为了方便我们模拟小学竖式加法（从个位开始计算），我们通常会把数字**逆序**存储到数组中。

*   **例子：** 数字 `12345`
    *   字符串形式：`"12345"`
    *   逆序存储到 `int` 数组 `a` 中：
        *   `a[0] = 5` (个位)
        *   `a[1] = 4` (十位)
        *   `a[2] = 3` (百位)
        *   `a[3] = 2` (千位)
        *   `a[4] = 1` (万位)

这样，数组的下标就对应着数字的位数（`a[i]` 代表第 `i` 位，从0开始）。

### 3. 高精度加法的核心思想：模拟竖式加法

还记得我们小时候是怎么做加法竖式的吗？

```
  123
+  45
-----
  168
```

我们是从个位开始，逐位相加，如果和超过10，就向高位进1。高精度加法正是模拟了这个过程！

**步骤分解：**

1.  **输入与存储：**
    *   使用字符数组（或字符串）读取两个大整数 `a` 和 `b`。
    *   将它们分别转换为 `int` 数组 `A` 和 `B`，并确保是**逆序存储**。
    *   例如，如果输入 `s1 = "123"`, `s2 = "45"`
        *   `A` 数组会是 `[3, 2, 1]` (长度 `lenA = 3`)
        *   `B` 数组会是 `[5, 4]` (长度 `lenB = 2`)

2.  **逐位相加与进位：**
    *   创建一个新的 `int` 数组 `C` 来存储结果。
    *   设置一个 `carry` 变量，初始值为 `0`，用来保存进位。
    *   循环遍历 `A` 和 `B` 数组，从下标 `0` 开始（即从个位开始），一直到两个数字中最长的位数。
    *   在每次循环中：
        *   取出当前位 `A[i]` 和 `B[i]` 的数字。如果某个数组已经越界（比如 `A` 长度是3，当 `i=3` 时 `A[3]` 不存在），就当作 `0` 处理。
        *   计算当前位的和：`sum = A[i] + B[i] + carry`。
        *   当前位的结果是 `sum % 10`，存入 `C[i]`。
        *   新的进位 `carry = sum / 10`。

3.  **处理最终进位：**
    *   循环结束后，如果 `carry` 仍然大于 `0`（例如 `99 + 1 = 100`，最后还有个进位），需要将这个 `carry` 作为结果的最高位，添加到 `C` 数组的末尾。

4.  **输出结果：**
    *   因为 `C` 数组是逆序存储的，所以输出时要从数组的末尾（最高位）向前遍历，逐位输出。

**例子：`123 + 45`**

*   `s1 = "123"`, `s2 = "45"`
*   `A = [3, 2, 1]`, `lenA = 3`
*   `B = [5, 4]`, `lenB = 2`
*   `C` 数组（足够大），`carry = 0`

| 步骤 | `i` | `A[i]` | `B[i]` | `carry` (上次) | `sum` = `A[i]` + `B[i]` + `carry` | `C[i]` = `sum % 10` | `carry` (本次) = `sum / 10` |
| :--- | :-: | :----: | :----: | :------------: | :--------------------------------: | :-----------------: | :--------------------------: |
| 1    | 0   | 3      | 5      | 0              | 3 + 5 + 0 = 8                      | 8                   | 0                            |
| 2    | 1   | 2      | 4      | 0              | 2 + 4 + 0 = 6                      | 6                   | 0                            |
| 3    | 2   | 1      | 0      | 0              | 1 + 0 + 0 = 1                      | 1                   | 0                            |

*   循环结束，`carry = 0`。
*   结果数组 `C = [8, 6, 1]`。
*   逆序输出：`C[2] C[1] C[0]` -> `168`。

非常棒，对不对？我们用计算机模拟了我们小学学过的加法，就解决了大数字计算的难题！

## 代码示例

下面是高精度加法的 C++ 代码实现。请注意代码风格、变量命名以及不使用 STL 容器的要求。

```cpp
#include <bits/stdc++.h>

using namespace std;

// 高精度加法函数
void highPrecisionAdd(char s1[], char s2[]) {
    // 定义两个 int 数组来存储数字的每一位（逆序）
    // 数组大小需要足够大，能够存储最长数字的位数 + 1（为了可能产生的进位）
    // 假设数字长度不超过 1000 位
    const int MAXN = 1005;
    int a[MAXN] = {0}; // 存储第一个数字
    int b[MAXN] = {0}; // 存储第二个数字
    int c[MAXN] = {0}; // 存储结果

    // 获取两个数字的长度
    int len1 = strlen(s1);
    int len2 = strlen(s2);

    // 将第一个字符串数字逆序存入数组 a
    for (int i = 0; i < len1; i++) {
        a[i] = s1[len1 - 1 - i] - '0'; // 字符转数字，并逆序存储
    }

    // 将第二个字符串数字逆序存入数组 b
    for (int i = 0; i < len2; i++) {
        b[i] = s2[len2 - 1 - i] - '0'; // 字符转数字，并逆序存储
    }

    // 确定结果数组的最大可能长度
    // 结果的位数不会超过两个加数中较长者的位数 + 1
    int maxLen = max(len1, len2);
    int carry = 0; // 进位变量

    // 模拟竖式加法，从低位到高位逐位相加
    for (int i = 0; i < maxLen; i++) {
        int sum = a[i] + b[i] + carry; // 当前位数字之和加上进位
        c[i] = sum % 10;                // 当前位结果
        carry = sum / 10;               // 更新进位
    }

    // 如果最高位还有进位，将其存入结果数组的下一位
    if (carry > 0) {
        c[maxLen] = carry;
        maxLen++; // 结果的长度增加一位
    }

    // 输出结果 (逆序存储，所以从最高位开始输出)
    for (int i = maxLen - 1; i >= 0; i--) {
        cout << c[i];
    }
    cout << endl;
}

int main() {
    char s1[1005]; // 定义字符数组来存储输入的数字字符串
    char s2[1005];

    // 读取两个大整数
    cin >> s1 >> s2;

    // 调用高精度加法函数进行计算并输出结果
    highPrecisionAdd(s1, s2);

    return 0;
}
```

**代码解析：**

1.  `#include <bits/stdc++.h>` 和 `using namespace std;`：这是信奥竞赛中常用的快速包含所有标准库和使用 `std` 命名空间的方法。
2.  `char s1[1005], s2[1005];`：定义了两个足够大的字符数组来存储输入的两个大整数。
3.  `int a[MAXN] = {0}, b[MAXN] = {0}, c[MAXN] = {0};`：定义了三个整型数组。`a` 和 `b` 用于存储输入数字的每一位（逆序），`c` 用于存储计算结果的每一位（也是逆序）。初始化为0很重要，确保未使用的位是0。
4.  `strlen(s1)`：获取字符串的长度。
5.  `a[i] = s1[len1 - 1 - i] - '0';`：这是将字符数字转换为整型数字并逆序存储的关键一步。
    *   `s1[len1 - 1 - i]`：通过 `len1 - 1 - i` 实现了从字符串末尾（个位）开始取字符。
    *   `- '0'`：将字符 `'0'` 到 `'9'` 转换为对应的整数 `0` 到 `9`。
6.  `maxLen = max(len1, len2);`：确定循环的次数，至少要处理到两个数字中最长者的位数。
7.  `carry = 0;`：进位变量，初始为0。
8.  `sum = a[i] + b[i] + carry;`：当前位的和。
9.  `c[i] = sum % 10;`：当前位的结果，即和的个位。
10. `carry = sum / 10;`：新的进位，即和的十位。
11. `if (carry > 0)`：循环结束后，如果还有进位，说明结果比原来多了一位，需要额外存储。
12. `for (int i = maxLen - 1; i >= 0; i--)`：由于结果 `c` 数组是逆序存储的，所以输出时要从最后一个有效位开始向前输出，才能得到正确的数字顺序。

## 练习题目

### 选择题/填空题

**题目一 (选择题)：**
在 C++ 高精度加法中，将字符串 `"789"` 存储到 `int` 数组 `a` 中以模拟竖式加法，正确的逆序存储方式是：
A. `a[0]=7, a[1]=8, a[2]=9`
B. `a[0]=9, a[1]=8, a[2]=7`
C. `a[0]=9, a[1]=7, a[2]=8`
D. `a[0]=7, a[1]=9, a[2]=8`

**题目二 (填空题)：**
假设我们正在计算 `15 + 8` 的高精度加法。当处理到个位时，`a[0]=5`, `b[0]=8`，初始 `carry=0`。计算后，当前位结果 `c[0]` 为 `____`，新的 `carry` 值为 `____`。

**题目三 (选择题)：**
高精度加法代码中，如果两个大整数的长度分别为 `len1` 和 `len2`，那么用于存储结果的数组 `c` 的最大有效长度（不包括空余位）通常是：
A. `len1 + len2`
B. `max(len1, len2)`
C. `min(len1, len2) + 1`
D. `max(len1, len2) + 1`

**题目四 (填空题)：**
模拟计算 `99 + 1`，最终结果数组（逆序）`c` 应该存储为 `[____, ____, ____]`。

**题目五 (选择题)：**
高精度加法代码中，为了将 `char` 类型的数字字符（例如 `'5'`）转换为 `int` 类型的数字（例如 `5`），应该使用的操作是：
A. `c + '0'`
B. `c - '0'`
C. `int(c)`
D. `c * 1`

---

### 答案与解析

**答案一：**
B. `a[0]=9, a[1]=8, a[2]=7`
**解析：** 逆序存储意味着个位放在数组下标0，十位放在下标1，以此类推。对于 `"789"`，9是个位，8是十位，7是百位，所以 `a[0]=9, a[1]=8, a[2]=7`。

**答案二：**
`c[0]` 为 `3`，新的 `carry` 值为 `1`。
**解析：** `sum = a[0] + b[0] + carry = 5 + 8 + 0 = 13`。当前位结果 `c[0] = sum % 10 = 13 % 10 = 3`。新的进位 `carry = sum / 10 = 13 / 10 = 1`。

**答案三：**
D. `max(len1, len2) + 1`
**解析：** 结果的位数最多比两个加数中最长的那个多一位（例如 `99 + 1 = 100`，两位数加一位数得到三位数）。所以需要 `max(len1, len2) + 1` 的空间。

**答案四：**
`[0, 0, 1]`
**解析：**
`99 + 1`
`a = [9, 9]`, `b = [1]`
*   `i=0`: `9 + 1 + 0 = 10` -> `c[0]=0, carry=1`
*   `i=1`: `9 + 0 + 1 = 10` -> `c[1]=0, carry=1`
*   循环结束，`carry=1`。将 `carry` 存入 `c[2]`。
最终 `c = [0, 0, 1]`。

**答案五：**
B. `c - '0'`
**解析：** 在 ASCII 码中，数字字符 `'0'` 到 `'9'` 是连续排列的。所以，`'5' - '0'` 的结果就是 `5`。
