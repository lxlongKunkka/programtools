# GESP 二级 (进阶语法)

## 教学目标
1. 熟练掌握多重分支结构与逻辑运算符。
2. 理解并掌握 `while`、`do-while` 循环及循环嵌套。
3. 初步掌握一维数组的使用。

## 核心知识点
- 语法：`if-else if`, `switch-case`, `while`, `do-while`。
- 逻辑运算：`&&` (与), `||` (或), `!` (非)。
- 循环控制：`break`, `continue`。
- 嵌套循环：双层循环的执行流程（如打印图形）。
- 数组基础：定义、索引访问、遍历。

## 专题列表

1. **循环进阶**
   > 深入掌握 while/do-while 循环，学习循环累加、统计与数位拆解，再通过双层循环、九九乘法表等嵌套结构例题，全面强化循环的实战能力。
2. **ASCII 编码与字符处理**
   > ASCII 编码表、字符与整数互转、字符类型 char 的使用
3. **数学函数**
   > C++ 标准库常用数学函数：max/min、sqrt、abs、pow、floor/ceil
4. **枚举**
   > 通过单层、双重、三重循环枚举，掌握穷举搜索的基本思路，培养逐步缩小解空间的意识，为后续枚举进阶打好基础。
5. **模拟**
   > 以"按题目描述直接模拟"为核心，通过模拟思想拆解、计数模拟、过程模拟三类例题，建立规范的模拟解题框架。
