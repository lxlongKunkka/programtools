## 知识点讲解

### 1. 什么是 `while` 循环？

`while` 循环，顾名思义，就是“当……（条件）成立时，就一直做……（循环体）”。它会先判断一个条件，如果条件为真（成立），就执行循环体里的代码；执行完后，再回到开头判断条件，直到条件为假（不成立）为止，循环才会结束。

### 2. `while` 循环的基本语法结构

`while` 循环的语法非常简洁：

```cpp
// 1. 初始化（在循环开始前，为循环条件中的变量赋初始值）
// ...

while (循环条件) { // 2. 循环条件：一个布尔表达式，结果为真或假
    // 3. 循环体：当循环条件为真时，要重复执行的代码
    // ...

    // 4. 更新语句：修改循环条件中变量的值，以便在某个时刻让循环条件变为假，从而终止循环
    // ...
}
// 循环结束后的代码
```

**执行流程图：**

1.  **准备阶段：** 执行循环前的初始化代码。
2.  **判断条件：** 检查 `while` 后面的“循环条件”是否为真。
3.  **条件为真：** 如果条件为真，则执行 `{}` 大括号内部的“循环体”代码。
4.  **更新变量：** 在循环体内部，通常会有一个“更新语句”，改变影响循环条件判断的变量的值。
5.  **再次判断：** 循环体执行完毕后，程序会再次回到第 2 步，重新判断“循环条件”。
6.  **条件为假：** 如果条件为假，则跳出 `while` 循环，执行 `while` 循环后面的代码。

### 3. `while` 循环的关键要素

为了让 `while` 循环正确工作并避免死循环，我们需要牢记三个关键要素：

*   **初始化：** 在 `while` 循环开始前，必须给循环条件中用到的变量一个初始值。
*   **循环条件：** 这是一个布尔表达式，当它为真时，循环继续；当它为假时，循环停止。
*   **更新语句：** 在循环体内部，必须有能够改变循环条件中变量值的语句。如果没有更新语句，或者更新语句无法使循环条件在某个时刻变为假，就会导致死循环。

### 4. `while` 循环 vs `for` 循环

| 特点           | `for` 循环                                   | `while` 循环                                 |
| :------------- | :------------------------------------------- | :------------------------------------------- |
| **适用场景**   | 通常用于已知循环次数的情况，例如从 1 数到 10。 | 通常用于循环次数不确定，但知道终止条件的情况。 |
| **结构特点**   | 初始化、条件、更新集中在一行。                 | 初始化在循环前，条件在 `while` 后，更新在循环体内部。 |
| **灵活性**     | 相对固定，适用于计数器控制的循环。             | 更加灵活，适用于各种复杂的终止条件。         |
| **互相转换**   | 任何 `for` 循环都可以改写成 `while` 循环，反之亦然。 |

**小贴士：** 当你不知道循环要执行多少次，但知道“什么时候停下来”时，优先考虑 `while` 循环。

### 5. 死循环 (Infinite Loop)

**什么是死循环？**
如果 `while` 循环的条件永远为真，那么循环就会一直执行下去，永不停止，这就是死循环。它会导致程序卡死，占用大量计算机资源。

**如何避免死循环？**
**核心在于：确保循环体内有更新语句，且这个更新语句能够让循环条件在某个时刻变为假。**

例如：
```cpp
int i = 1; // 初始化
while (i <= 5) { // 循环条件
    cout << i << endl;
    // 这里如果忘记了 i++; 就会变成死循环！
}
```
正确的做法是：
```cpp
int i = 1; // 初始化
while (i <= 5) { // 循环条件
    cout << i << endl;
    i++; // 更新语句，让 i 最终会大于 5，循环条件变为假
}
```

## 代码示例

让我们通过几个小例子来实际操作 `while` 循环吧！

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    cout << "--- 示例1: 从1数到10 ---" << endl;
    // 目标：使用 while 循环输出 1 到 10 的数字
    int i = 1; // 初始化：计数器从 1 开始
    while (i <= 10) { // 循环条件：当 i 小于或等于 10 时，继续循环
        cout << i << " "; // 循环体：输出当前的 i
        i++; // 更新语句：i 每次增加 1，最终会达到 11，使条件变为假
    }
    cout << endl << endl;

    cout << "--- 示例2: 计算 1 到 n 的和 ---" << endl;
    // 目标：用户输入一个正整数 n，计算从 1 到 n 的所有整数之和
    int n;
    cout << "请输入一个正整数 n: ";
    cin >> n;

    int s = 0; // 初始化：总和 s 为 0
    int j = 1; // 初始化：计数器 j 从 1 开始
    while (j <= n) { // 循环条件：当 j 小于或等于 n 时，继续循环
        s = s + j; // 循环体：将当前的 j 加到总和 s 中
        j++; // 更新语句：j 每次增加 1
    }
    cout << "1 到 " << n << " 的和是: " << s << endl << endl;

    cout << "--- 示例3: 输入数字直到输入0，计算非零数字的个数和总和 ---" << endl;
    // 目标：不断接收用户输入的数字，直到用户输入 0 为止。统计输入的非零数字的个数和它们的总和。
    int x;       // 用于存储用户输入的数字
    int cnt = 0; // 初始化：非零数字的个数
    int sum = 0; // 初始化：非零数字的总和

    cout << "请连续输入数字 (输入 0 结束):" << endl;
    cin >> x; // 第一次输入，作为循环的初始判断

    while (x != 0) { // 循环条件：当输入的数字不为 0 时，继续循环
        sum = sum + x; // 循环体：将当前非零数字加到总和中
        cnt++;         // 循环体：非零数字的个数增加 1
        cin >> x;      // 更新语句：再次输入一个数字，作为下一次循环的判断
    }
    cout << "共输入了 " << cnt << " 个非零数字。" << endl;
    cout << "这些非零数字的总和是: " << sum << endl << endl;

    cout << "--- 示例4: 数字逆序输出 ---" << endl;
    // 目标：输入一个正整数，将其各位数字逆序输出。例如输入 1234，输出 4321。
    int num;
    cout << "请输入一个正整数: ";
    cin >> num;

    int temp_num = num; // 使用一个临时变量进行操作，保留原始数字
    cout << num << " 逆序后的数字是: ";

    // 循环条件：当数字不为 0 时，说明还有位可以取出来
    while (temp_num > 0) {
        int digit = temp_num % 10; // 取出个位数字
        cout << digit;             // 输出这个数字
        temp_num = temp_num / 10;  // 更新语句：将数字去掉个位，继续处理剩余部分
    }
    cout << endl;

    return 0;
}
```

### 选择题/填空题

1.  以下 `while` 循环的代码片段中，会导致死循环的是哪一个？
    A.
    ```cpp
    int i = 0;
    while (i < 5) {
        cout << i << " ";
        i++;
    }
    ```
    B.
    ```cpp
    int j = 10;
    while (j > 0) {
        cout << j << " ";
        j--;
    }
    ```
    C.
    ```cpp
    int k = 1;
    while (k <= 3) {
        cout << k << " ";
        // k 没有改变
    }
    ```
    D.
    ```cpp
    int m = 5;
    while (m > 0) {
        cout << m << " ";
        m = m - 2;
    }
    ```

2.  `while` 循环的三个关键要素是：______、______ 和 ______。

3.  请问以下 `while` 循环会输出什么？

    ```cpp
    int a = 20;
    while (a % 3 != 0) { // 当 a 不能被 3 整除时，继续循环
        a--;
    }
    cout << a;
    ```
    A. 20
    B. 19
    C. 18
    D. 17

4.  如果一个问题需要重复执行一段代码，但我们**不确定**具体要重复多少次，只知道一个**终止条件**，那么最适合使用哪种循环结构？
    A. `if-else` 语句
    B. `for` 循环
    C. `while` 循环
    D. `switch` 语句

5.  将以下 `for` 循环改写成等价的 `while` 循环：
    ```cpp
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
    ```
    请填写 `while` 循环中的空缺部分：
    ```cpp
    int i = 1;      // 初始化
    while (_______) { // 循环条件

        _______;      // 更新语句
    }
    ```

### 答案与解析

**1. 选择题 1 答案：C**
*   **解析：**
    *   A、B、D 选项中的 `i++`、`j--`、`m = m - 2` 都会改变循环变量的值，最终使循环条件变为假，从而终止循环。
    *   C 选项中，变量 `k` 的值始终是 1，`k <= 3` 永远为真，所以会形成死循环。

**2. 填空题 2 答案：初始化、循环条件、更新语句**
*   **解析：** 这是 `while` 循环正确运行并避免死循环的三个核心组成部分。

**3. 选择题 3 答案：C**
*   **解析：**
    *   初始 `a = 20`。
    *   `20 % 3 != 0` (20 除 3 余 2，不等于 0)，条件为真，`a` 变为 19。
    *   `19 % 3 != 0` (19 除 3 余 1，不等于 0)，条件为真，`a` 变为 18。
    *   `18 % 3 != 0` (18 除 3 余 0，等于 0)，条件为假，循环终止。
    *   输出 `a` 的值，即 18。

**4. 选择题 4 答案：C**
*   **解析：** `while` 循环正是为解决循环次数不确定，但知道终止条件的问题而设计的。`for` 循环通常用于已知循环次数的情况。

**5. 填空题 5 答案：`i <= 5`，`i++`**
*   **解析：**
    *   `for` 循环的第一个部分 `int i = 1;` 对应 `while` 循环前的初始化 `int i = 1;`。
    *   `for` 循环的第二个部分 `i <= 5;` 对应 `while` 循环的条件 `i <= 5`。
    *   `for` 循环的第三个部分 `i++;` 对应 `while` 循环体内的更新语句 `i++;`。
