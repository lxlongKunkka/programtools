## 知识点讲解

### 1. 回顾单层 `for` 循环

还记得我们之前学过的 `for` 循环吗？它可以让我们重复执行一段代码。
例如，如果我们想打印从1到5的数字：

```cpp
for (int i = 1; i <= 5; ++i)
{
    cout << i << " ";
}
cout << endl; // 输出：1 2 3 4 5
```
这里，变量 `i` 从1开始，每次循环加1，直到 `i` 大于5为止。

### 2. 什么是嵌套循环？

嵌套循环，顾名思义，就是在一个循环的内部，又完整地包含了另一个循环。就像我们刚才说的俄罗斯套娃，或者像时钟的指针：
*   外层循环：时针（走一圈需要12小时）
*   内层循环：分针（时针走一步，分针要走完一圈，即60分钟）
*   更内层循环：秒针（分针走一步，秒针要走完一圈，即60秒）

**它的执行顺序是这样的：**
1.  外层循环开始执行。
2.  外层循环的循环体执行到内层循环时，内层循环会**从头到尾完整地执行一遍**。
3.  内层循环执行完毕后，外层循环继续执行剩下的部分。
4.  外层循环再次进入下一次迭代时，内层循环会再次从头到尾完整执行一遍。

用代码表示就是：

```cpp
for (int i = 1; i <= 3; ++i) // 外层循环
{
    cout << "外层循环第 " << i << " 次" << endl;
    for (int j = 1; j <= 2; ++j) // 内层循环
    {
        cout << "    内层循环第 " << j << " 次" << endl;
    }
}
```
大家可以想象一下它的输出，外层循环每走一步，内层循环就要走完它所有的步数！

### 3. 九九乘法表的结构分析

让我们仔细观察九九乘法表：
1 * 1 = 1
1 * 2 = 2   2 * 2 = 4
1 * 3 = 3   2 * 3 = 6   3 * 3 = 9
...
1 * 9 = 9   2 * 9 = 18  ... 9 * 9 = 81

我们可以发现：
*   它有9行。
*   每一行的式子数量是不同的，第 `i` 行有 `i` 个式子。
*   每个式子都是 `a * b = c` 的形式。
*   在第 `i` 行中，第二个乘数（也就是等号左边第二个数字）都是 `i`。而第一个乘数从1开始，一直到 `i`。

这正好符合嵌套循环的特点！我们可以这样设计：
*   **外层循环** 控制**行数**，也就是乘法式子中的**第二个乘数**，从1到9。
*   **内层循环** 控制**列数**，也就是乘法式子中的**第一个乘数**，从1到当前外层循环的数字。

例如，当外层循环到 `i = 3` 时（表示第三行，第二个乘数是3）：
内层循环 `j` 从1开始，到 `j <= 3`。
于是会打印：
`1 * 3 = 3`
`2 * 3 = 6`
`3 * 3 = 9`
这样就完美地实现了九九乘法表的逻辑！

## 代码示例

现在，让我们一起动手编写代码，实现九九乘法表吧！

```cpp
#include <bits/stdc++.h> // 包含所有常用头文件
using namespace std;     // 使用标准命名空间

int main()
{
    // 外层循环：控制行数，同时也是乘法式子中的第二个乘数（从1到9）
    for (int i = 1; i <= 9; ++i) // 变量 i 代表当前行，也代表乘法式子中的第二个数字
    {
        // 内层循环：控制当前行中的式子数量，同时也是乘法式子中的第一个乘数（从1到i）
        for (int j = 1; j <= i; ++j) // 变量 j 代表当前列，也代表乘法式子中的第一个数字
        {
            // 打印乘法表达式：j * i = (j*i)
            // 为了美观，每个式子后面加一个制表符 \t 或者空格，让它们分隔开
            // 这里我们用空格，并且为了让等号看起来对齐，可以调整输出格式
            // 对于 Level 2，先简单实现，后续可以优化对齐
            cout << j << "*" << i << "=" << i * j << " "; 
        }
        // 每当一行式子打印完毕后，就换行，准备打印下一行的式子
        cout << endl; 
    }

    return 0; // 程序正常结束
}
```

**代码解析：**
*   `for (int i = 1; i <= 9; ++i)`：这是外层循环。它会从 `i=1` 一直执行到 `i=9`。每次 `i` 增加1，就代表我们开始打印新的一行乘法式子。
*   `for (int j = 1; j <= i; ++j)`：这是内层循环。它在每次外层循环执行时都会从 `j=1` 开始，一直执行到 `j` 等于当前 `i` 的值。这意味着，当 `i=1` 时，内层循环只执行1次（`j=1`）；当 `i=5` 时，内层循环会执行5次（`j=1` 到 `j=5`），这完美地控制了每行的式子数量。
*   `cout << j << "*" << i << "=" << i * j << " ";`：这是打印乘法式子的语句。`j` 是第一个乘数，`i` 是第二个乘数，`i * j` 是它们的积。`" "` 是为了在每个式子之间添加一个空格，让输出更清晰。
*   `cout << endl;`：当内层循环（即当前行的所有式子）执行完毕后，`endl` 会让光标移动到下一行的开头，这样下一行的式子就会在新的一行打印出来。

运行这段代码，你就会看到熟悉的九九乘法表啦！是不是很神奇？

## 练习题目

### 选择题/填空题

1.  下面这段 C++ 代码的输出结果是什么？
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;

    int main()
    {
        for (int i = 1; i <= 2; ++i)
        {
            for (int j = 1; j <= 3; ++j)
            {
                cout << i << j << " ";
            }
            cout << endl;
        }
        return 0;
    }
    ```
    A. `11 12 13`
       `21 22 23`
    B. `11 21`
       `12 22`
       `13 23`
    C. `11 12 13 21 22 23`
    D. `11 12 13`
       `21 22 23`
       `31 32 33`

2.  在打印九九乘法表的代码中，如果外层循环控制变量是 `i`，内层循环控制变量是 `j`，要实现标准的九九乘法表（即每行式子数量逐渐增加），内层循环的条件通常是 `j <= ___`。

3.  嵌套循环的执行特点是：外层循环每执行一次，内层循环会（      ）。
    A. 执行一次
    B. 从头到尾完整执行一次
    C. 不执行
    D. 执行多次，但次数不确定

4.  下列哪个选项最能描述九九乘法表的结构特点？
    A. 每一行都有固定的9个乘法式子。
    B. 每一行的式子数量都比前一行多一个。
    C. 每一行的式子数量与该行对应的第二个乘数相同。
    D. 只有第一行有式子，其他行都是空的。

5.  如果想用嵌套循环打印一个 `3x4` 的矩形星号（3行4列），外层循环和内层循环的条件分别应该是什么？
    外层循环：`for (int i = 1; i <= ___ ; ++i)`
    内层循环：`for (int j = 1; j <= ___ ; ++j)`

<br>
<br>

### 答案与解析

1.  **答案：A**
    **解析：**
    *   当 `i = 1` 时，内层循环 `j` 从1到3。打印 `11 12 13`，然后换行。
    *   当 `i = 2` 时，内层循环 `j` 从1到3。打印 `21 22 23`，然后换行。
    所以输出是两行，每行三个数字。

2.  **答案：`i`**
    **解析：** 九九乘法表的特点是第 `i` 行有 `i` 个式子。所以内层循环的 `j` 应该从1循环到 `i`。

3.  **答案：B**
    **解析：** 这是嵌套循环最核心的执行特点。外层循环每迭代一次，都会触发内层循环从其起始条件开始，完整地执行一遍所有迭代。

4.  **答案：C**
    **解析：** 九九乘法表的第1行有1个式子（1*1），第2行有2个式子（1*2, 2*2），第 `i` 行有 `i` 个式子。这与该行对应的第二个乘数（外层循环变量 `i` 的值）是相同的。

5.  **答案：外层循环：`3`，内层循环：`4`**
    **解析：** 外层循环通常控制行数，所以是 `i <= 3`。内层循环通常控制列数，所以是 `j <= 4`。
