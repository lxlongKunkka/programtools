# break 与 continue

## 教学目标

亲爱的小小程序员们，欢迎来到“循环进阶”的课堂！今天我们要学习两个非常酷的“循环控制器”——`break` 和 `continue`。

**知识目标：**
1. 理解 `break` 关键字的作用：能够立即终止当前循环。
2. 理解 `continue` 关键字的作用：能够跳过当前循环的剩余部分，直接进入下一次循环。
3. 掌握 `break` 和 `continue` 的基本语法，并能正确地在 `for` 循环和 `while` 循环中使用它们。

**能力目标：**

3. 能够编写简洁高效的代码，实现有条件地提前结束循环或跳过某些迭代。

**素养目标：**
1. 培养严谨的逻辑思维能力，学会如何精确控制程序的执行流程。

## 知识点讲解

同学们，我们已经学习了 `for` 循环和 `while` 循环，它们能让程序重复做一些事情。但有时候，我们不希望循环一直走到最后，或者我们想在某些特定情况下跳过一些步骤。这时候，`break` 和 `continue` 就派上用场啦！

### 1. `break` 关键字：立即终止循环

`break` 就像一个“紧急停止”的按钮。当程序执行到 `break` 语句时，它会立刻跳出当前所在的循环（`for` 或 `while`），然后继续执行循环后面的代码。

**作用：** 立即终止循环。
**使用场景：**
* 找到了想要的目标，不需要继续搜索了。
* 遇到了错误或特殊情况，必须停止循环。
* 用户输入了特定的“结束”指令。

**语法：**
`break;` （通常放在 `if` 语句中，表示满足某个条件才终止）

**执行流程图：**
循环条件 -> 执行循环体 -> 遇到 `if` 条件 -> `if` 条件为真 -> 执行 `break` -> **跳出整个循环** -> 执行循环后的语句

### 2. `continue` 关键字：跳过本次循环

`continue` 就像一个“跳过”的指令。当程序执行到 `continue` 语句时，它会跳过当前循环中 `continue` 后面剩下的所有代码，直接进入下一次循环的判断条件（对于 `for` 循环，会先执行迭代语句，再判断条件；对于 `while` 循环，会直接判断条件）。

**作用：** 跳过当前循环的剩余部分，直接开始下一次循环。
**使用场景：**
* 某些数据不符合处理要求，但又希望继续处理其他数据。
* 过滤掉一些特殊情况，只处理符合条件的。

**语法：**
`continue;` （通常也放在 `if` 语句中，表示满足某个条件才跳过）

**执行流程图：**
循环条件 -> 执行循环体 -> 遇到 `if` 条件 -> `if` 条件为真 -> 执行 `continue` -> **跳过本次循环剩余代码，进入下一次循环判断** -> 执行循环体（如果条件仍为真）

### 3. `break` 和 `continue` 的区别与联系

| 特性     | `break`                                  | `continue`                               |
| -------- | ---------------------------------------- | ---------------------------------------- |
| **作用** | 立即终止整个循环                       | 终止当前循环的本次迭代，进入下一次迭代 |
| **结果** | 循环彻底结束                             | 循环继续进行，只是跳过了一小段代码     |
| **比喻** | 紧急撤离，游戏结束                       | 跳过陷阱，游戏继续                       |

**重要提示：**
* `break` 和 `continue` 都只对它们最近的那个循环有效。如果是嵌套循环，它们只会影响内层循环。
* 它们通常与 `if` 语句一起使用，这样才能根据条件来控制循环。

同学们，理解了这两个“魔法词”的原理，我们就可以开始用代码来实践它们了！

## 代码示例

### 示例一：使用 `break` 查找特定数字

我们要从 1 数到 10，但是一旦数到 5，就立即停止。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int i;
    cout << "使用 break 查找数字 5：" << endl;
    for (i = 1; i <= 10; ++i)
    {
        if (i == 5)
        {
            cout << "找到数字 5 了！立即停止循环。" << endl;
            break; // 当 i 等于 5 时，立即终止整个 for 循环
        }
        cout << "当前数字是: " << i << endl;
    }
    cout << "循环结束，程序继续执行。" << endl;
    return 0;
}
```
**运行结果：**
```
使用 break 查找数字 5：
当前数字是: 1
当前数字是: 2
当前数字是: 3
当前数字是: 4
找到数字 5 了！立即停止循环。
循环结束，程序继续执行。
```
**代码解析：**
* 循环从 `i=1` 开始。
* 当 `i` 等于 1、2、3、4 时，条件 `i == 5` 不成立，所以会打印 `i` 的值。
* 当 `i` 变成 5 时，条件 `i == 5` 成立，执行 `break;`。
* `break` 语句让程序立即跳出了 `for` 循环，不再执行 `i=5` 之后的任何循环体代码，也不会继续 `i=6, 7...` 的迭代。
* 之后程序会执行 `cout << "循环结束，程序继续执行。"`。

### 示例二：使用 `continue` 跳过偶数

我们要打印 1 到 10 之间所有的奇数，跳过偶数。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int i;
    cout << "使用 continue 打印 1 到 10 的奇数：" << endl;
    for (i = 1; i <= 10; ++i)
    {
        if (i % 2 == 0) // 如果 i 是偶数 (i 除以 2 余数为 0)
        {
            cout << "跳过偶数: " << i << endl;
            continue; // 跳过当前循环的剩余部分，直接进入下一次循环
        }
        cout << "打印奇数: " << i << endl;
    }
    cout << "循环结束，所有奇数已打印。" << endl;
    return 0;
}
```
**运行结果：**
```
使用 continue 打印 1 到 10 的奇数：
打印奇数: 1
跳过偶数: 2
打印奇数: 3
跳过偶数: 4
打印奇数: 5
跳过偶数: 6
打印奇数: 7
跳过偶数: 8
打印奇数: 9
跳过偶数: 10
循环结束，所有奇数已打印。
```
**代码解析：**
* 循环从 `i=1` 开始。
* 当 `i` 是奇数时（如 1, 3, 5, 7, 9），条件 `i % 2 == 0` 不成立，不执行 `continue`，程序会打印 `i` 的值。
* 当 `i` 是偶数时（如 2, 4, 6, 8, 10），条件 `i % 2 == 0` 成立，执行 `continue;`。
* `continue` 语句让程序立即跳过了当前循环中 `cout << "打印奇数: " << i << endl;` 这行代码，直接进入下一次 `i` 的迭代（`i` 会自增，然后判断循环条件）。

### 示例三：综合运用 `break` 和 `continue`

我们要计算 1 到 20 之间所有不是 3 的倍数的数的和。但是，如果和超过 50，就立即停止计算。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int i;
    int s = 0; // 用 s 存储和
    cout << "计算 1 到 20 之间不是 3 的倍数的数的和，和超过 50 则停止：" << endl;

    for (i = 1; i <= 20; ++i)
    {
        if (i % 3 == 0) // 如果 i 是 3 的倍数
        {
            cout << "跳过数字 (3 的倍数): " << i << endl;
            continue; // 跳过当前循环，不加到和里
        }

        s = s + i; // 将不是 3 的倍数的数加到和 s 中
        cout << "当前数字: " << i << ", 累加后的和: " << s << endl;

        if (s > 50) // 如果和 s 超过 50
        {
            cout << "和已经超过 50 (" << s << ")，停止计算！" << endl;
            break; // 立即终止整个循环
        }
    }
    cout << "最终的和是: " << s << endl;
    return 0;
}
```
**代码解析：**
* 循环从 `i=1` 到 `i=20`。
* 首先判断 `i` 是不是 3 的倍数。如果是，`continue` 跳过本次循环，不执行后面的累加和判断。
* 如果 `i` 不是 3 的倍数，则将 `i` 加到 `s` 中。
* 累加之后，检查 `s` 是否大于 50。如果大于 50，则 `break` 立即终止整个 `for` 循环。
* 这个例子很好地展示了 `continue` 过滤不符合条件的情况，而 `break` 在达到特定目标时提前结束循环的功能。

### 练习题目答案与解析

1.  **答案：B**
    **解析：** 当 `i` 等于 0, 1, 2 时，条件 `i == 3` 不成立，会打印 `i` 的值。当 `i` 等于 3 时，条件 `i == 3` 成立，执行 `break` 语句，循环立即终止。所以只会打印 0, 1, 2。

2.  **答案：B**
    **解析：**
    *   当 `i = 0` 时，`0 % 2 == 0` 成立，执行 `continue`，跳过 `cout << i << " ";`，进入下一次循环。
    *   当 `i = 1` 时，`1 % 2 == 0` 不成立，打印 `1`。
    *   当 `i = 2` 时，`2 % 2 == 0` 成立，执行 `continue`，跳过打印。
    *   当 `i = 3` 时，`3 % 2 == 0` 不成立，打印 `3`。
    *   当 `i = 4` 时，`4 % 2 == 0` 成立，执行 `continue`，跳过打印。
    *   所以最终输出 `1 3 `。

3.  **答案：D**
    **解析：**
    *   A 错误，`break` 和 `continue` 都只对最近的一层循环有效。
    *   B 错误，`break` 是完全终止循环，`continue` 是结束当前迭代。
    *   C 错误，`continue` 也通常与 `if` 语句配合使用。
    *   D 正确，准确描述了 `break` 和 `continue` 的作用。

4.  **填空处应填：`continue`**
