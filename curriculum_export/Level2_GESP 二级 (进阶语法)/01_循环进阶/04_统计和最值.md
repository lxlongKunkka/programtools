# 统计和最值

## 教学目标

知识目标  
- 理解如何使用循环结构统计数据个数  
- 掌握在循环中通过比较操作求最大值和最小值  

能力目标  
- 能运用 `for` 循环和条件判断，实现对一组数据的统计与最值分析  
- 学会合理初始化变量，并在循环中动态更新统计结果  

素养目标  
- 培养认真审题、细心调试的良好编程习惯  
- 体验算法思维：分析问题、设计变量、实现逻辑、验证结果

## 趣味引入

真实场景：  
班级运动会上，老师想知道全班同学跑步成绩中，有多少人跑进一分半内（小于等于90秒），以及最快和最慢成绩分别是多少。  

已知：每位同学跑步成绩（以秒计）  
未知：如何用程序统计“进90秒内的人数”，并找出“最快成绩”和“最慢成绩”？  

引出本节课主题：“统计和最值”  

===NEXT===

## 知识点讲解

1. 已知→未知  
   - 已知：一组成绩数据  
   - 未知：符合条件的计数、最大值、最小值  

2. 简单→复杂  
   - 先统计满足条件的个数  
   - 再在同一循环中同时更新最大值与最小值  

3. 理论→实践  
   - 初始化  
     - 计数器 `cnt` 设为 0  
     - 最大值 `max` 设为一个极小值，比如输入范围之外的最小值  
     - 最小值 `min` 设为一个极大值，比如输入范围之外的最大值  
   - 循环过程  
     - 读取单个元素 `a[i]`  
     - 如果满足条件，`cnt++`  
     - 比较 `a[i]` 与当前 `max`/`min`，必要时更新  
   - 循环结束后，`cnt`,`max`,`min` 即为答案

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n;
    cin >> n;              // 输入数据数量
    int a[1000];           // 最大支持1000个数据
    for (int i = 0; i < n; i++)
    {
        cin >> a[i];
    }

    int cnt = 0;           // 统计成绩 ≤ 90 的人数
    int maxv = -1;         // 初始化为不可能的最小值
    int minv = 1000000;    // 初始化为不可能的最大值

    for (int i = 0; i < n; i++)
    {
        // 统计进90秒内的人数
        if (a[i] <= 90)
        {
            cnt++;
        }

        // 更新最大值
        if (a[i] > maxv)
        {
            maxv = a[i];
        }

        // 更新最小值
        if (a[i] < minv)
        {
            minv = a[i];
        }
    }

    cout << "进90秒内的人数: " << cnt << endl;
    cout << "最快成绩: " << maxv << endl;
    cout << "最慢成绩: " << minv << endl;

    return 0;
}
```

## 练习题目

### 题目部分

1. （填空）在以下代码中，为了统计数组 `a` 中大于 100 的数的个数，计数器 `cnt` 应该初始化为 ________。  
   ```cpp
   int cnt = ___;
   for (int i = 0; i < n; i++)
   {
       if (a[i] > 100)
       {
           cnt++;
       }
   }
   ```

2. （选择）要在循环中找出 `a` 数组的最大值，最合适的初始化方式是  
   A. `int maxv = 0;`  
   B. `int maxv = a[0];`  
   C. `int maxv = INT_MIN;`  
   D. `int maxv = 10000;`  

3. （判断）如果要找最小值，下面初始化和更新方式正确的是  
   int minv = 100000;  
   if (a[i] < minv)  
   {  
       minv = a[i];  
   }  
   （  ）

4. （选择/进阶）一组数据中既有正数也有负数，要统计负数的个数并找出最大负数，初始化最大负数 `maxNeg` 最正确的是  
   A. `int maxNeg = 0;`  
   B. `int maxNeg = a[0];`  
   C. `int maxNeg = INT_MIN;`  
   D. `int maxNeg = -1;`  

### 答案与解析

1. 答案：0  
   解析：计数器初值应从 0 开始累加。

2. 答案：B 或 C  
   解析：  
   - B(`a[0]`)：直接从第一个元素开始比较，通用且安全。  
   - C(`INT_MIN`)：保证比任何元素都小，也可行。  

3. 答案：正确  
   解析：`minv` 初始化为足够大的值，保证第一个 `a[i]` 肯定小于它，从而正确更新。

4. 答案：C  
   解析：要查找最大负数，应初始化为最小的整型，以便所有负数都能更新。
