# 字符与整数的相互转换

## 深度知识点讲解

### 1. 计算机的“秘密语言”：ASCII码

计算机是“数字控”，它只认识0和1。我们平时看到的文字、符号，在计算机内部都必须先转换成数字才能存储和处理。**ASCII (American Standard Code for Information Interchange)** 编码就是这样一套“翻译规则”，它给世界上大部分常用的字符（包括英文字母、数字、标点符号等）都分配了一个唯一的整数编号。

*   **就像图书馆里每本书都有一个书名（字符）和一个唯一的编号（ASCII码）一样。** 你想找《哈利·波特》，计算机不会直接去找“哈利·波特”这几个字，它会根据《哈利·波特》的编号（比如10086号）去书架上查找。

**重要的小秘密：**
*   字符 **'0'** 的ASCII码是 **48**。
*   字符 **'1'** 的ASCII码是 **49**。
*   ...
*   字符 **'9'** 的ASCII码是 **57**。
    *   **发现了吗？** 数字字符 '0' 到 '9' 的ASCII码是连续的，而且依次递增！
*   字符 **'A'** 的ASCII码是 **65**。
*   字符 **'B'** 的ASCII码是 **66**。
*   ...
*   字符 **'Z'** 的ASCII码是 **90**。
    *   **同样！** 大写字母 'A' 到 'Z' 的ASCII码也是连续的，依次递增！
*   字符 **'a'** 的ASCII码是 **97**。
*   字符 **'b'** 的ASCII码是 **98**。
*   ...
*   字符 **'z'** 的ASCII码是 **122**。
    *   **惊喜不惊喜？** 小写字母 'a' 到 'z' 的ASCII码也是连续的，依次递增！

**这个“连续性”特点非常非常重要！** 它是我们进行字符处理时的“魔法棒”！

### 2. `char` 类型和 `int` 类型：字符与整数的“变身”

在C++中：
*   `char` 是一种数据类型，专门用来存储单个字符，比如 `'A'`, `'5'`, `'@'`。
*   `int` 是一种数据类型，专门用来存储整数，比如 `65`, `5`, `100`。

**核心思想：** `char` 类型变量虽然看起来存的是字符，但它在计算机内部存储的其实是这个字符对应的ASCII码（一个整数）！所以，`char` 和 `int` 之间有着天然的联系，可以互相“变身”。

### 3. 字符变整数 ( `char` -> `int` )

当你想把一个字符变成它对应的ASCII码，或者把一个数字字符变成它代表的数字时，就可以进行转换。

**方法一：隐式转换（自动变身）**
当你把一个 `char` 类型的变量赋值给一个 `int` 类型的变量时，C++会自动把 `char` 变量中存储的ASCII码值赋给 `int` 变量。
```cpp
char c = 'A'; // c 存储的是字符 'A'
int i = c;    // 隐式转换，i 存储的是 'A' 的ASCII码 65
std::cout << i << std::endl; // 输出 65
```

**方法二：强制类型转换（明确变身）**
你可以明确告诉C++，把这个 `char` 变量当作 `int` 来处理。
```cpp
char c = 'B';
int i = (int)c; // 强制类型转换，i 存储的是 'B' 的ASCII码 66
std::cout << i << std::endl; // 输出 66
```

**最最最重要的应用：数字字符转换为整数！**
如果我们有一个字符 `'5'`，我们想把它变成真正的整数 `5`，怎么办呢？
还记得我们说的ASCII码的连续性吗？
*   '0' 的ASCII码是 48
*   '1' 的ASCII码是 49
*   '2' 的ASCII码是 50
*   '3' 的ASCII码是 51
*   '4' 的ASCII码是 52
*   '5' 的ASCII码是 53

如果我们用 `'5'` 的ASCII码减去 `'0'` 的ASCII码，会发生什么？
`53 - 48 = 5`！
**Bingo！** 这就是我们想要的整数 `5`！

所以，**将数字字符转换为对应整数的“魔法公式”是： `数字字符 - '0'`**
```cpp
char c = '7'; // 字符'7'
int n = c - '0'; // '7'的ASCII码减去'0'的ASCII码
                 // 实际上是 55 - 48 = 7
std::cout << n << std::endl; // 输出 7
```
这个技巧在编程中非常常用，请同学们牢牢记住！

### 4. 整数变字符 ( `int` -> `char` )

当你想把一个整数（特别是ASCII码值）变成它对应的字符，或者把一个数字变成它对应的数字字符时，就可以进行转换。

**方法一：隐式转换（自动变身）**
当你把一个 `int` 类型的变量赋值给一个 `char` 类型的变量时，C++会自动把 `int` 变量中存储的整数值当作ASCII码，然后把对应的字符赋给 `char` 变量。
```cpp
int i = 65;   // i 存储的是整数 65
char c = i;   // 隐式转换，c 存储的是ASCII码 65 对应的字符 'A'
std::cout << c << std::endl; // 输出 A
```

**方法二：强制类型转换（明确变身）**
你可以明确告诉C++，把这个 `int` 变量当作 `char` 来处理。
```cpp
int i = 97;
char c = (char)i; // 强制类型转换，c 存储的是ASCII码 97 对应的字符 'a'
std::cout << c << std::endl; // 输出 a
```

**最最最重要的应用：整数转换为数字字符！**
如果我们有一个整数 `3`，我们想把它变成字符 `'3'`，怎么办呢？
还是利用ASCII码的连续性！
'0' 的ASCII码是 48
'1' 的ASCII码是 49
'2' 的ASCII码是 50
'3' 的ASCII码是 51

如果我们用整数 `3` 加上 `'0'` 的ASCII码，会发生什么？
`3 + 48 = 51`！
而 51 正是字符 `'3'` 的ASCII码！

所以，**将整数转换为对应数字字符的“魔法公式”是： `整数 + '0'`**
```cpp
int n = 8;     // 整数8
char c = n + '0'; // 8 加上 '0' 的ASCII码
                 // 实际上是 8 + 48 = 56，对应的字符是 '8'
std::cout << c << std::endl; // 输出 '8'
```
这个技巧同样非常常用，请同学们牢牢记住！

### 5. 常见误区

1.  **混淆字符和整数：** 字符 `'5'` 和整数 `5` 是完全不同的东西！`'5'` 是一个字符，它的ASCII码是53；`5` 是一个整数，就是数字五本身。
2.  **直接对数字字符进行数学运算：** `char c = '3' + '5';` 这样的代码不会得到字符 `'8'`！它会计算 `'3'` 的ASCII码 (51) 加上 `'5'` 的ASCII码 (53)，结果是 104。104对应的字符是 'h'，所以 `c` 会变成 `'h'`。这是错误的用法！
    *   **正确做法：** 应该先转换成整数再运算：`int n = ('3' - '0') + ('5' - '0');` 这样 `n` 才会是 `3 + 5 = 8`。

理解了这些，你就掌握了字符与整数转换的“核心魔法”！

## 典型例题精讲

### 例题 1：字母大小写转换

**题目描述：**
输入一个大写英文字母，请你将它转换为对应的小写英文字母并输出。

**输入样例：**
`F`

**输出样例：**
`f`

**思路分析：**
1.  首先，我们需要读取一个字符。
2.  思考大写字母和小写字母的ASCII码有什么关系？
    *   'A' 是 65，'a' 是 97
    *   'B' 是 66，'b' 是 98
    *   'C' 是 67，'c' 是 99
    *   我们发现，每个小写字母的ASCII码都比它对应的大写字母的ASCII码大一个固定的值。这个差值是 `97 - 65 = 32`。
3.  所以，如果我们有一个大写字母 `c`，要把它变成小写字母，只需要将它的ASCII码加上 32 即可。
4.  或者，更通用和易懂的方式是加上 `'a' - 'A'` 这个差值。
    `char small_c = big_c + ('a' - 'A');`

**代码实现：**

```cpp
#include <bits/stdc++.h> // 包含所有常用库，GESP常用

int main() {
    char c; // 定义一个字符变量用于存储输入的大写字母
    std::cin >> c; // 读取一个字符

    // 方法一：直接加上固定的ASCII码差值 32
    // char s = c + 32;

    // 方法二：更通用、易读的方式，计算 'a' 和 'A' 的ASCII码差值
    // 这样即使ASCII码标准有微小变化，代码也依然正确
    char s = c + ('a' - 'A'); // 将大写字母转换为小写字母

    std::cout << s << std::endl; // 输出转换后的小写字母

    return 0;
}
```

**时间复杂度分析：**
本程序只进行了一次字符读取和一次简单的加法运算，操作次数是常数，与输入无关。因此，时间复杂度为 **O(1)**。

### 例题 2：计算数字字符之和

**题目描述：**
输入两个数字字符（例如 '3' 和 '7'），请计算它们所代表的整数的和，并输出这个整数结果。

**输入样例：**
`3 7`

**输出样例：**
`10`

**思路分析：**
1.  首先，我们需要读取两个字符。
2.  这两个字符是 `'3'` 和 `'7'`，它们是字符类型，不能直接进行数学加法运算。
3.  我们需要将字符 `'3'` 转换为整数 `3`，将字符 `'7'` 转换为整数 `7`。
    *   还记得“魔法公式”吗？`数字字符 - '0'`。
    *   所以，`int n1 = c1 - '0';` 和 `int n2 = c2 - '0';`。
4.  将转换后的两个整数相加，得到它们的和。
5.  最后，输出这个整数和。

**代码实现：**

```cpp
#include <bits/stdc++.h> // 包含所有常用库，GESP常用

int main() {
    char c1, c2; // 定义两个字符变量，用于存储输入的数字字符
    std::cin >> c1 >> c2; // 读取两个字符

    // 将第一个数字字符转换为对应的整数
    int n1 = c1 - '0'; // 例如，如果c1是'3'，n1就是3 (51-48=3)

    // 将第二个数字字符转换为对应的整数
    int n2 = c2 - '0'; // 例如，如果c2是'7'，n2就是7 (55-48=7)

    // 计算这两个整数的和
    int sum = n1 + n2;

    std::cout << sum << std::endl; // 输出整数和

    return 0;
}
```

**时间复杂度分析：**
本程序只进行了一次字符读取和几次简单的减法、加法运算，操作次数是常数，与输入无关。因此，时间复杂度为 **O(1)**。

## 代码实现模板

下面是一个综合性的代码模板，展示了字符与整数相互转换的常用场景。

```cpp
#include <bits/stdc++.h> // 包含所有常用库，方便GSPP考试，等同于 #include <iostream> 等

int main() {
    // --------------------------------------------------------------------
    // 场景一：字符类型变量存储字符，本质是存储其ASCII码
    char chA = 'A'; // chA 存储字符 'A'，其内部值是 65
    char ch0 = '0'; // ch0 存储字符 '0'，其内部值是 48

    std::cout << "字符 '" << chA << "' 的ASCII码是: " << (int)chA << std::endl;
    std::cout << "字符 '" << ch0 << "' 的ASCII码是: " << (int)ch0 << std::endl;

    // --------------------------------------------------------------------
    // 场景二：字符（char）到整数（int）的转换
    // 1. 隐式转换：将 char 赋值给 int，自动转换为 ASCII 码
    int iA = chA; // iA 变为 65
    int i0 = ch0; // i0 变为 48
    std::cout << "隐式转换：chA ('" << chA << "') 转换为整数是: " << iA << std::endl;
    std::cout << "隐式转换：ch0 ('" << ch0 << "') 转换为整数是: " << i0 << std::endl;

    // 2. 强制类型转换：明确转换为 int
    char chX = 'X';
    int iX = (int)chX; // iX 变为 'X' 的ASCII码 88
    std::cout << "强制转换：chX ('" << chX << "') 转换为整数是: " << iX << std::endl;

    // 3. 特殊且常用：数字字符转换为对应的整数
    //    核心思想：利用数字字符ASCII码的连续性，减去 '0' 的ASCII码
    char digitChar = '5';
    int digitNum = digitChar - '0'; // '5' (53) - '0' (48) = 5
    std::cout << "数字字符 '" << digitChar << "' 转换为整数是: " << digitNum << std::endl;

    // --------------------------------------------------------------------
    // 场景三：整数（int）到字符（char）的转换
    // 1. 隐式转换：将 int 赋值给 char，自动转换为对应字符
    int asciiVal = 67; // 67 是 'C' 的ASCII码
    char chC = asciiVal; // chC 变为 'C'
    std::cout << "隐式转换：整数 " << asciiVal << " 转换为字符是: '" << chC << "'" << std::endl;

    // 2. 强制类型转换：明确转换为 char
    int asciiVal2 = 100; // 100 是 'd' 的ASCII码
    char chD = (char)asciiVal2; // chD 变为 'd'
    std::cout << "强制转换：整数 " << asciiVal2 << " 转换为字符是: '" << chD << "'" << std::endl;

    // 3. 特殊且常用：整数转换为对应的数字字符
    //    核心思想：利用数字字符ASCII码的连续性，加上 '0' 的ASCII码
    int numToChar = 9;
    char charToNum = numToChar + '0'; // 9 + '0' (48) = 57，57 是 '9' 的ASCII码
    std::cout << "整数 " << numToChar << " 转换为数字字符是: '" << charToNum << "'" << std::endl;

    // --------------------------------------------------------------------
    // 综合应用示例：读取一个数字字符，将其代表的数字加 2，再输出新的数字字符
    char inputD;
    std::cout << "\n请输入一个数字字符 (0-7): ";
    std::cin >> inputD;

    int tempNum = inputD - '0'; // 字符转整数
    tempNum = tempNum + 2;       // 整数加 2
    char outputD = tempNum + '0'; // 整数转字符

    std::cout << "原始字符: '" << inputD << "', 加2后的数字字符: '" << outputD << "'" << std::endl;

    return 0;
}
```

**拓展挑战答案：**

1.  ```cpp
    #include <bits/stdc++.h>
    int main() {
        char s[10]; // 用字符数组存储字符串，GESP二级不用string
        std::cin >> s;
        int n = 0;
        // 遍历字符串直到遇到字符串结束符 '\0'
        for (int i = 0; s[i] != '\0'; ++i) {
            n = n * 10 + (s[i] - '0'); // 每读一位，累加并乘以10
        }
        std::cout << n << std::endl;
        return 0;
    }
    ```
2.  ```cpp
    #include <bits/stdc++.h>
    int main() {
        int n;
        std::cin >> n;
        if (n == 0) { // 特殊处理0
            std::cout << '0' << std::endl;
            return 0;
        }
        char s[10]; // 用字符数组存储结果
        int i = 0;
        while (n > 0) {
            s[i] = (n % 10) + '0'; // 取个位数字并转换为字符
            n = n / 10;            // 去掉个位
            i++;
        }
        // 字符串是逆序存储的，需要反向输出
        for (int j = i - 1; j >= 0; --j) {
            std::cout << s[j];
        }
        std::cout << std::endl;
        return 0;
    }
    ```
