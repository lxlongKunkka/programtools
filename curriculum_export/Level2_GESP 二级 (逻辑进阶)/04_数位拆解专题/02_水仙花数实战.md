## 趣味引入

同学们，大家好！今天我们要一起探索一种非常有趣的数字——水仙花数！

大家有没有玩过数字游戏？比如，给你一个数字，让你猜它有什么特别之处？今天我们要猜的数字，它们可厉害了！

我们先来看一个神奇的三位数：153。
我们把它的每个位上的数字拆开：个位是3，十位是5，百位是1。
现在，我们把每个数字都进行一个特殊操作——立方（也就是自己乘以自己，再乘以自己），然后把结果加起来：
1的立方是 1 * 1 * 1 = 1
5的立方是 5 * 5 * 5 = 125
3的立方是 3 * 3 * 3 = 27
把它们加起来：1 + 125 + 27 = 153！

哇！发现了吗？这个三位数的每个位上的数字的立方和，竟然正好等于它自己！像这样神奇的数字，我们就称之为“水仙花数”（Narcissistic Number）。

是不是很有趣？那么，除了153，还有哪些三位数是水仙花数呢？我们能不能用我们学过的编程知识，把它们都找出来呢？这节课，我们就要一起编写程序，去发现这些隐藏在数字中的“水仙花”！

## 知识点讲解

要找到水仙花数，最核心的问题就是：**如何把一个三位数的个位、十位、百位数字拆解出来？**

我们以一个三位数 `n` 为例，比如 `n = 153`。

1.  **获取个位数字**：
    个位数字就是 `n` 除以 10 的余数。
    `d1 = n % 10;`
    例如：`153 % 10 = 3`。所以，个位是3。

2.  **获取十位数字**：
    要获取十位数字，我们首先需要把 `n` 的个位去掉，也就是 `n` 除以 10。
    `n / 10` 会得到一个整数，例如 `153 / 10 = 15`。
    现在，这个 `15` 的个位就是原来 `n` 的十位。所以，我们再对 `15` 进行求余操作。
    `d2 = (n / 10) % 10;`
    例如：`(153 / 10) % 10 = 15 % 10 = 5`。所以，十位是5。

3.  **获取百位数字**：
    类似地，要获取百位数字，我们需要把 `n` 的个位和十位都去掉，也就是 `n` 连续除以两次 10。
    `n / 100` 会得到一个整数，例如 `153 / 100 = 1`。
    对于三位数来说，`n / 100` 的结果就是百位数字。
    `d3 = n / 100;`
    例如：`153 / 100 = 1`。所以，百位是1。

**总结一下拆解方法：**
对于一个三位数 `n`：
*   个位 `d1 = n % 10;`
*   十位 `d2 = (n / 10) % 10;`
*   百位 `d3 = n / 100;`

拆解出各个位之后，我们就可以进行判断了：
**判断条件**：`d1 * d1 * d1 + d2 * d2 * d2 + d3 * d3 * d3 == n`

**如何查找所有三位数的水仙花数？**
三位数从100开始，到999结束。我们可以使用 `for` 循环，从100遍历到999，对每个数字都进行拆解和判断。

```cpp
for (int i = 100; i <= 999; ++i) {
    // 拆解数字 i
    // 计算立方和
    // 判断是否等于 i
    // 如果是，就输出 i
}
```

这就是我们解决“水仙花数”问题的核心思路。是不是很清晰呢？

## 代码示例

好的，现在我们把刚才讲解的思路，用C++代码实现出来。首先，我们编写一个函数来判断一个数是否为水仙花数，然后在 `main` 函数中循环查找。

```cpp
#include <bits/stdc++.h>

using namespace std;

// 函数：判断一个三位数是否为水仙花数
// 参数 n：待判断的三位数
// 返回值：如果是水仙花数返回 true，否则返回 false
bool check(int n)
{
    // 确保是三位数
    if (n < 100 || n > 999)
    {
        return false; // 不是三位数，直接返回false
    }

    // 拆解数字的各个位
    int d1 = n % 10;        // 获取个位
    int d2 = (n / 10) % 10; // 获取十位
    int d3 = n / 100;       // 获取百位

    // 计算各个位数字的立方和
    int s = d1 * d1 * d1 + d2 * d2 * d2 + d3 * d3 * d3;

    // 判断立方和是否等于原数
    if (s == n)
    {
        return true; // 是水仙花数
    }
    else
    {
        return false; // 不是水仙花数
    }
}

int main()
{
    cout << "所有三位数的水仙花数有：" << endl;

    // 遍历所有三位数（从100到999）
    for (int i = 100; i <= 999; ++i)
    {
        // 调用 check 函数判断当前数字是否为水仙花数
        if (check(i))
        {
            // 如果是，则输出该数字
            cout << i << endl;
        }
    }

    return 0;
}
```

**代码运行结果：**
```
所有三位数的水仙花数有：
153
370
371
407
```

**代码解析：**
1.  `#include <bits/stdc++.h>` 和 `using namespace std;`：这是C++编程的常用开头，包含了大部分标准库，方便我们使用输入输出等功能。
2.  `bool check(int n)` 函数：
    *   它接收一个整数 `n` 作为参数。
    *   首先判断 `n` 是否在100到999之间，确保它是一个三位数。
    *   `int d1 = n % 10;`：使用取模运算符 `%` 得到 `n` 的个位。
    *   `int d2 = (n / 10) % 10;`：先用 `/` 运算符去掉个位，再用 `%` 得到十位。
    *   `int d3 = n / 100;`：用 `/` 运算符直接得到百位。
    *   `int s = d1 * d1 * d1 + d2 * d2 * d2 + d3 * d3 * d3;`：计算三个位数字的立方和，存储在变量 `s` 中。
    *   `if (s == n)`：比较立方和 `s` 是否等于原始数字 `n`。
    *   根据比较结果返回 `true` 或 `false`。
3.  `int main()` 函数：
    *   `cout << "所有三位数的水仙花数有：" << endl;`：打印提示信息。
    *   `for (int i = 100; i <= 999; ++i)`：这是一个 `for` 循环，变量 `i` 从100开始，每次增加1，直到999结束。它会遍历所有三位数。
    *   `if (check(i))`：在循环中，我们对每一个数字 `i` 调用 `check` 函数进行判断。
    *   `cout << i << endl;`：如果 `check(i)` 返回 `true` (说明 `i` 是水仙花数)，就打印出这个数字 `i`。

通过这段代码，我们成功地找到了所有三位数的水仙花数！是不是很有成就感呢？

### 编程题

**基础练习：判断指定数字**
编写一个C++程序，从键盘输入一个三位数 `n`，判断它是否是水仙花数，并输出相应的结果（例如“是水仙花数”或“不是水仙花数”）。

**进阶挑战：寻找四位数水仙花数**
水仙花数也可以是四位数。一个四位数的水仙花数定义是：它的每个位上的数字的四次方之和等于它本身。
例如：1634 = 1^4 + 6^4 + 3^4 + 4^4 = 1 + 1296 + 81 + 256 = 1634。
请编写一个C++程序，找出并输出所有四位数的水仙花数（从1000到9999）。

---

### 练习题目答案与解析

**选择题/填空题答案**

1.  `x / 100`
    *   **解析**：对于三位数，例如153，153 / 100 = 1，即百位数字。
2.  `(x / 10) % 10`
    *   **解析**：例如153，153 / 10 = 15，然后 15 % 10 = 5，即十位数字。
3.  `==`
    *   **解析**：判断条件是立方和“等于”原数，所以使用相等运算符 `==`。
4.  B) 370
    *   **解析**：
        *   A) 123: 1^3+2^3+3^3 = 1+8+27 = 36 != 123
        *   B) 370: 3^3+7^3+0^3 = 27+343+0 = 370 == 370 (是水仙花数)
        *   C) 456: 4^3+5^3+6^3 = 64+125+216 = 405 != 456
        *   D) 500: 5^3+0^3+0^3 = 125+0+0 = 125 != 500
5.  `for (int i = 100; i <= 999; ++i)`
    *   **解析**：三位数从最小的100开始，到最大的999结束。

**编程题参考答案**

**基础练习：判断指定数字**
```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    int n;
    cout << "请输入一个三位数：";
    cin >> n;

    // 检查输入是否为三位数
    if (n < 100 || n > 999)
    {
        cout << "输入的不是一个三位数。" << endl;
        return 0;
    }

    // 拆解数字
    int d1 = n % 10;        // 个位
    int d2 = (n / 10) % 10; // 十位
    int d3 = n / 100;       // 百位

    // 计算立方和
    int s = d1 * d1 * d1 + d2 * d2 * d2 + d3 * d3 * d3;

    // 判断并输出结果
    if (s == n)
    {
        cout << n << " 是水仙花数。" << endl;
    }
    else
    {
        cout << n << " 不是水仙花数。" << endl;
    }

    return 0;
}
```

**进阶挑战：寻找四位数水仙花数**
```cpp
#include <bits/stdc++.h>

using namespace std;

// 函数：判断一个四位数是否为水仙花数
bool check_four_digit(int n)
{
    // 确保是四位数
    if (n < 1000 || n > 9999)
    {
        return false;
    }

    // 拆解数字的各个位
    int d1 = n % 10;          // 个位
    int d2 = (n / 10) % 10;   // 十位
    int d3 = (n / 100) % 10;  // 百位
    int d4 = n / 1000;        // 千位

    // 计算各个位数字的四次方和
    // 注意：d*d*d*d 也可以写成 d*d*d*d，为了避免使用 pow 函数
    int s = d1 * d1 * d1 * d1 +
            d2 * d2 * d2 * d2 +
            d3 * d3 * d3 * d3 +
            d4 * d4 * d4 * d4;

    // 判断四次方和是否等于原数
    if (s == n)
    {
        return true;
    }
    else
    {
        return false;
    }
}

int main()
{
    cout << "所有四位数的水仙花数有：" << endl;

    // 遍历所有四位数（从1000到9999）
    for (int i = 1000; i <= 9999; ++i)
    {
        // 调用 check_four_digit 函数判断当前数字是否为水仙花数
        if (check_four_digit(i))
        {
            // 如果是，则输出该数字
            cout << i << endl;
        }
    }

    return 0;
}
```

8208
