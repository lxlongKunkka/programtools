## 知识点讲解

要让计算机帮我们拆解数字，我们需要用到两个神奇的数学运算符：**整数除法 `/`** 和 **取模 `%`**。

### 1. 回顾 `/` 和 `%` 运算符

*   **整数除法 `/`**：
    *   在 C++ 中，如果两个整数相除，结果也会是整数，它会**自动舍弃小数部分**。
    *   例如：`10 / 3 = 3` (不是 3.333...)，`123 / 10 = 12`。
    *   它的作用就像是“分蛋糕”，把蛋糕分成几份，每份有多少，不考虑剩下的。

*   **取模 `%`**：
    *   取模运算符用来计算**除法的余数**。
    *   例如：`10 % 3 = 1` (10 除以 3 商 3 余 1)。
    *   `123 % 10 = 3` (123 除以 10 商 12 余 3)。
    *   它的作用就像是“看剩下多少”，只关心分完后还剩下多少。

### 2. 拆解个位数字

我们要取出 `n` 的个位数字，想想看，一个数除以 10，余数是什么？是不是正好就是它的个位？
*   例如：`123 % 10`，123 分成 10 份，每份 12，还剩下 `3`。
*   所以，**个位 = `n % 10`**。

### 3. 拆解十位数字

要取出 `n` 的十位数字，我们不能直接 `n % 100` 或者 `n % 10`，那样会出错。
我们需要先“去掉”个位，把这个数变成一个“没有个位”的数。怎么去掉个位呢？
*   把 `n` **整除 10**！
*   例如：`123 / 10 = 12`。现在 `12` 的个位，是不是就是原来 `123` 的十位呢？是的！
*   所以，我们先 `n / 10` 得到一个新的数，再对这个新数取 `10` 的模，就能得到十位数字了。
*   **十位 = `(n / 10) % 10`**。

### 4. 拆解百位数字

要取出 `n` 的百位数字，最简单的方法就是把 `n` **整除 100**。
*   例如：`123 / 100 = 1`。
*   所以，**百位 = `n / 100`**。

### 总结一下：
假设有一个三位数 `n`：
*   **个位数字 `a` = `n % 10`**
*   **十位数字 `b` = `(n / 10) % 10`**
*   **百位数字 `c` = `n / 100`**

理解了这些，我们就能让计算机轻松地完成数字拆解任务啦！

## 代码示例

现在，我们来编写一个 C++ 程序，输入一个三位数，然后输出它的个位、十位、百位数字。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    // 定义一个整数变量 n，用来存储用户输入的三位数
    int n;

    // 提示用户输入一个三位数
    cout << "请输入一个三位数：";
    // 从键盘读取用户输入，并存储到变量 n 中
    cin >> n;

    // 计算个位数字
    // 任何数除以10的余数就是它的个位
    int a = n % 10;

    // 计算十位数字
    // 先将 n 整除 10，去掉个位，得到一个新的数
    // 再对这个新的数取 10 的模，得到十位
    int b = (n / 10) % 10;

    // 计算百位数字
    // 将 n 整除 100，直接得到百位
    int c = n / 100;

    // 输出计算结果
    cout << "你输入的数字 " << n << " 的个位是：" << a << endl;
    cout << "你输入的数字 " << n << " 的十位是：" << b << endl;
    cout << "你输入的数字 " << n << " 的百位是：" << c << endl;

    return 0;
}
```

**运行示例：**
如果你输入 `456`，程序会输出：
```
请输入一个三位数：456
你输入的数字 456 的个位是：6
你输入的数字 456 的十位是：5
你输入的数字 456 的百位是：4
```

是不是很神奇？计算机按照我们的指令，准确地分出了每一位数字！

## 练习题目

### 选择题 & 填空题

1.  在 C++ 中，表达式 `25 % 10` 的结果是（ ）。
    A. 2
    B. 5
    C. 2.5
    D. 0
2.  要从一个整数 `n` 中取出它的个位数字，正确的 C++ 表达式是（ ）。
    A. `n / 10`
    B. `n % 10`
    C. `n - 10`
    D. `n * 10`
3.  要从一个三位数 `n` 中取出它的十位数字，正确的 C++ 表达式是（ ）。
    A. `n / 10 % 10`
    B. `n % 100 / 10`
    C. `n % 10`
    D. `n / 100`
4.  如果 `n = 789`，那么它的百位数字是 `____`，十位数字是 `____`，个位数字是 `____`。
5.  如果 `n = 305`，那么 `(n / 10) % 10` 的结果是 `____`。

---

### 答案与解析

1.  **答案：B. 5**
    **解析：** `25 % 10` 表示 25 除以 10 的余数。25 除以 10 等于 2 余 5，所以结果是 5。
2.  **答案：B. `n % 10`**
    **解析：** 任何整数除以 10 的余数就是它的个位数字。
3.  **答案：A. `n / 10 % 10`**
    **解析：**
    *   `n / 10` 会先将 `n` 的个位去掉（例如 123 变成 12）。
    *   然后对这个结果 `12 % 10`，就能得到原数的十位（12 除以 10 余 2，即 2）。
    *   选项 B `n % 100 / 10` 也可以得到十位（例如 123 % 100 = 23，23 / 10 = 2），但 `n / 10 % 10` 是更常用和直观的方法。
4.  **答案：百位数字是 7，十位数字是 8，个位数字是 9。**
    **解析：**
    *   百位：`789 / 100 = 7`
    *   十位：`(789 / 10) % 10 = 78 % 10 = 8`
    *   个位：`789 % 10 = 9`
5.  **答案：0**
    **解析：**
    *   `n = 305`
    *   `n / 10 = 305 / 10 = 30` (整数除法，舍弃小数)
    *   `30 % 10 = 0` (30 除以 10 余 0)
    *   所以，结果是 0。这说明 305 的十位数字是 0。
