## 深度知识点讲解

### 1. 什么是数位？

我们平时用的数字，都是由0到9这十个阿拉伯数字组成的。每个数字在数中占据的位置，就叫做“数位”。
比如数字 `123`：
*   `3` 在个位，表示 `3` 个一。
*   `2` 在十位，表示 `2` 个十。
*   `1` 在百位，表示 `1` 个百。

所以 `123` 其实是 `1 * 100 + 2 * 10 + 3 * 1`。
每个数位上的数字都有自己的“身份”和“价值”！

### 2. 计算机的“数字剥离术”：模运算 `%` 和 整除 `/`

计算机不会“看”，但它会“算”！我们可以利用两种特殊的运算，把数字一个一个地“揪”出来。

#### 2.1 揪出个位数字：模运算 `% 10`

还记得我们学过的**取模运算符 `%`** 吗？它用来计算两数相除的余数。
*   `10 % 3` 等于 `1` (10除以3，商3余1)。
*   `7 % 2` 等于 `1` (7除以2，商3余1)。

那么，如果一个数除以 `10`，它的余数会是什么呢？
*   `123 % 10`： 123 除以 10，商是 12，余数是 `3`。
*   `45 % 10`： 45 除以 10，商是 4，余数是 `5`。
*   `7 % 10`： 7 除以 10，商是 0，余数是 `7`。

**发现了吗？** 任何一个整数，它除以 `10` 的余数，就是它自己的**个位数字**！
这就像我们有一堆糖果，把它们每10个装一包，最后袋子里剩下零散的几颗，就是个位数！

所以，要取出数字 `n` 的个位，我们只需要 `n % 10`。

#### 2.2 扔掉个位数字：整除运算 ` / 10`

学过**整除运算符 `/`** 吧？它用来计算两数相除的商，并且会**自动丢弃小数部分**（只保留整数部分）。
*   `10 / 3` 等于 `3` (10除以3，商3余1，丢弃余数1)。
*   `7 / 2` 等于 `3` (7除以2，商3余1，丢弃余数1)。

那么，如果一个数整除 `10`，结果会是什么呢？
*   `123 / 10`： 123 除以 10，商是 `12`。
*   `45 / 10`： 45 除以 10，商是 `4`。
*   `7 / 10`： 7 除以 10，商是 `0`。

**发现了吗？** 任何一个整数，它整除 `10` 的结果，就是它**去掉了个位数字后剩下的部分**！
这就像我们把那堆糖果每10个装一包后，把零散的几颗糖（个位）扔掉，剩下的就是装好包的糖果（去掉个位后的数字）。

所以，要去掉数字 `n` 的个位，我们只需要 `n = n / 10`。

### 3. 神奇的循环拆解大法：从右往左“剥洋葱”

现在我们有了两个魔法：`% 10` 取个位，` / 10` 扔个位。如果我们要把一个多位数的每个数字都拆出来，怎么办呢？我们可以重复使用这两个魔法！

我们以数字 `123` 为例，看看它是怎么被“剥”出来的：

**步骤 1：处理 `123`**
*   取出个位：`123 % 10` 得到 `3`。 (我们得到了第一个数字！)
*   去掉个位：`123 / 10` 得到 `12`。 (原来的数变成了 `12`)

**步骤 2：处理 `12`**
*   取出个位：`12 % 10` 得到 `2`。 (我们得到了第二个数字！)
*   去掉个位：`12 / 10` 得到 `1`。 (原来的数变成了 `1`)

**步骤 3：处理 `1`**
*   取出个位：`1 % 10` 得到 `1`。 (我们得到了第三个数字！)
*   去掉个位：`1 / 10` 得到 `0`。 (原来的数变成了 `0`)

**停止！** 当我们的数字变成 `0` 的时候，就说明所有的数字都被我们“剥”出来啦！
这个重复的过程，是不是很适合用 `while` 循环来完成呢？

**循环条件：`while (n > 0)`**
只要数字 `n` 还没有变成 `0`，我们就继续剥！

**关键点：**
*   这个方法是从**右往左**，也就是从**个位**开始，依次取出数字的。
*   每次循环，我们都先用 `% 10` 得到当前数的个位，然后用 ` / 10` 更新这个数，为下一次循环做准备。

### 4. 常见误区：

*   **忘记更新数字 `n`**：如果只取了 `n % 10`，但没有 `n = n / 10`，那么 `n` 的值就不会改变，循环会一直进行下去，变成死循环！
*   **循环条件错误**：如果写成 `while (n != 0)` 也可以，但 `n > 0` 更直观，因为我们处理的通常是正整数。
*   **数字的顺序**：记住，这种方法是**从右往左**（从个位到最高位）取出数字的。如果你需要从左往右的顺序，就需要一些额外的处理（比如把数字存起来再倒序输出，或者利用数学方法）。

## 典型例题精讲

### 例题 1：求一个整数的各位数字之和

**题目描述**：
输入一个正整数 `n` (例如 `n = 12345`)，计算并输出它所有数字的和。

**输入样例**：
`12345`

**输出样例**：
`15` (因为 `1 + 2 + 3 + 4 + 5 = 15`)

**思路分析**：
1.  我们要把 `n` 的每个数字都取出来，然后累加到一个变量 `sum` 里。
2.  使用 `while (n > 0)` 循环，每次循环：
    *   用 `d = n % 10` 取出当前 `n` 的个位数字 `d`。
    *   将 `d` 加到 `sum` 中：`sum = sum + d`。
    *   用 `n = n / 10` 去掉 `n` 的个位，准备处理下一个数字。
3.  当循环结束（`n` 变为 `0`）时，`sum` 中就保存了所有数字的和。

**代码实现**：

```cpp
#include <iostream> // 引入输入输出库

int main() {
    int n; // 定义整数变量 n，用来存储输入的数字
    std::cin >> n; // 从键盘读取一个整数到 n

    int sum = 0; // 定义一个变量 sum，用来累加各位数字的和，初始值为 0

    // 当 n 大于 0 时，说明还有数字没有被拆解完，继续循环
    while (n > 0) {
        int d = n % 10; // 取出 n 的个位数字，赋值给 d
        // 比如 n=123时，d=3
        // 比如 n=12时，d=2
        // 比如 n=1时，d=1

        sum = sum + d; // 将取出的数字 d 加到 sum 中
        // 第一次循环：sum = 0 + 3 = 3
        // 第二次循环：sum = 3 + 2 = 5
        // 第三次循环：sum = 5 + 1 = 6

        n = n / 10; // 移除 n 的个位数字，更新 n 的值，准备下一次循环
        // 第一次循环：n = 123 / 10 = 12
        // 第二次循环：n = 12 / 10 = 1
        // 第三次循环：n = 1 / 10 = 0
    }

    std::cout << sum << std::endl; // 输出最终的数字之和

    return 0; // 程序结束
}
```

**复杂度分析**：
*   **时间复杂度**：循环的次数取决于数字 `n` 的位数。如果 `n` 有 `k` 位，那么循环会执行 `k` 次。对于一个 `k` 位的数字，`k` 大约是 `log10(n)`。所以时间复杂度是 `O(log n)`。对于 GESP 二级的题目，通常 `n` 不会太大（比如在 `int` 范围内，最多10位），这个速度非常快。
*   **空间复杂度**：我们只使用了几个变量 `n`, `sum`, `d` 来存储数据，这些变量的数量不会随着 `n` 的大小而变化，所以空间复杂度是 `O(1)` (常数空间)。

### 例题 2：统计一个整数中偶数数字的个数

**题目描述**：
输入一个正整数 `n` (例如 `n = 123456`)，统计并输出它所有数字中是偶数的数字有多少个。

**输入样例**：
`123456`

**输出样例**：
`3` (因为 `2`、`4`、`6` 是偶数，共3个)

**思路分析**：
1.  同样，我们需要把 `n` 的每个数字都取出来。
2.  定义一个计数器 `count`，初始为 `0`。
3.  在 `while (n > 0)` 循环中，每次取出数字 `d` 后：
    *   判断 `d` 是否为偶数。一个数是偶数的条件是 `d % 2 == 0`。
    *   如果是偶数，就让 `count` 加 `1`。
    *   最后，用 `n = n / 10` 去掉个位。
4.  循环结束后，输出 `count`。

**代码实现**：

```cpp
#include <iostream>

int main() {
    int n;
    std::cin >> n;

    int count = 0; // 定义计数器，初始为 0

    while (n > 0) {
        int d = n % 10; // 取出个位数字

        // 判断这个数字是不是偶数
        if (d % 2 == 0) { // 如果 d 除以 2 的余数是 0，说明 d 是偶数
            count = count + 1; // 偶数计数器加 1
        }

        n = n / 10; // 去掉个位数字
    }

    std::cout << count << std::endl; // 输出偶数数字的个数

    return 0;
}
```

**复杂度分析**：
*   **时间复杂度**：与例题1相同，循环次数取决于 `n` 的位数，为 `O(log n)`。
*   **空间复杂度**：同样只使用了常数个变量，为 `O(1)`。

    #include <iostream>
    #include <cmath> // 用于 pow 函数，计算10的幂

    int main() {
        int n;
        std::cin >> n;

        if (n < 10) { // 如果是1位数，直接输出
            std::cout << n << std::endl;
            return 0;
        }

        int last = n % 10; // 取出个位
        int temp = n;
        int count = 0; // 记录位数
        while (temp > 0) {
            temp /= 10;
            count++;
        }

        int first = n / static_cast<int>(pow(10, count - 1)); // 取出最高位
        
        // 构造中间部分：去掉首位和个位
        int middle = n % static_cast<int>(pow(10, count - 1)); // 去掉最高位
        middle = middle / 10; // 再去掉个位

        // 重新组合数字
        // 新的最高位 * 10^(count-1) + 中间部分 * 10^1 + 新的个位
        long long newN = static_cast<long long>(last) * static_cast<long long>(pow(10, count - 1)) + 
                         static_cast<long long>(middle) * 10 + 
                         first;

        std::cout << newN << std::endl;

        return 0;
    }
    ```

我们用了哪两个运算符来“剥洋葱”？（`% 10` 和 ` / 10`）
`% 10` 的作用是什么？（取出个位）
` / 10` 的作用是什么？（去掉个位）
这个剥洋葱的过程，我们用什么循环结构来完成？（`while (n > 0)`）

**作业布置**：
1.  **必做题**：
    输入一个正整数 `n`，统计并输出它所有数字中是奇数的数字有多少个。
    *   例如：输入 `12345`，输出 `3` (因为 `1`、`3`、`5` 是奇数)。
2.  **选做题**：
    输入一个正整数 `n`，判断它是否是“幸运数”。如果一个数的每个数字都是偶数，那么它就是幸运数。
    *   例如：输入 `246`，输出 `是`。
    *   例如：输入 `123`，输出 `否`。
3.  **预习与思考**：
    我们今天学习的数位分离是从右往左进行的。如果我想从左往右（从最高位开始）一个一个地获取数字，你觉得有什么办法可以实现吗？（可以尝试思考，不需要代码，下节课我们会继续探讨更复杂的数位处理方法）。
