## 知识点讲解

### 1. 数组回顾：数据小管家

还记得数组是什么吗？它就像一个“数据仓库”，可以存储**一系列相同类型**的数据。

*   **声明数组**：告诉计算机我们要创建多大的仓库。
    `int a[5];` // 声明一个名为 `a` 的整数数组，可以存放5个整数。
*   **数组下标**：每个柜子都有一个唯一的编号，我们称之为“下标”（index）。
    **特别注意：在C++中，数组的下标总是从 `0` 开始！**
    所以，一个有5个元素的数组 `a`，它的元素分别是 `a[0], a[1], a[2], a[3], a[4]`。
    最后一个元素的下标是 `数组长度 - 1`。

### 2. 数组遍历：逐个拜访

“遍历”是什么意思呢？简单来说，就是**挨个访问数组中的每一个元素**。就像你要检查每个储物柜里放了什么东西一样。

为什么要遍历呢？
*   **打印所有元素**：看看数组里都存了些什么。
*   **查找特定元素**：在数组中找到某个我们想要的值。
*   **修改元素值**：更新数组中某个位置的数据。
*   **统计或计算**：比如计算所有元素的总和、平均值、最大值、最小值等等。

那么，如何实现遍历呢？我们最常用的工具就是 **`for` 循环**！

```cpp
// 假设数组 a 有 n 个元素
for (int i = 0; i < n; i++)
{
    // 在这里，a[i] 就是当前访问到的元素
    // 我们可以对 a[i] 进行操作，比如打印、加和等
}
```

**重点解释：**
*   `int i = 0;`：循环变量 `i` 从 `0` 开始，因为数组下标从 `0` 开始。
*   `i < n;`：循环条件是 `i` 小于数组的长度 `n`。这意味着 `i` 会取到 `0, 1, 2, ..., n-1`。这正好覆盖了数组的所有合法下标！
*   `i++`：每次循环结束后，`i` 自动加1，去访问下一个元素。

### 3. 数组求和：累积财富

有了遍历的工具，求和就变得非常简单啦！求和的核心思想是：

1.  **准备一个“求和器”**：声明一个变量（比如 `s`），用来存放累加的结果。一开始，这个“求和器”里的值应该是 `0`。
2.  **挨个取出元素**：使用 `for` 循环遍历数组，每次循环都取出当前元素的值。
3.  **加到求和器里**：把取出的元素值加到“求和器” `s` 中。

```cpp
int s = 0; // 初始化求和变量，非常重要！
// 假设数组 a 有 n 个元素
for (int i = 0; i < n; i++)
{
    s = s + a[i]; // 或者更简洁地写成 s += a[i];
}
// 循环结束后，变量 s 中就存放着数组所有元素的总和了
```

同学们，是不是觉得数组和循环结合起来非常强大呢？

## 代码示例

### 示例一：输入并打印数组元素

这个例子会让你输入 `n` 个整数，并将它们存储到数组中，然后再把它们一个一个地打印出来。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n; // 声明一个整数变量 n，用于存储数组的长度
    cin >> n; // 从键盘读取数组的长度

    int a[100]; // 声明一个足够大的整数数组 a，这里假设最大不会超过100个元素

    // 循环遍历，将用户输入的 n 个整数存入数组
    cout << "请输入 " << n << " 个整数：" << endl;
    for (int i = 0; i < n; i++)
    {
        cin >> a[i]; // 将输入的第 i+1 个数存入 a[i]
    }

    // 循环遍历，打印数组中的所有元素
    cout << "数组中的元素是：" << endl;
    for (int i = 0; i < n; i++)
    {
        cout << a[i] << " "; // 打印 a[i]，并在每个数后面加一个空格
    }
    cout << endl; // 打印完所有元素后换行

    return 0;
}
```

### 示例二：计算数组元素的总和

这个例子会在输入 `n` 个整数并存入数组后，计算这些整数的总和，并打印出来。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n; // 声明一个整数变量 n，用于存储数组的长度
    cin >> n; // 从键盘读取数组的长度

    int a[100]; // 声明一个足够大的整数数组 a
    int s = 0;  // 声明并初始化求和变量 s 为 0

    // 循环遍历，将用户输入的 n 个整数存入数组，并同时累加求和
    cout << "请输入 " << n << " 个整数：" << endl;
    for (int i = 0; i < n; i++)
    {
        cin >> a[i];    // 将输入的数存入 a[i]
        s = s + a[i];   // 将当前元素 a[i] 加到总和 s 中
        // 也可以写成 s += a[i];
    }

    // 打印计算出的总和
    cout << "数组所有元素的总和是：" << s << endl;

    return 0;
}
```

**思考题：** 如果我们想在第二个例子中，先输入所有元素，然后分两次循环，第一次存入，第二次求和，代码应该怎么写呢？（其实就是把示例一和示例二的代码合并一下，把求和的循环放在输入循环之后）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n;
    cin >> n;

    int a[100];
    cout << "请输入 " << n << " 个整数：" << endl;
    for (int i = 0; i < n; i++)
    {
        cin >> a[i];
    }

    int s = 0; // 求和变量在所有元素输入完成后再初始化也可以
    for (int i = 0; i < n; i++)
    {
        s = s + a[i];
    }

    cout << "数组所有元素的总和是：" << s << endl;

    return 0;
}
```
这两种写法都是正确的，同学们可以根据具体需求选择。在求和这种场景下，在输入时就累加会更简洁一些，因为它不需要额外的循环。

### 练习题目答案与解析

**选择题/填空题答案：**

1.  **B. 0 到 9**
    *   **解析：** 数组下标从0开始，对于长度为10的数组，合法下标是 `0, 1, ..., 9`。

2.  **C. `for (int i = 0; i < N; i++) { /* ... */ }`**
    *   **解析：** 数组下标从0开始，到 `N-1` 结束。`i = 0` 访问第一个元素，`i < N` 确保 `i` 的最大值是 `N-1`，正好访问到最后一个元素。
        *   A 漏掉了 `arr[0]`。
        *   B 会访问到 `arr[N]`，这是越界访问。
        *   D 漏掉了 `arr[0]` 和 `arr[N-1]`。

3.  **C. 初始化求和变量 `s`，确保它从零开始累加，避免脏数据。**
    *   **解析：** 如果不初始化，变量 `s` 中可能会包含内存中残留的随机值（脏数据），导致最终的求和结果不正确。

4.  **100**
    *   **解析：** 数组 `a` 中的元素依次是 10, 20, 30, 40。循环会将它们累加到 `s` 中：
        *   `s = 0 + 10 = 10`
        *   `s = 10 + 20 = 30`
        *   `s = 30 + 30 = 60`
        *   `s = 60 + 40 = 100`
        最终输出 `100`。

5.  **5**
    *   **解析：** 对于长度为5的数组 `d`，合法下标是 `0, 1, 2, 3, 4`。循环条件 `i < 5` 正好能让 `i` 取到这些值。
