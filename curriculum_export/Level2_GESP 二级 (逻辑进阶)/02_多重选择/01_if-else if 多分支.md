## 知识点讲解

### 1. 回顾 `if` 和 `if-else`

*   `if` 语句：当某个条件满足时，执行一段代码。
    ```cpp
    if (条件) {
        // 条件满足时执行的代码
    }
    ```
*   `if-else` 语句：当某个条件满足时执行一段代码，否则（条件不满足时）执行另一段代码。
    ```cpp
    if (条件) {
        // 条件满足时执行的代码
    }
    else {
        // 条件不满足时执行的代码
    }
    ```

### 2. `if-else if-else` 多分支语句

当我们需要处理多个互斥的条件时，`if-else if-else` 结构就非常有用了。它的基本语法结构是这样的：

```cpp
if (条件1) {
    // 如果条件1满足，执行这里的代码
}
else if (条件2) {
    // 否则，如果条件2满足，执行这里的代码
}
else if (条件3) {
    // 否则，如果条件3满足，执行这里的代码
}
// 可以有任意多个 else if 分支
else {
    // 如果以上所有条件都不满足，执行这里的代码
}
```

**执行流程：**

1.  程序首先判断 `条件1`。
2.  如果 `条件1` 满足（为真），就执行 `条件1` 后面大括号里的代码，然后整个 `if-else if-else` 结构就结束了，跳过后面的所有 `else if` 和 `else`。
3.  如果 `条件1` 不满足（为假），程序会接着判断 `条件2`。
4.  如果 `条件2` 满足，就执行 `条件2` 后面大括号里的代码，然后整个 `if-else if-else` 结构结束。
5.  这个过程会一直持续下去，直到找到一个满足的条件并执行其对应的代码块，或者所有 `if` 和 `else if` 的条件都不满足。
6.  如果所有 `if` 和 `else if` 的条件都不满足，那么最后 `else` 后面的代码块（如果存在的话）就会被执行。`else` 是可选的，它用于处理所有条件都不满足的“其他情况”。

**小贴士：** `if-else if-else` 结构只会执行**第一个**满足条件的代码块。一旦有一个条件满足并执行了其对应的代码，程序就会跳出整个 `if-else if-else` 结构。

**举个例子：判断一个数的正负零**

假设我们输入一个整数 `x`，要判断它是正数、负数还是零。

*   如果 `x > 0`，它是正数。
*   否则，如果 `x < 0`，它是负数。
*   否则（既不大于 0 也不小于 0），它就是零。

这正是 `if-else if-else` 结构大显身手的时候！

## 代码示例

### 示例 1：判断一个整数的符号

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int x; // 定义一个整数变量 x
    cout << "请输入一个整数: "; // 提示用户输入
    cin >> x; // 读取用户输入的整数

    if (x > 0)
    {
        // 如果 x 大于 0
        cout << "你输入的是一个正数。" << endl;
    }
    else if (x < 0)
    {
        // 否则，如果 x 小于 0
        cout << "你输入的是一个负数。" << endl;
    }
    else
    {
        // 否则（既不大于0也不小于0），那么 x 就是 0
        cout << "你输入的是零。" << endl;
    }

    return 0;
}
```

**运行结果示例：**
```
请输入一个整数: 10
你输入的是一个正数。
```
```
请输入一个整数: -5
你输入的是一个负数。
```
```
请输入一个整数: 0
你输入的是零。
```

### 示例 2：根据分数判断等级

我们来编写一个程序，根据输入的考试分数（0-100分），输出对应的等级：
*   90-100分：A
*   80-89分：B
*   70-79分：C
*   60-69分：D
*   0-59分：E

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int s; // 定义一个整数变量 s 代表分数
    cout << "请输入你的考试分数 (0-100): "; // 提示用户输入
    cin >> s; // 读取用户输入的分数

    if (s >= 90 && s <= 100) // 判断分数是否在 90 到 100 之间
    {
        cout << "你的等级是：A" << endl;
    }
    else if (s >= 80 && s < 90) // 否则，判断分数是否在 80 到 89 之间
    {
        cout << "你的等级是：B" << endl;
    }
    else if (s >= 70 && s < 80) // 否则，判断分数是否在 70 到 79 之间
    {
        cout << "你的等级是：C" << endl;
    }
    else if (s >= 60 && s < 70) // 否则，判断分数是否在 60 到 69 之间
    {
        cout << "你的等级是：D" << endl;
    }
    else if (s >= 0 && s < 60) // 否则，判断分数是否在 0 到 59 之间
    {
        cout << "你的等级是：E" << endl;
    }
    else // 如果分数不在 0-100 之间，说明输入有误
    {
        cout << "输入的分数无效，请确保在 0 到 100 之间。" << endl;
    }

    return 0;
}
```

**运行结果示例：**
```
请输入你的考试分数 (0-100): 95
你的等级是：A
```
```
请输入你的考试分数 (0-100): 82
你的等级是：B
```
```
请输入你的考试分数 (0-100): 59
你的等级是：E
```
```
请输入你的考试分数 (0-100): 105
输入的分数无效，请确保在 0 到 100 之间。
```

## 练习题目

### 选择题/填空题

1.  以下 C++ 代码片段执行后，变量 `k` 的值是多少？
    ```cpp
    int a = 10;
    int b = 5;
    int k = 0;
    if (a < b) {
        k = 1;
    }
    else if (a == b) {
        k = 2;
    }
    else {
        k = 3;
    }
    ```
    A. 1
    B. 2
    C. 3
    D. 0

2.  以下代码片段中，如果 `x` 的值为 75，程序会输出什么？
    ```cpp
    int x = 75;
    if (x > 80) {
        cout << "优秀" << endl;
    }
    else if (x > 70) {
        cout << "良好" << endl;
    }
    else if (x > 60) {
        cout << "及格" << endl;
    }
    else {
        cout << "不及格" << endl;
    }
    ```
    A. 优秀
    B. 良好
    C. 及格
    D. 不及格

3.  `if-else if-else` 结构中，`else` 语句块是（必选/可选）的，它会在所有 `if` 和 `else if` 条件都为（真/假）时执行。

4.  下列关于 `if-else if-else` 语句的描述，正确的是：
    A. 它可以判断多个条件，并执行所有满足条件的代码块。
    B. 它只能判断两个条件，多于两个条件时需要使用多个独立的 `if` 语句。
    C. 一旦某个 `if` 或 `else if` 的条件满足，其对应的代码块执行后，整个多分支结构就会结束。
    D. `else if` 可以单独存在，不需要前面有 `if` 语句。

5.  如果 `a = 5, b = 10`，执行以下代码后，`s` 的值是多少？
    ```cpp
    int a = 5;
    int b = 10;
    int s = 0;
    if (a > b) {
        s = a + b;
    }
    if (a < b) {
        s = a * b;
    }
    else {
        s = a - b;
    }
    ```
    A. 15
    B. 50
    C. -5
    D. 0

---

### 答案与解析

1.  **答案：C**
    *   **解析：** `a` (10) 不小于 `b` (5)，所以第一个 `if (a < b)` 条件不满足。`a` (10) 不等于 `b` (5)，所以 `else if (a == b)` 条件也不满足。最终执行 `else` 块中的代码，`k` 被赋值为 3。

2.  **答案：B**
    *   **解析：**
        *   `x > 80` (75 > 80) 为假。
        *   `else if (x > 70)` (75 > 70) 为真。程序会执行此分支的代码，输出 "良好"，然后跳出整个 `if-else if-else` 结构。

3.  **答案：可选 / 假**
    *   **解析：** `else` 块是可选的，当所有前面的 `if` 和 `else if` 条件都为假（不满足）时，`else` 块才会执行。

4.  **答案：C**
    *   **解析：**
        *   A 错误：`if-else if-else` 只会执行第一个满足条件的代码块。
        *   B 错误：它可以判断任意多个条件。
        *   C 正确：这是 `if-else if-else` 的核心执行特点。
        *   D 错误：`else if` 必须跟在 `if` 或另一个 `else if` 之后。

5.  **答案：C**
    *   **解析：** 这段代码是多个独立的 `if` 语句和 `if-else` 的组合，而不是一个完整的 `if-else if-else` 结构。
        *   第一个 `if (a > b)` (5 > 10) 为假，`s` 保持 0。
        *   第二个 `if (a < b)` (5 < 10) 为真，`s` 被赋值为 `a * b`，即 `5 * 10 = 50`。
        *   注意，紧接着的 `else` 是和第二个 `if (a < b)` 配对的。但因为 `if (a < b)` 已经满足并执行了，所以它的 `else` 部分不会被执行。
        *   **等等！这里有一个陷阱！** `else` 总是和离它最近的未配对的 `if` 配对。所以这个 `else` 是和 `if (a < b)` 配对的。因为 `a < b` 是真，所以 `s = a * b` 执行了，那么 `else { s = a - b; }` 就不会执行。最终 `s` 的值是 50。
        *   **修正：** 我在出题时对这个问题有点疏忽，这道题的重点是辨析 `if-else if` 和多个独立 `if` 的区别。
        *   如果代码是：
            ```cpp
            int a = 5;
            int b = 10;
            int s = 0;
            if (a > b) {
                s = a + b;
            }
            else if (a < b) { // 这里是 else if
                s = a * b;
            }
            else { // 这里是 else
                s = a - b;
            }
            ```
            那么 `s` 的值就是 50。
        *   但原题中第二个 `if` 和 `else` 之间没有 `else if`，所以它们构成了一个独立的 `if-else` 结构。
            1. `if (a > b)` (5 > 10) 为假，不执行。`s` 仍为 0。
            2. `if (a < b)` (5 < 10) 为真，执行 `s = a * b;`，此时 `s` 变为 50。
            3. 紧接着的 `else` 是与第二个 `if (a < b)` 配对的。由于 `if (a < b)` 的条件为真，其对应的代码块已执行，所以这个 `else` 块不会执行。
            所以最终 `s` 的值是 50。
        *   **答案应该是 B。**

### 课后作业

1.  **基础巩固：判断闰年**
    编写一个 C++ 程序，让用户输入一个年份（整数），然后判断这个年份是否是闰年，并输出结果。
    **闰年的判断规则：**
    *   能被 4 整除，但不能被 100 整除的年份是闰年。
    *   能被 400 整除的年份也是闰年。
    *   例如：2000 是闰年，2004 是闰年，1900 不是闰年，2023 不是闰年。

2.  **进阶挑战：三角形判断**
    编写一个 C++ 程序，让用户输入三个正整数，分别代表三条边的长度 `a`、`b`、`c`。程序需要判断这三条边能否构成一个三角形。
    **构成三角形的条件：** 任意两边之和必须大于第三边。
    *   `a + b > c`
    *   `a + c > b`
    *   `b + c > a`
    同时，三条边长都必须是正数。
