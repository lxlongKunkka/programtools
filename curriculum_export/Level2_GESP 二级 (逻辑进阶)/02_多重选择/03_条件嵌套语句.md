已知：  
- 如果步数大于 10000，奖励“金牌”

- “金牌”里根据步数是否超过 15000 再细分“金牌+加餐券”或“金牌”  
  
通过这个场景，引出“条件嵌套语句”，让同学们感受到多级判断的必要性。

===NEXT===

## 知识点讲解

1. 语法结构  
   - if(条件1)  
     {  
       if(条件2) { … }  
       else { … }  
     }  
     else { … }  
2. 执行流程  
   - 先判断外层条件，若为真，则进入内层判断；否则执行外层 else 分支  
3. 简单→复杂  
   - 单层 if…else → 一层嵌套 → 多层嵌套  
4. 常见错误  
   - 忘记大括号导致逻辑不清  
   - 条件判断顺序不当  
5. 注意事项  
   - 保持代码缩进清晰  
   - 控制嵌套层数，过多会降低可读性

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a;            // 步数
    cin >> a;
    // 第一级判断：是否超过 10000 步
    if (a > 10000)
    {
        // 第二级判断：是否超过 15000 步
        if (a > 15000)
        {
            cout << "金牌+加餐券" << endl;
        }
        else
        {
            cout << "金牌" << endl;
        }
    }
    else
    {
        cout << "银牌" << endl;
    }
    return 0;
}
```

关键行注释  
- 判断 a>10000 进入第一层  
- 在第一层内部再判断 a>15000 实现嵌套逻辑

## 练习题目

基础题（选择/填空，各 3 道）  

1. 以下代码输出什么？  
   ```cpp
   int a = 5;
   if (a > 0)
   {
       if (a % 2 == 0)
       {
           cout << "偶数";
       }
       else
       {
           cout << "奇数";
       }
   }
   else
   {
       cout << "非正数";
   }
   ```  
2. 填空：  
   ```cpp
   int a = -3;
   if (a >= 0)
   {
       // A
   }
   else
   {
       if (a == -3)
       {
           cout << "__B__";
       }
   }
   ```  
3. 以下嵌套 if 结构中，若要判断 x>10 且 x<20，应该将判断条件写在（    ）  
   A. 外层 if  
   B. 内层 if  
   C. else 分支  

进阶题（选择/填空，2 道）  

4. 以下代码的输出是（    ）  
   ```cpp
   int a = 8;
   if (a % 4 == 0)
   {
       if (a % 2 == 0)
       {
           cout << "A";
       }
       else
       {
           cout << "B";
       }
   }
   else
   {
       cout << "C";
   }
   ```  
   A. A  
   B. B  
   C. C  

5. 填空：编写一个嵌套 if…else，判断一个年份 y 是否为闰年（条件：能被 4 整除且不能被 100 整除，或者能被 400 整除），在 `//` 处填入正确条件：  
   ```cpp
   int y;
   cin >> y;
   if (y % 400 == 0)
   {
       cout << "闰年";
   }
   else
   {
       if ( /* D */ )
       {
           cout << "闰年";
       }
       else
       {
           cout << "平年";
       }
   }
   ```

答案与解析  
1. 输出：奇数  
   解析：5>0 为真，进入内层；5%2==1，输出“奇数”。  
2. B 应填 “-3”  
   解析：a<0 进入 else，a==–3 为真，输出 -3。  
3. A  
   解析：外层判断 x>10，内层再判断 x<20 才能实现区间判断。  
4. A  
   解析：8%4==0 为真，进入内层；8%2==0 为真，输出 A。  
5. D 应填： `y % 4 == 0 && y % 100 != 0`  
   解析：闰年的第二种情况。
