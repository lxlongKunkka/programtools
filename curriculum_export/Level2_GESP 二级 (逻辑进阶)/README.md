# Level 2：GESP 二级 (逻辑进阶)

## 教学目标
1. 熟练掌握多重分支结构与逻辑运算符。
2. 理解并掌握 `while`、`do-while` 循环及循环嵌套。
3. 初步掌握一维数组的使用。

## 核心知识点
- 语法：`if-else if`, `switch-case`, `while`, `do-while`。
- 逻辑运算：`&&` (与), `||` (或), `!` (非)。
- 循环控制：`break`, `continue`。
- 嵌套循环：双层循环的执行流程（如打印图形）。
- 数组基础：定义、索引访问、遍历。
