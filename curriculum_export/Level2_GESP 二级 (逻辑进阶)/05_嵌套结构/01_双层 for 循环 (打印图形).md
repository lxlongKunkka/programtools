## 知识点讲解

同学们，我们已经知道单层 `for` 循环可以帮我们重复执行某个动作。现在，我们来看看如何让循环“套娃”，实现更复杂的重复。

### 1. 单层 `for` 循环回顾

还记得吗？一个 `for` 循环长这样：
```cpp
for (int i = 0; i < n; ++i) {
    // 循环体，这里面的代码会重复执行 n 次
    cout << "*";
}
cout << endl; // 循环结束后换行
```
这段代码会打印 `n` 个星号，然后换行。

### 2. 双层 `for` 循环的概念

双层 `for` 循环，顾名思义，就是一个 `for` 循环里面又嵌套了另一个 `for` 循环。

它的基本结构是这样的：
```cpp
for (int i = 0; i < n; ++i) { // 外层循环，通常控制“行”
    for (int j = 0; j < m; ++j) { // 内层循环，通常控制“列”或“行内的元素”
        // 循环体，这里面的代码会重复执行
    }
    // 外层循环体，内层循环结束后执行，通常是换行
}
```

**工作原理：**
1.  外层循环开始执行。
2.  外层循环每执行一次，内层循环就会**完整地**从头到尾执行一遍。
3.  等到内层循环全部执行完毕后，外层循环才会继续下一次迭代。

**举个例子：**
如果外层循环 `i` 从 0 到 1 (共 2 次)，内层循环 `j` 从 0 到 2 (共 3 次)。
-   当 `i = 0` 时：
    -   内层循环 `j` 会依次取 0, 1, 2，执行 3 次。
-   当 `i = 1` 时：
    -   内层循环 `j` 会依次取 0, 1, 2，执行 3 次。

所以，内层循环体总共执行了 `2 * 3 = 6` 次！是不是很神奇？

### 3. 使用双层 `for` 循环打印图形

#### 3.1 打印实心矩形

现在，我们想打印一个 `n` 行 `m` 列的实心矩形，比如 `n=3, m=4`：
```
****
****
****
```
**分析：**
-   我们需要打印 `n` 行。所以，外层循环用来控制行数，从 `i = 0` 到 `n-1`。
-   每一行都需要打印 `m` 个星号。所以，内层循环用来控制列数，从 `j = 0` 到 `m-1`。
-   每打印完一行（内层循环结束），我们需要换到下一行去打印。所以，在外层循环的内部、内层循环的外部，需要一个 `cout << endl;`。

#### 3.2 打印空心矩形

接下来，我们挑战一下更有趣的图案——空心矩形！比如 `n=5, m=5`：
```
*****
*   *
*   *
*   *
*****
```
**分析：**
-   同样需要 `n` 行 `m` 列。外层循环 `i` 控制行，内层循环 `j` 控制列。
-   关键在于：什么时候打印 `*`，什么时候打印空格 ` `？
-   只有在满足以下条件之一时，才打印 `*`：
    1.  当前是第一行 (`i == 0`)。
    2.  当前是最后一行 (`i == n - 1`)。
    3.  当前是第一列 (`j == 0`)。
    4.  当前是最后一列 (`j == m - 1`)。
-   如果以上条件都不满足，说明我们正在矩形的“内部”，就应该打印空格 ` `。
-   这需要用到我们学过的 `if-else` 条件判断语句！

通过这两个例子，你会发现，双层 `for` 循环是绘制二维图形的强大工具，而 `if-else` 语句则是让图案变得多样化的“魔法棒”！

## 代码示例

### 示例 1: 打印一个 `n` 行 `m` 列的实心矩形

```cpp
#include <bits/stdc++.h> // 包含大部分常用头文件

using namespace std; // 使用标准命名空间

int main()
{
    int n, m; // 声明两个整数变量 n 和 m，分别代表行数和列数
    
    // 提示用户输入行数和列数
    cout << "请输入行数 n: ";
    cin >> n;
    cout << "请输入列数 m: ";
    cin >> m;
    
    // 外层循环：控制行数，从第 0 行到第 n-1 行
    for (int i = 0; i < n; ++i)
    {
        // 内层循环：控制列数，每行打印 m 个星号
        for (int j = 0; j < m; ++j)
        {
            cout << "*"; // 打印一个星号，不换行
        }
        cout << endl; // 一行打印完毕后，换到下一行
    }
    
    return 0; // 程序正常结束
}
```

### 示例 2: 打印一个 `n` 行 `m` 列的空心矩形

```cpp
#include <bits/stdc++.h> // 包含大部分常用头文件

using namespace std; // 使用标准命名空间

int main()
{
    int n, m; // 声明两个整数变量 n 和 m，分别代表行数和列数
    
    // 提示用户输入行数和列数
    cout << "请输入行数 n: ";
    cin >> n;
    cout << "请输入列数 m: ";
    cin >> m;
    
    // 外层循环：控制行数，从第 0 行到第 n-1 行
    for (int i = 0; i < n; ++i)
    {
        // 内层循环：控制列数，每行打印 m 个字符 (星号或空格)
        for (int j = 0; j < m; ++j)
        {
            // 判断当前位置是否应该打印星号
            // 如果是第一行 (i==0) 或最后一行 (i==n-1)
            // 或者 是第一列 (j==0) 或最后一列 (j==m-1)
            if (i == 0 || i == n - 1 || j == 0 || j == m - 1)
            {
                cout << "*"; // 打印星号
            }
            else
            {
                cout << " "; // 否则打印空格
            }
        }
        cout << endl; // 一行打印完毕后，换到下一行
    }
    
    return 0; // 程序正常结束
}
```

### 练习题目答案与解析

**选择题/填空题答案**

1.  **A**
    *   **解析**：外层循环 `i` 从 0 到 1 (2 次)，内层循环 `j` 从 0 到 2 (3 次)。每次内层循环打印 3 个 `#`，然后外层循环换行。所以会打印 2 行，每行 3 个 `#`。

2.  **B**
    *   **解析**：在 C++ 中，数组和循环索引通常从 0 开始。所以第一行是 `i == 0`，最后一行是 `i == n - 1`。同理，第一列是 `j == 0`，最后一列是 `j == n - 1`。要满足其中任意一个条件就打印 `*`，所以用 `||` (或) 连接。

3.  **B**
    *   **解析**：外层循环通常控制行，内层循环控制列，这是打印二维图形的常见模式。

4.  **C**
    *   **解析**：`cout << endl;` 的作用是输出一个换行符，让光标移动到下一行的开头。它在内层循环结束后执行，确保每行星号打印完毕后能换行。

5.  填空1：`n` (或 `3`)，填空2：`m` (或 `4`)
    *   **解析**：外层循环控制行数，需要执行 `n` 次；内层循环控制列数，需要执行 `m` 次。
