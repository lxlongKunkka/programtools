## 深度知识点讲解

### 1. 什么是最大值 (max) 和最小值 (min)？

*   **最大值 (Maximum - max)**：在一组数中，最大的那个数就叫做这组数的最大值。它比组内其他所有数都大，或者至少不比它们小。
    *   例如：在数字 {5, 2, 8, 1} 中，最大值是 8。
*   **最小值 (Minimum - min)**：在一组数中，最小的那个数就叫做这组数的最小值。它比组内其他所有数都小，或者至少不比它们大。
    *   例如：在数字 {5, 2, 8, 1} 中，最小值是 1。

### 2. 比较的秘密：为什么需要比较？

要找出最大值或最小值，我们必须进行“比较”。比较就是判断两个数之间的大小关系。在 C++ 中，我们用比较运算符来完成这个任务：
*   `>` (大于)
*   `<` (小于)
*   `>=` (大于等于)
*   `<=` (小于等于)
*   `==` (等于)
*   `!=` (不等于)

**生活类比**：想象你手里拿着两颗糖果，你想知道哪颗更大。你会怎么做？你会把它们放在一起，用眼睛观察，然后就能判断出哪颗糖果更大了。这个“用眼睛观察”的过程，就相当于计算机里的“比较”。

**怎么想到的？**
如果我们有两个数 `a` 和 `b`，怎么才能找出它们中的最大值呢？
最直接的办法就是：
*   **如果 `a` 比 `b` 大**，那 `a` 就是最大值。
*   **否则（也就是 `a` 不比 `b` 大，可能是 `a` 小于 `b`，也可能是 `a` 等于 `b`）**，那 `b` 就是最大值。

用 C++ 的 `if-else` 语句来实现就是这样：

```cpp
#include <iostream> // 引入输入输出库

int main() {
    int a = 10; // 第一个数
    int b = 20; // 第二个数
    int m;      // 用来存放最大值

    if (a > b) { // 如果 a 比 b 大
        m = a;   // 那么最大值就是 a
    } else {     // 否则（a 不大于 b，即 a 小于等于 b）
        m = b;   // 那么最大值就是 b
    }

    std::cout << "两个数中较大的数是: " << m << std::endl; // 输出结果
    return 0;
}
```

同样的道理，找最小值也是一样：
*   **如果 `a` 比 `b` 小**，那 `a` 就是最小值。
*   **否则（也就是 `a` 不比 `b` 小，可能是 `a` 大于 `b`，也可能是 `a` 等于 `b`）**，那 `b` 就是最小值。

```cpp
#include <iostream>

int main() {
    int a = 10;
    int b = 20;
    int n; // 用来存放最小值

    if (a < b) { // 如果 a 比 b 小
        n = a;   // 那么最小值就是 a
    } else {     // 否则（a 不小于 b，即 a 大于等于 b）
        n = b;   // 那么最小值就是 b
    }

    std::cout << "两个数中较小的数是: " << n << std::endl;
    return 0;
}
```

### 3. C++ 的“魔法工具”：`std::max()` 和 `std::min()`

每次都写 `if-else` 语句来找最大值或最小值，是不是有点麻烦？C++ 早就为大家准备好了方便的“魔法工具”！它们就是 `std::max()` 和 `std::min()` 函数。

*   **`std::max(a, b)`**：这个函数会比较 `a` 和 `b`，然后直接返回它们中的最大值。
*   **`std::min(a, b)`**：这个函数会比较 `a` 和 `b`，然后直接返回它们中的最小值。

**它们藏在哪里？**
这两个函数定义在 C++ 的 `<algorithm>` 头文件中。所以，在使用它们之前，我们需要在代码开头加上 `#include <algorithm>`。不过，我们通常为了方便，会直接使用 `#include <bits/stdc++.h>`，它包含了几乎所有常用的标准库，当然也包括 `<algorithm>`。

**使用示例：**

```cpp
#include <iostream>
#include <algorithm> // 引入算法库，max和min函数就藏在这里！
// 或者直接 #include <bits/stdc++.h>

int main() {
    int a = 10;
    int b = 20;

    int m = std::max(a, b); // 直接找出 a 和 b 中较大的数
    int n = std::min(a, b); // 直接找出 a 和 b 中较小的数

    std::cout << "两个数中较大的数是: " << m << std::endl;
    std::cout << "两个数中较小的数是: " << n << std::endl;

    // 还可以直接在输出时使用
    std::cout << "max(5, 12) = " << std::max(5, 12) << std::endl;
    std::cout << "min(5, 12) = " << std::min(5, 12) << std::endl;

    return 0;
}
```

是不是非常简洁方便呢？这就像你不再需要自己去“比大小”，而是直接问一个“裁判”：这两个数谁更大？谁更小？

### 4. 怎么找三个数的最大/最小值？

`std::max()` 和 `std::min()` 函数一次只能比较两个数。那如果有三个数 `a`, `b`, `c`，我们怎么找最大值呢？

**思路**：我们可以把问题分解成两步：
1.  先找出 `a` 和 `b` 中的最大值。
2.  然后把这个最大值，再和 `c` 比较，就能得到最终的三个数的最大值。

**生活类比**：三个人比身高。
1.  小明和小红比，小明赢了。
2.  小明再和小亮比，小亮赢了。
3.  所以小亮是最高的！

**C++ 代码实现：**

```cpp
#include <iostream>
#include <algorithm> // 或者 #include <bits/stdc++.h>

int main() {
    int a = 10;
    int b = 20;
    int c = 15;

    // 找出 a, b, c 中的最大值
    int max_ab = std::max(a, b);      // 先找出 a 和 b 中较大的
    int max_abc = std::max(max_ab, c); // 再用 max_ab 和 c 比较，找出最终的最大值

    // 也可以写成一行：
    // int max_abc = std::max(std::max(a, b), c);

    std::cout << "三个数中最大的数是: " << max_abc << std::endl;

    // 找出 a, b, c 中的最小值
    int min_ab = std::min(a, b);      // 先找出 a 和 b 中较小的
    int min_abc = std::min(min_ab, c); // 再用 min_ab 和 c 比较，找出最终的最小值

    // 也可以写成一行：
    // int min_abc = std::min(std::min(a, b), c);

    std::cout << "三个数中最小的数是: " << min_abc << std::endl;

    return 0;
}
```

### 5. 常见误区

*   **忘记包含头文件**：使用 `std::max()` 或 `std::min()` 时，一定要记得 `include <algorithm>` 或 `bits/stdc++.h>`。否则编译器会报错，说找不到这些函数。
*   **误以为能直接比较多个数**：`std::max(a, b, c)` 这样的写法是错误的！`std::max` 和 `std::min` 默认只能比较两个参数。如果需要比较多个，必须像上面那样，通过嵌套的方式进行。

## 典型例题精讲

### 例题 1：找出两个数中的最大值

**题目描述**
输入两个整数 `a` 和 `b`，请输出这两个数中较大的那一个。

**输入格式**
一行，包含两个整数 `a` 和 `b`，中间用空格隔开。
例如：`10 20`

**输出格式**
一行，输出较大的数。
例如：`20`

**思路分析**
1.  **读取输入**：首先，我们需要使用 `std::cin` 读取用户输入的两个整数 `a` 和 `b`。
2.  **比较大小**：然后，我们可以使用 `std::max(a, b)` 函数来直接获得 `a` 和 `b` 中的最大值。
3.  **输出结果**：最后，使用 `std::cout` 将结果输出。

**代码实现**

```cpp
#include <iostream> // 用于输入输出
#include <algorithm> // 用于使用 std::max 和 std::min 函数
// 当然，也可以直接使用 #include <bits/stdc++.h> 来包含常用库

int main() {
    // 1. 定义两个整数变量，用于存储输入的数
    int a, b;

    // 2. 读取用户输入的两个整数
    std::cin >> a >> b;

    // 3. 使用 std::max 函数找出 a 和 b 中的最大值
    int m = std::max(a, b);

    // 4. 输出结果
    std::cout << m << std::endl;

    return 0; // 程序正常结束
}
```

**复杂度分析**
*   **时间复杂度**：`O(1)`。因为无论输入的数字多大，`std::max` 函数都只需要进行一次比较操作，是一个常数时间操作。
*   **空间复杂度**：`O(1)`。程序只使用了几个固定的变量来存储数据，不随输入规模变化，所以空间占用是常数。

### 例题 2：找出三个数中的最小值

**题目描述**
输入三个整数 `a`、`b` 和 `c`，请输出这三个数中最小的那一个。

**输入格式**
一行，包含三个整数 `a`、`b` 和 `c`，中间用空格隔开。
例如：`5 12 8`

**输出格式**
一行，输出最小的数。
例如：`5`

**思路分析**
1.  **读取输入**：读取三个整数 `a`、`b` 和 `c`。
2.  **比较大小**：`std::min` 函数一次只能比较两个数。
    *   我们可以先找出 `a` 和 `b` 中的最小值，记为 `min_ab`。
    *   然后，再用 `min_ab` 和 `c` 比较，找出它们中的最小值，这个结果就是三个数中的最小值。
    *   或者，更简洁地，直接嵌套调用 `std::min(std::min(a, b), c)`。
3.  **输出结果**：输出最终的最小值。

**代码实现**

```cpp
#include <iostream>  // 用于输入输出
#include <algorithm> // 用于使用 std::max 和 std::min 函数

int main() {
    // 1. 定义三个整数变量
    int a, b, c;

    // 2. 读取用户输入的三个整数
    std::cin >> a >> b >> c;

    // 3. 使用 std::min 函数找出 a, b, c 中的最小值
    // 方法一：分两步
    // int min_ab = std::min(a, b);       // 先找出 a 和 b 中较小的
    // int min_abc = std::min(min_ab, c); // 再用 min_ab 和 c 比较

    // 方法二：嵌套调用，更简洁
    int m = std::min(std::min(a, b), c);

    // 4. 输出结果
    std::cout << m << std::endl;

    return 0;
}
```

**复杂度分析**
*   **时间复杂度**：`O(1)`。进行了两次 `std::min` 操作，每次都是常数时间比较，所以总时间复杂度是常数。
*   **空间复杂度**：`O(1)`。只使用了几个固定变量。

### 例题 3：计算两个数的和与差的绝对值，并找出其中的最大值

**题目描述**
输入两个整数 `x` 和 `y`。
请计算它们的和（`x + y`）以及它们的差的绝对值（`|x - y|`）。
然后，输出这两个计算结果中的最大值。

**提示**：`|x - y|` 表示 `x` 减 `y` 的绝对值。在 C++ 中，可以使用 `std::abs()` 函数来求绝对值，它定义在 `<cmath>` 或 `<cstdlib>` 头文件中。为了方便，`#include <bits/stdc++.h>` 也能包含。

**输入格式**
一行，包含两个整数 `x` 和 `y`，中间用空格隔开。
例如：`5 3`

**输出格式**
一行，输出计算出的最大值。
例如：`8` (因为 `5+3=8`, `|5-3|=2`, `max(8, 2)=8`)

**思路分析**
1.  **读取输入**：读取两个整数 `x` 和 `y`。
2.  **计算和**：计算 `s = x + y`。
3.  **计算差的绝对值**：计算 `d = std::abs(x - y)`。
4.  **比较并取最大值**：使用 `std::max(s, d)` 找出 `s` 和 `d` 中的最大值。
5.  **输出结果**：输出最终的最大值。

**代码实现**

```cpp
#include <iostream>  // 用于输入输出
#include <algorithm> // 用于使用 std::max 函数
#include <cmath>     // 用于使用 std::abs 函数 (求绝对值)
// 也可以直接 #include <bits/stdc++.h>

int main() {
    // 1. 定义两个整数变量
    int x, y;

    // 2. 读取用户输入的两个整数
    std::cin >> x >> y;

    // 3. 计算它们的和
    int s = x + y;

    // 4. 计算它们的差的绝对值
    // std::abs() 可以用于整数、浮点数。对于整数，它在 <cstdlib> 或 <cmath> 中。
    int d = std::abs(x - y);

    // 5. 找出和与差的绝对值中的最大值
    int m = std::max(s, d);

    // 6. 输出结果
    std::cout << m << std::endl;

    return 0;
}
```

**复杂度分析**
*   **时间复杂度**：`O(1)`。进行了加法、减法、绝对值和 `std::max` 操作，都是常数时间操作。
*   **空间复杂度**：`O(1)`。使用了几个固定变量。

## 课堂互动

1.  **思考题：** 如果我们有 5 个数，`a, b, c, d, e`，请问怎么用 `std::max()` 函数找出它们中的最大值呢？需要调用几次 `std::max()`？
    *   **提示**：可以像找三个数最大值那样，一步步来。

#### 能力提升答案
1.  ```cpp
    #include <iostream>
    #include <algorithm>

    int main() {
        int x, y, z;
        std::cin >> x >> y >> z;
        if (x % 2 == 0) { // x 是偶数
            std::cout << std::max(y, z) << std::endl;
        } else {          // x 是奇数
            std::cout << std::min(y, z) << std::endl;
        }
        return 0;
    }
    ```
2.  ```cpp
    #include <iostream>
    #include <algorithm> // 允许使用 std::sort，但这里用 max/min 嵌套实现

    int main() {
        int a, b, c;
        std::cin >> a >> b >> c;

        int max_val = std::max(std::max(a, b), c); // 找出最大值

        // 找出次大值：
        // 方法一：从三个数中去掉最大值，然后从剩下的两个数中找出较大的
        // 但这里无法直接“去掉”，我们可以通过比较来判断
        int second_max_val;
        if (a == max_val) { // 如果a是最大值，那么次大值在b和c之间
            second_max_val = std::max(b, c);
        } else if (b == max_val) { // 如果b是最大值，那么次大值在a和c之间
            second_max_val = std::max(a, c);
        } else { // 如果c是最大值，那么次大值在a和b之间
            second_max_val = std::max(a, b);
        }

        // 注意：如果存在重复的最大值，例如 10 10 5，这种方法会得到 10 和 10。
        // 如果题目要求严格的次大值（不能和最大值相等），则需要更复杂的逻辑。
        // 对于 GESP 2 级，这种处理通常是可接受的。

        std::cout << "最大值: " << max_val << std::endl;
        std::cout << "次大值: " << second_max_val << std::endl;
        return 0;
    }
    ```

#### 拓展挑战答案
1.  ```cpp
    #include <iostream>
    #include <algorithm> // for std::max

    int main() {
        std::ios_base::sync_with_stdio(false);
        std::cin.tie(NULL);

        int H; // 山洞高度
        std::cin >> H;

        int N; // 货车数量
        std::cin >> N;

        int max_pass_h = 0; // 初始化通过的最高货车高度为0

        for (int i = 0; i < N; ++i) {

            std::cin >> h;
            if (h <= H) { // 如果货车能通过山洞
                max_pass_h = std::max(max_pass_h, h); // 更新能通过的最高货车高度
            }
        }

        std::cout << max_pass_h << std::endl;

        return 0;
    }
    ```
