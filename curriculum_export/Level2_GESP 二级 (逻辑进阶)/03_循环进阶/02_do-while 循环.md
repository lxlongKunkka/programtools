## 趣味引入

同学们，大家好！还记得我们之前学习的 `while` 循环吗？它就像一个“先检查再行动”的小管家，每次做事情前都会先看看条件是不是满足。

比如，我们要烧水，`while (水没开)` 就 `继续烧`。如果水一开始就是开的，那它就不会烧了。

但是，在我们的生活中，有没有这样的场景：无论如何，我们都得先做一件事，做完之后，再根据情况决定要不要继续做呢？

举个例子：小明想玩一个猜数字游戏，他需要先输入一个数字（这是第一次行动），然后程序会告诉他猜大了还是猜小了，如果没猜对，他会继续输入（重复行动）。即使小明第一次就猜对了，他也至少输入了一次数字对吧？

再比如，我们去银行取钱，至少要输入一次密码（即使是错的也要输一次），然后系统再判断密码对不对，不对就让你重输。

这种“先做一次，再判断是否重复”的循环结构，就是我们今天要学习的 `do-while` 循环！它是一个“先行动再检查”的小帮手，是不是很有趣呢？

## 知识点讲解

### 1. `while` 循环回顾

在学习 `do-while` 循环之前，我们先快速回顾一下 `while` 循环。
`while` 循环的特点是：**先判断条件，如果条件为真，则执行循环体；如果条件为假，则跳过循环体。**
这意味着，如果条件一开始就不满足，`while` 循环的循环体可能一次都不会执行。

```cpp
// 示例：while 循环
int i = 5;
while (i < 5) { // 条件 i < 5 为假
    cout << "这个不会被打印出来" << endl;
    i++;
}
cout << "while 循环结束，i = " << i << endl;
```
运行结果：
```
while 循环结束，i = 5
```
可以看到，循环体一次都没有执行。

### 2. `do-while` 循环的语法

`do-while` 循环的语法结构如下：

```cpp
do {
    // 循环体：需要重复执行的代码
} while (条件表达式); // 注意：这里有一个分号！
```

### 3. `do-while` 循环的执行流程

`do-while` 循环的执行流程和 `while` 循环正好相反：
1.  首先，**无条件地执行一次 `do` 后面的循环体**。
2.  执行完循环体后，**再判断 `while` 后面的条件表达式**。
3.  如果条件表达式为真（`true`），则返回第 1 步，继续执行循环体。
4.  如果条件表达式为假（`false`），则跳出循环，执行 `do-while` 循环后面的语句。

**核心特点：`do-while` 循环至少会执行一次循环体。**

### 4. `do-while` 循环的适用场景

当你的程序需要**至少执行一次**某个操作，然后再根据条件决定是否继续重复时，`do-while` 循环是最佳选择。
常见的应用场景有：
*   **用户输入验证**：无论用户第一次输入什么，我们都需要先获取一次输入，然后判断输入是否有效，如果无效则继续要求用户重新输入。
*   **菜单选择**：显示一个操作菜单，让用户选择。无论用户选择什么，菜单至少要显示一次。如果用户选择继续操作，则再次显示菜单。
*   **一些游戏逻辑**：比如猜数字游戏，玩家至少要猜一次。

让我们通过代码示例来进一步理解它。

## 代码示例

### 示例一：`do-while` 循环的基本用法

这个例子会打印数字 1 到 5。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int i = 1; // 初始化计数器
    do {
        cout << "当前数字是: " << i << endl; // 执行循环体
        i++; // 更新条件变量
    } while (i <= 5); // 判断条件，如果 i 小于等于 5，则继续循环

    cout << "循环结束！" << endl;

    return 0;
}
```

**代码解析：**
*   `int i = 1;`：定义一个整数变量 `i` 并初始化为 1。
*   `do { ... } while (i <= 5);`：这是一个 `do-while` 循环。
    *   `cout << "当前数字是: " << i << endl;`：首先打印 `i` 的值。
    *   `i++;`：然后 `i` 自增 1。
    *   `while (i <= 5);`：在执行完循环体后，检查 `i` 是否小于等于 5。
        *   当 `i` 是 1, 2, 3, 4, 5 时，条件都为真，循环会继续。
        *   当 `i` 变成 6 时，条件 `6 <= 5` 为假，循环结束。

### 示例二：使用 `do-while` 进行用户输入验证

要求用户输入一个正整数，如果输入不是正整数，则一直要求重新输入，直到输入正确为止。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; // 用于存储用户输入的数字

    do {
        cout << "请输入一个正整数 (大于0): "; // 提示用户输入
        cin >> n; // 获取用户输入
        if (n <= 0) { // 如果输入不符合要求
            cout << "输入错误！请确保输入的是一个正整数。" << endl;
        }
    } while (n <= 0); // 如果 n 小于等于 0，说明输入无效，继续循环

    cout << "你输入的正整数是: " << n << endl; // 输入正确，打印结果

    return 0;
}
```

**代码解析：**
*   `int n;`：声明一个变量 `n` 来接收用户的输入。
*   `do { ... } while (n <= 0);`：这是一个 `do-while` 循环。
    *   `cout << "请输入一个正整数 (大于0): ";` 和 `cin >> n;`：无论如何，程序都会先提示用户并获取一次输入。
    *   `if (n <= 0)`：如果用户输入了一个非正整数，会给出错误提示。
    *   `while (n <= 0);`：在获取输入后，检查 `n` 是否小于等于 0。
        *   如果 `n` 小于等于 0，条件为真，循环会再次执行，要求用户重新输入。
        *   如果 `n` 大于 0，条件为假，循环结束，程序继续执行后面的代码。

这个例子完美展现了 `do-while` 循环“至少执行一次”的特性，非常适合处理需要先进行一次操作，再根据结果判断是否继续的场景。

### 3. 代码演练：修改循环

老师会提供一段使用 `while` 循环的代码，请同学们尝试将其修改为 `do-while` 循环，并思考修改后程序的行为是否会有变化。

**原 `while` 循环代码：**
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    char c = 'n'; // 假设用户一开始就不想玩
    while (c == 'y') {
        cout << "欢迎来到游戏！" << endl;
        cout << "想继续玩吗？(y/n): ";
        cin >> c;
    }
    cout << "游戏结束！" << endl;
    return 0;
}
```
**任务**：将上述代码修改为 `do-while` 循环，使其至少显示一次“欢迎来到游戏！”。

---
**参考答案（小组任务2和代码演练）**
*   **问题一答案**：`while` 循环是先判断条件再执行循环体，循环体可能一次都不执行；`do-while` 循环是先执行一次循环体，再判断条件，循环体至少会执行一次。当需要确保循环体至少执行一次时，优先选择 `do-while`。
*   **问题二答案**：选择 `do-while` 循环。因为无论用户第一次输入什么，我们都需要先获取一次输入。如果第一次就输入了 0，`do-while` 也能正确处理，而 `while` 循环可能需要先在循环外获取一次输入。

*   **代码演练修改结果：**
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;

    int main() {
        char c; // 这里可以不用初始化，因为do会先执行一次
        do {
            cout << "欢迎来到游戏！" << endl;
            cout << "想继续玩吗？(y/n): ";
            cin >> c;
        } while (c == 'y'); // 只要用户输入 'y'，就继续玩
        cout << "游戏结束！" << endl;
        return 0;
    }
    ```
    **修改后的变化**：即使 `c` 初始值不是 `'y'`，程序也会先打印一次“欢迎来到游戏！”并询问用户是否继续。这符合“至少执行一次”的特性。

### 练习题目答案与解析

**选择题答案：**

1.  **C**
    *   A 错误，条件在循环体执行之后判断。
    *   B 错误，循环体至少执行一次。
    *   D 错误，`while` 关键字后面必须有分号。

2.  **A**
    *   `i` 初始化为 0。
    *   `do` 块执行：打印 `0`，`i` 变为 1。
    *   判断 `while (i < 0)`，即 `1 < 0`，条件为假。
    *   循环结束。所以只打印 `0`。

3.  **C**
    *   `do-while` 循环保证循环体至少执行一次，这正好符合“玩家至少猜一次”的需求。

4.  **B**
    *   A 选项：`i` 会自增，最终 `i` 会达到 5，循环结束。
    *   B 选项：`i` 始终为 0，`i < 5` 永远为真，导致无限循环。
    *   C 选项：`i` 会自减，最终 `i` 会达到 0，条件 `i > 0` 为假，循环结束。
    *   D 选项：`i` 在循环体中被赋值为 10，然后条件 `i < 5` 变为 `10 < 5`，为假，循环结束。

**编程题参考答案：**

1.  **基础题：倒数计时**
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;

    int main() {
        int n;

        // 输入验证，确保 n 是正整数
        do {
            cout << "请输入一个正整数作为倒数起点: ";
            cin >> n;
            if (n <= 0) {
                cout << "输入错误！请重新输入一个正整数。" << endl;
            }
        } while (n <= 0);

        cout << "开始倒数：" << endl;
        do {
            cout << n << endl; // 打印当前数字
            n--; // 数字自减
        } while (n >= 1); // 当 n 大于等于 1 时继续循环

        cout << "倒数结束！" << endl;

        return 0;
    }
    ```

2.  **进阶题：简单的菜单选择**

    ```cpp
        return 0;
    }
    ```
