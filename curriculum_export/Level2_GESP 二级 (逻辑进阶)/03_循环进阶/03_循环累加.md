## 趣味引入

小明每天坚持阅读英文单词，第一天背 1 个，第二天背 2 个，第 \(n\) 天要背 \(n\) 个。  
问题：如果他坚持 10 天，他一共背了多少个单词？  

已知  
- 第 \(i\) 天背的单词数是 \(i\)。  
- 从第 1 天到第 10 天共有 10 天。  

未知  
- 所有天数背单词数的总和。  

通过“已知每天的单词数，用循环把每天的单词数累加到总和里”，即可解决该问题。

===NEXT===

## 知识点讲解

1. 累加器的作用  
   - 在循环前定义一个变量 `sum` 用于保存总和，初始值为 0。  
2. `for` 循环结构  
   ```cpp
   for (int i = 起始值; i <= 结束值; i++)
   {
       // 循环体
   }
   ```  
3. 累加操作  
   - 在循环体内执行 `sum += i;`，即可把每次的 `i` 加入到 `sum`。  
4. 拓展  
   - 求偶数和：`if (i % 2 == 0) sum += i;`  
   - 改为 `while` 循环、`do-while` 循环实现同样的功能。  

示例思路由浅入深：  
  已知→实现从 1 到 \(n\) 的累加；  
  简单→复杂→加入条件；  
  理论→实践→调试。

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n;      // 输入的天数
    int sum;    // 累加器

    // 读取天数 n
    cin >> n;

    // 1. 求 1 到 n 的累加
    sum = 0;
    for (int i = 1; i <= n; i++)
    {
        sum += i;        // 把第 i 天的单词数加入总和
    }
    cout << "1到" << n << "的累加和是 " << sum << endl;

    // 2. 求 1 到 n 中的偶数和
    sum = 0;
    for (int i = 1; i <= n; i++)
    {
        if (i % 2 == 0)
        {
            sum += i;    // 只累加偶数
        }
    }
    cout << "1到" << n << "中偶数的累加和是 " << sum << endl;

    return 0;
}
```

关键行注释已在代码中标明，小朋友可以对照阅读。

## 练习题目

基础题（选择题/填空题）：

1. 填空：下列 `for` 循环用于累加 1 到 100 的和，空缺部分应填  
   ```cpp
   int sum = 0;
   for (int i = 1; i __ 100; i++)
   {
       sum __ i;
   }
   ```
2. 选择题：下面代码执行后，`sum` 的值是多少？  
   ```cpp
   int sum = 0;
   for (int i = 2; i <= 6; i += 2)
   {
       sum += i;
   }
   ```  
   A. 6       B. 8       C. 10       D. 12  
3. 填空：如果我们要累加从 1 到 n 的奇数，可以在 `for` 循环中使用 `if (i % __ == __)`。  
4. 选择题：下面代码段中的循环次数是？  
   ```cpp
   for (int i = 5; i > 0; i--)
   {
       // 循环体
   }
   ```  
   A. 4 次    B. 5 次    C. 6 次    D. 无限次  
5. 选择题：若 `n=4`，下面两段代码得到的 `sum` 是否相同？  
   (1) `for (int i=1;i<=n;i++) sum+=i;`  
   (2) `for (int i=4;i>=1;i--) sum+=i;`  
   A. 相同    B. 不同  

答案与解析：  

1. “i <=”、“+=”  
   解析：循环 i 从 1 到 100，sum += i 累加。  
2. C  
   解析：i 依次取 2、4、6，sum=2+4+6=12（选 D 应为 12）。  
3. 2, 1  
   解析：奇数的判断 `i % 2 == 1`。  
4. B  
   解析：i=5,4,3,2,1 共 5 次。  
5. A  
   解析：两个循环遍历的都是 {1,2,3,4}，顺序不同但累加结果相同。
