同学们好！今天我们来玩一个“条件判断”的小游戏。

想象一下，周末你想出去玩，但是妈妈给你设了几个条件：
**条件一**：外面不下雨。
**条件二**：你已经完成了所有家庭作业。

妈妈说：“你可以出去玩，只要**不下雨** **或者** 你已经完成了所有作业。”

请问，以下哪种情况你可以出去玩呢？
1.  外面下雨，作业没完成。
2.  外面下雨，作业完成了。
3.  外面不下雨，作业没完成。
4.  外面不下雨，作业完成了。

（等待学生回答，引导他们发现只要有一个条件满足就可以出去玩）

再比如，妈妈说：“如果你**没有**完成作业，就不能玩游戏。”
这里“没有”完成作业，就是把“完成作业”这个条件的结果取反。在编程里，这叫做“逻辑非”。

今天，我们就要学习 C++ 语言中是如何表达这些“或者”和“不”的，它们叫做逻辑运算符：逻辑或 `||` 和逻辑非 `!`。

## 知识点讲解

### 1. 布尔类型回顾

在开始学习逻辑运算符之前，我们先简单回顾一下布尔类型 `bool`。
`bool` 类型只有两个值：
*   `true`：表示“真”，通常对应数字 `1`。
*   `false`：表示“假”，通常对应数字 `0`。

我们的逻辑运算，就是针对这些“真”和“假”进行的。

### 2. 逻辑或 (`||`) 运算符

*   **含义**：`||` 表示“或者”的关系。当且仅当所有条件都为假时，整个表达式的结果才为假；只要有一个条件为真，整个表达式的结果就为真。
*   **真值表**：
    | 条件 A | 条件 B | A `||` B |
    | :----: | :----: | :------: |
    |  `true`  |  `true`  |   `true`   |
    |  `true`  |  `false` |   `true`   |
    |  `false` |  `true`  |   `true`   |
    |  `false` |  `false` |   `false`  |
*   **C++ 中的表示**：`||` (两个竖线)
*   **短路求值**：这是一个很重要的特性！如果 `||` 左边的表达式结果已经是 `true` 了，那么 C++ 编译器会很“聪明”，它就不会再去计算 `||` 右边的表达式了，因为无论右边是 `true` 还是 `false`，整个 `||` 表达式的结果都已经是 `true` 了。这就像你妈妈说“不下雨或者作业完成”，如果外面已经不下雨了，她就不需要再检查你作业有没有完成了。

    **举例**：
    `bool a = true;`
    `bool b = false;`
    `bool c = a || b; // c 的结果是 true`

    `int x = 5;`
    `int y = 10;`
    `if (x > 10 || y == 10)`
    `{ // 这个条件会成立，因为 y == 10 是 true`
    `}`

### 3. 逻辑非 (`!`) 运算符

*   **含义**：`!` 表示“非”或“相反”的关系。它会将一个布尔值取反，即把 `true` 变为 `false`，把 `false` 变为 `true`。
*   **真值表**：
    | 条件 A | `!` A |
    | :----: | :---: |
    |  `true`  | `false` |
    |  `false` | `true`  |
*   **C++ 中的表示**：`!` (一个感叹号)
*   **举例**：
    `bool p = true;`
    `bool q = !p; // q 的结果是 false`

    `int age = 15;`
    `if (!(age < 18))`
    `{ // 这个条件的意思是：如果“年龄不小于18”，也就是“年龄大于等于18”`
    `}`

### 4. 运算符优先级

*   在 C++ 中，`!` 运算符的优先级比较高，比算术运算符和关系运算符都高。
*   `||` 运算符的优先级比较低。
*   当你不确定优先级时，最好的方法是使用小括号 `()` 来明确运算顺序，这样代码会更清晰，也不容易出错。
    例如：`!(a > 5 || b < 3)` 比 `!a > 5 || b < 3` 更明确。

### 5. 结合 `if` 语句使用

逻辑运算符最常用的场景就是结合 `if` 语句来构建复杂的条件判断。

**小结**：
*   `||` (逻辑或)：“或者”，只要有一个真，结果就真。
*   `!` (逻辑非)： “不”，把真变假，把假变真。

## 代码示例

让我们通过一些 C++ 代码来实际操作一下。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    // 示例 1: 逻辑或 (||) 的使用
    int a = 10;
    int b = 5;
    int c = 12;

    // 判断 a 是否大于 10 或者 b 是否等于 5
    // a > 10 是 false
    // b == 5 是 true
    // false || true 的结果是 true
    if (a > 10 || b == 5) {
        cout << "示例1: 条件满足！a大于10 或者 b等于5。" << endl;
    } else {
        cout << "示例1: 条件不满足。" << endl;
    }

    // 判断 c 是否小于 10 或者 a 是否等于 10
    // c < 10 是 false
    // a == 10 是 true
    // false || true 的结果是 true
    if (c < 10 || a == 10) {
        cout << "示例1: 第二个条件也满足！c小于10 或者 a等于10。" << endl;
    } else {
        cout << "示例1: 第二个条件不满足。" << endl;
    }

    cout << "--------------------" << endl;

    // 示例 2: 逻辑非 (!) 的使用
    bool is_raining = true; // 假设外面正在下雨
    bool has_homework = false; // 假设作业已经完成

    // 如果外面没有下雨，我们就可以出去玩
    // !is_raining 的结果是 !true，也就是 false
    if (!is_raining) {
        cout << "示例2: 没有下雨，可以出去玩！" << endl;
    } else {
        cout << "示例2: 正在下雨，不能出去玩。" << endl;
    }

    // 如果没有完成作业，就不能玩游戏
    // !has_homework 的结果是 !false，也就是 true
    if (!has_homework) { // 意味着作业没完成
        cout << "示例2: 作业没完成，不能玩游戏！" << endl;
    } else {
        cout << "示例2: 作业完成了，可以玩游戏。" << endl;
    }

    cout << "--------------------" << endl;

    // 示例 3: 结合使用
    int score = 85;
    int age = 17;

    // 判断一个学生是否可以获得奖励：
    // 条件是：分数大于90分 或者 年龄小于18岁 并且 没有不及格的科目 (这里简化为没有不及格，即分数不小于60)
    // 假设我们有一个变量 g 表示是否及格，这里我们用 score >= 60 来模拟
    bool is_passed = (score >= 60);

    // 奖励条件：(score > 90 || age < 18) && is_passed
    // 这里我们只看 (score > 90 || age < 18) 部分
    // score > 90 是 false
    // age < 18 是 true
    // false || true 的结果是 true
    if (score > 90 || age < 18) {
        cout << "示例3: 学生满足奖励条件的一部分：分数高或年龄小！" << endl;
    } else {
        cout << "示例3: 学生不满足奖励条件的一部分。" << endl;
    }

    // 如果我们想表达“分数不小于60分”
    if (! (score < 60) ) { // ! (score < 60) 和 score >= 60 是等价的
        cout << "示例3: 分数不小于60分，及格了！" << endl;
    } else {
        cout << "示例3: 分数小于60分，没及格。" << endl;
    }

    return 0;
}
```

### 1. 小组讨论与判断

请同学们分成小组，讨论并判断以下表达式的结果是 `true` 还是 `false`。

**情景设定**：
*   `int h = 170;` (身高)
*   `int w = 65;` (体重)
*   `bool has_ticket = false;` (是否有票)

**请判断以下表达式的结果**：

1.  `h > 180 || w < 60`
2.  `!(h < 160)`
3.  `has_ticket || h > 160`
4.  `!has_ticket || w > 70`
5.  `!(h > 175 || w < 70)`

（给学生几分钟时间讨论，然后请不同小组的代表回答并解释原因。）

**参考答案**：
1.  `h > 180` (170 > 180) 是 `false`。`w < 60` (65 < 60) 是 `false`。所以 `false || false` 是 `false`。
2.  `h < 160` (170 < 160) 是 `false`。所以 `!false` 是 `true`。
3.  `has_ticket` 是 `false`。`h > 160` (170 > 160) 是 `true`。所以 `false || true` 是 `true`。
4.  `!has_ticket` 是 `!false`，即 `true`。`w > 70` (65 > 70) 是 `false`。所以 `true || false` 是 `true`。
5.  `h > 175` (170 > 175) 是 `false`。`w < 70` (65 < 70) 是 `true`。所以 `(h > 175 || w < 70)` 的结果是 `true`。那么 `!(true)` 就是 `false`。

## 练习题目

### 选择题/填空题

1.  在 C++ 语言中，表示逻辑“或”的运算符是哪个？
    A. `&`
    B. `&&`
    C. `|`
    D. `||`

2.  表达式 `!(5 > 3)` 的结果是什么？
    A. `true`
    B. `false`
    C. `0`
    D. `1`

3.  如果 `bool x = true;` 且 `bool y = false;`，那么表达式 `x || y` 的结果是？
    A. `true`
    B. `false`
    C. 编译错误
    D. 不确定

4.  请填写合适的 C++ 逻辑运算符，使得表达式 `(10 > 5 ____ 8 < 3)` 的结果为 `true`。
    A. `&&`
    B. `||`
    C. `!`
    D. `=`

5.  以下关于逻辑或 `||` 运算符的说法，哪一项是正确的？
    A. `||` 运算符两边的表达式必须都为 `true`，结果才为 `true`。
    B. `||` 运算符会先计算右边的表达式，再计算左边的表达式。
    C. 如果 `||` 运算符左边的表达式为 `true`，则右边的表达式可能不会被计算。
    D. `||` 运算符的优先级高于算术运算符。

<br><br><br>
---
### 答案与解析

1.  **答案**：D
    **解析**：在 C++ 中，`||` 表示逻辑“或”，`&&` 表示逻辑“与”，`!` 表示逻辑“非”。`|` 是位运算符中的“或”。

2.  **答案**：B
    **解析**：`5 > 3` 的结果是 `true`。对 `true` 进行逻辑非运算 `!true`，结果是 `false`。

3.  **答案**：A
    **解析**：逻辑或 `||` 的规则是，只要有一个操作数为 `true`，结果就为 `true`。这里 `x` 为 `true`，所以 `true || false` 的结果是 `true`。

4.  **答案**：B
    **解析**：`10 > 5` 的结果是 `true`。`8 < 3` 的结果是 `false`。
    如果使用 `&&` (与)，`true && false` 的结果是 `false`。
    如果使用 `||` (或)，`true || false` 的结果是 `true`。
    所以应该填写 `||`。

5.  **答案**：C
    **解析**：
    A. 错误，`||` 只要有一个为 `true`，结果就为 `true`。
    B. 错误，`||` 是从左向右计算的。
    C. 正确，这是 `||` 的短路求值特性。如果左边为 `true`，整个表达式的结果已经确定为 `true`，无需计算右边。
    D. 错误，`||` 的优先级低于算术运算符和关系运算符。例如 `a + b > 5 || c < 10` 会先计算 `a+b` 和 `c<10`，再计算 `||`。
