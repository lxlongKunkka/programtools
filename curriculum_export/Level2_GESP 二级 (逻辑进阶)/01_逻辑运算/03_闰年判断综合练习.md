## 知识点讲解

闰年的判断规则是计算机编程中一个经典的逻辑判断问题。它包含了“与”和“或”两种逻辑关系。

### 1. 闰年的判断规则

一个年份 `y` 是闰年，需要满足以下两个条件中的 **任意一个**：

*   **条件一（普通闰年）**：`y` 能被 4 整除，**并且** 不能被 100 整除。
    *   例如：2004年，能被4整除（2004 / 4 = 501），不能被100整除，所以它是闰年。
    *   例如：1999年，不能被4整除，所以不是闰年。
    *   例如：1900年，能被4整除（1900 / 4 = 475），但也能被100整除（1900 / 100 = 19），所以不符合这个条件。

*   **条件二（世纪闰年）**：`y` 能被 400 整除。
    *   例如：2000年，能被400整除（2000 / 400 = 5），所以它是闰年。
    *   例如：1900年，不能被400整除，所以不符合这个条件。

**总结一下：** 只要满足“条件一”或者“条件二”中的任何一个，这个年份就是闰年。

### 2. 逻辑运算符回顾

在 C++ 中，我们使用逻辑运算符来连接这些判断条件：

*   `&&` (逻辑与)：表示“并且”。只有当 `&&` 两边的条件都为真时，整个表达式才为真。
    *   例如：`y % 4 == 0 && y % 100 != 0` 表示“y能被4整除 **并且** y不能被100整除”。
*   `||` (逻辑或)：表示“或者”。只要当 `||` 两边的条件中有一个为真时，整个表达式就为真。
    *   例如：`(条件一) || (条件二)` 表示“满足条件一 **或者** 满足条件二”。

### 3. 将闰年规则转化为 C++ 表达式

根据上面的规则和逻辑运算符，我们可以将闰年的判断条件写成一个完整的 C++ 逻辑表达式：

` (y % 4 == 0 && y % 100 != 0) || (y % 400 == 0) `

*   `y % 4 == 0`：判断 `y` 能否被 4 整除。
*   `y % 100 != 0`：判断 `y` 能否被 100 整除（这里是不能被整除）。
*   `y % 400 == 0`：判断 `y` 能否被 400 整除。

同学们，记住这个表达式，它就是我们判断闰年的“魔法咒语”！

## 代码示例

现在，让我们把闰年的判断规则，用 C++ 代码实现出来吧！

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    // 声明一个整数变量y，用于存储输入的年份
    int y;

    // 提示用户输入年份
    cout << "请输入一个年份：";

    // 读取用户输入的年份
    cin >> y;

    // 使用if-else语句和逻辑运算符判断是否为闰年
    // 闰年规则：(能被4整除但不能被100整除) 或者 (能被400整除)
    if ((y % 4 == 0 && y % 100 != 0) || (y % 400 == 0))
    {
        // 如果满足闰年条件，输出“是闰年”
        cout << y << "年是闰年。" << endl;
    }
    else
    {
        // 如果不满足闰年条件，输出“不是闰年”
        cout << y << "年不是闰年。" << endl;
    }

    return 0;
}
```

**代码解析：**

1.  `#include <bits/stdc++.h>` 和 `using namespace std;`：这是我们编程的“魔法棒”，包含了大部分常用的功能，让我们能够方便地进行输入输出。
2.  `int y;`：我们定义了一个名为 `y` 的变量，用来存放用户输入的年份。
3.  `cout << "请输入一个年份：";`：在屏幕上显示提示信息，告诉用户应该做什么。
4.  `cin >> y;`：等待用户输入一个数字，并把这个数字存到变量 `y` 里面。
5.  `if ((y % 4 == 0 && y % 100 != 0) || (y % 400 == 0))`：这是判断闰年的核心逻辑！
    *   `y % 4 == 0`：判断年份 `y` 除以 4 的余数是否为 0（即能否被 4 整除）。
    *   `y % 100 != 0`：判断年份 `y` 除以 100 的余数是否不为 0（即不能被 100 整除）。
    *   `y % 400 == 0`：判断年份 `y` 除以 400 的余数是否为 0（即能否被 400 整除）。
    *   `&&` (逻辑与)：连接“能被4整除”和“不能被100整除”这两个条件，它们必须同时成立。
    *   `||` (逻辑或)：连接两个大的判断条件，只要其中一个成立，就说明是闰年。
6.  `cout << y << "年是闰年。" << endl;`：如果 `if` 语句中的条件为真，就执行这行代码，输出结果。
7.  `else { ... }`：如果 `if` 语句中的条件为假（不是闰年），就执行 `else` 后面的代码。

同学们，是不是很神奇？短短几行代码，就能帮我们解决一个有趣的实际问题！

## 练习题目

### 选择题/填空题

1.  以下哪个年份是闰年？
    A. 1900
    B. 2000
    C. 2100
    D. 2200

2.  判断一个年份 `y` 是否为闰年的 C++ 逻辑表达式中，哪个是正确的？
    A. `y % 4 == 0 || y % 100 != 0 && y % 400 == 0`
    B. `(y % 4 == 0 && y % 100 != 0) || (y % 400 == 0)`
    C. `y % 4 == 0 && y % 100 != 0 || y % 400 == 0`
    D. `(y % 4 == 0 || y % 400 == 0) && y % 100 != 0`

3.  如果年份 `y = 1900`，以下 C++ 表达式 `(y % 4 == 0 && y % 100 != 0) || (y % 400 == 0)` 的计算结果是：
    A. true (真)
    B. false (假)
    C. 1
    D. 0

4.  请填写代码，使程序能够正确判断输入的年份 `a` 是否为闰年。
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;
    int main() {
        int a;
        cin >> a;
        if (__________________________________________) { // 在此处填入判断条件
            cout << "Yes" << endl;
        } else {
            cout << "No" << endl;
        }
        return 0;
    }
    ```

5.  在 C++ 中，逻辑运算符 `&&` (与) 和 `||` (或) 之间，优先级更高的是 ____。

---

### 答案与解析

1.  **答案：** B
    **解析：**
    *   A. 1900：能被4整除，能被100整除，但不能被400整除，所以不是闰年。
    *   B. 2000：能被400整除，所以是闰年。
    *   C. 2100：能被4整除，能被100整除，但不能被400整除，所以不是闰年。
    *   D. 2200：能被4整除，能被100整除，但不能被400整除，所以不是闰年。

2.  **答案：** B
    **解析：**
    *   闰年规则是：(能被4整除且不能被100整除) **或者** (能被400整除)。
    *   选项 B 完美地体现了这种逻辑关系，通过括号明确了运算顺序。
    *   选项 A 和 C 都没有正确使用括号来组合条件，由于 `&&` 的优先级高于 `||`，它们的运算顺序会出错。
    *   选项 D 的逻辑是错误的，它表示“能被4整除或能被400整除，并且不能被100整除”，这不符合世纪闰年的规则。

3.  **答案：** B
    **解析：**
    *   当 `y = 1900` 时：
        *   `y % 4 == 0` (1900 % 4 == 0) 为 `true`。
        *   `y % 100 != 0` (1900 % 100 != 0) 为 `false`。
        *   `y % 400 == 0` (1900 % 400 == 0) 为 `false`。
    *   所以，表达式 `(true && false) || (false)` 变为 `false || false`，最终结果是 `false`。
    *   在 C++ 中，`false` 通常对应整数 `0`。

4.  **答案：** `(a % 4 == 0 && a % 100 != 0) || (a % 400 == 0)`
    **解析：** 填入标准的闰年判断逻辑表达式即可。

5.  **答案：** `&&` (逻辑与)
    **解析：** 在 C++ 中，逻辑与 `&&` 的优先级高于逻辑或 `||`。因此，在没有括号的情况下，`&&` 会先进行运算。为了避免优先级问题，我们通常会使用括号来明确运算顺序，就像在闰年判断表达式中那样。
