## 趣味引入

同学们，想象一下我们班级要组织一次春游活动，需要统计每位同学的详细信息：**姓名**、**年龄**和**身高**。

如果我们要存储一个同学的信息，可能需要这样几个变量：
```cpp
char n1[20]; // 存储第一个同学的姓名
int a1;      // 存储第一个同学的年龄
double h1;   // 存储第一个同学的身高
```
那如果有 30 个同学呢？难道我们要定义 `n1, a1, h1, n2, a2, h2, ..., n30, a30, h30` 这么多的变量吗？这太麻烦了，而且很容易搞混！

就算我们用数组来存储：
```cpp
char names[30][20]; // 存储30个同学的姓名
int ages[30];       // 存储30个同学的年龄
double heights[30]; // 存储30个同学的身高
```
这样是好多了，但新的问题来了：`names[0]`、`ages[0]`、`heights[0]` 它们分别代表同一个同学的信息吗？我们必须小心翼翼地保证数组下标的一致性，否则数据就乱套了！

有没有一种方法，能把一个同学的姓名、年龄、身高这些相关的信息，“打包”成一个整体呢？就像一个文件袋，里面装着同一个同学的所有资料。

有的！C++ 中的 **结构体 (struct)** 就是专门用来解决这个问题的！它能把不同类型但相互关联的数据组合在一起，形成一个新的数据类型，让我们的数据管理变得井井有条！

## 知识点讲解

### 1. 什么是结构体？

结构体（`struct`）是一种用户自定义的数据类型，它允许我们将不同类型（比如 `int`、`char` 数组、`double` 等）的相关数据项组合在一起，形成一个单一的实体。你可以把它想象成一个“蓝图”或者“模板”，用来描述一个复杂的数据对象应该包含哪些信息。

### 2. 如何定义结构体？

定义结构体就像在制作一个模板。我们使用 `struct` 关键字，后面跟着你为这个新数据类型起的名字（通常首字母大写，但小写也可以），然后在大括号 `{}` 里面列出这个结构体包含的所有成员变量。

**语法格式：**

```cpp
struct 结构体名称 {
    数据类型 成员变量1;
    数据类型 成员变量2;
    // ... 更多成员变量
}; // 注意：这里有一个分号！
```

**例子：定义一个“学生”结构体**

```cpp
struct Student {
    char n[20];  // 姓名 (使用字符数组存储字符串，因为我们不能使用STL容器)
    int a;       // 年龄
    double s;    // 成绩
}; // 定义了一个名为 Student 的结构体，它有三个成员：n, a, s
```
**小提示：**
*   定义结构体只是告诉编译器有这么一种数据类型，它长什么样子，但并没有真正创建任何变量，也没有分配内存。
*   每个成员变量都有自己的数据类型，可以是 `int`、`double`、`char` 数组，甚至是另一个结构体！

### 3. 如何声明结构体变量？

定义了结构体模板后，我们就可以像声明 `int` 变量一样，声明结构体变量了。声明结构体变量才会真正分配内存空间。

**语法格式：**

```cpp
结构体名称 变量名;
```

**例子：声明“学生”结构体变量**

```cpp
Student s1;       // 声明一个 Student 类型的变量 s1
Student students[50]; // 声明一个 Student 类型的数组，可以存储 50 个学生的信息
```
**小提示：**
*   `s1` 这个变量现在就代表了一个具体的学生，它内部包含了姓名、年龄和成绩这三部分信息。
*   `students` 数组的每个元素都是一个 `Student` 类型的变量，可以存储一个学生的完整信息。

### 4. 如何访问结构体成员？

要访问结构体变量内部的成员（比如获取学生的年龄，或者修改学生的成绩），我们使用 **`.` (点) 运算符**。

**语法格式：**

```cpp
结构体变量名.成员变量名
```

**例子：访问“学生”结构体成员**

```cpp
Student s1; // 假设我们已经声明了一个学生变量 s1

// 给 s1 的成员赋值
strcpy(s1.n, "Xiao Ming"); // 给姓名赋值 (注意：char 数组赋值要用 strcpy 函数)
s1.a = 12;                 // 给年龄赋值
s1.s = 95.5;               // 给成绩赋值

// 打印 s1 的成员信息
cout << "学生姓名: " << s1.n << endl;
cout << "学生年龄: " << s1.a << endl;
cout << "学生成绩: " << s1.s << endl;
```
**特别注意：关于字符串成员的赋值**

*   结构体中的 `char` 数组（例如 `char n[20];`）不能直接使用 `=` 运算符进行赋值，比如 `s1.n = "Li Hua";` 这种写法是错误的！
*   我们需要使用 C 语言标准库中的 `strcpy` 函数来复制字符串。`strcpy` 函数的用法是 `strcpy(目标字符数组, 源字符串);`。
*   使用 `strcpy` 函数需要包含 `<cstring>` 头文件（或者在 `<bits/stdc++.h>` 中已经包含了）。

### 5. 结构体变量的初始化

我们可以在声明结构体变量的同时，对其成员进行初始化。

**语法格式：**

```cpp
结构体名称 变量名 = {成员1的值, 成员2的值, ...};
```

**例子：初始化“学生”结构体变量**

```cpp
Student s2 = {"Wang Fang", 14, 88.0}; // 声明并初始化一个 Student 变量 s2

// 此时，s2.n 就是 "Wang Fang"，s2.a 是 14，s2.s 是 88.0
cout << "学生姓名: " << s2.n << endl;
cout << "学生年龄: " << s2.a << endl;
cout << "学生成绩: " << s2.s << endl;
```
**小提示：**
*   初始化的顺序必须和结构体定义中成员的顺序一致。
*   如果成员是 `char` 数组，直接用双引号括起来的字符串即可。

## 代码示例

下面我们来看一个完整的 C++ 程序，它演示了如何定义、声明、输入和输出结构体变量，以及如何处理结构体数组。

```cpp
#include <bits/stdc++.h> // 包含大部分常用头文件
using namespace std;     // 使用标准命名空间

// 1. 定义一个 Student 结构体
struct Student {
    char n[20];  // 姓名
    int a;       // 年龄
    double s;    // 成绩
};

int main()
{
    // --- 声明并使用单个结构体变量 ---
    cout << "--- 处理单个学生信息 ---" << endl;
    Student s1; // 声明一个 Student 类型的变量 s1

    // 输入 s1 的信息
    cout << "请输入第一个学生的姓名: ";
    cin >> s1.n; // 对于 char 数组，cin 可以直接读取字符串
    cout << "请输入第一个学生的年龄: ";
    cin >> s1.a;
    cout << "请输入第一个学生的成绩: ";
    cin >> s1.s;

    // 输出 s1 的信息
    cout << "第一个学生的信息如下:" << endl;
    cout << "姓名: " << s1.n << endl;
    cout << "年龄: " << s1.a << endl;
    cout << "成绩: " << s1.s << endl;
    cout << endl;

    // --- 声明并使用结构体数组 ---
    cout << "--- 处理多个学生信息 (结构体数组) ---" << endl;
    Student students[3]; // 声明一个 Student 类型的数组，可以存储 3 个学生的信息

    // 循环输入 3 个学生的信息
    for (int i = 0; i < 3; i++) {
        cout << "请输入第 " << i + 1 << " 个学生的姓名: ";
        cin >> students[i].n;
        cout << "请输入第 " << i + 1 << " 个学生的年龄: ";
        cin >> students[i].a;
        cout << "请输入第 " << i + 1 << " 个学生的成绩: ";
        cin >> students[i].s;
        cout << "--------------------" << endl;
    }

    // 循环输出 3 个学生的信息
    cout << "所有学生的信息如下:" << endl;
    for (int i = 0; i < 3; i++) {
        cout << "学生 " << i + 1 << ":" << endl;
        cout << "  姓名: " << students[i].n << endl;
        cout << "  年龄: " << students[i].a << endl;
        cout << "  成绩: " << students[i].s << endl;
    }
    cout << endl;

    // --- 查找年龄最大的学生 ---
    cout << "--- 查找年龄最大的学生 ---" << endl;
    if (3 > 0) { // 确保数组非空
        int max_a = students[0].a; // 假设第一个学生年龄最大
        int max_idx = 0;           // 记录年龄最大的学生的下标

        for (int i = 1; i < 3; i++) {
            if (students[i].a > max_a) {
                max_a = students[i].a;
                max_idx = i;
            }
        }

        cout << "年龄最大的学生是:" << endl;
        cout << "  姓名: " << students[max_idx].n << endl;
        cout << "  年龄: " << students[max_idx].a << endl;
        cout << "  成绩: " << students[max_idx].s << endl;
    } else {
        cout << "没有学生信息可供查找。" << endl;
    }


    return 0;
}
```

### 选择题

1.  下列关于结构体的描述，哪一项是**错误**的？
    A. 结构体是一种用户自定义的数据类型。
    B. 结构体可以将不同数据类型的数据组合在一起。
    C. 定义结构体时，会直接分配内存空间。
    D. 结构体成员可以通过 `.` 运算符访问。

2.  如果定义了 `struct Item { int id; double price; };`，并声明了 `Item item1;`，那么要给 `item1` 的 `id` 成员赋值为 100，正确的语句是？

    B. `id.item1 = 100;`
    C. `item1->id = 100;`
    D. `item1[id] = 100;`

3.  定义一个 `struct Product { char name[30]; int count; };`，并声明 `Product p;`。要将 `p` 的 `name` 成员赋值为 "Apple"，正确的代码是？
    A. `p.name = "Apple";`
    B. `strcpy(p.name, "Apple");`
    C. `p.name[] = "Apple";`
    D. `p.name = {'A', 'p', 'p', 'l', 'e', '\0'};`

4.  以下 C++ 代码片段中，结构体定义和变量声明都正确的是？
    A.
    ```cpp
    struct {

        int y;
    } Point;
    Point p1;
    ```
    B.
    ```cpp
    struct Point {
        int x;
        int y;
    };
    Point p1;
    ```
    C.
    ```cpp
    struct Point {
        int x;
        int y;
    ```


### 练习题目答案与解析

1.  **C. 定义结构体时，会直接分配内存空间。**
    *   **解析：** 结构体定义只是一个“蓝图”，它告诉编译器这种数据类型长什么样，但并不会分配内存。只有当声明结构体变量时，才会根据这个蓝图分配实际的内存空间。

2.  **A. `item1.id = 100;`**
    *   **解析：** 访问结构体成员使用 `.` (点) 运算符，然后直接进行赋值操作。`B` 顺序错误，`C` 是指针操作，`D` 是数组操作，都不适用于此。

3.  **B. `strcpy(p.name, "Apple");`**
    *   **解析：** 结构体中的字符数组（字符串）成员不能直接使用 `=` 运算符赋值。需要使用 `strcpy` 函数将源字符串复制到目标字符数组中。

4.  **B.**
    *   **解析：**
        *   A 选项在结构体定义时直接声明了变量 `Point`，导致 `Point p1;` 再次声明时会出错（或者 `Point` 本身就是变量名而不是类型名）。
        *   B 选项是标准的结构体定义和变量声明方式。
        *   C 选项 `struct p1;` 是错误的，应该直接 `Point p1;`。
        *   D 选项 `Point { ... };` 缺少 `struct` 关键字。

5.  **关联**
    *   **解析：** 结构体的核心作用就是将相互关联的数据项组织在一起。

6.  **.** (点)
    *   **解析：** 访问结构体成员需要使用 `.` (点) 运算符。
