## 知识点讲解

### 1. 结构体回顾

同学们，还记得结构体是什么吗？它就像我们自己定义的一种“新类型”，可以把不同的数据类型（比如 `int`、`string`、`char` 等）组合在一起，形成一个有意义的整体。

**定义一个学生结构体 `Student`：**

```cpp
struct Student {
    int id;         // 学号
    char name[20];  // 姓名（使用字符数组存储字符串，避免STL string）
    int c;          // 语文成绩
    int m;          // 数学成绩
    int e;          // 英语成绩
    int total;      // 总分
    double avg;     // 平均分
};
```

这样，一个 `Student` 类型的变量就能存储一个学生的所有信息了。

### 2. 为什么需要结构体数组？

当我们需要存储多个相同类型的结构体数据时，例如一个班级里所有学生的信息，如果一个一个地定义变量，会非常麻烦：
`Student s1, s2, s3, ..., s30;`

这显然不是一个好办法！就像我们存储多个整数时会用 `int a[100];` 一样，存储多个结构体时，我们就可以使用**结构体数组**！

结构体数组，顾名思义，就是以结构体作为数据类型的数组。它能让我们用一个统一的名字来管理一组结构体变量。

### 3. 结构体数组的定义

结构体数组的定义和普通数组类似，只是数据类型换成了我们自定义的结构体类型。

**语法：**
`struct 结构体名 数组名[大小];`

**例子：定义一个可以存储 50 个学生信息的数组**

```cpp
struct Student s[50]; // s 就是一个结构体数组，可以存储 50 个 Student 类型的数据
```
这里的 `s` 就是我们全班同学的“档案袋集合”！

### 4. 结构体数组的初始化

**a) 定义时初始化：**
可以在定义结构体数组时，直接为它的成员赋值。

```cpp
struct Student s[3] = {
    {101, "Alice", 90, 85, 92, 0, 0.0}, // 第一个学生的学号、姓名、三科成绩
    {102, "Bob", 88, 91, 80, 0, 0.0},   // 第二个学生
    {103, "Charlie", 95, 87, 89, 0, 0.0} // 第三个学生
};
```
**注意：** 如果结构体成员是字符数组，初始化时可以直接用字符串字面量。

**b) 逐个赋值（通过输入）：**
这在实际应用中更常见，通常我们会通过循环来读取用户输入的数据，然后赋值给结构体数组的各个成员。

```cpp
// 假设要输入 n 个学生的信息
int n;
// cin >> n;
// struct Student s[n]; // 变长数组 C++标准不支持，但一些编译器可能支持。
// 为了兼容性，通常会定义一个足够大的数组，或者在C++11后使用vector (但本级别不能用)
// 假设 n 不超过 100
struct Student s[100];

for (int i = 0; i < n; i++) {
    cout << "请输入第 " << i + 1 << " 个学生的学号、姓名、语文、数学、英语成绩：\n";
    cin >> s[i].id;
    cin >> s[i].name; // 注意：cin >> char_array 遇到空格会停止
    cin >> s[i].c >> s[i].m >> s[i].e;
    // 计算总分和平均分
    s[i].total = s[i].c + s[i].m + s[i].e;
    s[i].avg = s[i].total / 3.0;
}
```

### 5. 结构体数组的访问

访问结构体数组中的元素，需要先指定数组的下标，然后再通过成员访问符（`.`）来访问该元素的具体成员。

**语法：**
`数组名[下标].成员名`

**例子：**

```cpp
// 访问第一个学生的学号
cout << s[0].id << endl;

// 访问第二个学生的姓名
cout << s[1].name << endl;

// 输出所有学生的总分和平均分
for (int i = 0; i < n; i++) {
    cout << s[i].name << " 的总分是：" << s[i].total << "，平均分是：" << s[i].avg << endl;
}
```

### 6. 结构体数组的排序

我们经常需要对学生成绩进行排序，比如按总分从高到低排列。C++ 标准库中的 `sort` 函数可以很方便地实现这个功能。但是，`sort` 函数不知道如何比较两个 `Student` 结构体，所以我们需要告诉它比较的规则。

**方法：编写一个比较函数 `cmp`**

`cmp` 函数是一个布尔函数，它接收两个结构体变量作为参数，并返回 `true` 或 `false`，表示它们在排序中的相对顺序。

**例子：按总分从高到低排序**

```cpp
// 比较函数：按总分从高到低排序
bool cmp(struct Student x, struct Student y) {
    return x.total > y.total; // 如果 x 的总分大于 y 的总分，则 x 排在 y 前面
}

// 在主函数中使用 sort 函数进行排序
// sort(数组起始地址, 数组结束地址+1, 比较函数);
// sort(s, s + n, cmp); // 对 s 数组的前 n 个元素进行排序
```
如果想按学号从小到大排序，`cmp` 函数就是：
`return x.id < y.id;`

如果想按平均分从低到高排序，`cmp` 函数就是：
`return x.avg < y.avg;`

**注意：**
*   `sort` 函数位于 `<algorithm>` 头文件中，但我们使用 `<bits/stdc++.h>` 已经包含了。
*   `sort` 函数的第三个参数是比较函数的地址，直接写函数名 `cmp` 即可。

通过结构体数组，我们就能像管理一个个独立的“学生档案”一样，轻松管理全班同学的成绩啦！是不是很方便呢？

## 代码示例

### 示例1：学生成绩管理系统 (输入、计算、输出)

这个示例将演示如何定义一个学生结构体，然后使用结构体数组存储多个学生的信息，并计算他们的总分和平均分，最后输出所有学生的信息。

```cpp
#include <bits/stdc++.h>
using namespace std;

// 定义学生结构体
struct Student {
    int id;         // 学号
    char name[20];  // 姓名
    int c;          // 语文成绩
    int m;          // 数学成绩
    int e;          // 英语成绩
    int total;      // 总分
    double avg;     // 平均分
};

int main() {
    int n; // 学生数量
    cout << "请输入学生数量：";
    cin >> n;

    struct Student s[100]; // 定义一个结构体数组，最多可存储 100 个学生

    // 循环输入每个学生的信息
    for (int i = 0; i < n; i++) {
        cout << "\n请输入第 " << i + 1 << " 个学生的信息：\n";
        cout << "学号：";
        cin >> s[i].id;
        cout << "姓名：";
        cin >> s[i].name; // 注意：cin >> char_array 遇到空格会停止读取
        cout << "语文成绩：";
        cin >> s[i].c;
        cout << "数学成绩：";
        cin >> s[i].m;
        cout << "英语成绩：";
        cin >> s[i].e;

        // 计算总分和平均分
        s[i].total = s[i].c + s[i].m + s[i].e;
        s[i].avg = s[i].total / 3.0;
    }

    // 输出所有学生的信息及计算结果
    cout << "\n--- 学生成绩单 ---\n";
    cout << "学号\t姓名\t语文\t数学\t英语\t总分\t平均分\n";
    cout << "--------------------------------------------------------\n";
    for (int i = 0; i < n; i++) {
        cout << s[i].id << "\t";
        cout << s[i].name << "\t";
        cout << s[i].c << "\t";
        cout << s[i].m << "\t";
        cout << s[i].e << "\t";
        cout << s[i].total << "\t";
        cout << fixed << setprecision(2) << s[i].avg << "\n"; // fixed和setprecision需要<iomanip>
    }
    cout << "--------------------------------------------------------\n";

    return 0;
}
```

### 示例2：结构体数组排序 (按总分从高到低)

在上面的基础上，我们增加按总分排序的功能。

```cpp
#include <bits/stdc++.h>
using namespace std;

// 定义学生结构体
struct Student {
    int id;         // 学号
    char name[20];  // 姓名
    int c;          // 语文成绩
    int m;          // 数学成绩
    int e;          // 英语成绩
    int total;      // 总分
    double avg;     // 平均分
};

// 比较函数：按总分从高到低排序
bool cmp(struct Student x, struct Student y) {
    // 如果总分相同，则按学号从小到大排序（保持稳定）
    if (x.total == y.total) {
        return x.id < y.id;
    }
    return x.total > y.total; // 否则按总分从高到低排序
}

int main() {
    int n; // 学生数量
    cout << "请输入学生数量：";
    cin >> n;

    struct Student s[100]; // 定义一个结构体数组，最多可存储 100 个学生

    // 循环输入每个学生的信息
    for (int i = 0; i < n; i++) {
        cout << "\n请输入第 " << i + 1 << " 个学生的信息：\n";
        cout << "学号：";
        cin >> s[i].id;
        cout << "姓名：";
        cin >> s[i].name;
        cout << "语文成绩：";
        cin >> s[i].c;
        cout << "数学成绩：";
        cin >> s[i].m;
        cout << "英语成绩：";
        cin >> s[i].e;

        // 计算总分和平均分
        s[i].total = s[i].c + s[i].m + s[i].e;
        s[i].avg = s[i].total / 3.0;
    }

    // 使用 sort 函数对学生数组进行排序
    // s 是数组的起始地址，s + n 是数组结束地址的下一个位置
    // cmp 是我们自定义的比较函数
    sort(s, s + n, cmp);

    // 输出排序后的学生信息及计算结果
    cout << "\n--- 学生成绩单 (按总分降序排列) ---\n";
    cout << "学号\t姓名\t语文\t数学\t英语\t总分\t平均分\n";
    cout << "--------------------------------------------------------\n";
    for (int i = 0; i < n; i++) {
        cout << s[i].id << "\t";
        cout << s[i].name << "\t";
        cout << s[i].c << "\t";
        cout << s[i].m << "\t";
        cout << s[i].e << "\t";
        cout << s[i].total << "\t";
        cout << fixed << setprecision(2) << s[i].avg << "\n";
    }
    cout << "--------------------------------------------------------\n";

    return 0;
}
```

### 练习题目答案与解析

1.  **答案：D**
    **解析：** `struct Point` 是结构体的完整类型名，`p[10]` 表示数组名和大小。A选项省略了 `struct` 关键字，在 C++ 中虽然有时可以省略，但规范写法是带上 `struct`。B和C选项语法错误。

2.  **答案：B**
    **解析：** 访问结构体数组元素需要先用下标 `[ ]` 指定是数组中的第几个元素（数组下标从0开始，所以第3个是 `s[2]`），然后用成员访问符 `.` 访问该元素的具体成员 `id`。

3.  **答案：比较**
    **解析：** `sort` 函数需要一个比较函数来确定如何对自定义数据类型进行排序。

4.  **答案：Li**
    **解析：** `cin >> char_array` 读取字符串时，遇到空格、制表符或换行符就会停止。所以输入“Li Hua”时，`s[i].name` 只会读取到“Li”。要读取包含空格的字符串，可以使用 `cin.getline(s[i].name, sizeof(s[i].name));` 或者 `fgets(s[i].name, sizeof(s[i].name), stdin);`。

5.  **答案：B**
    **解析：** `return a.total < b.total;` 表示如果 `a` 的总分小于 `b` 的总分，则认为 `a` 应该排在 `b` 的前面。这符合从小到大（升序）的排序逻辑。
