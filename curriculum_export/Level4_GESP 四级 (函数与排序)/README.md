# Level 4：GESP 四级 (函数与排序)

## 教学目标
1. 掌握结构体 (`struct`) 的使用。
2. 理解指针与引用的基本概念。
3. 掌握简单的算法复杂度分析。

## 核心知识点
- 结构体：定义、成员访问、结构体数组。
- 排序：`std::sort` 的使用，自定义比较函数。
- 指针入门：地址概念，指针变量。
- 算法分析：时间复杂度 ($O(n)$, $O(n^2)$) 的直观理解。
