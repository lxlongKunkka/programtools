## 知识点讲解

### 1. 回顾二维数组

还记得我们上节课学过的二维数组吗？它就像一个表格，有行有列。我们可以用 `a[行下标][列下标]` 来访问表格中的每一个小格子。
例如，`int a[5][5];` 就声明了一个5行5列的整型二维数组。

### 2. 杨辉三角的结构与规律

我们再仔细观察一下杨辉三角：

```
第0行:      1
第1行:     1 1
第2行:    1 2 1
第3行:   1 3 3 1
第4行:  1 4 6 4 1
...
```

**规律总结：**
1.  **行数与元素个数：** 杨辉三角的第 `i` 行（我们从第0行开始计数），有 `i+1` 个数字。
    -   第0行有1个数字。
    -   第1行有2个数字。
    -   第2行有3个数字。
    -   ...
2.  **边界值：** 每行的第一个数字和最后一个数字都是 `1`。
    -   例如，第 `i` 行的第0个数字 `a[i][0]` 总是1。
    -   第 `i` 行的第 `i` 个数字 `a[i][i]` 总是1。
3.  **内部值：** 除每行的第一个和最后一个数字外，其余的每个数字都等于它**正上方**和**左上方**两个数字之和。
    -   例如，第 `i` 行的第 `j` 个数字 `a[i][j]` 等于 `a[i-1][j-1] + a[i-1][j]`。

### 3. 如何用二维数组表示杨辉三角

我们可以用一个二维数组 `a[N][N]` 来存储杨辉三角。
-   `a[i][j]` 就表示杨辉三角第 `i` 行的第 `j` 个数字。
-   因为第 `i` 行有 `i+1` 个数字，所以 `j` 的取值范围是 `0` 到 `i`。

### 4. 生成杨辉三角的算法思路

现在，我们有了二维数组和杨辉三角的规律，就可以开始设计算法了！

1.  **确定最大行数：** 我们需要知道要生成多少行的杨辉三角。假设用户输入 `n` 行。
2.  **声明二维数组：** 根据 `n` 的最大可能值，声明一个足够大的二维数组，例如 `int a[20][20];` (如果我们最多生成20行)。
3.  **循环填充数组：**
    *   **外层循环：** 从第0行到第 `n-1` 行，遍历每一行 `i`。
    *   **内层循环：** 在每一行 `i` 中，从第0列到第 `i` 列，遍历每一个数字 `j`。
    *   **判断赋值：**
        *   如果 `j` 是当前行的第一个数字（即 `j == 0`），或者 `j` 是当前行的最后一个数字（即 `j == i`），那么 `a[i][j] = 1`。
        *   否则（对于内部数字），`a[i][j] = a[i-1][j-1] + a[i-1][j]`。
4.  **打印输出：**
    *   为了让杨辉三角看起来更整齐，我们可以先打印一些空格，让每行居中。
    *   然后遍历二维数组，依次打印出 `a[i][j]` 的值。

**示例推演（生成前3行）：**

-   **`i = 0` (第0行):**
    -   `j = 0`：`j == 0` 成立，`a[0][0] = 1`。
    -   打印：`1`

-   **`i = 1` (第1行):**
    -   `j = 0`：`j == 0` 成立，`a[1][0] = 1`。
    -   `j = 1`：`j == i` 成立，`a[1][1] = 1`。
    -   打印：`1 1`

-   **`i = 2` (第2行):**
    -   `j = 0`：`j == 0` 成立，`a[2][0] = 1`。
    -   `j = 1`：`j` 既不等于0也不等于 `i`，使用公式：`a[2][1] = a[1][0] + a[1][1] = 1 + 1 = 2`。
    -   `j = 2`：`j == i` 成立，`a[2][2] = 1`。
    -   打印：`1 2 1`

是不是很简单呢？我们马上来用代码实现它！

## 代码示例

```cpp
#include <bits/stdc++.h> // 包含常用的头文件
using namespace std;       // 使用标准命名空间

const int MAXN = 20; // 定义一个常量，表示杨辉三角的最大行数，避免数组越界

int main() {
    int n; // 声明变量 n，用于存储用户输入的杨辉三角行数

    cout << "请输入要生成的杨辉三角的行数 (1-" << MAXN << "): "; // 提示用户输入
    cin >> n; // 读取用户输入的行数

    // 检查输入是否合法
    if (n <= 0 || n > MAXN) {
        cout << "行数输入不合法，请在 1 到 " << MAXN << " 之间选择。" << endl;
        return 1; // 退出程序，返回错误码
    }

    int a[MAXN][MAXN]; // 声明一个二维数组 a 来存储杨辉三角的数字

    // 开始生成杨辉三角
    for (int i = 0; i < n; i++) { // 外层循环：遍历每一行 (i 从 0 到 n-1)
        // 为了让杨辉三角居中显示，先打印一些空格
        // 第 i 行前面需要打印 (n - 1 - i) * 2 个空格（假设每个数字占2个字符宽度）
        // 这里为了简单，我们先打印 n-i 个空格，效果会稍微偏左，但易于理解
        for (int k = 0; k < (n - 1 - i) * 2; k++) { 
            cout << " ";
        }

        for (int j = 0; j <= i; j++) { // 内层循环：遍历当前行 i 中的每一个数字 (j 从 0 到 i)
            if (j == 0 || j == i) { // 如果是当前行的第一个数字或最后一个数字
                a[i][j] = 1; // 赋值为 1
            } else { // 如果是当前行中间的数字
                // 根据杨辉三角的规律：当前数字等于它正上方和左上方两个数字之和
                a[i][j] = a[i - 1][j - 1] + a[i - 1][j];
            }
            // 打印当前数字，并设置宽度，让数字对齐
            cout << setw(4) << a[i][j]; // setw(4) 表示占用4个字符的宽度，不足则用空格填充
        }
        cout << endl; // 每行打印完毕后换行
    }

    return 0; // 程序正常结束
}

```

**代码解释：**
1.  `const int MAXN = 20;`: 我们定义了一个常量 `MAXN`，表示二维数组的最大尺寸。这有助于我们控制杨辉三角的最大行数，并避免数组越界。
2.  `int a[MAXN][MAXN];`: 声明了一个 `20x20` 的二维数组，用来存储杨辉三角的数值。
3.  `for (int i = 0; i < n; i++)`: 外层循环控制行数。`i` 从 `0` 开始，表示第0行、第1行...直到第 `n-1` 行。
4.  `for (int k = 0; k < (n - 1 - i) * 2; k++)`: 这个循环用来打印每行前面的空格，让杨辉三角看起来居中对齐。行数越多，前面的空格越少。
5.  `for (int j = 0; j <= i; j++)`: 内层循环控制当前行中的列数。注意，第 `i` 行有 `i+1` 个数字，所以 `j` 要从 `0` 到 `i`。
6.  `if (j == 0 || j == i)`: 这是判断当前数字是不是每行的第一个或最后一个数字。如果是，它们的值就是 `1`。
7.  `else { a[i][j] = a[i - 1][j - 1] + a[i - 1][j]; }`: 这是杨辉三角的核心规律！当前数字等于它上一行的左上方数字和正上方数字的和。
8.  `cout << setw(4) << a[i][j];`: 使用 `setw(4)` 来设置输出宽度为4，这样每个数字都会占用4个字符的空间，不足4个字符的会在前面补空格，让数字对齐，输出更整齐。`setw` 需要 `#include <iomanip>`，但我们已经用了万能头文件 `#include <bits/stdc++.h>`，所以可以直接使用。
9.  `cout << endl;`: 每行打印完后换行，开始打印下一行。

运行这段代码，输入一个数字（比如 `5`），你就会看到一个漂亮的五行杨辉三角啦！

## 练习题目

### 选择题/填空题

1.  杨辉三角的第6行（从第0行开始计数）的第2个数字（从第0个数字开始计数）是多少？
    A. 1
    B. 5
    C. 10
    D. 15

2.  在用二维数组 `a[N][N]` 存储杨辉三角时，如果 `a[i][j]` 表示第 `i` 行第 `j` 列的数（都从0开始计数），那么 `a[i][j]` 的值通常由哪两个数相加得到？
    A. `a[i-1][j]` 和 `a[i-1][j+1]`
    B. `a[i][j-1]` 和 `a[i][j+1]`
    C. `a[i-1][j-1]` 和 `a[i-1][j]`
    D. `a[i+1][j]` 和 `a[i+1][j+1]`

3.  如果我们要用 `int a[M][M]` 这样的二维数组存储一个包含 `n` 行的杨辉三角（`n` 从1开始计数），那么 `M` 的最小值应该至少是 ______。
    （假设 `n` 最大为 `MAX_N`，那么 `M` 至少要能容纳 `MAX_N` 行 `MAX_N` 列）

4.  下列关于杨辉三角的描述，错误的是？
    A. 每行的第一个数和最后一个数都是1。
    B. 除第0行外，每个数都等于它上方相邻的两个数之和。
    C. 杨辉三角是左右对称的。
    D. 第 `n` 行（从第0行开始计数）有 `n` 个数字。

5.  如果杨辉三角的第4行是 `1 4 6 4 1`，那么它的第5行（从第0行开始计数）的中间两个数字分别是 ______ 和 ______。

---

### 答案与解析

1.  **答案：C**
    **解析：**
    第0行: 1
    第1行: 1 1
    第2行: 1 2 1
    第3行: 1 3 3 1
    第4行: 1 4 6 4 1
    第5行: 1 5 10 10 5 1
    第6行: 1 6 15 20 15 6 1
    第6行第2个数字（`a[6][2]`）是15。

2.  **答案：C**
    **解析：** 杨辉三角的规律是，当前数字等于它正上方（`a[i-1][j]`）和左上方（`a[i-1][j-1]`）两个数字之和。

3.  **答案：n (或 MAX_N)**
    **解析：** 如果要存储 `n` 行的杨辉三角，由于第 `i` 行有 `i+1` 个数字，且 `i` 最大为 `n-1`，所以数组的行和列至少要能达到 `n`。例如，要存储第 `n-1` 行的第 `n-1` 个数字 `a[n-1][n-1]`，那么数组的下标最大需要到 `n-1`，所以数组大小至少是 `n x n`。

4.  **答案：D**
    **解析：**
    A. 正确。这是杨辉三角的基本性质。
    B. 正确。这是杨辉三角的核心生成规律。
    C. 正确。杨辉三角的每一行都是对称的。
    D. 错误。第 `n` 行（从第0行开始计数）有 `n+1` 个数字。例如，第0行有1个数字，第1行有2个数字。

5.  **答案：10 和 10**
    **解析：**
    第4行: 1 4 6 4 1
    根据规律：
    第5行的第一个数字是1。
    第5行的第二个数字是 `1 + 4 = 5`。
    第5行的第三个数字是 `4 + 6 = 10`。
    第5行的第四个数字是 `6 + 4 = 10`。
    第5行的第五个数字是 `4 + 1 = 5`。
    第5行的第六个数字是1。
    所以第5行是 `1 5 10 10 5 1`。中间两个数字是 `10` 和 `10`。
