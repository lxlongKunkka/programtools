## 知识点讲解

在 Python 里，我们让电脑进行数学运算，需要用到一些特殊的符号，它们就是“运算符”。

1.  **加法运算符 `+`**
    *   和我们平时数学中的加号一样，用来求和。
    *   例如：`3 + 5` 的结果是 `8`。

2.  **减法运算符 `-`**
    *   和我们平时数学中的减号一样，用来求差。
    *   例如：`10 - 4` 的结果是 `6`。

3.  **乘法运算符 `*`**
    *   **注意啦！** 在编程中，我们不用 `x` 来表示乘法，而是用星号 `*`。
    *   例如：`2 * 6` 的结果是 `12`。

4.  **除法运算符 `/`**
    *   这个符号表示除法。
    *   **特别注意：** 在 Python 中，使用 `/` 进行除法运算，结果通常会是一个小数（浮点数），即使能整除也是小数。
    *   例如：`10 / 2` 的结果是 `5.0`。
    *   例如：`7 / 2` 的结果是 `3.5`。

5.  **整数除法运算符 `//`**
    *   有时候我们只关心除法结果的整数部分，不想要小数。比如 10 块饼干分给 3 个人，每人能分到几块完整的饼干？
    *   这时，我们就可以使用 `//` 运算符。它会向下取整，只保留商的整数部分。
    *   例如：`10 // 3` 的结果是 `3`。 (因为 10 除以 3 是 3 余 1，只取整数部分 3)
    *   例如：`7 // 2` 的结果是 `3`。

6.  **取模（求余数）运算符 `%`**
    *   小朋友们，你们知道“余数”是什么吗？比如 10 块饼干分给 3 个人，每人分 3 块，还剩下 1 块，这个 1 就是余数。
    *   在编程中，我们用 `%` 运算符来求余数。
    *   例如：`10 % 3` 的结果是 `1`。
    *   例如：`7 % 2` 的结果是 `1`。

7.  **运算符的优先级**
    *   就像我们在数学课上学的，先乘除，后加减，有括号先算括号里面的。
    *   在 Python 中也是一样：`*`、`/`、`//`、`%` 的优先级比 `+`、`-` 高。
    *   如果想改变运算顺序，可以使用小括号 `()`。
    *   例如：`2 + 3 * 4` 的结果是 `14` (先算 `3 * 4 = 12`，再算 `2 + 12 = 14`)。
    *   例如：`(2 + 3) * 4` 的结果是 `20` (先算 `2 + 3 = 5`，再算 `5 * 4 = 20`)。

## 代码示例

现在，让我们用 Python 来实际操作一下这些有趣的运算符吧！

```python
# 1. 加法运算
# 声明两个变量 a 和 b
a = 10
b = 5
s = a + b # 计算 a 加 b 的和，并将结果存入变量 s
print("加法运算：", s) # 输出结果

print("直接计算：", 7 + 3) # 也可以直接计算并输出

# 2. 减法运算
a = 20
b = 8
c = a - b # 计算 a 减 b 的差，并将结果存入变量 c
print("减法运算：", c) # 输出结果

print("直接计算：", 15 - 6)

# 3. 乘法运算
a = 4
b = 3
p = a * b # 计算 a 乘以 b 的积，并将结果存入变量 p
print("乘法运算：", p) # 输出结果

print("直接计算：", 5 * 7)

# 4. 除法运算 (结果是小数)
a = 10
b = 4
d = a / b # 计算 a 除以 b 的商
print("除法运算 (结果是小数)：", d) # 输出结果

print("直接计算：", 12 / 3) # 即使能整除，结果也是小数 4.0

# 5. 整数除法运算 (只取整数部分)
a = 10
b = 3
i = a // b # 计算 a 除以 b 的商的整数部分
print("整数除法运算：", i) # 输出结果

print("直接计算：", 17 // 5)

# 6. 取模运算 (求余数)
a = 10
b = 3
r = a % b # 计算 a 除以 b 的余数
print("取模运算 (求余数)：", r) # 输出结果

print("直接计算：", 20 % 6)

# 7. 运算符优先级示例
e = 2 + 3 * 4 # 先乘后加
f = (2 + 3) * 4 # 有括号先算括号里的
print("优先级示例 1 (2 + 3 * 4)：", e)
print("优先级示例 2 ((2 + 3) * 4)：", f)
```

## 练习题目

**一、选择题**

1.  在 Python 中，表示乘法运算的符号是哪个？
    A. `x`
    B. `*`
    C. `/`
    D. `+`

2.  `print(15 / 4)` 的输出结果是什么？
    A. `3`
    B. `3.0`
    C. `3.75`
    D. `4`

3.  `print(15 // 4)` 的输出结果是什么？
    A. `3`
    B. `3.0`
    C. `3.75`
    D. `4`

4.  如果 `a = 7`，`b = 3`，那么 `a % b` 的结果是？
    A. `0`
    B. `1`
    C. `2`
    D. `3`

5.  `print(5 + 2 * 3)` 的输出结果是什么？
    A. `11`
    B. `21`
    C. `17`
    D. `7`

**二、填空题**

1.  `print(12 - 5 + 3)` 的输出结果是 `_______`。
2.  `print(6 * 4 / 2)` 的输出结果是 `_______`。
3.  `print((10 + 2) // 3)` 的输出结果是 `_______`。
4.  计算 `25` 除以 `7` 的余数，可以使用运算符 `_______`，结果是 `_______`。

---

**答案与解析**

**一、选择题答案**

1.  **B. `*`**
    *   解析：在 Python 编程中，星号 `*` 用于表示乘法运算。

2.  **C. `3.75`**
    *   解析：`/` 是普通除法，结果是浮点数（小数）。15 除以 4 等于 3.75。

3.  **A. `3`**
    *   解析：`//` 是整数除法，只取商的整数部分。15 除以 4 等于 3 余 3，所以整数部分是 3。

4.  **B. `1`**
    *   解析：`%` 是取模运算符，用于求余数。7 除以 3 等于 2 余 1，所以余数是 1。

5.  **A. `11`**
    *   解析：根据运算符优先级，先乘除后加减。`2 * 3` 等于 `6`，然后 `5 + 6` 等于 `11`。

**二、填空题答案**

1.  `10`
    *   解析：`12 - 5 = 7`，`7 + 3 = 10`。

2.  `12.0`
    *   解析：乘除优先级相同，从左到右计算。`6 * 4 = 24`，`24 / 2 = 12.0`（注意普通除法结果是小数）。

3.  `4`
    *   解析：先计算括号里的 `10 + 2 = 12`，然后 `12 // 3 = 4`。

4.  `%`，`4`
    *   解析：取模运算符 `%` 用于求余数。25 除以 7 等于 3 余 4，所以结果是 4。
