## 知识点讲解

我们之前学习了加法 `+`、减法 `-`、乘法 `*` 和除法 `/`。今天我们要认识两位新朋友：**整除 `//`** 和 **取余 `%`**。

### 1. 整除运算符 `//`

*   **作用**：它用来计算两个数相除的**整数商**。也就是说，它只会保留商的整数部分，小数部分会被直接“扔掉”。
*   **读法**：我们可以叫它“双斜杠整除”或“地板除”。
*   **举例**：
    *   `10 // 3` 的结果是 `3` (因为10除以3等于3余1，它只取整数3)。
    *   `7 // 2` 的结果是 `3` (因为7除以2等于3余1，它只取整数3)。
    *   `8 // 4` 的结果是 `2` (因为8除以4等于2，没有余数，它也取整数2)。

### 2. 取余运算符 `%`

*   **作用**：它用来计算两个数相除的**余数**。
*   **读法**：我们可以叫它“百分号取余”或“模运算”。
*   **举例**：
    *   `10 % 3` 的结果是 `1` (因为10除以3等于3余1，它只取余数1)。
    *   `7 % 2` 的结果是 `1` (因为7除以2等于3余1，它只取余数1)。
    *   `8 % 4` 的结果是 `0` (因为8除以4等于2，没有余数，它就取余数0)。

### 3. 整除和取余的应用场景

这两个运算符在编程中非常有用，可以帮助我们解决很多问题：

*   **判断奇偶数**：一个数如果能被2整除（余数为0），它就是偶数；如果不能被2整除（余数为1），它就是奇数。
*   **时间转换**：比如把分钟转换成小时和分钟。
*   **数字拆分**：把一个数的个位、十位、百位分离出来。
*   **轮流操作**：比如每隔N个进行一次特殊操作。

## 代码示例

让我们用 Python 来实际操作一下，看看 `//` 和 `%` 是怎么工作的。

```python
# 示例1：分糖果问题
print("--- 分糖果问题 ---")
a = 10 # 糖果总数
b = 3  # 小朋友人数

# 计算每个小朋友分到的糖果数（整数商）
s = a // b
print(f"{a} 颗糖果分给 {b} 个小朋友，每个小朋友分到 {s} 颗。")

# 计算剩下的糖果数（余数）
y = a % b
print(f"还剩下 {y} 颗糖果。")

print("\n--- 更多整除例子 ---")
print(f"15 // 4 的结果是：{15 // 4}") # 15除以4等于3余3，只取3
print(f"20 // 6 的结果是：{20 // 6}") # 20除以6等于3余2，只取3
print(f"9 // 3 的结果是：{9 // 3}")   # 9除以3等于3，只取3

print("\n--- 更多取余例子 ---")
print(f"15 % 4 的结果是：{15 % 4}") # 15除以4等于3余3，只取3
print(f"20 % 6 的结果是：{20 % 6}") # 20除以6等于3余2，只取2
print(f"9 % 3 的结果是：{9 % 3}")   # 9除以3等于3，只取0

print("\n--- 应用：判断奇偶数 ---")
n = 7
if n % 2 == 0: # 如果 n 除以 2 的余数是 0，说明是偶数
    print(f"{n} 是偶数。")
else: # 否则是奇数
    print(f"{n} 是奇数。")

m = 12
if m % 2 == 0:
    print(f"{m} 是偶数。")
else:
    print(f"{m} 是奇数。")

print("\n--- 应用：时间转换 ---")
t = 135 # 总分钟数
h = t // 60 # 计算小时数
s = t % 60  # 计算剩余分钟数

print(f"{t} 分钟是 {h} 小时 {s} 分钟。")

```

**代码解释：**
*   `a = 10` 和 `b = 3`：我们定义了两个变量，用来存储糖果总数和小朋友人数。
*   `s = a // b`：这里使用了整除运算符 `//`，计算 `10` 除以 `3` 的整数商，结果是 `3`，并把 `3` 存到变量 `s` 里。
*   `y = a % b`：这里使用了取余运算符 `%`，计算 `10` 除以 `3` 的余数，结果是 `1`，并把 `1` 存到变量 `y` 里。
*   `if n % 2 == 0:`：这是一个条件判断语句。如果一个数 `n` 除以 `2` 的余数等于 `0`，那么它就是偶数，否则就是奇数。
*   `h = t // 60` 和 `s = t % 60`：这里将总分钟数 `t` 通过整除 `//` 得到小时数 `h`，通过取余 `%` 得到剩余的分钟数 `s`。

## 练习题目

**一、选择题**

1.  在 Python 中，用于计算整数商的运算符是哪个？
    A. `/`
    B. `*`
    C. `//`
    D. `%`

2.  表达式 `20 % 3` 的结果是多少？
    A. 6
    B. 2
    C. 0
    D. 3

3.  下列哪个表达式可以用来判断一个变量 `x` 是否为偶数？
    A. `x // 2 == 0`
    B. `x % 2 == 1`
    C. `x % 2 == 0`
    D. `x / 2 == 0`

4.  如果 `a = 15`，`b = 4`，那么 `a // b` 和 `a % b` 的结果分别是？
    A. `3, 3`
    B. `3, 4`
    C. `4, 3`
    D. `3.75, 3`

**二、填空题**

1.  在 Python 中，`17 // 3` 的结果是 \_\_\_\_\_\_。
2.  在 Python 中，`17 % 3` 的结果是 \_\_\_\_\_\_。
3.  要计算一个三位数 `n` 的个位数字，可以使用表达式 `n % ______`。
4.  要计算一个三位数 `n` 的百位数字，可以使用表达式 `n // ______`。

---

**答案与解析**

**一、选择题答案**

1.  **C. `//`**
    *   解析：`/` 是普通除法，结果可能是小数；`*` 是乘法；`%` 是取余；`//` 才是整除，只取整数商。

2.  **B. 2**
    *   解析：20 除以 3 等于 6 余 2。取余运算符 `%` 返回的是余数，所以结果是 2。

3.  **C. `x % 2 == 0`**
    *   解析：偶数的定义是能被2整除的数，即除以2余数为0。`x // 2 == 0` 只能判断 `x` 是否小于2；`x % 2 == 1` 判断的是奇数；`x / 2 == 0` 只有当 `x` 为0时才成立。

4.  **A. `3, 3`**
    *   解析：`15 // 4` 结果是 3 (15除以4等于3余3，取整数商3)。`15 % 4` 结果是 3 (取余数3)。

**二、填空题答案**

1.  **5**
    *   解析：17 除以 3 等于 5 余 2，整除 `//` 取整数商 5。

2.  **2**
    *   解析：17 除以 3 等于 5 余 2，取余 `%` 取余数 2。

3.  **10**
    *   解析：任何一个整数除以 10 的余数就是它的个位数字。例如 `123 % 10` 等于 `3`。

4.  **100**
    *   解析：一个三位数 `n` 除以 100 的整数商就是它的百位数字。例如 `123 // 100` 等于 `1`。
