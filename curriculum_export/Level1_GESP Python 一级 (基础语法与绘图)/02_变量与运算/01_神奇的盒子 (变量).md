## 知识点讲解

### 1. 什么是变量？——你的魔法盒子

**变量**，就像我们刚才说的，是计算机内存中用来存储数据（信息）的一个“小容器”或“小盒子”。每个盒子都有一个独特的“标签”或“名字”，我们通过这个名字来找到并使用盒子里的数据。

**核心特点：**
*   **有名字**：每个变量都有一个独一无二的名字，方便我们识别。
*   **装数据**：变量可以存储各种类型的数据，比如数字、文字等。
*   **会变化**：变量里存储的数据是可以改变的，这也是它为什么叫“变量”的原因。

### 2. 变量的命名规则——给盒子起名字的规矩

给你的魔法盒子起名字可不是随便的哦，Python有一些小规矩：

*   **开头必须是字母或下划线 `_`**：不能用数字开头。
*   **名字中可以包含字母、数字和下划线 `_`**。
*   **不能使用Python的“关键字”**：Python有一些特殊的词语有自己的用途，比如 `if`, `for`, `while` 等，我们不能用它们做变量名。（现在记住几个就行，以后会学到更多）
*   **区分大小写**：`a` 和 `A` 是两个不同的变量名！
*   **信奥小贴士**：在算法竞赛中，为了代码简洁和书写方便，我们经常会使用单个字母（如 `n`, `m`, `i`, `j`, `k`）或者非常简短的字母组合来作为变量名。

**好的变量名示例：**
*   `x`
*   `n`
*   `s`
*   `_a`
*   `y1`

**不好的变量名示例：**
*   `1n` (数字开头)
*   `for` (是关键字)
*   `my age` (有空格)

### 3. 变量的赋值——把东西放进盒子里

我们使用等号 `=` 来把数据“放进”变量这个“盒子”里，这个过程叫做**赋值**。

**语法格式：**
```
变量名 = 值
```

例如，把数字 `10` 放进名叫 `x` 的盒子里：
```python
x = 10
```

### 4. 变量的值——盒子里的宝贝

变量里存储的数据就是它的“值”。这个值是可以变化的。

例如，我们先给 `x` 赋值 `10`，然后又给 `x` 赋值 `20`。那么 `x` 里面最后的值就是 `20` 了。旧的 `10` 就会被新的 `20` 替换掉。

```python
x = 10  # x 盒子现在装的是 10
print(x) # 打印 x 盒子里面的东西，结果是 10

x = 20  # x 盒子现在装的是 20，把原来的 10 替换了
print(x) # 打印 x 盒子里面的东西，结果是 20
```

### 5. 基本数据类型（初步认识）——盒子能装什么？

变量这个魔法盒子能装各种各样的东西。最常见的有：

*   **整数 (int)**：没有小数点的数字，比如 `1`, `100`, `-5`。
    ```python
    a = 100
    ```
*   **浮点数 (float)**：有小数点的数字，比如 `3.14`, `0.5`, `-1.2`。
    ```python
    b = 3.14
    ```
*   **字符串 (str)**：文字信息，需要用单引号 `' '` 或双引号 `" "` 括起来。比如 `'你好'`, `"Python"`。
    ```python
    s = "编程真有趣"
    ```

## 代码示例

让我们来动手写一些代码，看看这些魔法盒子是怎么工作的！

```python
# 示例1：声明并给整数变量赋值
# 我们声明一个变量 n，并把整数 10 赋值给它
n = 10
print("变量 n 里面装的是：", n) # 输出 n 里面的值

# 示例2：声明并给浮点数变量赋值
# 我们声明一个变量 p，并把浮点数 3.14 赋值给它
p = 3.14159
print("变量 p 里面装的是：", p) # 输出 p 里面的值

# 示例3：声明并给字符串变量赋值
# 我们声明一个变量 s，并把字符串 "小明" 赋值给它
s = "小明"
print("变量 s 里面装的是：", s) # 输出 s 里面的值

# 示例4：改变变量的值
# 变量 x 最初装的是 5
x = 5
print("x 盒子最初装的是：", x)

# 现在我们把 x 盒子里的东西换成 12
x = 12
print("x 盒子现在装的是：", x) # 看看 x 盒子里的东西是不是变了？

# 示例5：一个综合应用，存储你的信息
# 声明一个变量 name 来存储你的名字
name = "编程小天才"
# 声明一个变量 age 来存储你的年龄
age = 8
# 声明一个变量 height 来存储你的身高（比如 1.35 米）
height = 1.35

print("我的名字是：", name)
print("我今年：", age, "岁")
print("我的身高是：", height, "米")
```

### 练习题目答案与解析

**选择题答案：**

1.  **C. `_score`**
    *   A. `2_name` 不合法，因为变量名不能以数字开头。
    *   B. `my-age` 不合法，因为变量名不能包含 `-` 符号（它在Python中是减号）。
    *   D. `class` 是Python的关键字，不能用作变量名。

2.  **B. `x = 25`**
    *   A. `x == 25` 是判断 `x` 是否等于 `25`，而不是赋值。
    *   C. `25 = x` 是错误的赋值方式，值不能放在等号的左边。
    *   D. `print(x, 25)` 是打印输出，不是赋值。

3.  **C. 30**
    *   `k = 10` (k 现在是 10)
    *   `k = k + 5` (k 现在是 10 + 5 = 15)
    *   `k = 2 * k` (k 现在是 2 * 15 = 30)

4.  **B. `s = '李华'`**
    *   字符串（文字）必须用单引号或双引号括起来。
    *   A. `s = 李华` 会报错，因为“李华”会被当作变量名而不是字符串。
    *   C. `s = 123` 赋值的是数字。
    *   D. `s = True` 赋值的是布尔值（真/假）。

**填空题答案：**

1.  数据 / 信息
2.  `=`
3.  浮点数 (float)
4.  数字，关键字

**编程题参考答案：**

**编程题 1：**

```python
# 声明变量并赋值
a = 12345
b = '三年二班'
c = 2

# 打印变量的值
print("我的学号是：", a)
print("我的班级是：", b)
print("我今天吃了：", c, "个苹果")
```

**编程题 2：**

```python
# 声明并初始化变量
x = 100
y = 200

print("交换前：")
print("x =", x)
print("y =", y)

# 交换 x 和 y 的值（Python 特色写法）
x, y = y, x

# 或者使用算术运算交换（不推荐，但作为知识点了解）
# x = x + y  # x 变成 300 (100+200)
# y = x - y  # y 变成 100 (300-200)
# x = x - y  # x 变成 200 (300-100)

print("\n交换后：")
print("x =", x)
print("y =", y)
```
