## 知识点讲解

### 1. 什么是条件判断？
条件判断就是根据一个事情是否成立（是“真”还是“假”），来决定接下来要做什么。比如“下雨了吗？”这是一个条件。如果答案是“真”（下雨了），我们就做一件事；如果答案是“假”（没下雨），我们就做另一件事。

### 2. 比较运算符
在Python中，我们用一些特殊的符号来表达条件，它们叫做**比较运算符**：

| 运算符 | 含义         | 举例           | 结果 (真/假) |
| :----- | :----------- | :------------- | :----------- |
| `==`   | 等于         | `5 == 5`       | `True` (真)  |
| `!=`   | 不等于       | `5 != 10`      | `True` (真)  |
| `>`    | 大于         | `10 > 5`       | `True` (真)  |
| `<`    | 小于         | `5 < 10`       | `True` (真)  |
| `>=`   | 大于等于     | `10 >= 10`     | `True` (真)  |
| `<=`   | 小于等于     | `5 <= 10`      | `True` (真)  |

这些运算符会返回 `True` (真) 或 `False` (假) 两种结果，这就是我们电脑做判断的依据。

### 3. `if-else` 双分支结构
`if-else` 结构就像一个岔路口，电脑会根据条件选择走哪条路。

*   **`if` 部分**：如果条件是 `True`（真），就执行 `if` 后面的代码。
*   **`else` 部分**：如果条件是 `False`（假），就执行 `else` 后面的代码。

**语法结构：**

```python
if 条件:
    # 如果条件为真，执行这里的代码
    # 注意：这里的代码需要缩进！
else:
    # 如果条件为假，执行这里的代码
    # 注意：这里的代码也需要缩进！
```

**重要提示：**
*   `if` 和 `else` 后面都必须跟着一个冒号 `:`。
*   冒号后面的代码必须**缩进**（通常是 4 个空格或一个 Tab 键）。Python就是靠缩进来判断哪些代码属于 `if`，哪些属于 `else`。
*   在 `if-else` 结构中，`if` 或 `else` 后面的代码块**只会有一个被执行**，不可能两个都执行，也不可能两个都不执行。

让我们通过一个例子来理解：判断一个数是偶数还是奇数。
*   **条件**：这个数能被2整除吗？（即 `n % 2 == 0`）
*   **如果**能被2整除，**那么**它是偶数。
*   **否则**（不能被2整除），**那么**它是奇数。

## 代码示例

### 示例 1：判断一个数是偶数还是奇数

```python
# 获取用户输入的整数
n = int(input("请输入一个整数："))

# 使用 if-else 判断 n 是偶数还是奇数
if n % 2 == 0:  # 如果 n 除以 2 的余数是 0，说明是偶数
    print(f"{n} 是偶数。") # f-string 格式化输出，方便显示变量值
else: # 否则，说明是奇数
    print(f"{n} 是奇数。")
```

### 示例 2：判断学生是否及格

```python
# 获取学生分数
s = int(input("请输入你的分数 (0-100)："))

# 判断分数是否及格（假设60分及格）
if s >= 60: # 如果分数大于等于 60
    print("恭喜你，你及格了！")
else: # 否则（分数小于 60）
    print("很遗憾，你需要继续努力！")
```

### 示例 3：比较两个数的大小

```python
# 获取两个数字
a = int(input("请输入第一个整数 a："))
b = int(input("请输入第二个整数 b："))

# 判断哪个数更大
if a > b: # 如果 a 大于 b
    print(f"数字 {a} 更大。")
else: # 否则（a 小于或等于 b）
    print(f"数字 {b} 更大或两者相等。")
```

### 编程题

1.  **基础练习：正数与非正数**
    编写一个程序，接收用户输入的一个整数 `k`。
    如果 `k` 是正数（大于0），则打印 "这是一个正数。"
    否则（`k` 小于或等于0），则打印 "这是一个非正数。"

2.  **基础练习：高温警示**
    编写一个程序，接收用户输入今天的气温 `t`（一个整数）。

3.  **进阶练习：购物折扣**
    小明去商店买东西，总价是 `p` 元。
    如果 `p` 大于等于 100 元，商店会给打九折（即价格乘以0.9）。
    否则，不打折。
    请编写一个程序，接收总价 `p`，然后计算并打印出小明最终需要支付的金额（保留两位小数）。

### 答案与解析

**选择题/填空题答案：**

1.  **B. `==`**
    *   解析：`=` 是赋值运算符，用于给变量赋值；`==` 才是比较运算符，用于判断是否相等。

2.  **A. `True`**
    *   解析：`5 + 3` 的结果是 `8`，`8 == 8` 显然是 `True`。

3.  **B. `未成年人`**
    *   解析：`age` 是 `16`，`16 >= 18` 的结果是 `False`，所以会执行 `else` 后面的代码。

4.  **C. `:` (冒号)**
    *   解析：`if` 条件和 `else` 关键词后面必须加上冒号，表示代码块的开始。

5.  **C. 缩进**
    *   解析：Python 使用严格的缩进来区分代码块，这是它的一大特色。

**编程题答案：**

1.  **正数与非正数**
    ```python
    k = int(input("请输入一个整数 k："))
    if k > 0:
        print("这是一个正数。")
    else:
        print("这是一个非正数。")
    ```

2.  **高温警示**
    ```python
    t = int(input("请输入今天的气温 t："))
    if t > 30:
        print("小心中暑，今天很热！")
    else:
        print("今天气温适宜。")
    ```

3.  **购物折扣**
    ```python
    p = float(input("请输入购物总价 p：")) # 购物总价可能是小数
    final_p = 0.0 # 初始化最终价格

    if p >= 100: # 如果总价大于等于100
        final_p = p * 0.9 # 打九折
    else: # 否则
        final_p = p # 不打折

    print(f"最终需要支付的金额是：{final_p:.2f} 元。") # 保留两位小数输出
    ```
