## 知识点讲解

### 1. 什么是布尔值（Boolean Value）？

在计算机科学中，“真”和“假”被称为布尔值（Boolean Value），以英国数学家乔治·布尔的名字命名。它只有两种可能的值：
*   **True** (真)：表示条件成立、正确、是。
*   **False** (假)：表示条件不成立、错误、否。

在 Python 语言中，这两个值是特殊的关键字，并且**首字母必须大写**：`True` 和 `False`。

### 2. 如何在 Python 中表示真与假？

直接使用 `True` 和 `False` 关键字：

```python
x = True
y = False
print(x)
print(y)
```

### 3. 比较运算符：判断的“魔法棒”

我们怎么让计算机判断一件事是真还是假呢？这就需要用到“比较运算符”了！它们就像魔法棒一样，能比较两个值，然后告诉你结果是 `True` 还是 `False`。

| 运算符 | 含义             | 举例           | 结果   |
| :----- | :--------------- | :------------- | :----- |
| `==`   | 等于             | `5 == 5`       | `True` |
| `!=`   | 不等于           | `5 != 10`      | `True` |
| `>`    | 大于             | `10 > 5`       | `True` |
| `<`    | 小于             | `5 < 10`       | `True` |
| `>=`   | 大于或等于       | `5 >= 5`       | `True` |
| `<=`   | 小于或等于       | `5 <= 10`      | `True` |

**注意：**
*   `==` 是判断“等于”，不要和 `=` （赋值运算符）搞混哦！`=` 是把右边的值放进左边的变量里，而 `==` 是比较左右两边是不是一样。
*   比较运算符不仅可以比较数字，还可以比较字符串等其他类型的数据。

### 4. 布尔值的应用：简单的条件判断（`if` 语句初探）

布尔值最常见的应用就是进行条件判断。当一个条件是 `True` 时，我们就执行某些操作；当条件是 `False` 时，就执行另外一些操作（或者不执行）。

这是一个最简单的 `if` 语句：

```python
a = 10
b = 5

if a > b:  # 如果 a 大于 b 这个条件为 True
    print("a 比 b 大！")
```
在这个例子中，`a > b` 的结果是 `True`，所以 `print("a 比 b 大！")` 这行代码就会被执行。

## 代码示例

让我们来动手写一些 Python 代码，看看布尔值和比较运算符是怎么工作的！

```python
# 示例 1：直接定义布尔值
print("--- 示例 1：直接定义布尔值 ---")
x = True
y = False
print(f"变量 x 的值是：{x}")
print(f"变量 y 的值是：{y}")
print(f"x 的类型是：{type(x)}") # 查看变量的类型

print("\n--- 示例 2：使用比较运算符 ---")
# 示例 2：使用比较运算符
a = 15
b = 8
c = 15

# 判断是否相等
s1 = (a == b)  # a 等于 b 吗？
print(f"{a} == {b} 的结果是：{s1}")

s2 = (a == c)  # a 等于 c 吗？
print(f"{a} == {c} 的结果是：{s2}")

# 判断是否不相等
s3 = (a != b)  # a 不等于 b 吗？
print(f"{a} != {b} 的结果是：{s3}")

# 判断大小
s4 = (a > b)   # a 大于 b 吗？
print(f"{a} > {b} 的结果是：{s4}")

s5 = (b < c)   # b 小于 c 吗？
print(f"{b} < {c} 的结果是：{s5}")

s6 = (a >= c)  # a 大于或等于 c 吗？
print(f"{a} >= {c} 的结果是：{s6}")

s7 = (b <= a)  # b 小于或等于 a 吗？
print(f"{b} <= {a} 的结果是：{s7}")

print("\n--- 示例 3：布尔值与变量 ---")
# 示例 3：将比较结果赋值给变量
is_adult = (18 >= 18) # 判断年龄是否大于等于18
print(f"18 >= 18 的结果是：{is_adult}")

is_passed = (90 > 60) # 判断分数是否及格
print(f"90 > 60 的结果是：{is_passed}")

print("\n--- 示例 4：布尔值在 if 语句中的应用 ---")
# 示例 4：布尔值在 if 语句中的应用
is_sunny = True
if is_sunny: # 如果 is_sunny 是 True
    print("今天天气真好，可以出去玩啦！")
else: # 否则 (is_sunny 是 False)
    print("今天下雨了，在家看书吧。")

temperature = 25
if temperature > 30:
    print("天气好热，开空调！")
else:
    print("温度适中，很舒服。")
```

## 练习题目

**一、选择题**

1.  在 Python 中，表示“真”的关键字是哪个？
    A.  `true`
    B.  `True`
    C.  `TRUE`
    D.  `1`

2.  下面哪一个 Python 表达式的结果是 `False`？
    A.  `10 == 10`
    B.  `7 != 8`
    C.  `5 > 10`
    D.  `20 >= 20`

3.  如果 `a = 5`，那么表达式 `a * 2 > 10` 的结果是？
    A.  `True`
    B.  `False`
    C.  `10`
    D.  `无法判断`

4.  下面哪一个比较运算符表示“不等于”？
    A.  `=`
    B.  `==`
    C.  `!=`
    D.  `<>`

**二、填空题**

1.  Python 中的布尔类型只有两个值，分别是 `____` 和 `____`。
2.  表达式 `(3 + 4) <= 7` 的结果是 `____`。
3.  如果 `x = True`，那么 `print(x)` 会输出 `____`。
4.  表达式 `type(False)` 的结果是 `____`。

---

**答案与解析**

**一、选择题答案**

1.  **B. `True`**
    *   解析：在 Python 中，布尔值 `True` 和 `False` 的首字母必须大写。

2.  **C. `5 > 10`**
    *   解析：
        *   A. `10 == 10` 结果是 `True`。
        *   B. `7 != 8` 结果是 `True`。
        *   C. `5 > 10` 结果是 `False` (因为 5 不大于 10)。
        *   D. `20 >= 20` 结果是 `True` (因为 20 等于 20，所以也满足大于或等于)。

3.  **B. `False`**
    *   解析：首先计算 `a * 2`，即 `5 * 2 = 10`。然后判断 `10 > 10`，结果是 `False` (因为 10 不大于 10)。

4.  **C. `!=`**
    *   解析：
        *   `=` 是赋值运算符。
        *   `==` 是等于运算符。
        *   `!=` 是不等于运算符。
        *   `<>` 在 Python 2 中表示不等于，但在 Python 3 中已被 `!=` 取代。

**二、填空题答案**

1.  `True`, `False`
    *   解析：布尔类型只有这两个值。

2.  `True`
    *   解析：先计算 `3 + 4` 得到 `7`。然后判断 `7 <= 7`，因为 7 等于 7，所以结果是 `True`。

3.  `True`
    *   解析：`print()` 函数会直接输出变量 `x` 所存储的布尔值 `True`。

4.  `<class 'bool'>`
    *   解析：`type()` 函数用于查看数据类型，`False` 是布尔类型，所以结果是 `<class 'bool'>`。
