## 知识点讲解

### 1. 什么是 `if` 条件语句？

`if` 语句是编程中用来实现“选择”或“分支”逻辑的工具。它的意思就是“如果某个条件成立，就做某件事情”。

### 2. 比较运算符：判断的“眼睛”

在 `if` 语句中，我们需要告诉程序“什么条件”才算成立。这就需要用到比较运算符。它们就像程序的“眼睛”，用来比较两个值的大小或是否相等。

| 运算符 | 含义           | 举例      | 结果 (True/False) |
| :----- | :------------- | :-------- | :---------------- |
| `==`   | 等于           | `a == 5`  | `a` 等于 5 吗？   |
| `!=`   | 不等于         | `a != 5`  | `a` 不等于 5 吗？ |
| `>`    | 大于           | `a > 5`   | `a` 大于 5 吗？   |
| `<`    | 小于           | `a < 5`   | `a` 小于 5 吗？   |
| `>=`   | 大于等于       | `a >= 5`  | `a` 大于等于 5 吗？|
| `<=`   | 小于等于       | `a <= 5`  | `a` 小于等于 5 吗？|

**小提示：** 记住 `==` 是判断“等于”，而 `=` 是赋值（把右边的值给左边的变量），它们是不一样的哦！

### 3. `if` 语句的基本结构

`if` 语句最简单的形式是这样的：

```
if 条件:
    # 如果条件为真（True），就执行这里的代码
    # 注意：这里的代码需要缩进！
```

*   `if`：关键词，表示“如果”。
*   `条件`：一个会返回 `True` 或 `False`（真或假）的表达式，通常使用比较运算符。
*   `:`：冒号，表示 `if` 语句的开始。
*   `缩进的代码块`：当条件为 `True` 时，程序会执行这部分代码。Python 中，缩进（通常是4个空格）非常重要，它告诉程序哪些代码属于 `if` 语句。

### 4. `if-else` 语句：有选择，也有备选

有时候，我们不仅想在条件成立时做一件事，还想在条件不成立时做另一件事。这时，`else` 就派上用场了。

```
if 条件:
    # 如果条件为真（True），就执行这里的代码
else:
    # 如果条件为假（False），就执行这里的代码
    # 注意：这里的代码也需要缩进！
```

*   `else`：关键词，表示“否则”。当 `if` 后面的条件不满足时，程序就会自动执行 `else` 后面的代码。

### 5. `if-elif-else` 语句：多重选择

如果我们需要判断多个条件，而不仅仅是“是”或“否”，该怎么办呢？`elif`（是 `else if` 的缩写）就可以帮助我们处理这种情况。

```
if 条件1:
    # 如果条件1为真，执行这里的代码
elif 条件2:
    # 如果条件1为假，但条件2为真，执行这里的代码
elif 条件3:
    # 如果条件1、2都为假，但条件3为真，执行这里的代码
else:
    # 如果以上所有条件都为假，执行这里的代码
```

*   `elif`：可以有多个 `elif`。程序会从上到下依次检查条件，只要有一个条件为真，就会执行对应的代码块，然后跳出整个 `if-elif-else` 结构。
*   `else`：是可选的，用来处理所有条件都不满足的“兜底”情况。

**重要提示：** Python 中，缩进是非常重要的！它决定了代码的结构和执行顺序。如果缩进错了，程序就会报错或者执行结果不符合预期。

## 代码示例

让我们用代码来实践一下吧！

### 示例 1：判断一个数是否大于 10

```python
# 获取用户输入的数字
n = int(input("请输入一个整数："))

# 使用 if 语句判断
if n > 10:
    print("你输入的数字", n, "大于 10。")

print("程序结束。") # 这句话无论条件是否满足都会执行
```

**运行结果示例：**
*   输入 `12`：
    ```
    请输入一个整数：12
    你输入的数字 12 大于 10。
    程序结束。
    ```
*   输入 `5`：
    ```
    请输入一个整数：5
    程序结束。
    ```

### 示例 2：判断一个数是奇数还是偶数

```python
# 获取用户输入的数字
n = int(input("请输入一个整数："))

# 使用 if-else 语句判断
# 知识点：n % 2 是求 n 除以 2 的余数
# 如果余数是 0，说明是偶数；否则是奇数
if n % 2 == 0: # 如果 n 除以 2 的余数等于 0
    print("你输入的数字", n, "是一个偶数。")
else: # 否则（余数不等于 0）
    print("你输入的数字", n, "是一个奇数。")
```

**运行结果示例：**
*   输入 `4`：
    ```
    请输入一个整数：4
    你输入的数字 4 是一个偶数。
    ```
*   输入 `7`：
    ```
    请输入一个整数：7
    你输入的数字 7 是一个奇数。
    ```

### 3. 示例 3：根据分数判断等级

```python
# 获取用户输入的分数
s = int(input("请输入你的分数（0-100）："))

# 使用 if-elif-else 语句判断等级
if s >= 90:
    print("恭喜你，你的等级是：优秀！")
elif s >= 80: # 如果 s 小于 90 但大于等于 80
    print("你的等级是：良好！")
elif s >= 60: # 如果 s 小于 80 但大于等于 60
    print("你的等级是：及格！")
else: # 如果 s 小于 60
    print("很抱歉，你的等级是：不及格。继续努力哦！")
```

**运行结果示例：**
*   输入 `95`：
    ```
    请输入你的分数（0-100）：95
    恭喜你，你的等级是：优秀！
    ```
*   输入 `72`：
    ```
    请输入你的分数（0-100）：72
    你的等级是：及格！
    ```
*   输入 `59`：
    ```
    请输入你的分数（0-100）：59
    很抱歉，你的等级是：不及格。继续努力哦！
    ```

### 练习题目答案与解析

**选择题答案：**

1.  **B. `==`**
    *   解析：`=` 是赋值运算符，`==` 才是等于运算符。

2.  **A. `A`**
    *   解析：`x` 的值是 15，`15 > 10` 这个条件是 `True`，所以会执行 `if` 语句块中的 `print("A")`。`else` 语句块不会执行。

3.  **B. `等于5`**
    *   解析：`y` 的值是 5。
        *   首先判断 `y % 2 == 0` (即 `5 % 2 == 0`)，结果是 `1 == 0`，为 `False`。
        *   接着判断 `elif y == 5` (即 `5 == 5`)，结果是 `True`。所以会执行 `print("等于5")`，然后整个 `if-elif-else` 结构结束，不再检查 `else`。

4.  **C. `a >= 10`**
    *   解析：如果 `a > 10` 为 `True`，说明 `a` 是一个比 10 大的数（例如 11, 12 等）。
        *   A. `a < 10` 显然是 `False`。
        *   B. `a == 10` 显然是 `False`。
        *   C. `a >= 10` 意味着 `a` 大于或等于 10。既然 `a > 10` 已经成立，那么 `a >= 10` 也一定成立。
        *   D. `a != 10` 也是 `True`，但 C 选项更准确地包含了 `a > 10` 的情况。不过，严格来说，如果 `a > 10`，那么 `a != 10` 也肯定是 `True`。但是题目问的是“也一定为 `True`”，C选项是更直接的包含关系。

**填空题答案：**

1.  `n > 0`
    *   解析：正数就是大于 0 的数。

2.  `==`
    *   解析：需要使用等于运算符 `==` 来判断字符串是否相等。

3.  `冬天`
    *   解析：当 `s` 不是 1、2、3 时，就默认是 4，也就是冬天。
