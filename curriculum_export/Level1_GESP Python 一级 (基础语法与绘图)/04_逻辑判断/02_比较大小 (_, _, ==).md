## 知识点讲解

电脑非常聪明，它有一套专门的“比较工具”，叫做“比较运算符”。我们今天要学习其中最常用的三个小帮手：

1.  **大于号：`>`**
    *   它的意思和我们数学中的大于号一模一样！
    *   当左边的数字比右边的数字大时，它就会说 `True`（对的），否则就说 `False`（错的）。
    *   **例子:** `5 > 3`，电脑会说 `True`（因为5确实比3大）。
    *   **例子:** `2 > 7`，电脑会说 `False`（因为2不比7大）。

2.  **小于号：`<`**
    *   它也和我们数学中的小于号一样！
    *   当左边的数字比右边的数字小时，它就会说 `True`，否则就说 `False`。
    *   **例子:** `1 < 10`，电脑会说 `True`（因为1确实比10小）。
    *   **例子:** `8 < 4`，电脑会说 `False`（因为8不比4小）。

3.  **等于号：`==`**
    *   这个要特别注意啦！在编程里，我们要判断两个东西是不是“相等”，要用**两个等号** `==`。
    *   还记得我们之前学过的**一个等号** `=` 是用来做什么的吗？（提问：`a = 5`是什么意思？）对啦，一个等号 `=` 是用来给变量“赋值”的，就是把一个值放进一个盒子里。
    *   而**两个等号** `==` 才是用来“比较”两个值是不是一样的。
    *   当左边的数字和右边的数字完全一样时，它就会说 `True`，否则就说 `False`。
    *   **例子:** `6 == 6`，电脑会说 `True`（因为6确实等于6）。
    *   **例子:** `10 == 12`，电脑会说 `False`（因为10不等于12）。

记住，这些比较运算的结果，永远只有两种可能：`True`（真）或者 `False`（假）！

## 代码示例

现在，我们一起来看看如何在 Python 中使用这些比较运算符吧！

```python
print("--- 比较大小的魔法时刻 ---")

# 例子1：使用大于号 >
a = 10
b = 5
print(f"a = {a}, b = {b}")
print(f"a > b 的结果是: {a > b}")  # 10 > 5，结果是 True

c = 3
d = 8
print(f"c = {c}, d = {d}")
print(f"c > d 的结果是: {c > d}")  # 3 > 8，结果是 False

print("\n----------------------")

# 例子2：使用小于号 <
x = 7
y = 15
print(f"x = {x}, y = {y}")
print(f"x < y 的结果是: {x < y}")  # 7 < 15，结果是 True

m = 20
n = 10
print(f"m = {m}, n = {n}")
print(f"m < n 的结果是: {m < n}")  # 20 < 10，结果是 False

print("\n----------------------")

# 例子3：使用等于号 ==
p = 100
q = 100
print(f"p = {p}, q = {q}")
print(f"p == q 的结果是: {p == q}")  # 100 == 100，结果是 True

r = 50
s = 51
print(f"r = {r}, s = {s}")
print(f"r == s 的结果是: {r == s}")  # 50 == 51，结果是 False

print("\n----------------------")

# 小挑战：让电脑判断你输入的数字
print("小挑战：输入一个数字，看看它是不是比10大！")
num = int(input("请输入一个整数: ")) # 接收用户输入的数字
print(f"你输入的数字 {num} 比 10 大吗？结果是: {num > 10}")
```

## 练习题目

**一、选择题**

1.  在 Python 中，判断 `a` 是否“大于” `b` 的正确运算符是：
    A. `=`
    B. `==`
    C. `>`
    D. `<`

2.  表达式 `10 < 5` 的结果是什么？
    A. `True`
    B. `False`
    C. `10`
    D. `5`

3.  如果 `x = 7`，那么表达式 `x == 7` 的结果是什么？
    A. `True`
    B. `False`
    C. `7`
    D. `x`

4.  下列哪个表达式的结果是 `False`？
    A. `8 > 6`
    B. `1 == 1`
    C. `3 < 3`
    D. `100 == 100`

5.  在 Python 中，用来给变量赋值的符号是：
    A. `==`
    B. `=`
    C. `>`
    D. `<`

**二、编程挑战**

1.  **比较身高**：
    编写一个 Python 程序，接收两个同学的身高（整数，单位厘米），然后判断第一个同学是否比第二个同学高。
    示例输入：
    `140`
    `135`
    示例输出：
    `第一个同学比第二个同学高吗？ True`

2.  **判断及格**：
    编写一个 Python 程序，接收一个学生的考试分数（整数）。判断这个分数是否达到了60分（即大于等于60分）。（提示：虽然我们今天主要学 `>` `<` `==`，但你可以试试用 `>=` 来表示“大于等于”，或者用 `>` 和 `==` 来组合判断哦！）
    示例输入：
    `75`
    示例输出：
    `这个学生及格了吗？ True`

---

**答案与解析**

**一、选择题答案**

1.  **C. `>`**
    *   解析：`>` 表示大于。`=` 是赋值，`==` 是等于，`<` 是小于。
2.  **B. `False`**
    *   解析：10 并不小于 5，所以结果是 `False`。
3.  **A. `True`**
    *   解析：`x` 的值是 7，所以 `7 == 7` 的结果是 `True`。
4.  **C. `3 < 3`**
    *   解析：`8 > 6` 是 `True`；`1 == 1` 是 `True`；`100 == 100` 是 `True`。只有 `3 < 3` 是 `False`（3不小于3）。
5.  **B. `=`**
    *   解析：一个等号 `=` 是赋值运算符，把右边的值赋给左边的变量。`==` 是比较是否相等的运算符。

**二、编程挑战参考答案**

1.  **比较身高**
    ```python
    h1 = int(input("请输入第一个同学的身高（厘米）: "))
    h2 = int(input("请输入第二个同学的身高（厘米）: "))

    result = h1 > h2
    print(f"第一个同学比第二个同学高吗？ {result}")
    ```

2.  **判断及格**
    ```python
    score = int(input("请输入学生的考试分数: "))

    # 方法一：使用 >= 运算符 (推荐)
    # is_pass = score >= 60
    # print(f"这个学生及格了吗？ {is_pass}")

    # 方法二：使用 > 和 == 运算符组合 (如果只想用今天学过的)
    is_pass = (score > 60) or (score == 60) # or 表示“或者”，只要满足一个条件就是True
    print(f"这个学生及格了吗？ {is_pass}")
    ```
