## 知识点讲解

### 1. 认识海龟画图 (Turtle Graphics)

海龟画图是 Python 语言中一个非常有趣的绘图模块。想象一下，你有一只小海龟，它背着一支画笔。你给它指令，比如“向前走100步”，它就会向前爬行100步，并在身后留下一条100步长的线。你再给它指令“向右转90度”，它就会原地转个圈，然后面向新的方向。

### 2. 导入海龟模块

在使用海龟画图之前，我们首先要告诉 Python，我们要用它的海龟功能。这就像你玩积木前，要先把积木盒打开一样。我们用 `import` 语句来完成这个任务：

```python
import turtle
```
这行代码的意思是“导入海龟模块”。通常，我们会给海龟起个小名，比如 `t`，这样每次命令它的时候就不用写 `turtle` 这么长了。我们可以这样写：
```python
import turtle
t = turtle.Turtle()  # 创建一只海龟，并给它起名叫 t
```
以后，我们就可以用 `t` 来代表这只海龟了。

### 3. 让海龟前进：`forward()`

海龟最基本的动作就是前进。它会沿着当前的方向向前移动一段距离，并在移动过程中画出一条线。

**命令格式：**
`t.forward(距离)`

这里的“距离”是一个数字，代表海龟要前进的像素点数。像素点是屏幕上最小的显示单位，数字越大，海龟走得越远，线就越长。

**举个例子：**
让海龟前进100个像素：
```python
t.forward(100)
```

### 4. 让海龟转向：`right()`

海龟不仅会前进，还会转弯！`right()` 命令可以控制海龟顺时针转动指定的角度。

**命令格式：**
`t.right(角度)`

这里的“角度”也是一个数字，代表海龟顺时针转动的度数。我们知道一个圆是360度。
- 如果是 `t.right(90)`，海龟会向右转90度，也就是一个直角。
- 如果是 `t.right(180)`，海龟会向右转180度，也就是转过身去，面朝相反方向。

**举个例子：**
让海龟向右转90度：
```python
t.right(90)
```

### 5. 保持画图窗口：`done()` 或 `exitonclick()`

当你画完图后，如果不告诉 Python 保持窗口，它可能会一闪而过，你就来不及看清了。
- `turtle.done()`：会保持画图窗口，直到你手动关闭它。
- `turtle.exitonclick()`：会保持画图窗口，直到你点击窗口中的任意位置，它才会关闭。

我们通常会选择其中一个放在代码的最后。

```python
turtle.done()  # 或者 turtle.exitonclick()
```

现在，我们把这些命令组合起来，看看能画出什么吧！

## 代码示例

### 示例 1：画一条简单的直线

```python
import turtle

# 创建一只海龟，它的名字是 t
t = turtle.Turtle()

# 让海龟前进 150 像素
t.forward(150)

# 保持画图窗口，直到手动关闭
turtle.done()
```
**运行效果：** 屏幕上会出现一条向右的直线。

### 示例 2：画一个直角（“L”形）

```python
import turtle

t = turtle.Turtle()

# 让海龟前进 100 像素
t.forward(100)

# 让海龟向右转 90 度
t.right(90)

# 再次让海龟前进 100 像素
t.forward(100)

# 保持画图窗口
turtle.done()
```
**运行效果：** 屏幕上会出现一个 L 形的图案。

### 示例 3：尝试画一个正方形

画正方形需要四条等长的边和四个直角。我们可以重复前进和转向的动作！

```python
import turtle

t = turtle.Turtle()

# 第一条边和第一次转弯
t.forward(100)
t.right(90)

# 第二条边和第二次转弯
t.forward(100)
t.right(90)

# 第三条边和第三次转弯
t.forward(100)
t.right(90)

# 第四条边和第四次转弯
t.forward(100)
t.right(90)

# 保持画图窗口
turtle.done()
```
**运行效果：** 屏幕上会出现一个完美的正方形！

### 示例 4：画一个等边三角形的一部分（思考角度）

一个等边三角形的内角是60度。但是海龟是转动“外角”才能形成图形的。一个外角是 `180 - 内角`。所以，如果是等边三角形，外角是 `180 - 60 = 120` 度。

```python
import turtle

t = turtle.Turtle()

# 画第一条边
t.forward(100)

# 顺时针转 120 度
t.right(120)

# 画第二条边
t.forward(100)

# 顺时针转 120 度
t.right(120)

# 画第三条边
t.forward(100)

# 保持画图窗口
turtle.done()
```
**运行效果：** 屏幕上会出现一个等边三角形。

## 练习题目

**请选择正确答案或填写正确内容：**

1.  在 Python 中使用海龟画图功能时，第一步通常是导入海龟模块。以下哪行代码是正确的导入方式？
    A. `import turtle_graphics`
    B. `include turtle`
    C. `import turtle`
    D. `using turtle`

2.  要让海龟向前移动 80 像素，应该使用哪个命令？
    A. `t.move(80)`
    B. `t.go(80)`
    C. `t.forward(80)`
    D. `t.step(80)`

3.  如果海龟当前方向是向右，执行 `t.right(90)` 命令后，海龟将面向哪个方向？
    A. 向上
    B. 向下
    C. 向左
    D. 仍然向右

4.  下面的 Python 代码会画出什么图形？
    ```python
    import turtle
    s = turtle.Turtle()
    s.forward(60)
    s.right(90)
    s.forward(60)
    turtle.done()
    ```
    A. 一条直线
    B. 一个正方形
    C. 一个直角（L形）
    D. 一个三角形

5.  在海龟画图程序中，为了防止画图窗口一闪而过，通常会在代码的最后加上 `turtle._______()`。请填空。

---

**答案与解析：**

1.  **C. `import turtle`**
    *   **解析：** 这是 Python 中导入海龟模块的标准语法。其他选项均不符合 Python 导入模块的规则。

2.  **C. `t.forward(80)`**
    *   **解析：** `forward()` 是海龟模块中用于控制海龟向前移动的命令，参数是移动的距离。

3.  **B. 向下**
    *   **解析：** 初始海龟默认面向右方（0度）。`right(90)` 表示顺时针转90度。从右转90度，就变成了向下。

4.  **C. 一个直角（L形）**
    *   **解析：** 代码让海龟前进60，然后右转90度，再前进60。这只完成了正方形的两条边和一个角，形成了一个直角形状。

5.  **`done` 或 `exitonclick`**
    *   **解析：** `turtle.done()` 和 `turtle.exitonclick()` 都可以用来保持画图窗口不关闭，直到用户手动操作。
