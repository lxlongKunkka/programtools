## 知识点讲解

### 1. 认识海龟画图 (Turtle Graphics)

海龟画图是Python语言中一个非常有趣和容易上手的图形库，特别适合初学者学习编程。你可以想象它是一只背着画笔的小乌龟，在屏幕上爬行，它爬过的路径就会留下痕迹，形成各种图形。

### 2. 请出小乌龟：导入模块

我们首先要告诉Python，我们要使用海龟画图功能。就像我们要玩玩具，得先把玩具拿出来一样。
```python
import turtle # 导入海龟模块
```
这行代码的意思是“请把海龟画图的工具箱给我拿过来！”

### 3. 给小乌龟取个名字：创建海龟对象

为了方便指挥，我们通常会给这只小乌龟取个名字，比如叫 `t`。
```python
t = turtle.Turtle() # 创建一只小海龟，并给它取名t
```
现在，我们就可以通过 `t` 来指挥小海龟了！

### 4. 小乌龟的基本行动指令

*   **向前走**：`t.forward(距离)`
    *   让小乌龟向前爬指定的像素距离。
    *   例如：`t.forward(100)` 让小乌龟向前走100像素。
*   **向后退**：`t.backward(距离)`
    *   让小乌龟向后退指定的像素距离。
    *   例如：`t.backward(50)` 让小乌龟向后退50像素。
*   **向左转**：`t.left(角度)`
    *   让小乌龟向左旋转指定的角度（以度为单位）。
    *   例如：`t.left(90)` 让小乌龟向左转90度。
*   **向右转**：`t.right(角度)`
    *   让小乌龟向右旋转指定的角度。
    *   例如：`t.right(60)` 让小乌龟向右转60度。

### 5. 小乌龟的画笔设置

*   **抬起画笔**：`t.penup()`
    *   小乌龟抬起画笔，移动时不再留下痕迹。
*   **放下画笔**：`t.pendown()`
    *   小乌龟放下画笔，移动时开始留下痕迹（这是默认状态）。
*   **设置画笔粗细**：`t.pensize(粗细值)`
    *   设置画笔的粗细，参数是一个数字，值越大越粗。
    *   例如：`t.pensize(3)` 将画笔粗细设置为3像素。
*   **设置画笔颜色**：`t.color("颜色名")`
    *   设置画笔的颜色。你可以使用常见的英文颜色名，如 "red", "blue", "green", "yellow", "purple", "black" 等。
    *   例如：`t.color("red")` 将画笔颜色设置为红色。
*   **设置画图速度**：`t.speed(速度值)`
    *   设置小乌龟画图的速度，参数是一个数字。
    *   `0`：最快 (瞬间完成)。
    *   `1`：最慢。
    *   `1-10`：数字越大速度越快。
    *   例如：`t.speed(5)` 设置中等速度。

### 6. 让画图窗口保持显示

当小乌龟画完图后，程序会很快结束，窗口可能会一闪而过。为了能好好欣赏我们的作品，我们需要告诉Python：“等等，别关窗口！”
```python
turtle.done() # 保持画图窗口显示，直到手动关闭
```
这行代码通常放在你所有画图指令的最后面。

## 代码示例

让我们来尝试用这些指令画出一些简单的几何图形吧！

### 示例 1：画一条线

```python
import turtle # 导入海龟模块

t = turtle.Turtle() # 创建一只小海龟，并给它取名t
t.speed(3) # 设置画图速度为中等

t.forward(150) # 小乌龟向前走150像素

turtle.done() # 保持画图窗口显示
```

### 示例 2：画一个正方形

一个正方形有四条边，每条边一样长，每个角都是90度。小乌龟每走一条边，就需要转90度。

```python
import turtle

t = turtle.Turtle()
t.speed(5) # 设置画图速度
t.pensize(3) # 设置画笔粗细
t.color("blue") # 设置画笔颜色

s = 100 # 正方形的边长

# 画第一条边
t.forward(s)
t.right(90) # 向右转90度

# 画第二条边
t.forward(s)
t.right(90)

# 画第三条边
t.forward(s)
t.right(90)

# 画第四条边
t.forward(s)
t.right(90)

turtle.done()
```
**思考：** 有没有更简单的方法来画重复的动作呢？我们可以用循环！

```python
import turtle

t = turtle.Turtle()
t.speed(5)
t.pensize(3)
t.color("blue")

s = 100 # 正方形的边长

for i in range(4): # 循环4次，因为正方形有4条边
    t.forward(s)
    t.right(90)

turtle.done()
```

### 示例 3：画一个等边三角形

一个等边三角形有三条边，每条边一样长，每个角都是60度。但是，小乌龟需要转的是外角，所以是 `180 - 60 = 120` 度。

```python
import turtle

t = turtle.Turtle()
t.speed(5)
t.pensize(2)
t.color("green")

s = 120 # 三角形的边长

for i in range(3): # 循环3次，因为三角形有3条边
    t.forward(s)
    t.left(120) # 向左转120度 (外角)

turtle.done()
```

### 示例 4：画一个长方形

长方形有两组对边相等，两个内角都是90度。

```python
import turtle

t = turtle.Turtle()
t.speed(5)
t.pensize(4)
t.color("purple")

s1 = 150 # 长边
s2 = 80  # 短边

# 画长边
t.forward(s1)
t.right(90)

# 画短边
t.forward(s2)
t.right(90)

# 画另一条长边
t.forward(s1)
t.right(90)

# 画另一条短边
t.forward(s2)
t.right(90)

turtle.done()
```
**思考：** 长方形也可以用循环来画吗？当然可以！

```python
import turtle

t = turtle.Turtle()
t.speed(5)
t.pensize(4)
t.color("purple")

s1 = 150 # 长边
s2 = 80  # 短边

for i in range(2): # 循环2次，每次画一条长边和一条短边
    t.forward(s1)
    t.right(90)
    t.forward(s2)
    t.right(90)

turtle.done()
```

## 练习题目

### 选择题

1.  要使用海龟画图功能，程序开头必须导入哪个模块？
    A. `import math`
    B. `import random`
    C. `import turtle`
    D. `import time`

2.  以下哪条指令是让小乌龟向前移动100像素？
    A. `t.move(100)`
    B. `t.go(100)`
    C. `t.forward(100)`
    D. `t.step(100)`

3.  如果小乌龟当前朝向右边，执行 `t.left(90)` 后，它的朝向会变成哪个方向？
    A. 上方
    B. 下方
    C. 左方
    D. 仍然是右方

4.  要让小乌龟画出的线条更粗，应该使用哪条指令？
    A. `t.color()`
    B. `t.speed()`
    C. `t.pensize()`
    D. `t.penup()`

5.  画一个正五边形，每走完一条边后，小乌龟需要向左或向右转多少度？
    A. 60度
    B. 72度
    C. 90度
    D. 108度

### 填空题

1.  要让小乌龟在移动时不留下痕迹，应该先执行 `t.___________()` 指令。
2.  画一个正方形，如果边长是 `s`，那么循环体中应该包含 `t.forward(s)` 和 `t.___________(90)`。
3.  `t.speed(0)` 表示小乌龟以___________的速度画图。
4.  为了让画图窗口在程序结束后不立即关闭，我们通常会在代码的最后一行加上 `turtle.___________()`。

---

### 练习题目答案与解析

**选择题答案：**

1.  **C. `import turtle`**
    *   解析：`import turtle` 是导入海龟画图模块的正确语法。

2.  **C. `t.forward(100)`**
    *   解析：`forward()` 是海龟画图中小乌龟向前移动的指令。

3.  **A. 上方**
    *   解析：小乌龟默认朝向右方（东方）。向左转90度会使其朝向上方（北方）。

4.  **C. `t.pensize()`**
    *   解析：`pensize()` 用于设置画笔的粗细。`color()` 设置颜色，`speed()` 设置速度，`penup()` 抬起画笔。

5.  **B. 72度**
    *   解析：正多边形的外角和是360度。正五边形有5条边，所以每个外角是 `360 / 5 = 72` 度。

**填空题答案：**

1.  **`penup`**
    *   解析：`t.penup()` 抬起画笔，移动不留痕迹。

2.  **`right`** 或 **`left`** (两者均可，取决于你选择哪个方向转)
    *   解析：画正方形需要转90度，可以是向右转 `t.right(90)`，也可以是向左转 `t.left(90)`。

3.  **最快**
    *   解析：`t.speed(0)` 是最快的速度，几乎瞬间完成。

4.  **`done`**
    *   解析：`turtle.done()` 用于保持画图窗口显示，直到用户手动关闭。
