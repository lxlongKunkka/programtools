## 知识点讲解

### 1. 让电脑“说话”——输出 (print)

我们想让电脑在屏幕上显示一些信息，比如“Hello, world!”，或者一个计算结果，这时候就需要用到 `print()` 函数。`print` 在英文里是“打印”的意思，在这里就是让信息显示在屏幕上。

#### 1.1 输出文字信息 (字符串)
如果你想让电脑说一句话，就把这句话用引号（单引号 `'` 或双引号 `"`）包起来，放到 `print()` 的括号里。

```python
print("你好，Python世界！")
print('我爱编程！')
```

#### 1.2 输出数字信息
如果你想让电脑显示一个数字，可以直接把数字放到 `print()` 的括号里。

```python
print(100)
print(3.14)
```

#### 1.3 输出变量的值
我们之前学过变量，变量就像一个盒子，可以存储数据。如果我们想看看盒子里装了什么，也可以用 `print()`。

```python
s = "小明"
n = 12
print(s)
print(n)
```

#### 1.4 输出多个信息
如果想一次性输出多个信息，可以用逗号 `,` 把它们隔开。`print()` 会自动在这些信息之间加上一个空格。

```python
s = "小红"
n = 8
print("我的名字是", s, "我今年", n, "岁。")
```

### 2. 让电脑“听话”——输入 (input)

光让电脑说话还不够，我们还想告诉电脑一些信息，比如我们的名字、我们想输入的数字等等。这时候，我们就需要用到 `input()` 函数。`input` 在英文里是“输入”的意思。

#### 2.1 获取文字输入 (字符串)
当程序执行到 `input()` 函数时，它会暂停，等待我们从键盘输入一些内容，然后按回车键。我们输入的内容会被 `input()` 函数接收，并作为一个结果返回。通常我们会用一个变量来接收这个结果。

```python
s = input()  # 程序会暂停，等待你输入，然后把输入的内容存到变量 s 里
print("你输入了：", s)
```

#### 2.2 带提示信息的输入
直接让程序等待输入有点“傻”，我们不知道要输入什么。`input()` 函数可以接受一个字符串作为参数，这个字符串会作为提示信息显示在屏幕上，告诉用户应该输入什么。

```python
s = input("请输入你的名字：")  # 会先显示“请输入你的名字：”，再等待输入
print("你好，", s, "！很高兴认识你！")
```

#### 2.3 `input()` 接收到的都是字符串！
这是非常重要的一点！无论你输入的是文字还是数字，`input()` 函数都会把它们当作“字符串”来处理。字符串就是一串文字。

例如，如果你输入了数字 `10`，`input()` 接收到的其实是字符串 `"10"`，而不是数字 `10`。

```python
s = input("请输入一个数字：") # 假设你输入了 10
print("你输入的是：", s)       # 输出：你输入的是： 10
print(type(s))                 # 输出：<class 'str'> (str 就是字符串的意思)
```

#### 2.4 将字符串转换为数字 (类型转换)
如果我们要对用户输入的数字进行数学计算（比如加减乘除），就必须把 `input()` 接收到的字符串数字转换成真正的数字类型。我们可以使用 `int()` 函数将其转换为整数，或者 `float()` 函数将其转换为小数。

```python
s = input("请输入你的年龄：") # 假设你输入了 10
n = int(s)                   # 将字符串 "10" 转换为整数 10
print("两年后你将是", n + 2, "岁。") # 这样就可以进行加法运算了
```

更常见的写法是把 `input()` 和 `int()` 结合起来：

```python
n = int(input("请输入一个整数：")) # 直接获取整数输入
m = int(input("请再输入一个整数："))
print("这两个数的和是：", n + m)
```

记住，`int()` 只能把看起来像整数的字符串（如 `"123"`）转换成整数。如果你输入了文字（如 `"你好"`）或者小数（如 `"3.14"`），`int()` 就会报错哦！

## 代码示例

让我们一起来看看这些知识点在实际代码中是怎么使用的吧！

### 示例1：使用 `print()` 输出各种信息
```python
# 输出一句问候语
print("欢迎来到Python编程世界！")

# 输出一个数字
print(2023)

# 输出一个小数
print(3.14159)

# 定义变量并输出它们的值
s = "小天才"
n = 7
print("我的名字是", s)
print("我今年", n, "岁了。")

# 输出多个信息，中间用逗号隔开
print("今天天气真好，我们一起学习Python吧！")
```

### 示例2：使用 `input()` 获取用户输入并打印
```python
# 获取用户的名字
s = input("请问你叫什么名字？")
print("你好，", s, "！")

# 获取用户最喜欢的颜色
c = input("你最喜欢的颜色是什么？")
print(s, "最喜欢的颜色是", c, "。")
```

### 示例3：获取数字输入并进行计算
```python
# 获取两个数字，并计算它们的和
print("这是一个简单的加法计算器。")

# 获取第一个数字
a = int(input("请输入第一个整数："))

# 获取第二个数字
b = int(input("请输入第二个整数："))

# 计算和并输出
sum_ab = a + b # 注意：变量名这里用了 sum_ab，是为了让大家更容易理解，信奥中通常会用 s 或者 c 等单字母
print("你输入的两个数的和是：", sum_ab)
```

### 示例4：综合应用——一个简单的自我介绍程序
```python
print("欢迎使用自我介绍生成器！")

# 获取姓名
s_name = input("请输入你的姓名：")

# 获取年龄
n_age = int(input("请输入你的年龄："))

# 获取爱好
s_hobby = input("请输入你的爱好：")

# 打印完整的自我介绍
print("--------------------")
print("大家好！我叫", s_name, "。")
print("我今年", n_age, "岁，我喜欢", s_hobby, "。")
print("很高兴认识大家！")
print("--------------------")
```

### 练习题目答案与解析

**选择题答案：**

1.  **C. `print()`**
    *   解析：`print()` 函数就是用来在控制台（屏幕）上输出信息的。

2.  **D. `input()`**
    *   解析：`input()` 函数用于接收用户从键盘输入的数据。

3.  **C. 字符串 (str)**
    *   解析：`input()` 函数获取的所有用户输入，无论看起来像数字还是文字，默认都是字符串类型。

4.  **B. `n = int(input("请输入一个数字："))`**
    *   解析：`input()` 获取的是字符串，如果需要进行数学计算，必须使用 `int()` 或 `float()` 将其转换为数字类型。选项 B 正确地使用了 `int()` 进行转换。

**填空题答案：**

1.  `我喜欢 Python 编程`
    *   解析：`print()` 函数用逗号分隔多个参数时，默认会在它们之间添加一个空格。

2.  `'小明'` (或 `"小明"`)
    *   解析：`input()` 函数将用户输入的内容以字符串的形式赋值给变量 `s`。

3.  `"Alice!"` (或 `'Alice!'` 或直接 `Alice` 但这通常意味着 Alice 是一个变量)
    *   解析：为了输出 `Hello, Alice!`，我们需要在逗号后面提供字符串 `"Alice!"`。

4.  `a + b`
    *   解析：`a` 和 `b` 已经被 `int()` 转换成了整数，所以可以直接进行加法运算 `a + b`。
