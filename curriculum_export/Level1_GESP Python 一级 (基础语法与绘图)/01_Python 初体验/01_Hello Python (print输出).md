## 知识点讲解

### 1. 什么是Python？

Python就像一种“国际通用语言”，你用它写下的指令，电脑都能听懂。它有很多强大的功能，可以用来做游戏、网站、人工智能等等。我们今天就从最简单的开始！

### 2. 我们的第一个“魔法指令”：`print()`

在Python里，我们想让电脑在屏幕上显示一些内容，就要用到一个叫做 `print()` 的“魔法指令”。
`print` 在英文里是“打印”或“输出”的意思。
`()` 就像一个说话的“嘴巴”，我们想让电脑说的话，就放在这个“嘴巴”里面。

**语法格式：**
`print(你想让电脑说的话)`

### 3. 让电脑说出文字（字符串）

当我们想让电脑说出一段文字，比如“Hello World!”、“我爱Python”的时候，这些文字需要用一对 **引号** 包裹起来。引号可以是单引号 `'`，也可以是双引号 `"`。它们的作用是告诉电脑：“嘿，这是一个文字，不是指令哦！”

**示例：**
*   `print("Hello Python!")`
*   `print('你好，世界！')`

### 4. 让电脑说出数字

如果我们要让电脑显示一个数字，比如123、3.14，这些数字就 **不需要** 用引号包裹。因为电脑知道123就是一个数字，它不会误以为是指令。

**示例：**
*   `print(123)`
*   `print(3.14159)`

### 5. 同时输出多个内容

如果我们想让电脑一次性说出好几段内容，中间可以用逗号 `,` 分隔。Python会很聪明地在它们之间自动加上一个空格。

**示例：**
*   `print("我的名字是小明", "我今年10岁")`
*   `print("2 + 3 =", 5)`

### 6. 常见的“小错误”（别担心，很容易改！）

*   **忘记加括号 `()`**：`print "Hello"` (这是错误的！)
*   **字符串忘记加引号 `""` 或 `''`**：`print(Hello)` (电脑会以为 `Hello` 是一个指令，然后报错！)
*   **引号不匹配**：`print("你好')` (一半双引号，一半单引号，电脑会很困惑！)

记住，当我们犯错时，电脑会给我们一些提示（报错信息），我们仔细看看就能找到问题所在！

## 练习题目

**一、选择题**

1.  以下哪一个 `print()` 语句是正确的，能让计算机输出“Python真棒！”？
    A. `print Python真棒！`
    B. `print("Python真棒！")`
    C. `print(Python真棒！)`
    D. `Print("Python真棒！")`

2.  如果想让计算机输出数字 `520`，以下哪种写法是正确的？
    A. `print("520")`
    B. `print(520)`
    C. `print('520')`
    D. 以上都正确

3.  观察以下代码：
    ```python
    print("我", "爱", "编程")
    ```
    它的输出结果是什么？
    A. 我爱编程
    B. 我 爱 编程
    C. "我", "爱", "编程"
    D. 报错

4.  以下哪种情况，字符串需要用引号包裹？
    A. 当字符串是数字时
    B. 当字符串是英文时
    C. 任何时候，只要是文本内容，都需要用引号包裹
    D. 只有在 `print()` 函数里才需要

**二、填空题**

1.  在Python中，我们使用 `______` 函数来在屏幕上显示信息。
2.  如果我们要输出文字“Hello”，需要用 `______` 或 `______` 将其包裹起来。
3.  `print(100)` 的输出结果是 `______`。
4.  `print("我的身高是", 145, "厘米。")` 这行代码会输出 `______`。

---

**答案与解析**

**一、选择题答案**

1.  **B**
    *   A选项缺少括号和引号。
    *   B选项语法正确，用双引号包裹字符串。
    *   C选项缺少引号。
    *   D选项 `Print` 的P是大写，Python对大小写敏感，`print` 是小写。

2.  **D**
    *   `print(520)` 直接输出数字，正确。
    *   `print("520")` 和 `print('520')` 输出的是字符串“520”，在屏幕上看起来和数字520一样，但在计算机内部，一个是数字类型，一个是字符串类型。对于初学者来说，这三种写法都能达到“在屏幕上显示520”的目的，因此都算正确。

3.  **B**
    *   `print()` 函数使用逗号分隔多个内容时，默认会在内容之间添加一个空格。

4.  **C**
    *   无论是中文、英文、符号，只要是文本内容（字符串），都需要用引号包裹。

**二、填空题答案**

1.  `print()`
2.  `单引号` (或 `'`)， `双引号` (或 `"`)
3.  `100`
4.  `我的身高是 145 厘米。` (注意逗号分隔后有空格)
