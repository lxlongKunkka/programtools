## 知识点讲解

### 1. 什么是“累加求和”？

“累加求和”就是把一串数字一个接一个地加起来，得到它们的总和。
例如：计算 1 到 5 的和，就是 `1 + 2 + 3 + 4 + 5 = 15`。

在编程中，我们通常会用一个特殊的变量来保存这个不断增加的总和，我们叫它**累加器**。

### 2. 累加器 (Accumulator) 变量

想象一下你的存钱罐，刚开始里面是空的（0元）。每当你得到一笔钱，就把它放进去。存钱罐里的钱就越来越多。
在编程中，这个“存钱罐”就是一个变量，我们通常给它起个名字，比如 `s` (sum 的缩写)。

-   **初始化**：刚开始，`s` 里面什么都没有，所以我们要让 `s = 0`。
-   **累加**：每当我们有一个新的数字 `i` 要加进来时，我们就让 `s = s + i`。这句话的意思是：把当前的 `s` 和新的数字 `i` 加起来，然后把结果再存回到 `s` 里。

### 3. `for` 循环：重复的力量

还记得我们前面学过的 `for` 循环吗？它就像一个勤劳的小工人，可以帮我们重复做很多事情。
当我们要累加一系列数字时，`for` 循环就可以帮我们一个一个地取出数字，并把它们加到累加器 `s` 中。

**`range()` 函数回顾：**
`range(a, b)` 会生成从 `a` 开始，到 `b-1` 结束的整数序列。
例如：
-   `range(5)` 会生成 `0, 1, 2, 3, 4`。
-   `range(1, 6)` 会生成 `1, 2, 3, 4, 5`。
-   `range(1, 11)` 会生成 `1, 2, 3, ..., 10`。

### 4. 结合：在循环中累加

现在，我们把累加器和 `for` 循环结合起来：

1.  **准备一个“存钱罐”**：`s = 0`
2.  **让小工人开始工作**：`for i in range(...)`，这里的 `i` 就是每次要加进来的数字。
3.  **每次把钱放进存钱罐**：`s = s + i` （或者简写为 `s += i`）
4.  **工作结束，看看总共有多少钱**：循环结束后，打印 `s`。

**重要提醒：** `s = s + i` 这句话必须在循环里面，这样它才能重复执行。而 `s = 0` 必须在循环外面，因为我们只需要把存钱罐清空一次。打印结果也应该在循环外面，我们只需要看最终的总和。

## 代码示例

让我们用 Python 来实现这些想法吧！

### 示例 1：计算 1 到 5 的和

```python
# 1. 准备一个“存钱罐”，初始化为 0
s = 0

# 2. 让小工人从 1 到 5 开始工作
# range(1, 6) 会生成 1, 2, 3, 4, 5
for i in range(1, 6):
    # 3. 每次把当前的数字 i 加到 s 中
    s = s + i  # 也可以写成 s += i
    print(f"当前数字是 {i}，累加后 s 变成了 {s}") # 观察s的变化

# 4. 小工人工作结束，看看总共有多少
print(f"从 1 到 5 的和是：{s}")
```

运行结果：
```
当前数字是 1，累加后 s 变成了 1
当前数字是 2，累加后 s 变成了 3
当前数字是 3，累加后 s 变成了 6
当前数字是 4，累加后 s 变成了 10
当前数字是 5，累加后 s 变成了 15
从 1 到 5 的和是：15
```

### 示例 2：计算 1 到用户输入 `n` 的和

```python
# 1. 准备一个“存钱罐”，初始化为 0
s = 0

# 2. 请用户输入一个数字 n
n = int(input("请输入一个正整数 n："))

# 3. 让小工人从 1 到 n 开始工作
# range(1, n + 1) 会生成 1, 2, ..., n
for i in range(1, n + 1):
    s += i  # 简写形式，和 s = s + i 是一样的

# 4. 小工人工作结束，看看总共有多少
print(f"从 1 到 {n} 的和是：{s}")
```

### 示例 3：计算用户输入的 `k` 个数字的总和

```python
# 1. 准备一个“存钱罐”，初始化为 0
s = 0

# 2. 请用户输入要加多少个数字
k = int(input("你想加多少个数字？请输入数量 k："))

# 3. 让小工人重复 k 次工作
for i in range(k): # range(k) 会生成 0, 1, ..., k-1，正好重复 k 次
    # 每次循环都请用户输入一个数字
    x = int(input(f"请输入第 {i + 1} 个数字："))
    s += x # 把用户输入的数字加到 s 中

# 4. 小工人工作结束，看看总共有多少
print(f"你输入的 {k} 个数字的总和是：{s}")
```

是不是很简单呢？有了循环，再多的数字我们也不怕啦！

## 练习题目

### 选择题或填空题

1.  以下 Python 代码段的运行结果是？
    ```python
    s = 0
    for i in range(1, 4):
        s += i
    print(s)
    ```
    A. 0
    B. 3
    C. 6
    D. 10

2.  如果想计算从 5 到 10 的所有整数的和，`for` 循环的 `range()` 函数应该怎么写？
    `for i in range(_____, _____)`：
    A. `range(5, 10)`
    B. `range(5, 11)`
    C. `range(6, 11)`
    D. `range(1, 11)`

3.  在累加求和的程序中，累加器变量 `s` 通常在循环开始前被初始化为 `____`。

4.  以下代码的运行结果是？
    ```python
    s = 1
    for i in range(2, 5):
        s = s * i
    print(s)
    ```
    A. 1
    B. 24
    C. 120
    D. 4

5.  如果我们要计算 5 个数字的总和，并且这 5 个数字由用户逐个输入，那么 `for` 循环的 `range()` 函数应该怎么写？
    `for i in range(_____)`：
    A. `range(5)`
    B. `range(1, 5)`
    C. `range(1, 6)`
    D. `range(0, 6)`

### 答案与解析

1.  **答案：C**
    **解析**：
    -   `s = 0`
    -   `i = 1` 时，`s = 0 + 1 = 1`
    -   `i = 2` 时，`s = 1 + 2 = 3`
    -   `i = 3` 时，`s = 3 + 3 = 6`
    -   循环结束，打印 `s`，结果为 6。

2.  **答案：B**
    **解析**：`range(a, b)` 会生成从 `a` 到 `b-1` 的序列。要包含 10，所以 `b` 应该是 11。

3.  **答案：0**
    **解析**：累加求和时，累加器最初应该没有值，即为 0，这样才能正确地从 0 开始加上后续的数字。

4.  **答案：B**
    **解析**：
    -   `s = 1` (注意这里是 1，不是 0)
    -   `range(2, 5)` 生成 2, 3, 4
    -   `i = 2` 时，`s = 1 * 2 = 2`
    -   `i = 3` 时，`s = 2 * 3 = 6`
    -   `i = 4` 时，`s = 6 * 4 = 24`
    -   循环结束，打印 `s`，结果为 24。这是一个连乘（阶乘）的例子。

5.  **答案：A**
    **解析**：`range(5)` 会生成 0, 1, 2, 3, 4，正好是 5 个数字，循环会执行 5 次，每次用户输入一个数字。选项 C `range(1, 6)` 也可以，但 `range(5)` 更简洁。
