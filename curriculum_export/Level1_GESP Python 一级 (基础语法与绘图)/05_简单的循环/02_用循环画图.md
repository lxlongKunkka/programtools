## 知识点讲解

### 1. `print()` 函数的小秘密：不换行输出

我们之前学过 `print()` 函数可以输出文字。默认情况下，每次 `print()` 完，光标都会自动跳到下一行，就像这样：

```python
print("你好")
print("世界")
```

输出：
```
你好
世界
```

但如果我们想让它们在同一行显示，比如画一排星星，怎么办呢？这时，我们需要告诉 `print()` 函数，输出完后不要换行！秘诀就是使用 `end=""` 参数。

```python
print("*", end="") # 输出一个星号后不换行
print("*", end="") # 继续输出一个星号后不换行
print("*")         # 输出一个星号后换行
```

输出：
```
***
```

看到了吗？`end=""` 就像告诉 `print()`：“等一下，我后面还有东西要打印，你先别急着换行！”

### 2. 单层 `for` 循环：画一条线

现在我们知道了怎么让星星排成一队。那如果我们要画10个星星呢？总不能写10次 `print("*", end="")` 吧？这时候，“循环”就派上用场了！

`for` 循环可以让我们指定一段代码重复执行多少次。它的基本结构是这样的：

```python
for i in range(n):
    # 这里是要重复执行的代码
```

其中：
*   `for i in range(n):` 表示让变量 `i` 从 0 开始，一直到 `n-1` 结束，循环 `n` 次。
*   `n` 就是你希望循环的次数。
*   冒号 `:` 不能少哦！
*   循环体（要重复执行的代码）需要缩进，通常是4个空格。

**例子：画一条由5个星星组成的直线。**

```python
n = 5
for i in range(n):
    print("*", end="")
print() # 循环结束后，额外打印一个空行，让光标跳到下一行，为接下来的输出做准备
```

这段代码会重复执行 `print("*", end="")` 5次，最后打印一个空行。

### 3. 嵌套 `for` 循环：画一个实心矩形

现在我们能画一条直线了。那如果我想画一个矩形呢？矩形是由很多行直线组成的！这意味着，我们需要：
1.  重复画很多条直线（外层循环控制行数）。
2.  每条直线里，又重复画很多个星星（内层循环控制列数）。

这就是“嵌套循环”！一个 `for` 循环里面再套一个 `for` 循环。

```python
for i in range(行数):     # 外层循环：控制画多少行
    for j in range(列数): # 内层循环：控制每行画多少个字符
        # 打印字符
    print()             # 内层循环画完一行后，记得换行！
```

**例子：画一个3行5列的实心矩形。**

```python
n = 3 # 行数
m = 5 # 列数

for i in range(n): # 外层循环，控制行
    for j in range(m): # 内层循环，控制列
        print("*", end="") # 在同一行打印星号
    print() # 每画完一行，就换行
```

### 4. 进阶：画一个实心直角三角形

掌握了矩形，我们再来试试画一个直角三角形。观察一下直角三角形的特点：
第一行：1个星星
第二行：2个星星
第三行：3个星星
...
第 `i` 行：`i` 个星星

这说明，内层循环打印星星的次数，是随着外层循环的行数变化的！

**例子：画一个5行的实心直角三角形。**

```python
n = 5 # 三角形的行数

for i in range(n): # 外层循环，控制行数，i 会是 0, 1, 2, 3, 4
    # 注意：如果 i 从 0 开始，那么 i+1 就代表第 1, 2, 3, 4, 5 行的星星数量
    for j in range(i + 1): # 内层循环，控制每行打印的星星数量
        print("*", end="")
    print() # 每画完一行，就换行
```

是不是很有趣？通过控制循环的次数和 `print()` 的 `end` 参数，我们就能画出各种各样的图案！

## 代码示例

### 示例1：画一条长为10的横线

```python
# 设定横线的长度
n = 10

# 使用for循环重复打印星号
for i in range(n):
    print("*", end="")

# 打印完所有星号后，换行
print()
```

### 示例2：画一个4行6列的实心矩形

```python
# 设定矩形的行数和列数
n = 4 # 行数
m = 6 # 列数

# 外层循环控制行
for i in range(n):
    # 内层循环控制列，打印每行的字符
    for j in range(m):
        print("#", end="") # 使用井号作为填充字符
    # 每画完一行，记得换行
    print()
```

### 示例3：画一个5行的实心直角三角形

```python
# 设定三角形的行数
n = 5

# 外层循环控制行数
for i in range(n): # i 的值依次为 0, 1, 2, 3, 4
    # 内层循环控制每行打印的字符数量
    # 第 i 行（从0开始计数），打印 i+1 个字符
    for j in range(i + 1):
        print("@", end="") # 使用@符号作为填充字符
    # 每画完一行，记得换行
    print()
```

### 练习题目答案与解析

1.  **答案：A**
    *   **解析：** `for i in range(3)` 会循环3次。
        *   第一次循环：`print("A", end="*")` 输出 `A*`，不换行。
        *   第二次循环：`print("A", end="*")` 输出 `A*`，不换行。
        *   第三次循环：`print("A", end="*")` 输出 `A*`，不换行。
        *   循环结束后：`print("B")` 输出 `B`，并换行。
        *   所以最终输出是 `A*A*A*B`。

2.  **答案：A**
    *   **解析：** 外层循环控制行数，有5行，所以 `range(5)`。内层循环控制列数，每行8个，所以 `range(8)`。

3.  **答案：C**
    *   **解析：**
        *   A. `print("***")` 会输出 `***` 然后换行。
        *   B. `for i in range(3): print("*")` 会输出3个星号，但每个星号都独占一行。
        *   C. `for i in range(3): print("*", end="")` 会重复打印3次 `*` 并且不换行，最终得到 `***` 并且光标停留在同一行。
        *   D. `print("*", end=""); print("*", end=""); print("*")` 会输出 `***`，但是最后一个 `print("*")` 会换行。

4.  **答案：B**
    *   **解析：** 外层循环 `for i in range(n)` 当 `n=4` 时，`i` 的值依次是 0, 1, 2, 3。外层循环会执行4次，每次循环都会打印一行，所以总共是4行。

5.  **答案：7**
    *   **解析：** `range(n)` 会从0到 `n-1` 循环 `n` 次。要打印7个字符，就需要循环7次，所以 `range(7)`。
