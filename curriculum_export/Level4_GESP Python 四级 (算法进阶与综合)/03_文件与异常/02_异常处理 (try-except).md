# 异常处理 (try-except)

## 教学目标
1. 理解 **异常处理 (try-except)** 的基本概念与用途。
2. 掌握 **异常处理 (try-except)** 的核心语法或逻辑。
3. 能够运用该知识点解决简单的编程问题。

## 核心知识点
- **定义**：什么是 异常处理 (try-except)？
- **语法**：标准写法与规范。
- **应用**：在实际编程中的应用场景。

> **注意**：本级别（GESP 4级）**严禁使用 STL 容器**（如 vector, stack, queue 等），必须使用原生数组和手写算法实现。但 **sort 函数** 是允许使用的。

## 教学重点与难点
- **重点**：掌握基本用法。
- **难点**：理解其背后的逻辑或算法思想。
