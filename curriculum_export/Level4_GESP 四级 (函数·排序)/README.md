# GESP 四级 (函数·排序)

## 教学目标
1. 掌握结构体 (`struct`) 的使用。
2. 理解指针与引用的基本概念。
3. 掌握简单的算法复杂度分析。

## 核心知识点
- 结构体：定义、成员访问、结构体数组。
- 排序：`std::sort` 的使用，自定义比较函数。
- 指针入门：地址概念，指针变量。
- 算法分析：时间复杂度 ($O(n)$, $O(n^2)$) 的直观理解。

## 专题列表

1. **函数编程**
2. **二维数组**
3. **结构体**
4. **指针与引用**
   > C++ 指针基础、值传递与引用传递的区别，GESP四级官方考点
5. **算法复杂度**
   > 时间复杂度与空间复杂度的概念、O 记号、常见复杂度量级分析
6. **排序算法**
7. **递推算法**
