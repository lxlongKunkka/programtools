# 结构体排序

## 教学目标

知识  
- 理解结构体（struct）的定义与成员访问  
- 掌握如何对结构体数组按照单个或多重关键字进行排序  
- 理解并能书写自定义比较函数（cmp）的格式与逻辑  

能力  
- 能够使用原生数组存储结构体并进行遍历、输入输出  
- 能使用手写算法（如冒泡/选择）对结构体数组按单字段排序  
- 能使用 `sort` 函数配合自定义 `cmp` 实现多关键字排序  

素养  
- 培养面向对象的思维方式，学会将复杂数据抽象为结构体  
- 培养良好的编码习惯：变量命名简洁、注释清晰、代码风格统一  
- 体会算法工程化：可复用、可扩展的排序模块设计

想象一下，学校要统计班级同学的运动会成绩，并按照分数从高到低排出名次。如果分数相同，则按学号从小到大排列。

## 知识点讲解

1. 结构体回顾  
   - 定义：使用 `struct` 将不同类型的数据组合成一个整体  
   - 访问：通过点运算符 `.` 访问成员  

2. 原生数组存储结构体  
   - `Student a[100];` 存储最多 100 名同学的信息  
   - 输入输出时，按下标遍历  

3. 手写排序（单关键字）  
   - 冒泡或选择排序思想不变，只需比较 `a[i].score` 即可  
   - 练习：如何将算法中的比较部分替换为对结构体成员的比较？  

4. `sort` 函数与自定义比较函数  
   - 头文件：`#include <bits/stdc++.h>` + `using namespace std;`  
   - 调用方式：`sort(a, a + n, cmp);`  
   - `cmp` 函数格式：  
     ```cpp
     bool cmp(类型 a, 类型 b)
     {
         // 若第一个关键字不相等，按第一个关键字排序
         // 否则按第二个关键字排序
     }
     ```  

5. 多关键字排序  
   - 第一关键字：分数（降序）  
   - 第二关键字：学号（升序）  
   - 在 `cmp` 中先判断分数，再判断学号  

6. 复杂度与稳定性  
   - `sort` 时间复杂度 $O(n\\log n)$，一般不保证稳定  
   - 若需稳定排序，可使用 `stable_sort`（本节不涉及）

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Student
{
    char name[20];
    int score;
    int id;
};

// 方法一：手写冒泡排序（单关键字：分数降序）
int main()
{
    int n;
    cin >> n;
    Student a[100];
    for (int i = 0; i < n; i++)
    {
        cin >> a[i].name >> a[i].score >> a[i].id;
    }
    // 冒泡排序
    for (int i = 0; i < n - 1; i++)
    {
        for (int j = 0; j < n - 1 - i; j++)
        {
            if (a[j].score < a[j + 1].score)
            {
                Student tmp = a[j];
                a[j] = a[j + 1];
                a[j + 1] = tmp;
            }
        }
    }
    // 输出结果
    for (int i = 0; i < n; i++)
    {
        cout << a[i].name << " " << a[i].score << " " << a[i].id << endl;
    }
    return 0;
}
```

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Student
{
    char name[20];
    int score;
    int id;
};

// 方法二：使用 sort + 自定义 cmp（多关键字排序）
// 主要逻辑：先按 score 降序；若相等，再按 id 升序
bool cmp(Student a, Student b)
{
    if (a.score != b.score)
    {
        return a.score > b.score;
    }
    return a.id < b.id;
}

int main()
{
    int n;
    cin >> n;
    Student a[100];
    for (int i = 0; i < n; i++)
    {
        cin >> a[i].name >> a[i].score >> a[i].id;
    }

    sort(a, a + n, cmp);  // 核心：调用自定义比较函数

    for (int i = 0; i < n; i++)
    {
        cout << a[i].name << " " << a[i].score << " " << a[i].id << endl;
    }
    return 0;
}
```

## 练习题目

### 基础题
1. 下列哪个 `cmp` 函数能够实现按 `id` 升序排序？  
   A.  
   ```cpp
   bool cmp(Student a, Student b)
   {
       return a.id > b.id;
   }
   ```  
   B.  
   ```cpp
   bool cmp(Student a, Student b)
   {
       return a.id < b.id;
   }
   ```  
   C.  
   ```cpp
   bool cmp(Student a, Student b)
   {
       return a.score < b.score;
   }
   ```  

2. 填空：在原生数组 `Student a[100];` 中，调用 `sort` 排序的语句是  
   `__________(a, a + n, cmp);`

### 进阶题
3. 已知结构体 `Book { char title[30]; int year; };`，请写出 `cmp`，实现先按 `year` 从旧到新排序，如果年份相同，按 `title` 字典序升序。  
   ```cpp
   bool cmp(Book a, Book b)
   {
       if (a.year != b.year)
       {
           return ____________;    // 填写条件
       }
       return ____________;        // 填写条件
   }
   ```

4. 下面关于 `sort(a, a + n, cmp)` 的说法，错误的是（  ）  
   A. 必须在头文件 `#include <bits/stdc++.h>` 后使用 `using namespace std;`  
   B. `cmp` 返回 `true` 表示前者“排在”后者之前  
   C. `sort` 保证在关键字相等时保持原数组顺序  
   D. `sort` 的时间复杂度平均为 $O(n\\log n)$  

### 答案与解析

1. 答案：B  
   解析：`a.id < b.id` 表示 id 小的排在前面，即升序。

2. 答案：sort  
   解析：`sort(a, a + n, cmp);` 使用原生指针区间调用算法。

3. 答案：  
   ```cpp
   return a.year < b.year;
   return strcmp(a.title, b.title) < 0;
   ```  
   解析：先按年份升序（小的先），相等时用 `strcmp` 判断字典序。

4. 答案：C  
   解析：`sort` 并不保证稳定性，关键字相等时顺序可能会改变。
