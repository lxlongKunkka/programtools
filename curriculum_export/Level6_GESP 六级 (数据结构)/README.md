# Level 6：GESP 六级 (数据结构)

## 教学目标
1. 掌握文件操作与流。
2. 理解并应用基础数据结构（栈、队列）。
3. 掌握基础的搜索算法。

## 核心知识点
- 文件IO：`freopen` 或 `fstream`。
- STL容器：`std::vector`, `std::stack`, `std::queue`。
- 搜索：深度优先搜索 (DFS) 基础，广度优先搜索 (BFS) 基础（迷宫问题）。
