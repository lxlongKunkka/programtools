## 知识点讲解

### 1. 什么是变量？

就像我们动画里看到的，计算机内存里有很多“小盒子”。这些“小盒子”就是我们今天要学习的**变量**。

*   **变量**是计算机程序中用来存储数据（比如数字、文字等）的一个“容器”或者“存储空间”。
*   每个变量都有一个独一无二的**名字**，就像你家里的抽屉，每个抽屉都有它放东西的标签，这样你才能找到想拿的东西。
*   变量里的数据是可以**改变**的！这也是它为什么叫“变量”——它里面装的东西可以变来变去。

### 2. `int` 类型的变量：整数的家

计算机里有各种各样的“盒子”，它们用来装不同类型的数据。今天我们学习的是最常用的一种盒子——`int` 类型。

*   `int` 是英文 `integer`（整数）的缩写。
*   `int` 类型的变量专门用来存储**整数**，比如 `1`、`10`、`-5`、`0`、`1000` 等等。
*   如果你想存储小数（比如 `3.14`），`int` 盒子就装不下咯，它会把小数部分丢掉！

### 3. 如何定义 `int` 变量？（创建你的盒子）

在 C++ 编程中，我们需要先告诉计算机我们要创建一个什么样的盒子，以及这个盒子叫什么名字。这个过程就叫做**定义变量**。

语法是：
```cpp
int 变量名;
```
*   `int`：告诉计算机，我要创建一个装整数的盒子。
*   `变量名`：你给这个盒子起的名字，比如 `a`、`b`、`c`、`score`、`age` 等等。在信奥比赛中，我们通常会用单个小写字母作为变量名，比如 `n`, `m`, `i`, `j`。
*   `;`：记住，每条 C++ 语句的末尾都要加上分号，就像我们说话时句号一样，表示一句话说完了。

**例子：**
```cpp
int a; // 定义了一个名为 a 的整数变量（创建了一个叫 a 的整数盒子）
int b; // 定义了一个名为 b 的整数变量
```

### 4. 如何给变量赋值？（往盒子里放东西）

定义好变量后，我们就可以往里面放数字了。这个操作叫做**赋值**。

语法是：
```cpp
变量名 = 值;
```
*   `变量名`：你要往哪个盒子里放东西。
*   `=`：在编程里，这个符号不是“等于”，而是“赋值”的意思。它表示把右边的“值”放进左边的“变量”里。
*   `值`：你要放进去的数字。

**例子：**
```cpp
a = 5; // 把数字 5 放到变量 a 里
b = 10; // 把数字 10 放到变量 b 里
```

### 5. 定义和赋值可以一起完成！

为了方便，我们可以在定义变量的同时就给它赋值：

语法是：
```cpp
int 变量名 = 值;
```
**例子：**
```cpp
int a = 5;  // 定义了一个名为 a 的整数变量，并把 5 放到它里面
int b = 10; // 定义了一个名为 b 的整数变量，并把 10 放到它里面
```

### 6. 如何使用变量？（看看盒子里有什么）

当我们想知道变量里存了什么数字时，可以使用 `cout` 命令把它打印出来。

语法是：
```cpp
cout << 变量名 << endl;
```

**例子：**
```cpp
cout << a << endl; // 会输出变量 a 里存储的数字
```

**重要提示：**
*   变量的值是可以改变的！你可以先给 `a` 赋值 `5`，然后又给 `a` 赋值 `10`。这个时候，`a` 里面原来的 `5` 就会被 `10` 替换掉。变量里永远只保存最新的那个值。

## 代码示例

让我们来看几个简单的 C++ 代码例子，看看这些知识点是怎么用的。

### 示例 1: 定义、赋值、输出

```cpp
#include <bits/stdc++.h> // 这是一个万能头文件，包含了我们常用的大部分功能
using namespace std;     // 告诉计算机我们使用标准命名空间，这样可以少写一些 std::

int main() { // 程序的入口，所有代码从这里开始执行
    // 定义一个名为 a 的整数变量（创建了一个名叫 a 的整数盒子）
    int a;

    // 给变量 a 赋值为 5（把数字 5 放进盒子 a）
    a = 5;

    // 输出变量 a 的值（看看盒子 a 里有什么）
    cout << a << endl; // endl 表示换行，让下一个输出显示在新的一行

    // 定义并初始化一个名为 b 的整数变量（创建盒子 b，并直接把 10 放进去）
    int b = 10;

    // 输出变量 b 的值
    cout << b << endl;

    // 改变变量 a 的值（把盒子 a 里的 5 替换成 20）
    a = 20;

    // 再次输出变量 a 的值，看看是不是变了
    cout << a << endl;

    return 0; // 程序运行成功结束
}
```
**运行结果:**
```
5
10
20
```

### 示例 2: 多个变量的简单运算

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int x = 3; // 定义整数变量 x 并赋值 3
    int y = 7; // 定义整数变量 y 并赋值 7
    int z;     // 定义整数变量 z，暂时不给它赋值

    // 将 x 和 y 的和赋值给 z
    // 计算机先算出 x + y 的结果 (3 + 7 = 10)，然后把 10 放到盒子 z 里
    z = x + y;

    // 输出每个变量的值，并加上一些文字说明，让结果更清楚
    cout << "x 的值是: " << x << endl;
    cout << "y 的值是: " << y << endl;
    cout << "x + y 的结果是: " << z << endl; // 输出 z 盒子里的数字

    return 0;
}
```
**运行结果:**
```
x 的值是: 3
y 的值是: 7
x + y 的结果是: 10
```

### 选择题/填空题

1.  在 C++ 中，如果我们要存储一个整数（比如 `99`），应该使用哪种数据类型？
    A. `float`
    B. `char`
    C. `int`

    A. `int score = 100;`
    B. `score = int 100;`
    C. `100 = score int;`
    D. `int 100 score;`

3.  请填写代码，使程序能正确输出 `result` 的值（`a` 减 `b` 的结果）。
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;
    int main() {
        int a = 5;
        int b = 3;
        int result;
        // 在这里填写代码，计算 a 减 b 的结果并存入 result
        result = ________;
        cout << result << endl;
        return 0;
    }
    ```

4.  阅读下面的代码，请问最终会输出什么？
    ```cpp
    #include <bits/stdc++.h>
    using namespace std;
    int main() {
        int x = 10;
        int y = 20;
        x = y;
        y = 30;
        cout << x << endl;
        return 0;
    }
    ```
    A. `10`
    B. `20`
    C. `30`
    D. `10 20`

5.  变量的“变量”体现在哪里？请用一句话简要说明。
    _________________________________________________

### 答案与解析

1.  **C. `int`**
    *   **解析:** `int` 专门用于存储整数。`float` 用于小数，`char` 用于单个字符，`string` 用于字符串（文字）。

2.  **A. `int score = 100;`**
    *   **解析:** 正确的定义并初始化变量的语法是 `数据类型 变量名 = 值;`。其他选项都不符合 C++ 的语法规则。

3.  **`a - b;`**
    *   **解析:** `result = a - b;` 表示将 `a` 的值减去 `b` 的值，然后把结果赋值给 `result` 变量。

4.  **B. `20`**
    *   **解析:**
        *   `int x = 10;` -> `x` 盒子是 `10`。
        *   `int y = 20;` -> `y` 盒子是 `20`。
        *   `x = y;` -> 把 `y` 盒子里的值 (`20`) 赋给 `x` 盒子。此时 `x` 盒子变成 `20`，`y` 盒子仍然是 `20`。
        *   `y = 30;` -> 把 `y` 盒子里的值 (`20`) 替换成 `30`。此时 `x` 盒子是 `20`，`y` 盒子变成 `30`。
        *   `cout << x << endl;` -> 输出 `x` 盒子里的值，即 `20`。

5.  **变量里存储的数据（值）在程序运行过程中可以被修改。**
    *   **解析:** 变量之所以叫“变量”，就是因为它所代表的存储空间里的数据不是固定不变的，可以在程序运行过程中根据需要进行更新和修改。
