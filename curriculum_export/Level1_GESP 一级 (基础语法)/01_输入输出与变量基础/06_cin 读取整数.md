# cin 读取整数

## 教学目标

*   **知识目标:**
    1.  理解计算机程序中“输入”的概念，认识 `cin` 是 C++ 中用于从键盘读取数据的标准输入流对象。
    2.  掌握 `cin` 读取整数类型数据（`int`）的基本语法，包括 `cin >> 变量名;`。
    3.  了解 `int` 类型变量的声明和作用，知道它是用来存储整数的“盒子”。
    4.  理解 `>>` 运算符（提取运算符）的作用，知道它将数据从输入流提取到变量中。
*   **能力目标:**
    1.  能够独立编写简单的 C++ 程序，实现从键盘输入一个或多个整数。
    2.  能够将输入的整数存储到不同的变量中，并进行简单的输出验证。
    3.  培养学生通过编程解决实际问题（如计算器输入数字）的初步能力。
*   **素养目标:**
    1.  培养学生对编程的兴趣和好奇心，认识到程序与用户交互的重要性。
    2.  提升逻辑思维能力，理解数据流动和存储的基本过程。
    3.  养成严谨细致的编程习惯，如正确书写语法、理解变量作用。

## 知识点讲解

同学们，准备好揭秘 `cin` 的魔法了吗？

1.  **什么是输入？**
    *   我们平时玩游戏、用软件，很多时候都需要我们“告诉”电脑一些信息，比如你的名字、分数、年龄等等。这些我们从键盘、鼠标等设备告诉电脑信息的过程，就叫做“输入”。
    *   今天我们主要学习从键盘输入数字。

2.  **`cin` 是什么？**
    *   在 C++ 语言里，`cin` (读作 "see-in") 是一个非常特别的“侦听器”或者说“接收器”。它专门负责从键盘接收你输入的数据。
    *   你可以把它想象成一个管道的入口，你输入的数据会顺着这个管道流进程序里。

3.  **`>>` 运算符：数据的搬运工**
    *   当我们输入数据后，数据会先进入 `cin` 这个“管道”。那么，怎么把管道里的数据取出来，放到我们程序里能够使用的“盒子”里呢？
    *   这时候就需要 `>>` 这个符号了，它叫做“提取运算符”或“输入运算符”。它就像一个神奇的“小铲子”或者“吸管”，能把 `cin` 管道里的数据“挖”出来，然后搬运到我们指定的变量里。

4.  **变量：存储数据的“盒子”**
    *   我们输入的数据，比如数字 `5`，总要找个地方存起来，才能在程序里使用，对不对？
    *   这个地方就是“变量”。你可以把它想象成一个贴着标签的空盒子，我们可以把数据放进去，以后随时都可以从这个盒子里取出数据来用。
    *   对于整数，我们通常使用 `int` 类型的变量来存储。`int` 就是 integer（整数）的缩写。
    *   **声明变量**：在使用变量之前，我们需要先告诉程序：“我要用一个盒子来存整数，这个盒子叫做 `a`！” 语法就是：`int a;`

5.  **`cin` 读取整数的魔法动画！**
    *   **想象一下：**
        1.  你按下键盘上的数字键，比如“5”。
        2.  这个“5”就像一个小小的数据，它会顺着一条看不见的管道（这就是 `cin`！）跑过来。
        3.  当它跑到 `cin` 管道的出口时，遇到了一个特殊的箭头（`>>`），这个箭头就像一个导向牌，指着一个叫做 `a` 的“盒子”。
        4.  于是，数字“5”就被 `>>` 运算符引导，安全地存放到我们之前声明的 `int a;` 这个变量“盒子”里了！
        5.  （*动画效果建议：可以画一个键盘，一个箭头从键盘指向一个标有“cin”的管道，管道末端有一个“>>”箭头指向一个标有“int a;”的方框，方框内随着输入显示数字。*）

6.  **基本语法：**
    *   要从键盘读取一个整数并存到变量 `a` 中，我们这样写：
        ```cpp
        int a; // 声明一个整数变量 a
        cin >> a; // 从键盘读取一个整数，存入变量 a
        ```
    *   如果想读取多个整数，比如两个整数分别存入变量 `a` 和 `b`：
        ```cpp
        int a, b; // 声明两个整数变量 a 和 b
        cin >> a >> b; // 依次读取两个整数，第一个存入 a，第二个存入 b
        ```
    *   **注意：** 当你输入多个数字时，数字之间可以用 **空格** 或 **回车** 分隔。

## 代码示例

让我们来动手实践一下！

**示例 1：读取一个整数并输出**

```cpp
#include <bits/stdc++.h> // 包含所有常用头文件
using namespace std;     // 使用标准命名空间

int main()
{
    int a; // 声明一个整数变量 a，用来存放我们输入的数字

    cout << "请你输入一个整数，然后按回车键："; // 提示用户输入

    cin >> a; // 使用 cin 读取用户输入的整数，并存入变量 a 中

    cout << "你输入的是：" << a << endl; // 输出我们刚刚读到的数字

    return 0; // 程序运行结束
}
```
*   **代码解读:**
    *   `int a;`：我们准备了一个叫做 `a` 的“盒子”来装整数。
    *   `cout << "..."`：这行代码是提示你，让你知道程序在等你输入。
    *   `cin >> a;`：当程序运行到这里时，它会暂停，等待你在键盘上输入一个整数，然后按下回车键。你输入的数字就会被存到 `a` 这个盒子里。
    *   `cout << a << endl;`：程序会把 `a` 盒子里的数字打印出来，证明它已经成功地收到了你的输入。

**示例 2：读取两个整数并计算它们的和**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a, b; // 声明两个整数变量 a 和 b

    cout << "请你输入两个整数，用空格或回车隔开，然后按回车键：";

    cin >> a >> b; // 依次读取两个整数，第一个存入 a，第二个存入 b

    int s = a + b; // 计算 a 和 b 的和，并存入新的变量 s 中

    cout << "你输入的第一个数是：" << a << endl;
    cout << "你输入的第二个数是：" << b << endl;
    cout << "它们的和是：" << s << endl; // 输出计算结果

    return 0;
}
```
*   **代码解读:**
    *   `int a, b;`：我们准备了两个盒子，一个叫 `a`，一个叫 `b`。
    *   `cin >> a >> b;`：这次我们连续使用了两次 `>>` 运算符，它会先从 `cin` 中提取第一个整数给 `a`，再提取第二个整数给 `b`。
    *   `int s = a + b;`：我们将 `a` 和 `b` 盒子里的数字取出来相加，结果放到 `s` 这个新盒子里。
    *   最后，程序会把 `a`、`b` 和它们的和 `s` 都展示给你看。

## 练习题目

### 选择题

1.  在 C++ 中，哪个关键字用于从键盘读取数据？
    A. `cout`
    B. `cin`
    C. `int`
    D. `main`

2.  要存储一个整数，我们应该声明哪种类型的变量？
    A. `double`
    B. `char`
    C. `int`
    D. `bool`

3.  在语句 `cin >> a;` 中，`>>` 符号的作用是什么？
    A. 输出数据
    B. 比较大小
    C. 将数据从输入流提取到变量中
    D. 进行数学运算

4.  下列哪个 C++ 语句能够正确地从键盘读取一个整数并存入变量 `n` 中？
    A. `cout << n;`
    B. `int n;`
    C. `cin << n;`
    D. `cin >> n;`

5.  如果程序需要读取两个整数 `x` 和 `y`，以下哪种输入方式是正确的？
    A. 用户输入 `10,20`
    B. 用户输入 `10 20`
    C. 用户输入 `10+20`
    D. 用户输入 `10;20`

### 编程题

1.  **基础练习：** 编写一个程序，让用户输入他最喜欢的数字，然后程序输出“你最喜欢的数字是：[用户输入的数字]”。

2.  **进阶练习：** 编写一个程序，让用户输入一个正方形的边长（整数），然后程序计算并输出这个正方形的周长。
    *   提示：周长 = 边长 × 4

---

### 练习题目答案与解析

1.  **答案：B**
    *   **解析：** `cin` (character in) 是 C++ 标准库中用于从标准输入设备（通常是键盘）读取数据的对象。`cout` 用于输出，`int` 是数据类型，`main` 是主函数。

2.  **答案：C**
    *   **解析：** `int` (integer) 是 C++ 中用于存储整数的数据类型。`double` 用于存储浮点数（小数），`char` 用于存储单个字符，`bool` 用于存储布尔值（真或假）。

3.  **答案：C**
    *   **解析：** `>>` 在 `cin` 语句中被称为提取运算符或输入运算符，它的作用是将 `cin` 这个输入流中的数据提取出来，并存放到其右侧的变量中。

4.  **答案：D**
    *   **解析：** `cin >> n;` 是从键盘读取数据并存入变量 `n` 的正确语法。`cout << n;` 是输出 `n` 的值。`int n;` 是声明变量。`cin << n;` 语法错误，`<<` 是输出运算符。

5.  **答案：B**
    *   **解析：** 当使用 `cin >> x >> y;` 读取多个整数时，用户输入的数据之间需要用空格或回车符进行分隔，程序才能正确识别不同的数字。其他分隔符（如逗号、加号、分号）会被视为输入的一部分或导致输入错误。

### 编程题参考答案

**1. 基础练习：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n; // 声明一个整数变量 n 来存储最喜欢的数字

    cout << "请输入你最喜欢的数字："; // 提示用户输入

    cin >> n; // 读取用户输入的数字，存入 n

    cout << "你最喜欢的数字是：" << n << endl; // 输出结果

    return 0;
}
```

**2. 进阶练习：**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a; // 声明一个整数变量 a 来存储正方形的边长
    int z; // 声明一个整数变量 z 来存储周长

    cout << "请输入正方形的边长（整数）："; // 提示用户输入

    cin >> a; // 读取用户输入的边长，存入 a

    z = a * 4; // 计算周长

    cout << "正方形的边长是：" << a << endl;
    cout << "正方形的周长是：" << z << endl; // 输出周长

    return 0;
}
```
