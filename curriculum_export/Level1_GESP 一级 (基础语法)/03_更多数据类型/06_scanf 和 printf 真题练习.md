### （一）导入：为什么要学 `scanf` 和 `printf`（5 分钟）
1. 输入输出是程序与用户交互的核心：程序需要接收用户输入的数据，也需要展示计算结果。

3. 真题引入：展示 2024 年 3 月一级真题第 12 题，明确 `scanf` 是核心输入语句，引出本节课重点。

### （二）核心知识点 1：输入函数 `scanf` 详解（20 分钟）
#### 1. 基本语法
- 功能：从键盘获取用户输入，存储到指定变量中。
- 必备头文件：`#include <stdio.h>` 或万能头文件 `#include <bits/stdc++.h>`。
- 语法格式：`scanf("格式控制符", 地址列表);`
    - 格式控制符：指定输入数据的类型（如 `%d` 对应整数）。
    - 地址列表：变量的内存地址，需用 `&` 符号（字符串数组除外）。

#### 2. 常用格式控制符（必考）
| 数据类型 | 格式符 | 功能 | 示例 |
|----------|--------|------|------|
| 整型（int） | `%d` | 十进制整数输入 | `scanf("%d", &n);`（n 为 int 变量） |
| 长整型（long long） | `%ld` | 十进制长整数输入 | `scanf("%ld", &d);` |
| 单精度浮点型（float） | `%f` | 浮点型数据输入 | `scanf("%f", &fl);` |
| 双精度浮点型（double） | `%lf` | 双精度数据输入 | `scanf("%lf", &db);` |
| 字符型（char） | `%c` | 单个字符输入 | `scanf("%c", &c);` |
| 字符串（char 数组） | `%s` | 字符串输入 | `scanf("%s", str);`（无需 & 符号） |

#### 3. 关键注意事项
- 地址符 `&`：除字符串数组外，所有变量都必须加 `&`，否则无法存储输入数据（考试高频错误点）。
- 输入分隔符：空格、制表符、回车均可作为输入分隔符，`scanf` 会自动跳过。
    - 示例：输入 `10 20` 或 `10\t20`（制表符）或分两行输入 10 和 20，均可正确读取两个整数。
- 数据类型匹配：输入数据类型必须与格式符一致，否则会读取失败（如用 `%d` 接收小数，只会读取整数部分）。

#### 4. 真题示例（2024 年 9 月一级第 11 题）
```cpp
int P;
printf("请输入正整数P:");
scanf("%d", &P); // 输入 14+7，仅读取 14
if (P%3|P%7)
    printf("第5行代码%d,%d",P%3,P%7);
else
    printf("第7行代码%2d",P%3&&P%7);
```
- 输出结果：第5行代码2,0。
- 解析：`scanf("%d")` 只读取 `+` 前的整数 14，14%3=2，14%7=0，满足 if 条件。

### （三）核心知识点 2：输出函数 `printf` 详解（30 分钟）
#### 1. 基本语法
- 功能：按指定格式输出数据或字符串。
- 语法格式：`printf("格式控制符/普通字符串", 输出列表);`
    - 普通字符串：双引号内的内容原样输出（如 `"我考了"`）。
    - 格式控制符：占位符，对应输出列表中的变量或表达式（如 `%d` 对应整数）。
    - 输出列表：需输出的变量、常量或表达式，与格式控制符一一对应。

#### 2. 基础用法示例
```cpp
// 1. 原样输出字符串
printf("我要学好C语言"); 
// 2. 输出常量+格式化
printf("我考了%d分", 666); 
// 3. 输出变量+格式化
int a = 90;
printf("我考了%d分", a); 
// 4. 多变量输出
int b = 85, c = 95;
printf("%d %d %d", a, b, c); 
```

#### 3. 高级格式化技巧（考试重点）
##### （1）宽度控制：`%md`
- 功能：指定输出整数的宽度为 m 位。
- 规则：不足 m 位时，左侧补空格；超过 m 位时，按实际长度输出。
- 示例：`printf("{%2d}", 5);` 输出 `{ 5}`（5 占 2 位，左侧补空格）。

##### （2）补零控制：`%0md`
- 功能：指定宽度为 m 位，不足时左侧补 0（而非空格）。
- 高频场景：时间格式化（如电子表、日期）。
- 示例：电子表时间转换（2024 年真题）
```cpp
int x = 3701; // 输入秒数
// 转换为 时:分:秒，不足两位补0
printf("%02d:%02d:%02d", x/3600, (x%3600)/60, x%60); 
// 输出：01:01:41
```

##### （3）小数保留：`%.nf`
- 功能：保留 n 位小数输出浮点型数据（`float` 用 `%f`，`double` 用 `%lf`）。
- 示例：保留 6 位小数输出（2024 年真题）
```cpp
float f = 2.3;
double d = 3.2;
printf("%.6f %.6lf", f, d); 
// 输出：2.300000 3.200000
```

##### （4）特殊符号：双 `%` 显示 `%`
- 规则：`printf` 中 `%` 是特殊字符，需用 `%%` 才能显示单个 `%`。
- 真题示例（2024 年 6 月一级第 4 题）
```cpp
printf("5%%2={%d}\n", 5%2); 
// 输出：5%2={1}（答案选 C）
```

#### 4. 常见错误规避
- 格式控制符与输出列表不匹配：如 `printf("%d", 3.14);` 会输出乱码。
- 忘记变量名大小写区分：C++ 区分大小写，`int first = 10; printf("%d", First);` 会编译报错（2025 年 3 月真题）。
- 字符串用单引号：`printf('Hello');` 错误，字符串必须用双引号（单引号仅用于单个字符）。

### （四）真题实战：高频题型解析（25 分钟）
#### 题型 1：时间/日期格式化（补零控制）
- 题目：输入一个秒数（不超过 24 小时），按 `xx:xx:xx` 格式输出（如输入 3701，输出 01:01:41）。
- 解题思路：
  1. 计算小时：`x / 3600`（1 小时 = 3600 秒）。
  2. 计算分钟：`(x % 3600) / 60`（剩余秒数换算为分钟）。
  3. 计算秒数：`x % 60`。
  4. 用 `%02d` 格式化，确保不足两位补零。

#### 题型 2：星期/月份计算（取余 + 分支）
- 题目 1：输入今天是星期几（1-6 表示周一到周六，7 表示周日）和天数 N，输出 N 天后是星期几（周日输出“星期天”，其余输出“星期X”）。
- 解题代码：
```cpp
int nowDay, afterDays, N;
cin >> nowDay >> afterDays;
N = nowDay + afterDays;
if (N % 7 == 0) // 7 的倍数为周日
    printf("星期天");
else
    printf("星期%d", N % 7);
```
- 题目 2：当前是 9 月，输入 N，输出 N 个月后的月份（如输入 3，输出 12 月；输入 4，输出 1 月）。
- 解题关键：`M = (9 + N) % 12`，若 `M == 0` 则输出 12 月，否则输出 M 月。

#### 题型 3：多类型变量输入输出（格式符匹配）
- 题目：读入一个字符、一个整数、一个单精度浮点数、一个双精度浮点数，按顺序用空格分隔输出，浮点数保留 6 位小数（2024 年真题）。
- 解题代码：
```cpp
char c;
int i;
float f;
double d;
scanf("%c%d%f%lf", &c, &i, &f, &d);
printf("%c %d %.6f %.6lf", c, i, f, d);
```
