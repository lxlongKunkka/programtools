## 知识点讲解

### 1. 布尔类型 (bool)

在 C++ 中，`bool` 是一种非常特殊的数据类型，它只能存储两个值：
*   `true`：表示“真”，通常可以理解为“是”或者“成立”。
*   `false`：表示“假”，通常可以理解为“否”或者“不成立”。

就像我们之前学习的 `int` 可以存整数，`char` 可以存字符一样，`bool` 就是专门存“真”和“假”的。

**如何声明和赋值布尔变量呢？**
和声明其他变量一样，我们使用 `bool` 关键字：

```cpp
bool b1; // 声明一个布尔变量 b1
b1 = true; // 给 b1 赋值为 true
bool b2 = false; // 声明并初始化一个布尔变量 b2 为 false
```

### 2. 比较运算符

现在，我们知道了如何存储“真”和“假”。那计算机怎么知道一个事情到底是“真”还是“假”呢？这就需要用到我们的**比较运算符**了！
比较运算符用来比较两个值，然后根据比较的结果，返回一个布尔值（`true` 或 `false`）。

我们有六个常用的比较运算符，它们就像是程序的“眼睛”，用来判断两个东西的关系：

| 运算符 | 含义           | 例子           | 结果解释                                       |
| :----- | :------------- | :------------- | :--------------------------------------------- |
| `==`   | 等于           | `5 == 5`       | 5 等于 5，结果是 `true`                      |
| `!=`   | 不等于         | `5 != 6`       | 5 不等于 6，结果是 `true`                    |
| `>`    | 大于           | `7 > 3`        | 7 大于 3，结果是 `true`                      |
| `<`    | 小于           | `2 < 10`       | 2 小于 10，结果是 `true`                     |
| `>=`   | 大于等于       | `8 >= 8`       | 8 大于等于 8，结果是 `true`                  |
| `<=`   | 小于等于       | `4 <= 5`       | 4 小于等于 5，结果是 `true`                  |

**小提示：**
*   `==` 是两个等号，表示“等于”。一个等号 `=` 是赋值运算符，表示“把右边的值赋给左边的变量”。它们可不一样哦！
*   比较运算符不仅可以比较数字，还可以比较变量。

**比较运算的结果：**
无论我们使用哪个比较运算符，它的运算结果都只会是 `true` 或者 `false`。这个结果可以直接赋值给一个 `bool` 变量，或者直接用在需要判断条件的地方（比如我们以后会学的 `if` 语句）。

## 代码示例

让我们通过一些代码例子来实际操作一下吧！

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    // 1. 布尔变量的声明和赋值
    bool b1 = true; // 声明并初始化一个布尔变量 b1 为 true
    bool b2;        // 声明一个布尔变量 b2
    b2 = false;     // 给 b2 赋值为 false

    cout << "b1 的值是: " << b1 << endl; // 输出 b1 的值
    cout << "b2 的值是: " << b2 << endl; // 输出 b2 的值
    // 小知识：在输出时，true 通常显示为 1，false 显示为 0

    cout << "===NEXT===" << endl; // 分隔符，方便查看

    // 2. 比较运算符的使用
    int a = 10;
    int b = 20;
    int c = 10;

    // 等于 (==)
    bool r1 = (a == b); // 10 等于 20 吗？
    cout << "a == b 的结果是: " << r1 << endl; // 应该输出 0 (false)

    bool r2 = (a == c); // 10 等于 10 吗？
    cout << "a == c 的结果是: " << r2 << endl; // 应该输出 1 (true)

    cout << "===NEXT===" << endl;

    // 不等于 (!=)
    bool r3 = (a != b); // 10 不等于 20 吗？
    cout << "a != b 的结果是: " << r3 << endl; // 应该输出 1 (true)

    bool r4 = (a != c); // 10 不等于 10 吗？
    cout << "a != c 的结果是: " << r4 << endl; // 应该输出 0 (false)

    cout << "===NEXT===" << endl;

    // 大于 (>)
    bool r5 = (b > a); // 20 大于 10 吗？
    cout << "b > a 的结果是: " << r5 << endl; // 应该输出 1 (true)

    bool r6 = (a > b); // 10 大于 20 吗？
    cout << "a > b 的结果是: " << r6 << endl; // 应该输出 0 (false)

    cout << "===NEXT===" << endl;

    // 小于 (<)
    bool r7 = (a < b); // 10 小于 20 吗？
    cout << "a < b 的结果是: " << r7 << endl; // 应该输出 1 (true)

    bool r8 = (b < a); // 20 小于 10 吗？
    cout << "b < a 的结果是: " << r8 << endl; // 应该输出 0 (false)

    cout << "===NEXT===" << endl;

    // 大于等于 (>=)
    bool r9 = (a >= c); // 10 大于等于 10 吗？
    cout << "a >= c 的结果是: " << r9 << endl; // 应该输出 1 (true)

    bool r10 = (b >= a); // 20 大于等于 10 吗？
    cout << "b >= a 的结果是: " << r10 << endl; // 应该输出 1 (true)

    cout << "===NEXT===" << endl;

    // 小于等于 (<=)
    bool r11 = (a <= c); // 10 小于等于 10 吗？
    cout << "a <= c 的结果是: " << r11 << endl; // 应该输出 1 (true)

    bool r12 = (a <= b); // 10 小于等于 20 吗？
    cout << "a <= b 的结果是: " << r12 << endl; // 应该输出 1 (true)

    cout << "===NEXT===" << endl;

    // 3. 结合输入进行判断
    int n;
    cout << "请输入一个整数: ";
    cin >> n;

    bool is_positive = (n > 0); // 判断 n 是否大于 0
    cout << n << " 大于 0 吗? " << is_positive << endl;

    bool is_even = (n % 2 == 0); // 判断 n 是否是偶数 (取余 2 等于 0)
    cout << n << " 是偶数吗? " << is_even << endl;

    return 0;
}
```

编写一个程序，从键盘输入一个学生的考试分数 `s`（整数）。

---

### 练习题目答案与解析

**选择题/填空题答案：**

1.  **C. `bool`**
    *   解析：`bool` 类型专门用来存储 `true` 或 `false`。

2.  **C. `7 >= 7`**
    *   解析：
        *   A. `5 == 6` 结果是 `false`。
        *   B. `10 < 8` 结果是 `false`。
        *   C. `7 >= 7` 结果是 `true` (7 大于等于 7 是成立的)。
        *   D. `20 != 20` 结果是 `false`。

3.  **A. `true`**
    *   解析：`12 / 3` 的结果是 `4`。所以 `(4 == 4)` 的结果是 `true`。

4.  **`false`**
    *   解析：`a % 5` 是 `15 % 5`，结果是 `0`。所以表达式变为 `(0 != 0)`，结果是 `false`。

5.  **`true`**
    *   解析：`3 * 4` 的结果是 `12`。所以表达式变为 `(12 > 10)`，结果是 `true`。

---

**编程题参考答案：**

**基础练习：判断大小**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a, b;
    cout << "请输入第一个整数 a: ";
    cin >> a;
    cout << "请输入第二个整数 b: ";
    cin >> b;

    bool r_greater = (a > b); // a 是否大于 b
    bool r_less = (a < b);    // a 是否小于 b
    bool r_equal = (a == b);  // a 是否等于 b

    cout << "a > b 的结果是: " << r_greater << endl;
    cout << "a < b 的结果是: " << r_less << endl;
    cout << "a == b 的结果是: " << r_equal << endl;

    return 0;
}
```
