# if / else 单分支与双分支

## 教学目标

**知识目标：**
1.  理解“条件”在编程中的含义，知道条件判断的结果只有“真”或“假”两种。
2.  掌握 `if` 语句（单分支结构）的语法和作用，知道它在条件为真时执行特定代码块。
3.  掌握 `if-else` 语句（双分支结构）的语法和作用，知道它在条件为真时执行一个代码块，条件为假时执行另一个代码块。

**素养目标：**

## 深度知识点讲解

### 1. 什么是条件？

在编程里，“条件”就像一个“是”或“否”的问题。比如：“这个数字大于10吗？” 答案要么是“是”（真），要么是“否”（假）。电脑会根据这个“真”或“假”的答案来决定接下来做什么。

**生活类比：**
想象你是一个机器人，你的任务是给小朋友分水果。
*   **条件1：** “这个小朋友喜欢苹果吗？”
    *   如果答案是“喜欢”（真），你就给他苹果。
    *   如果答案是“不喜欢”（假），你就不能给他苹果。

### 2. 单分支选择结构：`if` 语句（“如果...就...”）

当我们需要电脑在某个条件成立（为真）的时候，才去执行某段代码，否则就什么也不做时，我们就会用到 `if` 语句。它就像我们说的“如果...就...”一样。

**语法结构：**
```cpp
if (条件) {
    // 当条件为真时，执行这里的代码
}
```
*   `if` 是关键字，意思是“如果”。
*   `()` 小括号里放的就是我们的“条件”，比如 `a > 10`。
*   `{}` 大括号里放的是当条件为真时，电脑要执行的指令。我们称之为“代码块”。

**生活类比：**
你玩游戏，**如果**你打败了敌人，**就**会获得金币。
*   条件：`打败了敌人` （真/假）
*   代码块：`获得金币`

**电脑执行流程：**
1.  电脑首先检查 `()` 里的条件。
2.  如果条件是“真”（True），它就会进入 `{}` 里面，把里面的代码执行一遍。
3.  如果条件是“假”（False），它就会跳过 `{}` 里面的代码，继续执行 `if` 语句后面的代码。

**例子：**
检查一个数字是否是正数。如果是，就打印“这是一个正数！”。
```cpp
int n = 5;
if (n > 0) { // 条件：n大于0吗？ (5 > 0 是真)
    cout << "这是一个正数！" << endl; // 会执行
}
// 如果 n = -3，条件 (-3 > 0) 是假，就不会打印。
```

**关键点：**
*   `if` 语句只管“条件为真”时要做的事情。如果条件为假，它就“不干了”，直接跳过。
*   条件表达式必须放在小括号 `()` 里。
*   要执行的代码放在大括号 `{}` 里。即使只有一行代码，也建议加上大括号，这是好习惯，能避免很多错误。

**常见误区：**
*   **误区1：** 把 `==` (等于) 写成 `=` (赋值)。
    *   `if (a = 10)` 是错误的！`a = 10` 是把10赋值给a，并且这个赋值表达式本身的值就是10 (非0即真)。我们想比较是否相等，应该用 `if (a == 10)`。
*   **误区2：** 忘记条件表达式后的分号。
    *   `if (n > 0); { ... }` 这里的分号表示 `if` 语句在判断后立即结束，后面的 `{}` 块就变成了无论条件真假都会执行的普通代码块。这是个隐蔽的错误！正确的 `if` 语句后面是没有分号的。

### 3. 双分支选择结构：`if-else` 语句（“如果...就...，否则...就...”）

当我们需要电脑在某个条件成立（为真）的时候做一件事，而在条件不成立（为假）的时候做另一件事时，我们就会用到 `if-else` 语句。它就像我们说的“如果...就...，否则（就）...”一样。

**语法结构：**
```cpp
if (条件) {
    // 当条件为真时，执行这里的代码
} else {
    // 当条件为假时，执行这里的代码
}
```
*   `else` 是关键字，意思是“否则”。
*   `else` 后面不需要再写条件，因为它表示的是 `if` 条件不成立时的所有情况。

**生活类比：**
你准备出门，**如果**下雨了，**就**带伞；**否则**（如果没下雨），**就**戴帽子。
*   条件：`下雨了` （真/假）
*   `if` 代码块：`带伞`
*   `else` 代码块：`戴帽子`

**电脑执行流程：**
1.  电脑首先检查 `()` 里的条件。
2.  如果条件是“真”（True），它就会进入 `if` 后面的 `{}` 里，把里面的代码执行一遍，然后跳过 `else` 后面的 `{}`。
3.  如果条件是“假”（False），它就会跳过 `if` 后面的 `{}`，直接进入 `else` 后面的 `{}` 里，把里面的代码执行一遍。
4.  无论哪种情况，`if` 和 `else` 两个代码块中，只会有一个被执行。

**例子：**
判断一个数字是正数还是非正数。
```cpp
int n = -5;
if (n > 0) { // 条件：n大于0吗？ (-5 > 0 是假)
    cout << "这是一个正数！" << endl; // 不会执行
} else {
    cout << "这是一个非正数！" << endl; // 会执行
}
```

**关键点：**
*   `if-else` 语句保证了两个代码块中必有一个且只有一个会被执行。
*   `else` 后面不能有条件，它代表的是 `if` 条件的“反面”或“其余”情况。

### 4. 比较运算符

我们用来构造“条件”的符号叫做比较运算符，它们用来比较两个值的大小关系。
*   `>`：大于。例如 `a > b`，判断 `a` 是否大于 `b`。
*   `<`：小于。例如 `a < b`，判断 `a` 是否小于 `b`。
*   `==`：等于。例如 `a == b`，判断 `a` 是否等于 `b`。
*   `>=`：大于等于。
*   `<=`：小于等于。
*   `!=`：不等于。

GESP一级主要掌握前三个 `>`、`<`、`==` 即可。

## 典型例题精讲

### 例题1：判断是否及格 (单分支 `if`)

**题目描述：**
小明参加了一次编程测验，满分100分。如果他的分数达到60分（含60分）或以上，就算及格。请你编写一个程序，输入小明的分数，如果他及格了，就输出“恭喜你，及格了！”。

**思路分析：**
1.  首先，我们需要从键盘读取小明的分数。可以用一个整数变量 `s` 来存储。
2.  然后，我们需要判断这个分数 `s` 是否大于或等于60。
3.  如果条件 `s >= 60` 成立（为真），我们就打印那句祝贺的话。
4.  如果条件不成立（为假），也就是分数低于60，我们什么也不做，因为题目只要求及格时输出。这正好符合 `if` 单分支结构的特点。

**代码实现：**
```cpp
#include <bits/stdc++.h> // 包含所有常用库，方便同学们使用

int main() {
    // 告诉电脑，我们要用中文输出，防止乱码（部分编译器可能需要）
    // std::ios::sync_with_stdio(false);
    // std::cin.tie(NULL);

    int s; // 声明一个整数变量 s，用来存储分数
    std::cout << "请输入小明的分数: "; // 提示用户输入
    std::cin >> s; // 从键盘读取分数，存入变量 s

    // 判断分数是否及格
    if (s >= 60) { // 如果 s 大于或等于 60
        std::cout << "恭喜你，及格了！" << std::endl; // 就输出这句话
    }
    // 如果 s 小于 60，这里什么也不会发生，程序直接结束

    return 0; // 程序正常结束
}
```

**复杂度分析：**
*   时间复杂度：O(1)。程序只执行常数次操作（一次输入，一次判断，一次输出），与输入分数的大小无关。
*   空间复杂度：O(1)。程序只使用了常数个变量来存储数据。

### 例题2：判断奇偶数 (双分支 `if-else`)

**题目描述：**
输入一个整数，判断它是奇数还是偶数，并输出相应的结果。

**思路分析：**
1.  首先，我们需要从键盘读取一个整数。用变量 `n` 来存储。
2.  偶数的特点是能被2整除，也就是它除以2的余数是0。奇数就是不能被2整除，除以2的余数是1（对于正整数）。
3.  我们可以用取模运算符 `%` 来计算余数。`n % 2` 表示 `n` 除以 `2` 的余数。
4.  **如果** `n % 2 == 0` （余数是0，条件为真），**就**说明它是偶数，输出“偶数”。
5.  **否则**（如果余数不是0，条件为假），**就**说明它是奇数，输出“奇数”。这正好符合 `if-else` 双分支结构的特点。

**代码实现：**
```cpp
#include <bits/stdc++.h>

int main() {
    int n; // 声明一个整数变量 n
    std::cout << "请输入一个整数: ";
    std::cin >> n; // 从键盘读取整数，存入变量 n

    // 判断是奇数还是偶数
    if (n % 2 == 0) { // 如果 n 除以 2 的余数是 0 (偶数)
        std::cout << n << " 是偶数。" << std::endl;
    } else { // 否则 (余数不是 0，就是奇数)
        std::cout << n << " 是奇数。" << std::endl;
    }

    return 0;
}
```

**复杂度分析：**
*   时间复杂度：O(1)。程序只执行常数次操作。
*   空间复杂度：O(1)。程序只使用了常数个变量。

### 例题3：比较大小 (双分支 `if-else`)

**题目描述：**
输入两个整数 `a` 和 `b`，如果 `a` 大于 `b`，则输出“a比b大”；否则（即 `a` 小于或等于 `b`），输出“a不比b大”。

**思路分析：**
1.  需要从键盘读取两个整数，分别存入变量 `a` 和 `b`。
2.  **如果** `a > b` 这个条件成立（为真），**就**输出“a比b大”。
3.  **否则**（如果 `a > b` 不成立，即 `a` 小于或等于 `b`），**就**输出“a不比b大”。这又是典型的 `if-else` 结构。

**代码实现：**
```cpp
#include <bits/stdc++.h>

int main() {
    int a, b; // 声明两个整数变量 a 和 b
    std::cout << "请输入第一个整数 a: ";
    std::cin >> a; // 读取 a
    std::cout << "请输入第二个整数 b: ";
    std::cin >> b; // 读取 b

    // 比较 a 和 b 的大小
    if (a > b) { // 如果 a 大于 b
        std::cout << a << " 比 " << b << " 大。" << std::endl;
    } else { // 否则 (a 小于或等于 b)
        std::cout << a << " 不比 " << b << " 大。" << std::endl;
    }

    return 0;
}
```

**复杂度分析：**
*   时间复杂度：O(1)。
*   空间复杂度：O(1)。

## 教学评价与作业

**教学评价：**

1.  **课堂观察：** 观察学生在互动环节和即时练习中是否积极参与，能否正确理解并运用 `if` 和 `if-else` 结构。
2.  **代码检查：** 检查学生完成的即时练习代码，看是否存在语法错误（如 `=` 和 `==` 混淆，忘记大括号等），逻辑是否正确。
3.  **问题回答：** 根据学生对提问的回答，评估他们对“条件”概念和两种分支结构区别的掌握程度。

**课后作业：**

1.  **必做题：** 完成“分层练习题目”中的所有“基础巩固”和“能力提升”题目，并将代码保存好。
2.  **选做题（挑战）：** 尝试完成“拓展挑战”中的“判断闰年”题目。
3.  **思考与分享：**
    *   在家里，你还能找到哪些可以用“如果...就...”或者“如果...就...，否则就...”来描述的场景？例如，倒垃圾时，如果垃圾满了，就拿出去扔；否则，就继续用。
    *   尝试用文字描述你发现的这些场景，并想象一下如何用 `if` 或 `if-else` 来编写程序。

**预告：**
下节课我们将学习更复杂的条件判断，比如如何同时判断多个条件，或者在多个选择中做出决定，敬请期待！
