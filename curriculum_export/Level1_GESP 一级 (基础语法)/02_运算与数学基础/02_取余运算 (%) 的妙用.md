小明有 **10 颗糖果**，他想平均分给 **3 个好朋友**。每个小朋友能分到几颗呢？还剩下几颗糖果呢？

## 知识点讲解

### 1. 什么是取余运算 (%)？

取余运算，也叫做模运算，它的符号是 `%`。
当你想知道一个整数 `a` 除以另一个整数 `b` 之后，剩下的“零头”是多少时，就可以使用 `%` 运算符。

**语法格式**：`a % b`

-   `a` 被称为“被除数”。
-   `b` 被称为“除数”，**注意 `b` 不能是 0**，因为除数不能为 0，否则程序会报错！

**举个例子：**
-   `10 % 3` 的结果是 `1` (因为 `10` 除以 `3` 等于 `3` 余 `1`)
-   `7 % 2` 的结果是 `1` (因为 `7` 除以 `2` 等于 `3` 余 `1`)
-   `15 % 5` 的结果是 `0` (因为 `15` 除以 `5` 等于 `3` 余 `0`，正好整除)

### 2. 取余运算的规则

1.  **除数不能为 0**：这是最重要的规则！
2.  **结果的范围**：对于 `a % b`，当 `a` 和 `b` 都是正整数时，结果（余数）总是在 `0` 到 `b-1` 之间。
    -   比如 `a % 5`，结果只可能是 `0, 1, 2, 3, 4`。
3.  **结果的符号**：在 C++ 中，取余运算结果的符号与**被除数 `a` 的符号**相同。
    -   例如：`-10 % 3` 的结果是 `-1`。
    -   但我们 Level 1 阶段，主要处理正整数的取余运算，所以暂时不用担心负数的情况。

### 3. 取余运算的妙用

取余运算虽然简单，但它的用途可不小哦！

#### 妙用一：判断奇偶性 🕵️‍♀️

一个整数，如果能被 `2` 整除，那么它就是偶数；如果不能被 `2` 整除，余数是 `1`，那么它就是奇数。
-   **偶数**：`a % 2 == 0`
-   **奇数**：`a % 2 == 1`

这在很多程序中都非常有用！

#### 妙用二：获取数字的个位 🔢

任何一个正整数 `a` 对 `10` 取余，得到的结果就是它的个位数字。
-   例如：`123 % 10` 的结果是 `3` (个位是 `3`)
-   例如：`45 % 10` 的结果是 `5` (个位是 `5`)

是不是很方便？如果我们想获取十位、百位呢？
-   获取十位：先将数字除以 `10` (去掉个位)，再对 `10` 取余。
    -   例如：`123` 的十位是 `2`。我们可以先 `123 / 10 = 12`，再 `12 % 10 = 2`。
-   获取百位：先将数字除以 `100` (去掉个位和十位)，再对 `10` 取余。
    -   例如：`123` 的百位是 `1`。我们可以先 `123 / 100 = 1`，再 `1 % 10 = 1`。
    -   其实对于百位，`123 / 100` 直接得到 `1` 就行了，因为是整数除法。

#### 妙用三：判断能否整除 ✅

如果 `a % b` 的结果是 `0`，就说明 `a` 能被 `b` 整除。
-   例如：`100 % 25 == 0`，说明 `100` 能被 `25` 整除。
-   例如：`17 % 3 != 0`，说明 `17` 不能被 `3` 整除。

## 代码示例

让我们用 C++ 代码来实际操作一下取余运算吧！

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
    // 示例 1: 基本取余运算
    int a = 10;
    int b = 3;
    int r = a % b; // 10 除以 3，余数是 1
    cout << a << " % " << b << " = " << r << endl; // 输出: 10 % 3 = 1

    int c = 15;
    int d = 5;
    r = c % d; // 15 除以 5，余数是 0
    cout << c << " % " << d << " = " << r << endl; // 输出: 15 % 5 = 0

    cout << "--------------------" << endl;

    // 示例 2: 判断奇偶性
    int n;
    cout << "请输入一个整数: ";
    cin >> n;

    if (n % 2 == 0)
    {
        cout << n << " 是偶数。" << endl;
    }
    else
    {
        cout << n << " 是奇数。" << endl;
    }

    cout << "--------------------" << endl;

    // 示例 3: 获取数字的个位
    int x;
    cout << "请输入一个正整数: ";
    cin >> x;

    int g = x % 10; // 对 10 取余，得到个位
    cout << x << " 的个位是: " << g << endl;

    // 尝试获取十位 (需要先用除法)
    int s = (x / 10) % 10; // 先除以 10，再对 10 取余
    cout << x << " 的十位是: " << s << endl;

    // 尝试获取百位 (需要先用除法)
    int b1 = (x / 100) % 10; // 先除以 100，再对 10 取余
    cout << x << " 的百位是: " << b1 << endl;

    cout << "--------------------" << endl;

    // 示例 4: 判断能否被 3 整除
    int y;
    cout << "请输入一个整数，判断它能否被 3 整除: ";
    cin >> y;

    if (y % 3 == 0)
    {
        cout << y << " 能被 3 整除。" << endl;
    }
    else
    {
        cout << y << " 不能被 3 整除。" << endl;
    }

    return 0;
}
```

**代码运行结果示例 (根据用户输入可能不同):**
```
10 % 3 = 1
15 % 5 = 0
--------------------
请输入一个整数: 7
7 是奇数。
--------------------
请输入一个正整数: 123
123 的个位是: 3
123 的十位是: 2
123 的百位是: 1
--------------------
请输入一个整数，判断它能否被 3 整除: 9
9 能被 3 整除。
```

### 练习题目答案与解析

**选择题 / 填空题答案：**

1.  **C. `%`**
    *   **解析：** `/` 是整数除法，`*` 是乘法，`+` 是加法，`%` 才是取余运算。
2.  **B. `4`**
    *   **解析：** `25` 除以 `7` 等于 `3` (因为 `7 * 3 = 21`)，余数是 `25 - 21 = 4`。
3.  **B. `0`**
    *   **解析：** 偶数就是能被 `2` 整除的数，能被 `2` 整除的数余数当然是 `0`。
4.  **B. `num % 10`**
    *   **解析：** 对 `10` 取余是获取个位数字的常用方法。
5.  **C. `p % 4 == 0`**
    *   **解析：** 如果一个数能被 `4` 整除，那么它除以 `4` 的余数一定是 `0`。选项 A (`p / 4 == 0`) 只有当 `p` 是 `0, 1, 2, 3` 时才成立；选项 B (`p % 4 != 0`) 是不能被 `4` 整除；选项 D (`p == 4`) 只是判断 `p` 是否等于 `4`。
