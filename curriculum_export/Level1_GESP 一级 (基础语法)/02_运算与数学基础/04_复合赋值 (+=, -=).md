## 知识点讲解

### 1. 回顾普通赋值和算术运算
我们知道，最基本的赋值是使用 `=` 符号，比如 `int a = 10;`。
我们还学过加法 `+` 和减法 `-` 运算符。
当我们需要让一个变量的值“加上”或“减去”另一个值，然后把新结果再赋给这个变量时，通常我们会这样写：
```cpp
int a = 5;
a = a + 3; // a 的值会变成 5 + 3 = 8
```
或者
```cpp
int b = 10;
b = b - 4; // b 的值会变成 10 - 4 = 6
```
这种写法完全正确，也很容易理解。但是，聪明的程序员们发现，这种 `变量 = 变量 运算符 表达式;` 的模式太常用了，所以他们发明了一种更简洁的写法！

### 2. 认识复合赋值运算符 `+=`
当我们要写 `a = a + b;` 时，我们可以用复合赋值运算符 `+=` 来代替。
`a += b;` 的意思就是 `a = a + b;`。
它读作“a 加等于 b”。
是不是一下子就简洁了很多？

**举个例子：**
你现在有 10 颗糖果。
如果妈妈又给了你 5 颗，你的糖果数量会变成 `10 + 5 = 15`。
用普通写法是：`t = t + 5;` (假设 `t` 是糖果数量)
用复合赋值写法就是：`t += 5;`

### 3. 认识复合赋值运算符 `-=`
同理，当我们要写 `a = a - b;` 时，我们可以用复合赋值运算符 `-=` 来代替。
`a -= b;` 的意思就是 `a = a - b;`。
它读作“a 减等于 b”。

**举个例子：**
你现在有 15 颗糖果。
如果你吃了 3 颗，你的糖果数量会变成 `15 - 3 = 12`。
用普通写法是：`t = t - 3;`
用复合赋值写法就是：`t -= 3;`

### 4. 复合赋值运算符的好处
*   **代码更简洁**：一行代码代替了稍微长一点的写法。
*   **阅读更方便**：一眼就能看出这个变量自身发生了增减。
*   **减少错误**：减少了重复输入变量名的机会，从而减少了手误的可能性。

### 小结
复合赋值运算符 `+=` 和 `-=` 是对 `a = a + b;` 和 `a = a - b;` 这种常见模式的简化写法。它们让我们的代码更优雅、更高效！

## 代码示例

让我们通过几个小例子来看看复合赋值运算符的威力吧！

### 示例 1: 使用 `+=` 增加数值
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int a = 10; // 定义一个整数变量 a，初始值为 10
    cout << "a 的初始值是: " << a << endl;

    // 传统写法：a = a + 5;
    // a = a + 5;
    // cout << "a 加上 5 后的值 (传统写法) 是: " << a << endl;

    // 复合赋值写法：a += 5;
    a += 5; // 等价于 a = a + 5;，现在 a 的值是 10 + 5 = 15
    cout << "a 加上 5 后的值 (复合赋值) 是: " << a << endl;

    int b = 100;
    int c = 20;
    b += c; // 等价于 b = b + c;，现在 b 的值是 100 + 20 = 120
    cout << "b 加上 c 后的值是: " << b << endl;

    return 0;
}
```

### 示例 2: 使用 `-=` 减少数值
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int x = 20; // 定义一个整数变量 x，初始值为 20
    cout << "x 的初始值是: " << x << endl;

    // 传统写法：x = x - 7;
    // x = x - 7;
    // cout << "x 减去 7 后的值 (传统写法) 是: " << x << endl;

    // 复合赋值写法：x -= 7;
    x -= 7; // 等价于 x = x - 7;，现在 x 的值是 20 - 7 = 13
    cout << "x 减去 7 后的值 (复合赋值) 是: " << x << endl;

    int y = 50;
    int z = 15;
    y -= z; // 等价于 y = y - z;，现在 y 的值是 50 - 15 = 35
    cout << "y 减去 z 后的值是: " << y << endl;

    return 0;
}
```

### 示例 3: 综合运用，模拟购物
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int money = 100; // 小明有 100 元零花钱
    cout << "小明初始有 " << money << " 元。" << endl;

    // 买了一本故事书，花了 35 元
    money -= 35; // 等价于 money = money - 35;
    cout << "买书花了 35 元，现在还剩 " << money << " 元。" << endl;

    // 妈妈又给了小明 20 元奖励
    money += 20; // 等价于 money = money + 20;
    cout << "妈妈给了 20 元，现在有 " << money << " 元。" << endl;

    // 买了一包零食，花了 12 元
    money -= 12; // 等价于 money = money - 12;
    cout << "买零食花了 12 元，现在还剩 " << money << " 元。" << endl;

    cout << "最终，小明还剩 " << money << " 元。" << endl;

    return 0;
}
```

## 练习题目

### 选择题

1.  下列哪个表达式与 `x = x + y;` 的功能是相同的？
    A. `x = +y;`
    B. `x += y;`
    C. `x - = y;`
    D. `y += x;`

2.  如果 `int a = 5;`，执行语句 `a += 3;` 后，`a` 的值是多少？
    A. 3
    B. 5
    C. 8
    D. 15

3.  如果 `int b = 12;`，执行语句 `b -= 4;` 后，`b` 的值是多少？
    A. 4
    B. 8
    C. 12
    D. 16

4.  以下代码片段的输出结果是什么？
    ```cpp
    int n = 7;
    n += 3;
    n -= 5;
    cout << n << endl;
    ```
    A. 5
    B. 7
    C. 10
    D. 15

### 填空题

5.  将 `total = total - item_price;` 改写成复合赋值运算符的形式是 `total ______ item_price;`。

---

### 练习题目答案与解析

1.  **答案：B**
    *   **解析：** `x = x + y;` 表示将 `x` 的值加上 `y` 的值，再赋回给 `x`。复合赋值运算符 `+=` 正是用来实现这个功能的，`x += y;` 等价于 `x = x + y;`。

2.  **答案：C**
    *   **解析：** `a += 3;` 等价于 `a = a + 3;`。所以 `a` 的值会从 `5` 变为 `5 + 3 = 8`。

3.  **答案：B**
    *   **解析：** `b -= 4;` 等价于 `b = b - 4;`。所以 `b` 的值会从 `12` 变为 `12 - 4 = 8`。

4.  **答案：A**
    *   **解析：**
        *   `int n = 7;`：`n` 的初始值是 7。
        *   `n += 3;`：`n` 变为 `7 + 3 = 10`。
        *   `n -= 5;`：`n` 变为 `10 - 5 = 5`。
        *   最后输出 `n`，所以结果是 `5`。

5.  **答案：-=**
    *   **解析：** `total = total - item_price;` 表示 `total` 减去 `item_price`，再把结果赋给 `total`。这正是复合赋值运算符 `-=` 的作用，所以是 `total -= item_price;`。
