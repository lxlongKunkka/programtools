# GESP 一级 (基础语法)

零基础入门，掌握C++基本语法，培养计算思维，能够将简单数学问题转化为代码。

## 专题列表

1. **输入输出与变量基础**
   > 从第一个 C++ 程序，到变量的定义与赋值，再到用 cin/cout 与程序交互——搭建编程的第一块基石。
2. **运算与数学基础**
   > 整数与小数的四则运算、取余妙用、整除与取模规律，以及常见的数学操作，是一切算法的运算基础。
3. **更多数据类型**
4. **条件结构专题**
   > if/else 单分支与双分支、多分支嵌套、switch 语句的用法与经典例题
5. **循环结构专题**
   > for / while / do-while 三种循环、break 与 continue 的用法与经典题
