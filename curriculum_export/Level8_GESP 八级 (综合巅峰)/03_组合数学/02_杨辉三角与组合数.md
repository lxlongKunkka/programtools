## 知识点讲解

### 1. 组合数的定义与性质

我们先来认识一下什么是组合数。

**定义：** 从 n 个不同元素中，不考虑顺序地取出 k 个元素，称为一个组合。所有可能的组合的数量，就叫做组合数，记作 C(n, k) 或 $\binom{n}{k}$。

**公式：** 组合数 C(n, k) 的计算公式为：
C(n, k) = n! / (k! * (n-k)!)
其中，`!` 表示阶乘，例如 5! = 5 * 4 * 3 * 2 * 1。

**特殊值：**
*   C(n, 0) = 1 (从 n 个元素中取出 0 个，只有一种方法：一个都不取)
*   C(n, n) = 1 (从 n 个元素中取出 n 个，只有一种方法：全部取走)
*   C(n, 1) = n (从 n 个元素中取出 1 个，有 n 种方法)

**对称性：** C(n, k) = C(n, n-k)
这个性质很有趣，从 n 个元素中选 k 个出来，和从 n 个元素中选 n-k 个不选，方法数是等价的！

**帕斯卡恒等式 (Pascal's Identity)：** C(n, k) = C(n-1, k-1) + C(n-1, k)
这个恒等式是杨辉三角构造的数学基础，它告诉我们，从 n 个元素中选 k 个，可以分成两种情况：
1.  选中了某个特定元素：那么还需要从剩下的 n-1 个元素中选 k-1 个，即 C(n-1, k-1)。
2.  没有选中某个特定元素：那么需要从剩下的 n-1 个元素中选 k 个，即 C(n-1, k)。

### 2. 杨辉三角 (Pascal's Triangle)

杨辉三角是一个以三角形排列的二项式系数。它的构造方法非常简单：

1.  第一行（第 0 行）是 1。
2.  每一行的开头和结尾都是 1。
3.  中间的每个数是它“肩上”两个数之和。

让我们画一个杨辉三角的前几行：

```
        1         (C(0,0))
       1 1        (C(1,0), C(1,1))
      1 2 1       (C(2,0), C(2,1), C(2,2))
     1 3 3 1      (C(3,0), C(3,1), C(3,2), C(3,3))
    1 4 6 4 1     (C(4,0), C(4,1), C(4,2), C(4,3), C(4,4))
   1 5 10 10 5 1  (C(5,0), C(5,1), C(5,2), C(5,3), C(5,4), C(5,5))
```

仔细观察，你会发现杨辉三角的第 `n` 行（从 0 行开始计数）的第 `k` 个数（从 0 列开始计数），正好就是组合数 C(n, k)！

### 3. 组合数的计算方法

组合数 C(n, k) 的值增长非常快，即使 n 和 k 不大，结果也可能超出 `int` 类型的范围。因此，在计算时我们通常使用 `long long` 类型。

#### 方法一：动态规划 (递推法)

利用帕斯卡恒等式 C(n, k) = C(n-1, k-1) + C(n-1, k) 来填充一个二维数组。这种方法适用于需要计算多个组合数，或者 n 较小（例如 n <= 2000）的情况。

**思路：**
创建一个二维数组 `a[N][N]`，其中 `a[i][j]` 存储 C(i, j) 的值。
1.  初始化 `a[i][0] = 1` 和 `a[i][i] = 1`。
2.  对于 `i` 从 2 到 `n`，`j` 从 1 到 `i-1`，计算 `a[i][j] = a[i-1][j-1] + a[i-1][j]`。

#### 方法二：O(k) 单次计算法

当我们需要计算单个 C(n, k)，且 k 相对较小（例如 k <= 1000），而 n 可能很大时，可以直接利用组合数的展开式进行优化计算：
C(n, k) = (n * (n-1) * ... * (n-k+1)) / (k * (k-1) * ... * 1)

**思路：**
1.  C(n, k) = C(n, k-1) * (n-k+1) / k
2.  我们可以从 C(n, 0) = 1 开始，逐步计算 C(n, 1), C(n, 2), ..., 直到 C(n, k)。
3.  或者更直接地，通过循环 k 次，每次乘上分子项，除以分母项。需要注意的是，每次除法都必须是整除，否则会出错。幸运的是，C(n, k) 总是整数，所以这个过程总是能整除的。

#### 方法三：预处理阶乘与逆元 (模运算)

在很多算法竞赛题目中，组合数的结果会非常大，题目通常会要求将结果对一个大质数 P 取模。这时，我们不能直接使用除法，因为 (a / b) % P 不等于 (a % P / b % P)。我们需要用到**模逆元**。

**模逆元：** 如果 `a * x ≡ 1 (mod P)`，那么 `x` 就是 `a` 在模 P 意义下的乘法逆元，记作 `a^(-1)`。
所以，C(n, k) % P = (n! * (k!)^(-1) * ((n-k)!)^(-1)) % P

**思路：**
1.  **预处理阶乘：** 计算 `fact[i] = i! % P`。
2.  **预处理阶乘逆元：** 计算 `inv_fact[i] = (i!)^(-1) % P`。
    *   如果 P 是质数，可以使用**费马小定理**：`a^(P-2) ≡ a^(-1) (mod P)`。所以 `inv_fact[i] = power(fact[i], P-2, P)`。
    *   为了提高效率，可以先计算 `inv_fact[N]` (N是最大阶乘)，然后利用 `inv_fact[i-1] = inv_fact[i] * i % P` 倒推计算其他逆元。
3.  **计算 C(n, k) % P：** `C(n, k) % P = (fact[n] * inv_fact[k] % P * inv_fact[n-k] % P) % P`。

这种方法适用于 n 较大（例如 n <= 10^5 或 10^6），且需要多次查询组合数的情况。

## 代码示例

### 1. 生成杨辉三角并查询 C(n, k) (动态规划递推法)

```cpp
#include <bits/stdc++.h>
using namespace std;

const int N = 1005; // 定义最大行数，根据题目要求调整
long long a[N][N]; // 使用 long long 存储组合数，防止溢出

void init_pascal_triangle(int n_max) {
    // 初始化杨辉三角
    for (int i = 0; i <= n_max; ++i) {
        a[i][0] = 1; // 每行的第一个数是 1 (C(i,0))
        for (int j = 1; j <= i; ++j) {
            if (i == j) {
                a[i][j] = 1; // 每行的最后一个数是 1 (C(i,i))
            } else {
                a[i][j] = a[i-1][j-1] + a[i-1][j]; // 帕斯卡恒等式
            }
        }
    }
}

int main() {
    // 预处理杨辉三角，假设最大需要计算到第 10 行
    init_pascal_triangle(10); 

    cout << "杨辉三角前10行:" << endl;
    for (int i = 0; i <= 10; ++i) {
        for (int j = 0; j <= i; ++j) {
            cout << a[i][j] << " ";
        }
        cout << endl;
    }

    // 查询 C(n, k)
    int n = 5, k = 2;
    cout << "C(" << n << ", " << k << ") = " << a[n][k] << endl; // C(5,2) = 10

    n = 7, k = 3;
    cout << "C(" << n << ", " << k << ") = " << a[n][k] << endl; // C(7,3) = 35

    return 0;
}
```

### 2. 单次计算 C(n, k) (O(k) 优化递推法)

```cpp
#include <bits/stdc++.h>
using namespace std;

long long calculate_c_nk_ok(int n, int k) {
    if (k < 0 || k > n) {
        return 0; // k 不合法
    }
    if (k == 0 || k == n) {
        return 1;
    }
    if (k > n / 2) { // 利用对称性 C(n, k) = C(n, n-k) 优化，使 k 尽可能小
        k = n - k;
    }

    long long ans = 1;
    for (int i = 0; i < k; ++i) {
        ans = ans * (n - i) / (i + 1); // 逐步计算 C(n, i) = C(n, i-1) * (n-i+1) / i
    }
    return ans;
}

int main() {
    int n = 30, k = 5;
    cout << "C(" << n << ", " << k << ") = " << calculate_c_nk_ok(n, k) << endl; // C(30,5) = 142506

    n = 60, k = 10;
    cout << "C(" << n << ", " << k << ") = " << calculate_c_nk_ok(n, k) << endl; // C(60,10) = 7539402756

    return 0;
}
```

### 3. 模意义下计算 C(n, k) (预处理阶乘与逆元)

```cpp
#include <bits/stdc++.h>
using namespace std;

const int N_MAX = 1000005; // 最大 n 值
const long long MOD = 1000000007; // 模数，通常是一个大质数

long long fact[N_MAX];      // 存储阶乘
long long inv_fact[N_MAX];  // 存储阶乘的模逆元

// 快速幂函数：计算 (b^e) % mod
long long power(long long b, long long e, long long mod) {
    long long r = 1;
    b %= mod;
    while (e > 0) {
        if (e % 2 == 1) {
            r = (r * b) % mod;
        }
        b = (b * b) % mod;
        e /= 2;
    }
    return r;
}

// 预处理阶乘和阶乘逆元
void init_combinations(int n_max) {
    fact[0] = 1;
    inv_fact[0] = 1; // 0! 的逆元是 1 的逆元，即 1

    for (int i = 1; i <= n_max; ++i) {
        fact[i] = (fact[i-1] * i) % MOD;
    }
    
    // 计算 fact[n_max] 的逆元，然后倒推计算其他逆元
    // 根据费马小定理，a^(P-2) 是 a 在模 P 意义下的逆元
    inv_fact[n_max] = power(fact[n_max], MOD - 2, MOD);

    for (int i = n_max - 1; i >= 1; --i) {
        inv_fact[i] = (inv_fact[i+1] * (i + 1)) % MOD;
    }
}

// 计算 C(n, k) % MOD
long long calculate_c_nk_mod(int n, int k) {
    if (k < 0 || k > n) {
        return 0;
    }
    // C(n, k) = n! * (k!)^(-1) * ((n-k)!)^(-1) (mod P)
    long long numerator = fact[n];
    long long denominator_k = inv_fact[k];
    long long denominator_nk = inv_fact[n-k];
    
    return (((numerator * denominator_k) % MOD) * denominator_nk) % MOD;
}

int main() {
    // 预处理到 N_MAX-1
    init_combinations(N_MAX - 1); 

    int n = 100000, k = 50000;
    cout << "C(" << n << ", " << k << ") % " << MOD << " = " << calculate_c_nk_mod(n, k) << endl;

    n = 200000, k = 3;
    cout << "C(" << n << ", " << k << ") % " << MOD << " = " << calculate_c_nk_mod(n, k) << endl;

    return 0;
}
```

### 练习题目答案与解析

**选择题或填空题答案：**

1.  **B. 32**
    *   **解析：** 杨辉三角第 n 行（从第 0 行开始）的数字之和是 2^n。所以第 5 行的数字之和是 2^5 = 32。

2.  **B. 35**
    *   **解析：** C(7, 3) = 7! / (3! * (7-3)!) = 7! / (3! * 4!) = (7 * 6 * 5) / (3 * 2 * 1) = 35。

3.  **A. C(5, 3) + C(5, 4)**
    *   **解析：** 根据帕斯卡恒等式 C(n, k) = C(n-1, k-1) + C(n-1, k)，将 n=6, k=4 代入，得到 C(6, 4) = C(6-1, 4-1) + C(6-1, 4) = C(5, 3) + C(5, 4)。

4.  **B. O(k) 单次计算法**
    *   **解析：**
        *   A. 动态规划递推法需要一个 N*N 的二维数组，当 N=10^9 时内存无法承受。
        *   C. 预处理阶乘和逆元需要预处理到 N，同样当 N=10^9 时无法承受。
        *   D. 直接计算阶乘 10^9! 会导致数值过大，超出任何基本数据类型的表示范围，无法直接计算。
        *   B. O(k) 单次计算法 C(n, k) = (n * (n-1) * ... * (n-k+1)) / (k * (k-1) * ... * 1) 在 k 很小的情况下，乘法次数和除法次数都是 k 次，中间结果虽然可能很大，但通常 `long long` 可以承受，并且由于最终结果一定是整数，除法可以保证整除。

5.  **C. 质数**
    *   **解析：** 费马小定理 `a^(P-1) ≡ 1 (mod P)` 及其推论 `a^(P-2) ≡ a^(-1) (mod P)` 仅在 P 为质数时成立。
