# Level 8：GESP 八级 (综合巅峰)

## 教学目标
1. 掌握图论基础与常用算法。
2. 深入理解动态规划与搜索优化。
3. 具备综合运用算法解决复杂问题的能力。

## 核心知识点
- 图：图的存储（邻接矩阵/邻接表），图的遍历。
- 最短路径：Dijkstra 算法, Floyd 算法。
- 进阶DP：背包问题（0/1背包、完全背包），最长公共子序列 (LCS)。
- 综合能力：模拟大作业，复杂逻辑实现。
