## 知识点讲解  
1. “已知→未知”  
   - 已知：数组的前缀和可以 O(1) 查询前缀，但区间加减仍需两次前缀和。  
   - 未知：如果要支持动态更新，前缀和重算成本也很高。  
2. 数据结构原理  
   - 将数组分成左右两半，递归建立二叉树节点，每个节点存储对应区间的信息（如区间和）。  
   - 使用大小约为 4*n 的数组或 `vector` 存储树节点，节点编号遵循二叉堆性质：`node`，`2*node` 左子，`2*node+1` 右子。  
3. 操作流程  
   - 建树 build(node, l, r)：当 l==r 时为叶节点，否则递归构建并合并子节点数据。  
   - 查询 query(node, l, r, ql, qr)：完全覆盖、部分覆盖、无覆盖三种情况分别处理。  
   - 更新 update(node, l, r, idx, val)：定位到叶子并修改，再沿递归栈向上更新父节点。  
4. 复杂度  
   - 建树：O(n)  
   - 每次查询/更新：O(log n)  
5. 拓展  
   - 可改为区间更新 + 区间查询（懒标记）。

## 代码示例  
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 100000;
int n;
vector<int> a;
vector<int> tree;

// 建树：将 a[l..r] 建到 tree[node]
void build
(
int node,
int l,
int r
)
{
    if (l == r)
    {
        tree[node] = a[l];
        return;
    }
    int mid = (l + r) / 2;
    build(2 * node, l, mid);
    build(2 * node + 1, mid + 1, r);
    tree[node] = tree[2 * node] + tree[2 * node + 1];
}

// 区间求和：查询 a[ql..qr] 的和
int query
(
int node,
int l,
int r,
int ql,
int qr
)
{
    if (qr < l || r < ql)
    {
        return 0; // 无覆盖
    }
    if (ql <= l && r <= qr)
    {
        return tree[node]; // 完全覆盖
    }
    int mid = (l + r) / 2;
    int leftSum = query(2 * node, l, mid, ql, qr);
    int rightSum = query(2 * node + 1, mid + 1, r, ql, qr);
    return leftSum + rightSum;
}

// 单点更新：将 a[idx] 增加为 val
void update
(
int node,
int l,
int r,
int idx,
int val
)
{
    if (l == r)
    {
        tree[node] = val;
        return;
    }
    int mid = (l + r) / 2;
    if (idx <= mid)
    {
        update(2 * node, l, mid, idx, val);
    }
    else
    {
        update(2 * node + 1, mid + 1, r, idx, val);
    }
    tree[node] = tree[2 * node] + tree[2 * node + 1];
}

int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> n;
    a.resize(n);
    for (int i = 0; i < n; i++)
    {
        cin >> a[i];
    }
    tree.assign(4 * n, 0);
    build(1, 0, n - 1);

    int q;
    cin >> q;
    while (q--)
    {
        int type, x, y;
        cin >> type >> x >> y;
        if (type == 1)
        {
            // 更新操作
            update(1, 0, n - 1, x, y);
        }
        else
        {
            // 查询操作
            cout << query(1, 0, n - 1, x, y) << "\n";
        }
    }
    return 0;
}
```

## 练习题目  
1. 选择题/填空题  
   1) 若数组长为 n，在最坏情况下，线段树节点数组大小应至少是 ______ * n。  
   2) 在查询区间 [l, r] 时，如果当前节点区间完全无交集，应返回的“中立值”是 ______（用于区间和）。  
   3) 对于单点更新，线段树的时间复杂度是 O(   )。  
   4) 若要改为区间最小值查询，需要将“加和”操作改为 ______ 操作。  
   5) 下面关于线段树描述错误的是（   ）  
      A. 建树时间 O(n)  
      B. 查询/更新时间 O(log n)  
      C. 只适合静态区间查询  
      D. 可以维护区间最大/最小/和等多种信息  

---  

答案与解析  
1) 4；解析：线段树大小约 4n 最安全。  
2) 0；解析：区间和的中立元为 0。  
3) O(log n)；解析：更新沿树高递归。  
4) min；解析：将加法合并改为取最小值合并。  
5) C；解析：线段树可以支持动态更新，不限于静态。
