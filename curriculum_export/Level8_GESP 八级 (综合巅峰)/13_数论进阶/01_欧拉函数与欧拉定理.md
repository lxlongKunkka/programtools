## 趣味引入

同学们，还记得我们上次讲的“快速幂”吗？它可以帮我们计算像 `2^100 % 7` 这样的大指数幂取模。我们还学了一个神奇的费马小定理，它告诉我们：如果 `p` 是一个素数，`a` 不是 `p` 的倍数，那么 `a^(p-1) % p` 结果总是 `1`！这多酷啊，指数可以大大缩小！

比如，算 `2^100 % 7`。因为 `7` 是素数，`2` 不是 `7` 的倍数，根据费马小定理，`2^(7-1) % 7 = 2^6 % 7 = 1`。那么 `2^100 % 7` 就可以变成 `2^(100 % 6) % 7 = 2^4 % 7 = 16 % 7 = 2`。是不是很方便？

别担心！今天，我们就来认识两位数论界的“超级英雄”：**欧拉函数** 和 **欧拉定理**！它们能把费马小定理推广到任何模数 `m` 的情况，让我们的指数缩减魔法再次生效！准备好了吗？让我们一起揭开它们的神秘面纱！

## 深度知识点讲解

### 1. 欧拉函数 (Euler's Totient Function, φ(n))

#### 1.1 概念本质：互质的“小伙伴”

欧拉函数，通常用希腊字母 `φ` (读作 phi，音：fài) 表示，所以也叫 `phi` 函数。

**定义：** 对于一个正整数 `n`，欧拉函数 `φ(n)` 表示的是 **小于等于 `n` 且与 `n` 互质的正整数的个数**。

**生活类比：**
想象一下，你有一个秘密宝盒，上面有一个数字 `n`。只有能与 `n` **互质** 的钥匙才能打开它。那么 `φ(n)` 就是告诉你，在这个世界上，有多少把小于等于 `n` 的钥匙能打开你的宝盒。
**互质** 的意思就是它们除了 `1` 以外没有其他的公因数。

**举个例子：**
*   **φ(1)：** 小于等于 1 且与 1 互质的正整数是 `1`。所以 `φ(1) = 1`。
*   **φ(6)：** 小于等于 6 的正整数有 `1, 2, 3, 4, 5, 6`。
    *   `1` 与 `6` 互质 (`gcd(1, 6) = 1`)
    *   `2` 与 `6` 不互质 (`gcd(2, 6) = 2`)
    *   `3` 与 `6` 不互质 (`gcd(3, 6) = 3`)
    *   `4` 与 `6` 不互质 (`gcd(4, 6) = 2`)
    *   `5` 与 `6` 互质 (`gcd(5, 6) = 1`)
    *   `6` 与 `6` 不互质 (`gcd(6, 6) = 6`)
    所以，与 `6` 互质的数是 `1, 5`。`φ(6) = 2`。

#### 1.2 欧拉函数的计算方法

计算 `φ(n)` 有一些规律和公式，比一个一个数要高效得多！

**情况一：当 `n` 是一个素数 `p` 时**
*   `φ(p) = p - 1`
*   **为什么？** 因为素数 `p` 除了 `1` 和它本身，没有其他因数。所以，小于 `p` 的所有正整数 `1, 2, ..., p-1` 都与 `p` 互质。而 `p` 本身不与 `p` 互质。所以总共有 `p-1` 个。
*   **例子：** `φ(7) = 7 - 1 = 6` (与 7 互质的有 `1, 2, 3, 4, 5, 6`)。

**情况二：当 `n` 是一个素数的幂 `p^k` 时**
*   `φ(p^k) = p^k - p^(k-1)`
*   **为什么？** 小于等于 `p^k` 的数中，只有 `p` 的倍数才可能不与 `p^k` 互质。这些数是 `p, 2p, 3p, ..., (p^(k-1))p`。总共有 `p^(k-1)` 个。把这些数从总数 `p^k` 中减去，剩下的就是互质的了。
*   **例子：** `φ(8) = φ(2^3) = 2^3 - 2^(3-1) = 8 - 4 = 4` (与 8 互质的有 `1, 3, 5, 7`)。

**情况三：欧拉函数是积性函数**
*   如果 `a` 和 `b` 互质（即 `gcd(a, b) = 1`），那么 `φ(a * b) = φ(a) * φ(b)`。
*   **为什么？** 这个性质比较深刻，涉及到中国剩余定理和抽象代数，我们在这里可以先记住并理解它的应用。它告诉我们，如果一个数能分解成互质的两个因数，那么它的欧拉函数值也能分解。
*   **例子：** `φ(6) = φ(2 * 3)`。因为 `gcd(2, 3) = 1`，所以 `φ(6) = φ(2) * φ(3)`。
    *   `φ(2) = 2 - 1 = 1`
    *   `φ(3) = 3 - 1 = 2`
    *   所以 `φ(6) = 1 * 2 = 2`。这与我们上面枚举的结果一致！

**情况四：通用公式 (通过质因数分解)**
任何一个正整数 `n` 都可以被唯一地分解成若干个素数的幂的乘积：
`n = p1^a1 * p2^a2 * ... * pk^ak`
其中 `p1, p2, ..., pk` 是不同的素数，`a1, a2, ..., ak` 是正整数。

利用积性函数性质和素数幂的公式，我们可以推导出通用公式：
`φ(n) = φ(p1^a1) * φ(p2^a2) * ... * φ(pk^ak)`
`φ(n) = (p1^a1 - p1^(a1-1)) * (p2^a2 - p2^(a2-1)) * ... * (pk^ak - pk^(ak-1))`
`φ(n) = p1^a1 * (1 - 1/p1) * p2^a2 * (1 - 1/p2) * ... * pk^ak * (1 - 1/pk)`
`φ(n) = (p1^a1 * p2^a2 * ... * pk^ak) * (1 - 1/p1) * (1 - 1/p2) * ... * (1 - 1/pk)`
`φ(n) = n * (1 - 1/p1) * (1 - 1/p2) * ... * (1 - 1/pk)`

*   **推导思路：** 这个公式的本质是**容斥原理**。从总数 `n` 中减去 `p1` 的倍数，再减去 `p2` 的倍数，...，然后加上 `p1p2` 的倍数，等等。最终可以化简成这个漂亮的乘积形式。
*   **例子：** `φ(12)`
    *   `12 = 2^2 * 3^1`
    *   `p1 = 2, a1 = 2`
    *   `p2 = 3, a2 = 1`
    *   `φ(12) = 12 * (1 - 1/2) * (1 - 1/3)`
    *   `φ(12) = 12 * (1/2) * (2/3)`
    *   `φ(12) = 12 * (1/3) = 4`
    *   验证：小于等于 12 且与 12 互质的数有 `1, 5, 7, 11`。确实是 4 个！

**如何编程计算 `φ(n)`？**
最常用的方法是利用通用公式，通过对 `n` 进行质因数分解来计算。

```cpp
// 伪代码思路
int calculatePhi(int n) {
    int ans = n; // 初始化为 n
    // 遍历所有可能的质因子
    for (int i = 2; i * i <= n; ++i) { // 只需要遍历到 sqrt(n)
        if (n % i == 0) { // 如果 i 是 n 的一个质因子
            ans = ans / i * (i - 1); // 等价于 ans = ans * (1 - 1/i)
                                     // 避免浮点数，先除再乘
            while (n % i == 0) { // 把 n 中所有的 i 因子都除掉
                n /= i;
            }
        }
    }
    if (n > 1) { // 如果循环结束后 n 仍然大于 1，说明 n 自身是一个素数（或剩下一个大的素因子）
        ans = ans / n * (n - 1); // 对这个剩余的素因子进行处理
    }
    return ans;
}
```

#### 1.3 常见误区

1.  **误以为 `φ(n) = n - 1`：** 只有当 `n` 是素数时才成立。对于合数，`φ(n)` 会小于 `n-1`。
2.  **忘记 `n > 1` 的情况：** 在质因数分解循环结束后，如果 `n` 仍然大于 1，说明它是一个素数因子（例如 `n=10`，`i=2` 处理后 `n=5`，`5` 还是素数因子），需要额外处理。
3.  **计算 `ans = ans * (1 - 1/i)` 时的浮点数问题：** 应该写成 `ans = ans / i * (i - 1)` 来避免浮点数，因为 `i` 一定是 `ans` 的因子（因为 `ans` 最初是 `n`，而 `i` 是 `n` 的因子）。

### 2. 欧拉定理 (Euler's Theorem)

现在，我们有了欧拉函数这个强大的工具，就可以揭示欧拉定理的秘密了！

#### 2.1 定理内容

**欧拉定理：**
如果 `a` 和 `m` 是两个正整数，并且它们互质（即 `gcd(a, m) = 1`），那么：
`a^φ(m) ≡ 1 (mod m)`

**与费马小定理的关系：**
费马小定理 `a^(p-1) ≡ 1 (mod p)` 是欧拉定理的一个特例。
当 `m` 是素数 `p` 时，`φ(p) = p - 1`。所以欧拉定理就变成了 `a^(p-1) ≡ 1 (mod p)`，这正是费马小定理！

**应用：模幂的降幂**
当我们需要计算 `a^b % m`，并且 `gcd(a, m) = 1` 时，如果 `b` 非常大，我们可以利用欧拉定理将指数 `b` 降幂：
`a^b ≡ a^(b % φ(m)) (mod m)`

**为什么可以这样降幂？**
假设 `b = q * φ(m) + r`，其中 `r = b % φ(m)`。
那么 `a^b = a^(q * φ(m) + r) = (a^φ(m))^q * a^r`。
根据欧拉定理，`a^φ(m) ≡ 1 (mod m)`。
所以 `(a^φ(m))^q * a^r ≡ 1^q * a^r ≡ a^r (mod m)`。
即 `a^b ≡ a^(b % φ(m)) (mod m)`。

#### 2.2 证明思路 (简述)

1.  **构造集合：** 考虑一个与 `m` 互质的完全剩余系 `S = {x1, x2, ..., x_φ(m)}`，这些 `x_i` 都是小于 `m` 且与 `m` 互质的数。
2.  **乘以 `a`：** 构造新集合 `aS = {a*x1, a*x2, ..., a*x_φ(m)}`。
3.  **关键性质：**
    *   因为 `gcd(a, m) = 1` 且 `gcd(xi, m) = 1`，所以 `gcd(a*xi, m) = 1`。这意味着 `aS` 中的所有元素也与 `m` 互质。
    *   `aS` 中的所有元素在模 `m` 意义下是两两不同的。
    *   所以 `aS` 在模 `m` 意义下，是 `S` 的一个重新排列。
4.  **乘积相等：**
    `x1 * x2 * ... * x_φ(m) ≡ (a*x1) * (a*x2) * ... * (a*x_φ(m)) (mod m)`
    `product(xi) ≡ a^φ(m) * product(xi) (mod m)`
5.  **约分：** 因为 `product(xi)` 与 `m` 互质（每个 `xi` 都与 `m` 互质），所以我们可以将 `product(xi)` 从两边约去。
    `1 ≡ a^φ(m) (mod m)`
    证毕！

#### 2.3 扩展欧拉定理 (Extended Euler's Theorem)

欧拉定理有一个限制：`gcd(a, m) = 1`。如果 `a` 和 `m` 不互质怎么办？
例如 `2^10 % 6`，这里 `gcd(2, 6) = 2 ≠ 1`。欧拉定理不能直接用。
这时就需要 **扩展欧拉定理** 出场了！

**定理内容：**
对于任意正整数 `a, m` 和 `b`：
1.  如果 `gcd(a, m) = 1`，则 `a^b ≡ a^(b % φ(m)) (mod m)` (这就是普通欧拉定理)
2.  如果 `gcd(a, m) ≠ 1`，但 `b ≥ φ(m)`，则 `a^b ≡ a^(b % φ(m) + φ(m)) (mod m)`
3.  如果 `gcd(a, m) ≠ 1`，且 `b < φ(m)`，则 `a^b ≡ a^b (mod m)` (指数不能降，直接计算)

**关键点：`b % φ(m) + φ(m)`**
为什么当 `b ≥ φ(m)` 时，指数要加上 `φ(m)`？
这是为了确保指数 `b'` 足够大，使得 `a^(b')` 在模 `m` 意义下进入循环周期。
*   当 `gcd(a, m) = 1` 时，`a` 在模 `m` 意义下的幂次是周期性的，周期长度是 `φ(m)` 的因子。`φ(m)` 是最小的使得 `a^φ(m) ≡ 1 (mod m)` 成立的指数。
*   当 `gcd(a, m) ≠ 1` 时，`a` 的幂次序列 `a^1, a^2, a^3, ...` 在模 `m` 意义下，会先经过一段“非周期”的序列，然后才进入一个循环。
    例如 `2^1 % 6 = 2`, `2^2 % 6 = 4`, `2^3 % 6 = 2`, `2^4 % 6 = 4`, ...
    序列是 `2, 4, 2, 4, ...`。
    `φ(6) = 2`。
    如果 `b = 1`，`b < φ(6)`，直接 `2^1 % 6 = 2`。
    如果 `b = 2`，`b = φ(6)`，`2^(2 % 2 + 2) % 6 = 2^(0 + 2) % 6 = 2^2 % 6 = 4`。
    如果 `b = 3`，`b > φ(6)`，`2^(3 % 2 + 2) % 6 = 2^(1 + 2) % 6 = 2^3 % 6 = 8 % 6 = 2`。
    可以看到，`+ φ(m)` 的作用是把指数“拉”到足以进入循环的部分。

**总结一下：**
计算 `a^b % m` 时，步骤是：
1.  计算 `φ(m)`。
2.  比较 `b` 和 `φ(m)`。
    *   如果 `b < φ(m)`：直接计算 `a^b % m`。
    *   如果 `b ≥ φ(m)`：计算 `b' = b % φ(m) + φ(m)`，然后计算 `a^(b') % m`。

这个 `b'` 确保了指数足够大，使得 `a` 的幂次已经进入了循环节。

### 3. 快速幂 (Modular Exponentiation) 优化

无论是计算 `a^b % m` 还是 `a^(b % φ(m)) % m`，我们都需要使用快速幂算法。
快速幂的核心思想是利用二进制分解指数，将大指数幂运算转化为一系列乘法和平方运算。

```cpp
// 快速幂函数
long long power(long long b, long long e, long long m) {
    long long res = 1;
    b %= m; // 模数可能比底数小，先取模
    while (e > 0) {
        if (e % 2 == 1) { // 如果指数是奇数，乘以 b
            res = (res * b) % m;
        }
        b = (b * b) % m; // 底数平方
        e /= 2; // 指数减半
    }
    return res;
}
```

---

### 4. 整体流程

计算 `a^b % m`：
1.  计算 `φ(m)`。
    *   使用质因数分解法。
2.  处理指数 `b`。
    *   如果 `b` 字符串形式，需要比较长度判断是否 `b >= φ(m)`。
    *   如果 `b` 是普通整数，直接比较 `b` 和 `φ(m)`。
    *   设新的指数为 `e_new`。
    *   如果 `b < φ(m)`：`e_new = b`。
    *   如果 `b ≥ φ(m)`：`e_new = b % φ(m) + φ(m)`。
3.  使用快速幂计算 `power(a, e_new, m)`。

**思考：** 如果 `b` 是一个非常大的数，例如用字符串表示的 `10^100000`，我们怎么计算 `b % φ(m)` 和判断 `b >= φ(m)` 呢？
*   **判断 `b >= φ(m)`：** 可以将 `b` 的字符串长度与 `φ(m)` 的位数进行比较。如果 `b` 的位数多于 `φ(m)`，则 `b >= φ(m)`。如果位数相同，则逐位比较。
*   **计算 `b % φ(m)`：** 可以用字符串模拟大数取模运算。

```cpp
// 大数 b 字符串对 m 取模
long long stringMod(string s, long long m) {
    long long res = 0;
    for (char c : s) { // GESP 8 允许 C++11 范围for
        res = (res * 10 + (c - '0')) % m;
    }
    return res;
}

// 检查大数 b 字符串是否大于等于 m
bool checkGreaterOrEqual(string s, long long m) {
    string m_str = to_string(m); // GESP 8 允许 to_string
    if (s.length() > m_str.length()) return true;
    if (s.length() < m_str.length()) return false;
    return s >= m_str; // 字符串直接比较大小
}
```

这部分是 GESP 8 的难点，需要学生掌握大数处理。

### 例题 1：计算欧拉函数值

**题目：** 计算 `φ(100)` 和 `φ(720)`。

**思路分析：**
1.  **质因数分解：** 先将 `n` 分解为质因数的乘积。
2.  **应用公式：** 使用通用公式 `φ(n) = n * (1 - 1/p1) * (1 - 1/p2) * ...` 或者 `φ(n) = φ(p1^a1) * φ(p2^a2) * ...`。

**解答：**

**计算 `φ(100)`：**
1.  质因数分解 `100 = 2^2 * 5^2`。
2.  应用公式 `φ(n) = n * (1 - 1/p1) * (1 - 1/p2)`：
    `φ(100) = 100 * (1 - 1/2) * (1 - 1/5)`
    `φ(100) = 100 * (1/2) * (4/5)`
    `φ(100) = 100 * (4/10)`
    `φ(100) = 40`

**计算 `φ(720)`：**
1.  质因数分解 `720`：
    `720 = 72 * 10`
    `= (8 * 9) * (2 * 5)`
    `= (2^3 * 3^2) * (2 * 5)`
    `= 2^4 * 3^2 * 5^1`
2.  应用公式 `φ(n) = n * (1 - 1/p1) * (1 - 1/p2) * ...`：
    `φ(720) = 720 * (1 - 1/2) * (1 - 1/3) * (1 - 1/5)`
    `φ(720) = 720 * (1/2) * (2/3) * (4/5)`
    `φ(720) = 720 * (8 / 30)`
    `φ(720) = 720 * (4 / 15)`
    `φ(720) = (720 / 15) * 4`
    `φ(720) = 48 * 4`
    `φ(720) = 192`

**复杂度：**
质因数分解的时间复杂度是 `O(sqrt(n))`。对于每个因子，进行常数次乘除运算。所以计算单个 `φ(n)` 的时间复杂度是 `O(sqrt(n))`。

### 例题 2：欧拉定理应用 (模幂)

**题目：** 计算 `2^100 % 77`。

**思路分析：**
1.  **判断互质：** `gcd(2, 77) = 1`，因此可以使用欧拉定理。
2.  **计算 `φ(77)`：** `77 = 7 * 11`。`gcd(7, 11) = 1`。
    `φ(77) = φ(7) * φ(11) = (7-1) * (11-1) = 6 * 10 = 60`。
3.  **降幂：** 根据欧拉定理 `a^b ≡ a^(b % φ(m)) (mod m)`。
    `2^100 % 77 ≡ 2^(100 % 60) % 77`
    `100 % 60 = 40`。
    所以，`2^100 % 77 ≡ 2^40 % 77`。
4.  **快速幂计算：** 使用快速幂计算 `2^40 % 77`。

**解答：**
1.  `gcd(2, 77) = 1`，满足欧拉定理条件。
2.  `φ(77) = φ(7 * 11) = φ(7) * φ(11) = (7-1) * (11-1) = 6 * 10 = 60`。
3.  根据欧拉定理，`2^100 % 77 ≡ 2^(100 % 60) % 77`。
    `100 % 60 = 40`。
    所以，原式等价于计算 `2^40 % 77`。
4.  使用快速幂计算 `2^40 % 77`：
    *   `2^1 % 77 = 2`
    *   `2^2 % 77 = 4`
    *   `2^4 % 77 = 16`
    *   `2^8 % 77 = 16^2 % 77 = 256 % 77 = 256 - 3 * 77 = 256 - 231 = 25`
    *   `2^16 % 77 = 25^2 % 77 = 625 % 77 = 625 - 8 * 77 = 625 - 616 = 9`
    *   `2^32 % 77 = 9^2 % 77 = 81 % 77 = 4`
    `2^40 = 2^(32 + 8) = 2^32 * 2^8`
    `2^40 % 77 = (2^32 % 77 * 2^8 % 77) % 77`
    `= (4 * 25) % 77`
    `= 100 % 77`
    `= 23`

最终结果是 `23`。

**复杂度：**
计算 `φ(m)` 的时间复杂度是 `O(sqrt(m))`。
快速幂的时间复杂度是 `O(log(exp))`。
总复杂度 `O(sqrt(m) + log(exp))`。

### 例题 3：扩展欧拉定理应用 (模幂)

**题目：** 计算 `2^100 % 6`。

**思路分析：**
1.  **判断互质：** `gcd(2, 6) = 2 ≠ 1`。不能直接使用欧拉定理。需要使用扩展欧拉定理。
2.  **计算 `φ(6)`：** `6 = 2 * 3`。
    `φ(6) = φ(2) * φ(3) = (2-1) * (3-1) = 1 * 2 = 2`。
3.  **判断指数 `b` 与 `φ(m)` 的关系：**
    `b = 100`，`φ(6) = 2`。
    `b = 100 ≥ φ(6) = 2`。
    所以，使用扩展欧拉定理的第二种情况：`a^b ≡ a^(b % φ(m) + φ(m)) (mod m)`。
4.  **降幂：**
    `b' = 100 % φ(6) + φ(6)`
    `b' = 100 % 2 + 2`
    `b' = 0 + 2`
    `b' = 2`
    所以，`2^100 % 6 ≡ 2^2 % 6`。
5.  **计算：** `2^2 % 6 = 4 % 6 = 4`。

**解答：**
1.  `gcd(2, 6) = 2 ≠ 1`，所以使用扩展欧拉定理。
2.  计算 `φ(6) = φ(2 * 3) = φ(2) * φ(3) = 1 * 2 = 2`。
3.  指数 `b = 100`，`φ(6) = 2`。因为 `100 ≥ 2`，所以指数降为 `100 % φ(6) + φ(6)`。
    `e_new = 100 % 2 + 2 = 0 + 2 = 2`。
4.  原式等价于计算 `2^2 % 6`。
5.  `2^2 % 6 = 4 % 6 = 4`。

最终结果是 `4`。

**复杂度：** 同上，`O(sqrt(m) + log(exp))`。

### 例题 4：大指数与扩展欧拉定理

**题目：** 计算 `7^B % 10`，其中 `B` 是一个非常大的整数，例如 `B = "123456789123456789"`。

**思路分析：**
1.  **判断互质：** `gcd(7, 10) = 1`。可以使用普通欧拉定理。
2.  **计算 `φ(10)`：** `10 = 2 * 5`。
    `φ(10) = φ(2) * φ(5) = (2-1) * (5-1) = 1 * 4 = 4`。
3.  **处理大指数 `B`：** `B` 是字符串形式，我们需要计算 `B % φ(10)`，并判断 `B` 是否大于等于 `φ(10)`。
    *   `φ(10) = 4`。
    *   `B` 的位数明显比 `4` 多，所以 `B >= φ(10)` 成立。
    *   计算 `B % 4`。
4.  **降幂：** `7^B % 10 ≡ 7^(B % 4) % 10`。
5.  **快速幂计算。**

**解答：**
1.  `gcd(7, 10) = 1`，使用欧拉定理。
2.  `φ(10) = φ(2 * 5) = φ(2) * φ(5) = 1 * 4 = 4`。
3.  `B = "123456789123456789"`。
    `B` 的位数远大于 `φ(10) = 4` 的位数，所以 `B >= φ(10)`。
    计算 `B % 4`：一个数模 4 的结果只取决于其最后两位。`89 % 4 = 1`。
    所以 `B % 4 = 1`。
4.  根据欧拉定理，`7^B % 10 ≡ 7^(B % 4) % 10`。
    `≡ 7^1 % 10`
    `≡ 7 % 10`
    `= 7`

最终结果是 `7`。

**复杂度：**
计算 `φ(m)` 的时间复杂度是 `O(sqrt(m))`。
大数取模的时间复杂度是 `O(len(B))` (B 的位数)。
快速幂的时间复杂度是 `O(log(φ(m)))`。
总复杂度 `O(sqrt(m) + len(B))`。

## 代码实现模板

```cpp
#include <bits/stdc++.h> // GESP 推荐使用这个头文件

// 使用 long long 避免中间计算溢出
typedef long long ll;

// 1. 计算欧拉函数 φ(n)
ll calculatePhi(ll n) {
    ll res = n;
    for (ll i = 2; i * i <= n; ++i) {
        if (n % i == 0) {
            res = res / i * (i - 1); // 等价于 res *= (1 - 1.0/i)
            while (n % i == 0) {
                n /= i;
            }
        }
    }
    if (n > 1) { // 处理剩余的素因子（如果 n 最终是一个素数）
        res = res / n * (n - 1);
    }
    return res;
}

// 2. 快速幂 (a^e % m)
ll power(ll b, ll e, ll m) {
    ll res = 1;
    b %= m; // 底数先对模数取模，防止过大
    while (e > 0) {
        if (e % 2 == 1) { // 如果指数是奇数
            res = (res * b) % m;
        }
        b = (b * b) % m; // 底数平方
        e /= 2;          // 指数减半
    }
    return res;
}

// 3. 大整数字符串对小整数取模 (b % mod)
// 同时判断 b 是否大于等于 mod
// isGreaterOrEqual 标记如果大数 b 实际大于等于 mod
ll stringMod(std::string s, ll m, bool& isGreaterOrEqual) {
    ll res = 0;
    std::string m_str = std::to_string(m); // 将模数转为字符串方便比较

    // 初始假设 b < m
    isGreaterOrEqual = false;

    if (s.length() > m_str.length()) {
        isGreaterOrEqual = true;
    } else if (s.length() == m_str.length()) {
        if (s >= m_str) { // 字符串可以直接比较大小
            isGreaterOrEqual = true;
        }
    }

    // 计算 b % m
    for (char c : s) {
        res = (res * 10 + (c - '0')) % m;
    }
    return res;
}


int main() {
    // 示例：计算 a^b % m
    ll a, m;
    std::string b_str; // b 可能非常大，用字符串表示

    // 假设输入 a, b_str, m
    // std::cin >> a >> b_str >> m;
    a = 2;
    b_str = "100";
    m = 77;
    // a = 2, b_str = "100", m = 6; // 扩展欧拉定理示例
    // a = 7, b_str = "123456789123456789", m = 10; // 大指数示例

    ll phi_m = calculatePhi(m); // 计算欧拉函数 φ(m)

    bool b_is_ge_phi_m; // 标记 b 是否大于等于 φ(m)
    ll b_mod_phi_m = stringMod(b_str, phi_m, b_is_ge_phi_m); // 计算 b % φ(m)

    ll final_exponent; // 最终用于快速幂的指数

    // 根据扩展欧拉定理选择指数
    if (b_is_ge_phi_m) { // 如果 b >= φ(m)
        final_exponent = b_mod_phi_m + phi_m; // 指数 = (b % φ(m)) + φ(m)
    } else { // 如果 b < φ(m)
        // 此时 b_str 对应的数值就是 b_mod_phi_m (因为 b < φ(m))
        // 但为了严谨和统一，可以再用一次 stringMod，或者直接转成ll
        // GESP 8 允许 std::stoll 进行字符串到长整型转换
        final_exponent = std::stoll(b_str); // 此时指数就是 b 本身
    }

    // 计算最终结果
    ll ans = power(a, final_exponent, m);

    std::cout << "a = " << a << ", b = " << b_str << ", m = " << m << std::endl;
    std::cout << "phi(" << m << ") = " << phi_m << std::endl;
    std::cout << "Final exponent used: " << final_exponent << std::endl;
    std::cout << "Result: " << ans << std::endl;

    return 0;
}
```

**代码复杂度分析：**
*   `calculatePhi(m)`: `O(sqrt(m))`
*   `stringMod(b_str, phi_m, ...)`: `O(len(b_str))` (b 的位数)
*   `power(a, final_exponent, m)`: `O(log(final_exponent))`。`final_exponent` 的最大值约为 `2 * phi_m`，所以是 `O(log(phi_m))`。
*   **总时间复杂度：** `O(sqrt(m) + len(b_str) + log(phi_m))`。这对于 GESP 八级题目中的 `m` 通常在 `10^9` 级别，`b_str` 长度在 `10^5` 级别，是高效可行的。
