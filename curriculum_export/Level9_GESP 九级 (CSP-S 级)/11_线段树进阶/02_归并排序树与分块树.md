# 归并排序树与分块树

## 教学目标

知识目标  
- 理解归并排序树（Merge Sort Tree）和分块树（Sqrt-Decomposition）的基本结构与原理  
- 掌握两种数据结构的查询流程：区间第k小、区间内不大于给定值的元素个数  

能力目标  
- 能够基于线段树框架实现归并排序树  
- 能够基于分块思想实现分块树，并完成区间查询  
- 学会对比两种结构的适用场景与时间复杂度  

素养目标  
- 培养算法设计与分析能力，增强对复杂度的敏感性  
- 培养动手实现数据结构的能力，体验算法工程化

## 知识点讲解

1. 线段树复习  
   - 区间求和、区间最值等基础应用  
2. 归并排序树（Merge Sort Tree）  
   - 结构：在线段树每个节点存储对应区间的数据并保持排序  
   - 建树：自底向上合并左右子区间的有序数组 → 时间 O(n log n)  
   - 查询：在树上分裂区间，用二分统计不大于 x 的个数 → 单次 O(log n · log n)  
3. 分块树（Sqrt-Decomposition）  
   - 思想：将数组分成大小为 B 的若干块，每块内部保持有序  
   - 预处理：对每块排序 → O(n log B)  
   - 查询：跨块使用二分搜索，块内暴力或二分 → O(√n · log n)  
4. 复杂度对比与适用场景  
   - 归并排序树：n, q 均大时更优  
   - 分块树：n 中等，q 多时实现简单且常数低

## 代码示例

```cpp
#include <bits/stdc++.h>
using namespace std;

// 全局数组与归并排序树结构
int n;
vector<int> a;
vector<vector<int>> seg;

// 构建归并排序树
void build(int idx, int l, int r)
{
    if (l == r)
    {
        seg[idx] = { a[l] };
        return;
    }
    int mid = (l + r) >> 1;
    int lc = idx << 1, rc = lc | 1;
    build(lc, l, mid);
    build(rc, mid + 1, r);
    // 合并排序
    seg[idx].resize(r - l + 1);
    merge(seg[lc].begin(), seg[lc].end(),
          seg[rc].begin(), seg[rc].end(),
          seg[idx].begin());
}

// 区间查询：统计区间 [L,R] 中 ≤ x 的个数
int query(int idx, int l, int r, int L, int R, int x)
{
    if (r < L || R < l) return 0;
    if (L <= l && r <= R)
    {
        // 二分查找上界
        return int(upper_bound(seg[idx].begin(), seg[idx].end(), x)
                   - seg[idx].begin());
    }
    int mid = (l + r) >> 1;
    return query(idx << 1, l, mid, L, R, x)
         + query(idx << 1 | 1, mid + 1, r, L, R, x);
}

// 分块树全局
int B, nb;
vector<vector<int>> blk;
vector<int> bel; // bel[i] 表示 a[i] 属于第几块

// 构建分块树
void buildBlock()
{
    B = int(sqrt(n)) + 1;
    nb = (n + B - 1) / B;
    blk.assign(nb, vector<int>());
    bel.assign(n, 0);
    for (int i = 0; i < n; i++)
    {
        int id = i / B;
        bel[i] = id;
        blk[id].push_back(a[i]);
    }
    for (int i = 0; i < nb; i++)
    {
        sort(blk[i].begin(), blk[i].end());
    }
}

// 分块查询：统计 [L,R] 中 ≤ x 的个数
int queryBlock(int L, int R, int x)
{
    int res = 0;
    int bL = bel[L], bR = bel[R];
    if (bL == bR)
    {
        for (int i = L; i <= R; i++)
        {
            if (a[i] <= x) res++;
        }
        return res;
    }
    // 左侧残块
    int endL = (bL + 1) * B - 1;
    for (int i = L; i <= min(endL, n - 1); i++)
    {
        if (a[i] <= x) res++;
    }
    // 完整块
    for (int b = bL + 1; b < bR; b++)
    {
        res += int(upper_bound(blk[b].begin(), blk[b].end(), x)
                   - blk[b].begin());
    }
    // 右侧残块
    int startR = bR * B;
    for (int i = startR; i <= R; i++)
    {
        if (a[i] <= x) res++;
    }
    return res;
}

int main()
{
    // 示例主流程略
    return 0;
}
```

## 练习题目

### 题目部分

1. (选择) 归并排序树处理 n 个元素，构建其所需时间为  
   A. O(n)  
   B. O(n log n)  
   C. O(n log² n)  
   D. O(n²)  

2. (填空) 在归并排序树的单次区间≤x计数查询中，关键操作是____，该操作需要 O(log n) 时间。

3. (选择) 分块树查询区间≤x 个数的复杂度大致是  
   A. O(n)  
   B. O(√n)  
   C. O(√n · log n)  
   D. O(log² n)  

4. (填空) 如果希望支持区间第 k 小查询，一般做法是对答案范围二分，然后结合____ 数据结构完成计数查询。

5. (选择) 在以下场景下，推荐使用分块树而不是归并排序树：  
   A. n、q 都非常大（≥10⁵）  
   B. 只有少量查询，n 特别大  
   C. 需要频繁单点修改，查询也较多  
   D. q 很大但 n 只是中等（≈10⁴）

### 答案与解析

1. 答案：B  
   解析：归并排序树构建时，每层进行一次合并，合并总成本 ∑O(长度) = O(n log n)  

2. 答案：upper_bound  
   解析：在有序数组中二分查找上界，统计不大于 x 的元素个数。  

3. 答案：C  
   解析：分块树跨块时对每块二分 O(log n)，块数 O(√n)，整体 O(√n · log n)。  

4. 答案：归并排序树 / 分块树  
   解析：二分答案后要快速统计任意区间中 ≤ mid 的个数，通常用归并排序树或分块树。  

5. 答案：D  
   解析：n 中等时，块分解实现简单且常数小，适合 q 较多的场景。
