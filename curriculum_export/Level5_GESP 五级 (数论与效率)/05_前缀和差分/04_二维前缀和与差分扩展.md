## 趣味引入  
场景：学校想统计操场上不同区域的落叶数量，每天可能有多次落叶清理（增加或减少），并且需要随时查询任一区域的总落叶数。  
已知：每个格子的落叶数变动记录  
未知：如何在多次区域更新和查询中仍然保持高效？  

通过“已知单点→未知区域”的思路，引出“二维前缀和”“二维差分”两种方法。  

===NEXT===  

## 知识点讲解  

1. 二维前缀和（Prefix Sum）  
   - 定义：  
     S[i][j] 表示 (1,1) 到 (i,j) 矩形内所有元素之和  
   - 计算公式：  
     S[i][j] = a[i][j] + S[i-1][j] + S[i][j-1] - S[i-1][j-1]  
   - 区域查询：  
     sum(x1,y1,x2,y2) = S[x2][y2] - S[x1-1][y2] - S[x2][y1-1] + S[x1-1][y1-1]  

2. 二维差分（Difference Array）  
   - 原理：借助差分数组 D，单次更新 O(1)，最后一次性恢复  
   - 单次对矩形 (x1,y1)-(x2,y2) 增加 v：  
     D[x1][y1]     += v  
     D[x2+1][y1]   -= v  
     D[x1][y2+1]   -= v  
     D[x2+1][y2+1] += v  
   - 恢复原数组：对 D 做二维前缀和  

3. 复杂度对比  
   - 朴素每次枚举：O(n·m)／次  
   - 前缀和：预处理 O(n·m)，单次查询 O(1)  
   - 差分：单次更新 O(1)，恢复 O(n·m)

## 代码示例  
```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1005;
int n, m, q;
long long a[MAXN][MAXN];     // 原始数组（可选用于验证）
long long D[MAXN][MAXN];     // 差分数组
long long S[MAXN][MAXN];     // 前缀和数组

int main()
{
    // 输入行列数和操作数
    cin >> n >> m >> q;

    // 初始化差分数组为 0
    for (int i = 1; i <= n + 1; i++)
    {
        for (int j = 1; j <= m + 1; j++)
        {
            D[i][j] = 0;
        }
    }

    // 读取 q 次操作
    // op = 1 表示区域更新，op = 2 表示区域查询（离线示例）
    for (int t = 0; t < q; t++)
    {
        int op, x1, y1, x2, y2;
        long long v;
        cin >> op;
        if (op == 1)
        {
            cin >> x1 >> y1 >> x2 >> y2 >> v;
            // 差分更新
            D[x1][y1]       += v;
            D[x2 + 1][y1]   -= v;
            D[x1][y2 + 1]   -= v;
            D[x2 + 1][y2 + 1] += v;
        }
        else if (op == 2)
        {
            // 离线查询：先构造好前缀和再回答
        }
    }

    // 恢复原始数组：对差分数组做二维前缀和
    for (int i = 1; i <= n; i++)
    {
        for (int j = 1; j <= m; j++)
        {
            S[i][j] = D[i][j] + S[i - 1][j] + S[i][j - 1] - S[i - 1][j - 1];
        }
    }

    // 示例：输出恢复后的数组
    for (int i = 1; i <= n; i++)
    {
        for (int j = 1; j <= m; j++)
        {
            cout << S[i][j] << " ";
        }
        cout << "\n";
    }

    return 0;
}
```

## 练习题目  

基础题（选择或填空）  
1. 二维前缀和 S[i][j] 的递推公式为：  
   S[i][j] = a[i][j] + ____ + ____ - ____  
2. 查询矩形 (x1,y1)-(x2,y2) 的和公式为：  
   sum = S[x2][y2] - ____ - ____ + ____  
3. 对差分数组 D 进行单次区域更新时，下列哪几项需要修改（多选）：  
   A. D[x1][y1]  
   B. D[x1][y2+1]  
   C. D[x2+1][y1]  
   D. D[x2+1][y2+1]  
4. 构造 n×m 差分数组并做一次恢复的时间复杂度是？  
   A. O(n·m)  
   B. O(n + m)  
   C. O(1)  
   D. O(n·m·log(n·m))  

进阶题  
5. 如果要给 t 次区域更新都生效并随时在线查询，应当怎么做？  

请先独立完成，下面给出答案与解析。  

答案与解析  
1. S[i][j] = a[i][j] + S[i-1][j] + S[i][j-1] - S[i-1][j-1]  
2. sum = S[x2][y2] - S[x1-1][y2] - S[x2][y1-1] + S[x1-1][y1-1]  
3. 正确答案：A、B、C、D（都要修改）  
   解析：差分原理需要在四个拐角处调整增量  
4. 正确答案：A  
   解析：恢复是两层循环，复杂度 O(n·m)  
5. 在线场景下，需要对差分数组每次更新后，及时维护前缀和或使用树状数组、线段树等更高级结构。
