## 深度知识点讲解

### 1. 链表的本质：分散的连接

链表，顾名思义，就是像链条一样连接起来的列表。它不像数组那样，把所有数据紧密地排在一起。想象一下散落在房间各处的宝藏箱，每个宝藏箱里不仅有宝藏（数据），还有一张纸条告诉你下一个宝藏箱藏在哪里（指针）。你只需要知道第一个宝藏箱的位置，就能找到所有的宝藏。

*   **节点 (Node)：** 链表的基本组成单位。每个节点通常包含两部分：
    *   **数据域 (Data Field)：** 存放实际数据，比如一个数字、一个名字或者一个更复杂的结构体。
    *   **指针域 (Pointer Field)：** 存放一个“地址”，这个地址指向链表中的**下一个**节点。
*   **头指针 (Head Pointer)：** 一个特殊的指针，它指向链表的第一个节点。通过头指针，我们才能找到整个链表。如果头指针指向空（`nullptr`），则表示链表是空的。
*   **尾节点 (Tail Node)：** 链表的最后一个节点。它的指针域通常指向空（`nullptr`），表示后面没有节点了。

**为什么是 `nullptr`？**
`nullptr` 是C++11引入的空指针常量，用来表示一个指针不指向任何有效的内存地址。在链表中，它就像宝藏箱里的最后一张纸条，上面写着“这里是终点，没有下一个宝藏箱了！”

### 2. 链表与数组：两种存储哲学

| 特性         | 数组 (Array)                                  | 链表 (Linked List)                                |
| :----------- | :-------------------------------------------- | :------------------------------------------------ |
| **存储方式** | 内存中连续存储，像排队的学生。                  | 内存中分散存储，通过指针连接，像传话筒游戏。        |
| **大小**     | 固定大小（创建时确定），难以动态扩展。          | 动态大小，可以随时添加或删除节点，理论上无限扩展。  |
| **随机访问** | O(1) - 知道下标就能直接访问，速度极快。         | O(N) - 必须从头开始，一个一个节点地遍历才能找到。 |
| **插入/删除** | O(N) - 在中间插入/删除，需要移动大量元素。      | O(1) - 如果已经找到插入/删除位置，只需修改少数指针。 |
| **内存利用** | 空间连续，可能造成“外部碎片”（大块内存无法分配）。 | 空间分散，更灵活利用“内部碎片”（小块内存）。      |
| **额外开销** | 无额外开销（只存储数据）。                      | 每个节点需要额外存储一个或多个指针。              |

**总结：**
*   数组的“超能力”是**随机访问**和**紧凑存储**。
*   链表的“超能力”是**动态大小**和**高效的插入/删除**（在已知位置时）。

### 3. 核心操作：指针的魔法

链表的所有操作，都是围绕着**指针**的修改进行的。理解指针如何“指来指去”是关键。

#### 3.1 节点结构定义

这是链表的基础，就像我们给每个小朋友设计一个“纸条”模板。

```cpp
struct Node {
    int data;    // 数据域，存放节点的值
    Node* next;  // 指针域，指向下一个节点的地址

    // 构造函数，方便创建节点
    Node(int v) : data(v), next(nullptr) {}
};
```

#### 3.2 遍历链表 (Walking the Chain)

从头指针开始，沿着 `next` 指针一步步走下去，直到遇到 `nullptr`。

```cpp
void printList(Node* head) {
    Node* p = head; // 临时指针p，从头开始
    while (p != nullptr) { // 只要p不是空，就继续走
        cout << p->data << " "; // 打印当前节点的数据
        p = p->next;            // 走向下一个节点
    }
    cout << endl;
}
```
**思考：** 如果 `head` 是 `nullptr`，会发生什么？`printList` 函数会输出什么？

#### 3.3 查找节点 (Finding a Specific Link)

遍历链表，直到找到目标值或遍历结束。

```cpp
Node* findNode(Node* head, int target) {
    Node* p = head;
    while (p != nullptr) {
        if (p->data == target) {
            return p; // 找到目标，返回该节点指针
        }
        p = p->next;
    }
    return nullptr; // 没有找到
}
```

#### 3.4 插入节点 (Adding a Link)

这是链表最灵活的地方，但也是最容易出错的地方。关键在于**改变指针的顺序**。

**a) 头插法 (Prepend)：在链表头部插入新节点**
就像把新车厢加到火车头前面。
1.  创建新节点 `newNode`。
2.  让 `newNode` 指向原来的 `head`。
3.  更新 `head`，让它指向 `newNode`。

```cpp
// 假设 head 是指向链表头部的指针
void insertAtHead(Node* &head, int v) { // 注意这里head是引用，因为我们要修改它
    Node* newNode = new Node(v); // 1. 创建新节点
    newNode->next = head;        // 2. 新节点指向原来的头节点
    head = newNode;              // 3. 更新头指针，指向新节点
}
```
**画图理解：** `newNode` -> `old_head`；`head` -> `newNode`。

**b) 尾插法 (Append)：在链表尾部插入新节点**
就像把新车厢加到火车尾部。
1.  创建新节点 `newNode`。
2.  如果链表为空，新节点就是头节点。
3.  否则，遍历到尾节点，让尾节点指向新节点。

```cpp
void insertAtTail(Node* &head, int v) {
    Node* newNode = new Node(v);
    if (head == nullptr) { // 如果链表为空，新节点就是头节点
        head = newNode;
        return;
    }
    Node* p = head;
    while (p->next != nullptr) { // 遍历到最后一个节点
        p = p->next;
    }
    p->next = newNode; // 最后一个节点指向新节点
}
```
**画图理解：** `old_tail` -> `newNode`。

**c) 在指定节点后插入 (Insert After)：**
这是最通用的插入方式，也是最能体现链表灵活性的地方。
1.  找到要插入位置的前一个节点 `prevNode`。
2.  创建新节点 `newNode`。
3.  让 `newNode` 指向 `prevNode` 原来指向的节点。
4.  让 `prevNode` 指向 `newNode`。

```cpp
// 在 prevNode 之后插入值为 v 的节点
void insertAfter(Node* prevNode, int v) {
    if (prevNode == nullptr) { // 如果前一个节点为空，无法插入
        cout << "Error: prevNode cannot be nullptr." << endl;
        return;
    }
    Node* newNode = new Node(v);
    newNode->next = prevNode->next; // 1. 新节点指向 prevNode 原来的下一个节点
    prevNode->next = newNode;       // 2. prevNode 指向新节点
}
```
**画图理解：** `newNode` -> `prevNode->next`；`prevNode` -> `newNode`。**注意顺序，如果先改变 `prevNode->next`，那么 `prevNode` 原来的下一个节点就找不到了！**

#### 3.5 删除节点 (Removing a Link)

同样需要小心处理指针。删除节点后，**记得释放内存**，避免内存泄漏。

**a) 删除头部节点 (Delete Head)：**
1.  如果链表为空，直接返回。
2.  用一个临时指针 `temp` 记录当前头节点。
3.  更新 `head`，让它指向原来的第二个节点。
4.  释放 `temp` 指向的内存。

```cpp
void deleteHead(Node* &head) {
    if (head == nullptr) {
        return;
    }
    Node* temp = head;   // 1. 记录要删除的头节点
    head = head->next;   // 2. 头指针指向下一个节点
    delete temp;         // 3. 释放原头节点的内存
    temp = nullptr;      // 避免野指针
}
```

**b) 删除指定节点 (Delete Specific Node)：**
通常是删除值为 `v` 的节点，或者删除 `prevNode` 后面的节点。
这里我们以删除 `prevNode` 后的节点为例：
1.  找到要删除节点的前一个节点 `prevNode`。
2.  如果 `prevNode` 或 `prevNode` 的下一个节点为空，无法删除。
3.  用一个临时指针 `temp` 记录要删除的节点 (`prevNode->next`)。
4.  让 `prevNode` 指向 `temp` 原来指向的下一个节点 (`temp->next`)。
5.  释放 `temp` 指向的内存。

```cpp
// 删除 prevNode 后面的节点
void deleteAfter(Node* prevNode) {
    if (prevNode == nullptr || prevNode->next == nullptr) {
        // 无法删除：prevNode为空，或prevNode是尾节点
        cout << "Error: Cannot delete after this node." << endl;
        return;
    }
    Node* temp = prevNode->next; // 1. 记录要删除的节点
    prevNode->next = temp->next; // 2. prevNode 指向 temp 原来的下一个节点
    delete temp;                 // 3. 释放内存
    temp = nullptr;              // 避免野指针
}
```
**画图理解：** `prevNode` -> `temp->next`。**注意：如果你要删除的是头节点，这个函数就不适用了，需要用 `deleteHead`。**

### 4. 常见误区与注意事项

*   **空指针判断：** 在访问 `p->data` 或 `p->next` 之前，一定要先判断 `p` 是否为 `nullptr`，否则会引发运行时错误（空指针解引用）。
*   **指针顺序：** 插入和删除操作时，改变指针的顺序至关重要。画图是最好的辅助工具。
*   **内存泄漏：** 使用 `new` 分配的内存，在不再使用时一定要 `delete` 释放，否则会造成内存泄漏。
*   **头指针引用：** 如果函数内部需要修改 `head` 指针本身（比如头插、头删），那么 `head` 参数必须是引用类型 `Node* &head`。如果只是遍历或修改 `head` 指向的节点内容，`Node* head` 即可。

理解了这些基本操作和注意事项，你就能掌握链表的精髓啦！

## 典型例题精讲

### 例题一：链表反转 (Reverse a Linked List)

**题目描述：**
给定一个单向链表的头指针 `head`，请你将整个链表反转，并返回反转后的新链表头指针。

**示例：**
输入链表：`1 -> 2 -> 3 -> 4 -> 5 -> nullptr`
输出链表：`5 -> 4 -> 3 -> 2 -> 1 -> nullptr`

**思路分析：**
这就像把一串珠子从头到尾重新串一遍，让原来的“下一个”变成“上一个”。
我们不能一下子把所有指针都反过来，否则会迷失方向。我们需要**逐个节点**地反转指针。

想象一下有三个指针在链表上“跳舞”：
1.  `prev` (previous): 指向当前节点的前一个节点。一开始，还没有前一个节点，所以 `prev` 是 `nullptr`。
2.  `curr` (current): 指向当前正在处理的节点。一开始，`curr` 就是链表的 `head`。
3.  `next_node` (next_node): 指向当前节点的下一个节点。我们需要提前保存它，因为 `curr->next` 将会被修改。

**核心步骤：**
在一个循环中，每次处理一个 `curr` 节点：
1.  **保存 `curr` 的下一个节点：** `next_node = curr->next;` (这是为了防止我们改变 `curr->next` 后，就找不到原来的下一个节点了)。
2.  **反转 `curr` 的 `next` 指针：** `curr->next = prev;` (让当前节点指向它的前一个节点)。
3.  **移动 `prev` 和 `curr` 指针：**
    *   `prev = curr;` (当前节点处理完了，它就变成了下一个节点的前一个节点)。
    *   `curr = next_node;` (移动到下一个节点继续处理)。

当 `curr` 变为 `nullptr` 时，表示所有节点都处理完了。此时 `prev` 就指向了原链表的最后一个节点，也就是反转后链表的头节点。

**画图推演：**
初始：`nullptr <- prev   curr -> next_node -> ...`
             `1 -> 2 -> 3 -> nullptr`
             `prev=nullptr, curr=1, next_node` 还没取

1.  `curr = 1`
    *   `next_node = curr->next` (即 `2`)
    *   `curr->next = prev` (即 `1->next = nullptr`)
    *   `prev = curr` (即 `prev = 1`)
    *   `curr = next_node` (即 `curr = 2`)
    链表状态：`nullptr <- 1   2 -> 3 -> nullptr` (注意 `1` 已经反转，但 `2` 还没处理)

2.  `curr = 2`
    *   `next_node = curr->next` (即 `3`)
    *   `curr->next = prev` (即 `2->next = 1`)
    *   `prev = curr` (即 `prev = 2`)
    *   `curr = next_node` (即 `curr = 3`)
    链表状态：`nullptr <- 1 <- 2   3 -> nullptr`

3.  `curr = 3`
    *   `next_node = curr->next` (即 `nullptr`)
    *   `curr->next = prev` (即 `3->next = 2`)
    *   `prev = curr` (即 `prev = 3`)
    *   `curr = next_node` (即 `curr = nullptr`)
    链表状态：`nullptr <- 1 <- 2 <- 3   nullptr`

循环结束，返回 `prev`，即 `3`。

**代码实现：**

```cpp
#include <bits/stdc++.h>

using namespace std;

// 节点结构定义
struct Node {
    int data;
    Node* next;

    Node(int v) : data(v), next(nullptr) {}
};

// 打印链表函数
void printList(Node* head) {
    Node* p = head;
    while (p != nullptr) {
        cout << p->data << " ";
        p = p->next;
    }
    cout << endl;
}

// 链表反转函数
Node* reverseList(Node* head) {
    Node* prev = nullptr;    // 前一个节点指针，初始为nullptr
    Node* curr = head;       // 当前节点指针，从头开始
    Node* next_node = nullptr; // 临时保存下一个节点指针

    while (curr != nullptr) {
        next_node = curr->next; // 1. 保存当前节点的下一个节点
        curr->next = prev;      // 2. 反转当前节点的指针，指向前一个节点
        prev = curr;            // 3. prev 指针向前移动一位，指向当前节点
        curr = next_node;       // 4. curr 指针向前移动一位，指向下一个节点
    }
    return prev; // 当curr为nullptr时，prev就是反转后链表的头节点
}

int main() {
    // 创建一个链表: 1 -> 2 -> 3 -> 4 -> 5
    Node* head = new Node(1);
    head->next = new Node(2);
    head->next->next = new Node(3);
    head->next->next->next = new Node(4);
    head->next->next->next->next = new Node(5);

    cout << "原始链表: ";
    printList(head);

    Node* reversedHead = reverseList(head);

    cout << "反转后链表: ";
    printList(reversedHead);

    // 释放内存 (为了演示，实际竞赛中可能不严格要求)
    Node* p = reversedHead;
    while (p != nullptr) {
        Node* temp = p;
        p = p->next;
        delete temp;
    }
    reversedHead = nullptr; // 防止野指针

    return 0;
}
```
**复杂度分析：**
*   **时间复杂度：** 我们遍历了链表中的每个节点一次，对每个节点的操作都是常数时间。所以，时间复杂度是 O(N)，其中 N 是链表的节点数量。
*   **空间复杂度：** 我们只使用了几个额外的指针变量 (`prev`, `curr`, `next_node`)，它们占用的空间是常数。所以，空间复杂度是 O(1)。

### 例题二：合并两个有序链表 (Merge Two Sorted Linked Lists)

**题目描述：**
给定两个已经按升序排列的单向链表的头指针 `h1` 和 `h2`，请你将它们合并成一个新的有序链表，并返回新链表的头指针。合并后的链表也应该是升序的。

**示例：**
输入链表 `h1`: `1 -> 3 -> 5 -> nullptr`
输入链表 `h2`: `2 -> 4 -> 6 -> nullptr`
输出链表: `1 -> 2 -> 3 -> 4 -> 5 -> 6 -> nullptr`

**思路分析：**
这就像你有两叠已经按大小排好的扑克牌，你想把它们合并成一叠新的，并且新的一叠也依然是按大小排好的。

我们可以使用“双指针”技巧，同时遍历两个链表。每次比较两个链表当前节点的值，哪个小，就把哪个节点取出来，添加到新链表的末尾。

**核心步骤：**
1.  **创建虚拟头节点 (Dummy Head)：** 为了方便处理新链表的头部，我们通常会创建一个虚拟头节点 `dummyHead`。它的 `next` 指针将指向我们新链表的第一个实际节点。这样，无论是哪个链表的节点先被添加到新链表，我们都不需要特殊处理头节点。
2.  **当前指针 `p`：** 维护一个指针 `p`，它始终指向新链表的当前末尾节点。
3.  **比较并添加：**
    *   当 `h1` 和 `h2` 都不为空时，比较 `h1->data` 和 `h2->data`。
    *   如果 `h1->data <= h2->data`，则将 `h1` 的当前节点连接到 `p->next`，然后 `h1` 移动到下一个节点。
    *   否则，将 `h2` 的当前节点连接到 `p->next`，然后 `h2` 移动到下一个节点。
    *   无论哪种情况，`p` 都要移动到新添加的节点上 (`p = p->next;`)。
4.  **处理剩余节点：** 当其中一个链表遍历完（变为 `nullptr`）时，另一个链表可能还有剩余节点。因为这两个链表都是有序的，所以剩余的节点直接全部连接到新链表的末尾即可。
5.  **返回结果：** 新链表的头节点是 `dummyHead->next`。别忘了释放 `dummyHead` 本身的内存。

**画图推演：**
初始：
`h1: 1 -> 3 -> 5`
`h2: 2 -> 4 -> 6`
`dummyHead -> p` (p指向dummyHead)

1.  `h1->data (1)` < `h2->data (2)`
    *   `p->next = h1` (即 `dummyHead -> 1`)
    *   `h1 = h1->next` (即 `h1` 变为 `3`)
    *   `p = p->next` (即 `p` 变为 `1`)
    新链表: `dummyHead -> 1`

2.  `h1->data (3)` > `h2->data (2)`
    *   `p->next = h2` (即 `1 -> 2`)
    *   `h2 = h2->next` (即 `h2` 变为 `4`)
    *   `p = p->next` (即 `p` 变为 `2`)
    新链表: `dummyHead -> 1 -> 2`

3.  `h1->data (3)` < `h2->data (4)`
    *   `p->next = h1` (即 `2 -> 3`)
    *   `h1 = h1->next` (即 `h1` 变为 `5`)
    *   `p = p->next` (即 `p` 变为 `3`)
    新链表: `dummyHead -> 1 -> 2 -> 3`

4.  `h1->data (5)` > `h2->data (4)`
    *   `p->next = h2` (即 `3 -> 4`)
    *   `h2 = h2->next` (即 `h2` 变为 `6`)
    *   `p = p->next` (即 `p` 变为 `4`)
    新链表: `dummyHead -> 1 -> 2 -> 3 -> 4`

5.  `h1->data (5)` < `h2->data (6)`
    *   `p->next = h1` (即 `4 -> 5`)
    *   `h1 = h1->next` (即 `h1` 变为 `nullptr`)
    *   `p = p->next` (即 `p` 变为 `5`)
    新链表: `dummyHead -> 1 -> 2 -> 3 -> 4 -> 5`

6.  `h1` 为 `nullptr`，循环结束。
    *   将剩余的 `h2` (即 `6`) 直接连接到 `p` (即 `5`) 的后面。
    *   `p->next = h2` (即 `5 -> 6`)
    新链表: `dummyHead -> 1 -> 2 -> 3 -> 4 -> 5 -> 6`

最后返回 `dummyHead->next`。

**代码实现：**

```cpp
#include <bits/stdc++.h>

using namespace std;

// 节点结构定义 (与上例相同)
struct Node {
    int data;
    Node* next;

    Node(int v) : data(v), next(nullptr) {}
};

// 打印链表函数 (与上例相同)
void printList(Node* head) {
    Node* p = head;
    while (p != nullptr) {
        cout << p->data << " ";
        p = p->next;
    }
    cout << endl;
}

// 合并两个有序链表函数
Node* mergeTwoLists(Node* h1, Node* h2) {
    // 1. 创建一个虚拟头节点，方便处理新链表的开头
    Node* dummyHead = new Node(0); // 虚拟节点，数据值不重要
    Node* p = dummyHead;           // p 指针用于构建新链表

    // 2. 同时遍历两个链表，比较并添加节点
    while (h1 != nullptr && h2 != nullptr) {
        if (h1->data <= h2->data) {
            p->next = h1; // 将h1的当前节点连接到新链表
            h1 = h1->next; // h1向前移动
        } else {
            p->next = h2; // 将h2的当前节点连接到新链表
            h2 = h2->next; // h2向前移动
        }
        p = p->next; // p 指针始终指向新链表的当前末尾
    }

    // 3. 处理剩余节点：将未遍历完的链表直接连接到新链表末尾
    if (h1 != nullptr) {
        p->next = h1;
    } else if (h2 != nullptr) {
        p->next = h2;
    }

    // 4. 获取新链表的头节点并释放虚拟头节点
    Node* mergedHead = dummyHead->next;
    delete dummyHead; // 释放虚拟头节点内存
    dummyHead = nullptr; // 防止野指针

    return mergedHead;
}

int main() {
    // 创建第一个有序链表: 1 -> 3 -> 5
    Node* h1 = new Node(1);
    h1->next = new Node(3);
    h1->next->next = new Node(5);

    // 创建第二个有序链表: 2 -> 4 -> 6
    Node* h2 = new Node(2);
    h2->next = new Node(4);
    h2->next->next = new Node(6);

    cout << "链表 1: ";
    printList(h1);
    cout << "链表 2: ";
    printList(h2);

    Node* mergedHead = mergeTwoLists(h1, h2);

    cout << "合并后链表: ";
    printList(mergedHead);

    // 释放合并后链表的内存
    Node* p = mergedHead;
    while (p != nullptr) {
        Node* temp = p;
        p = p->next;
        delete temp;
    }
    mergedHead = nullptr; // 防止野指针

    return 0;
}
```
**复杂度分析：**
*   **时间复杂度：** 我们遍历了两个链表，每个节点最多被访问一次。如果链表 `h1` 的长度是 `N1`，链表 `h2` 的长度是 `N2`，那么总的时间复杂度是 O(N1 + N2)。
*   **空间复杂度：** 除了虚拟头节点和几个辅助指针外，我们没有创建新的节点，只是重新组织了原有节点的连接关系。所以，空间复杂度是 O(1)。
    *   *注意：* 如果题目要求创建一个全新的链表，而不是修改原链表的节点，那么空间复杂度将是 O(N1 + N2)，因为需要创建 N1 + N2 个新节点。但在本题的常见解法中，我们复用了原有节点。

## 代码实现模板

下面是一个包含链表基本操作的C++代码模板，你可以基于此进行修改和扩展。

```cpp
#include <bits/stdc++.h> // 包含常用头文件，如iostream等

using namespace std;

// 1. 节点结构定义
struct Node {
    int data;    // 数据域，存放节点的值
    Node* next;  // 指针域，指向下一个节点的地址

    // 构造函数，方便创建节点
    Node(int v) : data(v), next(nullptr) {}
};

// --- 辅助函数 ---

// 2. 打印链表
void printList(Node* head) {
    Node* p = head;
    cout << "链表内容: ";
    while (p != nullptr) {
        cout << p->data << " -> ";
        p = p->next;
    }
    cout << "nullptr" << endl;
}

// 3. 释放链表所有内存
void clearList(Node* &head) { // 注意：head是引用，因为可能需要将head置为nullptr
    Node* p = head;
    while (p != nullptr) {
        Node* temp = p; // 记录当前节点
        p = p->next;    // 移动到下一个节点
        delete temp;    // 释放当前节点的内存
        temp = nullptr; // 防止野指针
    }
    head = nullptr; // 链表清空后，头指针指向nullptr
    cout << "链表内存已释放." << endl;
}

// --- 链表基本操作 ---

// 4. 头插法：在链表头部插入新节点
void insertAtHead(Node* &head, int v) {
    Node* newNode = new Node(v);
    newNode->next = head;
    head = newNode;
    cout << "插入 " << v << " 到头部." << endl;
}

// 5. 尾插法：在链表尾部插入新节点
void insertAtTail(Node* &head, int v) {
    Node* newNode = new Node(v);
    if (head == nullptr) { // 如果链表为空，新节点就是头节点
        head = newNode;
        cout << "插入 " << v << " 到尾部 (链表为空，成为头节点)." << endl;
        return;
    }
    Node* p = head;
    while (p->next != nullptr) { // 遍历到最后一个节点
        p = p->next;
    }
    p->next = newNode; // 最后一个节点指向新节点
    cout << "插入 " << v << " 到尾部." << endl;
}

// 6. 在指定节点后插入：在 prevNode 之后插入值为 v 的节点
// 返回值为 true 表示成功，false 表示失败 (prevNode 为空)
bool insertAfter(Node* prevNode, int v) {
    if (prevNode == nullptr) {
        cout << "错误: 无法在空节点后插入." << endl;
        return false;
    }
    Node* newNode = new Node(v);
    newNode->next = prevNode->next;
    prevNode->next = newNode;
    cout << "在 " << prevNode->data << " 后插入 " << v << "." << endl;
    return true;
}

// 7. 删除头部节点
void deleteHead(Node* &head) {
    if (head == nullptr) {
        cout << "链表为空，无法删除头部." << endl;
        return;
    }
    Node* temp = head;
    head = head->next;
    cout << "删除头部节点: " << temp->data << endl;
    delete temp;
    temp = nullptr;
}

// 8. 删除指定节点后的节点 (删除 prevNode 的下一个节点)
// 返回值为 true 表示成功，false 表示失败
bool deleteAfter(Node* prevNode) {
    if (prevNode == nullptr || prevNode->next == nullptr) {
        cout << "错误: 无法删除指定节点后的节点 (prevNode为空或prevNode是尾节点)." << endl;
        return false;
    }
    Node* temp = prevNode->next; // 要删除的节点
    prevNode->next = temp->next; // prevNode 指向 temp 的下一个节点
    cout << "删除节点: " << temp->data << endl;
    delete temp;
    temp = nullptr;
    return true;
}

// 9. 查找节点：返回第一个值为 target 的节点指针
Node* findNode(Node* head, int target) {
    Node* p = head;
    while (p != nullptr) {
        if (p->data == target) {
            cout << "找到节点 " << target << "." << endl;
            return p;
        }
        p = p->next;
    }
    cout << "未找到节点 " << target << "." << endl;
    return nullptr;
}

int main() {
    Node* head = nullptr; // 初始化空链表

    printList(head); // 打印空链表

    insertAtTail(head, 10); // 10
    insertAtTail(head, 20); // 10 -> 20
    insertAtHead(head, 5);  // 5 -> 10 -> 20
    printList(head);

    Node* targetNode = findNode(head, 10);
    if (targetNode != nullptr) {
        insertAfter(targetNode, 15); // 5 -> 10 -> 15 -> 20
    }
    printList(head);

    deleteHead(head); // 10 -> 15 -> 20
    printList(head);

    targetNode = findNode(head, 15);
    if (targetNode != nullptr) {
        deleteAfter(targetNode); // 10 -> 15 (删除20)
    }
    printList(head);

    // 再次查找，尝试删除不存在的节点
    targetNode = findNode(head, 100);
    deleteAfter(targetNode); // 应该报错

    // 清理所有内存
    clearList(head);
    printList(head); // 再次打印，确认链表已空

    return 0;
}
```
