## 趣味引入  
情景：  
小明和小华在准备数学竞赛，需要对一组数字进行“拆分”，把每个合数拆成最小的“砖块”——质数。他们发现手算很慢，想用编程来加速。  

已知：  
- 质数是只能被 1 和自身整除的大于 1 的整数。  
- 合数是有除 1 和自身之外的其他因数的整数。  

未知：  
- 如何让程序高效地“拆砖”，找到一个数的所有质因数？  

通过引入“试除”的思想，我们一步步揭开算法奥秘！

===NEXT===

## 知识点讲解  
1. 质数与合数  
   - 质数定义：大于 1 且仅有两个因数（1 和自身）。  
   - 合数定义：大于 1 且除了 1 和自身，还有其他因数。  
2. 质因数分解  
   - 将一个合数分解成若干个质数相乘的形式。  
   - 例如：60 = 2² × 3 × 5。  
3. 朴素试除法思路  
   - 从最小的质数 2 开始，依次尝试除数 i（i = 2, 3, 4, …），只要 n % i == 0，就将 i 作为质因数，n /= i，并重复尝试同一 i。  
   - 当 i*i > n 时，如果剩下的 n 大于 1，则剩余 n 本身也是质因数。  
4. 时间复杂度分析  
   - 最坏情况下，需要尝试 2 到 √n 的所有 i，时间复杂度 O(√n)。  
   - 适用于中小规模整数的分解。  
5. 优化思路（拓展，不在本节详细讲解）  
   - 只试除 2 和奇数 i，或预先筛素数进行分解。

## 代码示例  
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    long long n;
    // 输入待分解的数
    cin >> n;

    // 存储质因数及其个数
    long long fac[30];
    int cnt[30];
    int k = 0;
    // 初始化计数
    for (int i = 0; i < 30; i++)
    {
        cnt[i] = 0;
    }

    // 试除法分解
    for (long long i = 2; i * i <= n; i++)
    {
        // 如果 i 是 n 的质因数
        while (n % i == 0)
        {
            fac[k] = i;      // 记录质因数 i
            cnt[k]++;        // 对应因数个数加一
            n /= i;          // 除去已分解的因数
        }
        if (cnt[k] > 0)
        {
            k++;             // 移动到下一个存储位置
        }
    }

    // 如果剩下的 n > 1，则它本身也是质因数
    if (n > 1)
    {
        fac[k] = n;
        cnt[k] = 1;
        k++;
    }

    // 输出结果
    for (int i = 0; i < k; i++)
    {
        cout << fac[i];
        if (cnt[i] > 1)
        {
            cout << "^" << cnt[i];
        }
        if (i + 1 < k)
        {
            cout << " * ";
        }
    }
    cout << endl;

    return 0;
}
```

关键行注释已在代码中标出，帮助理解每一步的目的。

## 练习题目  

### 基础题  
1. 填空：质因数分解中，只需要试除到 ____ （填符号表达式），即可保证正确分解。  
2. 选择题：以下哪个数是质数？  
   A. 51  
   B. 53  
   C. 57  
   D. 63  

### 进阶题  
3. 填空：对于 n=360，试除法分解后应得到 360 = ____。  
4. 选择题：朴素试除法的时间复杂度是 ____。  
   A. O(n)  
   B. O(log n)  
   C. O(√n)  
   D. O(n log n)  

### 答案与解析  
1. √n  
   解析：若 i>√n, 则另一因子必小于√n，已被搜过。  
2. B  
   解析：53 只能被 1 和 53 整除，是质数。  
3. 2^3 * 3^2 * 5  
   解析：360 ÷2=180 ÷2=90 ÷2=45；45÷3=15÷3=5，剩 5。  
4. C  
   解析：试除到 √n，最坏情况尝试了 √n 次除法。
