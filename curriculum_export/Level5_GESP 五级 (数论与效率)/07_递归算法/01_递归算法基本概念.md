## 教案内容

本章首先讲解递归的定义与性质，重点包括递归函数的基准情形（base case）和递推关系（recursive case），让学生理解如何将大问题分解为子问题。授课内容包括：1）递归的数学模型和调用栈工作原理；2）基准情形的重要性，避免无限递归；3）递归深度与栈空间的关系；4）手写递归示例：计算 n 的幂（f(n)=a×f(n-1)）；5）Allman 风格 C++ 代码模板：#include <bits/stdc++.h>
using namespace std;

int pow_n(int a, int n)
{
    if (n == 0)
    {
        return 1;
    }
    return a * pow_n(a, n - 1);
}
重点引导学生关注基准条件判断和返回值传递，难点在于理解函数调用的栈帧展开与回溯过程。
