# Level 5：GESP 五级 (数论与效率)

## 教学目标
1. 掌握常用的数学算法（质数、GCD等）。
2. 理解递归算法的思想与实现。
3. 掌握二维数组的高级应用。

## 核心知识点
- 数论基础：素数判定、最大公约数 (GCD)、最小公倍数 (LCM)。
- 递归：递归的定义、终止条件、递归栈（如斐波那契数列）。
- 位运算：按位与、或、异或、左移、右移。
