## 知识点讲解

### 1. 什么是树？

树是一种非常重要且应用广泛的非线性数据结构。它模拟了自然界中的树状结构，具有层级关系。

**基本定义**：
树（Tree）是n（n≥0）个节点的有限集合。
*   当 n=0 时，称为空树。
*   当 n>0 时，它满足以下条件：
    1.  有一个特殊的节点，称为**根节点**（Root Node），它没有父节点。
    2.  除根节点外，其余节点被分成 m（m≥0）个互不相交的有限集 T1, T2, ..., Tm，其中每一个集合又都是一棵树，并且称之为根节点的**子树**（Subtree）。

**树的特点**：
*   一个根节点。
*   除根节点外，每个节点有且只有一个父节点。
*   没有循环（即从任何一个节点出发，都不能通过沿着边回到该节点）。
*   节点之间具有明显的层级关系。

### 2. 树的基本术语

我们用一个图来帮助理解这些术语：

```
       A (根节点)
      /|\
     B C D
    /|   |
   E F   G
  /      |
 H       I
```

*   **节点（Node）**：树中的每一个元素都称为节点。例如：A, B, C, ..., I。
*   **根节点（Root Node）**：树中没有父节点的节点，是树的起点。例如：A。
*   **父节点（Parent Node）**：若一个节点包含子节点，则该节点是其子节点的父节点。例如：A 是 B, C, D 的父节点；B 是 E, F 的父节点。
*   **子节点（Child Node）**：若一个节点是另一个节点的后继，则该节点是其父节点的子节点。例如：B, C, D 是 A 的子节点；E, F 是 B 的子节点。
*   **兄弟节点（Sibling Node）**：拥有相同父节点的节点。例如：B, C, D 互为兄弟节点；E, F 互为兄弟节点。
*   **叶子节点（Leaf Node）**：没有子节点的节点。例如：C, F, H, I。
*   **分支节点（Internal Node）**：除叶子节点外的所有节点（包括根节点）。例如：A, B, D, E, G。
*   **度（Degree）**：
    *   **节点的度**：一个节点拥有的子树（或子节点）的数量。例如：A 的度为 3；B 的度为 2；C 的度为 0。
    *   **树的度**：树中所有节点度的最大值。例如：上面这棵树的度为 3（A 的度是 3）。
*   **路径（Path）**：从一个节点到另一个节点所经过的节点序列。例如：从 A 到 H 的路径是 A -> B -> E -> H。
*   **深度（Depth）**：
    *   **节点的深度**：从根节点到该节点的路径上的边的数量。根节点的深度为 0。
    *   **树的深度**：树中所有节点深度的最大值。例如：A 的深度为 0；B 的深度为 1；E 的深度为 2；H 的深度为 3。这棵树的深度为 3。
*   **高度（Height）**：
    *   **节点的高度**：从该节点到最远叶子节点的路径上的边的数量。叶子节点的高度为 0。
    *   **树的高度**：根节点的高度。例如：H 的高度为 0；E 的高度为 1；B 的高度为 2；A 的高度为 3。这棵树的高度为 3。
    *   **注意**：树的深度和高度通常是同一个概念，但有时定义略有不同（深度从0开始，高度从1开始）。在信奥中，我们通常认为根节点在第1层，深度从1开始计算，或从0开始。这里我们统一认为**根节点深度为0，高度为树中节点最大深度 + 1**。或者简单来说，**树的深度 = 树的高度**。
*   **层（Level）**：节点的深度加 1。根节点在第 1 层，其子节点在第 2 层，以此类推。

### 3. 什么是二叉树？

二叉树（Binary Tree）是一种特殊的树，它的每个节点最多只有两个子节点。这两个子节点通常被称为**左子节点**（Left Child）和**右子节点**（Right Child）。

**二叉树的特点**：
*   每个节点最多有两个子节点。
*   子节点分为左右，顺序不能颠倒。即使只有一个子节点，也要区分是左子节点还是右子节点。

### 4. 特殊的二叉树

*   **满二叉树（Full Binary Tree）**：
    *   除叶子节点外，每个节点都有两个子节点。
    *   所有叶子节点都在同一层上。
    *   形象地说，它是一个“完美”的三角形。

    ```
          A
         / \
        B   C
       / \ / \
      D  E F  G
    ```

*   **完全二叉树（Complete Binary Tree）**：
    *   除最后一层外，其他各层节点数都达到最大值。
    *   最后一层的节点都集中在左侧，可以不满。
    *   满二叉树一定是完全二叉树，但完全二叉树不一定是满二叉树。

    ```
          A
         / \
        B   C
       / \ /
      D  E F
    ```
    这是一个完全二叉树。

    ```
          A
         / \
        B   C
       /
      D
    ```
    这也是一个完全二叉树。

    ```
          A
         / \
        B   C
         \
          D
    ```
    这不是一个完全二叉树，因为 D 节点没有靠左排列。

### 5. 二叉树的性质

对于一个深度为 k (根节点深度为 0) 的二叉树：

1.  **第 i 层（i 从 0 开始）最多有 $2^i$ 个节点。**
    *   第 0 层（根节点）有 $2^0 = 1$ 个节点。
    *   第 1 层有 $2^1 = 2$ 个节点。
    *   第 i 层有 $2^i$ 个节点。
2.  **深度为 k 的二叉树（即共有 k+1 层），最多有 $2^{k+1}-1$ 个节点。**
    *   这是因为 $2^0 + 2^1 + ... + 2^k = 2^{k+1}-1$。
3.  **对任何一个二叉树，如果叶子节点数为 $n_0$，度为 2 的节点数为 $n_2$，则 $n_0 = n_2 + 1$。**
    *   这个性质很有趣，可以用来快速检查一个二叉树的结构。

### 6. 树和二叉树的存储表示（概念介绍）

*   **链式存储**：每个节点包含数据域和指向其子节点的指针（对于二叉树是左、右子节点指针）。这是最常用的方式。
*   **顺序存储**：使用数组来存储节点。这种方式主要适用于完全二叉树，因为完全二叉树的节点可以一一对应到数组的连续位置上，从而可以通过下标关系快速找到父节点和子节点。

    *   对于完全二叉树，如果节点 i 的下标从 1 开始：
        *   其父节点是 `i / 2`。
        *   其左子节点是 `2 * i`。
        *   其右子节点是 `2 * i + 1`。
    *   如果下标从 0 开始：
        *   其父节点是 `(i - 1) / 2`。
        *   其左子节点是 `2 * i + 1`。
        *   其右子节点是 `2 * i + 2`。

## 代码示例

我们将用 C++ 来定义二叉树的节点结构，并构建一个简单的二叉树。

```cpp
#include <bits/stdc++.h>
using namespace std;

// 定义二叉树的节点结构体
// 每个节点包含一个值 (int val)
// 和两个指针 (Node* left, Node* right)，分别指向左子节点和右子节点
struct Node {
    int v;         // 节点存储的值
    Node* l;       // 指向左子节点的指针
    Node* r;       // 指向右子节点的指针

    // 构造函数，方便创建新节点
    Node(int val) {
        v = val;
        l = nullptr; // 初始时，左右子节点都为空
        r = nullptr;
    }
};

// 函数：计算二叉树的高度
// 高度定义为从根节点到最远叶子节点的最长路径上的边数
// 空树高度为 -1，只有一个根节点的高度为 0
int getHeight(Node* root) {
    // 如果根节点为空，说明是空树，高度为 -1
    if (root == nullptr) {
        return -1;
    }
    // 递归计算左子树的高度
    int h_l = getHeight(root->l);
    // 递归计算右子树的高度
    int h_r = getHeight(root->r);

    // 当前节点的高度 = 左右子树中较高的那个高度 + 1 (加上当前节点到子节点的边)
    return max(h_l, h_r) + 1;
}

// 函数：计算二叉树的节点数量
int countNodes(Node* root) {
    // 如果根节点为空，节点数为 0
    if (root == nullptr) {
        return 0;
    }
    // 节点总数 = 1 (当前根节点) + 左子树节点数 + 右子树节点数
    return 1 + countNodes(root->l) + countNodes(root->r);
}

int main() {
    // 手动构建一个简单的二叉树
    //        1
    //       / \
    //      2   3
    //     / \
    //    4   5
    //   /
    //  6

    Node* n1 = new Node(1);
    Node* n2 = new Node(2);
    Node* n3 = new Node(3);
    Node* n4 = new Node(4);
    Node* n5 = new Node(5);
    Node* n6 = new Node(6);

    n1->l = n2; // 1 的左子节点是 2
    n1->r = n3; // 1 的右子节点是 3

    n2->l = n4; // 2 的左子节点是 4
    n2->r = n5; // 2 的右子节点是 5

    n4->l = n6; // 4 的左子节点是 6

    cout << "二叉树的高度 (根节点高度为0): " << getHeight(n1) << endl; // 预期输出 3
    cout << "二叉树的节点总数: " << countNodes(n1) << endl;     // 预期输出 6

    // 释放内存 (在实际程序中，需要遍历树来释放所有节点，这里为简化示例只释放根节点，实际操作需要更完善的内存管理)
    // 对于初学者，更重要的是理解概念和结构，内存释放可以后续深入学习
    // delete n1; // 这样做是不够的，因为它不会释放子节点

    return 0;
}
```

**代码解析**：
1.  **`struct Node`**：定义了二叉树节点的蓝图。每个节点存储一个整数 `v`（值），以及两个指针 `l` 和 `r`。`l` 指向左子节点，`r` 指向右子节点。如果某个子节点不存在，对应的指针就为 `nullptr`。
2.  **构造函数 `Node(int val)`**：当我们 `new Node(1)` 时，就会创建一个值为 1 的节点，并且它的左右子节点指针默认是空的。
3.  **`getHeight(Node* root)` 函数**：这是一个递归函数，用来计算树的高度。
    *   **基准情况**：如果 `root` 是 `nullptr`（空节点），说明这棵树不存在，它的高度定义为 -1。
    *   **递归步骤**：否则，它会分别计算左子树和右子树的高度，然后取两者中的最大值，再加 1（因为从当前节点到其子节点有一条边）。
4.  **`countNodes(Node* root)` 函数**：同样是递归函数，用来计算树中节点的总数。
    *   **基准情况**：如果 `root` 是 `nullptr`，节点数为 0。
    *   **递归步骤**：否则，节点总数等于 1（当前节点）加上左子树的节点数，再加上右子树的节点数。
5.  **`main` 函数**：我们手动创建了 6 个节点，并用指针把它们连接起来，形成了一个示例二叉树。然后调用 `getHeight` 和 `countNodes` 函数来验证。

    **解析**：树的根节点没有父节点，所以“每个节点都有且只有一个父节点”是错误的。

2.  **答案：B**
    **解析**：这是二叉树的一个重要性质。可以通过归纳法证明，或简单地考虑每个非叶子节点增加一个边，并最终连接到叶子节点。

3.  **答案：C**
    **解析**：深度为 k（根节点深度为0）的满二叉树共有 k+1 层。第 i 层有 $2^i$ 个节点。所以总节点数是 $2^0 + 2^1 + \dots + 2^k = 2^{k+1}-1$。

4.  **答案：否**
    **解析**：这不是完全二叉树。因为 C 节点的右子节点 E 存在，但 C 节点的左子节点缺失，不满足完全二叉树“最后一层节点靠左排列”的原则。

5.  **答案：1, 0**
    **解析**：
    *   节点 B 的深度：从根节点 A 到 B 经过 1 条边，所以深度是 1。
    *   节点 E 的高度：E 是叶子节点，叶子节点的高度定义为 0。

**基础题：二叉树的叶子节点计数**

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node {
    int v;
    Node* l;
    Node* r;

    Node(int val) {
        v = val;
        l = nullptr;
        r = nullptr;
    }
};

// 函数：计算二叉树的叶子节点数量
int countLeaves(Node* root) {
    // 如果根节点为空，没有叶子节点
    if (root == nullptr) {
        return 0;
    }
    // 如果当前节点是叶子节点 (左右子节点都为空)，则返回 1
    if (root->l == nullptr && root->r == nullptr) {
        return 1;
    }
    // 否则，递归计算左右子树的叶子节点数之和
    return countLeaves(root->l) + countLeaves(root->r);
}

int main() {
    // 构建示例二叉树
    Node* n1 = new Node(1);
    Node* n2 = new Node(2);
    Node* n3 = new Node(3);
    Node* n4 = new Node(4);
    Node* n5 = new Node(5);
    Node* n6 = new Node(6);

    n1->l = n2;
    n1->r = n3;
    n2->l = n4;
    n2->r = n5;
    n4->l = n6;

    cout << "叶子节点数量: " << countLeaves(n1) << endl; // 预期输出 3

    // 实际中需要更完善的内存释放
    return 0;
}
```

**进阶题：判断是否为满二叉树**

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node {
    int v;
    Node* l;
    Node* r;

    Node(int val) {
        v = val;
        l = nullptr;
        r = nullptr;
    }
};

// 辅助函数：计算二叉树的高度
int getHeight(Node* root) {
    if (root == nullptr) {
        return -1; // 空树高度为 -1
    }
    return max(getHeight(root->l), getHeight(root->r)) + 1;
}

// 辅助函数：计算二叉树的节点总数
int countNodes(Node* root) {
    if (root == nullptr) {
        return 0;
    }
    return 1 + countNodes(root->l) + countNodes(root->r);
}

// 函数：判断是否为满二叉树
bool isFullBinaryTree(Node* root) {
    // 空树也是满二叉树 (或根据具体定义，也可以认为不是，这里按节点数0的高度-1是满二叉树处理)
    // 但通常我们讨论的是非空树
    if (root == nullptr) {
        return true;
    }

    int h = getHeight(root); // 计算树的高度
    int n = countNodes(root); // 计算树的节点总数

    // 满二叉树的节点数 n = 2^(h+1) - 1
    // 这里使用位运算 1 << (h + 1) 来计算 2 的 (h+1) 次方
    return n == ((1 << (h + 1)) - 1);
}

int main() {
    // 示例 1: 满二叉树
    //        1
    //       / \
    //      2   3
    //     / \ / \
    //    4  5 6  7
    Node* m1 = new Node(1);
    Node* m2 = new Node(2);
    Node* m3 = new Node(3);
    Node* m4 = new Node(4);
    Node* m5 = new Node(5);
    Node* m6 = new Node(6);
    Node* m7 = new Node(7);

    m1->l = m2; m1->r = m3;
    m2->l = m4; m2->r = m5;
    m3->l = m6; m3->r = m7;

    cout << "示例 1 (满二叉树): " << (isFullBinaryTree(m1) ? "是" : "否") << endl; // 预期输出 是

    // 示例 2: 不是满二叉树 (但可能是完全二叉树)
    //        1
    //       / \
    //      2   3
    //     /
    //    4
    Node* c1 = new Node(1);
    Node* c2 = new Node(2);
    Node* c3 = new Node(3);
    Node* c4 = new Node(4);

    c1->l = c2; c1->r = c3;
    c2->l = c4;

    cout << "示例 2 (非满二叉树): " << (isFullBinaryTree(c1) ? "是" : "否") << endl; // 预期输出 否

    // 示例 3: 只有一个根节点 (也是满二叉树，高度0，节点1，1 = 2^(0+1)-1)
    Node* s1 = new Node(10);
    cout << "示例 3 (单个节点): " << (isFullBinaryTree(s1) ? "是" : "否") << endl; // 预期输出 是

    // 实际中需要更完善的内存释放
    return 0;
}
```
