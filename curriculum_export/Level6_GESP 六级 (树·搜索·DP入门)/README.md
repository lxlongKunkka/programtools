# GESP 六级 (树·搜索·DP入门)

## 教学目标
1. 掌握文件操作与流。
2. 理解并应用基础数据结构（栈、队列）。
3. 掌握基础的搜索算法。

## 核心知识点
- 文件IO：`freopen` 或 `fstream`。
- STL容器：`std::vector`, `std::stack`, `std::queue`。
- 搜索：深度优先搜索 (DFS) 基础，广度优先搜索 (BFS) 基础（迷宫问题）。

## 专题列表

1. **栈**
   > LIFO 结构、括号匹配、单调栈入门、表达式求值基础
2. **队列**
   > FIFO 结构、循环队列、双端队列 deque 的使用
3. **树结构**
4. **哈夫曼树**
   > 哈夫曼编码、最优二叉树的贪心构造与带权路径长度
5. **图论基础**
   > 介绍图的基本概念（有向图/无向图/权重）、邻接矩阵与邻接表两种存储方式，以及链式前向星的数组模拟实现，为 DFS/BFS 和最短路算法打好基础。
6. **DFS 深度优先搜索**
   > 系统学习 DFS 框架、连通块、路径枚举与剪枝
7. **BFS 广度优先搜索**
   > BFS 层序遍历、最短步数、迷宫最短路等经典题型
8. **动态规划基础**
   > 从记忆化搜索切入动态规划思想，掌握一维DP状态设计，再到经典线性DP模型：最长递增子序列(LIS)与最长公共子序列(LCS)，系统建立DP解题框架。
9. **背包 DP**
   > 0-1 背包、完全背包、多重背包及其变种
