## 深度知识点讲解

### 1. 栈是什么？（快速回顾）

我们已经知道，栈（Stack）是一种特殊的线性数据结构，它遵循“后进先出”（Last In, First Out，简称 LIFO）的原则。这就像我们前面说的叠盘子，最后放上去的盘子，总是最先被拿走。

栈有几个基本操作：
*   **入栈 (Push)**：将一个元素放到栈的顶部。
*   **出栈 (Pop)**：从栈的顶部移除一个元素。
*   **获取栈顶元素 (Top)**：查看栈顶部的元素，但不移除它。
*   **判断栈是否为空 (Empty)**：检查栈里有没有元素。
*   **获取栈的大小 (Size)**：查看栈中有多少个元素。

栈的实现通常用数组或者链表。在 GESP 考试中，我们通常使用**原生数组**来实现栈，并通过一个 `top` 变量来指示栈顶元素的位置。`top` 初始化为 `-1` 表示栈空。

### 2. 栈的应用思想：为什么是栈？

栈的 LIFO 特性让它在处理需要“回溯”或“逆序”操作的问题时显得尤为强大。
*   **顺序依赖性检查**：当我们需要检查一系列操作的顺序是否正确时，例如括号匹配，栈可以帮助我们“记住”之前遇到的某个状态，以便后续进行匹配检查。
*   **延迟处理**：在一些场景中，我们不能立即处理当前遇到的数据，需要先将它“保存”起来，等待后续条件满足时再处理，而处理的顺序往往是最近保存的优先。
*   **消除/抵消操作**：当某些操作可以互相抵消时，栈能有效地追踪最近的未抵消操作。

接下来，我们就用两个经典的例子来深入理解栈的这些应用思想。

### 3. 应用一：括号匹配

**问题描述：** 给定一个只包含 `(`, `)`, `[`, `]`, `{`, `}` 这六种字符的字符串，判断字符串中的括号是否有效。有效括号必须满足：
1.  左括号必须用相同类型的右括号闭合。
2.  左括号必须以正确的顺序闭合。
3.  空字符串也被认为是有效的。

**为什么用栈？**
想象一下，你正在读一句话，里面有很多括号。当你遇到一个左括号 `(` 时，你会“记住”它，并且期待在后面看到一个对应的右括号 `)`。如果再遇到一个左括号 `[`，你会先放下之前的 `(`，转而去期待 `]`。当你遇到一个右括号 `]` 时，你首先要检查的是**最近**遇到的那个左括号是不是 `[`。如果是，它们就匹配成功了，你可以把这两个括号“消掉”。如果不是，或者没有最近的左括号，那么就是错误的。

这种“最近遇到的左括号最先被匹配”的特点，是不是和栈的“后进先出”一模一样？
*   遇到左括号：把它“记住”，压入栈中。
*   遇到右括号：检查栈顶元素。
    *   如果栈是空的，说明没有左括号可以匹配，错误！
    *   如果栈顶元素能和当前右括号匹配，说明匹配成功，将栈顶元素弹出。
    *   如果栈顶元素不能和当前右括号匹配，说明类型不符，错误！

**生活类比：配对袜子**
想象你有一堆袜子，有左脚的，也有右脚的，颜色和款式各不相同。你的任务是把它们配成对。
*   你拿起一只左脚的红袜子，把它放在“待配对”的篮子里（**入栈**）。
*   你又拿起一只左脚的蓝袜子，也把它放在“待配对”的篮子里，放在红袜子的上面（**入栈**）。
*   现在你拿起一只右脚的蓝袜子。你要找和它配对的，你会先看篮子里**最上面**的那只（栈顶元素）。发现是左脚蓝袜子！太棒了，它们配成一对，你可以把它们从篮子里拿走（**出栈**）。
*   接着你拿起一只右脚的黄袜子。再看篮子顶部，发现是左脚红袜子。哦，它们不配！那么这堆袜子就是无效的。
*   最后，如果所有袜子都检查完了，篮子里还有袜子，说明有些左脚袜子没有找到搭档，也是无效的。

**核心思路：**
1.  遍历字符串中的每一个字符。
2.  如果字符是左括号（`(`、`[`、`{`），则将其压入栈中。
3.  如果字符是右括号（`)`、`]`、`}`）：
    *   首先检查栈是否为空。如果为空，说明没有对应的左括号，直接返回 `false`。
    *   如果栈不为空，则取出栈顶元素。
    *   检查栈顶元素与当前右括号是否匹配。例如，栈顶是 `(`，当前是 `)`，则匹配。
    *   如果匹配成功，则将栈顶元素弹出。
    *   如果匹配失败，则返回 `false`。
4.  字符串遍历结束后，检查栈是否为空。
    *   如果栈为空，说明所有左括号都找到了对应的右括号，返回 `true`。
    *   如果栈不为空，说明还有未匹配的左括号，返回 `false`。

### 4. 应用二：消除相邻重复项

**问题描述：** 给定一个字符串 `s`，反复删除相邻且相同的字符，直到不能再删除为止。返回最终的字符串。
例如：`s = "abbaca"`
1.  `"abbaca"` -> `bb` 相邻且相同，删除 -> `"aaca"`
2.  `"aaca"` -> `aa` 相邻且相同，删除 -> `"ca"`
3.  `"ca"` 没有相邻相同字符，停止。最终结果是 `"ca"`。

**为什么用栈？**
当我们在扫描字符串时，如果当前字符和我们已经“处理”过的字符中的**最后一个**字符相同，那么它们就应该被消除。这个“最后一个”字符，恰好就是栈顶元素。

**生活类比：堆积木**
想象你有一个长长的传送带，上面不断送来不同颜色的积木。你面前有一个架子，每次只能往架子最上面放一块积木。
*   当传送带送来一块**蓝色**积木时，你把它放到架子上（**入栈**）。
*   又送来一块**红色**积木，你放到蓝色积木上面（**入栈**）。
*   接着又送来一块**红色**积木。你发现它和架子最上面的那块积木（红色）颜色一样！那么这两块红积木就互相抵消了，你把架子最上面的红积木拿走，新来的这块也不放上去（**出栈**）。
*   如果送来一块**黄色**积木，和架子最上面的蓝色积木不同，就把它放上去（**入栈**）。

**核心思路：**
1.  创建一个空栈，用于存放最终不被消除的字符。
2.  遍历输入字符串 `s` 中的每一个字符 `c`。
3.  对于每个字符 `c`：
    *   如果栈不为空，并且栈顶元素与 `c` 相同，说明它们可以消除。执行 `pop` 操作，将栈顶元素移除。
    *   否则（栈为空，或者栈顶元素与 `c` 不同），将 `c` 压入栈中。
4.  遍历结束后，栈中剩下的元素就是最终的字符串。由于栈是 LIFO，我们需要将栈中的元素逆序取出，或者从栈底到栈顶构建字符串。

## 典型例题精讲

### 例题1：括号匹配

**题目描述：**
给定一个只包含字符 `'('`, `')'`, `'['`, `']'`, `'{'`, `'}'` 的字符串 `s`，判断输入字符串是否有效。
有效字符串需满足：
1.  左括号必须用相同类型的右括号闭合。
2.  左括号必须以正确的顺序闭合。
3.  空字符串可被认为是有效字符串。

**输入示例：**
`s = "([{}])"`

**输出示例：**
`true`

**思路分析：**
1.  我们需要一个栈来存储遇到的左括号。因为 GESP 6 级要求使用原生数组，所以我们定义一个 `char` 数组 `st` 和一个 `int` 变量 `top` 作为栈顶指针。`top = -1` 表示栈空。
2.  遍历输入字符串 `s` 中的每个字符 `c`。
3.  如果 `c` 是左括号（`(`、`[`、`{`），则将其 `push` 到栈中。
4.  如果 `c` 是右括号（`)`、`]`、`}`）：
    *   首先检查栈是否为空 (`top == -1`)。如果为空，说明没有左括号可以匹配，直接返回 `false`。
    *   如果栈不为空，我们检查栈顶元素 `st[top]` 与当前右括号 `c` 是否匹配。
        *   如果 `c` 是 `)`，则 `st[top]` 必须是 `(`。
        *   如果 `c` 是 `]`，则 `st[top]` 必须是 `[`。
        *   如果 `c` 是 `}`，则 `st[top]` 必须是 `{`。
    *   如果匹配成功，则执行 `pop` 操作 (`top--`)。
    *   如果匹配失败（类型不符），则直接返回 `false`。
5.  遍历完整个字符串后，最后检查栈是否为空 (`top == -1`)。如果为空，说明所有括号都成功匹配，返回 `true`。否则，说明还有未匹配的左括号，返回 `false`。

**代码实现：**

```cpp
#include <iostream>
#include <string>
#include <cstring> // For strlen

const int MAX_SIZE = 1005; // 栈的最大容量，根据题目字符串长度设定

// 栈的实现
char st[MAX_SIZE]; // 存储栈元素的数组
int t = -1;        // 栈顶指针，初始化为-1表示栈空

// 入栈操作
void push(char c) {
    if (t < MAX_SIZE - 1) { // 检查栈是否已满
        st[++t] = c;
    }
}

// 出栈操作
void pop() {
    if (t != -1) { // 检查栈是否为空
        t--;
    }
}

// 获取栈顶元素
char top() {
    if (t != -1) { // 检查栈是否为空
        return st[t];
    }
    return '\0'; // 栈空返回空字符
}

// 判断栈是否为空
bool empty() {
    return t == -1;
}

bool isValid(std::string s) {
    // 每次调用前重置栈，确保独立性
    t = -1; 

    for (char c : s) {
        if (c == '(' || c == '[' || c == '{') {
            push(c); // 遇到左括号，入栈
        } else { // 遇到右括号
            if (empty()) { // 栈为空，没有左括号可以匹配
                return false;
            }
            char p = top(); // 获取栈顶元素
            if ((c == ')' && p == '(') ||
                (c == ']' && p == '[') ||
                (c == '}' && p == '{')) {
                pop(); // 匹配成功，出栈
            } else { // 类型不匹配
                return false;
            }
        }
    }
    // 字符串遍历结束后，如果栈为空，说明所有括号都匹配成功
    return empty();
}

int main() {
    std::string s1 = "([{}])";
    std::string s2 = "({[})";
    std::string s3 = "{[]}";
    std::string s4 = "(()";
    std::string s5 = "";

    std::cout << s1 << " is valid: " << (isValid(s1) ? "true" : "false") << std::endl; // true
    std::cout << s2 << " is valid: " << (isValid(s2) ? "true" : "false") << std::endl; // false
    std::cout << s3 << " is valid: " << (isValid(s3) ? "true" : "false") << std::endl; // true
    std::cout << s4 << " is valid: " << (isValid(s4) ? "true" : "false") << std::endl; // false
    std::cout << s5 << " is valid: " << (isValid(s5) ? "true" : "false") << std::endl; // true

    return 0;
}
```

**复杂度分析：**
*   **时间复杂度：** 我们遍历了字符串 `s` 中的每个字符一次，每个字符最多进行一次入栈或出栈操作。栈的 `push`、`pop`、`top`、`empty` 操作都是 O(1) 的时间复杂度。因此，总的时间复杂度是 O(n)，其中 `n` 是字符串 `s` 的长度。
*   **空间复杂度：** 在最坏的情况下（例如 `((((((`），所有左括号都会被压入栈中，直到字符串结束。因此，栈的最大深度可以达到 `n`。所以，空间复杂度是 O(n)。

### 例题2：消除相邻重复项

**题目描述：**
给定一个字符串 `s`，反复删除相邻且相同的字符，直到不能再删除为止。返回最终的字符串。

**输入示例：**
`s = "abbaca"`

**输出示例：**
`"ca"`

**思路分析：**
1.  我们同样需要一个栈来存储最终不被消除的字符。使用原生数组 `st` 和 `top` 指针。
2.  遍历输入字符串 `s` 中的每个字符 `c`。
3.  对于每个字符 `c`：
    *   我们检查栈是否为空 (`empty()`)。
    *   如果栈不为空且栈顶元素 `top()` 与当前字符 `c` 相同，说明它们是相邻且重复的，应该被消除。执行 `pop()` 操作。
    *   否则（栈为空，或者栈顶元素与 `c` 不同），将当前字符 `c` 压入栈中 (`push(c)`)。
4.  遍历完整个字符串后，栈中剩下的元素就是最终的字符串。由于栈是 LIFO，栈底元素是最终字符串的第一个字符，栈顶元素是最后一个字符。我们需要将栈中的元素按从栈底到栈顶的顺序取出，或者将栈中元素复制到一个新的字符数组中，然后转换为字符串。

**代码实现：**

```cpp
#include <iostream>
#include <string>
#include <cstring> // For strlen

const int MAX_SIZE = 1005; // 栈的最大容量

// 栈的实现 (与例题1相同，为了代码完整性再次提供)
char st[MAX_SIZE];
int t = -1;

void push(char c) {
    if (t < MAX_SIZE - 1) {
        st[++t] = c;
    }
}

void pop() {
    if (t != -1) {
        t--;
    }
}

char top() {
    if (t != -1) {
        return st[t];
    }
    return '\0'; // 栈空返回空字符
}

bool empty() {
    return t == -1;
}

std::string removeDuplicates(std::string s) {
    // 每次调用前重置栈
    t = -1; 

    for (char c : s) {
        if (!empty() && top() == c) { // 如果栈不为空且栈顶元素与当前字符相同
            pop(); // 消除，栈顶元素出栈
        } else {
            push(c); // 否则，当前字符入栈
        }
    }

    // 栈中剩余的元素就是最终结果。
    // 由于栈是后进先出，我们需要按从栈底到栈顶的顺序构建字符串。
    // 最简单的方法是先将栈中元素复制到一个临时字符串/字符数组，然后反转。
    // 或者直接从栈底开始构建（但需要知道栈的实际大小）。
    // 这里我们直接从栈中取出，然后构建字符串。
    std::string result = "";
    for (int i = 0; i <= t; ++i) { // 从栈底到栈顶遍历
        result += st[i];
    }
    
    return result;
}

int main() {
    std::string s1 = "abbaca";
    std::string s2 = "azxxzy";
    std::string s3 = "qqq";
    std::string s4 = "abc";
    std::string s5 = "";

    std::cout << s1 << " -> " << removeDuplicates(s1) << std::endl; // abbaca -> ca
    std::cout << s2 << " -> " << removeDuplicates(s2) << std::endl; // azxxzy -> ay
    std::cout << s3 << " -> " << removeDuplicates(s3) << std::endl; // qqq -> q
    std::cout << s4 << " -> " << removeDuplicates(s4) << std::endl; // abc -> abc
    std::cout << s5 << " -> " << removeDuplicates(s5) << std::endl; // "" -> ""

    return 0;
}
```

**复杂度分析：**
*   **时间复杂度：** 我们遍历了字符串 `s` 中的每个字符一次，每个字符最多进行一次入栈或出栈操作。最后构建结果字符串时，也最多遍历一次栈中的元素。因此，总的时间复杂度是 O(n)，其中 `n` 是字符串 `s` 的长度。
*   **空间复杂度：** 在最坏的情况下（例如 `abcde`，没有重复项），所有字符都会被压入栈中。因此，栈的最大深度可以达到 `n`。所以，空间复杂度是 O(n)。

## 代码实现模板

这里提供一个基于原生数组实现的栈的通用模板，方便大家在解决问题时直接使用。

```cpp
#include <iostream> // 用于输入输出
#include <string>   // 用于字符串操作，如果需要
#include <cstring>  // 用于处理C风格字符串，例如strlen

// 定义栈的最大容量，根据题目数据范围调整
const int MAX_STACK_SIZE = 1005; 

// 栈的底层存储数组
char myStackData[MAX_STACK_SIZE]; 
// 栈顶指针，初始化为 -1 表示栈空
// top 指向栈顶元素，如果栈为空，则为 -1
int myStackTop = -1; 

// -------------------------------------------------------------------
// 栈的基本操作函数
// -------------------------------------------------------------------

// 1. 入栈操作 (Push)
// 将元素 c 压入栈中
void myPush(char c) {
    if (myStackTop < MAX_STACK_SIZE - 1) { // 检查栈是否已满
        myStackTop++;                  // 栈顶指针上移
        myStackData[myStackTop] = c;   // 将元素放入栈顶位置
    } else {
        // 栈已满，可以打印错误信息或抛出异常，但通常在竞赛中，我们会确保不会溢出
        // std::cerr << "Error: Stack overflow!" << std::endl;
    }
}

// 2. 出栈操作 (Pop)
// 从栈中移除栈顶元素
void myPop() {
    if (myStackTop != -1) { // 检查栈是否为空
        myStackTop--;       // 栈顶指针下移，表示栈顶元素被移除
    } else {
        // 栈已空，可以打印错误信息或抛出异常
        // std::cerr << "Error: Stack underflow!" << std::endl;
    }
}

// 3. 获取栈顶元素 (Top)
// 返回栈顶元素的值，但不移除
char myTop() {
    if (myStackTop != -1) { // 检查栈是否为空
        return myStackData[myStackTop];
    } else {
        // 栈已空，返回一个特殊值表示错误或空，例如 '\0'
        return '\0'; 
    }
}

// 4. 判断栈是否为空 (Empty)
// 返回 true 如果栈为空，否则返回 false
bool myEmpty() {
    return myStackTop == -1;
}

// 5. 获取栈中元素数量 (Size)
// 返回栈中元素的个数
int mySize() {
    return myStackTop + 1;
}

// -------------------------------------------------------------------
// 示例使用 (可选，实际应用中会封装到具体问题函数内)
// -------------------------------------------------------------------
int main() {
    std::cout << "Is stack empty? " << (myEmpty() ? "Yes" : "No") << std::endl; // Yes

    myPush('A');
    myPush('B');
    std::cout << "After pushing A, B. Top: " << myTop() << ", Size: " << mySize() << std::endl; // B, 2

    myPop();
    std::cout << "After popping. Top: " << myTop() << ", Size: " << mySize() << std::endl;     // A, 1

    myPush('C');
    std::cout << "After pushing C. Top: " << myTop() << ", Size: " << mySize() << std::endl;     // C, 2

    while (!myEmpty()) {
        std::cout << "Popping: " << myTop() << std::endl;
        myPop();
    }
    std::cout << "Is stack empty? " << (myEmpty() ? "Yes" : "No") << std::endl; // Yes

    return 0;
}
```

**使用说明：**
*   将 `myStackData`、`myStackTop` 和相关的 `myPush`、`myPop` 等函数复制到你的解决方案中。
*   根据题目要求，将 `char` 类型替换为 `int` 或其他需要存储的数据类型。
*   `MAX_STACK_SIZE` 需要根据题目给出的数据范围来设置，确保栈不会溢出。
*   在每次解决一个新问题时，如果栈是全局变量，记得在函数开头或测试用例之间重置 `myStackTop = -1;`，以避免上次测试的数据影响本次结果。

        *   当执行“重做”操作时，从“重做栈”弹出元素，并重新压入“操作栈”。
        *   这样，两个栈配合就能实现撤销和重做。

4.  **编程题 `isBalanced` 参考代码：**
    ```cpp
    #include <string>
    #include <cstring> // For strlen

    const int MAX_SIZE_B = 1005;
    char s_b[MAX_SIZE_B];
    int t_b = -1;

    void push_b(char c) { if (t_b < MAX_SIZE_B - 1) s_b[++t_b] = c; }
    void pop_b() { if (t_b != -1) t_b--; }
    bool empty_b() { return t_b == -1; }

    bool isBalanced(std::string s) {
        t_b = -1; // Reset stack
        for (char c : s) {
            if (c == '(') {
                push_b(c);
            } else if (c == ')') {
                if (empty_b()) return false;
                pop_b();
            }
        }
        return empty_b();
    ```

**能力提升答案：**

1.  **编程题 `removeAdjacent` 参考代码：** (与例题2代码相同)
    ```cpp
    #include <string>
    #include <cstring> // For strlen

    const int MAX_SIZE_RA = 1005;
    char s_ra[MAX_SIZE_RA];
    int t_ra = -1;

    void push_ra(char c) { if (t_ra < MAX_SIZE_RA - 1) s_ra[++t_ra] = c; }
    void pop_ra() { if (t_ra != -1) t_ra--; }
    char top_ra() { return s_ra[t_ra]; }
    bool empty_ra() { return t_ra == -1; }

    std::string removeAdjacent(std::string s) {
        t_ra = -1; // Reset stack
        for (char c : s) {
            if (!empty_ra() && top_ra() == c) {
                pop_ra();
            } else {
                push_ra(c);
            }
        }
        std::string result = "";
        for (int i = 0; i <= t_ra; ++i) {
            result += s_ra[i];
        }
        return result;
    }
    ```

2.  **编程题 `extendedIsValid` 参考代码：**
    ```cpp
    #include <string>
    #include <cstring> // For strlen

    const int MAX_SIZE_EI = 1005;
    char s_ei[MAX_SIZE_EI];
    int t_ei = -1;

    void push_ei(char c) { if (t_ei < MAX_SIZE_EI - 1) s_ei[++t_ei] = c; }
    void pop_ei() { if (t_ei != -1) t_ei--; }
    char top_ei() { return s_ei[t_ei]; }
    bool empty_ei() { return t_ei == -1; }

    bool extendedIsValid(std::string s) {
        t_ei = -1; // Reset stack
        for (char c : s) {
            if (c == '(' || c == '[' || c == '{') {
                push_ei(c);
            } else if (c == ')' || c == ']' || c == '}') {
                if (empty_ei()) return false;
                char p = top_ei();
                if ((c == ')' && p == '(') ||
                    (c == ']' && p == '[') ||
                    (c == '}' && p == '{')) {
                    pop_ei();
                } else {
                    return false;
                }
            }
            // 忽略其他字符（如小写英文字母），不做任何处理
        }
        return empty_ei();
    }
    ```

3.  **思考与实现：用两个栈实现队列**
    *   **enqueue (入队)**：直接将元素压入第一个栈 `s1`。
    *   **dequeue (出队)**：
        1.  如果第二个栈 `s2` 不为空，直接从 `s2` 弹出元素。
        2.  如果 `s2` 为空，则将 `s1` 中的所有元素依次弹出，并压入 `s2`。完成此操作后，`s1` 将为空，`s2` 中的元素顺序变为 `s1` 中元素的逆序（即 `s1` 的栈底元素现在是 `s2` 的栈顶）。
        3.  此时，如果 `s2` 仍然为空（意味着 `s1` 之前也为空），则队列为空，无法出队。
        4.  否则，从 `s2` 弹出栈顶元素。
    *   **top (获取队首元素)**：与 `dequeue` 类似，只是最后不弹出，只返回 `s2` 的栈顶元素。
    *   **empty (判断队列是否为空)**：当且仅当 `s1` 和 `s2` 都为空时，队列为空。
