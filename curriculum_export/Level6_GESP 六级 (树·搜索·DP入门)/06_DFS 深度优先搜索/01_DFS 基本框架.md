## 深度知识点讲解

### 1. DFS 是什么？

DFS，全称 Depth-First Search，深度优先搜索。它是一种用于遍历或搜索树或图的算法。它的核心思想是：沿着一个分支尽可能深地搜索，直到到达末端或不能再深入为止，然后回溯到上一个节点，继续搜索其他分支。

**生活类比：**
就像你在玩“一笔画”游戏，从一个点开始，选择一个方向，一直画下去，直到这条线不能再延伸（或者回到原点）。如果还有其他没画过的点，就回到上一个可以分叉的点，选择另一个方向继续画。

### 2. DFS 的工作原理与核心组件

DFS 算法的实现通常基于**递归**。想象一下，你站在一个节点上，面前有几条路：

1.  **标记访问**：首先，你得记住你已经来过这个节点了，不然你可能会原地打转，永远出不去。所以，我们需要一个 `visited` 数组来标记节点是否被访问过。
    *   `bool visited[MAXN];` // `MAXN` 是图的最大节点数。
2.  **处理当前节点**：你到达一个节点后，可以对它进行一些操作，比如打印它的名字、统计它、或者检查它是不是宝藏。
3.  **遍历邻居**：接着，你会查看所有与当前节点直接相连的邻居节点。
4.  **递归深入**：对于每一个**未被访问过**的邻居节点，你都会像施展魔法一样，瞬间“传送”到那个邻居节点，然后从那个邻居节点开始，重复上面的1、2、3步。这就是**递归**！
5.  **回溯**：当一个节点的所有邻居都被访问过（或者都已经被标记为“死胡同”），DFS 就会自动“返回”到调用它的上一个节点，继续探索那个节点的其他邻居。这个过程就叫做**回溯**。

**形象比喻：**
你是一只聪明的小蚂蚁，在复杂的蚁穴（图）里找食物。
1.  你进入一个房间（节点 `u`），首先在房间门口做个标记 (`visited[u] = true`)，表示你来过了。
2.  你在房间里找找看有没有食物（处理节点 `u`）。
3.  这个房间有几扇门通向其他房间（`u` 的邻居 `v`）。
4.  你选择一扇**没去过**的门 (`!visited[v]`)，直接钻进去。钻进去之后，你就变成了那个新房间里的小蚂蚁，重复上面1、2、3步（`dfs(v)` 递归调用）。
5.  如果你在一个房间里，所有门都通向你已经去过的房间，或者都是死路，那么你就会原路返回，爬出这个房间，回到你刚刚进来的那个房间（递归函数返回）。

### 3. 图的表示方法（邻接表）

在计算机中，我们如何表示迷宫的结构（图）呢？

*   **邻接矩阵 (Adjacency Matrix)**：用一个 `N x N` 的二维数组 `adj[u][v]` 来表示。如果 `adj[u][v] == 1`，表示 `u` 和 `v` 之间有边。这种方法简单直观，但当节点数 `N` 很大而边数 `M` 很少时，会浪费大量空间。
*   **邻接表 (Adjacency List)**：这是更常用的一种方法。对于每个节点 `u`，我们存储一个列表，里面存放所有与 `u` 相连的邻居节点。在 GESP 6 级中，我们通常使用**原生数组**来实现邻接表。

**原生数组实现邻接表：**
我们可以用两个数组来模拟：
*   `int g[MAXN][MAXN_DEGREE];`：`g[u]` 是一个数组，存储节点 `u` 的所有邻居。`MAXN_DEGREE` 是一个节点可能有的最大邻居数。
*   `int d[MAXN];`：`d[u]` 存储节点 `u` 实际有多少个邻居。

例如，节点 `u` 有邻居 `v1, v2, v3`。
那么 `g[u][0] = v1; g[u][1] = v2; g[u][2] = v3;` 并且 `d[u] = 3;`。
每次添加边 `(u, v)` 时，就 `g[u][d[u]++] = v;`。

### 4. DFS 的基本框架

```cpp
void dfs(int u)
{
    visited[u] = true; // 标记当前节点 u 已被访问
    // 在这里可以进行对节点 u 的操作，例如打印、计数、记录路径等
    // printf("%d ", u); // 示例：打印节点

    // 遍历所有与节点 u 相邻的节点 v
    // 注意：这里是伪代码，实际要根据邻接表的实现方式来遍历
    for (int i = 0; i < d[u]; ++i) // 遍历 u 的所有 d[u] 个邻居
    {
        int v = g[u][i]; // 获取 u 的第 i 个邻居

        // 如果邻居 v 没有被访问过
        if (!visited[v])
        {
            // 递归调用 DFS，继续深入探索 v
            dfs(v);
        }
    }
}
```

### 5. 常见误区与注意事项

*   **忘记 `visited` 数组**：这是最常见的错误！如果没有 `visited` 数组，DFS 可能会在图中无限循环，导致程序崩溃（栈溢出）或死循环。
*   **图的类型**：对于无向图，如果 `u` 和 `v` 之间有边，那么 `v` 也是 `u` 的邻居，`u` 也是 `v` 的邻居，所以需要在邻接表中双向添加。
*   **起始节点**：有些问题要求从特定节点开始 DFS，有些则要求遍历所有连通块，需要从每个未访问过的节点开始 DFS。
*   **栈溢出**：当图非常深（例如一条很长的链），或者节点数非常多时，递归深度可能超过系统限制，导致栈溢出。对于 GESP 6 级题目，通常不会遇到这种极端情况，但了解这个概念很重要。

## 典型例题精讲

### 例题一：图的深度遍历

**题目描述：**
给定一个无向图，包含 `N` 个节点和 `M` 条边。从给定的起始节点 `S` 开始，使用深度优先搜索（DFS）遍历所有可达节点，并按照访问顺序打印出来。

**输入格式：**
第一行包含两个整数 `N` 和 `M`，表示节点数和边数。
接下来 `M` 行，每行包含两个整数 `u` 和 `v`，表示节点 `u` 和 `v` 之间有一条边。
最后一行包含一个整数 `S`，表示起始节点。
（节点编号从 1 到 `N`）

**输出格式：**
一行，按照 DFS 访问顺序打印所有可达节点，节点之间用空格隔开。

**样例输入：**
```
5 4
1 2
1 3
2 4
3 5
1
```

**样例输出：**
```
1 2 4 3 5
```

**思路分析：**
1.  **图的表示**：由于 `N` 通常不会太大（例如 GESP 6 级可能在 100-1000 之间），我们使用原生数组实现的邻接表来存储图。`g[u][i]` 存储 `u` 的第 `i` 个邻居，`d[u]` 存储 `u` 的邻居数量。
2.  **访问标记**：使用 `bool visited[MAXN]` 数组来记录每个节点是否已被访问。
3.  **DFS 函数**：
    *   `dfs(u)`：表示当前正在访问节点 `u`。
    *   进入函数后，首先将 `visited[u]` 设为 `true`，并打印 `u`。
    *   然后，遍历 `u` 的所有邻居 `v`。
    *   如果 `v` 没有被访问过 (`!visited[v]`)，则递归调用 `dfs(v)`。
4.  **主函数**：读取 `N, M` 和所有边，构建邻接表。然后读取起始节点 `S`，调用 `dfs(S)` 开始遍历。

**代码实现：**

```cpp
#include <bits/stdc++.h> // 包含所有常用头文件

const int MAXN = 105; // 假设节点数最大100，这里稍微放大一点防止越界
int n, m;             // n: 节点数, m: 边数

// 图的邻接表表示
// g[u][i] 存储节点 u 的第 i 个邻居
// d[u] 存储节点 u 的度数（有多少个邻居）
int g[MAXN][MAXN]; // 每个节点最多有MAXN-1个邻居，实际存储可能更少
int d[MAXN];       // d[u] 表示节点 u 的当前邻居数量

// 访问标记数组，visited[u] 为 true 表示节点 u 已经被访问过
bool visited[MAXN];

// DFS 函数
// 参数 u: 当前正在访问的节点
void dfs(int u)
{
    visited[u] = true; // 标记当前节点 u 已被访问
    printf("%d ", u); // 访问节点 u 时，打印它

    // 遍历所有与节点 u 相邻的节点 v
    for (int i = 0; i < d[u]; ++i)
    {
        int v = g[u][i]; // 获取 u 的第 i 个邻居

        // 如果邻居 v 没有被访问过
        if (!visited[v])
        {
            // 递归调用 DFS，继续深入探索 v
            dfs(v);
        }
    }
}

int main()
{
    // 读取输入：节点数 n 和边数 m
    scanf("%d %d", &n, &m);

    // 读取 m 条边，构建图的邻接表
    for (int i = 0; i < m; ++i)
    {
        int u, v;
        scanf("%d %d", &u, &v);
        // 对于无向图，需要双向添加边
        g[u][d[u]++] = v; // u 的邻居添加 v
        g[v][d[v]++] = u; // v 的邻居添加 u
    }

    int s;
    scanf("%d", &s); // 读取起始节点

    dfs(s); // 从起始节点 S 开始 DFS

    printf("\n"); // 打印完所有节点后换行

    return 0;
}
```

**复杂度分析：**
*   **时间复杂度**：O(N + M)。每个节点 `u` 最多被访问一次，每次访问时会遍历它的所有邻居。对于无向图，每条边 `(u, v)` 会在 `u` 和 `v` 的邻接表中各出现一次，所以总共遍历的边数是 `2M`。因此，总时间复杂度是节点数和边数之和。
*   **空间复杂度**：O(N + M)。存储邻接表需要 O(N + M) 的空间，`visited` 数组需要 O(N) 的空间，递归调用栈最坏情况下（图是一条链）需要 O(N) 的空间。

### 例题二：连通块计数

**题目描述：**
给定一个无向图，包含 `N` 个节点和 `M` 条边。请计算这个图中有多少个独立的连通块。

**输入格式：**
第一行包含两个整数 `N` 和 `M`，表示节点数和边数。
接下来 `M` 行，每行包含两个整数 `u` 和 `v`，表示节点 `u` 和 `v` 之间有一条边。
（节点编号从 1 到 `N`）

**输出格式：**
一个整数，表示图中的连通块数量。

**样例输入：**
```
7 4
1 2
2 3
4 5
6 7
```

**样例输出：**
```
3
```

**思路分析：**
1.  **图的表示与访问标记**：同例题一。
2.  **DFS 函数**：`dfs(u)` 依然是用于从节点 `u` 开始，标记所有与 `u` 连通的节点。在这个问题中，`dfs` 函数不需要打印节点，只需要负责标记 `visited` 数组。
3.  **主函数**：
    *   初始化一个计数器 `count = 0`。
    *   从节点 1 遍历到节点 `N`。
    *   对于每个节点 `i`：
        *   如果 `i` **没有被访问过** (`!visited[i]`)，这说明我们发现了一个新的连通块！
        *   将 `count` 加 1。
        *   然后从 `i` 开始调用 `dfs(i)`，这会把所有与 `i` 连通的节点都标记为已访问。这样，在后续遍历中就不会重复计算这个连通块中的节点了。
    *   遍历结束后，`count` 就是连通块的总数。

**代码实现：**

```cpp
#include <bits/stdc++.h>

const int MAXN = 105;
int n, m;

int g[MAXN][MAXN];
int d[MAXN];

bool visited[MAXN];

// DFS 函数，这里只负责标记所有与 u 连通的节点
void dfs(int u)
{
    visited[u] = true; // 标记当前节点 u 已被访问

    for (int i = 0; i < d[u]; ++i)
    {
        int v = g[u][i];
        if (!visited[v])
        {
            dfs(v);
        }
    }
}

int main()
{
    scanf("%d %d", &n, &m);

    for (int i = 0; i < m; ++i)
    {
        int u, v;
        scanf("%d %d", &u, &v);
        g[u][d[u]++] = v;
        g[v][d[v]++] = u;
    }

    int count = 0; // 连通块计数器
    for (int i = 1; i <= n; ++i) // 遍历所有节点
    {
        if (!visited[i]) // 如果节点 i 未被访问过，说明它属于一个新的连通块
        {
            count++;     // 发现一个新的连通块，计数加一
            dfs(i);      // 从节点 i 开始 DFS，标记所有与 i 连通的节点
        }
    }

    printf("%d\n", count); // 打印连通块总数

    return 0;
}
```

**复杂度分析：**
*   **时间复杂度**：O(N + M)。外层循环遍历所有节点 `N` 次，每次遇到未访问节点时启动一次 DFS。所有 DFS 调用加起来，每个节点和每条边最多被访问一次。
*   **空间复杂度**：O(N + M)。同例题一。

## 代码实现模板

这是一个通用的 DFS 框架，包含了图的表示、`visited` 数组以及 DFS 函数的基本结构。你可以根据具体问题在这个框架上进行修改和填充。

```cpp

// 定义常量，例如最大节点数
const int MAXN = 1005; // 假设节点数最大1000。根据题目数据范围调整

// 全局变量，方便在不同函数中访问

// g[u][i] 存储节点 u 的第 i 个邻居

// 访问标记数组，visited[u] 为 true 表示节点 u 已经被访问过
bool visited[MAXN];

// DFS 函数
// 参数 u: 当前正在访问的节点
void dfs(int u)
{
    visited[u] = true; // 1. 标记当前节点 u 已被访问

    // 2. 在这里可以进行对节点 u 的操作，例如：
    //    - 打印节点编号：printf("%d ", u);
    //    - 统计可达节点数量：global_count++;
    //    - 检查是否是目标节点：if (u == target_node) found = true;

    // 3. 遍历所有与节点 u 相邻的节点 v
    for (int i = 0; i < d[u]; ++i) // 遍历 u 的所有 d[u] 个邻居
    {
        int v = g[u][i]; // 获取 u 的第 i 个邻居

        // 4. 如果邻居 v 没有被访问过
        if (!visited[v])
        {
            // 5. 递归调用 DFS，继续深入探索 v
            dfs(v);
        }
    }
    // 6. （可选）在所有邻居访问完毕，即将返回前，可以进行回溯时的操作
    //    例如：如果是在路径搜索中，可以从路径中移除当前节点
}

int main()
{
    // 在开始处理输入前，可以进行初始化操作
    // 对于全局变量，通常会自动初始化为0/false。
    // 如果有多组测试数据，或者需要清空，可以使用 memset 函数：
    // memset(visited, 0, sizeof(visited)); // 将所有 visited 标记为 false

    // 1. 读取输入：节点数 n 和边数 m
    scanf("%d %d", &n, &m);

    // 2. 读取 m 条边，构建图的邻接表
    for (int i = 0; i < m; ++i)
    {
        int u, v;
        scanf("%d %d", &u, &v);
        // 对于无向图，需要双向添加边
        g[u][d[u]++] = v; // 将 v 添加到 u 的邻居列表
        g[v][d[v]++] = u; // 将 u 添加到 v 的邻居列表
    }

    // 3. 根据题目要求，调用 DFS 函数
    //    - 如果要求从某个特定节点 S 开始遍历：
    //      int s_start_node = 1; // 假设从节点1开始
    //      dfs(s_start_node);

    //    - 如果要求遍历所有连通块（例如计算连通块数量）：
    //      int component_count = 0;
    //      for (int i = 1; i <= n; ++i) // 遍历所有节点
    //      {
    //          if (!visited[i]) // 如果节点 i 未被访问过，说明它属于一个新的连通块
```

**基础巩固 2 答案：**
```cpp
#include <bits/stdc++.h>

const int MAXN = 105;
int n, m;
int g[MAXN][MAXN];
int d[MAXN];
bool visited[MAXN];

void dfs(int u)
{
    visited[u] = true;
    for (int i = 0; i < d[u]; ++i)
    {
        int v = g[u][i];
        if (!visited[v])
        {
            dfs(v);
        }
    }
}

int main()
{
    scanf("%d %d", &n, &m);

    for (int i = 0; i < m; ++i)
    {
        int u, v;
        scanf("%d %d", &u, &v);
        g[u][d[u]++] = v;
        g[v][d[v]++] = u;
    }

    int a, b;
    scanf("%d %d", &a, &b); // 读取要判断的两个节点 A 和 B

    dfs(a); // 从节点 A 开始 DFS

    if (visited[b]) // 如果 B 在 DFS 结束后被标记为已访问，说明 A 和 B 连通
    {
        printf("Yes\n");
    }
    else
    {
        printf("No\n");
    }

    return 0;
}
```

```cpp
#include <bits/stdc++.h>

const int MAXR = 55; // 迷宫最大行数
const int MAXC = 55; // 迷宫最大列数

int r, c;
char maze[MAXR][MAXC];
bool visited[MAXR][MAXC];

int dx[] = {0, 0, 1, -1}; // 移动方向：右 左 下 上
int dy[] = {1, -1, 0, 0};

bool found_target = false; // 标记是否找到终点

void dfs(int x, int y)
{
    if (found_target) // 如果已经找到终点，提前退出
    {
        return;
    }
    if (x < 0 || x >= r || y < 0 || y >= c) // 越界
    {
        return;
    }
    if (maze[x][y] == '#') // 遇到墙壁
    {
        return;
    }
    if (visited[x][y]) // 已经访问过
    {
        return;
    }

    visited[x][y] = true; // 标记当前格子已访问

    if (maze[x][y] == 'E') // 找到终点
    {
        found_target = true;
        return;
    }

    // 尝试向四个方向移动
    for (int i = 0; i < 4; ++i)
    {
        int nx = x + dx[i];
        int ny = y + dy[i];
        dfs(nx, ny);
    }
}

int main()
{
    scanf("%d %d", &r, &c);

    int sx, sy; // 起点坐标
    for (int i = 0; i < r; ++i)
    {
        scanf("%s", maze[i]);
        for (int j = 0; j < c; ++j)
        {
            if (maze[i][j] == 'S')
            {
                sx = i;
                sy = j;
            }
        }
    }

    dfs(sx, sy); // 从起点开始 DFS

    if (found_target)
    {
        printf("Yes\n");
    }
    else
    {
        printf("No\n");
    }

    return 0;
}
```
