# 1005 - NOIP 2009 普及组初赛试题 - 题目与解析

1), 关于图灵机下面的说法哪个是正确的：
{{ select(1) }}
- A. 图灵机是世界上最早的电子计算机。
- B. 由于大量使用磁带操作，图灵机运行速度很慢。
- C. 图灵机是英国人图灵发明的，在二战中为破译德军的密码发挥了重要作用。
- D. 图灵机只是一个理论上的计算模型。

> **答案**: D
> **解析**: 1. 图灵机（Turing Machine）是由英国数学家艾伦·图灵（Alan Turing）在1936年提出的一个抽象数学模型。它不是一台具体的物理机器，而是一个用来定义“什么是可计算性”的理论工具。
2. 选项A错误：世界上第一台通用电子计算机通常认为是ENIAC（埃尼阿克），而图灵机只是理论模型。
3. 选项B错误：图灵机中的“磁带”是理论上的无限长纸带，用于存储符号。由于它是一个数学模型，讨论其物理运行速度没有意义。
4. 选项C错误：虽然艾伦·图灵在二战期间确实领导了破译德军Enigma密码的工作，并为此设计了名为“Bombe”的机电装置，但“图灵机”特指他在1936年论文中提出的理论计算模型，两者不能混为一谈。
5. 选项D正确：图灵机是计算机科学中最核心的理论模型，它证明了只要某种计算过程可以被分解为简单的指令，就可以用机械化的方式实现。这为后来通用计算机的出现奠定了理论基础。

2), 关于计算机内存下面的说法哪个是正确的：
{{ select(2) }}
- A. 随机存储器（RAM）的意思是当程序运行时，每次具体分配给程序的内存位置是随机而不确定的。
- B. 1 MB 1MB 内存通常是指 1 0 2 4 × 1 0 2 4 1024×1024 字节大小的内存。
- C. 计算机内存严格说来包括主存（memory）、高速缓存（cache）和寄存器（register）三个部分。
- D. 一般内存中的数据即使在断电的情况下也能保留 2 2 个小时以上。

> **答案**: B
> **解析**: A 选项错误：随机存储器（RAM）中的“随机”是指 CPU 在读取或写入数据时，访问内存中任何一个存储单元所需的时间都是相同的，与该单元所处的物理位置无关。这与程序运行时分配内存的方式是否“随机”没有关系。

B 选项正确：在计算机存储单位中，通常采用二进制计算。1 KB = 1024 字节（Bytes），1 MB = 1024 KB = 1024 × 1024 字节。

C 选项错误：在计算机体系结构中，通常所说的“内存”（Memory）或“内存储器”主要指主存（RAM）。寄存器（Register）位于 CPU 内部，高速缓存（Cache）是位于 CPU 和主存之间的缓冲区。虽然它们都属于存储层次结构，但严格意义上的“内存”通常不包含寄存器。

D 选项错误：内存（RAM）属于易失性存储器，其特点是必须在有电的情况下才能保存数据。一旦断电，内存中的数据会迅速消失，无法保留 2 小时。

3), 关于 BIOS 下面说法哪个是正确的：
{{ select(3) }}
- A. BIOS 是计算机基本输入输出系统软件的简称。
- B. BIOS 里包含了键盘、鼠标、声卡、显卡、打印机等常用输入输出设备的驱动程序。
- C. BIOS 一般由操作系统厂商来开发完成。
- D. BIOS 能提供各种文件拷贝、复制、删除以及目录维护等文件管理功能。

> **答案**: A
> **解析**: A 选项正确：BIOS 的全称是 Basic Input/Output System，中文译为“基本输入输出系统”。它是存放在主板 ROM（只读存储器）芯片中的一组程序，是计算机启动时加载的第一个软件。

B 选项错误：BIOS 仅包含维持计算机最基本运行的驱动程序（如键盘、简单的显示输出、软驱/硬盘的基本访问）。像声卡、高级显卡、打印机等复杂设备的完整驱动程序是由操作系统（如 Windows）负责加载和管理的。

C 选项错误：BIOS 是由主板厂商或专门的 BIOS 开发商（如 AMI、Phoenix、Award 等）编写的，而不是由操作系统厂商（如微软、苹果）开发的。

D 选项错误：文件的拷贝、复制、删除以及目录维护属于操作系统的“文件系统管理”功能，BIOS 并不具备这些功能。BIOS 的主要任务是硬件自检（POST）和引导操作系统。

4), 关于 CPU 下面哪个说法是正确的：
{{ select(4) }}
- A. CPU 全称为中央处理器（或中央处理单元）。
- B. CPU 可以直接运行汇编语言。
- C. 同样主频下，32 位的 CPU 比 16 位的 CPU 运行速度快一倍。
- D. CPU 最早是由 Intel 公司发明的。

> **答案**: A
> **解析**: A选项：正确。CPU 的全称是 Central Processing Unit，中文翻译为中央处理器或中央处理单元，它是计算机系统的运算和控制核心。

B选项：错误。CPU 只能直接识别和执行由 0 和 1 组成的机器语言（Machine Language）。汇编语言（Assembly Language）虽然是低级语言，但仍需要通过汇编过程（Assembler）转换成机器语言后，CPU 才能运行。

C选项：错误。CPU 的位数（如 16 位、32 位、64 位）是指 CPU 一次能处理的数据宽度（字长）。虽然在处理大数值运算时，位数高的 CPU 效率更高，但计算机的整体运行速度受到主频、架构、缓存、指令集优化等多种复杂因素影响，并不是简单的倍数关系。

D选项：错误。Intel 公司在 1971 年研制出了世界上第一款微处理器（Microprocessor）Intel 4004。但在微处理器出现之前，CPU 的概念就已经存在，早期的 CPU 是由大量的电子管或晶体管组成的巨大电路，并非由 Intel 发明。

5), 关于 ASCII，下面哪个说法是正确的：
{{ select(5) }}
- A. ASCII 码就是键盘上所有键的唯一编码。
- B. 一个 ASCII 码使用一个字节的内存空间就能够存放。
- C. 最新扩展的 ASCII 编码方案包含了汉字和其他欧洲语言的编码。
- D. ASCII 码是英国人主持制定并推广使用的。

> **答案**: B
> **解析**: 1. ASCII（American Standard Code for Information Interchange，美国信息交换标准代码）是基于拉丁字母的一套电脑编码系统。
2. 标准 ASCII 码使用 7 位二进制数来表示，范围是 0-127，共 128 个字符。扩展 ASCII 码使用 8 位二进制数，范围是 0-255。
3. 计算机中最小的存储单位是字节（Byte），1 个字节等于 8 位（bit）。因此，无论是标准还是扩展的 ASCII 码，都可以用一个字节的内存空间来存放。
4. 选项 A 错误：键盘上的许多功能键（如 F1-F12）、控制键（如 Shift、Alt、Ctrl）以及方向键等并没有对应的 ASCII 码。
5. 选项 C 错误：ASCII 码主要包含英文字母、数字、标点符号和控制字符。汉字字符数量庞大，需要使用专门的编码方案（如 GBK、Unicode/UTF-8 等），一个字节的 ASCII 编码无法容纳汉字。
6. 选项 D 错误：从名称中的“American”可以看出，ASCII 是由美国国家标准学会（ANSI）制定并推广使用的，而不是英国。

6), 下列软件中不是计算机操作系统的是：
{{ select(6) }}
- A. Windows
- B. Linux
- C. OS/2
- D. WPS

> **答案**: D
> **解析**: 1. 操作系统（Operating System, OS）是管理计算机硬件与软件资源的计算机程序，是计算机系统的核心和基石。
2. 选项 A：Windows 是由微软公司开发的图形化操作系统，是目前全球个人电脑上使用最广泛的操作系统。
3. 选项 B：Linux 是一种开源的类 Unix 操作系统，广泛应用于服务器、超级计算机、嵌入式设备以及个人电脑。
4. 选项 C：OS/2 是由 IBM 和微软公司早期共同开发（后期由 IBM 独立开发）的一种操作系统。
5. 选项 D：WPS（Word Processing System）是由金山软件公司开发的一款办公软件套装，包含文字处理、表格、演示文稿等功能。它属于应用软件，必须运行在操作系统（如 Windows 或 Linux）之上，其本身并不是操作系统。

因此，正确答案是 D。

7), 关于互联网，下面的说法哪一个是正确的：
{{ select(7) }}
- A. 新一代互联网使用的 IPv6 标准是 IPv5 标准的升级与补充。
- B. 互联网的入网主机如果有了域名就不再需要 IP 地址。
- C. 互联网的基础协议为 TCP/IP 协议。
- D. 互联网上所有可下载的软件及数据资源都是可以合法免费使用的。

> **答案**: C
> **解析**: A 选项错误：IPv6 是为了解决 IPv4 地址枯竭问题而设计的下一代互联网协议。IPv5 是一个实验性的流传输协议（ST-II），从未被广泛使用，IPv6 并不是 IPv5 的升级或补充。

B 选项错误：域名（Domain Name）是为了方便人类记忆而设计的，而计算机在互联网上通信必须依靠 IP 地址。DNS（域名系统）的作用就是将域名解析为 IP 地址。因此，即使有了域名，主机依然需要 IP 地址才能联网。

C 选项正确：TCP/IP（传输控制协议/网际协议）是互联网最基础、最核心的协议，它定义了电子设备如何连入因特网以及数据在它们之间传输的标准。

D 选项错误：互联网上的资源受到知识产权法律的保护。许多软件和数据是收费的或有版权限制的，未经授权下载或使用盗版资源是非法行为。

8), 关于 HTML 下面哪种说法是正确的：
{{ select(8) }}
- A. HTML 实现了文本、图形、声音乃至视频信息的统一编码。
- B. HTML 全称为超文本标记语言。

- D. HTML 也是一种高级程序设计语言。

> **解析**: 本题考查对 HTML（超文本标记语言）基本概念的理解。

D 选项错误：HTML 属于“标记语言”，它通过标签来描述数据的外观和结构，不具备编程语言的核心特征（如复杂的逻辑运算、循环控制、变量作用域等）。因此，它不被视为“高级程序设计语言”。常见的程序设计语言包括 C++、Java、Python 等。

9), 关于程序设计语言，下面哪个说法是正确的：
{{ select(9) }}
- A. 加了注释的程序一般会比同样的没有加注释的程序运行速度慢。
- B. 高级语言开发的程序不能使用在低层次的硬件系统如：自控机床或低端手机上。
- C. 高级语言相对于低级语言更容易实现跨平台的移植。
- D. 以上说法都不对。

> **答案**: C
> **解析**: A选项错误：注释是写给程序员阅读的，用于提高代码的可读性。在编译型语言（如C++）中，编译器在预处理阶段就会删除所有注释；在解释型语言（如Python）中，解释器也会忽略注释。因此，注释不会影响程序的运行速度。

B选项错误：高级语言（特别是C和C++）是嵌入式开发、工业控制（如自控机床）和移动设备开发的主流语言。只要目标硬件有对应的编译器或运行环境，高级语言就可以在其上运行。

C选项正确：低级语言（如汇编语言）直接依赖于特定CPU的指令集，换一个硬件平台通常需要重写。而高级语言抽象化了硬件细节，同一份源代码通常只需要在不同平台上重新编译（如C++）或直接在不同平台的虚拟机/解释器上运行（如Java、Python），即可实现跨平台移植。

D选项错误：因为C选项是正确的。

10), 已知大写字母 A 的 ASCII 编码为 65 （10 进制），则大写字母 J 的 10 进制 ASCII 编码为：
{{ select(10) }}
- A. 71
- B. 72
- C. 73
- D. 以上都不是

> **答案**: D
> **解析**: 1. 在 ASCII 编码体系中，英文字母（包括大写和小写）都是按照字母表的顺序连续排列的。
2. 已知大写字母 A 的 ASCII 码是 65。
3. 我们可以通过数数或者计算偏移量来得出 J 的编码：
   - A 是第 1 个字母，编码为 65。
   - B 是第 2 个字母，编码为 66。
   - C 是第 3 个字母，编码为 67。
   - D 是第 4 个字母，编码为 68。
   - E 是第 5 个字母，编码为 69。
   - F 是第 6 个字母，编码为 70。
   - G 是第 7 个字母，编码为 71。
   - H 是第 8 个字母，编码为 72。
   - I 是第 9 个字母，编码为 73。
   - J 是第 10 个字母，编码为 74。
4. 计算公式：J 的 ASCII 码 = A 的 ASCII 码 + (J 的字母序号 - A 的字母序号) = 65 + (10 - 1) = 65 + 9 = 74。
5. 选项 A (71)、B (72)、C (73) 均不正确，因此正确答案为 D。

11), 十进制小数 125.125 对应的 8 进制数是
{{ select(11) }}
- A. 100.1
- B. 175.175
- C. 175.1
- D. 100.175

> **答案**: C
> **解析**: 将十进制数转换为八进制数，需要将整数部分和小数部分分别进行转换：

1. 整数部分转换（125）：采用“除8取余法”
   - 125 ÷ 8 = 15，余数为 5
   - 15 ÷ 8 = 1，余数为 7
   - 1 ÷ 8 = 0，余数为 1
   将余数从下往上（逆序）排列，得到整数部分为 175。

2. 小数部分转换（0.125）：采用“乘8取整法”
   - 0.125 × 8 = 1.0
   取出整数部分 1，此时小数部分已变为 0，转换结束。
   得到小数部分为 .1。

3. 合并结果：
   将整数部分和小数部分合并，得到 175.1。

因此，十进制数 125.125 对应的八进制数是 175.1。正确选项为 C。

12), 有六个元素 FEDCBA 从左至右依次顺序进栈，在进栈过程中会有元素被弹出栈。问下列哪一个不可能是合法的出栈序列？
{{ select(12) }}
- A. EDCFAB EDCFAB
- B. DECABF DECABF
- C. CDFEBA CDFEBA
- D. BCDAEF BCDAEF

> **答案**: C
> **解析**: 本题考查栈（Stack）的基本性质，即“后进先出”（LIFO, Last In First Out）。

进栈顺序为：F, E, D, C, B, A。

我们逐一分析各选项的合法性：

A. EDCFAB：
1. F, E 进栈，弹出 E（输出 E）；
2. D 进栈，弹出 D（输出 D）；
3. C 进栈，弹出 C（输出 C）；
4. 弹出 F（输出 F）；
5. B, A 进栈，弹出 A（输出 A）；
6. 弹出 B（输出 B）。
序列为 EDCFAB，合法。

B. DECABF：
1. F, E, D 进栈，弹出 D（输出 D）；
2. 弹出 E（输出 E）；
3. C 进栈，弹出 C（输出 C）；
4. B, A 进栈，弹出 A（输出 A）；
5. 弹出 B（输出 B）；
6. 弹出 F（输出 F）。
序列为 DECABF，合法。

C. CDFEBA：
1. F, E, D, C 进栈，弹出 C（输出 C）；
2. 弹出 D（输出 D）；
3. 此时栈中从栈底到栈顶的元素依次为 [F, E]。根据后进先出原则，下一个出栈的元素必须是 E，而不可能是 F。
4. 选项 C 中 D 之后紧跟的是 F，这违反了栈的操作规则。
因此，CDFEBA 是不合法的。

D. BCDAEF：
1. F, E, D, C, B 进栈，弹出 B（输出 B）；
2. 弹出 C（输出 C）；
3. 弹出 D（输出 D）；
4. A 进栈，弹出 A（输出 A）；
5. 弹出 E（输出 E）；
6. 弹出 F（输出 F）。
序列为 BCDAEF，合法。

综上所述，只有 C 选项是不可能的出栈序列。

13), 表达式a*(b+c)-d的后缀表达式是：
{{ select(13) }}
- A. abcd*+-
- B. abc+*d-
- C. abc*+d-
- D. +*abcd

> **答案**: B
> **解析**: 将中缀表达式转换为后缀表达式（逆波兰表达式）的步骤如下：

1. 按照运算符的优先级对表达式进行加括号：
   原式：a * (b + c) - d
   第一步（括号内）：(b + c)
   第二步（乘法）：(a * (b + c))
   第三步（减法）：((a * (b + c)) - d)

2. 将运算符移动到对应括号的后面：
   ((a (b c)+)* d)-

3. 去掉所有括号，得到后缀表达式：
   abc+*d-

具体逻辑分析：
- 首先处理括号内的 `b + c`，得到 `bc+`。
- 接着处理乘法 `a * (bc+)`，将 `*` 放在操作数后面，得到 `abc+*`。
- 最后处理减法 `(abc+*) - d`，将 `-` 放在操作数后面，得到 `abc+*d-`。

因此，正确选项是 B。

14), 一个包含 n 个分支结点（非叶结点）的非空二叉树，它的叶结点数目最多为：
{{ select(14) }}
- A. 2 𝑛 + 1 2n+1
- B. 2 𝑛 − 1 2n−1
- C. 𝑛 − 1 n−1
- D. 𝑛 + 1 n+1

> **答案**: D
> **解析**: 在二叉树中，结点的度数是指该结点拥有的子树数量。我们可以根据二叉树的性质来推导叶子结点（度为0的结点）与分支结点（非叶结点）之间的关系：

1. 设 $n_0$ 为叶子结点的个数。
2. 设 $n_1$ 为度为1的结点个数。
3. 设 $n_2$ 为度为2的结点个数。
4. 根据题目定义，分支结点（非叶结点）的个数为 $n$，因此有：$n = n_1 + n_2$。
5. 根据二叉树的性质，叶子结点数 $n_0$ 与度为2的结点数 $n_2$ 之间存在固定关系：$n_0 = n_2 + 1$。
   （推导过程：总结点数 $N = n_0 + n_1 + n_2$；总边数 $E = N - 1$；同时 $E = 0 \times n_0 + 1 \times n_1 + 2 \times n_2$。联立方程可得 $n_0 = n_2 + 1$）。
6. 题目要求叶子结点 $n_0$ 的最大值。根据公式 $n_0 = n_2 + 1$，要使 $n_0$ 最大，必须使 $n_2$ 最大。
7. 在 $n = n_1 + n_2$ 的约束下，当 $n_1 = 0$（即没有度为1的结点）时，$n_2$ 取得最大值 $n$。
8. 将 $n_2 = n$ 代入公式，得到 $n_0 = n + 1$。

因此，一个包含 $n$ 个分支结点的非空二叉树，其叶子结点数目最多为 $n + 1$。

15), 快速排序最坏情况下的算法时间复杂度为：
{{ select(15) }}
- A. 𝑂 ( log ⁡ 2 𝑛 ) O(log 2 ​ n)
- B. 𝑂 ( 𝑛 ) O(n)
- C. 𝑂 ( 𝑛 log ⁡ 2 𝑛 ) O(nlog 2 ​ n)
- D. 𝑂 ( 𝑛 2 ) O(n 2 )

> **答案**: D
> **解析**: 快速排序（Quicksort）是一种基于分治思想的排序算法。其时间复杂度取决于基准值（pivot）的选择以及数据的初始状态：

1. 平均情况与最佳情况：如果每次划分都能将数组分成两个规模相当的子问题，其递归树的高度为 log n，每层处理的时间复杂度为 O(n)，因此平均和最佳时间复杂度为 O(n log n)。

2. 最坏情况：当每次选择的基准值都是当前序列中的最大值或最小值时（例如：数组已经是有序的，且每次都选择第一个或最后一个元素作为基准），会导致划分极度不均匀。此时，一个子问题包含 n-1 个元素，另一个子问题包含 0 个元素。
   - 递归深度将达到 n 层。
   - 第一层需要比较 n-1 次，第二层需要 n-2 次，以此类推。
   - 总的比较次数为 (n-1) + (n-2) + ... + 1 = n(n-1)/2。
   - 因此，最坏情况下的时间复杂度为 O(n²)。

综上所述，正确答案为 D。

17), 排序算法是稳定的意思是关键码相同的记录排序前后相对位置不发生改变，下列哪种排序算法是不稳定的：
{{ select(17) }}
- A. 冒泡排序
- B. 插入排序
- C. 归并排序
- D. 快速排序

> **答案**: D
> **解析**: 排序算法的稳定性是指：在待排序的记录序列中，存在多个具有相同关键字的记录，若经过排序后，这些记录的相对次序保持不变，则称这种排序算法是稳定的；否则称为不稳定的。

1. 冒泡排序 (Bubble Sort)：通过相邻元素的比较和交换来排序。当两个元素相等时，不进行交换，因此相同元素的相对位置不会改变，是稳定的。
2. 插入排序 (Insertion Sort)：将未排序的元素插入到已排序序列的适当位置。在遇到相等元素时，将其插入在相等元素的后面，相对位置不变，是稳定的。
3. 归并排序 (Merge Sort)：采用分治法，在合并（Merge）两个有序子序列的过程中，如果遇到相等的元素，通常规定优先取左侧子序列中的元素，从而保证相对位置不变，是稳定的。
4. 快速排序 (Quick Sort)：在分区（Partition）过程中，通过基准值（Pivot）将数组分为两部分，涉及非相邻元素的交换。这种跨越式的交换极有可能改变相同关键字记录的原始相对顺序。例如，序列 [5, 3, 3*, 2]，在某些分区实现下，第一个 3 可能会被交换到第二个 3* 的后面。因此，快速排序是不稳定的。

常见的排序算法稳定性总结：
- 稳定：冒泡排序、插入排序、归并排序、基数排序。
- 不稳定：快速排序、堆排序、选择排序、希尔排序。

18), 已知 n 个顶点的有向图，若该图是强连通的（从所有顶点都存在路径到达其他顶点），则该图中最少有多少条有向边？
{{ select(18) }}
- A. 𝑛 n
- B. 𝑛 + 1 n+1
- C. 𝑛 − 1 n−1
- D. 𝑛 ( 𝑛 − 1 ) n(n−1)

> **答案**: A
> **解析**: 1. 强连通图的定义：在有向图中，如果对于任意一对顶点 (u, v)，都存在从 u 到 v 的路径，同时也存在从 v 到 u 的路径，则称该图是强连通的。

2. 构造最少边数的情况：
   - 要使 n 个顶点的有向图强连通，最节省边的方法是构造一个包含所有顶点的单向环路。
   - 例如，顶点序列为 v1, v2, ..., vn。我们连接有向边：v1 -> v2, v2 -> v3, ..., vn-1 -> v n, vn -> v1。
   - 在这个环形结构中，从任何一个顶点出发，都可以沿着环到达其他任意顶点。
   - 此时边的总数正好是 n 条。

3. 为什么 n-1 条边不行：
   - 如果一个有向图只有 n-1 条边且要连接 n 个顶点，在不考虑边方向的情况下（即作为无向图），它最多是一棵树。
   - 在有向树中，必然存在至少一个入度为 0 的顶点（无法从其他点到达）或者至少一个出度为 0 的顶点（无法到达其他点）。
   - 只要存在这样的点，图就无法满足“强连通”的定义。因此，n-1 条边不足以构成强连通图。

4. 总结：
   - 无向连通图最少需要 n-1 条边（生成树）。
   - 有向强连通图最少需要 n 条边（单循环回路）。
   - 有向完全图（每对顶点间都有双向边）则需要 n(n-1) 条边。

因此，正确答案是 A。
