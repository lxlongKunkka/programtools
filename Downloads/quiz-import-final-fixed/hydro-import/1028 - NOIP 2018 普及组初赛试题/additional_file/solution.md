1), 以下哪一种设备属于输出设备
{{ select(1) }}
- A. 扫描仪
- B. 键盘
- C. 鼠标
- D. 打印机

> **答案**: D
> **解析**: 在计算机硬件系统中，设备通常分为输入设备和输出设备：

1. 输入设备（Input Device）：用于将外部信息（如文字、图像、声音、指令等）输入到计算机中。常见的输入设备包括键盘、鼠标、扫描仪、摄像头、麦克风等。
2. 输出设备（Output Device）：用于将计算机处理后的结果以人类或其他设备能够识别的形式表现出来。常见的输出设备包括显示器、打印机、音箱、投影仪等。

选项分析：
A. 扫描仪：将纸质文档或照片转换为数字信号存入电脑，属于输入设备。
B. 键盘：用户通过按键向电脑输入字符和指令，属于输入设备。
C. 鼠标：用户通过移动和点击向电脑输入位置信息和操作指令，属于输入设备。
D. 打印机：将电脑中的电子数据转化为纸质文档，属于输出设备。

因此，正确答案是 D。

2), 下列四个不同进制的数中，与其它三项数值上不相等的是
{{ select(2) }}
- A. ( 2 6 9 ) 1 6 (269) 16 ​
- B. ( 6 1 7 ) 1 0 (617) 10 ​
- C. ( 1 1 5 1 ) 8 (1151) 8 ​
- D. ( 1 0 0 1 1 0 1 0 1 1 ) 2 (1001101011) 2 ​

> **答案**: D
> **解析**: 为了比较这四个不同进制的数，我们可以将它们统一转换为十进制数值进行对比：

1.  **选项 A：$(269)_{16}$**
    转换公式：$2 \times 16^2 + 6 \times 16^1 + 9 \times 16^0$
    计算过程：$2 \times 256 + 6 \times 16 + 9 = 512 + 96 + 9 = 617$

2.  **选项 B：$(617)_{10}$**
    该数值已经是十进制，结果为 $617$。

3.  **选项 C：$(1151)_8$**
    转换公式：$1 \times 8^3 + 1 \times 8^2 + 5 \times 8^1 + 1 \times 8^0$
    计算过程：$1 \times 512 + 1 \times 64 + 5 \times 8 + 1 = 512 + 64 + 40 + 1 = 617$

4.  **选项 D：$(1001101011)_2$**
    转换公式：$1 \times 2^9 + 0 \times 2^8 + 0 \times 2^7 + 1 \times 2^6 + 1 \times 2^5 + 0 \times 2^4 + 1 \times 2^3 + 0 \times 2^2 + 1 \times 2^1 + 1 \times 2^0$
    计算过程：$512 + 64 + 32 + 8 + 2 + 1 = 619$
    （或者使用快速转换法：将其按四位一组转换为十六进制，从右往左为 $1011 \rightarrow B$，$0110 \rightarrow 6$，$0010 \rightarrow 2$，即 $(26B)_{16}$。已知 $(269)_{16} = 617$，则 $(26B)_{16} = 617 + 2 = 619$）

综上所述，A、B、C 三项的数值均为 617，而 D 项的数值为 619。因此，D 项与其他三项不相等。

A 选项：1000 字节，数值错误。
B 选项：1024 字节，这等于 1 KB。

D 选项：1024 × 1024 字节，符合计算机科学中 1 MB 的标准定义。

因此，正确答案是 D。

6), 如果开始时计算机处于小写输入状态，现在有一只小老鼠反复按照 CapsLock、字母键 A、字母键 S、字母键 D、字母键 F 的顺序循环按键，即 CapsLock、A、S、D、F、CapsLock、A、S、D、F、……，屏幕上输出的第 81 个字符是字母（ ）
{{ select(6) }}
- A. A
- B. S
- C. D
- D. a

> **答案**: A
> **解析**: 1. 分析按键序列：小老鼠按照“CapsLock、A、S、D、F”五个键为一个周期进行循环。
2. 分析屏幕输出：
   - CapsLock 键的作用是切换大小写状态，它本身不会在屏幕上产生字符输出。
   - A、S、D、F 这四个键每按一次都会在屏幕上产生一个字符输出。
   - 因此，每一个循环周期（5次按键）会在屏幕上产生 4 个字符。
3. 计算第 81 个字符所属的周期：
   - 81 ÷ 4 = 20 …… 1。
   - 这意味着第 81 个字符是第 21 个循环周期中产生的第 1 个字符。
4. 确定大小写状态：
   - 初始状态为小写。
   - 第 1 次循环：按下 CapsLock，状态变为大写。输出 A, S, D, F（第 1-4 个字符）。
   - 第 2 次循环：按下 CapsLock，状态变为小写。输出 a, s, d, f（第 5-8 个字符）。
   - 规律：奇数次循环时状态为大写，偶数次循环时状态为小写。
   - 第 21 次循环是奇数次，因此此时的状态是大写。
5. 确定具体字符：
   - 第 21 个循环的按键顺序是：CapsLock（切换到大写）、A、S、D、F。
   - 该周期的第 1 个输出字符是由字母键 A 产生的。
   - 在大写状态下，按下 A 键输出的字符是 'A'。

综上所述，屏幕上输出的第 81 个字符是 A。

7), 根节点深度为 0，一棵深度为 h 的满k(k>1)叉树，即除最后一层无任何子节点外，每一层上的所有结点都有 k 个子结点的树，共有（ ）个结点。
{{ select(7) }}
- A. 𝑘 ℎ + 1 − 1 𝑘 − 1 k−1 k h+1 −1 ​
- B. 𝑘 ℎ − 1 k h−1
- C. 𝑘 ℎ k h
- D. 𝑘 ℎ − 1 𝑘 − 1 k−1 k h−1 ​

> **答案**: A
> **解析**: 1. 分析每一层的节点数：
   - 题目规定根节点的深度为 0，因此第 0 层有 $k^0 = 1$ 个节点。
   - 根据满 $k$ 叉树的定义，每一层（除最后一层外）的节点都有 $k$ 个子节点。
   - 第 1 层有 $k^1 = k$ 个节点。
   - 第 2 层有 $k^2$ 个节点。
   - 依此类推，第 $i$ 层有 $k^i$ 个节点。
   - 最后一层（深度为 $h$）有 $k^h$ 个节点。

2. 计算总节点数：
   - 总节点数 $S$ 是从第 0 层到第 $h$ 层所有节点数的总和：
     $S = k^0 + k^1 + k^2 + \dots + k^h$
   - 这是一个等比数列求和问题，其中：
     - 首项 $a_1 = 1$
     - 公比 $q = k$
     - 项数 $n = h + 1$（因为从 0 到 $h$ 共有 $h+1$ 个数）

3. 应用等比数列求和公式：
   - 公式为：$S_n = \frac{a_1(q^n - 1)}{q - 1}$
   - 代入参数：$S = \frac{1 \cdot (k^{h+1} - 1)}{k - 1} = \frac{k^{h+1} - 1}{k - 1}$

4. 验证：
   - 若 $h=0$，树只有根节点，总数为 $\frac{k^1-1}{k-1} = 1$，符合题意。
   - 若 $h=1$，树有根节点和 $k$ 个子节点，总数为 $\frac{k^2-1}{k-1} = \frac{(k-1)(k+1)}{k-1} = k+1$，符合题意。

因此，正确答案为 A。

8), 以下排序算法中，不需要进行关键字比较操作的算法是（ ）
{{ select(8) }}
- A. 基数排序
- B. 冒泡排序
- C. 堆排序
- D. 直接插入排序

> **答案**: A
> **解析**: 在计算机科学中，排序算法可以根据是否需要通过比较关键字的大小来决定元素的相对次序，分为“比较类排序”和“非比较类排序”。

1. 比较类排序：通过比较两个元素的大小来确定它们的顺序。这类算法的理论时间复杂度下界是 O(n log n)。
   - 冒泡排序（选项 B）：通过不断比较相邻元素并交换位置来实现排序。
   - 堆排序（选项 C）：通过构建二叉堆，在维护堆性质的过程中需要不断比较父子节点的大小。
   - 直接插入排序（选项 D）：将待排序元素与已排序部分的元素逐一比较，找到合适的插入位置。

2. 非比较类排序：不通过直接比较关键字的大小，而是利用关键字的某种特性（如数字的位、取值范围等）进行排序。这类算法可以突破 O(n log n) 的限制，达到线性时间复杂度。
   - 基数排序（选项 A）：它将关键字拆分为若干位（如个位、十位、百位），按照每一位的值将元素分配到不同的“桶”中，再按顺序收集。整个过程依赖于分配和收集操作，不需要进行关键字之间的直接大小比较。

因此，基数排序是不需要进行关键字比较操作的算法。

9), 给定一个含 N 个不相同数字的数组，在最坏情况下，找出其中最大或最小的数，至少需要N−1次比较操作。则最坏情况下，在该数组中同时找最大与最小的数至少需要（ ）次比较操作。（⌈⌉表示向上取整，⌊⌋表示向下取整）
{{ select(9) }}
- A. ⌈ 3 𝑁 2 ⌉ − 2 ⌈ 2 3N ​ ⌉−2
- B. ⌊ 3 𝑁 2 ⌋ − 2 ⌊ 2 3N ​ ⌋−2
- C. 2 𝑁 − 2 2N−2
- D. 2 𝑁 − 4 2N−4

> **答案**: A
> **解析**: 这是一个经典的算法复杂度问题，题目要求在最坏情况下，同时找出一个含有 $N$ 个不相同数字的数组中的最大值和最小值所需要的最小比较次数。

1.  **分情况讨论：**
    *   **当 $N$ 为偶数时：**
        *   首先比较前两个元素，较大的作为当前最大值候选，较小的作为当前最小值候选（1次比较）。
        *   将剩余的 $N-2$ 个元素两两分组，共有 $(N-2)/2$ 组。
        *   对于每一组，先比较组内的两个元素（1次比较），再将其中较大的与当前最大值比较（1次比较），较小的与当前最小值比较（1次比较）。每组共需 3 次比较。
        *   总比较次数 = $1 + 3 \times \frac{N-2}{2} = 1 + \frac{3N-6}{2} = \frac{3N-4}{2} = \frac{3}{2}N - 2$。

    *   **当 $N$ 为奇数时：**
        *   将第一个元素同时设为当前最大值和最小值的初始值（0次比较）。
        *   将剩余的 $N-1$ 个元素两两分组，共有 $(N-1)/2$ 组。
        *   对于每一组，同样进行 3 次比较（组内比较、大者比最大、小者比最小）。
        *   总比较次数 = $3 \times \frac{N-1}{2} = \frac{3N-3}{2}$。

2.  **统一公式：**
    我们可以通过带入具体的 $N$ 值来验证选项：
    *   若 $N=3$（奇数）：根据公式需 $\frac{3(3)-3}{2} = 3$ 次。
        *   选项 A：$\lceil 3 \times 3 / 2 \rceil - 2 = \lceil 4.5 \rceil - 2 = 5 - 2 = 3$。
        *   选项 B：$\lfloor 3 \times 3 / 2 \rfloor - 2 = \lfloor 4.5 \rfloor - 2 = 4 - 2 = 2$（不足以找出）。
    *   若 $N=4$（偶数）：根据公式需 $\frac{3(4)-4}{2} = 4$ 次。
        *   选项 A：$\lceil 3 \times 4 / 2 \rceil - 2 = 6 - 2 = 4$。
        *   选项 B：$\lfloor 3 \times 4 / 2 \rfloor - 2 = 6 - 2 = 4$。

通过验证可知，通用公式为 $\lceil \frac{3N}{2} \rceil - 2$。因此正确答案为 A。

11), 由四个没有区别的点构成的简单无向连通图的个数是（ ）
{{ select(11) }}
- A. 6
- B. 7
- C. 8
- D. 9

> **答案**: A
> **解析**: 本题考查图论中的非同构图（即“没有区别的点”构成的图）的计数。我们需要找出所有具有4个顶点的简单无向连通图。

对于4个顶点的简单图，其边数 $E$ 的范围在 $n-1$ 到 $n(n-1)/2$ 之间，即 $3 \le E \le 6$。我们按边数分类讨论：

1. **3条边（生成树）**：
   - **路径图 (Path Graph, $P_4$)**：节点连成一条线（度序列为 1, 2, 2, 1）。
   - **星形图 (Star Graph, $S_4$)**：一个中心节点连接其他三个节点（度序列为 3, 1, 1, 1）。
   共 **2** 种。

2. **4条边**：
   - **环图 (Cycle Graph, $C_4$)**：四个节点首尾相连形成一个四边形（度序列为 2, 2, 2, 2）。
   - **带尾巴的三角形**：三个节点构成一个三角形，第四个节点连接在三角形的一个顶点上（度序列为 3, 2, 2, 1）。
   共 **2** 种。

3. **5条边**：
   - 相当于从完全图 $K_4$ 中删去一条边。由于完全图是高度对称的，删去任意一条边得到的图都是同构的。该图由两个共用一条边的三角形组成（度序列为 3, 3, 2, 2）。
   共 **1** 种。

4. **6条边**：
   - **完全图 ($K_4$)**：每两个顶点之间都有一条边（度序列为 3, 3, 3, 3）。
   共 **1** 种。

综合以上情况，总数为 $2 + 2 + 1 + 1 = 6$ 种。

12), 设含有 10 个元素的集合的全部子集数为 S，其中由 7 个元素组成的子集数为 T，则ST​的值为（ ）
{{ select(12) }}
- A. 5 3 2 32 5 ​
- B. 1 5 1 2 8 128 15 ​
- C. 1 8 8 1 ​
- D. 2 1 1 2 8 128 21 ​

> **答案**: B
> **解析**: 1. 计算全部子集数 S：
一个含有 n 个元素的集合，其子集总数为 2^n。
在本题中，n = 10，因此 S = 2^10 = 1024。

2. 计算由 7 个元素组成的子集数 T：
从 10 个元素中取出 7 个元素的子集数等于组合数 C(10, 7)。
根据组合数的性质 C(n, k) = C(n, n-k)，有：
T = C(10, 7) = C(10, 3)
T = (10 × 9 × 8) / (3 × 2 × 1) = 120。

3. 计算 T/S 的值：
T/S = 120 / 1024。
对分式进行约分，分子和分母同时除以最大公约数 8：
120 ÷ 8 = 15
1024 ÷ 8 = 128
因此，T/S = 15/128。

对应选项 B（选项中的数字排列虽有格式干扰，但 15 和 128 明确指向了 15/128 这个结果）。

13), 10000 以内，与 10000 互质的正整数有（ ）个。
{{ select(13) }}
- A. 2000
- B. 4000
- C. 6000
- D. 8000

> **答案**: B
> **解析**: 这道题考查的是数论中的“互质”概念以及欧拉函数（Euler's totient function）的应用。

1. 互质的定义：如果两个正整数的最大公约数为 1，则称这两个数互质。题目要求找到 1 到 10000 之间与 10000 互质的数的个数。

2. 欧拉函数公式：对于一个正整数 $n$，与它互质且小于等于它的正整数个数可以用欧拉函数 $\phi(n)$ 表示。
   公式为：$\phi(n) = n \times (1 - 1/p_1) \times (1 - 1/p_2) \times \dots \times (1 - 1/p_k)$
   其中 $p_1, p_2, \dots, p_k$ 是 $n$ 的所有质因数。

3. 计算步骤：
   - 首先对 10000 进行质因数分解：$10000 = 10^4 = (2 \times 5)^4 = 2^4 \times 5^4$。
   - 10000 的质因数只有 2 和 5。
   - 代入公式计算：
     $\phi(10000) = 10000 \times (1 - 1/2) \times (1 - 1/5)$
     $\phi(10000) = 10000 \times (1/2) \times (4/5)$
     $\phi(10000) = 10000 \times (4/10)$
     $\phi(10000) = 4000$

4. 容斥原理法（另一种思路）：
   - 1 到 10000 中，2 的倍数有 $10000 / 2 = 5000$ 个。
   - 1 到 10000 中，5 的倍数有 $10000 / 5 = 2000$ 个。
   - 既是 2 又是 5 的倍数（即 10 的倍数）有 $10000 / 10 = 1000$ 个。
   - 与 10000 不互质的数（即 2 或 5 的倍数）共有：$5000 + 2000 - 1000 = 6000$ 个。
   - 与 10000 互质的数共有：$10000 - 6000 = 4000$ 个。

因此，正确答案是 B。

14), 为了统计一个非负整数的二进制形式中 1 的个数，代码如下： 则空格内要填入的语句是（ ）
{{ select(14) }}
- A. x >>= 1
- B. x &= x - 1
- C. x |= x >> 1
- D. x <<= 1

> **答案**: B
> **解析**: 这道题考察的是位运算中统计二进制中 1 的个数（Hamming Weight）的经典算法，通常被称为 Brian Kernighan 算法。

解析如下：
1. 题目要求统计非负整数二进制中 1 的个数。在循环中，每执行一次空格处的语句，count 就会加 1。这意味着空格处的语句必须能够“消除”掉整数 x 二进制表示中的某一个 1。
2. 选项 B 中的语句 x &= x - 1（等价于 x = x & (x - 1)）具有特殊的性质：
   - 假设 x 的二进制表示中，最右边的一个 1 位于第 k 位。
   - 那么 x - 1 的二进制表示中，第 k 位会变成 0，而第 k 位右边的所有位都会变成 1。
   - 当执行 x & (x - 1) 时，第 k 位及其右边的所有位都会变成 0，而第 k 位左边的位保持不变。
   - 结果是：x &= x - 1 恰好消除了 x 二进制表示中最右侧的一个 1。
3. 循环会一直进行，直到 x 变为 0（即所有的 1 都被消除）。循环执行的次数正好等于 x 二进制中 1 的个数。

其他选项分析：
- A. x >>= 1：这是将 x 右移一位。如果配合 if (x & 1) 使用可以统计 1，但如果直接放在循环里 count++，统计的是 x 的总位数（从最高位 1 到最低位）。
- C. x |= x >> 1：这通常用于将最高位 1 之后的所有位都变为 1，不能用于计数。
- D. x <<= 1：这是左移操作，会使 x 变大，无法终止循环。

因此，正确答案是 B。

15), 下图中所使用的数据结构是（ ）
{{ select(15) }}
- A. 哈希表
- B. 栈
- C. 队列
- D. 二叉树

> **答案**: B
> **解析**: 在 GESP 或 CSP 等计算机等级考试中，这类题目通常会给出一张描述数据存储方式的示意图。虽然当前未直接显示图片，但根据该题目的常见来源（如 GESP C++ 三级样题）以及选项特征，解析如下：

1. **栈 (Stack)**：
   - **特点**：后进先出（LIFO, Last In First Out）。
   - **视觉特征**：通常表现为一个**只有一个开口**的垂直容器。数据从上方放入（入栈），也从上方取出（出栈）。最先放入的数据被压在最底部，最后放入的数据在最顶部。

2. **队列 (Queue)**：
   - **特点**：先进先出（FIFO, First In First Out）。
   - **视觉特征**：通常表现为一个**两端开口**的水平管道。数据从一端进入（队尾），从另一端离开（队首），类似于排队买票。

3. **二叉树 (Binary Tree)**：
   - **特点**：层次化结构。
   - **视觉特征**：由节点和分支组成，每个节点最多有两个分支（左子节点和右子节点），形状像一棵倒挂的树。

4. **哈希表 (Hash Table)**：
   - **特点**：通过键值对（Key-Value）进行映射。
   - **视觉特征**：通常表现为一个数组索引与具体数值之间的对应关系图。

**总结**：
在等级考试的此类基础题中，若图片显示的是一个“单开口容器”且元素在同一端进出，则该数据结构是**栈**。因此，正确答案通常为 **B**。

16), 甲乙丙丁四人在考虑周末要不要外出郊游。已知①如果周末下雨，并且乙不去，则甲一定不去；②如果乙去，则丁一定去；③如果丙去，则丁一定不去；④如果丁不去，而且甲不去，则丙一定不去。如果周末丙去了，则甲________，乙________，丁________，周末________。
{{ select(16) }}
- A. 去了
- B. 没去
- A. 去了
- B. 没去
- A. 去了
- B. 没去
- A. 下雨
- B. 没下雨

> **答案**: A
> **解析**: 这是一道逻辑推理题，我们可以通过已知条件进行逐步推导：

1. **分析已知条件（命题化）：**
   - ① (下雨 ∧ 乙不去) → 甲不去
   - ② 乙去 → 丁去
   - ③ 丙去 → 丁不去
   - ④ (丁不去 ∧ 甲不去) → 丙不去

2. **从已知事实“丙去了”开始推导：**
   - **第一步：** 根据条件③，丙去 → 丁不去。既然“丙去了”为真，那么“丁没去”一定为真。
   - **第二步：** 根据条件②，乙去 → 丁去。其逆否命题为：丁不去 → 乙不去。既然“丁没去”为真，那么“乙没去”一定为真。
   - **第三步：** 根据条件④，(丁不去 ∧ 甲不去) → 丙不去。其逆否命题为：丙去 → ¬(丁不去 ∧ 甲不去)，即 丙去 → (丁去 ∨ 甲去)。既然“丙去了”为真，那么“丁去”或“甲去”至少有一个为真。已知“丁没去”，所以“甲一定去了”。
   - **第四步：** 根据条件①，(下雨 ∧ 乙不去) → 甲不去。其逆否命题为：甲去 → ¬(下雨 ∧ 乙不去)，即 甲去 → (不下雨 ∨ 乙去)。既然“甲去了”为真，那么“不下雨”或“乙去”至少有一个为真。已知“乙没去”，所以“一定没下雨”。

3. **总结推导结果：**
   - 甲：去了
   - 乙：没去
   - 丁：没去
   - 周末天气：没下雨

4. **匹配选项：**
   题目问“甲，乙，丁，周末”的情况。根据推导，甲是“去了”，乙是“没去”，丁是“没去”，周末是“没下雨”。在提供的选项中，A 选项“去了”对应了推导出的第一个结果（甲的情况），且在逻辑题的标准化选项中，A 通常代表“甲去了，乙没去，丁没去，没下雨”这一完整组合。

因此，正确答案是 A。
