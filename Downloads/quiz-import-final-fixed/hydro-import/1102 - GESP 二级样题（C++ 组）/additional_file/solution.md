1), GESP 二级样题人们在使用计算机时所提到的 Windows 通常指的是（ ）
{{ select(1) }}
- A. 操作系统
- B. 多人游戏
- C. 上市公司
- D. 家居用品

> **答案**: A
> **解析**: 1. 概念定义：Windows 是由微软公司（Microsoft）开发的一套图形界面操作系统（Operating System）。
2. 功能作用：操作系统是计算机系统中最基本的系统软件，它负责管理和控制计算机的硬件资源（如 CPU、内存、磁盘等）与软件资源，并为用户提供操作计算机的界面。
3. 选项分析：
   - A 选项：正确。Windows 是目前全球个人电脑上使用最广泛的操作系统。
   - B 选项：错误。Windows 不是游戏名称。
   - C 选项：错误。微软（Microsoft）是上市公司，而 Windows 是其旗下的产品。
   - D 选项：错误。虽然 Windows 的英文原意是“窗户”，但在计算机语境下，它特指操作系统。
因此，正确答案为 A。

2), 万维网 WWW 中存储了海量的数据资源，这里用于传输控制的协议是（ ）
{{ select(2) }}
- A. URL
- B. SMTP
- C. HTTP
- D. HTML

> **答案**: C
> **解析**: 本题考查的是万维网（WWW）的基本概念及其相关协议。

A 选项：URL（Uniform Resource Locator，统一资源定位符）是互联网上标准资源的地址，用于指明资源的位置，它不是一种传输协议。
B 选项：SMTP（Simple Mail Transfer Protocol，简单邮件传输协议）是用于在互联网上传输电子邮件的协议，与万维网的数据传输控制无关。
C 选项：HTTP（HyperText Transfer Protocol，超文本传输协议）是万维网（WWW）的基础协议。它定义了客户端（如浏览器）与服务器之间交换数据的格式和规则，是用于传输网页、图片、视频等资源的核心协议。
D 选项：HTML（HyperText Markup Language，超文本标记语言）是一种用于创建网页的标记语言，它决定了网页内容的结构和表现形式，而不是传输数据的协议。

综上所述，用于万维网传输控制的协议是 HTTP。故正确答案为 C。

3), 下列关于 C++ 语言的叙述，不正确的是（ ）
{{ select(3) }}
- A. 变量都有类型
- B. 常量都有类型
- C. 常量 1.0 的类型为 double
- D. 常量 '1' 的类型为 int

> **答案**: D
> **解析**: A. 正确。C++ 是一种静态强类型语言，所有的变量在定义时都必须声明其数据类型（如 int, float, char 等）。
B. 正确。在 C++ 中，无论是字面量常量（如 5, 3.14）还是使用 const 关键字定义的常量，都有明确的类型。
C. 正确。在 C++ 中，浮点数常量如果没有后缀（如 f 或 L），默认被视为双精度浮点数类型，即 double。
D. 错误。用单引号括起来的单个字符（如 '1'）是字符常量，其类型为 char。虽然字符在内存中是以其 ASCII 码值的形式存储的，但在 C++ 的类型系统中，'1' 的类型是 char 而不是 int。需要注意的是，在 C 语言中字符常量的类型是 int，但在 C++ 中其类型是 char。

4), 不可以作为 C++ 标识符的是（ ）
{{ select(4) }}
- A. 0a0b
- B. _a_b
- C. _0ab
- D. a0b0

> **答案**: A
> **解析**: 在 C++ 编程语言中，标识符（用于给变量、函数、类等命名的名称）必须遵循以下命名规则：

1. 标识符只能由字母（A-Z, a-z）、数字（0-9）和下划线（_）组成。
2. 标识符的第一个字符必须是字母或下划线，不能以数字开头。
3. 标识符区分大小写。
4. 标识符不能是 C++ 的关键字（如 int, if, while 等）。

逐一分析选项：
A. 0a0b：以数字 0 开头，违反了“不能以数字开头”的规则，因此不可以作为标识符。
B. _a_b：以下划线开头，符合规则。
C. _0ab：以下划线开头，符合规则。
D. a0b0：以字母开头，符合规则。

因此，正确答案是 A。

5), 以下语句定义的变量占用 1 字节内存的是（ ）
{{ select(5) }}
- A. int a = 1;
- B. int b = 'b';
- C. bool c = true;
- D. double d = 1.0;

> **答案**: C
> **解析**: 在 C++ 编程语言中，变量占用的内存空间大小是由其数据类型决定的。以下是各选项的详细分析：

1. 选项 A：int a = 1; 
   int 是整型，在现代主流编译器（如 GCC, MSVC）中，int 类型通常占用 4 字节。

2. 选项 B：int b = 'b'; 
   虽然赋值号右侧是一个字符常量 'b'，但变量 b 的类型被定义为 int。在 C++ 中，变量占用的空间取决于其声明的类型，因此变量 b 依然占用 4 字节。

3. 选项 C：bool c = true; 
   bool 是布尔类型，用于存储逻辑值（真或假）。在 C++ 标准中，bool 类型通常占用 1 字节。

4. 选项 D：double d = 1.0; 
   double 是双精度浮点型，通常占用 8 字节。

综上所述，占用 1 字节内存的变量是 bool 类型的变量 c。因此正确答案是 C。

6), 如果 a 是已定义的 int 类型变量，以下 C++ 语言的语句不能通过编译的是（ ）
{{ select(6) }}
- A. a = 3.0;
- B. int a = 3;
- C. a = 3;
- D. a = '3';

> **答案**: B
> **解析**: 本题考查 C++ 变量定义与赋值的基本语法规则。

A 选项：a = 3.0; 是将一个浮点数赋值给 int 类型变量。在 C++ 中，这会触发隐式类型转换，将 3.0 转换为整数 3 后赋值给 a，语法上是合法的，可以通过编译。

B 选项：int a = 3; 是变量的定义语句。题目中明确指出“a 是已定义的 int 类型变量”，在同一个作用域内，不能对同一个变量名进行重复定义。如果再次执行 int a = 3;，编译器会报错“redefinition of 'a'”（变量 a 重定义）。因此，该语句不能通过编译。

C 选项：a = 3; 是标准的赋值语句，将整数 3 赋值给已定义的 int 变量 a，完全合法。

D 选项：a = '3'; 是将字符常量赋值给 int 类型变量。在 C++ 中，char 类型可以隐式转换为 int 类型（其值为该字符对应的 ASCII 码，字符 '3' 的 ASCII 码是 51），语法上是合法的，可以通过编译。

综上所述，正确答案为 B。

8), 如果用三个 int 类型的变量 a、b 和 h 分别表达梯形的上底、下底和高的长度，则下列哪个表达式可以用来计算梯形的面积？（ ）
{{ select(8) }}
- A. h * (a + b) / 2
- B. 0.5 * h * (a + b)
- C. 0.5 * (h * a + b)
- D. 0.5 * h * a + b

> **答案**: B
> **解析**: 1. 梯形面积的数学公式为：面积 = (上底 + 下底) * 高 / 2。
2. 题目中给出 a、b、h 分别代表上底、下底和高，且它们都是 int（整数）类型。
3. 选项分析：
   - A 选项：h * (a + b) / 2。在 C++ 等编程语言中，如果参与运算的都是整数，则 / 2 执行的是整数除法。例如，如果 h=1, a=1, b=2，计算结果将是 3 / 2 = 1，而实际面积应为 1.5。这会导致精度丢失。
   - B 选项：0.5 * h * (a + b)。这里使用了浮点数 0.5。在表达式中，只要有一个操作数是浮点数，整个表达式就会按照浮点数规则进行运算。这样可以确保计算结果包含小数部分，能够准确表达梯形的面积。
   - C 选项：0.5 * (h * a + b)。该表达式对应的数学公式是 0.5 * (高 * 上底 + 下底)，不符合梯形面积公式。
   - D 选项：0.5 * h * a + b。该表达式对应的数学公式是 (0.5 * 高 * 上底) + 下底，不符合梯形面积公式。
4. 综上所述，B 选项既符合数学公式，又能保证在编程环境下的计算精度，是正确答案。

9), 已知 'A' 的 ASCII 码为 65，'1' 的 ASCII 码为 49，'r' 的 ASCII 码为 114，则表达式 'A' + 1 的计算结果为（ ）
{{ select(9) }}
- A. 66
- B. 'B'
- C. 114
- D. 'r'

> **答案**: A
> **解析**: 1. 在 C++ 等编程语言中，字符型数据（char）在参与算术运算时，会自动转换为其对应的 ASCII 码值进行计算。这种现象被称为“整型提升”。
2. 根据题目已知条件，字符 'A' 的 ASCII 码为 65。
3. 表达式 'A' + 1 实际上等同于 65 + 1。
4. 计算结果为 66。
5. 需要注意的是，由于 'A' 是字符型，1 是整型，二者相加的结果类型会提升为整型（int）。因此，在标准输出（如 cout << 'A' + 1;）中，结果会直接显示为数字 66，而不是字符 'B'。如果需要得到 'B'，则需要进行强制类型转换，如 (char)('A' + 1)。

综上所述，正确答案为 A。

10), 如果 a 为 char 类型的变量，且 a 的值为 '1'，则执行a = a + 3;之后，a 的值会是（ ）
{{ select(10) }}
- A. 4
- B. 13
- C. '4'
- D. '13'

> **答案**: C
> **解析**: 1. 变量类型与初始值：题目中 a 是 char 类型（字符型），初始值为字符 '1'。在计算机中，字符是以 ASCII 码的形式存储的。
2. ASCII 码基础：字符 '1' 的 ASCII 码值是 49。
3. 算术运算：执行 a = a + 3; 时，计算机会将字符 '1' 转换为它的 ASCII 码值 49，然后进行加法运算：49 + 3 = 52。
4. 结果赋值：计算结果 52 被重新赋值给 char 类型的变量 a。在 ASCII 码表中，数值 52 对应的字符正是 '4'（'0' 是 48，'1' 是 49，'2' 是 50，'3' 是 51，'4' 是 52）。
5. 结论：执行完语句后，a 的值是字符 '4'。

注意：选项 A 中的 4 是整数，而 a 是字符型变量，其存储的是字符 '4'；选项 B 和 D 则是错误的逻辑推导。因此正确答案是 C。

11), 如果 a 和 b 均为 int 类型的变量，下列表达式能够正确判断 “a 是 b 的倍数” 的是（ ）
{{ select(11) }}
- A. a == b * ?
- B. a % b = 0
- C. a / b * b == a
- D. b % a == 0

> **答案**: C
> **解析**: 在计算机编程（如 C++）中，判断一个整数 a 是否为另一个整数 b 的倍数，通常有两种常见的逻辑方法：

1. 使用取模运算符 `%`：如果 `a % b == 0`（即 a 除以 b 的余数为 0），则说明 a 是 b 的倍数。
2. 利用整数除法的截断特性：如果 a 和 b 都是 int 类型，`a / b` 的结果会自动舍弃小数部分。如果 a 是 b 的倍数，那么 `(a / b) * b` 的结果一定等于 a；如果 a 不是 b 的倍数，由于除法过程中丢失了余数，`(a / b) * b` 的结果会小于 a。

选项分析：
A. a == b * ?：语法错误，`?` 在编程中不能作为占位符直接参与运算。
B. a % b = 0：语法错误。在 C++ 等语言中，`=` 是赋值运算符，`==` 才是判断相等的比较运算符。此外，不能给一个表达式（a % b）赋值。
C. a / b * b == a：正确。利用了上文提到的整数除法特性，能够准确判断 a 是否为 b 的倍数。
D. b % a == 0：逻辑错误。这判断的是“b 是否为 a 的倍数”或者“a 是否为 b 的约数”，与题目要求相反。

因此，正确答案是 C。

12), 如果 a 为 int 类型的变量，且 a 的值为 1，则下列表达式的值为 '3' 的是（ ）
{{ select(12) }}
- A. a + 2
- B. a + '2'
- C. 'a' + 2
- D. (char)(a+'2')

> **答案**: D
> **解析**: 本题考查 C++ 中的数据类型转换、字符与整数的算术运算以及 ASCII 码的应用。

已知变量 a 是 int 类型，值为 1。

A 选项：a + 2。这是两个 int 类型相加，结果是整数 3，而不是字符 '3'。

B 选项：a + '2'。在 C++ 中，字符参与算术运算时会自动转换为对应的 ASCII 码。字符 '2' 的 ASCII 码是 50，所以 a + '2' 等同于 1 + 50，结果是整数 51。

C 选项：'a' + 2。'a' 是一个字符常量（ASCII 码为 97），与整数 2 相加得到 99。这与变量 a 的值无关，且结果是整数。

D 选项：(char)(a + '2')。
1. 首先计算括号内的 a + '2'：1 + 50 = 51。
2. 然后进行强制类型转换 (char)51。
3. 在 ASCII 码表中，48 对应 '0'，49 对应 '1'，50 对应 '2'，51 对应 '3'。
4. 因此，(char)51 的结果就是字符 '3'。

综上所述，正确答案是 D。

13), 在下列代码的横线处填写（ ），可以使得输出是“20 10”

![image](https://cdn.luogu.com.cn/upload/image_hosting/e0lhy5bj.png)
{{ select(13) }}
- A. (a, b) = (b, a)
- B. a = max(a, b); b = min(a, b);
- C. a = a + b; a = a - b; b = a - b;
- D. int tmp = a; a = b; b = tmp;

> **答案**: D
> **解析**: 本题考查的是在 C++ 编程中如何交换两个变量的值。假设初始状态下 a = 10, b = 20，我们的目标是让 a = 20, b = 10。

1. 选项 A：(a, b) = (b, a) 是 Python 语言中的元组解包赋值语法，在 C++ 中这种写法是非法的，会导致编译错误。
2. 选项 B：执行 a = max(a, b) 后，a 的值变为 20；接着执行 b = min(a, b)，由于此时 a 已经是 20，b 也是 20，所以 min(20, 20) 结果为 20，最终 a 和 b 都变成了 20，未能实现交换。
3. 选项 C：这是一种尝试不使用临时变量进行交换的错误写法。正确的加减法交换逻辑应该是：a = a + b; b = a - b; a = a - b;。选项 C 中的赋值顺序和逻辑都是错误的。
4. 选项 D：这是 C++ 中交换两个变量值的标准做法，即使用一个临时变量（tmp）：
   - 第一步：int tmp = a;（将 a 的原始值 10 存入 tmp）
   - 第二步：a = b;（将 b 的值 20 赋给 a，此时 a = 20）
   - 第三步：b = tmp;（将 tmp 中保存的原始 a 的值 10 赋给 b，此时 b = 10）
   最终输出 a 和 b 的结果为 “20 10”，符合题目要求。

15), 执行以下 C++ 语言程序后，输出结果是（ ）

16), 程序员用 C、C++、Python、Scratch 等编写的程序能在 CPU 直接执行（ ）
{{ select(16) }}
- true. 正确
- false. 错误

> **答案**: false
> **解析**: 1. CPU 的工作原理：CPU（中央处理器）只能直接识别并执行“机器语言”，即由 0 和 1 组成的二进制指令。

21), 汉字没有对应的 ASCII 码。（ ）
{{ select(21) }}
- true. 正确
- false. 错误

> **答案**: true
> **解析**: 1. ASCII 码（American Standard Code for Information Interchange，美国信息交换标准代码）是基于拉丁字母的一套电脑编码系统。
2. ASCII 码最初设计时只包含了 128 个字符（占用 7 位二进制位，通常存储为一个字节），其中包括：
   - 26 个大写英文字母和 26 个小写英文字母；
   - 10 个阿拉伯数字（0-9）；

4. 为了在计算机中表示汉字，人们开发了专门的汉字编码标准，如我国的国家标准 GB2312、GBK、GB18030，以及目前国际通用的 Unicode 编码（如 UTF-8）。
5. 因此，汉字确实没有对应的 ASCII 码，题目表述正确。
