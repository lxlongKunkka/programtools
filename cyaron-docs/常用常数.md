# 常用常数
CYaRon 提供了一些常用的常数。

1. `PI`

即圆周率的值。`3.1415926...`

2. `E`

即自然底数的值。`2.7182818...`

3. `ALPHABET_SMALL`

一个字符串，包含所有的小写字母。`"abcdefghijklmnopqrstuvwxyz"`

4. `ALPHABET_CAPITAL`

一个字符串，包含所有的大写字母。`"ABCDEFGHIJKLMNOPQRSTUVWXYZ"`

5. `ALPHABET`

一个字符串，包含所有的字母。`"abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ"`

6. `NUMBERS`

一个字符串，包含所有的数字。`"0123456789"`
