### 第 5 题

如果 `a` 和 `b` 均为 `int` 类型的变量，下列表达式不能正确判断"a 等于 0 且 b 等于 0"的是（ ）。

A. `a == 0 && b == 0`

B. `a == b == 0`

C. `!(a) && !(b)`

D. `(a == 0) + (b == 0) == 2`

---

**答案：B**

*来源：GESP 2023-03 一级*

**解析：**
B 的问题：`a == b == 0` 等价于 `(a == b) == 0`（左结合）。假设 a=0, b=0，则 `(0==0)==0` → `1==0` → false！反而是错的。
A 正确（逻辑与），C 正确（!a 当 a=0 时为 true），D 正确（两个 true 相加=2，等于 2 为真）。
