# 计算机基础题目导入说明

## 概述

本文件夹包含78道计算机基础测试题，涵盖计算机历史、硬件、软件、网络等基础知识。

## 文件说明

- `计算机基础测试题.md` - 题目源文件（Markdown格式）
- `../server/scripts/import-computer-basics-quiz.js` - 导入脚本

## 导入步骤

### 1. 确保MongoDB正在运行

本地MongoDB：
```bash
# Windows (需要以管理员权限运行)
net start MongoDB

# macOS/Linux
sudo systemctl start mongod
# 或
mongod --dbpath /path/to/data
```

### 2. 预览导入（不写入数据库）

```bash
node server/scripts/import-computer-basics-quiz.js
```

预期输出：
```json
{
  "inputFile": "...",
  "paper": "computer-basics-7000",
  "questions": 78,
  "single": 78,
  "warnings": 0,
  "apply": false
}
```

### 3. 执行实际导入

```bash
node server/scripts/import-computer-basics-quiz.js --apply
```

成功后会输出：
```json
{
  "papersUpserted": 1,
  "questionsUpserted": 78
}
```

## 故障排查

### MongoDB连接失败

错误信息：`MongooseServerSelectionError: connect ECONNREFUSED`

解决方法：
1. 确认MongoDB服务正在运行
2. 检查 `server/.env` 中的 `MONGODB_URI` 配置
3. 确认MongoDB监听在正确的端口（默认27017）

### 题目解析失败

如果warnings > 0，检查：
1. 题目格式是否符合规范（参考答案格式等）
2. 选项标记是否正确（A-E）
3. 题干和选项之间的空行
