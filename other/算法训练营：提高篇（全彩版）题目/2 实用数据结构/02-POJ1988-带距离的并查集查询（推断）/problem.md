# POJ1988 带距离的并查集查询（推断）

## 题目描述
维护若干个元素组成的堆，每个元素初始单独成堆。支持两种操作：

- 将元素 $x$ 所在的整堆放到元素 $y$ 所在整堆的上方；
- 查询某个元素下方有多少个元素。

## 输入格式
第一行输入一个整数 $n$，表示操作数。

接下来 $n$ 行，每行是以下两种操作之一：

- M x y：把包含 $x$ 的整堆移动到包含 $y$ 的整堆上方；
- C x：查询元素 $x$ 下方的元素数量。

## 输出格式
对于每个查询操作，输出一行一个整数。
