# POJ3368 区间最高频数（推断）

## 题目描述
给定一个非递减整数序列，回答若干次区间查询。对于每个查询区间 $[l,r]$，求该区间中出现次数最多的数的出现次数。

## 输入格式
输入包含多组数据。每组数据第一行输入一个整数 $n$。

当 $n=0$ 时输入结束。

第二行输入一个整数 $q$，表示查询次数。

第三行输入长度为 $n$ 的非递减序列。

接下来 $q$ 行，每行输入两个整数 $l,r$。

## 输出格式
对于每次查询，输出一行一个整数，表示区间内的最高频数。
