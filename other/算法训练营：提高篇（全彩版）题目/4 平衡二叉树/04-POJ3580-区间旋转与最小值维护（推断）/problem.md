# POJ3580 区间旋转与最小值维护（推断）

## 题目描述
给定一个整数序列，支持以下操作：

- ADD l r d：将区间 $[l,r]$ 内每个数都加上 $d$；
- REVERSE l r：将区间 $[l,r]$ 翻转；
- REVOLVE l r t：将区间 $[l,r]$ 循环右移 $t$ 位；
- INSERT pos x：在指定位置插入一个整数；
- DELETE pos：删除指定位置的整数；
- MIN l r：查询区间 $[l,r]$ 的最小值。

## 输入格式
第一行输入一个整数 $n$，表示初始序列长度。

第二行输入 $n$ 个整数，表示初始序列。

第三行输入一个整数 $m$，表示操作数。

接下来 $m$ 行，每行输入一条操作，格式如题目描述所示。

## 输出格式
对于每个 MIN 操作，输出一行一个整数，表示对应区间的最小值。

## 说明
该题面根据 AC 代码行为推断生成，默认代码使用 Splay 维护序列并支持区间加、翻转、旋转和区间最小值查询，建议后续人工复核。
