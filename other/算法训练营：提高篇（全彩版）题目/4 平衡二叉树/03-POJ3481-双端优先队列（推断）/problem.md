# POJ3481 双端优先队列（推断）

## 题目描述
维护一个空集合，支持以下操作：

- 1 k p：插入一个编号为 $k$、优先级为 $p$ 的元素；
- 2：输出并删除当前优先级最高的元素编号；如果集合为空，输出 0；
- 3：输出并删除当前优先级最低的元素编号；如果集合为空，输出 0。

## 输入格式
输入由若干行操作组成，每行首先输入一个整数表示操作类型。

- 当操作类型为 1 时，后续再输入两个整数 $k,p$；
- 当操作类型为 2 或 3 时，无额外输入；
- 当输入为 0 时，表示结束。

## 输出格式
对于每个 2 或 3 操作，输出一行一个整数，表示被删除元素的编号；若集合为空则输出 0。
