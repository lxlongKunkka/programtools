# POJ1236 强连通分量补边（推断）

## 题目描述
给定一个有向图，图中共有 $n$ 个点。现在需要回答两个问题：

- 至少选择多少个起点发出信息，才能让所有点最终都收到信息；
- 至少再添加多少条有向边，才能使整张图变成强连通图。

## 输入格式
第一行输入一个整数 $n$，表示点数。

接下来 $n$ 行描述每个点的出边：第 $i$ 行输入若干个整数，表示从点 $i$ 出发可以到达的点，最后以 0 结束。

## 输出格式
输出两行：

- 第一行输出第一个问题的答案；
- 第二行输出第二个问题的答案。

## 说明
该题面根据 AC 代码行为推断生成，默认代码先求强连通分量，再统计缩点后入度为 0 和出度为 0 的分量数量，建议后续人工复核。
