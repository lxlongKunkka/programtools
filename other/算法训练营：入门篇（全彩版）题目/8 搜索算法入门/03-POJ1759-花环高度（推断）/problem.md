# POJ1759 花环高度（推断）

## 题目描述
有一串共 $n$ 个花环节点，第一个节点的高度为 $A$，其余节点的高度满足递推关系：

$$h_i = 2h_{i-1} - h_{i-2} + 2 \quad (i \ge 3)$$

你需要确定第二个节点的最小可能高度，使得所有节点的高度都严格大于 $0$。在满足条件时，输出最后一个节点的高度。

## 输入格式
每组输入一行，包含一个整数 $n$ 和一个实数 $A$，分别表示节点数量和第一个节点的高度。

## 输出格式
对于每组数据，输出最后一个节点的高度，保留两位小数。
