# POJ1915 骑士移动（推断）

## 题目描述
在一个 $L \times L$ 的国际象棋棋盘上，给定骑士的起点坐标和终点坐标，求骑士从起点移动到终点所需的最少步数。

骑士每次移动遵循国际象棋规则，可以走到 8 个可能的位置之一。

## 输入格式
第一行输入一个整数 $N$，表示测试数据组数。

对于每组数据：

- 第一行输入一个整数 $L$，表示棋盘大小；
- 第二行输入两个整数，表示起点坐标；
- 第三行输入两个整数，表示终点坐标。

## 输出格式
对于每组数据，输出一行一个整数，表示最少步数。

## 说明
该题面根据 AC 代码行为推断生成，代码采用 BFS 求解最短路，建议后续人工复核。
