# POJ3237 树链剖分路径最大值与取反（推断）

## 题目描述
给定一棵带权树，支持以下操作：

- CHANGE i v：把第 $i$ 条边的权值修改为 $v$；
- NEGATE u v：把路径 $u$ 到 $v$ 上所有边权同时取相反数；
- QUERY u v：查询路径 $u$ 到 $v$ 上边权最大值。

## 输入格式
第一行输入测试组数。对于每组数据，先输入结点数和树边信息，随后输入若干条操作，直到读到 DONE 结束。

## 输出格式
对于每个 QUERY 操作，输出一行一个整数。

## 说明
该题面根据 AC 代码行为推断生成，默认代码使用树链剖分加线段树维护路径信息，建议后续人工复核。
