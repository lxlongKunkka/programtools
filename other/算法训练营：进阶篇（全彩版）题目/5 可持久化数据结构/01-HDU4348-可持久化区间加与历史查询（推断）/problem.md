# HDU4348 可持久化区间加与历史查询（推断）

## 题目描述
给定一个初始数列，支持以下操作：

- C l r d：在当前版本中把区间 $[l,r]$ 的每个数都加上 $d$，并生成一个新版本；
- Q l r：查询当前版本区间 $[l,r]$ 的元素和；
- H l r t：查询历史版本 $t$ 中区间 $[l,r]$ 的元素和；
- B t：将当前版本回退到版本 $t$。

## 输入格式
输入数列长度、操作数、初始数列以及若干条操作。

## 输出格式
对于每个查询操作，输出一行一个整数。
