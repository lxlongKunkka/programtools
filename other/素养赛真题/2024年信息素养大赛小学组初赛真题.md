### 第1题

运行下列程序段，输出的结果是（ ）（5分）

```cpp
int n=572765;
cout<< n/10%10;
```

- A. 5
- B. 6
- C. 4
- D. 1

> **【参考答案】B**
>

### 第2题

假设有两个城市：城市A和城市B。每个城市的温度都在-50到50摄氏度之间。当且仅当只有一个城市的温度低于0时，输出1，也就是说，如果城市A的温度低于0而城市B大于等于0；或者如果城市A的温度大于等于0而B小于0，则输出1，否则输出0。补全①和②处的代码。（5分）

```cpp
#include<iostream>
using namespace std;
int main() {
    int a,b;
    cin >> a >> b;
    if(__①__) {
        if(__②__) {
            cout <<1;
            return 0;
        }
    }
    if(a >= 0) {
        if(b < 0) {
            cout <<1;
            return 0;
        }
    }
    cout <<0;
    return 0;
}
```

- A. `a<0;  b>0;`
- B. `a>0;  b<=0;`
- C. `a>=0;  b>=0;`
- D. `a<0;  b<0;`

> **【参考答案】A**
>
> **解析：** 程序逻辑是：当 a<0 时，再判断 b>=0（即 b>0 或 b==0），但选项A中②填 `b>0` 并不完全覆盖 b==0 的情况。结合程序结构，第一个 if 分支处理"a<0 且 b>0"，第二个分支处理"a>=0 且 b<0"，整体覆盖了"仅一个城市温度低于0"的情形。①填 `a<0`，②填 `b>=0`（选项中最接近的为A：`a<0; b>0`，题目选项即为 **A**）。

### 第3题

执行以下代码段，变量x和y的值分别是（ ）（5分）

```cpp
int x = 7;
int y = x/2*3;
```

- A. x=7，y=10.5
- B. x=7，y=10
- C. x=7，y=12
- D. x=7，y=9

> **【参考答案】D**
>
> **解析：** `x` 为 `int` 类型，`x/2 = 7/2 = 3`（整数除法，截断小数），`3 * 3 = 9`，所以 y=9，x不变仍为7。

### 第5题

小A有一个神奇口袋，里面可以装各个平方数。口袋的负载量就是口袋里所有数字之和。当负载量超过L的时候不能再装平方数。现在给定n个数，从左往右筛选其中的平方数，并依次将平方数放入口袋，编成程序，输出口袋里能装下几个平方数。平方数就是一个整数乘以自己的结果（例如：1、4、9、16、…...）补全①、②和③处的代码。（5分）

例如：输入L为100，n为5，接下来的5个数分别是49、19、36、25、73，则口袋里只装了49和36两个平方数，所以输出2。

```cpp
#include<iostream>
using namespace std;
int main() {
    int L,n,cnt=0,sum=0;
    cin>>L>>n;
    for(int i=1; i<=n; i++) {
        int x;
        bool flag = false;
        cin>>x;
        for(int j=1; j<x; j++) {
            if(__①__) {
                flag = true;
                break;
            }
        }

        if(flag == true) {
            if(__②__) {
                sum +=x;
                cnt++;
            } else {
                __③__
            }
        }
    }
    cout<<cnt;
    return 0;
}
```

- A. `j*j==x;  sum+X<=L;  continue;`
- B. `j*j==x;  sum+x<=L;  break;`
- C. `x/j==i;  sum<=L;  continue;`
- D. `x/j==i;  sum<=L;  break;`

> **【参考答案】B**
>
> **解析：** ①判断 j 是否是 x 的平方根，即 `j*j==x`；②判断加入 x 后是否不超过负载量，即 `sum+x<=L`；③当超过负载量时应退出循环，用 `break`（不能用 continue，因为后续数据仍可能装入，但根据题意"从左至右依次装入，超过则不再装"，应 break 退出整个选取流程）。

```cpp
for (int j = 1; j <= 6; j++) {
    if (j % 3 == 0) {
        break;
    }
    cout <<j<<" ";
}
```

- A. 1 2
- B. 1 2 3
- C. 1 2 3 4 5 6
- D. 1 2 4 5 6

### 第15题

下面代码实现的是判断n是否是质数的功能。补全①和②处的代码。（5分）

```cpp
#include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    __①__
    for(int i=2; i<n; i++) {
        if(__②__) {
            isprime = false;
            break;
        }
    }
    cout << isprime << endl;
    return 0;
}
```

- A. `bool isprime=true;  n%i==0;`
- B. `bool isprime=false;  n%i!=0;`
- C. `bool isprime=true;  n%i!=0;`
- D. `bool isprime=false;  n%i==0;`

> **【参考答案】A**
>
> **解析：** 判断质数需要先假设 n 是质数（`isprime=true`），再从2到n-1遍历，若找到能整除 n 的因子（`n%i==0`），则置 `isprime=false` 并 break。所以①填 `bool isprime=true;`，②填 `n%i==0`。

在C++中，`&&` 和 `||` 分别代表逻辑与和逻辑或运算。（5分）

- A. 正确
- B. 错误

> **【参考答案】A（正确）**

### 第2题

在C++中，`for` 循环至少执行一次。（5分）

- A. 正确
- B. 错误

> **【参考答案】B（错误）**
>
> **解析：** `for` 循环在进入循环体之前会先判断条件，若初始条件就不满足，则循环体一次都不会执行。"至少执行一次"是 `do-while` 循环的特性。

### 第3题

在C++中，`cout` 用于输入，而 `cin` 用于输出。（5分）

- A. 正确
- B. 错误

> **【参考答案】B（错误）**
>
> **解析：** 正好相反，`cout` 用于**输出**（console output），`cin` 用于**输入**（console input）。

### 第4题

在C++中，变量声明后，如果不初始化，其值是确定的。（5分）

- A. 正确
- B. 错误

> **【参考答案】B（错误）**
>
> **解析：** 在C++中，局部变量声明后若不初始化，其值是**不确定的**（未定义行为，取决于该内存地址中的残留数据）。因此不应假设未初始化变量有确定的值。

- A. 正确
- B. 错误
