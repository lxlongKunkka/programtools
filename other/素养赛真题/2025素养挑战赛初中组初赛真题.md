### 第1题

以下关于变量定义的C++语句中存在编译错误的是哪项？

- A. `long long pi=3.14;`
- B. `int a=27, b[10], c[3][3];`
- C. `int x, y, z=10, 11, 12;`
- D. `bool f=false;`

> **【参考答案】A**
>
> **解析：** `long long` 是整型，不能直接赋值浮点数 `3.14`，会产生编译错误（精度丢失警告或错误）。选项C虽然写法奇怪，但 `z=10,11,12` 实际上是逗号表达式，语法上不会报错；A中long long赋3.14会隐式截断，部分编译器会报错。实际上A是最明显的类型不匹配错误。

```cpp
string s;

```

- A. hello world

### 第7题

补全以下代码，将数组a按升序排列。下列选项中哪一项正确？

```cpp
int a[7] = {7, 1, 4, 2, 2, 3, 6};
int N = 7;
for (int i=0; i<N-1; i++) {
    for (int j=0; j<_①_; j++) {
        if (a[j] > a[j+1]) swap(a[j], a[j+1]);
    }
}
```

- A. N-i-1
- B. i-1
- C. i
- D. N

> **【参考答案】A**
>
> **解析：** 这是冒泡排序。每次外层循环 `i` 结束后，最大的 `i+1` 个元素已经排到末尾，内层循环只需比较到 `N-i-1` 处。填入 `N-i-1` 可避免越界并提升效率。

### 第8题

以下说法正确的是？

- A. 执行代码 `cout << 13.8%2;` 会输出 1.8
- B. 将一个浮点数赋值给一个char类型的变量会出现运行时错误
- C. 如果代码中不含有 `#include<iostream>` 将无法通过编译
- D. C++可以定义无返回值且无参数函数

> **【参考答案】D**
>
> **解析：**
> - A错误：`%` 取模运算符不能用于浮点数，`13.8%2` 会编译报错；
> - B错误：浮点数赋给char会隐式截断，是编译警告而非运行时错误；
> - C错误：如果不使用cout/cin等，不包含iostream也能编译通过；
> - D正确：C++可以定义 `void func() {}` 这样无返回值无参数的函数。

下列C++代码中哪个语句运行结果是7？

- A. `cout << (char)7;`
- B. `cout << int(4.3333333 + 2.6666666);`
- C. `cout << 66/9;`
- D. `cout << 15/2.0;`

> **【参考答案】C**
>
> **解析：**
> - A：`(char)7` 输出ASCII码为7的控制字符，不是数字7；

> - C：`66/9` 整数除法 = `7`，**正确**；
> - D：`15/2.0 = 7.5`，不是7。

### 第10题

输入一个DNA序列：由字符A、C、G和T组成的字符串。补充下列代码找到最长的连续相同字符子串。下列选项中哪一项正确？

```cpp
string s;
cin >> s;
int len=_①_, ans = 1;
for(int i=_②_; i<s.size(); i++) {
    if(s[i]==s[_③_]) len++;
    else len = 1;
    if(_④_) ans=len;
}
cout << ans;
```

- A. 1, 1, i-1, len>ans
- B. 0, 0, i+1, len>ans
- C. 1, 1, i-1, len\<ans
- D. 0, 0, i+1, len\<ans

> **【参考答案】A**
>
> **解析：**
> - `len` 初始为 `1`（当前连续长度至少为1）；
> - 循环从 `i=1` 开始；
> - 比较当前字符 `s[i]` 与前一个字符 `s[i-1]`；
> - 当 `len > ans` 时更新最大值。
>
> 填入：①1，②1，③i-1，④len>ans。

### 第12题

从一个2×2网格的左上角出发，若只允许向右或向下移动，恰好有如下6条路径可以到达右下角。

补全以下代码，计算对于9×9的网格，有多少条路径可以到达右下角？

```cpp
int g[10][10] = {};
_①_;
for (int i=0; i<10; i++) {
    for (int j=0; j<10; j++) {
        if(i-1>=0) g[i][j] += g[i-1][j];
        _②_ g[i][j] += g[i][j-1];
    }
}
cout << g[9][9];
```

- A. `g[0][0]=1, if(j-1>=0)`
- B. `g[1][1]=1, if(j-1>=0)`
- C. `g[0][0]=1, else if(j-1>=0)`
- D. `g[1][1]=1, else if(j-1>=0)`

> **【参考答案】A**
>
> **解析：** 动态规划求网格路径数，起点为左上角 `g[0][0]=1`。对于每个格子，路径数 = 上方格子路径数 + 左方格子路径数。所以 ①填 `g[0][0]=1`，②填 `if(j-1>=0)`（独立判断，不用else if）。

### 第13题

有n个人，第i个人的当前头发长度为 $L_i$（1≤i≤N）。每个人的头发每天增长1。请计算并输出从第一天开始，第一次出现头发长度至少为t的人数≥p是第几天？

```cpp
int n, t, p, L[100], cnt;
cin >> n >> t >> p;
for (int i=0; i<n; i++) cin >> L[i];
for (int i=1; i<t; i++) {
    cnt=0;
    for (int j=0; j<n; j++) {
        if(_①_) cnt++;
    }
    if(_②_) {
        cout << i << endl;
        break;
    }
}
```

- A. `L[i]+j>=t, cnt>=p`
- B. `L[]+i>t, cnt>p`
- C. `L[j]+i>=t, cnt>=p`
- D. `L[i]+j>t, cnt>p`

> **【参考答案】C**
>
> **解析：** 外层循环变量 `i` 表示第几天，内层循环变量 `j` 表示第几个人。第j个人第i天的头发长度为 `L[j]+i`，条件是 `>=t`（至少为t），所以①为 `L[j]+i>=t`。条件"人数≥p"对应②为 `cnt>=p`。

### 第14题

执行下列程序，输出结果为？

```cpp
#include <iostream>
using namespace std;
int main() {
    int i=1, j=1;
    int x=i++, y=++j;
    cout << i << " " << j << " " << x << " " << y;
    return 0;
}
```

- A. 1 2 2 1
- B. 2 2 1 2
- C. 1 2 2 2
- D. 2 2 2 2

> **【参考答案】B**
>
> **解析：**
> - `x = i++`：后置自增，先将i的值（1）赋给x，再将i加1。所以 `x=1`，`i=2`；
> - `y = ++j`：前置自增，先将j加1变为2，再赋给y。所以 `j=2`，`y=2`；
> - 输出：`i=2, j=2, x=1, y=2`，即 `2 2 1 2`。

### 第15题

补全下列代码，计算n（1≤n≤1×10⁹）的阶乘中后缀0的个数。下列选项中哪一项正确？

例如5!=1×2×3×4×5=120，它有1个后缀0；20!=2432902008176640000，它有4个后缀0。

**提示：** n!中每个后缀0都是由一个因子2和一个因子5相乘得到的，而在阶乘中因子2的个数远多于因子5的个数，因此只需统计因子5的个数。

```cpp
int n, i=_①_, ans=0;
cin >> n;
while(_②_) {
    i *= 5;
    ans += _③_;
}
cout << ans;
```

- A. 5, i<=n, i
- B. 1, i<=n/5, n/i
- C. 1, i\<n/5, n/i
- D. 5, i\<n, i

> **【参考答案】B**
>
> **解析：** 统计n!中因子5的个数：
> - 5的倍数贡献1个因子5：⌊n/5⌋个；
> - 25的倍数额外贡献1个：⌊n/25⌋个；
> - 125的倍数再额外贡献1个：⌊n/125⌋个……
>
> 代码思路：i从1开始，每次 `i*=5`，累加 `n/i`。循环条件为 `i<=n/5`（即 `i*5<=n`，确保 `n/i>=1` 有意义）。选B：①1，②i<=n/5，③n/i。

### 第1题

C++中，逻辑与`&&`的优先级高于逻辑或`||`，因此`true||false&&true`等价于`true||(false&&true)`。

- A. 正确
- B. 错误

> **【参考答案】A（正确）**
>
> **解析：** C++中运算符优先级：`&&` 高于 `||`，所以 `true||false&&true` 先计算 `false&&true=false`，再计算 `true||false=true`，确实等价于 `true||(false&&true)`，说法正确。

选择排序的时间复杂度在最好的情况下是O(n)。

- A. 正确
- B. 错误

> **【参考答案】B（错误）**
>
> **解析：** 选择排序无论数据初始状态如何，都需要两层嵌套循环进行比较，时间复杂度始终为 **O(n²)**，不存在O(n)的最好情况（不同于冒泡排序可以优化到O(n)）。
