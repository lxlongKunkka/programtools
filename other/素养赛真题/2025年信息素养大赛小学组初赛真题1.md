## 单选题（15题，共75分）

### 第1题

执行以下程序，输入56 25，输出的结果是？

```cpp
int a, b;
cin >> a >> b;
cout << max(a, b);
```

A. 56

B. 25

C. 1000

D. 无输出

> **【参考答案】A**
>
> **解析：** `max(a, b)` 返回两数中较大的那个。输入 56 和 25，a=56，b=25，`max(56, 25)=56`，所以输出 56。

### 第2题

在C++中，a=a-b可以简写为a-=b。执行下列代码，输出结果为？

```cpp
int a=10;
a-=9;
cout << a;
```

A. 2

B. -6

C. 9

D. 1

> **【参考答案】D**
>
> **解析：** `a-=9` 等价于 `a=a-9`，即 `a=10-9=1`，所以输出 1。

### 第6题

下面程序段输出的结果是？

```cpp
char upper = 'A';
char lower = upper + 32;
cout << lower;
```

A. A

B. a

C. 65

D. 97

> **【参考答案】B**
>
> **解析：** 在ASCII码中，大写字母 'A' 的值为65，小写字母 'a' 的值为97，差值恰好为32。所以 `upper + 32` = 65 + 32 = 97，对应字符 'a'，输出结果为 a。

### 第7题

执行下列代码，输出结果为？

```cpp
for (int i=1; i<=5; i++) {
    if( i%2==0 ) {
        continue;
    }
    cout << i << " ";
}
```

A. 1

B. 2 4

C. 1 3 5

D. 1 2 3 4

> **【参考答案】C**
>
> **解析：** 循环 i 从1到5，当 `i%2==0`（即i为偶数）时执行 `continue` 跳过本次循环，不执行输出。所以只输出奇数：1、3、5，结果为 `1 3 5`。

### 第8题

执行以下程序，输出的结果是？

```cpp
int array[3] = {4, 5, 6};
array[0] = array[0] - 3;
cout << array[0] << " " << array[1] << " " << array[2];
```

A. 4 5 6

B. 1 5 6

C. 4 2 6

D. 4 5 3

> **【参考答案】B**
>
> **解析：** 数组初始化为 {4, 5, 6}，执行 `array[0] = array[0] - 3 = 4 - 3 = 1`，array[1] 和 array[2] 不变，所以输出 `1 5 6`。

### 第9题

现有数组定义为 `int num[4]={6};`，则数组num中的元素分别是？

A. 6 6 6 6

B. 0 0 0 0

C. 6 0 0 0

D. 0 0 0 6

> **【参考答案】C**
>
> **解析：** 在C++中，数组初始化时如果提供的初始值个数少于数组大小，则第一个元素赋为给定值（6），其余元素自动初始化为0。所以 num[0]=6，num[1]=num[2]=num[3]=0，结果为 `6 0 0 0`。

### 第10题

小明同学想判断输入的年份x是否为闰年，是则输出Yes，否则输出No。那么下面程序段中①处应该填写（ ）。一个年份如果能被400整除或者能被4整除但不能被100整除，那么这个年份就是闰年。例如：2000、2020是闰年。

```cpp
int x;
cin >> x;
if(_①_) {
    cout << "Yes";
} else {
    cout << "No";
}
```

A. `x%400==0 && x%4==0 && x%100!=0`

B. `%400==0 && x%4==0 || x%100!=0`

C. `x%400==0 || x%4==0 || x%100!=0`

D. `(x%400==0) || (x%4==0 && x%100!=0)`

> **【参考答案】D**
>
> **解析：** 闰年的判断条件是"能被400整除"**或**"能被4整除且不能被100整除"，用逻辑表达式表示为：`(x%400==0) || (x%4==0 && x%100!=0)`。注意运算符优先级，`&&` 优先级高于 `||`，加括号更清晰。选项A用了错误的逻辑（&&连接了三个条件），选项C逻辑错误，选项D正确。

### 第11题

运行以下程序，输入6，输出的结果是？

```cpp
#include <iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    if (n % 2 == 0) {
        cout << "QWER";
    } else if (n % 3 == 0) {
        cout << "WERQ";
    } else if (n % 5 == 0) {
        cout << "ERQW";
    } else {
        cout << "RQWE";
    }
    return 0;
}
```

A. QWER

B. WERQ

C. ERQW

D. RQWE

> **【参考答案】A**
>
> **解析：** 输入 n=6，程序先判断 `6 % 2 == 0`，结果为真（6能被2整除），立即执行 `cout << "QWER"` 并退出 if-else 结构。注意 if-else if 结构中，一旦第一个条件满足就不再检查后续条件。

### 第12题

下图为n=5时输出的图案，输入一个正整数n，输出一个n+1行特殊的直角三角形，则①处应补充的代码为？

```
*
**
****
******
********
**********
```

```cpp
int n;
cin >> n;
cout << "*" << endl;
for (int i=1; i<=n; i++) {
    for (int j=1; _①_; j++) {
        cout << "*";
    }
    cout << endl;
}
```

A. `j<=2*i`

B. `j<=i`

C. `j<=2*i-1`

D. `j<=2*i+1`

> **【参考答案】A**
>
> **解析：** 观察图案规律（不含第一行的单独 `*`）：
> - i=1：** （2个）= 2×1
> - i=2：**** （4个）= 2×2
> - i=3：****** （6个）= 2×3
> - i=4：******** （8个）= 2×4
> - i=5：********** （10个）= 2×5
>
> 第i行输出 2×i 个 `*`，所以循环条件为 `j<=2*i`。

### 第15题

运行以下程序，输出的结果是？

```cpp
int a[6] = {1, 2, 3, 4, 5, 6};
for (int i=0; i<=5; i++) {
    cout << a[i] << "+";
}
```

A. 1 2 3 4 5 6

B. 1+2+3+4+5+6+

C. 1+2+3+4+5+6

D. 1, 2, 3, 4, 5, 6,

> **【参考答案】B**
>
> **解析：** 循环从 i=0 到 i=5，每次输出 `a[i]` 后紧跟一个 `+`，没有空格分隔。所以输出为 `1+2+3+4+5+6+`（最后一个数字6后面也有一个`+`）。

### 第1题

在C++中，&&表示逻辑运算符"或"。

A. 正确

B. 错误

> **【参考答案】B（错误）**
>
> **解析：** 在C++中，`&&` 表示逻辑运算符**"与"（AND）**，而不是"或"。逻辑"或"的运算符是 `||`。

### 第2题

在C++中，可以通过这种方式定义数组并初始化：`int c[3]={1, 2, 3, 4, 5}`。

A. 正确

B. 错误

> **【参考答案】B（错误）**
>
> **解析：** 数组 `c[3]` 只有3个元素，但初始化列表 `{1, 2, 3, 4, 5}` 提供了5个值，初始值个数超过了数组大小，这在C++中会导致编译错误。

### 第3题

在C++语言中，一维数组的下标是从0开始。

A. 正确

B. 错误

> **【参考答案】A（正确）**
>
> **解析：** 在C++（以及C语言）中，数组下标从0开始。例如定义 `int a[5]`，其元素依次为 a[0]、a[1]、a[2]、a[3]、a[4]，合法下标范围是 0～4。

### 第4题

定义 `int arr[10];` 后，执行 `arr[10]=5;` 不会导致数组越界。

A. 正确

B. 错误

> **【参考答案】B（错误）**
>
> **解析：** `int arr[10]` 定义了10个元素，合法下标为 0～9。`arr[10]` 访问的是第11个位置，已经超出数组范围，属于**数组越界**，会导致未定义行为（可能引起程序崩溃或数据错误）。

### 第5题

下面代码段最终的值是-1。

```cpp
int i=5;
while(i>0) {
    i-=2;
}
```

A. 正确

B. 错误

> **【参考答案】B（错误）**
>
> **解析：** 逐步追踪：
> - 初始：i=5，5>0，执行 i-=2，i=3
> - i=3，3>0，执行 i-=2，i=1
> - i=1，1>0，执行 i-=2，i=-1
> - i=-1，-1>0 为假，循环结束
>
> 循环结束时 i=-1，但题目说"最终的值是-1"——**答案是正确的！**
>
> ⚠️ **注意：** 实际上此题答案应为 **A（正确）**，循环结束后 i 的值确实是 -1。图片中标注的参考答案 B 有误，请以实际运行结果为准。
