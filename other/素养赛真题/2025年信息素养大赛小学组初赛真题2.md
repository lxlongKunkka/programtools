### 第1题

在C++中，表示逻辑运算符"或"的是？

- A. `||`
- B. `&`
- C. `==`
- D. `@`

> **【参考答案】A**
>
> **解析：** 在C++中，逻辑运算符"或"用 `||` 表示；逻辑"与"用 `&&` 表示；`&` 是按位与；`==` 是相等判断；`@` 不是C++运算符。

### 第2题

执行下列代码段，输出结果为？

```cpp
cout << 10%3;
```

- A. 1
- B. 2
- C. 5
- D. 10

> **【参考答案】A**
>
> **解析：** `%` 是取余（取模）运算符。`10 % 3` 表示10除以3的余数，`10 = 3×3 + 1`，余数为 **1**。

### 第3题

下列选项中，输出结果为0的是？

- A. `cout<<"5-5";`
- B. `cout<<5-5;`
- C. `cout<<2*3;`
- D. `cout<<7/2;`

> **【参考答案】B**
>
> **解析：**
> - A：输出字符串 `"5-5"`，即输出 `5-5` 这个字符串，不是0。

> - C：`2*3 = 6`，输出6。
> - D：整数除法 `7/2 = 3`（向下取整），输出3。

### 第6题

下面程序输出的结果是"1 4 7 10"，补全①处的代码。下列选项中哪一项正确？

```cpp
#include <iostream>
using namespace std;
int main () {
    for(int i=1; i<=10; _①_) {
        cout<<i<<" ";
    }
    return 0;
}
```

- A. `i++`
- B. `i*=2`
- C. `i+=3`
- D. `i*2`

> **【参考答案】C**
>
> **解析：** 输出序列为 `1 4 7 10`，差值为3，说明每次 `i` 增加3，即 `i+=3`。
> - `i++`：输出1,2,3,...,10，不符合。
> - `i*=2`：输出1,2,4,8，不符合。
> - `i+=3`：输出1,4,7,10，**符合**。
> - `i*2`：这只是表达式，不改变i的值，会死循环。

- A. `number/1000`
- B. `number/100%10`
- C. `number/10%10`
- D. `number%10`

> **【参考答案】C**

### 第10题

以下程序的功能是对一个正整数n进行数位分离，并按逆序打印每一位，处应该填写？

```cpp
#include<iostream>
using namespace std;
int main() {
    int n;
    cin >> n;
    while(_①_) {
        int d = n%10;
        n /= 10;
        cout << d << " ";
    }
    return 0;
}
```

- A. `n>0`
- B. `n>=0`
- C. `n<0`
- D. `n<=0`

> **【参考答案】A**
>
> **解析：** 程序功能是对正整数n逐位分离（从个位开始逆序输出）。循环条件应为 `n>0`：当n大于0时继续循环取余并整除。若用 `n>=0`，当n变为0时还会多输出一个0，不符合预期。

### 第11题

使用双重for循环结构，求 1! + 2! + 3! + 4! .. +n!（1<=n<=10），则①处应补充的代码是？

```cpp

    int mul = 1;
    for (int j = 1; j <= i; j++) {
        _①_
    }
    sum += mul;
```

- A. `mul+=j`
- B. `mul*=j`
- C. `mul*=i`

### 第12题

以下程序段输出的结果为？

```cpp
int cnt = 0;
for (int i=0; i<5; i++) {
    for (int j=0; j<4; j++) {
        for (int k=0; k<2; k++) {
            cnt++;
        }
    }
}
cout << cnt << endl;
```

- A. 40
- B. 60
- C. 90
- D. 120

> **【参考答案】A**
>
> **解析：** 三重循环，`cnt` 累加次数 = `i`的循环次数 × `j`的循环次数 × `k`的循环次数 = `5 × 4 × 2 = 40`。

### 第1题

在C++中，`==` 可以用来判断两数相等。

- A. 正确
- B. 错误

> **【参考答案】A（正确）**
>

### 第2题

在C++中，一维数组里元素的值可以相等。

- A. 正确
- B. 错误

> **【参考答案】A（正确）**
>
> **解析：** 数组中各元素的值可以相同，例如 `int a[5] = {1,1,1,1,1};` 是完全合法的。数组下标（索引）不同，但值可以相等。

### 第3题

在C++中，执行语句 `int x[2]= {6}; cout << x[1];` 输出结果为9。

- A. 正确
- B. 错误

> **【参考答案】B（错误）**
>
> **解析：** `int x[2] = {6};` 初始化数组，`x[0]=6`，`x[1]` 未显式初始化，默认为 **0**。所以 `cout << x[1]` 输出的是 **0**，而不是9。

---

### 第4题

一个质数的因数个数一定是2个。

- A. 正确
- B. 错误

> **【参考答案】A（正确）**
>
> **解析：** 质数（素数）的定义是：只有1和本身两个因数的大于1的整数。因此质数的因数个数一定恰好是2个。

### 第5题

定义 `int arr[10];` 后，执行 `arr[10]= 5;` 不会导致数组越界。

- A. 正确
- B. 错误

> **【参考答案】B（错误）**
>
> **解析：** `int arr[10]` 定义了一个有10个元素的数组，下标范围为 `arr[0]` 到 `arr[9]`。访问 `arr[10]` 超出了合法范围，会导致**数组越界**（undefined behavior），是错误的。
