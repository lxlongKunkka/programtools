C++ 小数变量: float 与 double

处理带小数点的数字

为什么需要小数？

生活中很多数据不是整数：

  • 身高：1.75 米
  • 体重：45.5 公斤
  • 圆周率：3.14159...

int 只能存整数，存这些数会丢失精度（变成 1, 45, 3）。

float 与 double

C++ 提供了两种存小数的盒子：

类型 全称 精度 (有效数字) 推荐场景
float 单精度浮点数 约 6-7 位 省内存时用
double 双精度浮点数 约 15-16 位 默认推荐

为了计算更准确，竞赛和工程中通常首选 double。

定义与赋值

double pi = 3.14159;
double height = 1.75;
float weight = 45.5f; // float 后面通常加 f

输入输出和 int 一样：

double a;
cin >> a;
cout << a;

小数除法

只要运算中有一个是小数，结果就是小数。

cout << 5 / 2;     // 输出 2 (整数除法)
cout << 5.0 / 2;   // 输出 2.5 (小数除法)
cout << 5 / 2.0;   // 输出 2.5
cout << 1.0 / 3;   // 输出 0.33333...

技巧：想得到小数结果，把其中一个数写成 x.0

科学计数法 (了解)

对于很大或很小的数，可以用 e 表示。

double a = 1e9;   // 1 乘以 10 的 9 次方 (1000000000)
double b = 3e-2;  // 3 乘以 10 的 -2 次方 (0.03)

随堂练习

计算圆的面积。

公式：$S = \pi \times r \times r$

double pi = 3.14159;
double r;
cout << "请输入半径: ";
cin >> r;
cout << "面积是: " << pi * r * r;
