赋值、复合运算与特殊符号

=, +=, (), ,

赋值运算符 =

在 C++ 中，= 不是 "相等"，而是 "赋值"。

意思是：把右边的值，计算出来，交给左边的变量。

int a = 10; // 把 10 给 a
a = 20;     // 把 20 给 a，覆盖原来的 10
a = a + 5;  // 先算 a+5 (25)，再把 25 给 a

注意：10 = a; 是错误的！左边必须是变量。

复合赋值运算符 (偷懒神器)

程序员很懒，不想写 a = a + 5 这么长。

于是发明了 +=。

int money = 100;
money += 50;  // 等价于 money = money + 50;
// 现在 money 是 150

常见的复合运算符

符号 例子 等价于
+= a += b a = a + b
-= a -= b a = a - b
*= a *= b a = a * b
/= a /= b a = a / b
%= a %= b a = a % b

小括号 ()

作用：改变运算优先级。

和数学里一样，括号里的先算。

int x = 2 + 3 * 4;   // 先乘除后加减 -> 2 + 12 = 14
int y = (2 + 3) * 4; // 有括号先算括号 -> 5 * 4 = 20

当你不确定优先级时，加上括号总没错！

逗号 ,

作用：分隔符。

最常见用于一行定义多个变量。

int a, b, c;       // 定义三个变量
int x = 1, y = 2;  // 定义并初始化

(进阶知识：逗号也可以作为运算符连接表达式，从左到右执行，取最后一个值，但初学阶段很少用。)
