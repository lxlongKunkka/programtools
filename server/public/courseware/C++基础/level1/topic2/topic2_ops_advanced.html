除法与取余

特殊的除法

/ 除法
% 取余

在 C++ 中，这两个符号对于整数有着特殊的规则。

整数除法 /

当两个整数相除时，结果只保留整数部分。

小数部分会被直接扔掉（不是四舍五入）！

cout << 6 / 2;   // 输出 3
cout << 7 / 2;   // 输出 3 (0.5 被扔掉了)
cout << 1 / 3;   // 输出 0 (0.333... 被扔掉了)

重点：整数 / 整数 = 整数

取余运算 %

% 用于计算除法后的余数。

cout << 10 % 3;  // 输出 1 (10除以3，商3余1)
cout << 8 % 2;   // 输出 0 (整除，余数为0)
cout << 2 % 5;   // 输出 2 (2除以5，商0余2)

应用场景：判断奇偶数（n % 2）、判断倍数。

危险操作：除以零

在数学中，除数不能为 0。

在编程中，除以 0 会导致程序崩溃 (Crash)！

int a = 10;
int b = 0;
cout << a / b; // ❌ 危险！程序会停止运行
永远不要让除数变成 0
