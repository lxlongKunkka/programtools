Topic 3: 循环结构

Lesson 4: break 与 continue

循环的指挥官：随时停止或跳过

break：彻底终止循环

遇到 break，立即跳出当前循环，不再执行后续次数。

for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // 当 i 是 5 时，直接结束循环
    }
    cout << i << " ";
}
// 输出: 1 2 3 4 (5及之后都不输出了)

continue：跳过本次循环

遇到 continue，跳过本次循环体剩余代码，直接进入下一次循环。

for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // 当 i 是 3 时，跳过打印，直接去 i++
    }
    cout << i << " ";
}
// 输出: 1 2 4 5 (只有 3 没打印)

应用场景对比

break 场景

寻找第一个满足条件的数。

例如：在 1-100 中找到第一个能被 17 整除的数，找到就停。

continue 场景

排除特定条件的数。

例如：打印 1-100，但不要打印 7 的倍数（逢 7 过）。

交互练习：逢 7 过 (continue)

打印 1-20，遇到含 7 或 7 的倍数跳过

交互练习：寻找幸运数字 (break)

在 1-100 中寻找第一个 > 80 且是 9 的倍数的数
