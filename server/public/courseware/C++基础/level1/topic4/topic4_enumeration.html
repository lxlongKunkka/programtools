什么是枚举？

枚举（Enumeration），也叫穷举或暴力破解。

核心思想：一一列举所有可能的情况，并判断是否满足条件。

三要素：

  1. 确定范围 (循环起点和终点)
  2. 确定条件 (if 判断)
  3. 处理结果 (输出或统计)

案例：寻找水仙花数

水仙花数：一个 3 位数，其各位数字的立方和等于它本身。

例如：$153 = 1^3 + 5^3 + 3^3 = 1 + 125 + 27$

for (int i = 100; i <= 999; i++) { // 1. 确定范围：所有3位数
    int a = i / 100;     // 百位
    int b = i / 10 % 10; // 十位
    int c = i % 10;      // 个位
    
    // 2. 判断条件
    if (a*a*a + b*b*b + c*c*c == i) {
        cout << i << " "; // 3. 处理结果
    }
}

案例：百钱百鸡 (简化版)

公鸡 5 元，母鸡 3 元，小鸡 1 元。用 100 元买 100 只鸡。

// x:公鸡, y:母鸡, z:小鸡
for (int x = 0; x <= 20; x++) { // 公鸡最多20只
    for (int y = 0; y <= 33; y++) { // 母鸡最多33只
        int z = 100 - x - y; // 小鸡数量由总数决定
        
        // 价格条件检查
        if (5*x + 3*y + 1*z == 100) {
            cout << "公鸡:" << x << " 母鸡:" << y << " 小鸡:" << z << endl;
        }
    }
}
