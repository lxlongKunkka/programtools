Topic 3: 循环结构

Lesson 7: 数位分离

数字的拆解术：% 10 和 / 10

核心技巧

int n = 12345;

// 1. 取末位：n % 10
int last = n % 10; // 5

// 2. 去末位：n / 10
n = n / 10; // 1234

重复这两个步骤，直到 n 变为 0，就可以处理每一位数字。

示例：求各位数字之和

int n = 2024;
int sum = 0;
while (n > 0) {
    int digit = n % 10; // 取出末位 (4, 2, 0, 2)
    sum += digit;       // 累加
    n /= 10;            // 去掉末位
}
cout << "各位和: " << sum << endl; // 8

示例：数字反转

将 123 变成 321

int n = 123;
int rev = 0;
while (n > 0) {
    int digit = n % 10;
    rev = rev * 10 + digit; // 核心公式：旧反转数*10 + 新位
    n /= 10;
}
cout << "反转后: " << rev << endl;
