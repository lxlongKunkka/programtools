Lesson 5: 分支嵌套（if嵌套if）

什么是分支嵌套？

在一个 if/else 代码块内部，再写 if/else 语句

1. 先判断外层条件是否成立
2. 外层成立 → 才会执行内层判断
3. 外层不成立 → 内层代码直接跳过

核心：层级逻辑，外层是内层的执行前提

嵌套if的基本语法

#include <iostream>
using namespace std;

int main() {
    int age = 25;
    string gender = "女";
    
    // 外层：判断是否成年
    if (age >= 18) {
        cout << "已成年" << endl;
        // 内层：成年的前提下，判断性别
        if (gender == "女") {
            cout << "成年女性" << endl;
        } else {
            cout << "成年男性" << endl;
        }
    } else {
        cout << "未成年" << endl;
    }
    return 0;
}

示例1：先判断性别，再判断年龄

// 需求：判断是否为成年女性
int age = 20;
string gender = "女";

if (gender == "女") {
    // 内层：女性前提下判断年龄
    if (age >= 18) {
        cout << "成年女性" << endl;
    } else {
        cout << "未成年女性" << endl;
    }
} else {
    cout << "非女性" << endl;
}

缩进规范：内层代码缩进4个空格/1个Tab，清晰区分层级

示例2：三数排序（找出最大值）

int a = 12, b = 25, c = 18;
int max_num;

// 外层：先比较a和b
if (a > b) {
    // 内层：a>b的前提下，比较a和c
    if (a > c) {
        max_num = a; // a最大
    } else {
        max_num = c; // c最大
    }
} else {
    // 内层：b>=a的前提下，比较b和c
    if (b > c) {
        max_num = b; // b最大
    } else {
        max_num = c; // c最大
    }
}
cout << "最大值：" << max_num << endl; // 输出25
