Lesson 3: if-else 语句（双分支结构）

if-else 语句的基本结构

双分支：条件为真执行if块，为假执行else块（二选一）

// 语法格式
if (条件表达式) {
    // 条件为真时执行
    语句块1;
} else {
    // 条件为假时执行
    语句块2;
}

if-else 互斥逻辑

if (条件真)
执行if代码块
else代码块跳过
else (条件假)
执行else代码块
if代码块跳过

核心：两个代码块永远只会执行其中一个

示例：判断成绩及格/不及格

#include <iostream>
using namespace std;

int main() {
    int score = 75;
    if (score >= 60) {
        cout << "成绩及格！" << endl;
    } else {
        cout << "成绩不及格！" << endl;
    }
    return 0;
}

score=55时 → 输出"成绩不及格！"

示例：判断奇偶数

int num = 7;
if (num % 2 == 0) {
    cout << num << "是偶数" << endl;
} else {
    cout << num << "是奇数" << endl; // 会输出
}

核心逻辑：能被2整除（余数为0）是偶数，否则是奇数
