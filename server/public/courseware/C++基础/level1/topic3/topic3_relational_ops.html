Lesson 1: 关系运算符

常用关系运算符

用于比较两个值的大小/相等关系，返回布尔值（真/假）

> 大于
< 小于
>= 大于等于
<= 小于等于
== 等于
!= 不等于

关系运算符的使用

#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 5;
    cout << (a > b) << endl;   // 输出 1 (真)
    cout << (a < b) << endl;   // 输出 0 (假)
    cout << (a == b) << endl;  // 输出 0 (假)
    cout << (a != b) << endl;  // 输出 1 (真)
    return 0;
}

C++中，真用1表示，假用0表示

核心易错点：区分 = 和 ==

// 赋值运算符 = 
int num = 8;  // 将8赋值给变量num

// 关系运算符 ==
if (num == 8) {  // 判断num是否等于8
    cout << "相等";
}
❌ 错误示例：if (num = 8) { ... }
这是赋值操作，不是判断！无论num原本值是多少，该条件永远为真

字符的相等性判断

字符比较基于ASCII码值

char c1 = 'A', c2 = 'a';
cout << (c1 == c2) << endl;  // 输出0 ('A'ASCII=65, 'a'=97)
cout << (c1 < c2) << endl;   // 输出1 (65 < 97)
