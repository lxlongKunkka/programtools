Topic 2: 复杂条件构建

Lesson 7: 逻辑运算符（&& / || / !）

逻辑运算符：组合多个条件

&& 
逻辑与 (AND)
所有条件都成立才为真
有一个假则整体为假
||
逻辑或 (OR)
任意一个条件成立即为真
全假才为假
!
逻辑非 (NOT)
对条件取反
真→假，假→真

核心特性：短路求值

一旦能确定整体结果，后续条件不再判断 
#include <iostream>
using namespace std;

int main() {
    // && 短路：第一个条件假，第二个不执行
    int a = 5;
    if (a > 10 && ++a > 0) { // a>10为假，++a不执行
        cout << "执行";
    }
    cout << a << endl; // 输出5（a未自增）

    // || 短路：第一个条件真，第二个不执行
    int b = 5;
    if (b < 10 || ++b > 0) { // b<10为真，++b不执行
        cout << "执行";
    }
    cout << b << endl; // 输出5（b未自增）
    return 0;
}

实战示例1：判断闰年

闰年规则：能被4整除且不能被100整除，或能被400整除

#include <iostream>
using namespace std;

int main() {
    int year = 2024;
    // 组合条件：(能被4整除 && 不能被100整除) || 能被400整除
    bool isLeapYear = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
    
    if (isLeapYear) {
        cout << year << "年是闰年" << endl; // 输出2024年是闰年
    } else {
        cout << year << "年不是闰年" << endl;
    }
    return 0;
}

实战示例2：判断大写元音字母

规则：是大写字母（A-Z）且是元音（A/E/I/O/U）

#include <iostream>
using namespace std;

int main() {
    char ch = 'E';
    // 组合条件：大写字母 && 元音字母
    bool isUpperVowel = (ch >= 'A' && ch <= 'Z') && 
                        (ch == 'A' || ch == 'E' || ch == 'I' || ch == 'O' || ch == 'U');
    
    if (isUpperVowel) {
        cout << ch << "是大写元音字母" << endl; // 输出E是大写元音字母
    } else {
        cout << ch << "不是大写元音字母" << endl;
    }
    return 0;
}
