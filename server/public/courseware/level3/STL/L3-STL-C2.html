C++ Set 核心课程

竞赛解题实战版 · 有序去重的查找神器

为什么需要Set？

先思考：用vector解题时，你是否遇到过这些效率/便捷性问题？
Vector的痛点
✘ 查找元素需要遍历（O(n)），大数据量效率低
✘ 去重需要手动实现（遍历+判断），代码繁琐
✘ 有序输出需要额外sort，多一步操作
✘ 数据流去重+查找场景，代码冗余
Set的核心优势
✔ 自动去重：插入重复元素无效，无需手动处理
✔ 有序存储：默认升序，无需额外排序
✔ 高效查找：O(logn)时间复杂度，远超vector
✔ 简洁API：insert/find/erase一站式操作
核心结论：Set是“有序+唯一”的关联容器，专为“去重+快速查找”场景设计，竞赛中替代vector解决效率问题！
Set底层基于红黑树实现，无需理解底层，只需掌握“什么时候用、怎么用”！

Set基础语法（必掌握）

1. 基础准备

// 1. 包含头文件（必须！）
#include <set>
// 2. 使用命名空间（简化代码）
using namespace std;

2. 定义与初始化

// 方式1：空set（最常用）
set s;                // 存储int类型的空set
set str_s;         // 存储string类型的空set

// 方式2：数组/vector转set（自动去重+排序）
int a[] = {3, 1, 2, 2, 4};
set s1(a, a+5);       // 结果：{1,2,3,4}（去重+升序）

vector v = {5, 3, 5, 1};
set s2(v.begin(), v.end()); // 结果：{1,3,5}

3. 核心操作（竞赛高频）

操作 语法 说明 时间复杂度 插入元素 s.insert(val) 自动去重+排序，重复插入无效 O(logn) 查找元素 s.find(val) 返回迭代器，未找到则为s.end() O(logn) 判断存在 s.count(val) 返回0（不存在）/1（存在） O(logn) 删除元素 s.erase(val) / s.erase(it) 删除指定值/迭代器指向元素 O(logn) 基础判断 s.size() / s.empty() 元素个数 / 是否为空 O(1) 边界元素 s.begin() / s.rbegin() 最小值（首元素）/ 最大值（尾元素） O(1)
易错点：Set不支持下标访问！s[0]、s.at(0)均会报错，只能用迭代器/范围for遍历！

Set基础实战

实战1：自动去重+有序输出
题目：输入若干整数（以-1结束），用set存储并升序输出（去重）。
#include <iostream>
#include <set>
using namespace std;

int main() {
    set s;
    int num;
    
    // 输入数据（以-1结束）
    while (cin >> num && num != -1) {
        s.insert(num);  // 自动去重+排序
    }
    
    // 方式1：迭代器遍历
    cout << "迭代器遍历：";
    for (set::iterator it = s.begin(); it != s.end(); it++) {
        cout << *it << " ";
    }
    cout << endl;
    
    // 方式2：范围for遍历（C++11+，推荐）
    cout << "范围for遍历：";
    for (auto x : s) {
        cout << x << " ";
    }
    cout << endl;
    
    return 0;
}
// 输入：5 2 3 2 5 -1
// 输出：2 3 5
实战2：快速查找+动态增删
题目：输入一个数，存在则删除，不存在则插入，最后输出set大小和最大值。
#include <iostream>
#include <set>
using namespace std;

int main() {
    set s = {1, 3, 5, 7, 9};
    int num;
    cout << "请输入一个整数：";
    cin >> num;
    
    // 判断是否存在
    if (s.count(num)) {
        s.erase(num); // 存在则删除
        cout << num << " 已删除" << endl;
    } else {
        s.insert(num); // 不存在则插入
        cout << num << " 已插入" << endl;
    }
    
    cout << "Set大小：" << s.size() << endl;
    if (!s.empty()) {
        cout << "最大值：" << *s.rbegin() << endl; // rbegin()是反向迭代器
    }
    
    return 0;
}
技巧：s.rbegin()可直接获取最大值，无需遍历，比vector更高效！

Set进阶变体（竞赛高频）

1. unordered_set（无序高效版）

set
✓ 有序存储（升序）
✓ 查找O(logn)
✓ 底层红黑树
✓ 适合需要有序的场景
unordered_set
✗ 无序存储
✓ 查找O(1)（平均）
✓ 底层哈希表
✓ 适合仅需去重+快速查找
// unordered_set使用示例（需包含）
#include <iostream>
#include <unordered_set>
using namespace std;

int main() {
    unordered_set us = {3,1,2,4};
    // 无序输出：顺序不确定（如1 3 2 4）
    for (auto x : us) {
        cout << x << " ";
    }
    
    // 查找效率更高
    if (us.count(2)) {
        cout << "\n2 存在" << endl;
    }
    return 0;
}

2. multiset（允许重复版）

核心特性：有序存储，但允许重复元素，count()返回实际出现次数
#include <iostream>
#include <set>
using namespace std;

int main() {
    multiset ms;
    ms.insert(5);
    ms.insert(5);
    ms.insert(3);
    
    cout << "5出现次数：" << ms.count(5) << endl; // 输出2
    cout << "所有元素：";
    for (auto x : ms) {
        cout << x << " "; // 输出：3 5 5
    }
    return 0;
}

3. 自定义排序（降序/规则）

// 降序排列的set
#include <iostream>
#include <set>
using namespace std;

int main() {
    // greater指定降序
    set> s = {3,1,4,2};
    for (auto x : s) {
        cout << x << " "; // 输出：4 3 2 1
    }
    return 0;
}

Set/Vector/Unordered_set 选型指南

需求场景 优先选择 核心原因 效率对比（数据量1e5） 去重+有序+查找 set 有序+O(logn)查找，满足排序需求 查找：~1ms 去重+无需有序+快速查找 unordered_set O(1)平均查找，效率最高 查找：~0.1ms 频繁修改+随机访问+排序 vector 下标访问+sort灵活，适合频繁修改 查找：~10ms（遍历） 允许重复+有序 multiset 有序+支持重复元素，count()返回实际次数 查找：~1ms 数据量小（<1000） vector 无需额外容器开销，简单够用 无明显差异
实战：集合交集运算
题目：求两个数组的交集（不重复），按升序输出。
#include <iostream>
#include <set>
#include <algorithm> // set_intersection需要
#include <vector>
using namespace std;

int main() {
    int a[] = {1,3,5,7,9};
    int b[] = {2,3,6,7,8};
    set s1(a, a+5), s2(b, b+5);
    vector res;
    
    // 求交集（结果存入vector）
    set_intersection(s1.begin(), s1.end(),
                     s2.begin(), s2.end(),
                     back_inserter(res));
    
    cout << "交集：";
    for (auto x : res) {
        cout << x << " "; // 输出：3 7
    }
    return 0;
}
选型口诀：有序去重选set，无序高效选unordered_set，修改访问选vector！

Set避坑指南（解题必看）

坑1：下标访问Set
错误代码：set s={1,2,3}; cout << s[0];（编译报错）
解决方案：改用迭代器/范围for遍历，或先转vector（vector v(s.begin(), s.end())）
坑2：迭代器失效
错误：auto it = s.begin(); s.erase(it); *it;（erase后it失效）
解决方案：删除时保存下一个迭代器：it = s.erase(it)
注意：insert不会导致其他迭代器失效
坑3：混淆set和multiset的count()
set的count()：仅返回0/1（是否存在）
multiset的count()：返回元素实际出现次数
解决方案：根据是否允许重复选择容器，调用前明确容器类型
坑4：unordered_set自定义类型无哈希函数
错误：unordered_set s;（Person为自定义类型，无哈希）
解决方案：要么改用set，要么为自定义类型编写哈希函数（竞赛少用）
坑5：盲目用set替代vector
小数据量下，vector遍历（O(n)）比set查找（O(logn)）更快
解决方案：数据量<1e4时，优先用vector+sort+find
避坑核心：理解Set的特性（无下标、有序/无序），根据场景选择，不盲目替代！

综合实战 & 课程总结

综合实战：数据流去重判断
题目：输入若干整数，每次输入判断是否重复，重复输出“重复”，否则加入并输出“新增”。
// 用unordered_set实现高效查找
#include <iostream>
#include <unordered_set>
using namespace std;

int main() {
    unordered_set us;
    int num;
    
    cout << "请输入若干整数（输入-1结束）：" << endl;
    while (cin >> num && num != -1) {
        if (us.count(num)) {
            cout << num << " 重复" << endl;
        } else {
            us.insert(num);
            cout << num << " 新增" << endl;
        }
    }
    
    cout << "最终去重结果：";
    for (auto x : us) {
        cout << x << " ";
    }
    return 0;
}
  • Set核心知识点
    • 基础：有序+唯一，无下标访问，insert/find/erase
    • 变体：unordered_set（无序高效）、multiset（允许重复）
    • 选型：有序选set，无序选unordered_set，小数据选vector
    • 避坑：不下标访问，注意迭代器失效，不盲目替代vector
    • 课程核心：Set的价值在于“有序去重+高效查找”，竞赛中精准选型才能发挥最大优势！
