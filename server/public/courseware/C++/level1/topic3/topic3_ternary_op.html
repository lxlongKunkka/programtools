Lesson 4: 三目运算符（if-else简写）

三目运算符基本语法

唯一的三元运算符，是 if-else 的简洁写法

// 语法格式
条件表达式 ? 表达式1 : 表达式2;

// 执行逻辑：
// 条件为真 → 执行表达式1
// 条件为假 → 执行表达式2

核心：一行代码实现二选一逻辑

三目运算符 vs if-else

if-else 写法
int max;
if (a > b) {
    max = a;
} else {
    max = b;
}
三目运算符写法
// 一行搞定
int max = a > b ? a : b;

示例1：求两个数的最大值

#include <iostream>
using namespace std;

int main() {
    int a = 15, b = 8;
    // 三目运算符求最大值
    int max_num = a > b ? a : b;
    cout << "最大值：" << max_num << endl; // 输出15
    
    // 直接输出（更简洁）
    cout << "最大值：" << (a > b ? a : b) << endl;
    return 0;
}

示例2：简单状态判断

int score = 58;
// 判断及格状态
string status = score >= 60 ? "及格" : "不及格";
cout << "成绩状态：" << status << endl; // 输出不及格

// 判断奇偶性
int num = 7;
string type = num % 2 == 0 ? "偶数" : "奇数";
cout << num << "是" << type << endl; // 输出7是奇数

适合：简单的二选一赋值场景
