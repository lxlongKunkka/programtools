Topic 3: 循环结构

循环嵌套

循环中的循环：复杂问题的分解

什么是循环嵌套？

在一个循环的内部，再包含另一个完整的循环结构

// 外层循环
for (int i = 1; i <= 3; i++) {
    // 内层循环
    for (int j = 1; j <= 2; j++) {
        cout << "i=" << i << ", j=" << j << endl;
    }
}

执行顺序：外层循环执行1次，内层循环执行完整的一轮

嵌套循环的执行流程

以上一页代码为例，执行步骤：

  1. i=1 时，j=1 → 输出 "i=1, j=1"
  2. i=1 时，j=2 → 输出 "i=1, j=2"
  3. i=2 时，j=1 → 输出 "i=2, j=1"
  4. i=2 时，j=2 → 输出 "i=2, j=2"
  5. i=3 时，j=1 → 输出 "i=3, j=1"
  6. i=3 时，j=2 → 输出 "i=3, j=2"

总执行次数：外层次数 × 内层次数 = 3 × 2 = 6 次

典型应用：打印图形

打印 4 行 5 列的星号矩阵：

// 控制行数
for (int row = 1; row <= 4; row++) {
    // 控制每行的列数
    for (int col = 1; col <= 5; col++) {
        cout << "* ";
    }
    cout << endl; // 换行
}

输出结果：

* * * * * 
* * * * * 
* * * * * 
* * * * *

进阶应用：九九乘法表

for (int i = 1; i <= 9; i++) {      // 乘数
    for (int j = 1; j <= i; j++) {  // 被乘数，不超过乘数
        cout << j << "*" << i << "=" << j*i << "\t";
    }
    cout << endl;
}

注意内层循环的结束条件与外层变量相关

交互练习：打印直角三角形
