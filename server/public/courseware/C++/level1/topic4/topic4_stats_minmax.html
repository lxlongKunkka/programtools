Topic 3: 循环结构

Lesson 6: 统计与最值

数据侦探：找最大、找最小、数个数

计数器 (Counter)

统计满足特定条件的数据个数。

int count = 0; // 1. 初始化计数器
for (int i = 1; i <= 10; i++) {
    int score;
    cin >> score;
    if (score >= 60) {
        count++; // 2. 满足条件，计数加 1
    }
}
cout << "及格人数: " << count << endl;

擂台法找最大值 (Max)

假设第一个人是擂主，后面的人依次挑战，谁强谁当擂主。

int maxVal = -1; // 初始化为一个很小的数(或第一个数)
int n;
cin >> n; // 输入个数
for (int i = 0; i < n; i++) {
    int x;
    cin >> x;
    if (x > maxVal) { // 挑战成功？
        maxVal = x;   // 换擂主
    }
}
cout << "最大值是: " << maxVal << endl;

找最小值 (Min)

原理相同，只是比较符号相反。

int minVal = 10000; // 初始化为一个很大的数
// ... 循环中 ...
if (x < minVal) {
    minVal = x;
}

⚠️ 最小值初始值一定要足够大，最大值初始值要足够小！
