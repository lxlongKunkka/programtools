C 风格输入输出

scanf & printf

(更快、更强大的格式控制)

为什么要学这个？

  • 速度快：处理百万级数据时，比 cin/cout 快很多。
  • 格式强：保留小数、补前导零非常方便。

所需头文件：

#include <cstdio>
// 或者 #include <stdio.h>

printf (Print Formatted)

语法：printf("格式字符串", 变量1, 变量2...);

核心思想是占位符。

int a = 10;
int b = 20;
// %d 是整数的占位符
printf("a是%d, b是%d", a, b); 
// 输出: a是10, b是20

常用占位符 (背诵全文)

类型 占位符 注意
int %d Decimal
long long %lld Long Long Decimal
float %f Float
double %lf Long Float (输入必用，输出可用%f)
char %c Character

printf 的超能力：格式控制

double pi = 3.1415926;

// 1. 保留小数 (四舍五入)
printf("%.2f", pi);  // 输出 3.14
printf("%.4f", pi);  // 输出 3.1416

// 2. 补前导零 (数字时钟常用)
int time = 5;
printf("%02d", time); // 输出 05 (不足2位补0)

// 3. 设置域宽 (右对齐)
printf("%5d", 123);   // 输出 "  123" (总宽5)

scanf (Scan Formatted)

语法：scanf("格式字符串", &变量);

致命重点：变量前面必须加 & (取地址符)！

int a;
long long b;
// 这里的 & 不能少！
scanf("%d %lld", &a, &b);

(除了读字符串数组外，读基本类型都要加 &)

scanf 的坑

1. 严格匹配：格式串里有什么，输入就得有什么。

scanf("%d,%d", &a, &b); 
// 用户输入必须带逗号：10,20
// 如果输入 10 20 会读取失败

2. double 输入：必须用 %lf，不能用 %f。

double d;
scanf("%f", &d);  // ❌ 错误！读不进去
scanf("%lf", &d); // ✅ 正确
