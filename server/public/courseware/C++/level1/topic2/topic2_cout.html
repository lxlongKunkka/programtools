C++ 输出流: cout

让计算机开口说话

什么是 cout？

cout (Character Output) 是 C++ 中用于向屏幕（标准输出设备）打印信息的工具。

你可以把它想象成一个传声筒，把你的程序想说的话传送到屏幕上。

基本语法

cout << "Hello World";
  • cout：输出命令。
  • <<：插入运算符，像一个箭头，把数据“送给”cout。
  • "Hello World"：要输出的内容，字符串必须用双引号包裹。
  • ;：语句结束的分号，千万别忘了！

连续输出

你可以用多个 << 拼接不同的内容：

cout << "我今年" << 12 << "岁了";

屏幕显示：

我今年12岁了

换行：endl

如果想让输出换到下一行，可以使用 endl (End Line) 或 \n。

cout << "第一行" << endl;
cout << "第二行";

屏幕显示：

第一行
第二行

关于 std::

cout 和 endl 其实住在 std (Standard) 这个家里。

如果不写 using namespace std;，你就得这么写：

std::cout << "Hello" << std::endl;

为了偷懒，我们通常在程序开头加上：

using namespace std;
