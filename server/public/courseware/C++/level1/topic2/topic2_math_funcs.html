常用数学函数

abs, sqrt, floor, ceil, round

头文件

要使用这些数学工具，必须先告诉编译器。

请在程序开头加上：

#include <cmath>

或者使用万能头文件：

#include <bits/stdc++.h>

绝对值 abs()

作用：把负数变成正数，正数不变。

数学符号：$|x|$

cout << abs(-5);    // 输出 5
cout << abs(10);    // 输出 10
cout << abs(-3.14); // 输出 3.14

常用于计算两个数的“距离”：abs(a - b)

平方根 sqrt()

作用：求开根号。

数学符号：$\sqrt{x}$

cout << sqrt(16);  // 输出 4
cout << sqrt(2);   // 输出 1.41421...

注意：参数必须是非负数！sqrt(-1) 会报错 (NaN)。

取整三剑客

把小数变成整数，我们有三种不同的方法。

函数 含义 记忆口诀
floor() 向下取整 地板 (Floor) 在下
ceil() 向上取整 天花板 (Ceiling) 在上
round() 四舍五入 最常用的

地板与天花板

double x = 3.14;
double y = 3.99;

// floor: 只要有小数，统统舍去 (往小了取)
cout << floor(x); // 3
cout << floor(y); // 3

// ceil: 只要有小数，统统进位 (往大了取)
cout << ceil(x);  // 4
cout << ceil(y);  // 4

四舍五入 round()

符合我们数学上的四舍五入规则。

cout << round(3.4); // 3 (舍)
cout << round(3.5); // 4 (入)
cout << round(3.6); // 4 (入)
