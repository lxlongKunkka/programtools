C++ 基础运算 (上)

加法、减法与乘法

算术运算符

+ 加法
- 减法
* 乘法

注意：乘法使用的是星号 *，而不是 x。

加法与减法

用法和数学课上完全一样。

int a = 10;
int b = 5;

cout << a + b;  // 输出 15
cout << a - b;  // 输出 5
cout << a + 100; // 输出 110

乘法运算

计算机键盘上没有 × 号，我们用 * (Shift + 8) 代表乘法。

int price = 5;
int count = 3;
int total = price * count; // 5 * 3 = 15

cout << "总价是: " << total;

运算优先级

和数学一样，先乘除，后加减。

如果有括号 ()，先算括号里面的。

cout << 2 + 3 * 4;    // 先算 3*4=12，再加2，结果 14
cout << (2 + 3) * 4;  // 先算 2+3=5，再乘4，结果 20

难点：不要把括号写成中文的 （）！

随堂练习

编写程序计算长方形的周长。

公式：(长 + 宽) * 2

int length = 10;
int width = 5;
// 请补全代码
int c = (length + width) * 2;
cout << c;
