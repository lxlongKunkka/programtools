2. 核心原理：层层推进

BFS 的核心思想类似于水波纹扩散。

  • 从起点开始，首先访问所有距离为 1 的点。
  • 然后访问所有距离为 2 的点。
  • 以此类推，像地毯式搜索一样向外扩展。

或者想象成朋友圈：

  • 先找你的直接朋友（第1层）。
  • 再找朋友的朋友（第2层）。

3. 数据结构：队列 (Queue)

为了实现"层层推进"，我们需要一个先进先出 (FIFO) 的容器。

  • 入队 (push): 发现新节点时，加入队尾。
  • 出队 (pop): 处理当前节点时，从队首取出。
  • 队首 (front): 当前正在探索的中心点。

5. 代码实现：图的 BFS

使用邻接表 (Adjacency List) 和 `std::queue`。


void bfs(int startNode, vector>& adj, int n) {
    vector visited(n, false);
    queue q;

    // 1. 起点入队并标记
    visited[startNode] = true;
    q.push(startNode);

    while (!q.empty()) {
        // 2. 取出队首
        int u = q.front();
        q.pop();
        cout << u << " "; // 处理节点

        // 3. 遍历所有邻居
        for (int v : adj[u]) {
            if (!visited[v]) {
                visited[v] = true; // 关键：入队时立即标记
                q.push(v);
            }
        }
    }
}

7. 代码实现：迷宫最短路径


struct Point { int x, y, dist; };

int bfs_maze(vector>& grid, Point start, Point end) {
    int rows = grid.size(), cols = grid[0].size();
    vector> visited(rows, vector(cols, false));
    queue q;
    
    q.push({start.x, start.y, 0});
    visited[start.x][start.y] = true;
    
    int dx[] = {-1, 1, 0, 0};
    int dy[] = {0, 0, -1, 1};
    
    while (!q.empty()) {
        Point curr = q.front(); q.pop();
        
        if (curr.x == end.x && curr.y == end.y) return curr.dist;
        
        for (int i = 0; i < 4; i++) {
            int nx = curr.x + dx[i];
            int ny = curr.y + dy[i];
            
            if (nx >= 0 && nx < rows && ny >= 0 && ny < cols && 
                !visited[nx][ny] && grid[nx][ny] == 0) {
                visited[nx][ny] = true;
                q.push({nx, ny, curr.dist + 1});
            }
        }
    }
    return -1; // 无法到达
}

9. 对比：BFS vs DFS

特性 BFS (广度优先) DFS (深度优先)
数据结构 队列 (Queue) 栈 (Stack) / 递归
搜索方式 层层扩散，由近及远 一条路走到黑，回溯
最短路径 保证最短 (无权图) 不保证最短
空间复杂度 较大 (需存储整层节点) 较小 (仅存储路径)
