# C++ 是什么？

## 1. C++ 的历史
C++ 是一种通用的编程语言，由 **Bjarne Stroustrup**（本贾尼·斯特劳斯特卢普）于 1979 年在贝尔实验室开始设计开发。
最初它被称为 "C with Classes"（带类的 C），后来在 1983 年更名为 C++。
C++ 是在 C 语言的基础上发展而来的，它保留了 C 语言的高效性，同时增加了**面向对象编程**（OOP）的特性。

## 2. 为什么要学习 C++？
C++ 是世界上最流行的编程语言之一，广泛应用于：
- **系统软件**：操作系统（如 Windows, Linux）、驱动程序。
- **游戏开发**：绝大多数 3A 大作（如《魔兽世界》、《绝地求生》）的引擎都是用 C++ 写的。
- **高性能应用**：金融交易系统、图像处理、人工智能底层框架。
- **嵌入式系统**：智能家电、汽车电子。

## 3. C++ 与其他语言的对比
| 特性 | C++ | Python | Java |
| :--- | :--- | :--- | :--- |
| **运行速度** | 非常快（编译型） | 较慢（解释型） | 中等（虚拟机） |
| **学习难度** | 较高 | 较低 | 中等 |
| **应用领域** | 底层开发、游戏、高性能计算 | 数据分析、AI、Web 开发 | 企业级应用、安卓开发 |

## 4. 我们的开发环境
在接下来的课程中，我们将使用 **Dev-C++** 或者 **Visual Studio Code** 作为我们的编程工具。
它们可以帮助我们编写代码、检查错误，并运行程序看到结果。

## 5. 总结
C++ 是一门强大且高效的语言，虽然入门稍难，但学会它将为你打下坚实的计算机科学基础！
