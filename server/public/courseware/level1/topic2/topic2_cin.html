C++ 输入流: cin

让程序“听懂”你的话

什么是 cin？

cin (Character Input) 是 C++ 中用于从键盘读取数据的工具。

如果说 cout 是程序的嘴巴，那么 cin 就是程序的耳朵。

它允许程序暂停下来，等待用户输入信息。

基本语法

int a;
cin >> a;
  • cin：输入命令。
  • >>：提取运算符，像一个箭头，把键盘输入的数据“送进”变量里。
  • a：接收数据的变量。

注意：箭头方向和 cout 是相反的！
cout << (送出去), cin >> (拿进来)

输入的过程

cout << "请输入你的年龄: ";
int age;
cin >> age;  // 程序在这里暂停
cout << "明年你就 " << age + 1 << " 岁了";

当程序运行到 cin 时：

  1. 光标闪烁，等待输入。
  2. 用户输入数字并按下回车键。
  3. 数字被存入变量 age，程序继续执行。

一次输入多个数据

你可以用多个 >> 连续接收输入。

int a, b;
cin >> a >> b;

用户输入时，可以用空格或回车来分隔两个数字。

10 20
(或者)
10
20

实战：加法计算器

让我们做一个真正的计算器！

#include <iostream>
using namespace std;

int main() {
    int num1, num2;
    
    cout << "请输入两个数字: ";
    cin >> num1 >> num2;
    
    int sum = num1 + num2;
    cout << "它们的和是: " << sum << endl;
    
    return 0;
}
