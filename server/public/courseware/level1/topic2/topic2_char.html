C++ 字符变量: char

存储单个符号的容器

什么是字符？

char (Character) 用于存储单个字符。

它可以是字母、数字符号、标点符号等。

'A' 'x' '5' '#'

注意：只能存一个！不能存 "Hello"。

定义与赋值

字符必须用单引号 ' ' 包裹。

char level = 'A';
char gender = 'M';
char symbol = '+';

错误示范：

char c = "A";  // ❌ 错误！双引号是字符串
char c = 'AB'; // ❌ 错误！只能存一个字符

ASCII 码 (神奇的数字)

计算机内部其实只认识数字。

每个字符都有一个对应的数字编号，叫 ASCII 码。

  • 'A' 对应 65
  • 'a' 对应 97
  • '0' 对应 48
char c = 'A';
cout << (int)c; // 输出 65 (强制转换成整数查看)

字符也能运算？

因为字符本质是数字，所以它们可以加减！

char c = 'A';
c = c + 1;
cout << c; // 输出 'B' (65 + 1 = 66, 66是'B')

应用：大小写转换 ('a' - 'A' = 32)。

随堂练习

编写程序，输入一个大写字母，输出它的小写字母。

提示：小写字母比大写字母的 ASCII 码大 32。

char big;
cout << "请输入大写字母: ";
cin >> big;
char small = big + 32;
cout << "对应的小写是: " << small;
