数位分离

如何把一个整数“拆”开？

核心工具：% 和 /

两大核心工具

1. 取个位 (模 10)

任何整数 % 10，都能得到它的最后一位。

123 % 10  ->  3
4590 % 10 ->  0

2. 删个位 (除 10)

任何整数 / 10，都能砍掉最后一位。

123 / 10  ->  12
4590 / 10 ->  459

实战演练：拆解 385

我们要依次取出 5, 8, 3。

1. 取个位：385 % 10 = 5 (得到 5)

2. 删个位：385 / 10 = 38 (剩下 38)

3. 取个位：38 % 10 = 8 (得到 8)

4. 删个位：38 / 10 = 3 (剩下 3)

5. 取个位：3 % 10 = 3 (得到 3)

6. 删个位：3 / 10 = 0 (结束)

三位数的通用公式

假设有一个三位数 n (例如 385)

int n = 385;

int ge  = n % 10;          // 个位: 5
int shi = n / 10 % 10;     // 十位: 38 % 10 = 8
int bai = n / 100;         // 百位: 3

解释：

  • 十位：先除以10变成两位数(38)，再取个位(8)。
  • 百位：直接除以100，砍掉后两位(85)，剩下(3)。

随堂练习：各位数之和

输入一个三位数，输出它各位数字相加的和。

例如：输入 123，输出 6 (1+2+3)

int n;
cin >> n;
int g = n % 10;        // 个位
int s = n / 10 % 10;   // 十位
int b = n / 100;       // 百位
cout << g + s + b;

进阶：数字反转

输入 123，输出 321。

思路：把拆出来的数字，重新组合。

int n = 123;
int g = n % 10;      // 3
int s = n / 10 % 10; // 2
int b = n / 100;     // 1

// 重新组合: 个位*100 + 十位*10 + 百位
int reverse = g * 100 + s * 10 + b; 
cout << reverse; // 321
