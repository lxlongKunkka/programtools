Lesson 2: if 语句（单分支结构）

if 语句的基本结构

单分支：满足条件才执行代码，不满足则跳过

// 语法格式
if (条件表达式) {
    // 条件为真时执行的代码块
    语句1;
    语句2;
}

// 示例：判断是否打印
if (10 > 5) {
    cout << "条件为真，执行代码"; // 会输出
}

if 语句执行流程

1. 计算条件表达式的值（真/假）
2. 若为真 → 执行花括号内代码
3. 若为假 → 跳过花括号内代码
4. 继续执行后续代码

示例：判断是否成年

#include <iostream>
using namespace std;

int main() {
    int age = 20;
    // 判断年龄是否≥18
    if (age >= 18) {
        cout << "你已成年！" << endl; // 会执行
    }
    cout << "程序结束";
    return 0;
}

如果age=16，花括号内代码不会执行

示例：判断正数

int num = 8;
if (num > 0) {
    cout << num << "是正数" << endl; // 会输出
}

num = -5;
if (num > 0) {
    cout << num << "是正数" << endl; // 不会输出
}
注意：花括号 {} 可以省略（仅一行代码时），但不推荐！
省略后只有紧跟的第一行代码属于if语句
