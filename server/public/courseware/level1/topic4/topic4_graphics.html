Lesson 9: 图形打印 (嵌套循环)

用代码画画：行与列的艺术

嵌套循环原理

外层循环控制行 (Row)，内层循环控制列 (Col)。

for (int i = 1; i <= 3; i++) { // 外层：3行
    for (int j = 1; j <= 5; j++) { // 内层：每行5个
        cout << "*";
    }
    cout << endl; // 每行结束换行
}
/* 输出：
*****
*****
*****
*/

直角三角形

规律：第 i 行有 i 个星号。

for (int i = 1; i <= 5; i++) { // 5行
    for (int j = 1; j <= i; j++) { // 内层循环次数随 i 变化
        cout << "*";
    }
    cout << endl;
}
/* 输出：
*
**
***
****
*****
*/

九九乘法表

for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {
        cout << j << "*" << i << "=" << i*j << "\t";
    }
    cout << endl;
}
