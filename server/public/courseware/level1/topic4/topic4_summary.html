挑战1：质数判断 (Prime Check)

质数：只能被 1 和它本身整除的数 (如 2, 3, 5, 7)。

int n = 13;
bool isPrime = true; // 假设是质数
for (int i = 2; i < n; i++) { // 尝试除以 2 到 n-1
    if (n % i == 0) {
        isPrime = false; // 发现能整除，说明不是质数
        break; // 找到一个因子就够了，不用继续找
    }
}
if (isPrime && n > 1) cout << "是质数";
else cout << "不是质数";

挑战2：最大公约数 (GCD)

辗转相除法 (欧几里得算法)

int a = 24, b = 18;
while (b != 0) {
    int temp = a % b;
    a = b;
    b = temp;
}
cout << "最大公约数: " << a << endl; // 6
