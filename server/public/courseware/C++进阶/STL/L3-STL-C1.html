C++ Vector 核心课程

竞赛解题实战版 · 动态数组从入门到精通

为什么需要Vector？

先思考：用普通数组解题时，你是否遇到过这些问题？
普通数组的痛点
✘ 长度固定，无法动态扩容
✘ 输入数据量未知时，易越界/浪费空间
✘ 插入/删除元素需手动移动数据
✘ 无内置长度判断，需手动记录
Vector的优势
✔ 动态扩容，按需分配空间
✔ 无需预设长度，适配任意数据量
✔ 内置增删方法，操作更便捷
✔ size()/empty() 安全判断长度
核心结论：竞赛解题中，Vector几乎可以完全替代普通数组，更安全、更高效！
Vector本质是「动态数组」——底层仍是连续内存，但自动管理扩容和内存分配。

Vector基础语法（必掌握）

1. 基础准备

// 1. 包含头文件（必须！）
#include <vector>
// 2. 使用命名空间（简化代码）
using namespace std;

2. 定义与初始化

// 方式1：空vector（最常用）
vector v;                // 存储int类型的空vector
vector str_v;         // 存储string类型的空vector

// 方式2：定长初始化（指定长度+初始值）
vector v1(5, 0);         // 5个元素，每个元素初始值为0
vector v2(10);           // 10个元素，默认初始值为0

// 方式3：数组转vector
int a[] = {1,2,3,4,5};
vector v3(a, a+5);       // 从数组a的第0个到第4个元素初始化

3. 核心访问操作

操作 语法 说明 解题场景 尾部添加 v.push_back(val) 向vector末尾添加元素 输入数据、动态收集元素 尾部删除 v.pop_back() 删除vector最后一个元素 栈模拟、数据筛选 元素访问 v[i] / v.at(i) 下标访问，at()有越界检查 遍历、修改元素 长度判断 v.size() 返回vector元素个数 循环遍历、边界判断 空判断 v.empty() 是否为空（size()==0） 避免访问空vector
易错点：空vector直接用v[0]会越界！务必先判断v.empty()或v.size()>0！

Vector基础实战

实战1：动态收集数据并求和
题目：输入若干整数（以-1结束），存入vector，输出所有元素的和。
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector v;
    int num, sum = 0;
    
    // 输入数据（以-1结束）
    while (cin >> num && num != -1) {
        v.push_back(num);  // 尾部添加元素
    }
    
    // 遍历求和
    for (int i = 0; i < v.size(); i++) {
        sum += v[i];
    }
    
    cout << "元素个数：" << v.size() << endl;
    cout << "元素和：" << sum << endl;
    return 0;
}
实战2：定长初始化与修改
题目：创建长度为5、全0的vector，修改第3个元素为10，输出所有元素。
#include <iostream>
#include <vector>
using namespace std;

int main() {
    // 初始化：5个元素，初始值0
    vector v(5, 0);
    
    // 修改第3个元素（下标从0开始，第3个是v[2]）
    v[2] = 10;
    
    // 范围for循环遍历（C++11+）
    for (auto x : v) {
        cout << x << " ";
    }
    // 输出：0 0 10 0 0
    return 0;
}
技巧：范围for循环（for(auto x : v)）适合仅读取元素的场景，代码更简洁！

Vector进阶操作（解题效率翻倍）

1. 排序与反转（高频！）

#include <iostream>
#include <vector>
#include <algorithm>  // sort/reverse需要的头文件
using namespace std;

int main() {
    vector v = {5, 2, 8, 1, 9};
    
    // 升序排序（默认）
    sort(v.begin(), v.end());
    // 输出：1 2 5 8 9
    
    // 降序排序
    sort(v.rbegin(), v.rend());
    // 输出：9 8 5 2 1
    
    // 反转vector
    reverse(v.begin(), v.end());
    // 输出：1 2 5 8 9
    
    return 0;
}

2. 任意位置增删

#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector v = {1, 2, 3, 4, 5};
    
    // 在第2个位置（v[1]后）插入6
    v.insert(v.begin() + 2, 6);
    // 结果：1 2 6 3 4 5
    
    // 删除第4个元素（v[3]）
    v.erase(v.begin() + 3);
    // 结果：1 2 6 4 5
    
    // 清空所有元素（size()变为0）
    v.clear();
    
    return 0;
}

3. 容量优化（提升效率）

reserve(n)：提前预留n个元素的空间，避免多次扩容（适合已知数据量）
// 已知要存储1e5个元素，提前预留空间
vector v;
v.reserve(100000);  // 预留1e5个空间，避免扩容耗时
for (int i = 0; i < 100000; i++) {
    v.push_back(i); // 无扩容开销，效率更高
}
注意：insert/erase中间元素会移动数据（O(n)时间），解题中尽量用尾部操作！

二维Vector（矩阵操作核心）

1. 二维Vector定义

#include <iostream>
#include <vector>
using namespace std;

int main() {
    // 方式1：空二维vector
    vector> v;
    
    // 方式2：3行4列，初始值0（最常用）
    vector> v1(3, vector(4, 0));
    
    // 方式3：动态添加行
    vector> v2;
    v2.push_back(vector{1, 2, 3});  // 第一行
    v2.push_back(vector{4, 5, 6});  // 第二行
    
    return 0;
}

2. 二维Vector遍历

// 遍历3行4列的二维vector
vector> v(3, vector(4, 0));
// 修改第2行第3列（v[1][2]）为5
v[1][2] = 5;

// 方式1：普通双重循环
for (int i = 0; i < v.size(); i++) {       // 遍历行
    for (int j = 0; j < v[i].size(); j++) { // 遍历列
        cout << v[i][j] << " ";
    }
    cout << endl;
}

// 方式2：范围for循环
for (auto row : v) {
    for (auto val : row) {
        cout << val << " ";
    }
    cout << endl;
}
实战：二维vector求和
题目：创建3行4列的二维vector，计算所有元素的和、每行的和。
核心：二维vector的每行长度可以不同（不规则矩阵），解题中默认是规则矩阵！

Vector避坑指南（解题必看）

坑1：下标越界
错误代码：vector v; cout << v[0];（空vector访问）
解决方案：先判断!v.empty() 或 i < v.size()
调试技巧：用v.at(i)替代v[i]，越界时抛异常定位错误
坑2：混淆resize和reserve
resize(n)：改变元素个数（创建/删除元素），会初始化值
reserve(n)：仅预留空间，不改变元素个数（size()不变）
解题建议：已知数据量用reserve(n)，需要初始化用resize(n, val)
坑3：迭代器失效
错误：auto it = v.begin(); v.push_back(10); *it;（扩容后迭代器失效）
解决方案：扩容后重新获取迭代器，避免迭代器跨push_back使用
坑4：二维vector行未初始化
错误：vector> v(3); v[0][0] = 1;（行是空的）
解决方案：初始化每行：v[i].resize(m, 0) 或直接定义v(n, vector(m, 0))
避坑口诀：先判空再访问，提前reserve提效率，尾部操作更高效！

综合实战 & 课程总结

综合实战：偶数前移，奇数后移
题目：输入若干整数，将偶数移到前面，奇数移到后面，保持相对顺序。
// 解法：双vector分离，再合并
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector v, even, odd;
    int num;
    
    // 输入数据
    while (cin >> num) {
        if (num % 2 == 0) {
            even.push_back(num); // 偶数
        } else {
            odd.push_back(num);  // 奇数
        }
    }
    
    // 合并：偶数 + 奇数
    v = even;
    for (auto x : odd) {
        v.push_back(x);
    }
    
    // 输出结果
    for (auto x : v) {
        cout << x << " ";
    }
    return 0;
}
