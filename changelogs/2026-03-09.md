# 2026-03-09 工作日志

## 主题：题目导出包优化 & NFLSOJ 抓取质量提升

---

### 背景

programtools 支持从 NFLSOJ（nflsoi.cc:20035，基于 SYZOJ）抓取题目并生成标准化下载包（含 problem_zh.md、run.py、data_generator.py、solution.md 等）。本次集中修复了一批影响导出质量的 bug，包括 LaTeX 公式残缺、sample.zip 处理方式、样例编号退化、图片占位符未还原等问题。

---

### 工作内容

#### 1. `sample.zip` 处理方式调整（多次迭代）

| 阶段 | 方式 |
|------|------|
| 原始 | 浏览器端用 JSZip 解压 sample.zip 内容到 `additional_file/` |
| 中间版 | 直接把 sample.zip 放在 zip 根目录，run.py 启动时解压到 `additional_file/` |
| 最终版 | sample.zip 放入 `additional_file/sample.zip`，不再解压；run.py 不做任何解压操作 |

- **run.py 解压逻辑 Python 作用域 bug**：函数靠后有 `import zipfile`（打包段），导致 Python 把 `zipfile` 视为整个函数的局部变量，在前面用时报"未赋值"。修复：删去函数体内重复的 `import zipfile`，顶层已导入。

#### 2. MD 文件末尾追加 sample 链接

有 `sample.zip` 时，`problem_zh.md`、`problem_zh_TW.md`、`problem_en.md` 末尾追加：
```
[sample](file://sample.zip)
```

- 初版写成 `[sample]:(file://sample.zip)`（多余冒号），本次修正为标准 Markdown 链接格式。

#### 3. NFLSOJ LaTeX 公式空格丢失（`\le n` → `\len`）

**根本原因**：`decodeMmlChildren()` 把 MML 子节点拼接时用 `join('')`，导致 `\le` + `n` = `\len`（LaTeX 命令后紧跟字母，被解析为新命令）。

**修复**（`server/routes/nflsoj.js`）：
```js
// 之前
return getMmlChildrenArr(el, $).join('')
// 之后
return getMmlChildrenArr(el, $).join(' ')
```

#### 4. 翻译结果图片占位符未还原（`IMGPH0` 残留）

**根本原因**：服务端把图片替换为 `IMGPH0IMGPH` 发给 AI，AI 有时只保留 `IMGPH0`（丢弃尾部 `IMGPH`），原来用 `.split('IMGPH0IMGPH').join(原图)` 精确匹配，遇到截断形式就还原失败。

**修复**（`server/routes/ai.js`）：改用正则，同时兼容完整和截断两种形式：
```js
str.replace(/IMGPH(\d+)IMGPH|IMGPH(\d+)(?!\d)/g, (match, a, b) => {
  const key = `IMGPH${a ?? b}IMGPH`
  return imageMap[key] || match
})
```

#### 5. 翻译后样例编号全变成 input1/output1

**根本原因**：`server/routes/ai.js` 后处理逻辑中有一段代码会把所有 `input1` 块强行合并为一个，同时把 `input\d+` 连续块全部清空，导致多样例退化为单个编号。

**修复**：完整删除该段破坏性后处理逻辑（约 25 行）。

**同步更新 prompt**（`server/prompts/translate.js`）：
```
- ## 样例 (使用 ```input1 ```output1 ```input2 ```output2 格式，多样例依次递增编号)
```

#### 6. `problem_zh_TW.md` 加入导出包

- 批量导出和单任务导出均添加 `problem_zh_TW.md`（原始抓取内容）。
- `run.py` 打包段也将其打入最终 `***ed.zip`，并加入排除扫描列表。

#### 7. 标题中 "A. " 前缀点号去除

NFLSOJ 题目标题形如 `"A. 鸭子与按钮"`，下载包文件夹名不应含点。

```js
title = title.replace(/^([A-Za-z0-9]+)\. /, '$1 ')
```
