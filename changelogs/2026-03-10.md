### 工作内容

#### 1. 章节视频功能（完整实现）

为课程系统新增视频播放支持，覆盖数据库、后端、编辑器、学生端全链路。

**数据库** (`server/models/CourseLevel.js`)
- `chapterSchema` 新增 `videoUrl: { type: String }` 字段

**后端路由** (`server/routes/course.js`)
- POST `/levels/:id/topics/:topicId/chapters`：解构并写入 `videoUrl`
- PUT `/levels/:id/topics/:topicId/chapters/:chapterId`：更新时同步写入 `videoUrl`

**编辑器** (`src/components/editor/ChapterEditor.vue`)
- 新增视频链接输入框，支持 Bilibili 链接 / 纯 BV 号 / COS 直链
- `isBilibili(url)` 方法：识别 `bilibili.com`、`b23.tv`、纯 BV 号（`/^BV[a-zA-Z0-9]+$/`）
- 实时预览提示（已识别为 Bilibili 视频 / 直链视频）

**课程编辑面板** (`src/components/course/CourseEditorPanel.vue`)
- `editingChapter` 初始化、`fetchChapterContent`、`saveChapter` 均同步处理 `videoUrl`

**学生端** (`src/pages/ChapterDetail.vue`)
- `availableTabs` 计算属性：根据 `resourceUrl` / `content` / `videoUrl` 决定显示哪些 Tab，**与 `contentType` 完全解耦**
- 新增视频 Tab，切换到视频时显示播放区域

#### 2. Bilibili 视频防护层

参考 nebula-edu 实现，嵌入 Bilibili 播放器时添加多层保护：

| 元素 | 作用 |
|------|------|
| `.bilibili-click-blocker-top` | 覆盖顶部 12%，拦截 B站 Logo/标题点击跳转 |
| `.bilibili-rclick-blocker` | 覆盖 72% 区域，`bottom:28%` 留出控件栏+倍速面板，`@contextmenu.prevent` 禁右键，`@mousedown` 左键穿透 |
| `handleBilibiliOverlay` | 左键点击时临时 `pointer-events:none`（200ms），让点击穿透到 iframe |
| `requestBilibiliFullscreen` | 通过 `$refs.bilibiliIframeRef.requestFullscreen()` 实现外部全屏按钮 |

- Bilibili 工具栏最终只保留"⛶ 全屏播放"按钮（重播/倍速因跨域无法生效，已移除）

#### 3. 直链视频倍速控制

直链视频（COS / mp4）下方显示倍速工具栏：

- 支持 1x / 1.25x / 1.5x / 1.75x / 2x
- `setSpeed(speed)` 直接设置 `video.playbackRate`
- `reapplySpeed()` 监听 `loadedmetadata`，切换视频后自动还原所选倍速
- 当前倍速高亮显示（`.active` 样式）

#### 4. PPT Prompt 反引号转义修复 (`server/prompts/ppt.js`)

【技术要求】第 2 条中描述 CSS 属性的反引号未加 `\` 转义，导致模板字符串提前终止。

```js
// 修复前（字符串截断）
`body { background-color }` 和 `.slide { width / height }`

// 修复后
\`body { background-color }\` 和 \`.slide { width / height }\`
```
