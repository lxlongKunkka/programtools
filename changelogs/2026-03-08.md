# 2026-03-08 工作日志

## 主题：AtCoder ABC 题目 GESP 知识点标签体系

---

### 背景

programtools 平台收录了 AtCoder ABC 系列题目（共 2279 道），原先只有人工维护的 gesp 等级标签（gesp1~gesp10），缺乏具体知识点标签。本次工作通过抓取 atcoder-tags 社区标签，建立与 GESP 知识体系的映射，并批量写入数据库。

---

### 工作内容

#### 1. 抓取 AtCoder 社区标签
- 脚本：`scripts/scrape_atcoder_tags.js`
- 方式：按分类页（/explain?tag=xxx）抓取，通过代理 `127.0.0.1:10808`
- 结果：**3870 道** ABC 题目，**101 个**唯一标签，保存至 `scripts/atcoder_abc_tags.json`

#### 2. 建立 AtCoder → GESP 知识点映射
- 脚本：`scripts/gen_gesp_problem_list.js`（含完整 TAG_MAP）
- 101 个 AtCoder 标签 → **69 个 GESP 知识点**
- 标签处理决策：
  - 过滤（不映射）：`Easy`、`思维`、`小波矩阵`、`矩阵树定理`、`Marathon`、`April-Fool` 等
  - 规范化：`DP基础 / DP进阶 → DP`、`字符串处理 → 字符串`
  - 新增至 GESP：`折半搜索`（加入 gesp9）
- 输出：`scripts/atcoder_abc_tags_with_gesp.json`（含 `gesp_labels` 和 `gesp_levels`）

#### 3. 更新 GESP_TAGS.yaml
- 在 gesp9「基础算法」分类下新增「**折半搜索**」

#### 4. 批量写入数据库
- 脚本：`scripts/apply_atcoder_gesp_tags.js`
- 连接 Hydro MongoDB，`domainId: 'atcoder'`，`docType: 10`（仅题目）
- 规则：只追加（`$addToSet`），不修改已有 gesp 等级标签
- 结果：**2279** 道题中匹配 1350 道，**618 道**新增了知识点标签，732 道已有标签跳过

#### 5. 生成参考文档
| 文件 | 内容 |
|------|------|
| `scripts/atcoder_tags_to_gesp.md` | AtCoder 标签 ↔ GESP 知识点双向对照表 |
| `scripts/gesp_problem_list.md` | 按 69 个知识点分类的题目列表，含难度色标（灰/茶/绿/水/青/蓝/黄/橙） |
| `scripts/atcoder_problem_gesp_tags.md` | 3870 道题的 GESP 标签一览 |
| `scripts/atcoder_tagged_problems.tsv` | 当前 DB 中 1624 道已打标签题目清单 |

#### 6. 排查 DB 文档数量差异
- 问题：脚本查到 3100 条，UI 显示 2279 道题
- 原因：Hydro `document` 集合混合多种类型，`docType=10` 才是题目
- 各 docType 分布：10（题目）2279 / 11（题解）430 / 30（作业）285 / 其他若干
