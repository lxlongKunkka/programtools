## 主题：课程体系结构重组 & 知识点 DAG 修复

---

### 背景

上一阶段完成了 L1-L10 共 382 章的课程内容填充后，发现知识点依赖图（GESP_TAGS.mmd）中的边存在严重视觉问题：箭头倒置、大量跨多列的长弧线、同列节点间的零宽渲染。与此同时，课程章节的专题归属与学习顺序也存在不合理处（快速幂、组合数学、矩阵快速幂分级过高），章节 id 因多次迁移出现错位。

---

### 工作内容

#### 1. DAG 边重写（GESP_TAGS.mmd）

**问题**：边集存在以下三类问题：
- 跨 3+ 级远程边（如 `A2→G46`、`C11→G6`），bezier 曲线穿越多列形成极长斜线
- 同列内部边（如 `E11→E12`、`G35→G38`），x 方向接近 0，变成竖向短线
- 过时节点引用（旧 E4/F2/F3/G24 等迁移后未更新）

**修复原则**：所有边严格限制为 **相邻级（N→N+1）或最多跨一级（N→N+2）**，删除全部同列边和远程跨列边。

**结果**：边总数从 ~90 条减至 ~65 条，DAG 视觉效果大幅改善。

---

#### 2. 节点排列顺序优化（GESP_TAGS.mmd）

**问题**：g5/g6/g7/g8 各列内节点定义顺序与上游节点垂直位置不匹配，导致 bezier 曲线出现大幅向下弯曲（如函数(B5, g4-top) → 递归(C13, g5-bottom)）。

**修复**：按「接收同一父节点的下游节点优先排在列上方」原则，重排 g5/g6/g7/g8 四列节点：
- g5：STL → 数论 → 高精度 → 前缀和/差分 → 二分 → 双指针 → 贪心 → 递归 → 分治
- g6：栈 → 队列 → 树 → 哈夫曼树 → 图论基础 → DFS → BFS → DP → 线性DP → 背包DP
- g7：DFS/BFS进阶 → 快速幂/倍增/LCA → 离散化/ST表 → 哈希 → 单调栈/队列 → 区间/树上/二维DP
- g8：组合数学/数论 → 并查集/优先队列 → 图论 → 树上算法 → 树状数组/线段树

---

#### 3. GESP_TAGS.yaml 标签顺序同步

将 gesp4~gesp8 的标签列表顺序对齐课程体系主题顺序（Level4~Level8 的专题排列），确保 YAML 与课程结构一一对应，便于题目打标签时参考。

---

#### 4. 专题跨级迁移（DB + YAML + MMD 三处同步）

| 操作 | 原位置 | 新位置 | 说明 |
|------|--------|--------|------|

| 组合数学 + 快速幂合并入数学基础 | Level5 独立专题 | Level5 数学基础 | 删除旧「排列组合」章节（重复），合并为 10 章的统一数学基础专题 |
