# 图编辑器使用指南

## 访问地址
http://localhost:5173/graph-editor

## 功能特性

### 基本操作
- **添加节点**: 双击画布空白处 或 点击左侧"➕ 添加节点"按钮
- **删除节点/边**: 右键点击要删除的元素
- **移动节点**: 左键拖拽节点
- **选中元素**: 单击节点或边

### 边操作
1. **手动添加边**:
   - 在左侧输入起点和终点节点 ID
   - 可选：输入权重（需先勾选"显示权重"）
   - 点击"添加边"按钮

2. **切换有向/无向**: 勾选/取消勾选顶部的"有向图"选项

### 批量导入
支持标准图格式：
```
节点数 边数
u1 v1 w1
u2 v2 w2
...
```

例如：
```
5 6
1 2 3
1 3 2
2 4 5
3 4 1
3 5 4
4 5 2
```

### 布局选项
- **默认布局**: 力导向布局（节点自动分散）
- **层次布局**: 适合有向无环图（DAG）
- **环形布局**: 节点均匀分布在圆周上

### 导出功能

1. **边列表格式** - 用于 OJ 题目输入
   ```
   5 6
   1 2 3
   1 3 2
   ...
   ```

2. **邻接表** - 文本格式
   ```
   1: 2, 3
   2: 4
   3: 4, 5
   ...
   ```

3. **邻接矩阵** - 矩阵格式
   ```
   5
   0 3 2 0 0
   0 0 0 5 0
   ...
   ```

4. **JSON** - 包含完整图数据
   ```json
   {
     "directed": false,
     "nodes": [...],
     "edges": [...]
   }
   ```

## 使用场景

### 1. 图论题目测试
- 快速创建测试用例
- 导出边列表直接用于 OJ 输入
- 可视化验证图的连通性

### 2. 算法教学演示
- 可视化图的结构
- 展示最短路径、拓扑排序等算法输入
- 学生更直观理解图论概念

### 3. 数据生成
- 为 CyaRon 生成图数据
- 快速验证手工设计的图案例
- 导出多种格式供不同工具使用

## 快捷操作提示

- **双击空白** → 添加节点
- **右键元素** → 删除
- **拖拽节点** → 调整位置
- **Ctrl+C** → 复制导出结果（在导出弹窗中）

## 技术栈
- Vue 3
- vis-network (图可视化)
- 响应式设计，支持大规模图编辑
